Opciones de juegoen Ubuntu

Esta sección describe las diferentes opciones disponibles que controlan cómo se compilarán tus proyectos de juegos en Ubuntu(Linux). Las diferentes secciones son:

General

En esta pestaña se establece el nombre de su juego tal y como se mostrará mientras se ejecuta, y también puede proporcionar un correo electrónico de contacto y una descripción (corta y larga), así como una URL para la página de inicio del juego o del desarrollador.

Aquí también tienes la opción es para habilitar Vapor en tu juego. Si habilitas esta opción, asegúrate de que tienes instalado el SDK de Steamworks (ver aquí) y que has añadido el ID de la aplicación de Steam en las Opciones Generales del juego.

Por último, tienes la opción de Desactivar la caja de arena del sistema de archivos. Marcando esta opción se deshabilitará la caja de arena de GameMaker Studio 2 para guardar y cargar archivos, permitiendo que sus juegos accedan a los archivos desde cualquier parte del sistema que ejecuta el juego. Esta opción está desmarcada por defecto, ya que recomendamos mantener el sandbox de GameMaker Studio 2 para los archivos en todos los sistemas y cuando se marca, algunas áreas de guardado/carga pueden seguir sin estar disponibles dependiendo de los permisos a nivel de sistema operativo. Para obtener más información sobre la caja de arena, consulte las páginas sobre el sistema de archivos de GameMaker Studio 2.

Gráficos

Las opciones gráficas son las que debes configurar para determinar cómo tu juego utilizará la tarjeta gráfica de tu máquina Ubuntu de destino. Se incluyen las siguientes opciones para que las modifiques:

  • Iniciar en pantalla completa: Si esta opción está marcada, el juego se iniciará en modo de pantalla completa, de lo contrario, se iniciará en modo de ventana. Esta opción está desactivada por defecto.
  • Permitir el cambio de pantalla completa: Con esta opción marcada, el usuario puede cambiar de pantalla completa a ventana y viceversa utilizando los atajos estándar del sistema operativo Ubuntu. Esta opción está desactivada por defecto.
  • Interpolar los colores entre píxeles: Activa la interpolación lineal, que básicamente "suaviza" los píxeles. Para gráficos de píxeles nítidos, debería estar desactivado, pero si tiene bonitas mezclas alfa y gráficos con bordes suavizados, es mejor dejarlo activado.Está desactivado por defecto.
  • Mostrar Cursor: Cuando esta opción está activada, se mostrará el cursor del sistema operativo por defecto en tu juego. Si se desactiva, mientras el ratón esté sobre la ventana del juego, no habrá cursor visible a menos que haya creado uno como parte del proyecto del juego (está activado por defecto).
  • Utiliza la sincronización para evitar el tearing: Esto activa o desactiva la sincronización v. Ten en cuenta que si tienes un juego con una velocidad de sala de 120 y el jugador tiene un monitor con una tasa de refresco de 60, al activar esta opción también se bloqueará la velocidad del juego a 60. Esta opción está desactivada por defecto.
  • Permitir el cambio de tamaño de la ventana: Marcar esta opción permite al usuario cambiar el tamaño de la ventana del juego (la opción Ventana sin bordes debe estar desactivada para que esto funcione). Esta opción está desactivada por defecto.
  • Escala: Aquí puede elegir mantener la relación de aspecto (de modo que una habitación de 4:3 se verá en "buzón" en una de 16:9) o escalar completamente (estirando la imagen para que se ajuste a la pantalla completa).

    ADVERTENCIA Ladesactivaciónde la superficie de aplicación deshabilitará todas las opciones de escalado establecidas en las Opciones de Juego de Windows hasta que se vuelva a activar. Consulte la sección La superficie de aplicación para obtener más detalles.

Por último, existe la opción de establecer el tamaño de la página de la textura. El tamaño por defecto (y más compatible) es 2048x2048, pero puedes elegir entre 256x256 y 8192x8192. En también hay un botón marcado Vista previa que generará las páginas de texturas para esta plataforma y luego abrirá una ventana para que puedas ver cómo quedan. Esto puede ser muy útil si desea ver cómo se estructuran las páginas de textura estructuradas y evitar tener páginas de textura más grandes (o más pequeñas) de lo necesario. Para más información sobre las páginas de texturas, consulta aquí.

NOTA: Ten en cuenta que cuanto mayor sea el tamaño de la página de texturas, menos compatible será tu juego en los PC con especificaciones inferiores.

Imágenes

Aquí puede añadir una pantalla de bienvenida para su juego, y establecer el icono del juego que se utilizará. El icono debe ser una imagen de 64x64 píxeles en el formato .png, y la pantalla de inicio debe ser al menos del mismo tamaño que la primera habitación de tu juego (o su puerto de vista) y ser también de formato .png. La pantalla de inicio se mostrará mientras se carga el juego, y puede activarse o desactivarse marcando la opción Usar pantalla de bienvenida (que está desactivada por defecto).

Vale la pena señalar que GameMaker Studio 2 tiene una herramienta generadora de imágenes del proyecto que se puede utilizar para crear automáticamente todas las imágenes necesarias para todas las diferentes plataformas de destino de su juego para las que se compila el juego. Si utilizas esta herramienta, debes revisar las imágenes creadas para asegurarte de que son las que necesitas.
