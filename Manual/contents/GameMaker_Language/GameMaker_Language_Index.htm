Índice deidiomas de GameMaker

El Lenguaje GameMaker (también llamado simplemente GML) es el lenguaje de scripting propietario de GameMaker Studio 2. Este lenguaje está estructurado para permitir a los usuarios crear sus juegos de una manera intuitiva y flexible a la vez que ofreciendo toda la potencia de cualquier otro lenguaje de programación importante. También es la base de Drag and Drop™ y puede utilizarse junto con éste si es necesario.
