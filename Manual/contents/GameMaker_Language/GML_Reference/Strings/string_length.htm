longitud_de_cadena

Esta función devuelve el número de caracteres que componen una cadena dada. Puede ser útil para cosas como calcular cuándo limitar la longitud de los caracteres de una entrada de texto personalizada (por ejemplo, limitar el nombre de un jugador a 10 caracteres). Recuerde que esto es diferente de ancho_de_cadena() en que mide el número de caracteres de la cadena, no su anchura como se dibuja en la pantalla en píxeles.

Sintaxis:

string_length(string);

Argumento Descripción
cadena La cadena para medir el número de caracteres de.

Devuelve:

Real

Ejemplo:

if (string_length(nombre) > 10)
{
nombre = string_copy(nombre, 1, 10);
}

Esto comprobará si la cadena de nombre es mayor de diez caracteres y, si lo es, copiará y utilizará sólo los diez primeros caracteres.
