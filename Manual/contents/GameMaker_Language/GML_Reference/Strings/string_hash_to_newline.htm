string_hash_to_newline

Esta función devuelve una cadena en la que el símbolo "#" se ha convertido en una nueva línea. Así que una cadena formateada como:

"Hola#Mundo"

Se imprimiría en la pantalla como:

Hola
Mundo

IMPORTANTE! Esta función se proporciona para la compatibilidad de importación entre GameMaker Studio 2 y las versiones anteriores del software y, como tal, esta función no debe utilizarse excepto cuando sea necesario para replicar una funcionalidad obsoleta. En su lugar debería utilizar los caracteres de escape para marcar una nueva línea. Para obtener más información sobre la compatibilidad de importación, consulte aquí.

Sintaxis:

string_hash_to_newline(string);

Argumento Descripción
cadena La cadena a convertir en varias líneas

Devuelve:

Cadena

Ejemplo:

var str = string_hash_to_newline("Hola#Mundo");
draw_text(32, 32, str);

El código anterior convierte la cadena con el símbolo hash en una cadena dividida en dos líneas y luego la imprime en la pantalla.
