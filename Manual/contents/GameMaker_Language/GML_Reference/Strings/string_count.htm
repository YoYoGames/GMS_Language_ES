cuenta_de_cadenas

Esta función devolverá la cantidad de veces que la subcadena dada aparece dentro de una cadena específica. De este modo, puede comprobar cuántas veces se repite una sola letra o una frase en una sección del texto almacenado.

Sintaxis:

string_count(substr, str);

Argumento Descripción
sustrato La subcadena a comprobar en la cadena.
str La cadena a comprobar.

Devuelve:

Real

Ejemplo:

str1 = "Hola Mundo";
ocount = string_count( "o", str1 );

Esto establecerá la variable ocount al número de "o" en str1, en este caso 2.
