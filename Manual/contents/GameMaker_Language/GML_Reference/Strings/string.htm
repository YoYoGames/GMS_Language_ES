cadena

Con esta función puede convertir cualquier número real en una cadena. Si el número real es un entero, se guardará sin decimales, de lo contrario, se guardará con dos decimales. Si necesita más decimales, utilice la función string_format(). También tenga en cuenta que el uso de esta función en una variable que almacena una matriz, una estructura de datos o una estructura convertirá el contenido de estas variables en una cadena que puede ser enviada a la consola o guardada en un archivo para su depuración.

Sintaxis:

string(val);

Argumento Descripción
cadena El número real que se convertirá en una cadena.

Devuelve:

Cadena

Ejemplo:

draw_text(100, 100, "Puntuación: " + string(puntuación) + " / Salud: " + string(salud));

El código anterior utiliza la función de cadena para dibujar tanto números reales como cadenas (draw sólo acepta una cadena o un real, pero no ambos).
