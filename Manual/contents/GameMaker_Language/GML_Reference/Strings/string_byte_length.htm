string_byte_length

Esta función devuelve el número de bytes de una cadena, pero debe tener en cuenta que debido a que se mantienen como UTF8, esto no será igual a su longitud de cadena.

Sintaxis:

string_byte_length(cadena);

Argumento Descripción
cadena La cadena para medir el número de bytes de.

Devuelve:

Real

Ejemplo:

bytesize = string_byte_length("Hola Mundo");

Esto establecería bytesize al número de bytes en "Hello World".
