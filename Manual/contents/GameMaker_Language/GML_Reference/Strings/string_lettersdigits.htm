letrasdigitoscadenas

Esta función devolverá una copia de una cadena dada con todo menos sus letras y dígitos eliminados, lo que significa que puede utilizarse para eliminar cualquier carácter no deseado (como "#" o "?") de, por ejemplo, un nombre de usuario o una contraseña.

NOTA: Esta función sólo devuelve el alfabeto inglés de 26 letras de la A a la Z (mayúsculas o minúsculas) y los números del 0 al 9.

Sintaxis:

string_lettersdigits(string);

Argumento Descripción
cadena La cadena a convertir en letras y dígitos.

Devuelve:

Cadena

Ejemplo:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
draw_text(32,32, "¡Contraseña inválida! ¡Sólo letras y números, por favor!");
}

El código anterior comprobará la longitud de una cadena con la longitud de la misma cadena, pero con todos los símbolos eliminados. Si no son iguales, se dibuja un mensaje en la pantalla.
