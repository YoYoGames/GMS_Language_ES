string_delete

Puede utilizar esta función para eliminar una parte específica de una cadena. Por lo tanto, usted proporciona la cadena de entrada y la posición inicial y final dentro de esa cadena para eliminar caracteres (el índice comienza en 1) y la función devolverá una nueva cadena sin esa sección en ella.

Sintaxis:

string_delete(str, index, count);

Argumento Descripción
str La cadena a copiar y borrar.
índice La posición del primer carácter a eliminar.
cuenta El número de caracteres a eliminar.

Devuelve:

Cadena

Ejemplo:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Esto hará que str2 sea "Hola Mundo", ya que elimina la "o" extra de "Hola", es decir, la cadena cuenta desde la primera letra 5 lugares, y luego elimina 1 carácter).
