ord

Esta función toma una cadena de entrada de un solo carácter y devuelve el valor Unicode (UTF8) para ese carácter. Tenga en cuenta que cuando se utiliza con las funciones teclado_comprobación* la cadena de entrada sólo puede tener un carácter de longitud y sólo puede ser un número del 0 al 9 o un carácter romano en mayúsculas de la A a la Z.

Sintaxis:

ord(string);

Argumento Descripción
cadena La cadena con la que se busca el código Unicode.

Devuelve:

Real

Ejemplo:

if keyboard_check(ord("W"))
{
y -= 4;
}

Esto moverá la instancia de llamada cuatro píxeles hacia arriba si se mantiene pulsada la tecla W.
