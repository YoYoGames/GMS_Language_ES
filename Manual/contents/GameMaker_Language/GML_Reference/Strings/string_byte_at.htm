string_byte_at

Devuelve el valor del byte crudo como un valor real en una posición dada en la cadena dada.

Sintaxis:

string_byte_at(str, index);

Argumento Descripción
str La cadena a comprobar.
índice La posición desde la que se obtiene el byte.

Devuelve:

Real

Ejemplo:

newbyte = string_byte_at("Hola Mundo", 5);

Esto establecerá newbyte en el valor de byte crudo de la sexta letra de "Hello World".
