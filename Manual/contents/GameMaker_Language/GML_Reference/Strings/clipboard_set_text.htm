clipboard_set_text

Esta función establece el portapapeles para mantener la cadena definida. Puede establecerla como una cadena vacía "" para eliminar el texto del portapapeles.

NOTA: Esta función sólo es válida para el Windows objetivo.

Sintaxis:

clipboard_set_text(string);

Argumento Descripción
cadena El texto (en forma de cadena) que debe contener el portapapeles.

Devuelve:

N/A

Ejemplo:

if clipboard_has_text()
{
str = clipboard_get_text();
clipboard_set_text(");
}

El código anterior comprueba el portapapeles en busca de texto y, si lo contiene, lo lee como una cadena en la variable "str". Por último, el portapapeles se borra estableciendo una cadena vacía.
