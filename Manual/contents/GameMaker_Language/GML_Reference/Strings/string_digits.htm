dígitos_de_cadena

Puede utilizar esta función para analizar una cadena dada y obtener cualquier número de ella. Por ejemplo, digamos que tiene este texto - "Tengo 81 años". Con esta función obtendrías una cadena de retorno de "81".

Sintaxis:

string_digits(cadena);

Argumento Descripción
cadena La cadena de la que obtener los dígitos.

Devuelve:

Cadena

Ejemplo:

var t_str = string_digits(input_str);
age = real(t_str);

El código anterior tomará la cadena de entrada, la despojará de todos los caracteres que no sean números y, a continuación, establecerá la variable edad para mantener el valor numérico real de la cadena de retorno (así, si la cadena de entrada fuera -por ejemplo- "Tengo 18 años", la función devolvería "18").
