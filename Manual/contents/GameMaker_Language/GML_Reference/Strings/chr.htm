chr

Esta función devuelve una cadena que contiene el carácter que se relaciona con el código Unicode de entrada para su visualización. Este carácter depende de la página de códigos del conjunto de caracteres de las fuentes de dibujo actuales y, si no se establece ninguna fuente, utilizará la página de códigos por defecto para la máquina.

Sintaxis:

chr(val);

Argumento Descripción
val El valor del código Unicode para obtener la cadena

Devuelve:

Cadena

Ejemplo:

mystring = chr(53) + chr(48);

Esto establecería mystring en "50" (una cadena, no un entero) ya que chr(53) es "5" y chr(48) es "0".
