copiar_cadenas

Con esta función puede seleccionar fácilmente un número de caracteres de una cadena para copiarlos en otra. El primer carácter de una cadena siempre se indexa como 1 y no como 0, como es de esperar, por lo que para copiar (por ejemplo) los cinco primeros caracteres de la cadena se tendría string_copy(str, 1, 5). A continuación se ofrece otro ejemplo.

Sintaxis:

string_copy(str, index, count);

Argumento Descripción
str La cadena a copiar.
índice La posición del primer carácter de la cadena a copiar (numerada a partir de 1)
cuenta El número de caracteres, a partir de la posición del primero, a copiar

Devuelve:

Cadena

Ejemplo:

nombre = cadena_de_teclado;
si (longitud_de_cadena(nombre) > 15)
{
cadena_de_teclado = cadena_copia(nombre, 1, 15);
}

El código anterior permitirá al jugador introducir una cadena a través del teclado, que se almacena en la variable nombre. Esta variable se comprueba para ver si excede la longitud máxima de 15 caracteres y si lo hace, la cadena_del_teclado se sustituye por una copia de 15 caracteres de la cadena del nombre. Esto limita efectivamente la entrada del jugador a 15 caracteres.
