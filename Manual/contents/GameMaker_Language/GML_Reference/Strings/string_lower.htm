string_lower

Con esta función puede forzar que una cadena contenga sólo caracteres en minúsculas.

NOTA: Esta función sólo detecta el alfabeto inglés de 26 letras de la A a la Z.

Sintaxis:

string_lower(string);

Argumento Descripción
cadena La cadena a convertir en minúsculas.

Devuelve:

Cadena

Ejemplo:

str1 = "Hola Mundo";
str2 = string_lower(str1);

El código anterior establecerá str2 como "hola mundo".
