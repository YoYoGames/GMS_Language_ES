string_pos_ext

Esta función devolverá la posición de los caracteres de una instancia de una subcadena dentro de una cadena, buscando hacia adelante en la cadena desde la posición dada como posición inicial. La función devolverá 0 si no se encuentra la cadena buscada, o la posición del primer carácter de la cadena de búsqueda si se encuentra. Tenga en cuenta que para el soporte de legado las cadenas están indexadas desde 1, por lo que 1 es la primera posición de la cadena, no 0 como podría esperarse.

Sintaxis:

string_pos_ext(substr, str, start_pos);

Argumento Descripción
sustrato La subcadena a buscar en la cadena.
str La cadena.
start_pos La posición de partida para la búsqueda.

Devuelve:

Entero

Ejemplo:

if (string_pos_ext(",", text, 20) != 0)
{
string_insert(nombre, texto, string_pos_ext(",", texto, 20));
}

El código anterior busca en la cadena almacenada en la variable "texto" una coma después del 20º carácter, y si la encuentra inserta la subcadena "nombre" en esa posición.
