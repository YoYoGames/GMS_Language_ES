real

Esta función puede utilizarse para convertir una cadena dada en un número real. Cuando se utiliza esta función, se tienen en cuenta los números, los signos menos, los puntos decimales y las partes exponenciales de la cadena, mientras que otros caracteres (como las letras) provocarán un error de error. Si sabe, o sospecha, que una cadena puede tener otros caracteres, puede utilizar string_digits() para eliminar todos los caracteres no numéricos, antes de utilizar esta función para convertir la cadena resultante en un número real.

Sintaxis:

real(cadena);

Argumento Descripción
cadena La cadena a convertir en un valor real.

Devuelve:

Real

Ejemplo:

var t_str = string_digits(input_str);
age = real(t_str);

El código anterior tomará la cadena de entrada, la despojará de todos los caracteres que no sean números y luego establecerá la variable "edad" para mantener el valor numérico real de la cadena de retorno.
