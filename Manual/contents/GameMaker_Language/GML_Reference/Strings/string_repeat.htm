string_repeat

Esta función simplemente devuelve la misma cadena repetida un número determinado de veces sobre sí misma.

Sintaxis:

string_repeat(str, count);

Argumento Descripción
str La cadena a repetir.
cuenta El número de veces que se repite la cadena.

Devuelve:

Cadena

Ejemplo:

str1 = "Hola Mundo";
str2 = string_repeat(str1, 3);

El código anterior establecerá en str2 el texto "Hello WorldHello WorldHello World".
