string_height_ext

Esta función devolverá la altura (en píxeles) de la cadena de entrada, teniendo en cuenta la separación entre líneas y el ancho del salto de línea (que se define como el número de píxeles que puede ocupar la cadena antes de que se inserte un salto de línea). Es muy útil para calcular las distancias entre los elementos de texto en función de la más alta de las letras que componen la cadena, así como la propia longitud de la cadena cuando se reparte en varias líneas como se dibujaría con dibujar_texto_texto() utilizando la fuente definida actualmente. La separación y la anchura pueden establecerse en -1 para obtener el espaciado por defecto.

Sintaxis:

string_height_ext(string, sep, w);

Argumento Descripción
cadena La cadena para medir la altura de.
sep La distancia en píxeles entre las líneas de texto como si la cadena se estuviera dibujando.
w La anchura máxima (en píxeles) de la cadena antes de un salto de línea, como si la cadena fuera dibujada.

Devuelve:

Real

Ejemplo:

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100);

El código anterior obtendrá la altura de la cadena dada, teniendo en cuenta la separación entre líneas y el ancho del salto de línea, y luego dibujará dos líneas de texto, utilizando la altura total de la cadena devuelta como separador.
