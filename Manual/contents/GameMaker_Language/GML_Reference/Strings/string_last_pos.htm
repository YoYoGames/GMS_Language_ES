string_last_pos

Esta función devolverá la posición del carácter de una instancia de una subcadena dentro de una cadena, buscando desde el final de la cadena hasta el principio (por lo que es la inversa de string_pos(). La función devolverá 0 si la cadena buscada no se encuentra, o la posición del primer carácter de la cadena buscada si se encuentra. Tenga en cuenta que para el soporte de legado las cadenas están indexadas desde 1, por lo que 1 es la primera posición de la cadena, no 0 como podría esperarse.

Sintaxis:

string_last_pos(substr, str);

Argumento Descripción
sustrato La subcadena a buscar en la cadena.
str La cadena.

Devuelve:

Entero

Ejemplo:

if (string_last_pos(",", text) != 0)
{
string_insert(nombre, texto, string_last_pos(",", texto));
}

El código anterior busca en la cadena almacenada en la variable "texto" una coma, comenzando la búsqueda desde el final de la cadena. Si encuentra una, inserta la subcadena "nombre" en esa posición.
