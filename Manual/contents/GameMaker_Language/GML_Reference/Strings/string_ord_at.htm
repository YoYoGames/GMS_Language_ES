cadena_ord_at

Puede utilizar esta función para devolver un código de carácter específico en una posición concreta dentro de una cadena, con el índice empezando en 1 para el primer carácter. Si no se encuentra ningún carácter o la cadena es más corta que el valor dado al índice, se devuelve -1.

Sintaxis:

string_ord_at(str, index);

Argumento Descripción
str La cadena a comprobar.
índice La posición de la que obtener el código de carácter.

Devuelve:

Real

Ejemplo:

str = "Hola Mundo";
char_code = string_ord_at(str, 7);

Esto obtendrá el código del séptimo carácter (donde "H" cuenta como el primero) en la cadena "str" y lo almacenará en la variable "char_code".
