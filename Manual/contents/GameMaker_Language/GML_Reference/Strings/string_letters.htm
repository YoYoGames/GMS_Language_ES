letras_de_cadena

Con esta función puede eliminar todos los caracteres que no se clasifican como letras. Esto es útil para evitar que la gente introduzca caracteres no deseados en una entrada de texto (como introducir "#" para forzar una nueva línea).

NOTA: Esta función sólo detecta y devuelve el alfabeto inglés de 26 letras de la A a la Z (mayúsculas o minúsculas).

Sintaxis:

cadena_de_letras(cadena);

Argumento Descripción
cadena La cadena a convertir en letras.

Devuelve:

Cadena

Ejemplo:

nombre_de_usuario = cadena_de_letras(nombre_de_usuario);

El código anterior establecerá la variable "nombre de usuario" para que sólo contenga los caracteres de la cadena original (si la cadena original era "MacSw33ny", la cadena devuelta sería "MacSwny").
