formato_de_cadena

Convierte un número real en una cadena utilizando su propio formato, donde puede elegir cuántos "lugares" se guardan en la cadena y cuántos decimales se guardan también. Ambos pueden ser muy útiles, algunos juegos prefieren mostrar una puntuación como un número determinado de dígitos, mientras que el control sobre los decimales puede ser bueno para una alta precisión que los dos decimales de cadena() no pueden proporcionar. Si el número de lugares especificado es mayor que el valor a mostrar y/o el número más los lugares decimales que se han especificado es menor que el total de lugares, entonces se añadirán espacios antes del valor para compensar la diferencia (véase el ejemplo siguiente). Se añadirán ceros a la derecha del punto decimal si el valor dado es menor que el total y el número de decimales a incluir. El formato por defecto es sin espacios extra a la izquierda, y sólo dos decimales a la derecha, por ejemplo "265,73".

Sintaxis:

string_format(val, tot, dec);

Argumento Descripción
val El número real que se convertirá en una cadena.
tot El número total de posiciones del número principal a mostrar. Se insertarán ceros o espacios para que coincida con esto.
dec El número de decimales a incluir.

Devuelve:

Cadena

Ejemplo:

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

Esto establecerá str1 como " 1234", str2 como "3 .1415926535 " y str3 como " 3.14159 ".
