string_char_at

Puede utilizar esta función para devolver un carácter específico en una posición concreta dentro de una cadena, con el índice empezando en 1 para el primer carácter. Si no se encuentra ningún carácter o la cadena es más corta que el valor dado al índice, se devuelve una cadena vacía "" se devuelve una cadena vacía.

Sintaxis:

string_char_at(str, index);

Argumento Descripción
str La cadena a comprobar.
índice La posición desde la que obtener el personaje.

Devuelve:

Cadena

Ejemplo:

str1 = "Hola Mundo";
str2 = string_char_at(str1, 7);

Esto pondrá en str2 el séptimo carácter ("H" contando como el primero) de la cadena str1, en este caso "W".
