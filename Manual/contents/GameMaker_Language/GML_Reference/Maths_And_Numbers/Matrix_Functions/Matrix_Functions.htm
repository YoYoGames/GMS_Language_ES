Funciones matriciales

Una matriz es una colección de números dispuestos en un número fijo de filas y columnas. Normalmente los números son reales, pero no siempre tiene que ser así. La siguiente imagen muestra en general cómo se construye una matriz con cuatro filas y cuatro columnas:

La fila superior es la fila 1, la columna más a la izquierda es la columna 1, y esta matriz es una matriz 4x4 porque tiene cuatro filas y cuatro columnas (se pueden construir matrices de otros tamaños). Al describir matrices, el formato es siempre filas / columnas, y cada número que compone una matriz se llama elemento de la matriz. Los elementos de una matriz tienen ubicaciones específicas, descritas por su posición de fila y columna, donde la esquina superior izquierda de la matriz es la fila 1 columna 1. En el ejemplo de la matriz anterior, el elemento de la fila 1 columna 1 es el valor "1". El elemento de la fila 2 columna 3 es el valor valor "4,8".

Las matrices son muy importantes para muchas cosas, pero se utilizan sobre todo en el tratamiento del espacio 3D, por ejemplo, se pueden utilizar para establecer la vista de la cámara o para trasladar o transformar un modelo. Las siguientes funciones son para usar una pila de matrices, que es similar a una pila DS, pero diseñada para usar sólo con matrices. Es una forma práctica de aplicar múltiples operaciones matriciales (como las transformaciones) una tras otra al crear una escena 3D.

  • matrix_stack_clear
  • matrix_stack_set
  • matrix_stack_push
  • matrix_stack_pop
  • matrix_stack_top
