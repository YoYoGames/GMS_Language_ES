matrix_build_projection_perspective_fov

Esta función construye una matriz de proyección en perspectiva basada en el campo de visión, utilizando los parámetros especificados que se indican a continuación.

Sintaxis:

matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar);

Argumento Descripción
fov El ángulo del campo de visión.
aspecto La relación de aspecto del campo de visión.
znear El plano de recorte cercano.
zfar El plano de recorte lejano.

Devuelve:

Índice de la matriz

Ejemplo:

projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

El código anterior crea una matriz de proyección del campo de visión que se almacena en una variable. Esta matriz se utiliza entonces para configurar la cámara asignada al puerto de vista[0].
