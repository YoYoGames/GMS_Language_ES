matrix_build_projection_ortho

Esta función construye una matriz de proyección ortográfica basada en los parámetros especificados que se indican a continuación (este es el método de proyección por defecto que se utiliza cuando se crea una habitación en GameMaker Studio 2 sin cambiar nada). A veces es necesario cambiar de una proyección en perspectiva a una proyección ortográfica que es para lo que sirve esta función. Normalmente se utiliza para dibujar una superposición, por ejemplo, para mostrar la puntuación u otros aspectos, ya que esto da una vista "plana" de los elementos dibujados (es decir: sin perspectiva) en un juego 3D. Vea la imagen de abajo para tener una idea de la diferencia entre las vistas ortográficas y en perspectiva.

NOTA: Es posible que también tenga que desactivar temporalmente la eliminación de la superficie oculta si quiere que la información se dibuje independientemente del valor de la profundidad actual.

Sintaxis:

matrix_build_projection_ortho(width, height, znear, zfar);

Argumento Descripción
w La anchura de la proyección.
h La altura de la proyección.
znear El plano de recorte cercano.
zfar El plano de recorte lejano.

Devuelve:

Índice de la matriz

Ejemplo:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

El código anterior crea una nueva matriz de vista y una matriz ortográfica, almacena sus IDs en variables locales y luego las utiliza para establecer las matrices de vista y proyección para la cámara asignada al puerto de vista[0].
