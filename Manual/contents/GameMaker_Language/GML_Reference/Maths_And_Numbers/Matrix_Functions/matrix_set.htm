conjunto_de_matriz

Esta función toma un array de 16 valores, correspondientes a un tipo de matriz 4x4 dado, donde los elementos [0 - 3] serían la fila 1, los elementos [4 -7] serían la fila 2, etc... (ver la imagen en la página principal). Se puede crear una matriz de este tipo utilizando los botones matrix_build() o matrix_get() o simplemente construyendo la matriz tú mismo y pasándola a la función. Los tipos de matriz disponibles disponibles son la vista, la proyección y el mundo, para lo cual se debe utilizar una de las siguientes constantes:

Constante Descripción
vista_de_matriz La matriz de vista actual
proyección_de_matriz La matriz de proyección actual
matrix_world La matriz mundial actual

Sintaxis:

matrix_set(tipo, matriz);

Argumento Descripción
tipo El tipo de matriz para obtener los valores de (ver las constantes listadas arriba)
matriz Los datos de la matriz como un array

Devuelve:

NA

Ejemplo:

matrix_set(matrix_world, m_array);

El código anterior establecerá los valores de la matriz del mundo actual a los almacenados en la matriz del array "m_array".
