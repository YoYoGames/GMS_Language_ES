matrix_stack_is_empty

Esta función permite comprobar si la pila de matrices está vacía (devuelve true) o no (devuelve false).

Sintaxis:

matrix_stack_is_empty();

Devuelve:

Booleano

Ejemplo:

if !matrix_stack_is_empty()
{
matrix_stack_clear();
}

El código anterior comprueba la pila de la matriz para ver si está vacía y si no lo está la borra.
