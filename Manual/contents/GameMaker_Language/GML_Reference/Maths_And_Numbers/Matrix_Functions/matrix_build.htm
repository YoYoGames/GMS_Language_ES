matrix_build

Esta función se puede utilizar para crear su propia matriz personalizada y devolverá un valor de índice para esta nueva matriz que debe ser almacenado en una variable para su futura referencia y uso. La matriz en sí se crea como una matriz de 16 valores donde los primeros 4 elementos son la fila 1, los segundos 4 elementos son la fila 2, etc... de una matriz 4x4.

NOTA: Cuando se construye una nueva matriz de esta manera el orden de operación es YXZ.

Sintaxis:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);

Argumento Descripción
x El componente x del vector de traslación.
y El componente y del vector de traslación.
z El componente z del vector de traslación.
xrotación El ángulo a rotar alrededor del eje x (en grados °).
yrotación El ángulo a rotar alrededor del eje y (en grados °).
xzotación El ángulo a rotar alrededor del eje z (en grados °).
xscale La cantidad de la escala x.
yscale La cantidad de la escala y.
zscale La cantidad de la escala z.

Devuelve:

Índice de la matriz (matriz 4x4)

Ejemplo:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

El código anterior construirá una nueva matriz transformada y almacenará el índice de la matriz resultante en la variable "t_matrix".
