distancia_al_objeto

Esta función calcula la distancia desde el borde del cuadro delimitador de la instancia que llama hasta el borde más cercano de la instancia más próxima del objeto especificado. El objeto puede ser un índice de objeto o un ID de instancia específico, así como el palabra clave y la distancia se devuelve en píxeles. Tenga en cuenta que si alguno de los objetos no tiene un sprite o una máscara definida, los resultados serán incorrectos.

Sintaxis:

distancia_al_objeto(obj);

Argumento Descripción
obj El objeto a comprobar.

Devuelve:

Real.

Ejemplo:

if (distance_to_object(obj_Player) < range)
{
canshoot = true;
}

El código anterior comprobará la distancia al objeto jugador y si es menor que el valor almacenado en la variable "range" la variable "canshoot" se pone a true.
