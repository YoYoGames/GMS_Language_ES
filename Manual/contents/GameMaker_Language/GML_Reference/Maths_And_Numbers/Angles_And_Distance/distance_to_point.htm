distancia_al_punto

Esta función calcula la distancia desde el borde de la caja delimitadora de la instancia que llama hasta la posición x/y especificada en la habitación, con el valor de retorno en píxeles. Tenga en cuenta que si el objeto que llama no tiene sprite o no tiene definida una máscara, los resultados serán incorrectos.

Sintaxis:

distancia_al_punto(x, y);

Argumento Descripción
x La posición x a comprobar.
y La posición y a comprobar.

Devuelve:

Real

Ejemplo:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
canshoot = true;
}

El código anterior comprobará la distancia a la posición x/y del objeto jugador y si es menor que el valor almacenado en la variable "range" la variable "canshoot" se pone a true.
