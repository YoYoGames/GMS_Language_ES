diferencia_de_ángulo

Esta función devolverá la menor diferencia de ángulos entre dos ángulos como un valor entre -180 y 180 grados (donde un ángulo positivo es antihorario y un ángulo negativo es horario).

Sintaxis:

diferencia_de_ángulo(ang1, ang2)

Argumento Descripción
ang1 El primer ángulo a utilizar.
ang2 El segundo ángulo a utilizar.

Devuelve:

Real

Ejemplo:

var pd = punto_dirección(x, y, ratón_x, ratón_y);
var dd = ángulo_diferencia(ángulo_de_la_imagen, pd);
ángulo_de_la_imagen -= min(abs(dd), 10) * sign(dd);

El código anterior obtendrá el ángulo de dirección desde la instancia hasta el cursor del ratón, luego obtendrá la diferencia entre ese ángulo y el actual ángulo_de_imagen, utilizando este valor para rotar lentamente hacia el ratón.
