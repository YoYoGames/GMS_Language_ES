punto_dirección

Esta función devuelve la dirección de un vector formado por los componentes especificados [x1,y1] y [x2,y2] en relación con las coordenadas fijas x/y de la sala. Por ejemplo, en la imagen de abajo si queremos obtener la dirección desde la posición de la nave del jugador a la posición del enemigo para poder disparar un misil al enemigo, entonces usaríamos esta función (el código exacto está en el ejemplo debajo de la imagen):

Sintaxis:

punto_dirección(x1, y1, x2, y2)

Argumento Descripción
x1 La coordenada x del primer componente del vector
y1 La coordenada y del primer componente del vector
x2 La coordenada x del segundo componente del vector
y2 La coordenada y del segundo componente del vector

Devuelve:

Real

Ejemplo:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile))
{
direction = point_direction(x, y, ex, ey);
}

El código anterior obtendrá las coordenadas x e y del enemigo más cercano y luego las pasará a un objeto bala para utilizarlo en la función punto_dirección para establecer su dirección de desplazamiento correctamente.
