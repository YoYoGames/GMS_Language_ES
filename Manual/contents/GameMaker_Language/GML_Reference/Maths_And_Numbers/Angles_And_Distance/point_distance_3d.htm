punto_distancia_3d

Esta función toma los componentes suministrados del vector y devuelve la longitud (distancia) del vector. Funciona exactamente igual que punto_distancia() pero con la adición de factorizar el valor z (profundidad) para su uso en el espacio 3D.

Sintaxis:

punto_distancia_3d(x1, y1, z1, x2, y2, z2);

Argumento Descripción
x1 La coordenada x del primer componente del vector
y1 La coordenada y del primer componente del vector
z1 La coordenada z del primer componente del vector
x2 La coordenada x del segundo componente del vector
y2 La coordenada y del segundo componente del vector
z2 La coordenada z del segundo componente del vector

Devuelve:

Real

Ejemplo:

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
{
ex = inst.x;
ey = inst.y;
ez = inst.z;
if point_distance_3d(x, y, z, ex, ey, ez) < 200
{
instance_create_layer(x, y, "Bullets", obj_Missile)
}
}

El código anterior obtendrá las coordenadas x e y y z del enemigo más cercano y luego las utilizará para comprobar la distancia (longitud) del vector formado por ellas y las coordenadas del jugador. Si el valor es inferior a 200, el objeto jugador creará una instancia de "obj_Misil".
