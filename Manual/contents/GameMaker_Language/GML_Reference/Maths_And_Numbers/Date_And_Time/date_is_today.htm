fecha_es_hoy

Esta función devolverá true si el valor de la fecha es el día que se está comprobando (es decir, hoy), o false en caso contrario. Esta función puede ser muy útil para cosas como los huevos de pascua en tus juegos, o para desbloquear contenido de temporada. Tenga en cuenta que esta función se verá afectada por la zona horaria establecida (por defecto es la hora local) que puede cambiar utilizando la función date_set_timezone() función.

Sintaxis:

date_is_today(date);

Argumento Descripción
fecha La fecha a utilizar.

Devuelve:

Booleano

Ejemplo:

if date_is_today(global.Halloween)
{
global.Max_Levels = 200;
}

El código anterior comprobará la fecha almacenada en la variable global "Halloween" para ver si es hoy, y si lo es establece otra variable global con un nuevo valor.
