date_valid_datetime

Con esta función puede comprobar un valor de fecha y hora para ver si es válido (devuelve true) o no (devuelve false). Tenga en cuenta que esta función sólo considerará que una fecha válida es posterior al 1/1/1970 y que cualquier fecha anterior devolverá devolverá false, por lo que lo más temprano que se puede comprobar sería:

date_valid_datetime(1970, 01, 01, 0, 0, 0);

Sintaxis:

date_valid_datetime(año, mes, día, hora, minuto, segundo);

Argumento Descripción
año El año que hay que comprobar.
mes El mes para comprobarlo.
día El día para comprobarlo.
hora La hora de la comprobación.
minuto El minuto para comprobarlo.
segundo El segundo a comprobar.

Devuelve:

Booleano

Ejemplo:

if date_valid_datetime(2011, 9, 15, 10, 3, 30)
{
mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
}

Esto establecerá "mydatetime" en el 15 de septiembre de 2011, 10:03.30, si es un valor válido (que lo es).
