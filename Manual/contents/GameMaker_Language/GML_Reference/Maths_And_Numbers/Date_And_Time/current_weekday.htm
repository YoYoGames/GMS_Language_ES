día_de_la_semana_actual

Esta variable de sólo lectura devolverá el día de la semana como valor, donde el domingo es 0 y el sábado es 6.

Sintaxis:

current_weekday;

Devuelve:

Real

Ejemplo:

var day;
switch(current_weekday)
{
caso 0: día = "domingo"; break;
caso 1: día = "lunes"; break;
caso 2: día = "martes"; break;
caso 3: día = "miércoles"; break;
caso 4: día = "jueves"; break;
case 5: day = "Friday"; break;
case 6: day = "Saturday"; break;
}
draw_text(32, 32, "Hoy es " + día +";)

El código anterior utiliza el valor de current_weekday para establecer una variable con el día correcto en texto, y luego lo dibuja para que el usuario lo vea.
