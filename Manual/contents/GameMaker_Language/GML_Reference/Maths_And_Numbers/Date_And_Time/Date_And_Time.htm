Fecha y hora

GameMaker Studio 2 tiene una gran cantidad de funciones y variables relacionadas con la obtención de la fecha y la hora y el cálculo de otros valores basados en ellos. Por defecto, estas funciones devuelven la hora local como pero se puede utilizar la siguiente función para cambiar entre la hora local y la hora UTC:

  • date_set_timezone
  • date_get_timezone

Las siguientes variables incorporadas pueden utilizarse para obtener componentes específicos de fecha y hora:

  • hora_actual
  • segundo_actual
  • minuto_actual
  • hora_actual
  • día_actual
  • día_de_la_semana_actual
  • mes_actual
  • año_actual
