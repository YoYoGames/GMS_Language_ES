fecha_horario_cadena

Con esta función puedes crear una cadena que contenga la fecha y hora dadas, formateada para el sistema o dispositivo que esté ejecutando el juego cuando se llame a la función.

Sintaxis:

date_datetime_string(date);

Argumento Descripción
fecha La fecha a utilizar.

Devuelve:

Cadena

Ejemplo:

str = date_datetime_string(date_current_datetime());

Esto establecería "str" en algo como "15 de septiembre de 2011, 11:33.00" dependiendo de la configuración del sistema para la visualización de la fecha/hora y la fecha y hora actuales.
