tiempo_delta

El propósito del delta timing es eliminar los efectos del lag o la lentitud de los ordenadores que tratan de manejar gráficos complejos o mucho código. Es un valor que se puede añadir a la velocidad de los objetos para que finalmente se muevan a la misma velocidad, independientemente del lag. Esto se consigue en GameMaker Studio 2 utilizando la variable de sólo lectura delta_time en cada paso, ya que mide el tiempo que ha pasado entre un paso y el siguiente en microsegundos (1microsegundo es la 1.000.000ª de segundo). Por lo tanto, la variable delta_time se puede utilizar para calcular cuánto más rápido (por ejemplo) tiene que moverse un personaje del juego para compensar un pico de retraso en el juego.

Sintaxis:

tiempo_delta

Devuelve:

Real (entero)

Ejemplo:

var _dt = delta_time / 1000000;
speed = spd * dt;

El código anterior establecerá la velocidad de la instancia utilizando el tiempo delta para corregir el retraso. La variable "_dt" se utilizaría para mantener el valor del tiempo delta calculado previamente, de modo que el ratio pueda utilizarse para multiplicar el valor de la velocidad base (mantenido en la variable "spd") y así obtener una velocidad consistente para la instancia.
