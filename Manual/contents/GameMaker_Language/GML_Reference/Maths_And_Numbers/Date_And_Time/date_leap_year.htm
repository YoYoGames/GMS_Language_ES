fecha_año_superior

Esta función devolverá true si el componente de año del valor datetime dado es un año bisiesto o false en caso contrario. Esta función puede ser muy útil para cosas como los huevos de pascua en tus juegos, o para desbloquear contenido especial.

Sintaxis:

date_leap_year(date);

Argumento Descripción
fecha La fecha a utilizar.

Devuelve:

Booleano

Ejemplo:

if date_leap_year(date_current_datetime())
{
if !global.ExtraContent
{
global.ExtraContent = true;
}
}

El código anterior comprobará la fecha actual para ver si el año es bisiesto o no. Si lo es, establece una variable global.
