date_get_timezone

Obtiene la zona horaria base que se utiliza para el resto de las funciones de fecha y hora. Esta zona horaria puede ser local (como la establece el sistema) o UTC, y la función devolverá una de las siguientes constantes:

Constante Descripción
timezone_local utilizar la zona horaria local establecida por el sistema
timezone_utc utilizar el Tiempo Universal Coordinado

Sintaxis:

date_get_timezone();

Devuelve:

Constante

Ejemplo:

if (date_get_timezone() != timezone_utc)
{
date_set_timezone(timezone_utc);
}

Este código comprueba la configuración de la zona horaria base del juego y, si no es UTC, la cambia.
