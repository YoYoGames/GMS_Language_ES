hora_actual

Esta variable de sólo lectura devolverá el número de milisegundos que han pasado desde que se inició el juego.

Sintaxis:

hora_actual;

Devuelve:

Real

Ejemplo:

if (current_time > 600000)
{
msg = show_question_async("¿Desea valorar?");
}

El código anterior comprueba si han pasado más de 10 minutos antes de hacer una pregunta al usuario.
