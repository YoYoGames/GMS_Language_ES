A veces quiere especificar algo más que números para una selección aleatoria, o los números que quiere no están en ningún orden real o dentro de algún rango establecido. En estos casos, se utilizaría choose() para generar un resultado aleatorio. Por ejemplo, digamos que quieres crear un objeto con un sprite aleatorio al principio, entonces podrías usar esta función para establecer el índice del sprite a uno de un conjunto de sprites dados. Tenga en cuenta que puede tener tantos como necesite (tenga en cuenta que más argumentos significarán que la función será más lenta de analizar).

NOTA: Esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido a que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para facilitar la depuración del código. Para evitar este comportamiento utilice randomise() al inicio de tu juego.

Sintaxis:

choose(val0, val1, val2... max_val);

Argumento Descripción
val0... max_val Un valor de entrada que puede ser una cadena, un entero, una variable o una constante.

Devuelve:

Uno de los argumentos dados

Ejemplo:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

El código anterior utiliza choose para establecer una serie de propiedades para la instancia.
