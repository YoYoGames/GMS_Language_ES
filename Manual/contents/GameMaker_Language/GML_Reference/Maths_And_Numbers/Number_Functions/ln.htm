ln

El logaritmo natural ln(n) es la cantidad de tiempo necesaria para alcanzar un determinado nivel de crecimiento continuo, donde n es el nivel alcanzado. Así, si queremos saber cuántas unidades de tiempo necesitamos para conseguir un crecimiento de 20, utilizaremos ln (20), que devuelve 2,99 unidades de tiempo para conseguir esa cantidad de crecimiento.

Sintaxis:

ln(n);

Argumento Descripción
n El valor de entrada.

Devuelve:

Real

Ejemplo:

alarma[0] = ln(edad) * velocidad_de_habitación;

El código anterior utiliza el logaritmo natural del valor almacenado en la variable "edad" para establecer una alarma.
