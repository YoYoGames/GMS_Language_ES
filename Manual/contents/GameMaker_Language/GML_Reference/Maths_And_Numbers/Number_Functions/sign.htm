firmar

Esta función devuelve si un número es positivo, negativo o ninguno de los dos y devuelve 1, -1, 0 respectivamente. Por ejemplo - sign(458) devolverá 1, sign(-5) devolverá -1 y sign(0) devolverá 0.

Sintaxis:

signo(n);

Argumento Descripción
n El número para obtener el signo de.

Devuelve:

Entero

Ejemplo:

y += sign(y - mouse_y);

El código anterior añadirá 1, -1 o 0 a y dependiendo del resultado de y - mouse_y.
