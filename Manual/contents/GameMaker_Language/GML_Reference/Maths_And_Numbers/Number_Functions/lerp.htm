lerp

Con esta función puedes encontrar el valor que equivale a la posición entre otros dos valores para un porcentaje determinado. Así que si haces, por ejemplo

lerp(0, 10, 0.5)

obtendría el valor de retorno de 5, que es el 50% de los valores de entrada. También puede extrapolar utilizando esta función, suministrando un valor positivo o negativo para la cantidad de interpolación de manera que haciendo algo como:

lerp(0, 10, 2)

devolverá un valor de 20.

Sintaxis:

lerp(a, b, amt)

Argumento Descripción
a El primer valor.
b El segundo valor.
amt La cantidad a interpolar.

Devuelve:

Real

Ejemplo:

xx = lerp(x, x + hspeed, room_speed);
yy = lerp(y, y + vspeed, room_speed);

El código anterior utiliza la función de interpolación lineal para predecir a dónde se habrá movido una instancia después de un segundo de tiempo de juego.
