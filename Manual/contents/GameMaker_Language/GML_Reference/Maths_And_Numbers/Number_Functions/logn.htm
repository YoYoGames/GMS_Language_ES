logn

Esta función es similar a la log2(n) y log10(n) sólo que se estipula el valor logarítmico de la base. Por ejemplo, logn(5,25) devolverá cuántos 5 necesitamos multiplicar para obtener 25 (que es 2).

Sintaxis:

logn(n, val);

Argumento Descripción
n La base de troncos.
val El valor de entrada.

Devuelve:

Real

Ejemplo:

logval = logn(5, num);

El código anterior obtiene el log5 del valor almacenado en "num".
