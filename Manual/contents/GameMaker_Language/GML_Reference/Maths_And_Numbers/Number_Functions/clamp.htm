abrazadera

Con esta función puede mantener un valor de entrada entre un rango especificado.

Sintaxis:

clamp(val, min, max)

Argumento Descripción
val El valor a sujetar.
min El valor mínimo entre el que hay que sujetar.
max El valor máximo a sujetar entre.

Devuelve:

Real

Ejemplo:

velocidad = clamp(velocidad, 1, 10);

El código anterior fijará la velocidad para que nunca caiga por debajo de 1 o pase de 10.
