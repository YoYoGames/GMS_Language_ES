abs

Esta función devuelve el valor absoluto del argumento de entrada, por lo que si es un valor positivo se mantendrá igual, pero si es negativo se multiplicará por -1 para hacerlo positivo.

Sintaxis:

abs(val)

Argumento Descripción
val El número para convertirlo en absoluto.

Devuelve:

Real

Ejemplo:

x += abs (x - mouse_x)

Esto añadirá una cantidad igual al valor absoluto de la posición x actual de la instancia y la posición x del ratón.
