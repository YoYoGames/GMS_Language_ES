sqrt

Si se multiplica un número por sí mismo, se obtiene el cuadrado de ese número, pero a veces se quiere hacer lo contrario y obtener la raíz cuadrada de un número. Así que para encontrar qué número se ha multiplicado por sí mismo para obtener un valor positivo dado, utilizamos esta función. Por ejemplo: sqrt(9) devolverá 3 ya que 3*3=9.

Sintaxis:

sqrt(val);

Argumento Descripción
val El número para obtener la raíz cuadrada.

Devuelve:

Real

Ejemplo:

num = sqrt(exp);

El código anterior establecerá la variable "num" para mantener la raíz cuadrada del valor contenido en "exp".
