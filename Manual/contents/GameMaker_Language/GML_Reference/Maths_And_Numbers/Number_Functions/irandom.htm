irandom

Esta función tan útil sólo devuelve números enteros. Así, por ejemplo, para obtener un número aleatorio del 0 al 9 puede utilizar irandom(9) y devolverá un número del 0 al 9 inclusive. Tenga en cuenta que también se pueden utilizar flotadores, pero el valor superior después del punto será excluido, por lo que irandom(9 .7) devolverá un valor de 0 a 9 solamente. La función tiene un límite superior de $7fffffffffffffLL, por lo que hay que tener cuidado si se utilizan números muy grandes.

NOTA: Esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido a que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para facilitar la depuración del código. Para evitar este comportamiento utilice randomise() al inicio de tu juego.

Sintaxis:

irandom(n);

Argumento Descripción
n El rango superior del que se seleccionará el número aleatorio.

Devuelve:

Entero

Ejemplo:

si (irandom(9) == 1)
{
puntuación += 100;
}

Esto producirá una posibilidad entre diez (ya que se incluye el 0) de sumar 100 a la puntuación.
