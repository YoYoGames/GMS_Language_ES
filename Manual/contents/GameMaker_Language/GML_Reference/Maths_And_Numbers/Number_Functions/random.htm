al azar

Esta función es buena para las probabilidades en las que no es necesario devolver un entero (número entero). Por ejemplo, random(100) devolverá un valor de 0 a 99, ¡pero ese valor puede ser 22,56473! También puede utilizar números reales y no enteros en esta función así - random(0.5), que devolverá un valor entre 0 y 0.4999999.

NOTA: Esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido a que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para facilitar la depuración del código. Para evitar este comportamiento utilice randomise() al inicio de su juego.

Sintaxis:

al azar(n);

Argumento Descripción
n El rango superior del que se seleccionará el número aleatorio.

Devuelve:

Real

Ejemplo:

si (random(10) >= 9)
{
puntuación += 100;
}

Esto producirá aproximadamente una posibilidad entre diez de sumar 100 a la puntuación.
