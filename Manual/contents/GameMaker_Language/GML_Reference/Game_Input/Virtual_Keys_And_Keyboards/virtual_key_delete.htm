virtual_key_delete

Si tu juego tiene diferentes salas o instancias para controlar los menús y el juego y otras cosas, es probable que necesites cambiar la posición y los mapas de las teclas virtuales en algún momento. Para ello puedes utilizar esta función para borrar las antiguas antes de crear las nuevas (si es necesario). La función requiere que proporcione el ID de la llave virtual a borrar - como lo devuelve virtual_key_add().

Tenga en cuenta que las llaves virtuales se eliminarán automáticamente de la sala cuando se cambie de sala, por lo que esta función sólo es necesaria cuando desee eliminar manualmente las llaves antes de que la propia sala haya terminado.

Sintaxis

virtual_key_delete(index);

Argumento Descripción
índice El índice de la llave virtual a borrar.

Devuelve:

N/A

Ejemplo:

virtual_key_delete(global.Left);

El código anterior borra la clave virtual indexada en la variable global "global.Left".
