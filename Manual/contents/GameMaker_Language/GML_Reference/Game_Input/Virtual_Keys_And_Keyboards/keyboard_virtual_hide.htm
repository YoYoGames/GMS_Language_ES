keyboard_virtual_hide

Esta función se puede utilizar para ocultar el teclado virtual en el dispositivo que ejecuta el juego. Al llamar a esta función se generará un Evento Asíncrono del Sistema, en el que el mapa DS async_load DS map se rellenará con los siguientes pares clave/valor:

  • " tipo_de_evento" - será "estado del teclado virtual" cuando sea activado por las funciones del teclado virtual.
  • "screen_height" - la altura del teclado virtual (en píxeles). Será 0 si el teclado es invisible.
  • "keyboard_status" - el estado actual del teclado, devuelto como una de las siguientes cadenas:
    • "oculto"
    • "oculto"
    • "mostrando"
    • "visible"

Sintaxis

keyboard_virtual_hide();

Devuelve:

N/A

Ejemplo:

si input == true
{
input = false;
keyboard_virtual_hide();
}

El código anterior ocultará el teclado virtual del sistema operativo si la variable dada no es falsa.
