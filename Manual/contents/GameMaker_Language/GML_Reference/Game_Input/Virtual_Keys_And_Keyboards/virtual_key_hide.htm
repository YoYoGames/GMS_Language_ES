virtual_key_hide

Una vez que haya creado su llave virtual para sus dispositivos, es posible que necesite depurar y probar que está correctamente posicionada y funciona adecuadamente y para ello llamaría a la función virtual_key_show(). Sin embargo, puede necesitar activar y desactivar esta vista, por lo que también puede ocultar la llave virtual con esta función, utilizando el índice de la llave virtual que desea ocultar (previamente creado y almacenado utilizando virtual_key_add()). Una vez que esté satisfecho con las cosas, se recomienda que cree y coloque su propio gráfico en su área.

Sintaxis

virtual_key_hide(index);

Argumento Descripción
índice El índice de la llave virtual a ocultar.

Devuelve:

N/A

Ejemplo:

si global.Debug
{
virtual_key_show(global.Left);
}
else
{
virtual_key_hide(global.Left);
}

El código anterior comprueba la variable global "global.Debug" y si es verdadero entonces la tecla virtual indexada en la variable "global.Left" se dibujará en la pantalla, y si es falso entonces la tecla se ocultará.
