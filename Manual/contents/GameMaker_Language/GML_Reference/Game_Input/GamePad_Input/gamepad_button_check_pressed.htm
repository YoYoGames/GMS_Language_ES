gamepad_button_check_pressed

Esta función devolverá true o false dependiendo de si se detecta que el botón del gamepad ha sido pulsado o no. Tenga en cuenta que esta función sólo se activará una vez para el botón la primera vez que se pulse. Para que para que se active de nuevo, el botón debe soltarse y volverse a pulsar. Si necesitas comprobar una pulsación continua del botón debes utilizar la función gamepad_button_check().

Si está comprobando un botón analógico, la comprobación tendrá en cuenta el ajuste del umbral y sólo devolverá true cuando el valor bruto de entrada supere el umbral dado (puede obtener este valor bruto utilizando la función gamepad_button_value()).

Sintaxis:

gamepad_button_check_pressed(device, button);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.
botón Qué constante del botón del gamepad hay que comprobar.

Devuelve:

Booleano

Ejemplo:

if gamepad_button_check_pressed(0, gp_start)
{
audio_play_sound(snd_Start, 0, false);
room_goto(rm_Level_1);
}

El código anterior detectará si el botón "start" del gamepad conectado a la "ranura" 0 del dispositivo ha sido pulsado o no y cambiará de sala si lo ha hecho.
