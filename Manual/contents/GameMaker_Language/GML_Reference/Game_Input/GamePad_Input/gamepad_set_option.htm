gamepad_set_option

Esta función se puede utilizar para establecer cualquiera de las opciones disponibles del gamepad. Se proporciona la "ranura" del gamepad para establecer la opción, así como la cadena de opciones a establecer y el valor a utilizar. La cadena de opciones disponible dependerá de la plataforma en la que el proyecto se está ejecutando en, como se indica a continuación:

Opción Plataforma Descripción Valor
"allow_rotation" tvOS Establece si el Siri Remote estará bloqueado por rotación (es decir, sólo puede ser utilizado verticalmente) o no. Cuando está desbloqueado (un valor de true), la rotación del control cambiará la entrada, de manera que una rotación de 90º a la izquierda hará que el lado "derecho" del panel táctil vuelva a estar "arriba", etc. de la almohadilla táctil vuelva a estar "arriba", etc... de manera que toda la entrada será relativa a la orientación del dispositivo. Cuando se bloquea (un valor de false) entonces la entrada no cambiará en relación a la posición del controlador y siempre se tratará como si el controlador estuviera en posición vertical. Booleano
"dpad_absolute" tvOS Establece si el panel táctil del Siri Remote acepta entradas absolutas o relativas. En modo absoluto (un valor de verdadero) el centro del panel táctil se considera la posición (0,0) - es decir, donde el stick analógico está en reposo o donde el centro del dpad - y el movimiento alrededor de esta posición generará la entrada. Cuando se establece como relativa (un valor de false ), la posición (0,0) se considerará donde se haga el toque inicial en el pad y toda la entrada se generará desde esa posición, no desde el centro del dpad. se generará desde esa posición, no desde el centro del pad. Booleano

Sintaxis:

gamepad_set_option(device, option_name, value);

Argumento Descripción
dispositivo Qué "ranura" de dispositivo de gamepad se debe establecer.
nombre_de_la_opción El nombre de la opción a establecer (una cadena, véase la tabla anterior)
valor El valor a establecer la opción (puede ser booleano, real, entero o cadena - véase la tabla anterior)

Devuelve:

N/A

Ejemplo:

for(var i = 0; i < 12; ++i;)
{
if gamepad_is_connected(i) && gamepad_get_description(i) == " tvOS SiriRemote"
{
if gamepad_get_option(i, "allow_rotation") == false
{
gamepad_set_option(i, "allow_rotation", true);
}
}
}

El código anterior recorre todas las ranuras del gamepad y busca el "Siri Remote" en la plataforma tvOS . Si se detecta uno, configura el mando para permitir la rotación de la entrada.
