gamepad_remove_mapping

Esta función se puede utilizar para eliminar el mapeo del dispositivo actual del índice de ranura del gamepad dado. Una vez llamada, el índice de la ranura tendrá que ser reasignado usando la función gamepad_test_mapping() si quieres ser capaz de utilizar las constantes del gamepad para detectar la entrada correctamente (la entrada directa siempre puede ser recuperada utilizando el gamepad_axis/button/hat_count() y gamepad_axis/button/hat_value() ).

Sintaxis:

gamepad_remove_mapping(index);

Argumento Descripción
índice De qué "ranura" del índice del gamepad hay que quitar el mapeo.

Devuelve:

N/A

Ejemplo:

si remap == true
{
gamepad_remove_mapping(global.PadIndex);
}

El código anterior eliminará el mapeo de la ranura de índice del gamepad dado en base al valor de una variable.
