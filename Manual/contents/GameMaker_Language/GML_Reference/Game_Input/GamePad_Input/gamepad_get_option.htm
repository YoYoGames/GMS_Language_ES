gamepad_get_option

Esta función se puede utilizar para recuperar el valor de la opción dada. Se suministra el "slot" del gamepad a comprobar, así como la cadena de opciones y la función devolverá el valor asociado. La cadena de opciones disponible dependerá de la plataforma en la que el proyecto se está ejecutando en, como se indica a continuación:

Opción Plataforma Descripción Devuelve
"allow_rotation" tvOS Devuelve si el Siri Remote estará bloqueado por rotación (es decir, sólo se puede utilizar en vertical) o no. Cuando está desbloqueado, la rotación del mando cambiará la entrada, de forma que una rotación de 90º a la izquierda hará que el lado "derecho" del panel táctil de la almohadilla táctil, etc... de manera que todas las entradas serán relativas a la orientación del dispositivo. Booleano
"dpad_absolute" tvOS Devuelve si el panel táctil del Siri Remote acepta entradas absolutas o relativas. En el modo absoluto, el centro del panel táctil se considera la posición (0,0), es decir, donde el stick analógico está en reposo o donde estaría el centro del dpad - y el movimiento alrededor de esta posición generará la entrada. Cuando se establece como relativa, entonces la posición (0,0) se considerará donde se haga el toque inicial en el pad y toda la entrada se generará desde esa posición, no el centro del pad. Booleano

Sintaxis:

gamepad_get_option(device, option_name);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.
nombre_de_la_opción El nombre de la opción de la que obtener el valor (una cadena, véase la tabla anterior)

Devuelve:

Int, Bool, Real, String, N/A (depende de la plataforma)

Ejemplo:

for(var i = 0; i < 12; ++i;)
{
if gamepad_is_connected(i) && gamepad_get_description(i) == " tvOS SiriRemote"
{
if gamepad_get_option(i, "allow_rotation") == false
{
gamepad_set_option(i, "allow_rotation", true);
}
}
}

El código anterior recorre todas las ranuras del gamepad y busca el "Siri Remote" en la plataforma tvOS . Si se detecta uno, configura el mando para permitir la rotación de la entrada.
