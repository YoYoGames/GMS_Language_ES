gamepad_button_count

Esta función devolverá el número total de botones disponibles para el gamepad conectado a la "ranura" del dispositivo dado.

Sintaxis:

gamepad_button_count(device);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.

Devuelve:

Real

Ejemplo:

b_num = gamepad_button_count(0);

El código anterior devolverá el número de botones disponibles en el gamepad conectado a la "ranura" 0 del dispositivo y almacenará el valor en la variable "b_num".
