gamepad_get_device_count

Esta función te dirá una de dos cosas. O bien el número de game pads conectados, o bien el número de "ranuras" disponibles para que se conecten los game pads. El valor real de retorno dependerá de la plataforma y la configuración interna de esa plataforma y, como tal, esta función debe utilizarse junto con la función gamepad_is_connected() para asegurarse del número exacto de pads conectados en cada momento.

Sintaxis:

gamepad_get_device_count();

Devuelve:

Real

Ejemplo:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
if gamepad_is_connected(i)
{
global.gp[i] = true;
}
else
{
global.gp[i] = false;
}
}

El código anterior recorre los pads de juego disponibles (o ranuras de gamepad) y luego comprueba cada uno de ellos para ver si hay un gamepad conectado. el valor devuelto se utiliza entonces para establecer una matriz global en verdadero o falso para su uso en futuras comprobaciones.
