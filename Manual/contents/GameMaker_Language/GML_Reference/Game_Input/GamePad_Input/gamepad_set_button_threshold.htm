gamepad_set_button_threshold

Esta función permite establecer el umbral actual de los botones analógicos de un dispositivo determinado. El umbral por defecto para todos los botones analógicos es 0,5, con un rango de 0 a 1. El umbral define en qué momento el botón se considera como "pulsado" para los juegos que requieren que actúen como un botón digital. Tenga en cuenta que esta función afectará a los estados de comprobación, pulsación y liberación de los botones analógicos, pero no afectará al valor devuelto por la función gamepad_button_check(), que siempre devolverá el valor bruto del botón.

Sintaxis:

gamepad_set_button_threshold(device, threshold);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.
umbral El nuevo valor del umbral (de 0 a 1, por defecto 0,5).

Devuelve:

N/A

Ejemplo:

if gamepad_get_button_threshold(0) != 0.5
{
gamepad_set_button_threshold(0, 0.5);
}

El código anterior comprueba el umbral del botón analógico del gamepad conectado a la "ranura" 0 del dispositivo y si no es el valor por defecto de 0,5, se establece en ese valor.
