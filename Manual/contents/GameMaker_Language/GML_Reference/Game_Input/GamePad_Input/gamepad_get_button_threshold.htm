gamepad_get_button_threshold

Esta función permite detectar la configuración actual del umbral de los botones analógicos de un dispositivo determinado. El umbral por defecto para todos los botones analógicos es 0,5, con un rango de 0 a 1. El umbral define en qué momento el botón se considera considerado como "pulsado" para los juegos que requieren que actúen como un botón digital.

Sintaxis:

gamepad_get_button_threshold(device);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.

Devuelve:

Real

Ejemplo:

if gamepad_get_button_threshold(0) != 0.5
{
gamepad_set_button_threshold(0, 0.5);
}

El código anterior comprueba el umbral del botón analógico del gamepad conectado a la "ranura" 0 del dispositivo y si no es el valor por defecto de 0,5, se establece en ese valor.
