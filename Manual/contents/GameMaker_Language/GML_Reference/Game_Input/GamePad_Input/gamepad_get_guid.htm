gamepad_get_guid

Con esta función puedes recuperar el GUID del gamepad conectado al índice de la ranura dada. Se suministra la ranura del gamepad a comprobar (de 0 a 11), y la función devolverá una de las siguientes cadenas:

  • "ninguno"si no hay ningún GUID disponible o no hay ningún gamepad conectado
  • "índice del dispositivo fuera de rango" si el índice de la ranura del gamepad está fuera de rango
  • La cadena GUID, algo así como "050000005e040000fd020000ffff3f00", si la ranura tiene un gamepad asignado y puede obtener el GUID para él

Esta función se suele utilizar junto con gamepad_get_description() para generar parte de la cadena SDL necesaria para reasignar los controladores en Windows, Android y macOS.

Sintaxis:

gamepad_get_guid(índice);

Argumento Descripción
índice Qué índice de "ranura" del gamepad hay que comprobar

Devuelve:

Cadena

Ejemplo:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

El código anterior obtiene las cadenas de descripción del GUID y del nombre, luego las concatena y almacena la cadena final en una variable global para futuras referencias.
