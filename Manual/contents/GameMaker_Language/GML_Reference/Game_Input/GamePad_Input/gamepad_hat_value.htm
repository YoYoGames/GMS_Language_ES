gamepad_hat_value

Con esta función puedes obtener el valor actual de un "sombrero" del gamepad. Cada valor del sombrero es una máscara de bits para las diferentes direcciones donde:

  • Arriba = 1
  • Derecho = 2
  • Abajo = 4
  • Izquierda = 8

Tenga en cuenta que se pueden combinar (por ejemplo, si se suministra un índice de sombrero de 3 se estaría comprobando hacia arriba y hacia la derecha) pero sólo se pueden pulsar como máximo 2 bits a la vez. La función devolverá un valor real entre 0 y 1, donde 0 es no pulsado y 1 es totalmente pulsado (y puede haber valores intermedios dependiendo de si el gamepad soporta entrada analógica para sombreros).

Sintaxis:

gamepad_hat_value(dispositivo, hatindex);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.
hatindex Qué sombrero (o sombreros) del gamepad hay que comprobar.

Devuelve:

Real

Ejemplo:

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
caso 1: y -= 3; break;
caso 2: x += 3; break;
caso 3: y -= 3; x += 3; break;
caso 4: x -= 3; break;
caso 6: y += 3; x += 3; break;
caso 8: y += 3; break;
caso 9: y -= 3; x -= 3; break;
caso 12: y += 3; x -= 3; break;
}

El código anterior almacena el estado del sombrero "0" en una variable local, luego comprueba cuál es el valor de retorno y actúa en consecuencia.
