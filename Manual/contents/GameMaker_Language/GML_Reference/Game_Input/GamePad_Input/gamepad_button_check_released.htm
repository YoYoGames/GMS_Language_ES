gamepad_button_check_released

Esta función devolverá true o false dependiendo de si se detecta que el botón del gamepad ha sido soltado o no. Tenga en cuenta que esta función sólo se activará una vez para el botón en el momento en que se haya soltado. Para que se active de nuevo, el botón debe ser pulsado y liberado una vez más. Si está comprobando un botón analógico, la comprobación tendrá en cuenta el ajuste del umbral y sólo devolverá true cuando el valor bruto de la entrada esté por debajo del umbral dado (puede obtener este valor bruto utilizando la función gamepad_button_value()).

Sintaxis:

gamepad_button_check_released(device, button);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.
botón Qué constante del botón del gamepad hay que comprobar.

Devuelve:

Booleano

Ejemplo:

if gamepad_button_check_released(0, gp_select)
{
audio_play_sound(snd_Button, 0, false);
global.Pause = !global.Pause;
}

El código anterior detectará si el botón "select" del gamepad conectado al dispositivo "slot" 0 ha sido pulsado o no y activará la variable global "Pause".
