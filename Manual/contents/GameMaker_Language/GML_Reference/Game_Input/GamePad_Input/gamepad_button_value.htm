gamepad_button_value

Con esta función puedes obtener el valor actual de un botón analógico, de 0 a 1, donde 0 es sin presión y 1 es con presión completa. Se suministra el índice de la ranura del gamepad a comprobar, junto con una constante de botón (como se indica aquí) o un valor entero entre 0 y gamepad_button_count() -1.

Tenga en cuenta que esto devolverá el valor bruto del botón, y no tiene en cuenta la configuración del umbral (ver aquí para más información sobre los umbrales).

Sintaxis:

gamepad_button_value(dispositivo, botón);

Argumento Descripción
dispositivo Qué "ranura" del dispositivo gamepad hay que comprobar.
botón Qué constante del botón del gamepad hay que comprobar.

Devuelve:

Real

Ejemplo:

velocidad = gamepad_button_value(0, gp_shoulderrb) * 4;

El código anterior utiliza el valor de disparo analógico del gamepad conectado a la "ranura" 0 del dispositivo para establecer la velocidad de la instancia.
