datos_del_evento

Esta variable es de alcance global y se utiliza para mantener un mapa DS cuando se utiliza en los eventos de gestos, y -1 en todas las demás veces. El contenido real del mapa DS dependerá del tipo de evento gestual que lo haya desencadenado, así que consulta las secciones individuales para esos eventos.

Sintaxis:

datos_del_evento;

Devuelve:

Real (valor entero)

Ejemplo:

isFlick = event_data[? "isflick"];
if (isFlick)
{
flickVelX = event_data[? "diffX"];
flickVelY = event_data[? "diffY"];
}
else
{
flickVelX = 0;
flickVelY = 0;
}

El código anterior se toma del evento de gesto Drag End y comprueba si la ventilación es un evento "flick", y si lo es extrae los datos necesarios del mapa event_data DS y lo utiliza para establecer algunas variables. Si un evento "flick" no se detecta un evento "flick", entonces las mismas variables se ponen a 0.
