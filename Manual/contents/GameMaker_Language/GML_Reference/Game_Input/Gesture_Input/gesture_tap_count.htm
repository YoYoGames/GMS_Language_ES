gestual_tap_count

Esta función se utiliza para establecer si el conteo de toques está activado o desactivado. Cuando se establece en true el conteo de toques está habilitado, y cuando se establece en false está deshabilitado, aunque se puede comprobar en qué estado se encuentra utilizando la función gesture_get_tap_count(). Cuando está habilitado, cada evento de toque tendrá una entrada adicional en el mapa DS "tapcount" que habrá registrado el número de toques, y cuando está habilitado significa que todos los eventos de toque se activarán, es decir: dos toques activarán tanto el evento de un solo toque como el de dos toques, y el evento de dos toques. es decir, dos toques desencadenarán tanto el evento de un solo toque como el evento de doble toque, siendo el recuento de toques del evento de un solo toque 1 y el recuento de toques del evento de doble toque 2. El valor de la cuenta de toques se restablecerá a 0 una vez que haya transcurrido el tiempo establecido para la detección de un toque doble (véase la función gesture_double_tap_time()). Si el recuento de toques está desactivado, el toque inicial no se registrará hasta que haya pasado el tiempo de doble toque y no se haya detectado un segundo toque. Tenga en cuenta que esto está activado por defecto.

Sintaxis:

gesture_tap_count(enable);

Argumento Descripción
activar Activar (establecer como verdadero) o desactivar (establecer como falso) el conteo de toques.

Devuelve:

N/A

Ejemplo:

if !gesture_get_tap_count()
{
getsure_tap_count(true);
}

El código anterior comprueba si el recuento de grifos está activado y, si no lo está, se activa.
