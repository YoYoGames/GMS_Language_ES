Entrada gestual

Las funciones de entrada de dispositivos están muy bien para tratar con simples toques en la pantalla o conseguir la orientación del dispositivo, pero a veces querrás que tus proyectos detecten cosas más complejas, como ser capaz de arrastrar o "mover" una instancia a través de una habitación. Estas cosas se pueden hacer con las funciones de dispositivo, pero para hacer las cosas más fáciles GameMaker Studio 2 tiene una gama completa de Eventos de Gesto para detectar dobles toques, arrastre, gestos, pellizcos y rotaciones, tanto a nivel de instancia como global, así como así como devolver información sobre el evento. Las funciones que se enumeran aquí están diseñadas para ofrecerte un mayor control sobre cómo y cuándo se activan estos eventos.

IMPORTANTE! Estas funciones no funcionarán en la plataforma de destino HTML5. No darán un error si se utilizan, sino que simplemente se ignorarán.

Están disponibles las siguientes funciones gestuales diferentes:

  • gesture_get_drag_time
  • gesto_obtener_distancia_de_arrastramiento
  • gestual_get_flick_speed
  • gesture_get_double_tap_time
  • gesture_get_double_tap_distance
  • gesture_get_tap_count
  • gesto_obtener_distancia_de_pellizco
  • gesto_obtener_ángulo_de_pellizco_hacia_el_final
  • gesto_obtener_ángulo_de_pellizco_alejado
  • gesto_obtener_tiempo_de_rotación
  • gesto_obtener_ángulo_de_rotación
  • tiempo_de_arrastramiento_del_gesto
  • distancia_de_arrastramiento_del_gesto
  • velocidad_de_giro
  • gesto_doble_tap_tiempo
  • gesto_doble_tap_distancia
  • gestual_tap_count
  • gesto_pinch_distancia
  • gesto_pellizco_ángulo_hacia_el_futuro
  • gesto_pellizco_ángulo_alejado
  • tiempo_de_rotación_del_gesto
  • ángulo_de_giro_del_gesto
