dispositivo_ratón_y

Esta función devuelve la posición y de un toque en el dispositivo. Si está ejecutando esto en los módulos HTML5 o PC y Mac, entonces este valor se actualiza constantemente, siempre y cuando el dispositivo (normalmente un ratón) esté conectado, sin embargo para los dispositivos móviles, esto sólo se actualizará mientras se toque la pantalla y ten en cuenta que el número máximo de toques que se pueden detectar dependerá en gran medida del dispositivo que se utilice y del sistema operativo que ejecute. También hay que tener en cuenta que en HTML5 sólo el "dispositivo 0" funcionará correctamente y otros dispositivos pueden dar valores erróneos.

Sintaxis:

device_mouse_y(device);

Argumento Descripción
dispositivo El dispositivo (de 0 a n) que se está comprobando.

Devuelve:

Real

Ejemplo:

if device_mouse_check_button(0, mb_left)
{
if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
{
pressed = true;
}
else
{
pressed = false;
}
}

El código anterior comprueba si el dispositivo está siendo presionado y, si es así, sondea la posición y del dispositivo para ver si está dentro de los parámetros. Si lo está, establece la variable "pressed" como verdadera, de lo contrario la establece como falsa.
