device_mouse_y_to_gui

Esta función devuelve la posición y del toque en el dispositivo en relación con la capa GUI. Dado que la capa de la GUI puede configurarse con un tamaño diferente independientemente de la resolución de la pantalla y del tamaño de la sala, a veces puede ser complicado obtener el punto exacto de la capa GUI en el que un usuario ha tocado la pantalla. Con esta función puedes obtener esa posición basándote en el tamaño real de la GUI, haciendo que los botones de la GUI y los elementos del HUD sean más fáciles de crear e interactuar. Si estás ejecutando esto en los módulos HTML5 o en los módulos de PC y Mac, entonces este valor se actualiza constantemente, siempre y cuando el dispositivo (normalmente un ratón) esté conectado, sin embargo, para los dispositivos móviles, esto sólo se actualizará mientras se esté tocando la pantalla, y ten en cuenta que el número máximo de toques que se pueden detectar dependerá mucho del dispositivo que se utilice y del sistema operativo que ejecute. También hay que tener en cuenta que en HTML5 sólo funcionará correctamente el "dispositivo 0" y otros dispositivos pueden dar valores erróneos.

Sintaxis:

device_mouse_y_to_gui(device);

Argumento Descripción
dispositivo El dispositivo (de 0 a n) que se está comprobando.

Devuelve:

Real

Ejemplo:

if device_mouse_check_button(0, mb_left)
{
if device_mouse_y_to_gui(0) > 0 && device_mouse_y_to_gui(0) < 32
{
pressed = true;
}
si no
{
pressed = false;
}
}

El código anterior comprueba si el ratón del dispositivo está siendo presionado y si es así, entonces sondea la posición x del dispositivo en la capa de la GUI para ver si está dentro de los parámetros. Si lo está, establece la variable "pressed" como verdadera, de lo contrario la pone en falso.
