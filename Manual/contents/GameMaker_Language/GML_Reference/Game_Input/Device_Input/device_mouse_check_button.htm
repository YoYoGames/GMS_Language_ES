device_mouse_check_button

Esta función devuelve true o false dependiendo de si el botón del ratón dado se mantiene pulsado en el dispositivo dado. Para el botón del ratón puede suministrar una de las constantes listadas en esta página.

Esta función puede usarse para pantallas táctiles, y el argumento del dispositivo puede ser de 0 a n donde n es el número máximo de "dedos" que pueden estar tocando la pantalla a la vez (el número máximo de toques que pueden dependerá en gran medida del dispositivo que se utilice y del sistema operativo que ejecute, pero la mayoría de los dispositivos deberían detectar al menos hasta 4). Tenga en cuenta que mb_right sólo se detectará si se detecta un toque doble (este comportamiento se puede desactivar utilizando la función device_mouse_dbclick_enable()).

Sintaxis:

device_mouse_check_button(device, button);

Argumento Descripción
dispositivo El dispositivo (de 0 a n) que se está comprobando
botón El botón del dispositivo que se está comprobando.

Devuelve:

Booleano

Ejemplo:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
room_goto(rm_Menu);
}

El código anterior comprueba si los dos toques se mantienen pulsados al mismo tiempo y si lo hacen entonces va a otra habitación.
