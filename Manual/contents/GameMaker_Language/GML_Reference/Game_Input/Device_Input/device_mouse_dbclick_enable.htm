device_mouse_dbclick_enable

Esta función se puede utilizar para establecer que el dispositivo detecte un doble toque del mb_left (botón izquierdo del ratón) como un toque del mb_right (botón derecho del ratón) o no. Por defecto, está configurado como verdadero, lo que significa que cada vez que el usuario toca la pantalla del dispositivo dos veces de forma rápida y consecutiva, el dispositivo detecta el doble toque. pantalla del dispositivo dos veces rápida y consecutivamente, el valor de retorno es el mismo que si se hubiera pulsado el botón derecho del ratón. Cuando esto está activado, el primer toque será detectado como mb_left, y el segundo como mb_right, así que asegúrese de que cualquier código que utilices tenga esto en cuenta.

Sintaxis:

device_mouse_dbclick_enable(bool);

Argumento Descripción
bool Activar(verdadero) o desactivar(falso) la detección de doble clic.

Devuelve:

Booleano

Ejemplo:

si os_type == os_windows || os_type == os_mac
{
device_mouse_dbclick_enable(false);
}

El código anterior comprueba si el dispositivo que ejecuta el juego es un PC con Windows o un Mac y, si es alguno de ellos, desactiva la función de doble toque.
