device_mouse_raw_y

Esta función devuelve la posición y bruta del toque en un dispositivo. Esto significa que devuelve la definición real del dispositivo de la posición y que se está tocando, no la de GameMaker Studio 2, y como tal ignorará cosas como posición de la vista y la escala. Tenga en cuenta que el número máximo de toques del dispositivo que se puede detectar dependerá en gran medida del dispositivo que se utiliza y el sistema operativo que se ejecuta

NOTA: Esta función depende en gran medida del dispositivo y debe experimentar primero con el módulo y el dispositivo de destino deseados para ver qué se devuelve exactamente.

Sintaxis:

device_mouse_raw_y(device);

Argumento Descripción
dispositivo El dispositivo (de 0 a n) que se está comprobando.

Devuelve:

Real

Ejemplo:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1));
y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1));
}

El código anterior comprueba si el dispositivo1 y el dispositivo2 están siendo pulsados, y si lo están calcula la posición media de las coordenadas x/y entre cada punto de pulsación.
