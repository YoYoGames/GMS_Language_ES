device_mouse_check_button_pressed

Esta función devuelve true o false dependiendo de si el dispositivo que se especifica ha sido "tocado" (pulsado) o no. El argumento dispositivo se refiere al número de toques, que puede ser de 0 a n y el número máximo de toques que se pueden detectar dependerá mucho del dispositivo que se utilice y del sistema operativo que ejecute (la mayoría de los dispositivos soportan al menos 4 toques simultáneos). Esta función sólo se activa una vez por la acción de pulsar, y las constantes listadas en esta página pueden ser usadas para comprobar los botones del ratón. Tenga en cuenta que mb_right sólo se detectará si se detecta un toque doble y se mantiene en el segundo toque (este comportamiento se puede desactivar utilizando la función device_mouse_dbclick_enable()).

Sintaxis:

device_mouse_check_button_pressed(device, button);

Argumento Descripción
dispositivo El dispositivo (de 0 a n) que se está comprobando.
botón El botón del dispositivo que se está comprobando.

Devuelve:

Booleano

Ejemplo:

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

El código anterior comprueba si el dispositivo ha sido pulsado y establece una variable a true si lo ha hecho.
