Entrada del ratón

La entrada del ratón es aceptada en todas las plataformas (en los dispositivos móviles se acepta como un toque de pantalla simple - si necesitas usar multitáctil, debes usar las funciones específicas del dispositivo) y tiene algunas constantes que se utilizan para especificar los botones que se pulsan. Estas constantes se muestran en la siguiente tabla:

Constante Descripción
mb_left El botón izquierdo del ratón
mb_middle El botón central del ratón (esto puede no ser válido para todas las plataformas de destino)
mb_right El botón derecho del ratón
mb_side1* Botón lateral del ratón 1
mb_side2* Botón lateral del ratón 2
mb_any Cualquiera de los botones del ratón
mb_none Sin botón de ratón


* NOTA: Los botones mb_side1 y mb_side2 sólo están disponibles en Windows, macOS, Ubuntu y HTML5.

Existen las siguientes funciones para los controles estándar de los botones del ratón:

  • botón_del_ratón
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • botón_del_ratón
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • ratón_y

Tenga en cuenta que también hay un conjunto de funciones de ventana relacionadas con el uso del ratón en los objetivos de escritorio:

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
