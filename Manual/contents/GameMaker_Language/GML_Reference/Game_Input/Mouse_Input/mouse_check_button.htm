mouse_check_button

Esta función devolverá true si el botón del ratón que se está comprobando está pulsado o false si no lo está. El botón del ratón a comprobar se suministra a partir de una de las siguientes constantes:

Constante Descripción
mb_left El botón izquierdo del ratón
mb_middle El botón central del ratón (esto puede no ser válido para todas las plataformas de destino)
mb_right El botón derecho del ratón
mb_side1* Botón lateral del ratón 1
mb_side2* Botón lateral del ratón 2
mb_any Cualquiera de los botones del ratón
mb_none Sin botón de ratón


* NOTA: Los botones mb_side1 y mb_side2 sólo están disponibles en Windows, macOS, Ubuntu y HTML5.

Sintaxis:

mouse_check_button(numb);

Argumento Descripción
numb Qué constante del botón del ratón hay que comprobar.

Devuelve:

Booleano

Ejemplo:

if mouse_check_button(mb_left)
{
instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

El código anterior buscará el botón izquierdo del ratón y cada paso que se mantenga pulsado creará una instancia del objeto indexado en "obj_Star".
