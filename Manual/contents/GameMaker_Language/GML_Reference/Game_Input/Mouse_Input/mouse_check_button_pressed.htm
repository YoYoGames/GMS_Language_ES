mouse_check_button_pressed

Esta función devolverá true si el botón del ratón que se está comprobando ha sido pulsado o false si no lo ha sido. Esta función sólo se activará una vez para cualquier botón del ratón cuando se pulse por primera vez y para activarla de nuevo el botón deberá ser soltado y pulsado de nuevo. Tenga en cuenta que se considerará activada mientras dure el paso, y para todas las instancias que tengan algún evento de ratón o que utilicen esta misma función.

Se suministra el botón del ratón para comprobar de una de las siguientes constantes:

Constante Descripción
mb_left El botón izquierdo del ratón
mb_middle El botón central del ratón (esto puede no ser válido para todas las plataformas de destino)
mb_right El botón derecho del ratón
mb_side1* Botón lateral del ratón 1
mb_side2* Botón lateral del ratón 2
mb_any Cualquiera de los botones del ratón
mb_none Sin botón de ratón


* NOTA: Los botones mb_side1 y mb_side2 sólo están disponibles en Windows, macOS, Ubuntu y HTML5.

Sintaxis:

mouse_check_button_pressed(numb);

Argumento Descripción
numb Qué constante del botón del ratón hay que comprobar.

Devuelve:

Booleano

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
puntuación += 50;
}

El código anterior comprobará si se ha pulsado el botón izquierdo del ratón y, en caso afirmativo, añadirá 50 a la puntuación.
