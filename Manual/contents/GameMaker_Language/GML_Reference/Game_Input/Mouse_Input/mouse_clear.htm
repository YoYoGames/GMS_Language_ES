mouse_clear

Esta función borrará el estado actual del botón del ratón dado. Esto significa que la comprobación de que se mantiene pulsado no devolverá true hasta que el jugador suelte el botón y lo pulse de nuevo (pero el estado de liberación seguirá siendo detectado si el borrado se realiza mientras el botón del ratón se mantiene pulsado).

Puede suministrar una de las constantes del botón del ratón que aparecen en esta página.

Sintaxis:

mouse_clear(botón);

Argumento Descripción
botón Qué constante del botón del ratón hay que comprobar.

Devuelve:

Booleano

Ejemplo:

mouse_clear(mb_any);

El código anterior borrará el estado de inactividad de todos los botones del ratón.
