mouse_check_button_released

Esta función devolverá true si el botón del ratón que se está comprobando ha sido liberado o false si no lo ha sido. Esta función sólo se activará una vez para cualquier botón del ratón cuando se suelte y para activarla de nuevo el botón deberá ser pulsado y soltado de nuevo. El botón del ratón que se va a comprobar se puede indicar con una de las siguientes constantes:

Constante Descripción
mb_left El botón izquierdo del ratón
mb_middle El botón central del ratón (esto puede no ser válido para todas las plataformas de destino)
mb_right El botón derecho del ratón
mb_side1* Botón lateral del ratón 1
mb_side2* Botón lateral del ratón 2
mb_any Cualquiera de los botones del ratón
mb_none Sin botón de ratón


* NOTA: Los botones mb_side1 y mb_side2 sólo están disponibles en Windows, macOS, Ubuntu y HTML5.

Sintaxis:

mouse_check_button_released(numb);

Argumento Descripción
numb Qué constante del botón del ratón hay que comprobar.

Devuelve:

Booleano

Ejemplo:

if mouse_check_button_released(mb_right)
{
speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

El código anterior comprobará si se ha soltado el botón derecho del ratón y, en caso afirmativo, establecerá la velocidad de la instancia a una décima parte de la distancia entre la posición actual x/y y la posición x/y del ratón.
