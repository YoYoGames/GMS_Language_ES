mouse_wheel_down

Esta función devuelve true si la rueda del ratón está girando hacia abajo y false en caso contrario.

Sintaxis:

mouse_wheel_down();

Devuelve:

Booleano

Ejemplo:

si mouse_wheel_down()
{
y += 10;
}

Esto mueve la instancia actual hacia abajo en la pantalla si la rueda del ratón se gira hacia arriba.
