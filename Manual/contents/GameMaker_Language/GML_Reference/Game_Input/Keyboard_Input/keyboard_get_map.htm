keyboard_get_map

A veces puede desear obtener el código ascii de una tecla asignada (para ver si ya está asignada, por ejemplo), que es lo que devuelve esta función. La función tomará un valor de código de tecla como el devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes vk_* listadas en la página principal de Entrada de Teclado.

Sintaxis:

keyboard_get_map(key);

Argumento Descripción
clave Esta es la clave de la que se desea obtener el valor mapeado

Devuelve:

Real

Ejemplo:

if keyboard_get_map(ord("A")) = ord("A")
{
keyboard_set_map(ord("A"), keyboard_lastkey);
}

El código de ejemplo anterior primero comprobará si "A" ha sido asignada a otra tecla, y si no lo ha sido, la asignará a la última tecla que el usuario haya pulsado.
