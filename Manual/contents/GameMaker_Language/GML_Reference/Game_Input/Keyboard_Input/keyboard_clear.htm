teclado_limpio

Con esta función puedes borrar el estado actual del teclado, lo que significa esencialmente que si la tecla está siendo mantenida presionada, ya no será reconocida hasta que sea liberada de nuevo (lo que tampoco activará el evento Keyboard Key Released en esta ocasión) y se vuelva a pulsar. La función tomará un valor de código de tecla devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes vk_* listadas en la página principal de Entrada de Teclado.

Sintaxis:

keyboard_clear(key);

Argumento Descripción
clave La clave de la claridad.

Devuelve:

N/A

Ejemplo:

keyboard_clear(vk_space);

El código anterior borra el estado de la barra espaciadora.
