tecla_del_teclado

Con esta variable puedes obtener el código de la tecla que se está pulsando en ese momento y devolverá 0 si no se está pulsando ninguna tecla cuando se haga la comprobación.

Sintaxis:

tecla_del_teclado;

Devuelve:

Real

Ejemplo:

switch (keyboard_key)
{
case vk_numpad1: arma = arma[0][0]; break;
case vk_numpad2: arma = arma[1][0]; break;
case vk_numpad3: arma = arma[2][0]; break;
}

El código anterior utiliza el valor de la variable keyboard_key para establecer una variable con el mismo valor que un array.
