teclado_comprobación

Con esta función puedes comprobar si una tecla está pulsada o no. A diferencia de la función keyboard_check_pressed() o keyboard_check_released() que sólo se activan una vez cuando se pulsa o se suelta la tecla, esta función se activa cada vez que se mantiene la tecla pulsada. La función tomará un valor de código de tecla devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes vk_* listadas en la página principal de Entrada de Teclado.

Sintaxis:

keyboard_check(key);

Argumento Descripción
clave La clave para comprobar el estado de baja de.

Devuelve:

Booleano

Ejemplo:

si keyboard_check(vk_left)
{
x -= 5;
}

El código anterior comprobará si se está pulsando la tecla de flecha y moverá la instancia 5 píxeles a la izquierda cada vez que devuelva true.
