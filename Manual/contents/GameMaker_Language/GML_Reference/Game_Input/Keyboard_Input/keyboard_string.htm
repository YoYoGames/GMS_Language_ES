cadena_de_teclado

Esta variable contiene una cadena con los últimos (como máximo) 1024 caracteres escritos en el teclado. Esta cadena sólo contendrá los caracteres imprimibles tecleados, pero responderá correctamente a la pulsación de la tecla de retroceso borrando el último carácter. Esta variable no es de sólo lectura y puedes cambiarla, por ejemplo para ponerla a "" (una cadena vacía) si ya la has manejado, y puedes usar las Funciones de Cadena para manipularla. Tenga en cuenta que cuando se utiliza el Teclado Virtual en pantalla, sólo esta variable se actualizará con la entrada del teclado.

Sintaxis:

cadena_de_teclado;

Devuelve:

Cadena

Ejemplo:

if string_length(cadena_del_teclado) > 15
{
cadena_del_teclado = cadena_copia(cadena_del_teclado, 1, 15);
}

El código anterior limitará la longitud de la cadena del teclado a 15 caracteres, eliminando los que superen ese límite copiando los primeros quince caracteres en la variable.
