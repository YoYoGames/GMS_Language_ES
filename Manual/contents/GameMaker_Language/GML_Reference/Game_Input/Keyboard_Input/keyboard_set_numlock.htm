keyboard_set_numlock

Puede utilizar esta función para activar o desactivar el bloqueo numérico del teclado (si está activado es verdadero y si está desactiv ado es falso ).

NOTA: Esta funcionalidad sólo está disponible en las construcciones de destino de Windows y no funcionará en ningún otro dispositivo.

Sintaxis:

keyboard_set_numlock(valor);

Argumento Descripción
valor Establecer esto como verdadero para "on" y falso para "off"

Devuelve:

N/A

Ejemplo:

si keyboard_get_numlock()
{
keyboard_set_numlock(false);
}
si no
{
keyboard_set_numlock(true);
}

El código de ejemplo anterior obtendrá el estado de la tecla numberlock y si está activada (true) la pondrá en off (false) y viceversa.
