keyboard_check_pressed

Con esta función puedes comprobar si una tecla ha sido pulsada o no. A diferencia de la función keyboard_check() esta función sólo se ejecutará una vez por cada vez que se pulse la tecla, por lo que para que se active de nuevo la tecla debe ser liberada primero y luego presionada de nuevo. La función tomará un valor de código de tecla como el devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes vk_* listadas en la página principal de Entrada de Teclado.

Sintaxis:

keyboard_check_pressed(key);

Argumento Descripción
clave La tecla para comprobar el estado pulsado de.

Devuelve:

Booleano

Ejemplo:

si keyboard_check_pressed(vk_anykey)
{
room_goto_next();
}

El código anterior avanzará a la siguiente sala si el jugador pulsa cualquiera de las teclas del teclado (funcionando como un aviso de "Pulse cualquier tecla para continuar").
