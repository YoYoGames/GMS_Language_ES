keyboard_check_released

Con esta función puedes comprobar si una tecla ha sido liberada o no. A diferencia de la función keyboard_check() esta función sólo se ejecutará una vez por cada vez que se levante la tecla, así que para que se active de nuevo la tecla debe ser presionada primero y luego liberada de nuevo. La función tomará un valor de código de tecla como el devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes vk_* listadas en la página principal de Entrada de Teclado.

Sintaxis:

keyboard_check_released(key);

Argumento Descripción
clave La clave para comprobar el estado de liberación de.

Devuelve:

Booleano

Ejemplo:

if keyboard_check_released(ord("P"))
{
instance_create_layer(0, 0, "Controllers", obj_Pause);
}

El código anterior comprobará si la tecla "P" ha sido liberada y si es así, creará una instancia de "obj_Pause".
