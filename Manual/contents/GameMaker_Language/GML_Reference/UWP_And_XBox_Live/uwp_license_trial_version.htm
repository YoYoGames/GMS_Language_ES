uwp_license_trial_version

Esta función se puede utilizar para comprobar si el juego está bajo una licencia de prueba o no. Si lo está, la función devolverá true, o false en caso contrario.

Sintaxis:

uwp_license_trial_version();

Devuelve:

Booleano

Ejemplo:

si uwp_license_trial_version()
{
global.LevelCap = 10;
}

El código anterior comprueba si la aplicación está bajo una licencia de prueba y, si es así, establece una variable global.
