uwp_show_help

Esta función lanza la aplicación de ayuda de Xbox One que a su vez mostrará el archivo de ayuda de tu juego (que es un archivo en formato HTML que has subido previamente al Portal de Desarrolladores de XBox). Ten en cuenta que al llamar a esta función se especifica un ID de usuario para que se pueda utilizar la información de la configuración regional del usuario al mostrarla. Puedes recuperar el ID de usuario con las funciones xboxlive_get_user(), xboxlive_get_activating_user(), o uwp_license_trial_user().

Sintaxis:

uwp_show_help(user_id);

Argumento Descripción
user_id El puntero del ID de usuario para el que se abre el archivo de ayuda.

Devuelve:

Booleano

Ejemplo:

if gamepad_button_check_pressed(0, gp_start)
{
if uwp_license_trial_version()
{
var _uid = uwp_license_trial_user;
uwp_show_help(_uid);
}
}

El código anterior comprueba si se pulsa un botón y luego comprueba la aplicación para ver si se está ejecutando con una licencia de prueba. Si lo está, entonces obtiene el ID de usuario de la licencia y abre el archivo de ayuda para ese usuario.
