xboxlive_get_user

Con esta función puedes recuperar el puntero del ID de usuario para el usuario indexado. Si el usuario no existe, la función devolverá la constante pointer_null en su lugar. Puedes encontrar el número de usuarios actualmente conectados con la función xboxlive_get_user_count().

IMPORTANTE: Esta función sólo debe usarse con xboxlive_get_user_count() - no la uses con gamepads. Utilice xboxlive_user_for_pad() en su lugar.

Sintaxis:

xboxlive_get_user(index);

Argumento Descripción
índice El índice (un entero) para obtener el ID de usuario.

Devuelve:

Puntero

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
}

Lo anterior recorre todos los usuarios registrados y almacena su puntero de identificación único en una matriz.
