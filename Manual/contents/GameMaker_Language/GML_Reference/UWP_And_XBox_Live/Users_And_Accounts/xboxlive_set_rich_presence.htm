xboxlive_set_rich_presence

Esta función establecerá la cadena de presencia enriquecida para el usuario dado. Una cadena de presencia enriquecida muestra la actividad de un usuario en el juego después del nombre del juego que el usuario está jugando, separado por un guión. Esta cadena se muestra debajo del Gamertag de un jugador en la lista de "Amigos y Clubes", así como en el perfil de usuario de Xbox Live del jugador.

Cuando se utiliza esta función se necesita suministrar el puntero de ID de usuario para el usuario, y luego se puede marcar el usuario como actualmente activo en el juego o no (usando true/false). Si se establece en falso, entonces la cadena de presencia rica será con "/afk" o algo apropiado. El siguiente argumento es el ID de la cadena de presencia enriquecida que se va a mostrar, y finalmente puedes (opcionalmente) suministrar una cadena service_config_id. Tenga en cuenta que éste es un argumento opcional, ya que si ha llamado a xboxlive_stats_setup() no es necesario pasar el service_config_id aquí.

Para obtener más información sobre la presencia enriquecida y cómo configurar las cadenas que se utilizarán en el centro de socios, consulte la siguiente documentación de Microsoft:

  • Visión general de las cadenas de presencia

Sintaxis:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id])

Argumento Descripción
user_id El puntero ID del usuario a comprobar.
is_user_active Marcar el usuario como activo o no.
cadena_de_presencia_rica El ID de la cadena de presencia enriquecida que se va a utilizar (tal y como se define en el Centro de Socios - máximo 50 caracteres)
service_config_id OPCIONAL] Esta es la cadena config_id para el juego.

Devuelve:

N/A

Ejemplo:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

El código anterior obtiene el puntero de ID de usuario para el usuario asignado a la ranura [0] del gamepad, y luego establece la cadena de presencia rica para ese usuario.
