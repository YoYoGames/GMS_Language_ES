xboxlive_user_for_pad

Esta función devolverá el puntero de ID de usuario asociado con el valor del índice del gamepad dado, o pointer_null si no hay ningún usuario asociado. Tenga en cuenta que esta función sólo debe utilizarse con índices de gamepad, y no con xboxlive_get_user_count().

Sintaxis:

xboxlive_user_for_pad(pad_index);

Argumento Descripción
pad_index El índice (un entero) de la ranura del gamepad a comprobar

Devuelve:

Puntero

Ejemplo:

for(var i = 0; i < 11; ++i;)
{
var u_id = xboxlive_user_for_pad(i);
if u_id != pointer_null
{
pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
}
else
{
pad_name[i] = "Press Play";
}
}

El código anterior recorre todas las ranuras disponibles de los pads y comprueba si hay un puntero de ID de usuario asociado a él. Si lo hay, se establece un array con el nombre del usuario, de lo contrario el array se establece con algún texto por defecto.
