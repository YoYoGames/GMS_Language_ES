xboxlive_user_is_active

Con esta función puedes comprobar si el ID de usuario dado está en la lista de usuarios que actualmente utilizan la consola, y la función devolverá true si lo está, o false en caso contrario. Puedes obtener un puntero de ID de usuario con la función xboxlive_get_user().

Sintaxis:

xboxlive_user_is_active(user_id);

Argumento Descripción
user_id El puntero ID del usuario a comprobar.

Devuelve:

Booleano

Ejemplo:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
var _uid = xboxlive_get_user(i);
if xboxlive_user_is_active(_uid)
{
global.Player_ID[global.PlayerNum++] = _uid;
}
}

El código anterior recorre las cuentas de usuario y comprueba si alguna de ellas está activa. Si lo están, su ID de usuario se añade a un array global.
