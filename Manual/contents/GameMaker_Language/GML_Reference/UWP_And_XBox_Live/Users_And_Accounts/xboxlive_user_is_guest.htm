xboxlive_user_is_guest

Esta función devolverá true si el puntero de ID de usuario dado es un usuario invitado y false si no lo es. Puedes obtener un puntero de ID de usuario con la función xboxlive_get_user().

Sintaxis:

xboxlive_user_is_guest(user_id);

Argumento Descripción
user_id El puntero ID del usuario a comprobar.

Devuelve:

Booleano

Ejemplo:

if xboxlive_user_is_guest(user_id[0])
{
global.user_name[0] = "GUEST";
}
else
{
global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]);
}

Lo anterior almacena el puntero de identificación del usuario activador en una variable global.
