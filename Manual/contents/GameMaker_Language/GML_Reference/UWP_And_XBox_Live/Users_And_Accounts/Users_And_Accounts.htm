Usuarios y cuentas

Esta sección es para los usuarios que se dirigen a las plataformas UWP o Xbox One y describe las funciones de XBox Live disponibles para tratar con usuarios y cuentas de usuario.

La Xbox One permite que varios usuarios inicien sesión en una consola simultáneamente. Cada usuario tiene asociado un mando específico y los datos de las partidas guardadas también son específicos del usuario. Para aprovechar al máximo este sistema, GameMaker Studio 2 introduce una serie de de funciones exclusivas para XBox One relacionadas con la asociación entre mandos y usuarios, que permiten consultar información sobre el usuario y también controlar cómo se gestionan los datos de guardado por usuario. Puedes encontrar más información para desarrolladores sobre este sistema en el libro blanco oficial del sitio de desarrolladores de Microsoft.

NOTA: Tenga en cuenta que el ID de usuario es único para cada individuo y se perpetúa a través de diferentes consolas, por lo que si el mismo usuario se registra en dos dispositivos diferentes este ID de usuario será el mismo.

Las siguientes funciones existen para que las utilices con las cuentas de XBox Live, tanto en la XBox One como con la exportación regular de UWP (ten en cuenta que esta es la funcionalidad mínima permitida para UWP bajo el Programa de Creadores, como se indica en este artículo del servicio de ayuda):

  • xboxlive_user_is_signed_in
  • xboxlive_user_is_signing_in
  • xboxlive_gamertag_for_user
  • xboxlive_show_account_picker

Las siguientes funciones sólo son válidas cuando se dirige a la XBox One usando UWP con una cuenta ID@XBox válida:

  • xboxlive_get_user_count
  • xboxlive_get_user
  • xboxlive_get_activating_user
  • xboxlive_user_is_guest
  • xboxlive_user_is_active
  • xboxlive_user_is_remote
  • xboxlive_user_id_for_user
  • xboxlive_sponsor_for_user
  • xboxlive_set_rich_presence
  • xboxlive_gamedisplayname_for_user
  • xboxlive_user_for_pad
  • xboxlive_pad_for_user
  • xboxlive_pad_count_for_user
  • xboxlive_agegroup_for_user
  • xboxlive_gamerscore_for_user
  • xboxlive_show_profile_card_for_user
  • xboxlive_reputación_de_usuario
  • xboxlive_sprite_add_from_gamerpicture
  • xboxlive_generate_player_session_id
