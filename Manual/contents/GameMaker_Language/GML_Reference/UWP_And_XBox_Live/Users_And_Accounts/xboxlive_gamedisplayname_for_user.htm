xboxlive_gamedisplayname_for_user

Con esta función puede recuperar el nombre para mostrar - como una cadena - del puntero de ID de usuario dado. Tenga en cuenta que si el usuario es local, entonces el valor devuelto es simplemente una cadena de su nombre de pantalla, pero si se trata de un usuario remoto, la cadena será será su gamertag que puede ser utilizado, por ejemplo, para silenciar/desilenciar en el chat de voz.

Sintaxis:

xboxlive_gamedisplayname_for_user(user_id);

Argumento Descripción
user_id El puntero ID del usuario a comprobar.

Devuelve:

Cadena

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

El código anterior obtiene el puntero de ID de usuario para cada cuenta de usuario disponible y luego almacena el puntero junto con el nombre de pantalla en dos matrices.
