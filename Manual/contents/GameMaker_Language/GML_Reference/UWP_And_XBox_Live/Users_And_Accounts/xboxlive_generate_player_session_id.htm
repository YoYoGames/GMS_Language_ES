xboxlive_generate_player_session_id

Esta función generará una cadena única (que contiene un GUID) para la sesión actual del jugador. Esto puede ser utilizado con la función xboxlive_fire_event().

Sintaxis:

xboxlive_generate_player_session_id();

Devuelve:

Cadena

Ejemplo:

if !xboxlive_user_is_signed_in()
{
if !xboxlive_user_is_signing_in()
{
xboxlive_show_account_picker();
}
}
else
{
global.GamerTag = xboxlive_gamertag_for_user();
}

El código anterior comprueba si un usuario ha iniciado sesión en XBox Live y, si lo ha hecho, el código obtendrá el gamertag del usuario y lo almacenará en una variable global para su uso futuro. Si no están registrados, el código comprueba si están en el proceso de inicio de sesión, en cuyo caso no sucederá nada más, y si no lo están, entonces se abrirá el selector de cuentas, pidiéndoles que se registren.
