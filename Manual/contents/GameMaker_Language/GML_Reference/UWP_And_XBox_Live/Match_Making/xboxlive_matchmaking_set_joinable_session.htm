xboxlive_matchmaking_set_joinable_session

Esta función establece una sesión (previamente creada) para que esté disponible para que otros usuarios se unan a través de la interfaz de usuario del sistema. Un usuario sólo tiene una sesión a la que puede unirse a la vez, y cuando abandona la sesión (o la sesión termina) ésta se borra, sin embargo puede Sin embargo, puede borrarla manualmente introduciendo -1 en el argumento de la sesión.

Sintaxis:

xboxlive_matchmaking_set_joinable_session(local_user, session_that_is_joinable);

Argumento Descripción
usuario_local El puntero del ID de usuario local.
sesión_que_es_unible El ID de la sesión que se puede unir, o -1.

Devuelve:

N/A

Ejemplo:

if (global.session_ID != -1)
{
xboxlive_matchmaking_set_joinable_session(xboxlive_user_for_pad(0), global.session_ID);
}

El código anterior comprueba si hay un valor de ID de sesión válido, y si se detecta uno, establece que la sesión es unificable.
