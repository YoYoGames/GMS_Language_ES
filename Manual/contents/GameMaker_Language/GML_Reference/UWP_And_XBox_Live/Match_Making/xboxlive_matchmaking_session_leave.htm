xboxlive_matchmaking_session_leave

Esta función hace que el usuario que creó o encontró la sesión especificada la abandone. Puede obtener el puntero del ID de la sesión del async_load mapa DS que se devuelto en el Evento Social Asíncrono cuando creó o encontró una sesión (ver xboxlive_matchmaking_create() para más detalles).

Sintaxis:

xboxlive_matchmaking_session_leave(session_id);

Argumento Descripción
session_id El puntero de ID de sesión de la sesión a abandonar.

Devuelve:

N/A

Ejemplo:

if gamepad_button_check_pressed(0, gp_start)
{
xboxlive_matchmaking_session_leave(global.SessionID);
}

El código anterior comprueba si se ha pulsado un botón del gamepad y si se detecta uno, el usuario saldrá de la sesión de matchmaking dada.
