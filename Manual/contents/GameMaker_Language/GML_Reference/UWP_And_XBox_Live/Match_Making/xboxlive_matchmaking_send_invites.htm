xboxlive_matchmaking_send_invites

Esta función hace aparecer el diálogo del sistema de Xbox Live para permitir a un usuario enviar invitaciones a su juego. Se especifica el usuario local que debe controlar el diálogo (y cuya lista de amigos será utilizada por el diálogo), el valor de ID de sesión único para de la sesión a la que quieren invitar a la gente, y algún texto por defecto para mostrar en el diálogo. El ID de sesión se obtiene del mapa async_load DS generado por la función función xboxlive_matchmaking_create().

La función devolverá -1 si los argumentos no son válidos o el usuario no ha iniciado sesión en Xbox Live, o un valor superior a 0 en caso contrario.

Sintaxis:

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

Argumento Descripción
de_usuario El ID de usuario local para controlar el diálogo.
sesión_para_invitar_en El ID de sesión único para la sesión a la que se va a unir.
text_to_add_to_invite Un breve texto para añadir a la invitación.

Devuelve:

Entero

Ejemplo:

if gamepad_button_check_pressed(0, gp_face1)
{
var _uid = xboxlive_user_for_pad(0);
xboxlive_matchmaking_send_invites(_uid, global.session_ID, "¡Ven y únete a mi partida!");
}

El código anterior comprueba si se ha pulsado un botón del gamepad y, si se detecta uno, se abrirá el diálogo de Xbox Live para invitar a la sesión.
