xboxlive_matchmaking_session_get_users

Esta función creará y rellenará un mapa DS con los detalles de los usuarios de la sesión especificada, o -1 si hay un error. Puedes obtener el puntero del ID de la sesión desde el async_load mapa DS que se devuelve en el Evento Social Asíncrono cuando se ha creado o encontrado una sesión (ver xboxlive_matchmaking_create() para más detalles). El mapa DS devuelto tendrá los siguientes pares clave/valor:

  • " num_results" - contiene el número de usuarios en la sesión
  • " userId<index> "- contiene el ID del usuario (< index> es un valorde 0 a "num_results" - 1)
  • " userIsOwner<index> "- contiene 1 si el usuario es el anfitrión, en caso contrario 0 (< index> es un valorde 0 a "num_results" - 1)

Sintaxis:

xboxlive_matchmaking_session_get_users(session_id);

Argumento Descripción
session_id El puntero de identificación de la sesión que se utilizará

Devuelve:

ID de la solicitud (entero)

Ejemplo:

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
{
for (var i = 0; i < session_map[? "num_results"]; i++)
{
if (session_map[? "userIsOwner" + string(i)] == 1)
{
global.OwnerID = session_map[? "userId" + string(i)];
}
}
}

Lo anterior recuperará los datos de usuario de todos los usuarios que participan en la sesión de matchmaking y establecerá una variable global con el ID del propietario de la sesión.
