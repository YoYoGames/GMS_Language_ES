uwp_is_suspender

Esta función devolverá true para una sola trama (y false en caso contrario) de forma similar a os_is_paused(). El juego tiene entonces un segundo para guardar el estado del juego antes de que sea potencialmente terminado por el sistema. Una vez que el juego ha terminado de guardar los datos debe llamar a uwp_suspender() para suspenderlo limpiamente. El juego probablemente también debería entrar en un estado de pausa para que si es reanudado por el sistema (si no ha sido completamente terminado) el jugador no sea lanzado directamente a una situación de juego.

Sintaxis:

uwp_is_suspender();

Devuelve:

Booleano

Ejemplo:

if uwp_is_suspending()
{
scr_Save_Game_Data();
uwp_suspender();
}

El código anterior comprueba si la aplicación va a entrar en suspensión y si lo hace llama a un script para guardar los datos del juego antes de suspenderlo.
