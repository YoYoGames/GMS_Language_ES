UWPA y XBox Live

En esta sección se describen las funciones que se utilizan específicamente cuando se apunta a la plataforma de destino UWP. El conjunto inicial de funciones sirve para controlar el comportamiento general del juego al crear una aplicación UWP:

  • uwp_suspender
  • uwp_is_suspender
  • uwp_is_constrained
  • uwp_fue_terminado
  • uwp_fue_cerrado_por_usuario
  • uwp_show_help
  • uwp_license_trial_version
  • uwp_license_trial_user
  • uwp_license_trial_time_remaining
  • uwp_check_privilege

UWP también permite crear juegos para la plataforma XBox One utilizando la API de XBox Live mediante las siguientes funciones. Estas funciones requieren que hayas marcado la opción Habilitar XBox Live en las Opciones de Juego de UWP. Algunas de estas funciones requieren que el usuario inicie sesión en XBox Live, lo que GameMaker Studio 2 intentará hacer automáticamente cuando ejecute el proyecto. Este inicio de sesión "silencioso" generará un Evento Asíncrono del Sistema que puede ser comprobado para ver si fue exitoso o no antes de continuar. Ten en cuenta que, a diferencia de la exportación a la consola Xbox One, la exportación a UWP para XBox Live sólo permite el inicio de sesión de un usuario a la vez.

IMPORTANTE! El objetivo de la UWP sólo tiene un soporte mínimo para XBox Live cuando se configura como parte del Programa de Creadores, que se limita a algunas funciones básicas de usuario y cuenta (ver aquí para más información sobre la configuración). Para poder utilizar toda la funcionalidad que se indica aquí, necesitarás tener una cuenta ID@XBox.
