xboxlive_stats_delete_stat

Esta función se puede utilizar para eliminar una estadística del gestor de estadísticas para el ID de usuario dado. Se suministra el ID de usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user()y luego la cadena de cadena de estadísticas a eliminar. Esto borra el valor de la estadística y lo elimina del gestor de estadísticas, lo que significa que ya no será devuelto por las funciones xboxlive_stats_get_stat_names() o xboxlive_stats_get_stat().

La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito

Sintaxis:

xboxlive_stats_delete_stat(user_id, stat);

Argumento Descripción
user_id El puntero de ID de usuario para borrar la estadística de
stat La estadística a eliminar (una cadena)

Devuelve:

Real

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
user_id[i] = xboxlive_get_user(i);
xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

El código anterior recorre los usuarios conectados y borra la estadística especificada del gestor de estadísticas para cada uno de ellos.
