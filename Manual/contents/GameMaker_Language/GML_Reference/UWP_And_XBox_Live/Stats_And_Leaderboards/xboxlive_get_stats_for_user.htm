xboxlive_get_stats_for_user

Esta función puede utilizarse para recuperar datos sobre estadísticas específicas de los servidores de Xbox Live. Se suministra el ID de usuario devuelto por la función xboxlive_get_user()y luego el ID de configuración del servicio de juegos (como se muestra en la consola XDP), y finalmente las estadísticas requeridas. Puede solicitar hasta un máximo de 14 estadísticas, pero esto no garantiza que obtenga 14 resultados de estadísticas, ya que hay un límite en la longitud total de la solicitud y por lo que el número máximo de estadísticas depende de la longitud de los nombres de las propias estadísticas.

La función devolverá 0 si la solicitud de estadísticas fue enviada o -1 si hubo un error, y el callback disparará un Evento Asíncrono del Sistema. Este evento tendrá el mapa especial DS async_load que deberá ser analizado para la siguiente clave:

  • "tipo_de_evento" - debe contener la cadena "resultado_de_estado" siel evento fue activado por esta función

Si el tipo de evento está relacionado con esta función, también habrá la siguiente clave adicional:

  • " usuario"- contiene el ID del usuario para las estadísticas solicitadas.

También puede haber un conjunto extra de pares clave-valor, donde la clave es el nombre de la estadística solicitada, y el valor el valor de esa estadística en el servidor. Tenga en cuenta que si una estadística nunca ha sido creada para este usuario (porque no ha disparado los eventos que lo establecen), no se puede devolver ningún valor para esa estadística (ds_map_exists() puede utilizarse para comprobar las estadísticas en el mapa async_load ).

Si la solicitud falla debido a que la longitud de la solicitud es demasiado grande, debería haber un mensaje en la salida de la consola indicando el código de error:

xboxlive_get_stats_for_user - se produjo una excepción al obtener resultados - 0x80190190

Cuando esto ocurre, el mapa DS de eventos asíncronos debe tener una clave de "éxito" con un valor de "0", y en este caso se debe intentar recuperar menos estadísticas en una sola llamada.

Sintaxis:

xboxlive_get_stats_for_user(user_id, serviceconfig_id, statname1, ...);

Argumento Descripción
user_id El puntero de identificación del usuario.
serviceconfig_id El ID del archivo de configuración del servicio
statname1 (2, 3, etc...) Los nombres de las estadísticas para las que se recupera la información.

Devuelve:

Real

Ejemplo:

var _uid = xboxlive_get_user(0);
var _configid = "00000000-0000-0000-0000-000000000000";
xboxlive_get_stats_for_user(xbuser, _configid, "GameProgress", "CurrentMode");

El código anterior obtiene el ID del usuario y lo utiliza para solicitar información sobre las estadísticas específicas de ese usuario.
