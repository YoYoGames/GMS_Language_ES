Estadísticas y tablas de clasificación

Las estadísticas de Xbox One (logros y tablas de clasificación) son funcionales cuando se utiliza UWP y tienen algunas funciones especiales para tratarlas, aunque también utilizan algunas de las funciones estándar incorporadas para las tablas de clasificación. El sistema de estadísticas de Xbox Live de 2017 simplifica todo el procedimiento para crear logros y tablas de clasificación, permitiéndote crear y utilizar los diferentes sistemas desde GameMaker Studio 2. La principal diferencia entre el sistema de 2017 y los sistemas anteriores es que ya no tienes que definir todas las estadísticas que utilizas en el juego en XDP, y tampoco necesitas vincular el desbloqueo de logros a un evento/regla de estadísticas (ya que se desbloquean mediante una llamada directa a una función).

NOTA: Aunque no se requiere un archivo de manifiesto con este sistema, puedes configurar algunas estadísticas en XDP/MDC. Las que se configuren allí se considerarán estadísticas destacadas, lo que significa que puedes tener tablas de clasificación globales y las estadísticas aparecen en la página del panel de control de los juegos. Para más información, consulta el artículo de MS Configuring Featured Stats or Leaderboards on Universal Dev Center with Data Platform 2017.

Antes de llamar a cualquiera de las funciones de estadísticas listadas a continuación, debes asegurarte de que el usuario está conectado (ver las funciones de usuario listadas arriba) y luego configurar el sistema de estadísticas y añadir el usuario a él usando estas funciones:

  • xboxlive_stats_setup
  • xboxlive_stats_add_user

El resto de las funciones de las estadísticas de 2017 son las siguientes:

  • xboxlive_stats_remove_user
  • xboxlive_stats_flush_user
  • xboxlive_stats_set_stat_real
  • xboxlive_stats_set_stat_int
  • xboxlive_stats_set_stat_string
  • xboxlive_stats_delete_stat
  • xboxlive_stats_get_stat_names
  • xboxlive_stats_get_stat
  • xboxlive_stats_get_leaderboard
  • xboxlive_stats_get_social_leaderboard
  • xboxlive_achievements_set_progress
  • xboxlive_get_stats_for_user
  • xboxlive_read_player_leaderboard
  • xboxlive_fire_event
