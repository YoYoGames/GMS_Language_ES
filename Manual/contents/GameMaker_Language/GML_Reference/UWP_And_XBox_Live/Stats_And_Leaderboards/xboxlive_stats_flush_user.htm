xboxlive_stats_flush_user

Esta función se puede utilizar para vaciar los datos de las estadísticas de un determinado usuario desde el gestor de estadísticas, al servidor en vivo, asegurando que el servidor está al día con los valores actuales. Según la documentación de XBox, los desarrolladores deben tener cuidado de no llamar a esto con demasiada frecuencia, ya que la XBox limitará las solicitudes, y el sistema operativo de la XBox también vaciará automáticamente las estadísticas aproximadamente cada 5 minutos.

NOTA: La eliminación del usuario puede devolver un error si las estadísticas que se han establecido en el usuario no son válidas (como los nombres de las estadísticas que contienen caracteres numéricos no alfabéticos).

Para utilizar la función, se proporciona el ID de usuario devuelto por (por ejemplo) la función xboxlive_get_user()y un valor de prioridad (que será 1 para prioridad alta y 0 para prioridad baja). La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito. La función también generará una llamada de retorno que activará un Evento Asíncrono del Sistema. Este evento tendrá el mapa especial DS async_load que deberá ser analizado para las siguientes claves:

  • " id" - Mantendrá la constante achievement_stat_event
  • " eventname" - Mantendrá la cadena " StatisticUpdateComplete"
  • " userid" - El ID de usuario asociado a la solicitud
  • " error" - 0 si tiene éxito, algún otro valor si ha habido un error
  • " errormessage" - Una cadena con un mensaje de error, si hay alguno disponible

Ten en cuenta que las estadísticas se vaciarán automáticamente cuando elimines un usuario del gestor de estadísticas. Consulta xboxlive_stats_remove_user() para más información.

Sintaxis:

xboxlive_stats_flush_user(user_id, priority);

Argumento Descripción
user_id El identificador de usuario (un puntero) a utilizar
prioridad El valor de la proridad (0 o 1)

Devuelve:

Real

Ejemplo:

for(var i = 0; i < array_length(user_id); ++i;)
{
xboxlive_stats_flush_user(user_id[i], 0);
}

El código anterior recorre los ID de las cuentas de usuario almacenadas y luego vacía los datos del gestor de estadísticas para cada usuario.
