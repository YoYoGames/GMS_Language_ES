xboxlive_stats_remove_user

Esta función puede ser utilizada para eliminar (desregistrar) un usuario dado del gestor de estadísticas, realizando una descarga de los datos de las estadísticas al servidor en vivo. Según la documentación de XBox, el juego no tiene que eliminar el usuario del gestor de estadísticas, ya que el sistema operativo de la XBox vaciará periódicamente las estadísticas de todos modos. Si proporcionas el ID del usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user()y la función devolverá -1 si hubo un error o si el ID de usuario es inválido, o cualquier otro valor si la función fue llamada con éxito.

NOTA: La eliminación del usuario puede devolver un error si las estadísticas que se han establecido en el usuario no son válidas (como los nombres de las estadísticas que contienen caracteres numéricos no alfabéticos).

La función generará una llamada de retorno que desencadenará un evento asíncrono del sistema. Este evento tendrá el mapa especial DS async_load que deberá ser analizado en busca de las siguientes claves:

  • " id" - Mantendrá la constante achievement_stat_event
  • " eventname" - Mantendrá la cadena " LocalUserRemoved"
  • " userid" - El ID de usuario asociado a la solicitud
  • " error" - 0 si tiene éxito, algún otro valor si ha habido un error
  • " errormessage" - Una cadena con un mensaje de error, si hay alguno disponible

Ten en cuenta que si quieres vaciar los datos de las estadísticas en el servidor en vivo en cualquier momento sin eliminar el usuario, puedes utilizar la función xboxlive_stats_flush_user().

Sintaxis:

xboxlive_stats_remove_user(user_id);

Argumento Descripción
user_id El identificador de usuario (un puntero) para eliminar

Devuelve:

Real

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
xboxlive_stats_remove_user(user_id[i]);
}

El código anterior recorre los ID de las cuentas de usuario almacenadas y elimina (desregistra) cada usuario del gestor de estadísticas.
