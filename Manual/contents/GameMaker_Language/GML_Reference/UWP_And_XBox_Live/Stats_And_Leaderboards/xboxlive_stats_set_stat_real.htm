xboxlive_stats_set_stat_real

Esta función se puede utilizar para establecer el valor de una estadística para el ID de usuario dado. Se suministra el ID de usuario devuelto por (por ejemplo) la función xboxlive_get_user()y la cadena de estadísticas a establecer(si la cadena de estadísticas no existe ya, se creará una nueva y se establecerá el valor dado) y un valor (un real) para establecer la estadística. Tenga en cuenta que el nombre de la estadística debe ser sólo alfanumérico, sin símbolos ni espacios.

Al establecer el valor de la estadística, cualquier valor anterior será anulado, por lo tanto, depende de ti determinar si el valor de la estadística debe ser actualizado o no (es decir, comprobar que la puntuación más alta es realmente la más alta) comparando el valor de la estadística actual con el nuevo valor antes de fijarlo.

La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito.

Sintaxis:

xboxlive_stats_set_stat_real(user_id, stat, value);

Argumento Descripción
user_id El puntero de identificación del usuario para establecer la estadística para
stat La estadística a establecer (una cadena)
valor El valor a establecer la estadística (un real)

Devuelve:

Real

Ejemplo:

if (game_over == true)
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
}
}

El código anterior comprueba una variable y si es verdadera , entonces comprobará el valor de la estadística "PercentDone". Si este valor es inferior a 100, se genera un valor y la estadística se establece en ese valor.
