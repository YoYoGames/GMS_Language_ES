xboxlive_stats_get_stat_names

Esta función se puede utilizar para recuperar todas las estadísticas definidas del gestor de estadísticas para un usuario determinado. Se suministra el ID del usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user(), y la función devolverá una matriz de cadenas que contiene las estadísticas del usuario. Si se produce un error o el usuario no tiene estadísticas, el array se devolverá igualmente pero tendrá un número de elementos de 0.

Sintaxis:

xboxlive_stats_get_stat_names(user_id);

Argumento Descripción
user_id El puntero del ID de usuario para obtener los nombres de las estadísticas de

Devuelve:

Array

Ejemplo:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

El código anterior recupera todas las estadísticas definidas para el puntero de ID de usuario almacenado en la variable dada, y luego recorre la matriz y elimina la estadística especificada del gestor de estadísticas.
