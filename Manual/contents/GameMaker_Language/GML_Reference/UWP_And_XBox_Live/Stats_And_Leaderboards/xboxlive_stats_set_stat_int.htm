xboxlive_stats_set_stat_int

Esta función puede utilizarse para establecer el valor de una estadística para el ID de usuario dado. Se suministra el ID de usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user()y la cadena de estadísticas a establecer(si la cadena de estadísticas no existe ya, se creará una nueva y se establecerá el valor dado) y un valor (un entero) para establecer la estadística. Ten en cuenta que el nombre de la estadística debe ser sólo alfanumérico, sin símbolos ni espacios.

Al establecer el valor de la estadística, cualquier valor anterior será anulado, por lo tanto, depende de ti determinar si el valor de la estadística debe ser actualizado o no (es decir, comprobar que la puntuación más alta es realmente la más alta) comparando el valor de la estadística actual con el nuevo valor antes de fijarlo.

La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito.

Sintaxis:

xboxlive_stats_set_stat_int(user_id, stat, value);

Argumento Descripción
user_id El puntero de identificación del usuario para establecer la estadística para
stat La estadística a establecer (una cadena)
valor El valor a establecer la estadística (un entero)

Devuelve:

Real

Ejemplo:

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

El código anterior recupera el valor de la estadística "GamesPlayed", y luego establece la misma estadística a ese valor más 1.
