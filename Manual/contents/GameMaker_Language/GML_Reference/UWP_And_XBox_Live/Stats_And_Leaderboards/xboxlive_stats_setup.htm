xboxlive_stats_setup

Esta función necesita ser llamada antes de que puedas usar cualquiera de las otras funciones de estadísticas de Xbox Live, y simplemente inicializa las bibliotecas requeridas en el sistema. El argumento "user_id" es el ID de usuario bruto devuelto por la función xboxlive_get_user(), mientras que el argumento "service_config" y "title_id" es el ID único de tu juego en el Xbox Live Dev Center.

Sintaxis:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Argumento Descripción
user_id El puntero ID del usuario a comprobar.
service_config_id Esta es la cadena config_id para el juego.
title_id El ID único de tu juego en el Centro de Desarrollo de Xbox.

Devuelve:

N/A

Ejemplo:

var user = xboxlive_get_user(0); xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

El código anterior inicializa el sistema de estadísticas para el ID de usuario dado.
