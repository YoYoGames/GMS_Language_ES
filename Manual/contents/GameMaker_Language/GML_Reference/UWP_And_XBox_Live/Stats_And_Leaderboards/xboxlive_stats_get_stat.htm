xboxlive_stats_get_stat

Esta función puede utilizarse para recuperar un único valor de estadísticas del gestor de estadísticas para un usuario determinado. Se suministra el ID del usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user(), y luego la cadena de estadísticas como se definió cuando se creó utilizando una de las funciones xboxlive_stats_set_stat_*. El valor de retorno puede ser una cadena o un real (dependiendo de la estadística que se esté comprobando) o la constante GML undefined si la estadística no existe o ha habido un error.

Sintaxis:

xboxlive_stats_get_stat(user_id, stat);

Argumento Descripción
user_id El puntero del ID de usuario para obtener los nombres de las estadísticas de
stat La estadística a establecer (una cadena)

Devuelve:

Real, cadena o indefinido

Ejemplo:

if (game_over == true)
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
}
}

El código anterior comprueba una variable y si es verdadera , entonces comprobará el valor de la estadística "PercentDone". Si este valor es inferior a 100, se genera un valor y la estadística se establece en ese valor.
