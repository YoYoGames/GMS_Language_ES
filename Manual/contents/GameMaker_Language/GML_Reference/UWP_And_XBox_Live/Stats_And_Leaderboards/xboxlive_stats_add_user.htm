xboxlive_stats_add_user

Esta función se puede utilizar para añadir un puntero de ID de usuario dado al gestor de estadísticas. Esto debe hacerse antes de usar cualquiera de las otras funciones de estadísticas para sincronizar automáticamente el juego con el servidor de XBox Live y recuperar los últimos valores. Se suministra el ID de usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user()y la función devolverá -1 si hubo un error o si el ID de usuario es inválido, o cualquier otro valor si la función fue llamada con éxito.

La función generará una llamada de retorno que activará un Evento Asíncrono del Sistema. Este evento tendrá el mapa especial DS async_load que deberá ser analizado en busca de las siguientes claves

  • " id" - Mantendrá la constante achievement_stat_event
  • " eventname" - Mantendrá la cadena " LocalUserAdded"
  • " userid" - El ID de usuario asociado a la solicitud
  • " error" - 0 si tiene éxito, algún otro valor si ha habido un error
  • " errormessage" - Una cadena con un mensaje de error, si hay alguno disponible

Sintaxis:

xboxlive_stats_add_user(user_id);

Argumento Descripción
user_id El ID de usuario (un puntero) para añadir

Devuelve:

Real

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
xboxlive_stats_add_user(user_id[i]);
}

El código anterior obtendrá el número de cuentas de usuario disponibles, luego hará un bucle a través de ellas y asignará el ID de la cuenta a un array, así como registrará el usuario con el gestor de estadísticas.
