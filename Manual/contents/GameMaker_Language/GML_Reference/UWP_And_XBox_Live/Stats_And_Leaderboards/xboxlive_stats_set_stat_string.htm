xboxlive_stats_set_stat_string

Esta función se puede utilizar para establecer una estadística en una cadena para el ID de usuario dado. Se suministra el ID de usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user()y, a continuación, la cadena de estadísticas a establecer(si la cadena de estadísticas no existe ya, se creará una nueva y se establecerá el valor dado) y, a continuación, la cadena a la que se establecerá la estadística. Tenga en cuenta que el nombre de la estadística debe ser sólo alfanumérico, sin símbolos ni espacios, pero la cadena que se establezca puede tener espacios y caracteres especiales.

Al establecer la cadena de estadísticas, cualquier cadena anterior será anulada, por lo tanto, depende de ti determinar si la cadena de estadísticas debe ser actualizada o no, comparando la cadena de estadísticas actual con la nueva antes de establecerla.

La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito.

Sintaxis:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argumento Descripción
user_id El puntero de identificación del usuario para establecer la estadística para
stat La estadística a establecer (una cadena)
valor El valor a establecer la estadística (una cadena)

Devuelve:

Real

Ejemplo:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

El código anterior establece la estadística "LevelName" a la cadena almacenada en la variable global dada.
