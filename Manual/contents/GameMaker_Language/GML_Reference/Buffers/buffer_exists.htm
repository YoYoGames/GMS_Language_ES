buffer_exists

Esta función se puede utilizar para comprobar una variable para ver si tiene un valor válido de ID de búfer o no. Si es así, la función devolverá true, de lo contrario, devolverá false.

Sintaxis:

buffer_exists(buffer)

Argumento Descripción
buffer El índice del búfer a comprobar.

Devuelve:

Booleano

Ejemplo:

if buffer_exists(buff)
{
buffer_delete(buff);
}

El código anterior comprueba si la variable "buff" contiene un ID de búfer y, en caso afirmativo, el búfer se elimina.
