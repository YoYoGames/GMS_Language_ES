buffer_base64_decode

Con esta función puedes decodificar una cadena codificada en base64 (creada con la función buffer_base64_encode() ) en un búfer. Esta función creará el búfer (como un búfer de 1 byte alineado "grow") y devolverá el índice único del búfer, que se utilizará en todas las llamadas posteriores a la función.

NOTA: Es importante que elimines de la memoria cualquier recurso creado dinámicamente como éste cuando ya no lo necesites para evitar fugas de memoria, así que cuando termines con el buffer que has creado deberás liberarlo de nuevo utilizando buffer_delete().

Sintaxis:

buffer_base64_decode(string);

Argumento Descripción
cadena La cadena codificada en base64 a decodificar

Devuelve:

Real

Ejemplo:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

El código anterior abrirá un archivo ini y luego leerá una cadena del mismo en la función de decodificación. Esta función devolverá un índice de búfer, que se almacena en la variable "buff", que contiene los datos previamente codificados y guardados. El archivo ini se cierra.
