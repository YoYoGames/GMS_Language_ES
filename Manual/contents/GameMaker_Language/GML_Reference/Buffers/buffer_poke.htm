búfer_punto

Con la función buffer_write() se pueden escribir datos en el búfer dado en la posición de "búsqueda" actual, y cada dato avanza esta posición en los bytes que se escriben o leen. Sin embargo, puede Sin embargo, puede ser necesario cambiar una pieza de datos sin querer cambiar la posición de búsqueda actual, y es entonces cuando se utiliza esta función. Simplemente se suministra a la función un índice del buffer, y luego la posición de desplazamiento desde el del buffer (en bytes) dentro de ese buffer para escribir, así como el tipo de datos y el valor a escribir.

Sintaxis:

buffer_poke(buffer, offset, type, value);

Argumento Descripción
buffer El índice del buffer a utilizar.
offset La posición de desplazamiento (en bytes) dentro del buffer para escribir los datos dados.
tipo El tipo de datos que se va a escribir en el buffer (ver la lista de constantes aquí).
valor Los datos a añadir al buffer, de acuerdo con el tipo especificado.

Devuelve:

N/A

Ejemplo:

buffer_poke(buff, 3, buffer_u8, color_get_blue(image_blend));

El código anterior añadirá el valor del componente azul del color utilizado para la mezcla de imágenes en el buffer indexado en la variable "buff", en la tercera posición del buffer y como un valor de 8 bits sin signo.
