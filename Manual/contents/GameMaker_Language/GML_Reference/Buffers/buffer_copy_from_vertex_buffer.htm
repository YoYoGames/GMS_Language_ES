búfer_copia_de_vértice_buffer

Esta función puede utilizarse para copiar algunos (o todos) los datos de vértices almacenados en un búfer de vértices en un búfer regular previamente creado. Cuando se copia desde un búfer de vértices a un búfer regular con esta función, ambos búferes deben haber sido previamente creados (con los botones vertex_create_buffer() y buffer_create() por ejemplo). Se puede especificar el rango de datos de vértices que que desea copiar en el búfer, donde el vértice inicial puede estar en cualquier lugar entre 0 y el número de vértices -1, y puede dar el número de vértices a partir de ese punto para copiar. Puedes utilizar la función vertex_get_number() en el buffer de vértices para obtener el número total de vértices almacenados. Por último, se da el índice del búfer en el que se van a copiar los datos de los vértices, así como un desplazamiento de datos para definir la posición en la que se van a copiar los datos de los vértices en el búfer de destino.

Sintaxis:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Argumento Descripción
búfer_de_vértices El índice del buffer de vértices del que se va a copiar.
inicio_vértice El vértice inicial.
número_de_vertices El número total de vértices a utilizar.
dest_buffer El índice del buffer a copiar.
dest_offset La posición de desplazamiento para copiar los datos (en bytes).

Devuelve:

N/A

Ejemplo:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

El código anterior copiará los datos de los vértices almacenados en el buffer de vértices indexado en la variable "model_buffer", y luego los pegará en el buffer indexado en la variable "player_buffer".
