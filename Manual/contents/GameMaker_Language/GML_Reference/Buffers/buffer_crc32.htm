buffer_crc32

Esta función toma los datos de entrada de un búfer y devuelve un hash de suma de comprobación crc32. Se especifica el ID del búfer que se va a utilizar, luego un valor de desplazamiento (en bytes) para saber dónde comenzar, y luego un tamaño (de nuevo en bytes) para la región que se va a hashear, y la función devolverá un valor entero de 32 bits para esa región.

Sintaxis:

buffer_crc32(buffer, offset, size);

Argumento Descripción
buffer El índice del buffer a utilizar.
offset El valor del desplazamiento de los datos.
tamaño El tamaño del buffer.

Devuelve:

Int 32

Ejemplo:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

El código anterior creará un valor hash de suma de comprobación crc32 para los datos completos almacenados en el buffer indexado por la variable "buff", y almacenará el valor hash entero devuelto en la variable "check_val".
