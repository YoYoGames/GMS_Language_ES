buffer_load_partial

Esta función cargará algunos de los datos del buffer que fueron previamente guardados usando la función buffer_save() en un buffer ya creado. Se da el id del buffer previamente creado en el que se va a cargar, luego el archivo del buffer guardado que se va a cargar, y luego el desplazamiento desde el inicio del buffer (en bytes) desde el que se desea cargar los datos. Los siguientes argumentos sirven para establecer la longitud de los datos del buffer (en bytes) desde el punto de desplazamiento inicial que se desea cargar y el punto de desplazamiento al que cargar los datos en el buffer (de nuevo, en bytes).

Sintaxis:

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

Argumento Descripción
buffer El índice del buffer a cargar.
nombre de archivo El nombre del archivo a cargar.
offset El desplazamiento dentro del búfer para cargar (en bytes).
src_len La longitud de la parte del buffer a cargar (en bytes).
dest_offset El offset donde empezar a poner los datos parciales en el nuevo buffer (en bytes).

Devuelve:

Real

Ejemplo:

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

El código anterior creará un nuevo búfer "grow" y luego cargará en él una parte de los datos guardados en el archivo "save.dat".
