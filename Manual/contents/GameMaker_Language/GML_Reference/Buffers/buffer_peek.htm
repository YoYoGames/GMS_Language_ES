buffer_peek

Con la función buffer_read() puedes leer datos del buffer dado en la posición de "búsqueda" actual, con cada pieza de datos que se lee avanzando esta posición por los bytes que se leen o escriben. Sin embargo, puede ser necesario leer un dato determinado sin querer cambiar la posición de búsqueda actual, y es entonces cuando se utiliza esta función. Simplemente se suministra a la función un identificador de búfer, y luego una posición de desplazamiento (desde del inicio del búfer) dentro de ese búfer para leer, así como el tipo de datos que desea leer.

NOTA: El uso de un tipo de datos incorrecto para los datos que se leen dará lugar a valores erróneos.

Si la función tiene éxito, se devolverá un valor (Real, cadena, int, etc...) dependiendo del tipo de datos, mientras que un fallo devolverá simplemente 0.

Sintaxis:

buffer_peek(buffer, offset, type);

Argumento Descripción
buffer El índice del buffer a utilizar.
offset La posición de desplazamiento (en bytes) dentro del buffer para leer los datos dados.
tipo El tipo de datos que se va a leer del buffer (ver la lista de constantes aquí).

Devuelve:

Real/String o 0 si falla.

Ejemplo:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

El código anterior obtendrá tres valores de tres posiciones diferentes dentro del buffer indexado en la variable "buff" y luego utilizará esos valores para establecer la mezcla de imágenes de la instancia.
