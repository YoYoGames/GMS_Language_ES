buffer_base64_encode

Esta función convertirá los datos del búfer dado en una cadena codificada en formato base64. Este es un esquema de codificación comúnmente utilizado para cualquier medio que necesite ser almacenado o transferido a través de Internet como texto, y hace que la salida ilegible para el ojo humano. Para utilizarlo es necesario especificar un búfer ya creado, el valor de offset (que es el punto dentro del búfer en el que se desea empezar a codificar) así como el tamaño, en bytes, de la memoria del búfer a codificar.

Sintaxis:

buffer_base64_encode(buffer, offset, size);

Argumento Descripción
buffer El índice del buffer a utilizar.
offset El valor de desplazamiento de los datos.
tamaño El tamaño del buffer.

Devuelve:

Cadena

Ejemplo:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

El código anterior creará la codificación de los datos completos almacenados en el buffer indexado por la variable "buff", y almacenará la cadena devuelta en la variable local "b_str".
