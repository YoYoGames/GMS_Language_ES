guardar_buffer

Con esta función puedes guardar el contenido de un buffer en un archivo, listo para ser leído de nuevo en la memoria usando la función buffer_load() función

NOTA: En HTML5 el contenido del buffer se guardará como cadenas codificadas en base64 cuando se utilice esta función.

Sintaxis:

buffer_save(buffer, filename);

Argumento Descripción
buffer El índice del buffer a guardar.
nombre de archivo El nombre del archivo a guardar como.

Devuelve:

N/A

Ejemplo:

buffer_save(buff, "Player_Save.sav");

Guarda el contenido actual del buffer con el id almacenado en la variable "buff" en un archivo.
