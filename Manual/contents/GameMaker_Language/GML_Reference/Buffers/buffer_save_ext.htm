buffer_save_ext

Con esta función puedes guardar parte del contenido de un buffer en un archivo, listo para ser leído de nuevo en la memoria usando la función buffer_load() función. El "offset" define la posición inicial dentro del buffer para guardar (en bytes), y el "tamaño" es el tamaño del área del buffer que se guardará a partir de ese offset (también en bytes).

Sintaxis:

buffer_save_ext(buffer, filename, offset, size);

Argumento Descripción
buffer El índice del buffer a guardar.
nombre de archivo El nombre del archivo a guardar como.
offset El desplazamiento dentro del búfer desde el que se va a guardar (en bytes).
tamaño El tamaño del área del buffer a guardar (en bytes).

Devuelve:

N/A

Ejemplo:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Guarda en un archivo parte del contenido actual del buffer con el id almacenado en la variable "buff".
