tamaño_del_buffer

Esta función devolverá el tamaño (en bytes) de cualquiera de las constantes de datos dadas (enumeradas aquí).

Sintaxis:

buffer_sizeof(type);

Argumento Descripción
tipo El tipo de datos que se va a comprobar (ver la lista de constantes aquí).

Devuelve:

N/A

Ejemplo:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

El código anterior calcula primero el tamaño del búfer a crear multiplicando el tipo de datos de 8 bits sin signo por 12 (ya que utilizaremos el búfer para contener 12 datos), y luego utiliza este valor para establecer un búfer fijo con el tamaño correcto.
