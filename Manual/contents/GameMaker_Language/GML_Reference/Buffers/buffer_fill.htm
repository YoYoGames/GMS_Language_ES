relleno_del_buffer

Esta función se puede utilizar para llenar un búfer previamente creado con un tipo de datos y un valor determinados. Los datos con los que se rellena el buffer deben estar de acuerdo con el argumento "tipo" de esta función, lo que significa que no se puede intentar rellenar con una cadena y utilizar el tipo entero de 16 bits sin signo, por ejemplo. Las constantes de tipo son las mismas que las utilizadas por las funciones buffer_read() y buffer_write() . El "tamaño" es el tamaño del buffer (en bytes) que se desea rellenar, mientras que el "offset" es el valor de desplazamiento (también en bytes) desde el inicio del buffer para comenzar el relleno.

Sintaxis:

buffer_fill(buffer, offset, type, value, size);

Argumento Descripción
buffer El índice del buffer a llenar.
offset El valor del desplazamiento de los datos (en bytes).
tipo El tipo de datos que se va a escribir en el buffer (ver la lista de constantes aquí).
valor Los datos a escribir.
tamaño El tamaño del buffer (en bytes) que se desea llenar.

Devuelve:

N/A

Ejemplo:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

El código anterior encuentra el inicio del buffer con el id almacenado en la variable "buff" y escribe en él una serie de valores enteros de 16 bits con signo.
