buffer_base64_decode_ext

Con esta función puede decodificar una cadena codificada en base64 (creada con la función buffer_base64_encode() ) en un buffer. A diferencia de la función buffer_base64_decode(), ésta no creará un búfer por ti, sino que ya deberías haber creado el búfer (ver buffer_create()), cuyo id utilizaría con esta función. El "offset" es la posición dentro del buffer para decodificar la cadena dada (en bytes).

Sintaxis:

buffer_base64_decode_ext(buffer, string, offset);

Argumento Descripción
buffer El índice del búfer en el que se decodificará la cadena.
cadena La cadena codificada en base64 a decodificar.
offset El valor del desplazamiento de los datos.

Devuelve:

N/A

Ejemplo:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

El código anterior creará un búfer y almacenará el identificador único para él en la variable "buff", luego abrirá un archivo ini y leerá una cadena de él en la variable local "str". Esta cadena se decodifica en el búfer recién creado antes de cerrar el archivo ini de nuevo.
