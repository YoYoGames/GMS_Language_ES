búfer_copia

Esta función puede utilizarse para copiar un segmento (o todos) de los datos almacenados en un búfer a otro. Cuando se utilizan dos buffers y se copia de uno a otro, ambos buffers deben haber sido creados previamente con la función buffer_create() (por ejemplo), y se puede especificar un desplazamiento de datos (en bytes) para el punto de inicio de los datos a copiar desde el buffer de origen en relación con el inicio del buffer, así como otro desplazamiento de datos para definir la posición a la que copiar los datos en la memoria intermedia de destino.

NOTA: No se puede copiar en el mismo búfer.

Sintaxis:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

Argumento Descripción
src_buffer El índice del búfer del que se va a copiar.
src_offset El desplazamiento de datos desde el que se empieza a copiar (en bytes).
tamaño El tamaño de los datos a copiar (en bytes).
dest_buffer El índice del buffer a copiar.
dest_offset La posición de desplazamiento para copiar los datos (en bytes).

Devuelve:

N/A

Ejemplo:

buffer_copy(buff1, 0, 2048, buff2, 2048);

El código anterior copiará los datos almacenados en el buffer indexado en la variable "buff1", y luego los pegará en el buffer indexado en la variable "buff2", pero desplazado 2048 bytes.
