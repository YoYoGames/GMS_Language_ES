buffer_md5

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se utiliza habitualmente para comprobar la integridad de los datos. Esta función tomará los datos de entrada de un búfer dado (o parte del búfer) y devolverá el hash MD5 de 32 caracteres hexadecimales que es único para esos datos. De esta manera se puede generar una clave segura que puede ser almacenada y utilizada para comprobar la integridad de la información que se envía a (o se recibe de) un servidor externo (por ejemplo). Cuando se aplica esto a los búferes utilizando esta función se debe especificar el id del búfer a utilizar, luego un valor de desplazamiento (en bytes) para donde comenzar, y luego un tamaño (nuevamente en bytes) para la región a ser hash.

NOTA: MD5 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

buffer_md5(buffer, offset, size);

Argumento Descripción
buffer El índice del buffer a utilizar.
offset El valor de desplazamiento de los datos.
tamaño El tamaño del buffer.

Devuelve:

Cadena

Ejemplo:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

El código anterior creará un hash md5 para los datos completos almacenados en el buffer indexado por la variable "buff", y almacenará el hash devuelto en la variable "check_string".
