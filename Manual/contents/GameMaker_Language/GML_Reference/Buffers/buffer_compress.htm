buffer_compress

Con esta función puede comprimir parte (o todo) un buffer utilizando la compresión zlib. Se suministra el ID del búfer a comprimir (devuelto por buffer_create()), el desplazamiento dentro del buffer a utilizar en bytes, y el tamaño de los datos del buffer a comprimir (también en bytes). La función devolverá un nuevo valor de ID de búfer para el búfer comprimido, o un valor menor que 0 si ha fallado por alguna razón. Esta función no alterará el búfer original.

Sintaxis:

buffer_compress(buffer, offset, size);

Argumento Descripción
buffer El índice del buffer a comprimir.
offset El desplazamiento dentro del buffer a comprimir (en bytes).
tamaño El tamaño del área del buffer a comprimir (en bytes).

Devuelve:

Int (valor del ID del búfer)

Ejemplo:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

El código anterior creará un búfer y lo llenará con los datos de una cadena. Este búfer se comprime y se guarda, y tanto el búfer de origen como el comprimido se eliminan
