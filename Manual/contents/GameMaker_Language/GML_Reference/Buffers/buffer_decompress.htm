buffer_decompress

Con esta función puede descomprimir un búfer previamente comprimido utilizando la compresión zlib. Usted proporciona el ID del búfer a descomprimir, y la función devolverá un nuevo ID de búfer que contiene los datos sin comprimir sin comprimir. Si la descompresión ha fallado (por ejemplo, si se suministra un búfer que no ha sido comprimido) entonces la función devolverá un valor menor que 0.

Sintaxis:

buffer_decompress(buffer);

Argumento Descripción
buffer El índice del buffer a descomprimir.

Devuelve:

ID del búfer

Ejemplo:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

El código anterior cargará primero un buffer guardado, luego lo descomprimirá y finalmente leerá los datos de la cadena del buffer descomprimido en una variable global.
