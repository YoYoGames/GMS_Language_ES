buffer_load_ext

Esta función cargará los datos del buffer que se guardaron previamente usando las funciones buffer_save() en un búfer ya creado. Se da el id del buffer previamente creado para cargarlo, luego el archivo del buffer guardado para cargar, y finalmente el offset desde el inicio del buffer (en bytes). y, por último, el desplazamiento desde el inicio del búfer (en bytes) en el que desea cargar los datos.

Sintaxis:

buffer_load_ext(buffer, filename, offset);

Argumento Descripción
buffer El índice del buffer a cargar.
nombre de archivo El nombre del archivo desde el que se va a cargar.
offset El desplazamiento dentro del búfer para cargar (en bytes).

Devuelve:

N/A

Ejemplo:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

El código anterior obtendrá primero la posición del final del buffer indexado en la variable "player_buffer" y luego cargará los datos del dado en esa posición (tenga en cuenta que este ejemplo sólo funcionará con "grow" o "wrap" tipos de búfer).
