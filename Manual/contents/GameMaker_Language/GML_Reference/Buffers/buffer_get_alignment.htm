buffer_get_alignment

Con esta función puedes obtener la alineación de un byte para el ID del buffer dado.

Sintaxis:

buffer_get_alignment(buffer);

Argumento Descripción
buffer El índice del búfer a comprobar.

Devuelve:

Real

Ejemplo:

alineación = buffer_get_alignment(buff);

El código anterior obtendrá la alineación del buffer a partir del valor indexado en la variable "buff" y lo almacenará en una variable.
