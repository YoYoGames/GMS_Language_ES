buffer_get_address

Con esta función puedes obtener un puntero a la dirección del buffer alineado. Esto es principalmente para su uso con extensiones, ya que puede pasar este valor a través de ellos, lo que les permite acceder a los datos del búfer. Tenga en cuenta que hay algunas cosas que no puedes hacer con el buffer cuando accedes desde una extensión:

  • No puedes liberar la memoria.
  • No se puede cambiar el tamaño de la memoria.
  • No puede escribir más allá de la dirección del buffer más el tamaño del buffer (puede usar la función buffer_get_size() para esto) o obtendrá un error por exceso de límites.

Sintaxis:

buffer_get_address(buffer);

Argumento Descripción
buffer El índice del buffer a utilizar.

Devuelve:

puntero

Ejemplo:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

El código anterior obtendrá la dirección de memoria del buffer indexado en la variable "buff_model" y luego obtendrá la dirección de memoria para el final del buffer utilizando la función buffer_get_size (), almacenando ambos valores en variables locales para para su uso posterior.
