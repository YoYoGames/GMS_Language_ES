buffer_seek

Esta función se puede utilizar para desplazarse por un búfer, encontrando el inicio, el final o una posición relativa a la que se utilizó por última vez al leer o escribir datos. El valor "offset" es el desplazamiento (en bytes) que se debe añadir a la posición de búsqueda dada, por ejemplo, si la base es relativa y el offset es 4, entonces la posición del buffer se moverá a lo largo de 4 bytes desde su posición actual. Tenga en cuenta lo siguiente:

  • Puede utilizar valores negativos para el desplazamiento para buscar hacia atrás a través del buffer, así como valores positivos.
  • Si el búfer es del tipo "envolvente" y se desplaza más allá del final del búfer, la posición de búsqueda también se envolverá.
  • Si el búfer no es del tipo "envolvente", la búsqueda se sujetará al principio o al final del búfer, incluso cuando el desplazamiento lleve la búsqueda fuera de los límites del búfer.

Se aceptan las siguientes constantes como argumento "base" para buscar:

Constante Descripción
buffer_seek_start El inicio del buffer
buffer_seek_relative Una posición relativa a la posición actual de lectura/escritura
buffer_seek_end El final del buffer

Sintaxis:

buffer_seek(buffer, base, offset);

Argumento Descripción
buffer El índice del buffer a utilizar.
base La posición base a buscar.
offset El valor del desplazamiento de los datos.

Devuelve:

N/A

Ejemplo:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

El código anterior encuentra el inicio del buffer con el id almacenado en la variable "buff" y escribe en él una serie de valores enteros de 16 bits con signo.
