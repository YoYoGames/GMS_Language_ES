buffer_async_group_option

Con esta función puede establecer algunas opciones específicas de la plataforma para el grupo de búferes que se está guardando. Las opciones disponibles son las siguientes:

  • "subtítulo"o "slottitle" -El valor de esta opción sería una cadena, que se mostrará al usuario cuando gestione sus datos guardados en el SO. Esto sólo es importante cuando se guardan los datos, no cuando se cargan de nuevo.
  • " showdialog" - El valor de esta opción es un booleano verdadero o falso. Si se establece en verdadero, cuando se guarda o carga GameMaker Studio 2 mostrará la interfaz de usuario del sistema, de lo contrario se hará un quicksave/quickload sin mostrar la interfaz de usuario. Normalmente sólo se necesita esto si usted está apoyando múltiples ranuras de guardar para permitir al usuario elegir una ranura, pero si sólo admite una ranura por usuario, establecer esto a falso.
  • "savepadindex" - Para esta opción se especificaría el índice del pad del jugador que está guardando o cargando y el sistema escribirá datos en la carpeta de guardado de este jugador y los leerá.
  • " saveslotsize" - Esta opción requiere que se especifique el tamaño real en bytes que se quiere guardar (para poder hacer un buffer_seek() y buffer_tell() para obtener eso, por ejemplo). Tenga en cuenta que no es obligatorio suministrar este valor, ya que todos los guardados tienen preasignado un espacio mínimo, que normalmente es de unos 10MB.

IMPORTANTE! Esta función sólo es válida actualmente para los módulos de destino PSVita, PS4 y XBox One. En todos los demás objetivos no hará nada.

Sintaxis:

buffer_async_group_option(opción, valor);

Argumento Descripción
opción La opción de fijar.
valor El valor a establecer (puede ser una cadena o un real, dependiendo de la opción).

Devuelve:

N/A

Ejemplo:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "¡Todos tus haggis están guardados aquí!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

El código anterior inicia un grupo de búferes y luego establece las opciones del grupo antes de establecer 4 archivos para guardar asincrónicamente. A continuación, se finaliza la definición del grupo (momento en el que comenzará el guardado).
