buffer_set_used_size

Esta función es principalmente para su uso dentro de las extensiones, y le permite establecer el tamaño "usado" del búfer dado, que es el número de bytes que se han escrito en él.

Cuando se escriben datos en un buffer desde una extensión, GameMaker no sabe qué parte del buffer fue llenada por el código de la extensión y no es capaz de leer esos datos. Esta función puede ser llamada por la extensión para decirle al motor cuántos bytes de datos se escribieron en el buffer, para que el motor pueda leer esos datos.

Sintaxis:

buffer_set_used_size(buffer, size);

Argumento Descripción
buffer El índice del buffer a utilizar.
tamaño El número de bytes a establecer como tamaño "usado".

Devuelve:

N/A

Ejemplo:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

Las funciones anteriores serían llamadas en una extensión para escribir algunos datos en un buffer de GameMaker (a través de su dirección de memoria), y luego establecer el número de bytes que se escribieron en él para que el motor pueda leer esos datos.
