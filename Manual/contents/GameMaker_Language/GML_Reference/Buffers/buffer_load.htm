carga_del_buffer

Esta función se utiliza para cargar un búfer que se ha guardado previamente utilizando las funciones buffer_save() funciones, así como cualquier archivo incluido o archivos cargados externamente. Devolverá el ID de un nuevo buffer que es creado por la función como un buffer "grow" (ver aquí) con una alineación de bytes de 1. Este ID debe ser almacenado en una variable y utilizado en todas las llamadas posteriores de la función a este buffer. llamadas de la función a este búfer. Si la carga falla por cualquier razón, la función devolverá -1 y se mostrará un mensaje en la ventana de salida del compilador diciendo que la carga ha fallado.

Tenga en cuenta que en HTML5 cualquier búfer cargado desde el almacenamiento local será decodificado usando base64 (ya que se guardan como cadenas base64), sin embargo, el uso de esta función en Archivos Incluidos o cualquier archivo externo no utilizará la decodificación base64, lo que debería estar bien para la mayoría de los casos.

NOTA: Es importante que elimines de la memoria cualquier recurso creado dinámicamente como éste cuando ya no lo necesites para evitar fugas de memoria, así que cuando termines con el buffer que has creado deberás liberarlo de nuevo utilizando buffer_delete().

Sintaxis:

buffer_load(filename);

Argumento Descripción
nombre de archivo El nombre del archivo desde el que se va a cargar.

Devuelve:

real

Ejemplo:

player_buffer = buffer_load("Player_Save.sav");

El código anterior cargará un buffer previamente guardado en la memoria, creando un nuevo buffer. El índice de este nuevo buffer se almacena en la variable "player_buffer".
