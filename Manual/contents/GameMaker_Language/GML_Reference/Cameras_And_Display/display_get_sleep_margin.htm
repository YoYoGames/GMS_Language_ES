display_get_sleep_margin

Esta función se puede utilizar para obtener el valor del margen de reposo actual utilizado para la temporización de renderizado de su juego, y devolverá un valor en milisegundos. Para más información sobre la sincronización de la pantalla, por favor, vea display_set_timing_method().

Sintaxis:

display_get_sleep_margin();

Devuelve:

Real

Ejemplo:

si display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
si display_get_sleep_margin() != 20
{
display_set_sleep_margin(20);
}
}

El código anterior comprobará el método de temporización y, si no está establecido como tm_sleep , se establecerá como tal y el margen de suspensión se establecerá en 20.
