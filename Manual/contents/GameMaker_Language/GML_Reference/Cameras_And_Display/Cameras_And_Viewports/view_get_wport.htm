view_get_wport

Esta función se puede utilizar para recuperar la anchura del puerto de vista dado.

Sintaxis:

view_get_wport(view_port)

Argumento Descripción
ver_puerto El puerto de visualización al que se dirige (0 - 7)

Devuelve:

Real

Ejemplo:

si view_get_wport(0) != (display_get_width () / 2)
{
view_set_wport(0, display_get_width() / 2);
}

El código anterior comprobará la anchura del puerto de la vista[0] y si no es la mitad de la anchura de la pantalla se establecerá en ese valor.
