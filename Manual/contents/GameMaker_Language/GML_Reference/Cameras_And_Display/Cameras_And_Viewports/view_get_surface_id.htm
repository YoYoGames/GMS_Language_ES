view_get_surface_id

Esta función se puede utilizar para recuperar el valor de ID único de la superficie asignada al puerto de vista dado (devolverá -1 si no se ha asignado ninguna superficie).

Sintaxis:

view_get_surface_id(view_port)

Argumento Descripción
ver_puerto El puerto de visualización al que se dirige (0 - 7)

Devuelve:

Real

Ejemplo:

si view_get_surface_id(0) == -1
{
view_set_surface_id(0, global.vSurf);
}

El código anterior comprobará si se ha asignado una superficie al puerto de la vista[0] y, si no lo ha hecho, se asignará una.
