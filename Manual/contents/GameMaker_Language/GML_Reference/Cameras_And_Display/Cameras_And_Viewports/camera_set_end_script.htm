camera_set_end_script

Esta función puede ser usada para establecer una función de script que será llamada al final de cada cuadro de juego en el que la cámara sea asignada a un puerto de vista visible y activo, después de que todo para esa vista cámara se haya renderizado. Se le da el valor único del ID de la cámara (tal y como lo devuelven las diferentes camera_create() funciones) y el ID de la función a llamar. El orden en que las funciones adjuntas a las cámaras y el renderizado real de la vista de la cámara es el siguiente:

  • Las cámaras de todos los puertos de vista visibles y activos tienen su función de actualización llamada
  • Entonces, para cada vista:
    • Se llama a la función de inicio de la cámara para esa vista
    • Se ejecutan los eventos de dibujo para esa vista
    • Se llama a la función de finalización de la cámara
  • Pasar a la siguiente vista visible activa y repetir...

Sintaxis:

camera_set_end_script(camera_id, script)

Argumento Descripción
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara.
guión El script para ejecutar cada fotograma del juego

Devuelve:

N/A

Ejemplo:

camera_set_end_script(view_camera[0], endCamera);

El código anterior establece el script final para la cámara asignada al puerto de vista [0].
