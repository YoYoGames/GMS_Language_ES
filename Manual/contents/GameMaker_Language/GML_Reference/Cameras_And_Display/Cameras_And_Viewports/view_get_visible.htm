view_get_visible

Esta función se puede utilizar para comprobar la visibilidad del puerto de vista dado. La función devolverá true si es visible y false si no lo es.

Sintaxis:

view_get_visible(view_port)

Argumento Descripción
ver_puerto El puerto de visualización al que se dirige (0 - 7)

Devuelve:

Booleano

Ejemplo:

if !view_get_visible(0)
{
view_set_visible(0, true);
}

El código anterior comprobará si el puerto de la vista[0] es visible y, si no lo es, se establece como visible.
