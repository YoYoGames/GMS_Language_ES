camera_set_begin_script

Esta función se puede utilizar para establecer una función de script que será llamada al principio de cada fotograma del juego en el que la cámara esté asignada a un puerto de vista visible y activo. Se da el valor único de la cámara ID de la cámara (tal y como la devuelven las diferentes camera_create() funciones) y el ID de la función a llamar. El orden en el que los scripts se adjuntan a las cámaras y el renderizado real de la vista de la cámara es el siguiente:

  • Las cámaras de todos los puertos de vista visibles y activos tienen su función de actualización llamada
  • Luego, para cada vista:
    • Se llama a la función de inicio de la cámara para esa vista
    • Se ejecutan los eventos de dibujo para esa vista
    • Se llama a la función de finalización de la cámara
  • Pasar a la siguiente vista visible activa y repetir...

Sintaxis:

camera_set_begin_script(camera_id, script)

Argumento Descripción
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara.
guión La función de script para ejecutar cada fotograma del juego

Devuelve:

N/A

Ejemplo:

camera_set_begin_script(view_camera[0], beginCamera);

El código anterior establece la función de inicio de script para la cámara asignada al puerto de vista [0].
