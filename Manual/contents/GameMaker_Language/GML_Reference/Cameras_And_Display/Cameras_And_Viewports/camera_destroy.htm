cámara_destrucción

Con esta función puedes destruir cualquier cámara. Cuando se llama a la función se suministra el valor único del ID de la cámara, que se obtiene de las funciones camera_create_* o del view_camera si se utiliza el Room Editor para configurar el puerto de vista y la vista. Nunca debes destruir una cámara que esté actualmente asignada a una vista visible, a menos que estés asignando una nueva cámara a esa vista en el mismo paso, y siempre debes destruir cualquier cámara que ha creado a través de código cuando ya no es necesario por su juego para evitar fugas de memoria, y también puede destruir las cámaras por defecto si usted tiene cualquier asignado en el editor de la sala, pero debe asignar una nueva cámara al puerto de vista (o desactivar o deshabilitarla), de lo contrario obtendrás resultados extraños.

Sintaxis:

camera_destroy(camera_id)

Argumento Descripción
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara.

Devuelve:

Real (valor único de identificación de la cámara)

Ejemplo:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

El código anterior destruye la cámara actualmente asignada al puerto de vista [0] y luego crea una nueva cámara y asigna su ID a ese puerto.
