view_set_surface_id

Con esta variable se puede establecer el contenido de un puerto de vista para dibujar a una superficie. Cuando se trabaja con superficies, a menudo se requiere que se capture toda la región visible de la pantalla a la superficie, por lo que se asignaría a un puerto de vista utilizando esta función. Esto significa que todo lo que se muestra en el puerto de vista elegido se dibujará ahora a la superficie asignada y el contenido de ese puerto de vista ya no se mostrará, lo que significa que tendrá que

  • Habilitar una nueva vista y dibujar la superficie sólo en esa vista (usando vista_actual para comprobar qué vista se está dibujando)
  • Dibuja la superficie en el evento Draw GUI de una instancia, ya que la capa GUI es independiente de las vistas.

Cuando se utiliza esta función se da el índice del puerto de la vista (de 0 a 7) y un índice de superficie (ya sea el superficie_aplicación o el valor del índice único devuelto por la función surface_create()) o, si se ha asignado previamente una superficie y se desea eliminarla, un valor de -1. Para más información sobre la asignación del puerto de vista a una superficie, véase también la variable view_surface_id.

Sintaxis:

view_set_surface_id(view_port, surf)

Argumento Descripción
ver_puerto El puerto de visualización al que se dirige (0 - 7)

Devuelve:

Real

Ejemplo:

si view_get_surface_id(0) == -1
{
view_set_surface_id(0, global.vSurf);
}

El código anterior comprobará si se ha asignado una superficie al puerto de la vista[0] y, si no lo ha hecho, se asignará una.
