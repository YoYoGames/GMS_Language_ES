view_set_visible

Esta función se puede utilizar para establecer la visibilidad del puerto de vista dado. La función toma el índice del puerto de vista (de 0 a 7) y un booleano true si es visible y false si no lo es.

Sintaxis:

view_set_visible(view_port, visible)

Argumento Descripción
ver_puerto El puerto de visualización al que se dirige (0 - 7)
visible Conmutación de la visibilidad(true es visible y false es invisible)

Devuelve:

N/A

Ejemplo:

if !view_get_visible(0)
{
view_set_visible(0, true);
}

El código anterior comprobará si el puerto de la vista[0] es visible y, si no lo es, se establece como visible.
