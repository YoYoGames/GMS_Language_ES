camera_get_view_target

Esta función puede utilizarse para recuperar el objetivo de seguimiento de la cámara dada. El valor devuelto puede ser un índice de objeto, un ID de instancia o -1 si no se ha establecido ningún objetivo de seguimiento.

Sintaxis:

camera_get_view_target(camera_id)

Argumento Descripción
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara

Devuelve:

Índice del objeto, ID de la instancia, o -1 si no se establece ningún objetivo.

Ejemplo:

var target = camera_get_view_target(view_camera[0]);
if target != obj_Player
{
camera_set_view_target(view_camera[0], obj_Player);
}

El código anterior recupera el objetivo de seguimiento de la cámara asignada al puerto de vista[0] y luego lo comprueba para ver si coincide con el índice del objeto "obj_Player". Si no es así, la cámara de la vista se configura para seguir una instancia de ese objeto.
