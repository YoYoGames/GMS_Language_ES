view_get_xport

Esta función se puede utilizar para recuperar la posición x del puerto de vista dado.

Sintaxis:

view_get_xport(view_port)

Argumento Descripción
ver_puerto El puerto de visualización al que se dirige (0 - 7)

Devuelve:

Real

Ejemplo:

if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2)
{
view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2));
}

El código anterior comprobará la posición x del puerto de la vista[0] y si no está donde tiene que estar se pone en esa posición.
