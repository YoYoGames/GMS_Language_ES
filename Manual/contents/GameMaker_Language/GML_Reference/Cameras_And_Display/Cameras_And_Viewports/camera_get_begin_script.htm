camera_get_begin_script

Esta función puede utilizarse para recuperar el ID de la función de script asignada como script de inicio para la cámara dada. Si no se asigna ninguna función de script, la función devolverá -1.

Sintaxis:

camera_get_begin_script(camera_id)

Argumento Descripción
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara

Devuelve:

ID de la función de script o -1 si no hay script asignado

Ejemplo:

var scr = camera_get_begin_script(camera_view[0]);
if scr != cutScene
{
camera_set_update_script(camera_view[0], cutScene);
}

El código anterior comprueba la función de script asignada como script de inicio para la cámara asignada al puerto de vista[0] y si no es "cutScene" se establece en esa función.
