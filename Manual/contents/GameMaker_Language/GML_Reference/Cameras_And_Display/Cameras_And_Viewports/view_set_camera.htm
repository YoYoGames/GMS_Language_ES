view_set_camera

Esta función establece una cámara en un puerto de vista específico. Debe indicar el puerto de visualización que desea establecer (de 0 a 7), y proporcionar el valor único del ID de la cámara (devuelto por las funciones camera_create() o cuando se utiliza view_get_camera()). Si da un valor de -1 como el ID de la cámara, entonces está eliminando una cámara del puerto de vista y tenga en cuenta que si ese puerto de vista está habilitado y visible puede obtener algunos resultados impredecibles.

Sintaxis:

view_set_camera(view_port, camera_id)

Argumento Descripción
ver_puerto El puerto de visualización al que se dirige (0 - 7)
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara

Devuelve:

N/A

Ejemplo:

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

El código anterior creará una nueva cámara y le asignará el puerto de vista[0].
