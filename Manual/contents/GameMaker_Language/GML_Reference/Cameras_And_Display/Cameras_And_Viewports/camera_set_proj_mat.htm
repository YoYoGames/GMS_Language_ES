camera_set_proj_mat

Esta función establece la matriz de proyección para una cámara determinada. Se le da el valor único del ID de la cámara (devuelto por las diferentes funciones camera_create() funciones) y una matriz de vista a aplicar. Puede encontrar más información sobre la creación de matrices de proyección en la sección Funciones de Matriz, específicamente matrix_build_projection_perspective() y matrix_build_projection_ortho(). Tenga en cuenta que si su cámara realiza un seguimiento automático de objetos - es decir, ha sido creada con camera_create_view() con un índice de objeto/identificación de instancia que no sea -1, o está configurando una cámara definida en el Editor de Salas, o está configurando la cámara por defecto - entonces esta matriz se sobrescribirá en el siguiente fotograma del juego.

Sintaxis:

camera_set_proj_mat(camera_id, matrix)

Argumento Descripción
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara.
matriz El ID único de la matriz devuelto cuando se creó la matriz.

Devuelve:

N/A

Ejemplo:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

El código anterior crea una nueva cámara y la asigna al puerto de vista[0]. Esta cámara tiene entonces sus matrices de vista y proyección establecidas.
