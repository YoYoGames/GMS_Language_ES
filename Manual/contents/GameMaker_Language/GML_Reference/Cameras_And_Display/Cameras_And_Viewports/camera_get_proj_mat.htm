camera_get_proj_mat

Esta función puede utilizarse para recuperar la matriz de proyección. La función devuelve el valor del ID de la matriz que puede utilizarse en otras funciones de matriz o para establecer la matriz de proyección de otra cámara (utilizando camera_set_proj_mat()).

Sintaxis:

camera_get_proj_mat(camera_id)

Argumento Descripción
camera_id El valor único del ID de la cámara devuelto cuando se creó la cámara

Devuelve:

ID de la matriz (real)

Ejemplo:

mat = camera_get_proj_mat(camera_view[0]);

El código anterior obtiene la matriz de proyección para la cámara asignada al puerto de vista[0].
