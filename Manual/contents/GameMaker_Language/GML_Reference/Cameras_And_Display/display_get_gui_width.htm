display_get_gui_width

Con esta función se puede obtener el ancho (en píxeles) de la GUI como se utiliza en el evento Draw GUI.

Sintaxis:

display_get_gui_width();

Devuelve:

Real

Ejemplo:

ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0);

El código anterior establecerá un anuncio que se mostrará en la parte superior derecha de la pantalla.
