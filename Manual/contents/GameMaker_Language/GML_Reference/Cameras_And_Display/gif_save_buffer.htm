gif_save_buffer

Con esta función puedes guardar una animación GIF. Se suministra el índice del GIF (devuelto por la función gif_open()) y la función devolverá un buffer de crecimiento alineado a 1 byte con los datos del GIF. Tenga en cuenta que los datos finales del GIF final habrá sido paletizado utilizando la paleta universal 884 (ver aquí para más información).

Sintaxis:

gif_save_buffer(gif_index);

Argumento Descripción
gif_index El ID del gif a guardar

Devuelve:

Entero (ID del búfer)

Ejemplo:

si save_gif == true
{
si count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
else
{
global.capture_buff = gif_save_buffer(gif_image);
count = 0;
save_gif = false;
}
count++;
}

El código anterior creará un archivo de imagen GIF con 30 fotogramas tomados de la superficie de la aplicación y luego lo guardará en un buffer.
