manija_de_ventana

Con esta función se puede obtener el valor del id interno de Windows (el HWND, un puntero). Esta función sólo es útil para los escritores de extensiones que necesitan el manejador de la ventana para llamar a la API de Windows en el código DLL (el puntero devuelto debe (el puntero devuelto debe ser convertido en una cadena y luego en C++ simplemente convertirlo en un HWND). La siguiente tabla muestra las plataformas soportadas junto con lo que devuelven:

Plataforma Devuelve
Windows Ventana HWND
macOS Clase NSWindow
Ubuntu(Linux) Mango de XWindow
HTML5 Lienzo ID

Sintaxis:

window_handle();

Devuelve:

Puntero

Ejemplo:

win_id = windows_handle();

El código anterior almacenará el id de la ventana del juego en una variable.
