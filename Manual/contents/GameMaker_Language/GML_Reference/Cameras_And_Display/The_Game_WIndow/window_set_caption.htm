window_set_caption

Con esta función puedes cambiar o establecer el título de la ventana de la sala en la que te encuentras. Esta leyenda aparece en la parte superior de la ventana, junto al icono del juego, cuando el juego no está en modo de pantalla completa.

Sintaxis:

window_set_caption(caption);

Argumento Descripción
título El nuevo título.

Devuelve:

N/A

Ejemplo:

si window_get_caption() != ""
{
window_set_caption("");
}

El código anterior comprobará el título de la ventana para ver si tiene texto o no, y si lo tiene lo establece como una cadena vacía para que no se muestre ningún título.
