ventana_set_max_width

Esta función se puede utilizar para establecer un ancho máximo de ventana para su juego. Si activas la opción de redimensionamiento de la ventana en las Opciones de Juego para la plataforma de destino, entonces el jugador puede redimensionar la ventana del juego a cualquier tamaño que desee, sin embargo usando esta función puedes limitar el ancho máximo al tamaño que especifiques. Si desea volver al comportamiento por defecto (es decir, sin mínimo), entonces utilice un valor de -1.

NOTA: Esta función sólo está disponible en el Windows, macOS y Linux plataformas.

Sintaxis:

window_set_max_width(width);

Argumento Descripción
ancho La anchura máxima en píxeles para la ventana del juego.

Devuelve:

N/A

Ejemplo:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

El código anterior establecerá la anchura y la altura mínima y máxima de la ventana del juego.
