window_view_mouse_get_x

Esta función devolverá la posición x del ratón en relación con la vista seleccionada.

NOTA: Para las funciones habituales del ratón, consulte la sección sobre la entrada del ratón

Sintaxis:

window_view_mouse_get_x( id );

Argumento Descripción
id El id de la vista con la que se va a comparar la posición del ratón.

Devuelve:

Real

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
var xx, yy;
xx = window_view_mouse_get_x(0);
yy = window_view_mouse_get_y(0);
if xx > 0 && xx < 32 && yy > 0 && yy < 32
{
b_press[0] = true;
}
}

El código anterior comprobará si se ha pulsado un botón del ratón, y si es así, obtendrá la posición del ratón en relación con la vista[0] y la comparará para ver si una variable debe establecerse como verdadera.
