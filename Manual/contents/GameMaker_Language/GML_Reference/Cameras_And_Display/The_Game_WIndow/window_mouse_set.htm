window_mouse_set

Con esta función puedes cambiar o fijar la posición del ratón dentro de la ventana del juego, lo que puede ser útil para juegos FPS, por ejemplo. La función sólo funcionará mientras el juego esté enfocado y usando alt + tab se desbloqueará el ratón.

NOTA: Para las funciones habituales del ratón, consulte la sección sobre la entrada del ratón

Sintaxis:

window_mouse_set(x, y);

Argumento Descripción
x La coordenada x en la ventana.
y La coordenada y en la ventana.

Devuelve:

N/A

Ejemplo:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

El código anterior centraría el ratón en la ventana del juego.
