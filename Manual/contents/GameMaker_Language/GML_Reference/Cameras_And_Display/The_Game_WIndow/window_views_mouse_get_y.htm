window_views_mouse_get_y

Esta función devuelve la coordenada y del ratón con respecto a todas las vistas activas y devuelve el mismo valor ratón_y.

NOTA: Para las funciones habituales del ratón, véase la sección sobre la entrada del ratón.

Sintaxis:

window_views_mouse_get_y();

Devuelve:

Real

Ejemplo:

myy = window_views_mouse_get_y();

Esto establecería miy a la coordenada y del ratón en la ventana relativa a todas las vistas activas.
