display_set_ui_visibility

Esta función puede utilizarse para mostrar u ocultar la interfaz de usuario del sistema en Android sólo. La función requiere que proporcione uno o más indicadores del sistema como un valor entero. Cuando se utiliza más de una bandera, es necesario combinarlas utilizando la función "o", como se muestra en el siguiente ejemplo. Puede encontrar una lista de banderas del sistema Android aquí.

Sintaxis:

display_set_ui_visibility(flags);

Argumento banderas
banderas Las banderas de Android SYSTEM a utilizar (como un valor entero).

Devuelve:

N/A

Ejemplo:

var flags = 1024|4096;
display_set_ui_visibility(flags);

El código anterior utilizará las banderas del sistema Android "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" y "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" para establecer la visibilidad de la pantalla.
