mostrar_obtener_orientación

Esta función devolverá una de las dos constantes que tiene GameMaker Studio 2 para indicarle si el dispositivo que ejecuta el juego está siendo sostenido en modo horizontal o vertical (véase la tabla siguiente). Tenga en cuenta que esta función puede no detectar correctamente la orientación del dispositivo cuando se utiliza en el módulo de destino HTML5. Sin embargo, esto es fácilmente imitado por el uso de la siguiente secuencia de comandos:

return (browser_width < browser_height);

Devolverá true para el retrato y false para el paisaje.

Sintaxis:

display_get_orientation()

Devuelve:

Constante

Constante Descripción
mostrar_paisaje El dispositivo se sostiene horizontalmente, es decir, el borde más largo es de izquierda a derecha, y el botón de menú está a la derecha.
mostrar_paisaje_inclinado Como arriba, sólo que ahora el botón del menú está a la izquierda.
mostrar_retrato El dispositivo se sujeta verticalmente, es decir, el borde más largo va de arriba a abajo y el botón del menú está en la parte inferior.
display_portrait_flipped Como en el caso anterior, sólo que ahora el botón del menú está en la parte superior.

Ejemplo:

si display_get_orientation() == display_landscape
{
global.Config = 0;
}
else
{
global.Config = 1;
}

El código anterior comprueba la orientación del dispositivo y establece una variable global en función del valor devuelto.
