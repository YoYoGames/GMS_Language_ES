guardar_pantalla

Con esta función puedes guardar una captura de pantalla del juego tal y como aparece actualmente en la ventana del juego. El archivo se guardará en el directorio de trabajo del juego, que en Windows se encuentra en <drive>:/Usuarios/Su nombre/AppData/Local/[Nombre del juego]/ y en Mac sería ~/Librería/Application Support/[Nombre del juego]/. La imagen capturada será el renderizado final de la superficie de la aplicación del juego, o (si la superficie de la aplicación ha sido desactivada) será una imagen de toda la pantalla o ventana. Es importante tener en cuenta que la función está diseñada para ser llamada en el evento Draw GUI End, que debería dar resultados consistentes en todas las plataformas. Utilizarla en cualquier otro evento puede no funcionar como se espera, y puede dar diferentes resultados a través de diferentes objetivos e incluso dispositivos. Si necesita más control sobre lo que se guarda, debe utilizar la función screen_save_part().

NOTA: Esta función no funciona en el objetivo HTML5.

Sintaxis:

screen_save(fname);

Argumento Descripción
fname El nombre del archivo para la captura de pantalla a guardar

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("S"))
{
screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

El código anterior toma una captura de pantalla del juego cuando el usuario presiona la tecla "S". Esta captura de pantalla se guarda en un directorio específico y se numera mediante una variable que luego se incrementa.
