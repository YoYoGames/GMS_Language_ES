display_reset

Esta función restablece los ajustes de la pantalla a los que se establecieron cuando se inició el juego, pero también permite cambiar el nivel actual de antialiasing a pantalla completa que se está utilizando y si se utiliza la sincronización vertical. Los niveles de antialiasing disponibles son 0, 2, 4 y 8, siendo el valor de inicio por defecto 0, y el ajuste por defecto de v-sync es falso (off). Activarv-sync puede proporcionar una experiencia de juego más suave, pero también necesitará más potencia de procesamiento, por lo que su impacto debe ser considerado cuidadosamente antes de usarlo, y lo mismo ocurre con el antialiasing, donde cuanto más alto sea el número, más procesamiento se requiere.

Como no todos los dispositivos de destino son iguales, algunos pueden no soportar el antialiasing 8x o 4x, por ejemplo, y por ello hay una variable de sólo lectura disponible para obtener los diferentes niveles de AA que el dispositivo que ejecuta el juego puede mostrar:

display_aa

Esta variable devolverá un valor entero basado en la configuración de bits para los diferentes niveles. Así que para sólo 2xAA, esto reportará 2, para la disponibilidad de 2x y 4x reportará 6. Para 8 y 4 informará 12. Para los 3 niveles (2,4 y 8) informará informará de 14.

Sintaxis:

display_reset(aa, vsync);

Argumento Descripción
aa El nivel de filtrado antialiasing (0, 2, 4 u 8).
vsync Activa (true) o desactiva (false) la sincronización vertical.

Devuelve:

Real

Ejemplo:

si (display_aa > 12)
{
display_reset(8, true);
}

El código anterior establecerá el nivel de anti-aliasing a 8 si es compatible y cambiará v-sync a on.
