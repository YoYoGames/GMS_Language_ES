gif_add_surface

Con esta función puede guardar un cuadro en un archivo GIF desde una superficie. Se suministra el ID del archivo GIF (devuelto por la función gif_open()) y un ID de superficie (devuelto por la función surface_create()), así como un tiempo de retardo entre fotogramas. El tiempo de retardo se calcula en 1/100 de segundo, pero tenga en cuenta que, una vez creado el GIF, si lo utiliza con fines promocionales, por ejemplo, los distintos navegadores pueden interpretar el retardo de los fotogramas de forma ligeramente diferente. diferente - vea aquí para más detalles. Puede (opcionalmente) suministrar los valores de desplazamiento X e Y de la superficie que se utiliza para añadir al al GIF, en cuyo caso la parte dibujada en el GIF comenzará desde el desplazamiento y no desde la posición por defecto (0,0), permitiéndole seleccionar sólo una sección de la superficie a añadir. También puede proporcionar un valor de cuantificación, que es un número entero entre 0 y 3, donde 0 es la cuantificación completa y 3 es la cuantificación cero (los valores más bajos tendrán una degradación posterior de la calidad, pero crean un GIF más pequeño). Tenga en cuenta que no hay mucha diferencia entre la calidad por defecto de 2, y la más alta de 3, y el uso de 3 ralentizará significativamente la creación del GIF, por lo que se debe tener cuidado al utilizar el valor máximo.

Tenga en cuenta que no hay límites incorporados en el número de fotogramas o en el tamaño de esos fotogramas en su GIF, pero si utiliza demasiada memoria entonces la función fallará y devolverá -1, de lo contrario devolverá 0.

Sintaxis:

gif_add_surface(gif_index, surface, delay_time, [xoffset], [yoffset]);

Argumento Descripción
gif_index El ID del gif al que añadir los datos de la superficie
superficie El ID de la superficie a utilizar como marco añadido
tiempo_de_retraso El tiempo de retardo de la trama (en centésimas de segundo)
[xoffset] Opcional] El desplazamiento a lo largo del eje X de la superficie a añadir desde
[yoffset] Opcional] El desplazamiento a lo largo del eje Y de la superficie a añadir desde
[cuantificación] Opcional] La cantidad de cuantificación de 0 a 3 (0 calidad mínima, 3 calidad máxima, por defecto es 2)

Devuelve:

Entero

Ejemplo:

si save_gif == true
{
si count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
else
{
gif_save(gif_image, "GameCapture.gif");
count = 0;
save_gif = false;
}
count++;
}

El código anterior creará un archivo de imagen GIF con 30 fotogramas tomados de la superficie de la aplicación y luego lo guardará.
