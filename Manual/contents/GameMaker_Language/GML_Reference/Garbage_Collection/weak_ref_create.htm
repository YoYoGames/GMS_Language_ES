weak_ref_create

Con esta función puedes crear una referencia débil a una estructura que puede ser utilizada para comprobar si la estructura está todavía "viva" (referenciada) o no en el juego. Se suministra la referencia referencia a la estructura que quieres rastrear, y la función devolverá otra estructura que es una referencia débil a esa estructura. Ten en cuenta que puedes comprobar si una referencia es "fuerte" o "débil" utilizando la función instanceof(), ya que una referencia fuerte devolverá "struct" o el nombre de la función constructora que creó la estructura, o "weakref" si es una referencia débil. También hay que tener en cuenta que la estructura de referencia débil tendrá una variable "ref" a la que se puede acceder para obtener la cadena de referencia a la estructura en cuestión, a menos que se haya recogido el garabato, en cuyo caso devolverá undefined.

Sintaxis:

weak_ref_create(struct_to_track);

Argumento Descripción
struct_to_track La estructura para la que se quiere crear una referencia débil

Devuelve:

Estructurar

Ejemplo:

inventory_ref = weak_ref_create(inventory);

El código anterior crea una referencia débil a una estructura y la almacena en una variable de instancia para su uso posterior.
