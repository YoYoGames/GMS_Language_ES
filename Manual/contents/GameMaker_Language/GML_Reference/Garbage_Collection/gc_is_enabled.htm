gc_is_enabled

Con esta función puedes comprobar si el recolector de basura está habilitado o no. La función devolverá true si está habilitado o false en caso contrario.

Sintaxis:

gc_is_enabled();

Devuelve:

Bool

Ejemplo:

if (!gc_is_enabled())
{
gc_enable(true);
}

El código anterior comprueba si el recolector de basura está habilitado y si no lo está lo habilita.
