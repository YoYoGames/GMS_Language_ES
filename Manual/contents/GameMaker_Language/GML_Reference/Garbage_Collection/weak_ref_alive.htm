weak_ref_create

Con esta función puede comprobar la referencia débil a una estructura para ver si sigue "viva" o no. Se suministra la referencia débil a comprobar (tal y como la devuelve la función weak_ref_create()), y la función devolverá true si la estructura sigue siendo referenciada en algún lugar o false si no lo está y ha sido recolectada por la basura. Tenga en cuenta que si suministra un valor que no es una referencia débil, la función devolverá undefined.

Sintaxis:

weak_ref_alive(weak_ref);

Argumento Descripción
weak_ref La referencia débil a la estructura que desea comprobar.

Devuelve:

Booleano (o indefinido)

Ejemplo:

if weak_ref_alive(inventory_ref)
{
inventory = -1;
}

El código anterior comprueba una referencia débil a una estructura y, si sigue viva, la variable que la referencia se pone a -1, desreferenciando la estructura y permitiendo su recogida de basura.
