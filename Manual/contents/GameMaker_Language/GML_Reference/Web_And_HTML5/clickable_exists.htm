clickable_exists

Esta función devuelve si se ha creado un icono DOM clicable con el índice especificado existe o no. Tenga en cuenta que el valor utilizado para la comprobación debe haber sido inicializado previamente o de lo contrario obtendrá un error que hará que GameMaker Studio 2 se cierre.

Sintaxis:

clickable_exists(index);

Argumento Descripción
índice El índice de la pieza seleccionable que se va a comprobar.

Devuelve:

Booleano

Ejemplo:

if !clickable_exists(home_but)
{
home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

El código anterior comprueba la variable "home_but" para ver si ya existe, y si no existe crea un icono DOM clicable en la posición (32, 32) de la página en la que se está ejecutando el lienzo del juego. El icono utiliza el sprite referenciado desde la página de textura como "spr_MS_Home" y cuando se hace clic en el icono se abrirá una nueva ventana para la URL especificada y con las propiedades definidas para esa ventana.
