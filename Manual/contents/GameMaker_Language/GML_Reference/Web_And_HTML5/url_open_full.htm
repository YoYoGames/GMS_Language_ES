url_open_full

Esto abrirá la URL especificada en el navegador del dispositivo de destino elegido o, si está utilizando el módulo HTML5, en el navegador actualmente abierto. El parámetro "target" que se especifica es el mismo que el valor "name" estándar de JavaScript cuando se utiliza el método open(). cuando se utiliza el método open() (tenga en cuenta que todo lo que no sea " _self" puede hacer que el navegador se bloquee, o que pregunte al usuario si desea permitirlo) y las "opciones" son las mismas que el parámetro "specs" de JavaScript para controlar qué propiedades debe mostrar la nueva ventana/pestaña (puede que no todos los navegadores soporten todas las características).

Los objetivos válidos son:

Objetivo Descripción
En blanco Abre el documento vinculado en una nueva ventana o pestaña (esto no funcionará si las ventanas emergentes están bloqueadas por el usuario, en cuyo caso puede utilizar las funciones clickable_* en su lugar).
_self Abre el documento vinculado en el mismo marco en el que se ha hecho clic (es el predeterminado).
_padre Abre el documento vinculado en el marco principal.
Arriba Abre el documento vinculado en el cuerpo completo de la ventana.

Las opciones válidas son:

Parámetros Descripción
'height=[px]' La altura de la ventana, siendo el valor mínimo 100.
'width=[px]' La anchura de la ventana, siendo el valor mínimo 100.
'left=[px]' La posición izquierda de la ventana.
'top=[px]' La posición superior de la ventana (sólo en IE).
'location=[boolean]' Mostrar o no el campo de la dirección (por defecto es 1).
'menubar=[boolean]' Mostrar o no la barra de menús (por defecto es 1).
'resizable=[boolean]' Si la ventana es redimensionable o no (por defecto es 1).
'scrollbars=[boolean]' Mostrar o no las barras de desplazamiento (por defecto es 1).
'status=[boolean]' Añadir o no una barra de estado (por defecto es 1).
'titlebar=[boolean]' Mostrar o no la barra de título. Esto se ignora a menos que la aplicación que llama sea una aplicación HTML o un cuadro de diálogo de confianza (por defecto es 1);
'toolbar=[boolean]' Mostrar o no la barra de herramientas del navegador (por defecto, sí).

Sintaxis:

url_open_full(url, target, options);

Argumento Descripción
url La URL (dirección del sitio web) a la que enlazar.
objetivo Esta es la zona de destino para abrir la URL (ver descripción).
opciones Opciones estándar del navegador (ver descripción).

Devuelve:

N/A

Ejemplo:

url_open_full("http://yoyogames.com", "_blank", "resizable=0, height=200, scrollbars=0");

Esto abriría la página de inicio de YoYo Games en una nueva ventana que no puede ser redimensionada, tiene una altura de 200 píxeles y no tiene barras de desplazamiento.
