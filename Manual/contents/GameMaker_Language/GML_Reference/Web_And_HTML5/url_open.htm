url_open

Esto abrirá la URL especificada en el navegador del dispositivo de destino elegido o, si está utilizando el módulo HTML5, en el navegador actualmente abierto.

Sintaxis:

url_open(url);

Argumento Descripción
url La URL (dirección del sitio web) a la que enlazar.

Devuelve:

N/A

Ejemplo:

url_open("http://yoyogames.com");

Esto abriría la página de inicio de YoYo Games en la ventana actual.
