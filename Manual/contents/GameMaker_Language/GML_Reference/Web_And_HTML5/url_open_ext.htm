url_open_ext

Esto abrirá la URL especificada en el navegador del dispositivo de destino elegido o, si está utilizando el módulo HTML5, en el navegador actualmente abierto. El parámetro "target" que se especifica es el mismo que el valor "name" estándar de JavaScript cuando se utiliza el método open(), y hay que tener en cuenta que todos, excepto los de la categoría "target", se abren en el navegador. cuando se utiliza el método open(), y debe tener en cuenta que todo lo que no sea " _self" puede hacer que el navegador se bloquee, o que pregunte al usuario si desea permitirlo.

Los objetivos válidos son:

Objetivo Descripción
En blanco Abre el documento vinculado en una nueva ventana o pestaña (esto no funcionará si las ventanas emergentes están bloqueadas por el usuario, en cuyo caso puede utilizar las funciones clickable_* en su lugar).
_self Abre el documento vinculado en el mismo marco en el que se ha hecho clic (es el predeterminado).
_padre Abre el documento vinculado en el marco principal.
Arriba Abre el documento vinculado en el cuerpo completo de la ventana.

Sintaxis:

url_open_ext(url, target);

Argumento Descripción
url La URL (dirección del sitio web) a la que enlazar.
objetivo Este es el área de destino para abrir la URL.

Devuelve:

N/A

Ejemplo:

url_open_ext("http://yoyogames.com", "_blank");

Esto abrirá la página de inicio de YoYo Games en una nueva ventana.
