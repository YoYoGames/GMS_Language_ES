evento_analítico

Esta función enviará el texto especificado al proveedor de analítica que hayas configurado a través de las Opciones de Juego HTML5. Esta función se puede utilizar para crear una analítica personalizada para rastrear algo fuera del ámbito del proveedor que se está utilizando.

Sintaxis:

analytics_event(string);

Argumento Descripción
cadena Una cadena para enviar al proveedor

Devuelve:

N/A

Ejemplo:

si hs_new
{
analytics_event("Nuevo hiscore de " + string (score));
}

El código anterior comprobará una variable para ver si es verdadera y, si lo es, se activará un evento especial de análisis con la cadena especificada.
