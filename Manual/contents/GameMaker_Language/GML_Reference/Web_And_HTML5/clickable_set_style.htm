clickable_set_style

Esta función le permite establecer las propiedades de estilo CSS para el botón dado a través de los pares clave/valor en el mapa Mapa DS. Es necesario haber creado previamente el elemento botón (usando clickable_add()) y el mapa DS y suministrar los índices almacenados de cada uno como argumentos.

Sintaxis:

clickable_set_style(index, map)

Argumento Descripción
índice El índice del icono clicable para estilizar.
mapa El índice del Mapa DS para establecer el estilo.

Devuelve:

N/A

Ejemplo:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

El código anterior cambiará el estilo del botón pulsable con el índice almacenado en la variable "botón".
