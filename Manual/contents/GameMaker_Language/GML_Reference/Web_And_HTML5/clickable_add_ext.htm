clickable_add_ext

Con esta función se puede crear un icono personalizado, sobre el que se puede hacer clic, que actúa como un "botón" fuera del lienzo del juego, pero dentro de la propia ventana del navegador. La función devolverá un valor de identificación que debe ser almacenado en una variable para ser utilizado en todas funciones de botón posteriores que hagan referencia al nuevo icono. Esta función es muy útil para evitar que el navegador cree una ventana emergente al hacer clic en el botón, ya que todo se hace a través del propio GameMaker Studio 2.

Esta función funciona exactamente igual que la función clickable_add() donde debe establecer la posición relativa al (0,0) de la ventana del navegador, dar la entrada de la página de textura para el sprite que desea que tenga el icono del botón (ver: sprite_get_tpe()), especificar una URL a abrir o un script (desde el Navegador de Activos de GameMaker Studio 2) a abrir o ejecutar y luego establecer el objetivo y los parámetros a utilizar. Sin embargo, esta función también tiene dos opciones adicionales que le permiten establecer el alfa del icono de 0 (totalmente transparente) a 1 (totalmente opaco) y la escala del icono, que es un valor multiplicador aplicado al sprite utilizado. Así, un valor de 1 es la escala por defecto, mientras que (por ejemplo) 0,5 sería la mitad de la escala y 2 sería el doble.

También vale la pena señalar que si el argumento de la URL que pasas es el nombre de una función de script (es decir, no comienza con "http://") Y el script se llama gmcallback_* entonces puedes obtener una alimentación directa de un evento html5. Los nombres de las funciones de script de GameMaker Studio 2 no están ofuscados si comienzan con gmcallback_.

Sintaxis:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Argumento Descripción
x La posición x dentro de la ventana.
y La posición y dentro de la ventana.
tpe La entrada de la página de textura para el sprite que se va a utilizar.
url La URL (dirección de la página web) a la que enlazar, o el script a ejecutar.
objetivo Esta es la zona de destino para abrir la URL (ver descripción).
params Los diferentes parámetros del icono (ver descripción).
alfa El alfa de la imagen del icono (por defecto 1).
escala La escala del icono (por defecto 1).

Devuelve:

Real

Ejemplo:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

El código anterior crea un icono DOM sobre el que se puede hacer clic en la posición (32, 32) de la página en la que se está ejecutando el lienzo del juego. El icono utiliza el sprite referenciado desde la página de textura como "spr_MS_Home" y cuando se hace clic en el icono se abrirá una nueva ventana para la URL especificada y con las propiedades definidas para esa ventana. el botón también se escalará al doble del tamaño del sprite original y a la mitad de un alfa de imagen de 1.
