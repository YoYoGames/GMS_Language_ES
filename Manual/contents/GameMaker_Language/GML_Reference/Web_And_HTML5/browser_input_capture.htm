browser_input_capture

Con esta función puedes establecer si la ventana del navegador debe capturar toda la entrada (establecerla en false) o si el juego debe capturar la entrada (establecerla en true). Ten en cuenta que esta función es para usarla sólo con el módulo HTML5.

Sintaxis:

browser_input_capture(enable)

Devuelve:

N/A

Ejemplo:

browser_input_capture(true);

El código anterior configura el juego para capturar todas las entradas del navegador en la plataforma de destino HTML5.
