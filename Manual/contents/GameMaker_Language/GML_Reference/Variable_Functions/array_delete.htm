array_delete

Con esta función puede eliminar un valor (o varios) de una matriz en cualquier posición. La función requiere que proporcione una variable que contenga el array, el índice (posición) en el array desde el que se va a borrar, así como el número de valores que se van a eliminar.

Sintaxis:

array_delete(variable, índice, número);

Argumento Descripción
variable La variable que contiene el array.
índice El índice (posición) en el array del que se va a borrar el valor o valores.
número El número de valores a eliminar.

Devuelve:

N/A

Ejemplo:

array_delete(score_array, 0, 10);

El código anterior borrará los 10 primeros valores de la matriz dada empezando por el índice 0 (por lo que los índices 0 a 9 serán eliminados de la matriz).
