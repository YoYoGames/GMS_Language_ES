array_equals

Con esta función puedes comprobar si dos matrices son iguales (equivalentes o iguales). Se dan los dos arrays a comprobar, y la función devolverá true si son iguales (equivalentes o iguales) o false si no lo son. Tenga en cuenta que esto no es lo mismo que comprobar si dos arrays son iguales utilizando ==, que no comprobará si los dos arrays contienen valores equivalentes, sino sólo si los arrays están referenciando el mismo array inicial. Por ejemplo:

var a = [1,2,3,4];
var b = [1,2,3,4];
if (a == b)
{
show_debug_message( "Esto nunca se disparará, ya que a y b no hacen referencia al MISMO array" );
}
if (array_equals(a, b))
{
show_debug_message( "Esto se disparará ahora" );
}

Sintaxis:

array_equals(var1, var2);

Argumento Descripción
var1 El índice del primer array.
var2 El índice del segundo array.

Devuelve:

Booleano

Ejemplo:

if !array_equals(matriz_de_inventario, matriz_de_artículo)
{
var len = array_length(inventory_array);
array_copy(item_array, 0, inventory_array, 0, len);
}

El código anterior comprobará dos matrices para ver si contienen valores equivalentes, y si no lo hacen entonces el código copiará todo el contenido de la matriz "inventory_array" a la matriz "item_array".
