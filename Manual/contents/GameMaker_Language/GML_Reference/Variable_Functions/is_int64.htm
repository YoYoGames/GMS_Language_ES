is_int64

Esta función devuelve si una variable dada es un entero de 64 bits o no. En algunos casos se quiere comprobar y ver qué tipo de datos tiene una variable en GameMaker Studio 2 y es entonces cuando se utilizaría esta función. Devuelve true o false dependiendo de si el valor es un int64 o no.

Sintaxis:

is_int64(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

if !is_int64(val)
{
show_debug_message("¡No es un entero de 64 bits!");
}

El código anterior comprueba la variable "val" para ver si contiene un int64 y si no lo es entonces muestra un mensaje en la consola de depuración.
