is_bool

Esta función devuelve si una variable dada es un booleano(verdadero o falso) o no. En algunos casos se quiere comprobar y ver si una variable en GameMaker Studio 2 tiene un valor booleano y es entonces cuando se utiliza esta función.

Sintaxis:

is_bool(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

si is_bool(val)
{
global.Sound = val;
}
else
{
global.Sound = true;
}

El código anterior comprueba la variable "val" para ver si es un número real y, si lo es, lo utiliza para establecer una variable global.
