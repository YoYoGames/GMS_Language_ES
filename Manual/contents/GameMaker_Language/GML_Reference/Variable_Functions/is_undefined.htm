is_undefined

Esta función devuelve si una variable dada está definida o no. En algunos casos se quiere comprobar y ver qué tipo de datos tiene una variable en GameMaker Studio 2 y es entonces cuando se utilizaría esta función. Devuelve true o false dependiendo dependiendo de si el valor está definido o no.

NOTA: Esta función no puede utilizarse para verificar la existencia de una variable. Utilice variable_instance_exists() o variable_global_exists() en su lugar.

Sintaxis:

is_undefined(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
show_debug_message("¡La entrada del mapa no existe!");
}

El código anterior comprueba la variable "val" para ver si es indefinida o no y muestra un mensaje de depuración si lo es.
