variable_instance_get

Con esta función puede obtener el valor de una variable con nombre dada. Usted suministra el valor único del ID de la instancia (que se puede encontrar en las Propiedades de la Instancia en el editor de la sala o se devuelve cuando se llama a la función instance_create_layer()), así como el nombre de la variable para obtener el valor de una cadena (ver el código de ejemplo más abajo). La función devolverá el valor de la variable, o undefined si la variable no existe.

IMPORTANTE! Si la variable que está obteniendo no existe entonces la función devolverá la palabra clave undefined y puede obtener errores que detendrán el funcionamiento del juego, así que si tiene dudas utilice la función variable_instance_exists primero.

Sintaxis:

variable_instance_get(instance_id, name);

Argumento Descripción
instance_id El valor de identificación único de la instancia a utilizar
nombre El nombre de la variable a obtener (como una cadena)

Devuelve:

Valor (cualquier tipo de datos) o indefinido (si la variable nombrada no existe)

Ejemplo:

if variable_instance_exists(id, "shields")
{
var ss = variable_instance_get(id, "shields");
}
else
}
var ss = -1;
}

El código anterior comprobará si una variable existe y, si lo hace, el valor que contiene se recupera y se almacena en una variable local. Si no existe, la variable local se pone a -1.
