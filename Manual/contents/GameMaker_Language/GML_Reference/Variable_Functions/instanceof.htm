de

Esta función se puede utilizar para obtener el nombre de la función que se utilizó para crear una estructura cuando ésta se creó utilizando una función constructora y la función nuevo operador. Se suministra la variable con la referencia del struct a comprobar y la función devolverá una cadena con el nombre de la función o undefined. Tenga en cuenta que si pasa a la función un literal de estructura (es decir, una estructura que fue creada sin usar una función constructora) entonces simplemente devolverá la cadena "struct". Esta función también puede utilizarse para comprobar si una referencia a un struct es un referencia débil o no, en cuyo caso la función devolverá la cadena "weakref" en lugar del nombre de la función que creó la estructura. Para más información, véase la función weak_ref_create()).

Sintaxis:

instanceof(struct_id);

Argumento Descripción
struct La referencia de la estructura a utilizar.

Devuelve:

Cadena o indefinido

Ejemplo ampliado:

En este ejemplo debemos definir primero la función que se utilizará como constructor de la estructura:

function init_struct(_a, _b, _c) constructor
{
a = _a;
b = _b;
c = _c;
}

Esta función puede utilizarse junto con el operador new para crear una estructura y rellenarla con las variables establecidas con los valores de los argumentos utilizados en la función:

mystruct = new init_struct(10, 100, "Hello World");

Entonces podemos obtener el nombre de la función que se utilizó así:

var _name = instanceof(mystruct);
if is_string(_name)
{
show_debug_message(_nombre);
}
