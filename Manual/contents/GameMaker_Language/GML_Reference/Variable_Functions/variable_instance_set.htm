variable_instance_set

Con esta función puede establecer el valor de una variable dada en una instancia. Se suministra el valor único del ID de la instancia (que se puede encontrar en las Propiedades de la Instancia en el editor de salas, o se devuelve cuando se llama a la función instance_create_layer()), así como el nombre de la variable para establecer el valor como una cadena (ver el ejemplo más abajo), y finalmente el valor a establecer (puede ser cualquier tipo de dato válido). Si la variable no existe ya en la instancia, se creará y se le asignará el valor.

Sintaxis:

variable_instance_set(instance_id, name, val);

Argumento Descripción
instance_id El valor de identificación único de la instancia a utilizar
nombre El nombre de la variable a establecer (como una cadena)
val El valor al que debe ajustarse la variable

Devuelve:

N/A

Ejemplo:

if !variable_instance_exists(id, "shields")
{
variable_instance_set(id, "shields", 0);
}

El código anterior comprobará si existe una variable de instancia en la instancia de llamada y, si no existe, se crea y se pone a 0.
