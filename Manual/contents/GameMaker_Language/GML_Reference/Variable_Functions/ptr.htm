ptr

Esta función intentará convertir un valor dado en un tipo de dato puntero, donde el valor debe ser un real, un string, un int64, un int32, o un ptr. Cualquier otra cosa hará que el juego se bloquee con un mensaje de error.

Sintaxis:

ptr(n);

Argumento Descripción
n El valor a convertir.

Devuelve:

Puntero

Ejemplo:

if !is_ptr(val)
{
val = ptr(application_surface);
}

El código anterior comprueba la variable "val" para ver si contiene un puntero y si no lo contiene se le asigna uno.
