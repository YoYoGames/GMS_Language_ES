is_infinity

Esta función devuelve si una variable dada es infinita (un número infinito) o no, devolviendo verdadero si lo es, y falso si no lo es.

Sintaxis:

es_infinito(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

if is_infinity(global.value)
{
show_debug_message("¡El valor es infinito!");
}

El código anterior comprueba la variable global "value" para ver si es infinita o no y muestra un mensaje de depuración si lo es.
