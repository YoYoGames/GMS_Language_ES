array_length_1d(Descatalogado)

ATENCIÓN! Esta función está obsoleta, ya que los arrays ya no están limitados a sólo 1 o 2 dimensiones, y como tal esta función sólo se suministra para el apoyo de los proyectos heredados. Todos los nuevos proyectos deben utilizar la forma actual de crear y acceder a arrays multidimensionales (ver aquí para más información).

Con esta función puedes obtener la longitud (número de entradas) de un array 1D. Para matrices 2D deberías utilizar las funciones altura_de_array_2d y array_length_2d para obtener la longitud de un array 1D.

IMPORTANTE!!! Si el array tiene más de 32.000 entradas esta función devolverá un valor erróneo y no debe ser utilizada.

Sintaxis:

array_length_1d(array_index);

Argumento Descripción
array_index El índice del array a comprobar.

Devuelve:

Real

Ejemplo:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
{
a[i] = -1;
}

El código anterior hará un bucle a través de una matriz y establecerá cada entrada a -1.
