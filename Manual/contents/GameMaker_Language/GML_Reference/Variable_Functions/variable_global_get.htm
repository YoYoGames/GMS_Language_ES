variable_global_get

Con esta función puede obtener el valor de una variable global con nombre. Se suministra el nombre de la variable global para obtener el valor como una cadena (véase el código de ejemplo a continuación) y la función devolverá el valor de la variable global o undefined si la variable no existe.

IMPORTANTE! Si la variable global que está obteniendo no existe entonces la función devolverá la palabra clave undefined y puede obtener errores que detendrán el funcionamiento del juego, así que en caso de duda utilice la función variable_global_exists primero.

Sintaxis:

variable_global_get(nombre);

Argumento Descripción
nombre El nombre de la variable global a obtener (como una cadena)

Devuelve:

Valor (cualquier tipo de datos) o indefinido (si la variable nombrada no existe)

Ejemplo:

if variable_global_exists("enemy_num")
{
show_debug_message("número_enemigo = " + string(variable_global_get("número_enemigo"));)
}

El código anterior comprobará si existe una variable global y, si lo hace, se mostrará en la consola.
