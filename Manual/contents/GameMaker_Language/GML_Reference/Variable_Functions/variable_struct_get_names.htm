variable_struct_get_names

Con esta función puedes recuperar un array poblado con los nombres de las variables de una estructura. Se pasa la referencia a la estructura para comprobarla, y cada entrada del array será una cadena con los nombres de las variables que contiene la estructura contiene.

Sintaxis:

variable_struct_get_names(struct);

Argumento Descripción
struct La referencia de la estructura a comprobar.

Devuelve:

Matriz (cada entrada es una cadena)

Ejemplo:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables para struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
show_debug_message(str);
}

El código anterior recuperará una matriz de los nombres de las variables para la estructura dada y luego los mostrará junto con sus valores en la salida de depuración.
