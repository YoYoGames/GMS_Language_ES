variable_struct_exists

Con esta función puede comprobar si una variable existe dentro de la estructura dada o no. Se suministra la referencia a la estructura y el nombre de la variable que se desea comprobar en forma de cadena (véase el código de ejemplo siguiente). La función devolverá true si existe una variable con el nombre dado para la estructura y false en caso contrario.

Sintaxis:

variable_struct_exists(struct, name);

Argumento Descripción
struct La referencia de la estructura a comprobar
nombre El nombre de la variable struct que se va a comprobar (como una cadena)

Devuelve:

Booleano

Ejemplo:

if !variable_struct_exists(mystruct, "shields"))
{
mystruct.shields = 0;
}

El código anterior comprobará si la variable llamada "shields" existe en la estructura dada y si no existe entonces se crea y se inicializa a 0.
