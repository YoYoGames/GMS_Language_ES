array_get

Con esta función puedes recuperar el valor de un índice en un array. La función requiere que proporcione una variable que contenga el array y el índice para obtener el valor dentro de ese array. Tenga en cuenta que si el índice de la matriz está fuera de los límites entonces el juego se bloqueará con un error. Esta función también se puede utilizar para matrices multidimensionales, siempre y cuando se especifique qué dimensión se quiere obtener cuando se proporciona el índice de la matriz, siguiendo este patrón:

// Matriz 1D
array_get(mi_matriz, 0);
// Matriz 2D
array_get(mi_matriz[0], 0);
// Matriz 3D
array_get(mi_matriz[0][0], 0);
// etc.

Sintaxis:

array_get(variable, índice);

Argumento Descripción
variable La variable que contiene el array.
índice El índice del array del que se va a obtener el valor.

Devuelve:

Cualquier tipo de dato válido que pueda contener un array.

Ejemplo:

for (var i = 0; i < 10; ++i;)
{
show_debug_message(array_get(my_array, i));
}

El código anterior mostrará en la consola los 10 primeros elementos de lo dado.
