is_vec4

En algunos casos se quiere comprobar qué tipo de datos tiene una variable en GameMaker Studio 2 y es entonces cuando se utiliza esta función. Devuelve true o false dependiendo de si el valor es un vector de 4 valores o no.

Sintaxis:

is_vec4(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

if !is_vec4(val)
{
show_debug_message("¡No es un vector de 4 valores!");
}

El código anterior comprueba la variable "val" para ver si contiene un vec4 y si no lo es entonces muestra un mensaje en la consola de depuración.
