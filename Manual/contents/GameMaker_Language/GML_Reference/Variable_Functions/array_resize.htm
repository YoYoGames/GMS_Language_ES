array_resize

Con esta función puedes redimensionar una dimensión de matriz existente para que tenga un nuevo tamaño. Se suministra el array a redimensionar así como el nuevo número de índices para el array, y la función redimensionará ese array. Tenga en cuenta que esta función está diseñada para redimensionar un array a una longitud menor, ya que se puede redimensionar simplemente estableciendo un nuevo índice en el array. Dicho esto, si la utilizas para aumentar el tamaño de una matriz, los nuevos índices se establecerán en el valor predeterminado de 0.

Esta función también se puede utilizar para matrices multidimensionales, siempre que se especifique qué dimensión se quiere redimensionar al suministrar el índice de la matriz, siguiendo este patrón:

// Redimensionar la primera dimensión de la matriz
array_resize(my_array, 10);

// Redimensionar la segunda dimensión de la matriz
array_resize(my_array[0], 10);

// Redimensionar la tercera dimensión de la matriz
array_resize(my_array[0][0], 10);

// etc.

Sintaxis:

array_resize(array_index, new_size);

Argumento Descripción
array_index El índice del array a redimensionar.
nuevo_tamaño El nuevo tamaño del array (un entero, empezando por 0).

Devuelve:

N/A

Ejemplo:

si array_length(a) > 10
{
array_resize(a, 10);
}

El código anterior comprueba la longitud de un array y si tiene más de 10 índices entonces se redimensiona.
