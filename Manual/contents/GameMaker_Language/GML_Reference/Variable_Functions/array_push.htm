array_push

Con esta función se puede empujar un valor (o valores) al final de una matriz sin tener que conocer la longitud de la misma. La función requiere que se proporcione una variable que contenga la matriz, así como al menos un valor para empujar, aunque puede proporcionar opcionalmente más argumentos y todos ellos serán introducidos en la matriz en orden consecutivo.

Sintaxis:

array_push(variable, valor, [valor], [valor], [etc...]);

Argumento Descripción
variable La variable que contiene el array.
valor El valor que se va a empujar al final de la matriz
[valor], [valor], [etc...] OPCIONAL! Otros valores que se introducen en el array

Devuelve:

N/A

Ejemplo:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

El código anterior empujará cuatro valores al final de la matriz dada.
