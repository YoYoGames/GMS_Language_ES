array_create

Con esta función puedes crear un array 1D de un tamaño determinado. Le dices a la función la longitud del array a crear, y te devolverá el "handle" del array que puedes asignar a una variable. Las matrices creadas de este modo tendrán cada entrada se inicializará a 0 a menos que se especifique un valor de inicialización (opcional). Si se especifica el valor extra para la inicialización del array, todos los índices del nuevo array se pondrán a ese valor en lugar de a 0, pero hay que tener en cuenta que la función tendrá una mayor sobrecarga de rendimiento en este caso.

Sintaxis:

array_create(tamaño, [valor]);

Argumento Descripción
tamaño El tamaño del array a crear.
valor El valor a utilizar para inicializar todos los índices del array (opcional).

Devuelve:

Referencia de la matriz

Ejemplo:

instance_array = array_create(100, noone);

El código anterior creará un nuevo array de 100 entradas, cada una de ellas con la palabra clave noone, y lo asignará a la variable "instance_array".
