variable_instance_exists

Con esta función puede comprobar si una variable de ámbito de instancia existe o no. Se proporciona el valor de ID de instancia único (que puede encontrarse en las Propiedades de Instancia en el editor de espacios, o se devuelve cuando se llama a la función instance_create_layer()) así como el nombre de la variable a comprobar como una cadena (véase el código de ejemplo más abajo). La función devolverá true si existe una variable con el nombre dado para la instancia y false en caso contrario.

Sintaxis:

variable_instance_exists(instance_id, name);

Argumento Descripción
instance_id El valor de identificación único de la instancia a comprobar
nombre El nombre de la variable a comprobar (como cadena)

Devuelve:

Booleano

Ejemplo:

if !variable_instance_exists(id, "shields"))
{
escudos = 0;
}

El código anterior comprobará si la variable llamada "escudos" existe en la instancia que ejecuta el código y, si no existe, se crea y se inicializa a 0.
