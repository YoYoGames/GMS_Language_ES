es_método

Esta función puede utilizarse para comprobar si una variable es una variable de método (devolverá true) o no (en cuyo caso devolverá false).

Sintaxis:

is_method(n);

Argumento Descripción
n La variable a comprobar.

Devuelve:

Booleano

Ejemplo:

if is_method(get_vec)
{
show_debug_message("¡Variable de método!");
}

El código anterior comprueba una variable para ver si es un método, y si la función devuelve true, entonces un mensaje de depuración es enviado a la consola.
