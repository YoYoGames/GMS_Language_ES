es_estructura

Esta función comprueba si el valor suministrado es una estructura. Devuelve true si lo es, en caso contrario devuelve false.

Tenga en cuenta que las variables de los métodos también devolverán true, y las instancias de los objetos devolverán false.

Sintaxis:

is_struct(val);

Argumento Descripción
val El valor a comprobar.

Devuelve:

Booleano

Ejemplo:

si is_struct(a)
{
delete(a);
}

El código anterior comprueba una variable para ver si es una estructura, y si la función devuelve true, la estructura se elimina.
