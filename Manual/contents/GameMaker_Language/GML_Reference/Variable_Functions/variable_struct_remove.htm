variable_struct_remove

Con esta función se puede eliminar una variable de una estructura. Se suministra el ID de la estructura de la que se va a eliminar la variable y el nombre de la cadena de la variable que se va a eliminar.

Sintaxis:

variable_struct_set(struct, name);

Argumento Descripción
struct La referencia a la estructura para eliminar la variable de
nombre El nombre de la variable a eliminar (como cadena)

Devuelve:

N/A

Ejemplo:

if variable_struct_exists(mystruct, "shields")
{
variable_struct_remove(mystruct, "shields");
}

El código anterior comprobará si la variable dada existe en la estructura dada y si lo hace entonces se elimina.
