bool

Esta función intentará convertir un valor dado en un tipo de dato booleano, donde el valor será devuelto como verdadero si es mayor que 0,5 y falso en caso contrario.

Sintaxis:

bool(n);

Argumento Descripción
n El valor a convertir.

Devuelve:

Booleano

Ejemplo:

if !is_bool(val)
{
val = bool(val);
}

El código anterior comprueba la variable "val" para ver si es un booleano y si no lo es entonces se convierte en uno.
