variable_struct_get

Con esta función puede obtener el valor de una variable con nombre dentro de una estructura. Se proporciona la referencia a la estructura y el nombre de la variable que se desea obtener como una cadena (véase el código de ejemplo siguiente). La función devolverá el valor que tiene la variable o undefined si la variable nombrada no existe.

IMPORTANTE! Si la variable que está obteniendo no existe entonces la función devolverá la palabra clave undefined y puede obtener errores que detendrán el funcionamiento del juego, así que si tiene dudas utilice la función variable_struct_exists() primero.

Sintaxis:

variable_struct_get(struct, name);

Argumento Descripción
struct La referencia de la estructura a utilizar
nombre El nombre de la variable a obtener (como una cadena)

Devuelve:

Valor (cualquier tipo de datos) o indefinido

Ejemplo:

if variable_struct_exists(mystruct, "shields")
{
var ss = variable_struct_get(mystruct, "shields");
}
else
}
var ss = -1;
}

El código anterior comprobará si existe una variable en la estructura dada y, si lo hace, el valor que contiene se recupera y se almacena en una variable local. Si no existe, la variable local se pone a -1.
