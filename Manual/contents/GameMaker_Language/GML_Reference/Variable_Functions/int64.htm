int64

Esta función intentará convertir un valor dado en un entero de 64 bits, donde el valor debe ser un real, una cadena, un int64, un int32, o un ptr. Cualquier cosa Cualquier otra cosa hará que el juego se bloquee con un mensaje de error. Puedes comprobar si una variable contiene un int64 utilizando la función is_int64().

Sintaxis:

int64(val);

Argumento Descripción
val El valor a convertir.

Devuelve:

Entero de 64 bits

Ejemplo:

steam_handle= int64(global.fileReadString);

El código anterior convierte el valor de la variable global en un entero de 64 bits.
