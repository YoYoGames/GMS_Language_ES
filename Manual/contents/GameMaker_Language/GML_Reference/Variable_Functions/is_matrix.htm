es_matriz

Esta función devuelve si una variable dada se refiere a una matriz o no. En algunos casos se quiere comprobar y ver qué tipo de datos tiene una variable en GameMaker Studio 2 y es entonces cuando se utilizaría esta función. Devuelve true o false dependiendo de si el valor es una matriz o no.

Sintaxis:

is_matrix(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

if !is_matrix(val)
{
show_debug_message("¡No es una matriz válida!");
}

El código anterior comprueba la variable "val" para ver si contiene una matriz y si no lo es entonces muestra un mensaje en la consola de depuración.
