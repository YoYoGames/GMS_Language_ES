array_pop

Con esta función puedes recuperar un valor del final de un array sin tener que conocer la longitud del mismo. Se suministra la variable que contiene la matriz, y la función devolverá el contenido de la última entrada de la matriz. Si la matriz está vacío, la función devolverá undefined.

Sintaxis:

array_pop(variable);

Argumento Descripción
variable La variable que contiene el array.

Devuelve:

Valor (cualquier tipo de datos) o indefinido

Ejemplo:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

El código anterior extraerá el último valor de la matriz dada y lo dibujará junto con algún texto en la pantalla.
