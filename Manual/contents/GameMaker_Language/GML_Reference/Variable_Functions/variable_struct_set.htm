variable_struct_set

Con esta función puede establecer el valor de una variable dada en una estructura. Se suministra la referencia a la estructura, así como el nombre de la variable a establecer el valor como una cadena (véase el código de ejemplo a continuación), y, finalmente, el valor a establecer (puede ser cualquier tipo de dato válido). Si la variable no existe ya en la estructura, se creará y se le asignará el valor.

Sintaxis:

variable_struct_set(struct, name, val);

Argumento Descripción
struct La referencia de la estructura para establecer
nombre El nombre de la variable a establecer (como una cadena)
val El valor al que debe ajustarse la variable

Devuelve:

N/A

Ejemplo:

if !variable_struct_exists(mystruct, "shields")
{
variable_struct_set(mystruct, "shields", 0);
}

El código anterior comprobará si la variable dada existe en la estructura dada y si no existe entonces se crea y se pone a 0.
