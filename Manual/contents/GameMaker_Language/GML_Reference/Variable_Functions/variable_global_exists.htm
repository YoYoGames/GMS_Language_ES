variable_global_exists

Con esta función puedes comprobar si una variable de ámbito global existe o no. Se suministra el nombre de la variable global a comprobar como una cadena (véase el código de ejemplo más abajo) y la función devolverá true si una variable global con el nombre dado existe o falso en caso contrario.

Sintaxis:

variable_global_exists(nombre);

Argumento Descripción
nombre El nombre de la variable global a comprobar (como cadena)

Devuelve:

Booleano

Ejemplo:

if !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

El código anterior comprobará si la variable global llamada "enemy_num" existe y si no existe se crea y se le asigna un valor.
