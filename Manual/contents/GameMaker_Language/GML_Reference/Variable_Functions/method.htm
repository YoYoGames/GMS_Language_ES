método

Con esta función se puede vincular cualquier función que haya sido previamente definida a una instancia o estructura determinada, creando una variable de método que puede ser utilizada posteriormente. Se suministra el ID de la instancia a utilizar ( debe ser una instancia que esté activa y en la sala, y no puede ser un ID de objeto) o una referencia de estructura, así como el ID de la función que se quiere vincular. La función devolverá un método que puede ser llamado desde la variable a la que está asignada (ver el ejemplo de código más abajo). Hay que tener en cuenta que se pueden enlazar funciones incorporadas y funciones definidas por el usuario, y que también se pueden suministrar funciones definidas por el usuario. y también puede proporcionar undefined como argumento de instancia/estructura, lo que significa que el ámbito actual auto ámbito actual se utilizará para la vinculación.

Sintaxis:

method(struct_ref_or_instance_id, function);

Argumento Descripción
struct_ref_or_instance_id La referencia única o valor de identificación de la estructura o instancia a utilizar (o indefinido para uno mismo)
función El ID de la función a utilizar

Devuelve:

Método

Ejemplo:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
enemy_func = method(_inst, enemy_ai);
}

El código anterior comprobará si existe una instancia en la posición indicada y, si es así, se vincula una función a la instancia y se devuelve como la variable de método "enemy_func".
