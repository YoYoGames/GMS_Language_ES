array_height_2d(Descatalogado)

ATENCIÓN! Esta función está obsoleta, ya que los arrays ya no están limitados a sólo 1 o 2 dimensiones, y como tal esta función sólo se suministra para el apoyo de los proyectos heredados. Todos los nuevos proyectos deben utilizar la forma actual de crear y acceder a arrays multidimensionales (ver aquí para más información).

Con esta función puedes obtener la altura (número de entradas) de la primera dimensión de un array 2D. Se suministra el array a comprobar y la salida de la función indica el número de entradas iniciales que contiene. Se puede obtener el número de entradas para la segunda dimensión del array utilizando la función array_length_2d.

Sintaxis:

array_height_2d(array_index);

Argumento Descripción
array_index El índice del array a comprobar.

Devuelve:

Real

Ejemplo:

for (var i = 0; i < array_height_2d(a); ++i;)
{
for (var j = 0; j < array_length_2d(a, i); ++j;)
{
a[i, j] = -1;
}
}

El código anterior hará un bucle a través de una matriz 2D y establecerá cada entrada a -1.
