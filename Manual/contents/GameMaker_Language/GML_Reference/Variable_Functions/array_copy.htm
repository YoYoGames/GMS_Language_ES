array_copy

Con esta función puedes copiar todo o parte de un array en otro array en cualquier posición. Para ello es necesario proporcionar los arrays de origen y destino (ambos deben haber sido creados previamente), así como una posición dentro del array de origen para copiar y una posición dentro de la matriz de destino para copiar. Por último, hay que especificar la longitud del array (o la longitud de la parte que se desea) a copiar. Si los datos que se copian superan la longitud de la matriz de destino, ésta se ampliará para aceptar los datos. se extenderá para aceptar los datos.

Esta función también se puede utilizar para matrices multidimensionales, siempre y cuando se especifique qué dimensión se quiere copiar al suministrar el índice de la matriz, siguiendo este patrón:

// Copiar de la primera dimensión de una matriz
// a la segunda dimensión de una matriz
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copiar de la tercera dimensión de una matriz
// a la primera dimensión de una matriz
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc.

Sintaxis:

array_copy(dest, dest_index, src, src_index, length);

Argumento Descripción
dest El ID del array a copiar.
dest_index El índice dentro del array al que se va a copiar.
src El ID del array del que se va a copiar.
src_index El índice con el array desde el que empezar a copiar.
longitud La longitud (número de índices del array) a copiar.

Devuelve:

N/A

Ejemplo:

if !array_equals(matriz_de_inventario, matriz_de_artículo)
{
var len = array_length(inventory_array);
array_copy(item_array, 0, inventory_array, 0, len);
}

El código anterior comprobará dos matrices para ver si contienen valores equivalentes, y si no lo hacen entonces el código copiará todo el contenido de la matriz "inventory_array" a la matriz "item_array".
