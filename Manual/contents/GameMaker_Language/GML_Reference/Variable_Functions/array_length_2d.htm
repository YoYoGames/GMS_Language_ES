array_length_2d(Descatalogado)

ATENCIÓN! Esta función está obsoleta, ya que los arrays ya no están limitados a sólo 1 o 2 dimensiones, y como tal esta función sólo se suministra para el apoyo de los proyectos heredados. Todos los nuevos proyectos deben utilizar la forma actual de crear y acceder a arrays multidimensionales (ver aquí para más información).

Con esta función puedes obtener la longitud (número de entradas) de la segunda dimensión de un array. Se proporciona el número de entradas de la primera dimensión y la función devolverá el número de entradas de la segunda dimensión que tiene el array (para encontrar la longitud de la primera dimensión utilice la función altura_de_array_2D()). La función devolverá 0 si la variable dada no es un array o 1 si la variable es un array 1D (ya que todavía hay 1 fila).

Sintaxis:

array_length_2d(array_index, n);

Argumento Descripción
array_index El índice del array a comprobar.
n La entrada del array para obtener la longitud.

Devuelve:

Real

Ejemplo:

for (var i = 0; i < array_height_2d(a); ++i;)
{
for (var j = 0; j < array_length_2d(a, i); ++j;)
{
a[i, j] = -1;
}
}

El código anterior hará un bucle a través de una matriz 2D y establecerá cada entrada a -1.
