variable_struct_names_count

Con esta función puede encontrar el número total de variables definidas para una estructura. Se suministra el ID de la estructura a comprobar, y la función devolverá un valor entero para el número de variables encontradas, o (si no existe ninguna estructura con el ID dado) -1.

Sintaxis:

variable_struct_names_count(struct_id);

Argumento Descripción
struct_id El valor de ID único de la estructura a comprobar.

Devuelve:

Entero

Ejemplo:

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

El código anterior recuperará el número de variables en la estructura dada y mostrará un mensaje de depuración en la salida de la consola con este valor.
