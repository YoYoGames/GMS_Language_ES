is_ptr

Esta función devuelve si una variable dada es un puntero o no. En algunos casos se quiere comprobar y ver qué tipo de datos tiene una variable en GameMaker Studio 2 y es entonces cuando se utilizaría esta función. Devuelve true o false dependiendo dependiendo de si el valor es un puntero o no.

Sintaxis:

es_ptr(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

if !is_ptr(val)
{
show_debug_message("¡No es una textura válida!");
}

El código anterior comprueba la variable "val" para ver si contiene un puntero y si no lo es entonces muestra un mensaje en la consola de depuración.
