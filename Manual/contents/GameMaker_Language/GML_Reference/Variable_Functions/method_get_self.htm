method_get_self

Con esta función se puede recuperar el ID de la instancia o la referencia de la estructura que es el self contexto utilizado cuando se llama al método. Si la variable no es un método, la función devolverá undefined. Tenga en cuenta que la función también puede devolver la constante pointer_null, en cuyo caso se está utilizando el yo actual en el momento de la llamada.

Sintaxis:

method_get_self(método);

Argumento Descripción
método La variable del método a comprobar.

Devuelve:

Integer, undefined, pointer_null

Ejemplo:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

El código anterior obtiene el autocontexto de la variable del método dado y lo muestra en la consola.
