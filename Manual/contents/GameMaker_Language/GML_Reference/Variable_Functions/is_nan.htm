is_nan

Esta función devuelve si una variable dada es NaN (no es un número) o no, devolviendo true si lo es, y false si no lo es.

Sintaxis:

es_nan(n);

Argumento Descripción
n El argumento a comprobar.

Devuelve:

Booleano

Ejemplo:

if is_nan(global.value)
{
show_debug_message("El valor no es un número");
}

El código anterior comprueba la variable global "valor" para ver si es un número o no y muestra un mensaje de depuración si no lo es.
