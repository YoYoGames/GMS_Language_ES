array_set

Con esta función puedes establecer el valor de un índice en un array a un valor. La función requiere que proporcione una variable que contenga el array, así como el índice a establecer y el valor a establecer. Esta función también puede utilizarse para matrices multidimensionales siempre que se especifique la dimensión que se desea establecer al proporcionar el índice de la matriz, siguiendo este patrón:

// Matriz 1D
array_set(mi_matriz, 0, 100);
// Matriz 2D
array_set(mi_matriz[0], 0, 100);
// Matriz 3D
array_set(mi_matriz[0][0], 0, 100);
// etc.

Sintaxis:

array_set(variable, índice, valor);

Argumento Descripción
variable La variable que contiene el array.
índice El índice de la matriz para establecer el valor.
valor El valor a fijar.

Devuelve:

N/A

Ejemplo:

for (var i = 0; i < 10; ++i;)
{
array_set(score_array, i, i*100));
}

El código anterior establecerá los 10 primeros elementos de la matriz dada con un valor.
