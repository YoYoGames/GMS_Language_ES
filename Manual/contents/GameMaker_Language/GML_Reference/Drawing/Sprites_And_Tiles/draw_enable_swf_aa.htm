draw_enable_swf_aa

Con esta función puede activar o desactivar el antialiasing (AA) para los sprites vectoriales en formato SWF. El AA simplemente suaviza los bordes de las imágenes vectoriales para darles un aspecto más agradable. La cantidad de AA utilizada dependerá del valor establecido mediante la función draw_set_swf_aa_level(). Por defecto está desactivado.

Sintaxis:

draw_enable_swf_aa(enable);

Argumento Descripción
activar Activar(true) o desactivar(false) el AA para todos los sprites del SWF.

Devuelve:

N/A

Ejemplo:

si draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

El código anterior comprobará el valor de AA para los sprites en formato SWF, y si es 0 habilita el AA y establece el valor a 0,5.
