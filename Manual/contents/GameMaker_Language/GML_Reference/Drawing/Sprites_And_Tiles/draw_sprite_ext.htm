dibujar_espiga_ext

Esta función dibujará el sprite dado como en la función dibujar_sprite() pero con opciones adicionales para cambiar la escala, la mezcla, la rotación y el alfa del sprite dibujado. El cambio de estos valores no modifica el recurso de ninguna manera (sólo cómo se dibuja), y puede utilizar cualquiera de las variables de sprite disponibles en lugar de valores directos para todos los argumentos de la función. NOTA: La mezcla de colores sólo se recomienda para el objetivo HTML5 cuando WebGL está habilitado, aunque puede establecer el color de mezcla si no está habilitado y mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un sprite duplicado sprite que se almacena en la caché y se utiliza cuando se requiere. Esto está lejos de ser óptimo y si se utilizan múltiples cambios de color, se ralentizará el rendimiento de los juegos a menos que se active WebGL. el tamaño de la caché de la fuente para tratar de limitar esto si es necesario utilizando la función sprite_set_cache_size().

NOTA: Esta función puede no funcionar como se espera cuando se utilizan sprites de animación de esqueleto, y puede encontrar que la función sólo dibuja el primer cuadro de la pose por defecto. Deberías usar las funciones draw_skeleton_* en su lugar.

Sintaxis:

draw_sprite_ext( sprite, subimg, x, y, xscale, yscale, rot, color, alpha );

Argumento Descripción
sprite El índice del sprite a dibujar.
subimg La subimagen (fotograma) del sprite a dibujar (image_index o -1 se correlacionan con el fotograma actual de la animación en el objeto).
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
yscale La escala vertical del sprite como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
pudrirse La rotación del sprite. 0= derecho hacia arriba, 90= rotación de 90 grados en sentido contrario a las agujas del reloj, etc...
color El color con el que se mezcla el sprite. c_white es para mostrarlo normalmente.
alfa El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).

Devuelve:

N/A

Ejemplo:

draw_sprite_ext(sprite_index, image_index, x, y, image_xscale, image_yscale, image_angle, image_blend, image_alpha);

Esto dibujará el sprite asignado a la instancia con todos sus valores por defecto (esencialmente lo mismo que usar draw_self).
