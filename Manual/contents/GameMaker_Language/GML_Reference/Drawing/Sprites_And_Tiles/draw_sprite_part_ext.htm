draw_sprite_part_ext

Esta función dibujará una parte del sprite elegido en la posición dada siguiendo las mismas reglas que por draw_sprite_part()sólo que ahora puedes escalar la parte, mezclar un color con ella, o cambiar su alfa cuando al dibujarla en la pantalla (igual que cuando se dibuja un sprite con draw_sprite_ext()). Debe tener en cuenta que si la página de la textura permite el recorte automático, esta función puede no funcionar como se espera, ya que el espacio extra "vacío" alrededor del sprite habrá sido eliminado para crear la página de textura. Para resolver este problema, tendrá que establecer la configuración de la página de textura (en el Editor de grupos de texturas) para desactivar el Recorte automático.

NOTA: Al dibujar con esta función, el desplazamiento x e y del sprite se ignoran y la parte del sprite se dibujará con la esquina superior izquierda en la posición x / y especificada en la sala.

NOTA: Esta función sólo es útil para sprites de mapa de bits y no funcionará con sprites SWF o JSON (Spine).

NOTA: La mezcla de colores sólo se recomienda para el objetivo HTML5 cuando WebGL está habilitado, aunque todavía se puede establecer el color de mezcla si no está habilitado y se mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un sprite duplicado sprite que se almacena en la caché y se utiliza cuando se requiere. Esto está lejos de ser óptimo y si se utilizan múltiples cambios de color, se ralentizará el rendimiento de los juegos a menos que se active WebGL. el tamaño de la caché de la fuente para tratar de limitar esto si es necesario utilizando la función sprite_set_cache_size().

Sintaxis:

draw_sprite_part_ext(sprite, subimg, left, top, width, height, x, y, xscale, yscale, color, alpha);

Argumento Descripción
sprite El índice del sprite a dibujar.
subimg La subimagen (fotograma) del sprite a dibujar (image_index o -1 se correlacionan con el fotograma actual de la animación en el objeto).
a la izquierda La posición x en el sprite de la esquina superior izquierda del área a dibujar.
top La posición y en el sprite de la esquina superior izquierda del área a dibujar.
ancho El ancho del área a dibujar.
altura La altura del área a dibujar.
x La coordenada x de donde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
yscale La escala vertical del sprite, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
color El color con el que se mezcla el sprite. c_white es para mostrarlo normalmente.
alfa El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).

Devuelve:

N/A

Ejemplo:

draw_sprite_part_ext(sprite_index, image_index, 8, 8, sprite_width-16, sprite_height-16, x, y, 2, 0.5, c_black, 1);

Esto dibujará el sprite asignado a las instancias (sprite_index) y su fotograma actual de animación (image_index), sin embargo recortará un margen de 8px de los cuatro lados del sprite. Entonces se estirará al doble de su anchura habitual, pero a la mitad de su altura habitual. altura, y aunque el alfa sigue siendo 1, se mezclará con negro (convirtiéndolo en una silueta).
