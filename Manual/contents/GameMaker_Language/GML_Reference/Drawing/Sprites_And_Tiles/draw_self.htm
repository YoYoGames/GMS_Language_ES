dibujar_autoridad

Esta función dibuja el sprite asignado a la instancia exactamente como se dibujaría si el evento draw no contuviera ningún código o acción, y reflejará y los cambios que se hayan realizado en las variables del sprite en otros eventos.

NOTA: Si la instancia no tiene ningún sprite asignado, esta función arrojará un error.

Sintaxis:

draw_self();

Devuelve:

N/A

Ejemplo:

draw_self();

Esto hace que la instancia se dibuje a sí misma con las propiedades definidas por las variables del sprite incorporado.
