dibujar_mapa_de_tilema

Esta función puede utilizarse para dibujar un tilemap dado en cualquier lugar de la sala. Se le da el ID del elemento tilemap (que se obtiene cuando se crea un elemento tilemap usando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y luego da la posición x/y para el dibujo (en la habitación). Tenga en cuenta que esto simplemente dibujará el tilemap en el punto especificado, utilizando la profundidad de la capa de la instancia que llama a la función. No mueve el tilemap - ni lo cambia de ninguna manera - y no importa si el tilemap está marcado como visible o no.

Sintaxis:

draw_tilemap(tilemap_element_id, x, y);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener la posición y de la celda
x La posición x dentro de la habitación para dibujar
y La posición y dentro de la habitación para dibujar

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

El código anterior obtiene el ID de la capa y luego lo utiliza para obtener un ID de mapa de azulejos específico que a su vez se utiliza para dibujar el mapa de azulejos en la posición del ratón.
