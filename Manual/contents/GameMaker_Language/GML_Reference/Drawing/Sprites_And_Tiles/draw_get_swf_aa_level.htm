draw_get_swf_aa_level

Esta función puede utilizarse para obtener el nivel de antialiasing (AA) para los sprites vectoriales en formato SWF. El valor de retorno estará entre 0 y 1 y muestra cómo de "suaves" serán dibujados los bordes de estos sprites. Puede establecer el nivel de AA utilizando la función draw_set_swf_aa_level().

Sintaxis:

draw_get_swf_aa_level();

Devuelve:

Real

Ejemplo:

si draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

El código anterior comprobará el valor de AA para los sprites en formato SWF, y si es 0 habilita el AA y establece el valor a 0,5.
