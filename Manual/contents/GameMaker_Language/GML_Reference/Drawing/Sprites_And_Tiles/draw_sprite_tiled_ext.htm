dibujar_espiga_etiqueta_acodada_ext

Esta función tomará un sprite y lo distribuirá repetidamente a lo largo de toda la vista (o la habitación si no hay vista definida), comenzando desde las coordenadas que se dan en la función y con cada mosaico escalado, mezclado con color y con el alfa que se defina (estas propiedades son las mismas que las utilizadas en dibujar_sprite_ext()). Esta función es sólo para proyecciones 2D (ortográficas), y no funcionará correctamente cuando se utilice una proyección de cámara 3D.

NOTA: La mezcla de colores sólo se recomienda para el objetivo HTML5 cuando WebGL está habilitado, aunque se puede establecer el color de mezcla si no está habilitado y se mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un sprite duplicado sprite que se almacena en la caché y se utiliza cuando se requiere. Esto está lejos de ser óptimo y si se utilizan múltiples cambios de color, se ralentizará el rendimiento de los juegos a menos que se active WebGL. el tamaño de la caché de la fuente para tratar de limitar esto si es necesario utilizando la función sprite_set_cache_size().

Sintaxis:

draw_sprite_tiled_ext(sprite, subimg, x, y, xscale, yscale, color, alpha);

Argumento Descripción
sprite El índice del sprite a dibujar.
subimg La subimagen (fotograma) del sprite a dibujar (image_index o -1 se correlacionan con el fotograma actual de la animación en el objeto).
x La coordenada x de donde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite. Un multiplicador ' 1 = escala normal, 0,5 es la mitad, etc.
yscale La escala vertical del sprite. Un multiplicador ' 1 = escala normal, 0,5 es la mitad, etc.
color El color con el que se mezcla el sprite. c_white es para mostrarlo normalmente.
alfa El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).

Devuelve:

N/A

Ejemplo:

draw_sprite_tiled_ext(sprite_index, image_index, x, y, 2, 2, c_red, 0.5);

Esto dibujará el sprite asignado a la instancia (sprite_index) y su cuadro de animación actual (image_index) en la posición x e y propia de la instancia, pero escalado al doble del tamaño normal, mezclado en rojo y con la mitad del alfa normal. El sprite se colocará en mosaico horizontal y verticalmente a través de la vista.
