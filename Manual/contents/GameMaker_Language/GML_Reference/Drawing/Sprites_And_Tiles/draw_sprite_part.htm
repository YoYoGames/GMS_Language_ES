dibujar_parte_de_espalda

Con esta función puedes dibujar parte de cualquier sprite en una posición determinada dentro de la sala. Al igual que con dibujar_sprite() puedes especificar un sprite y una sub-imagen para dibujar, entonces debes dar las coordenadas relativas dentro del sprite del área a seleccionar para dibujar. Esto significa que una posición izquierda de 0 y una posición superior de 0 sería la esquina superior izquierda del sprite y todas las coordenadas posteriores deberían tomarse desde esa posición. La imagen siguiente muestra un ejemplo de cómo funciona esto:

Debe tener en cuenta que si la página de textura permite el recorte automático, esta función puede no funcionar como se espera, ya que el espacio extra "vacío" alrededor del sprite habrá sido eliminado para crear la página de textura. Para resolver este problema, tendrá que establecer la configuración de la página de textura (en el Editor de grupos de texturas) para desactivar la opción Recorte automático.

NOTA: Al dibujar con esta función, el desplazamiento x e y del sprite se ignoran y la parte del sprite se dibujará con la esquina superior izquierda en la posición x / y especificada en la sala.

NOTA: Esta función sólo es útil para sprites de mapa de bits y no funcionará con sprites SWF o JSON (Spine).

Sintaxis:

draw_sprite_part(sprite, subimg, left, top, width, height, x, y);

Argumento Descripción
sprite El índice del sprite a dibujar.
subimg La subimagen (fotograma) del sprite a dibujar (image_index o -1 se correlacionan con el fotograma actual de la animación en el objeto).
a la izquierda La posición x en el sprite de la esquina superior izquierda del área a dibujar.
top La posición y en el sprite de la esquina superior izquierda del área a dibujar.
ancho El ancho del área a dibujar.
altura La altura del área a dibujar.
x La coordenada x de donde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.

Devuelve:

N/A

Ejemplo:

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y );

Esto dibujará el sprite asignado a las instancias (sprite_index) y su fotograma actual de animación (image_index), sin embargo recortará un margen de 4px de la anchura en ambos lados, y un margen de 8 píxeles de la altura desde la parte inferior del sprite original de 24x24 original de 24x24 píxeles.
