draw_sprite_pos

Con esta función se puede dibujar un sprite distorsionado sobre el área definida por las cuatro coordenadas de las esquinas. Los dos primeros argumentos son el sprite a dibujar y la subimagen del sprite (la misma que en la función dibujar_sprite()) pero los siguientes son los que definen la posición de cada una de las cuatro esquinas de la caja delimitadora del sprite dado. Estas coordenadas deben darse en el sentido de las agujas del reloj, de modo que la primera coordenada es la superior izquierda, luego la superior derecha, luego la inferior derecha y finalmente la inferior izquierda. También puede establecer un valor para el alfa del sprite para dibujarlo con transparencia. La imagen siguiente ilustra cómo funciona esta función:

ATENCIÓN! La imagen de arriba es sólo para fines ilustrativos, y si usted utiliza esta función en un sprite, obtendrá resultados diferentes y puede experimentar "cizallamiento" de la textura debido a la forma en que un sprite se construye a partir de un quad de primitivas.

NOTA: Esta función sólo es útil para sprites de mapa de bits y no funcionará con sprites SWF o JSON (Spine).

NOTA: Cuando utilice esta función, debe tener desactivado el ajuste Recorte automático para las páginas de textura.

Sintaxis:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

Argumento Descripción
sprite El índice del sprite a dibujar.
subimg La subimagen (fotograma) del sprite a dibujar (image_index o -1 se correlacionan con el fotograma actual de la animación en el objeto).
x1 La primera coordenada x.
y1 La primera coordenada y.
x2 La segunda coordenada x.
y2 La segunda coordenada y.
x3 La tercera coordenada x.
y3 La tercera coordenada y.
x4 La cuarta coordenada x.
y4 La cuarta coordenada y.
alfa El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).

Devuelve:

N/A

Ejemplo:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

El código anterior dibujará el sprite asociado a la instancia que ejecuta el código distorsionado alrededor de la posición x / y de la instancia y con un alfa totalmente opaco.
