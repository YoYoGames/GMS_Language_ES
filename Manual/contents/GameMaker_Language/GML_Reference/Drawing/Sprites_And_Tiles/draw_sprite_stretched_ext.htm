dibujar_espiga_extendida

Esta función hace exactamente lo mismo que la función dibujar_espiga_estirada() con la capacidad añadida de establecer la mezcla de colores y el valor alfa para el sprite cuando se dibuja (similar a la función dibujar_sprite_ext()).

NOTA: La mezcla de colores sólo se recomienda para el objetivo de HTML5 cuando WebGL está habilitado, aunque puede establecer el color de mezcla si no está habilitado y mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un sprite duplicado sprite que se almacena en la caché y se utiliza cuando se requiere. Esto está lejos de ser óptimo y si se utilizan múltiples cambios de color, se ralentizará el rendimiento de los juegos a menos que se active WebGL. el tamaño de la caché de la fuente para tratar de limitar esto si es necesario utilizando la función sprite_set_cache_sizee().

Sintaxis:

draw_sprite_stretched_ext(sprite, subimg, x, y, w, h, color, alpha);

Argumento Descripción
sprite El índice del sprite a dibujar.
subimg La subimagen (fotograma) del sprite a dibujar (image_index o -1 se correlacionan con el fotograma actual de la animación en el objeto).
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
w El ancho del área que ocupará el sprite estirado.
h La altura del área que ocupará el sprite estirado.
color El color con el que se mezcla el sprite. c_white es para mostrarlo normalmente.
alfa El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).

Devuelve:

N/A

Ejemplo:

draw_sprite_stretched_ext(sprite_index, image_index, x, y, sprite_width, sprite_height / 2, c_white, 0.5);

Esto dibujará el sprite asignado a la instancia y su sub-imagen con la esquina izquierda en la posición x /y de la instancia. Su anchura es la misma que la del sprite, y la altura es la del sprite dividida por dos. También se mezclará normalmente pero tendrá un valor alfa parcialmente transparente.
