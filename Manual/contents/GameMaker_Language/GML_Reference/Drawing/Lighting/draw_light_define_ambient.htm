dibujar_luz_definir_ambiente

Esta función se utiliza para controlar la luz ambiental de una escena, que es la luz que se tiene en una escena incluso sin haber definido ninguna fuente de luz puntual o direccional. Es efectivamente el color y el brillo (u oscuridad) general de una escena. El color por defecto es c_black.

Sintaxis:

dibujar_luz_definir_ambiente(col);

Argumento Descripción
col El color a utilizar (ya sea una constante, un real o un valor hexadecimal).

Devuelve:

N/A

Ejemplo:

dibujar_luz_definir_ambiente(c_blanco);

El código anterior definirá que la iluminación ambiental es blanca.
