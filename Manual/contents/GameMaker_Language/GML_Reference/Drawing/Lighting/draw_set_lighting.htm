draw_set_lighting

Esta función se utiliza para habilitar todos los efectos de iluminación. Por defecto está desactivado(falso).

Sintaxis:

draw_set_lighting(enable);

Argumento Descripción
activar activar o desactivar toda la iluminación(verdadero o falso)

Devuelve:

N/A

Ejemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

El código anterior habilitará la iluminación para toda la escena, luego definirá una luz blanca direccional en el espacio de la habitación, y finalmente encenderá esa luz.
