draw_light_get

Esta función obtendrá los parámetros de luz especificados como una matriz con los siguientes 6 elementos -

  • 0] = activado / desactivado(verdadero / falso)
  • [1] = tipo de luz (ver constantes más abajo)
  • [2] = posición x
  • [3] = posición y
  • [4] = posición z
  • 5] = radio de la luz (sólo para luces puntuales, será 0 para las direccionales)
  • [6] = color claro (un real)

El tipo de luz puede ser una de las dos constantes siguientes:

Constante Descripción
lighttype_dir La luz es una luz direccional
punto_de_luz La luz es una luz puntual

Sintaxis:

draw_light_get(ind);

Argumento Descripción
ind El número de índice de la luz (de 0 a 7)

Devuelve:

Real

Ejemplo:

light_a = draw_light_get(1);
if light_a[5] < 200
{
light_a[5] += 5;
draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

El código anterior obtendrá los valores utilizados para definir la luz indexada como "1", luego comprobará el radio y si es menor de 200 se incrementará y el radio de la luz se ajustará al nuevo valor.
