draw_light_enable

Esta función se utiliza para activar una luz definida. Cuando se define una luz posicional o direccional se le debe asignar un número de índice que luego es utilizado por esta función para encender o apagar la luz. Por defecto está desactivada(falso).

NOTA: Sólo hay 8 luces de hardware disponibles, por lo que sólo se pueden activar 8 luces definidas a la vez (aunque se pueden definir más).

Sintaxis:

draw_light_enable(ind, enable);

Argumento Descripción
ind El número de índice de la luz (de 0 a 7)
activar Activar o desactivar toda la iluminación(verdadero o falso)

Devuelve:

N/A

Ejemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

El código anterior habilitará la iluminación para toda la escena, luego definirá una luz blanca direccional en el espacio de la habitación, y finalmente encenderá esa luz.
