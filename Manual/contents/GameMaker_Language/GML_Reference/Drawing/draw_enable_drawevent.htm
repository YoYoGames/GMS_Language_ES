draw_enable_drawevent

Con esta función puedes elegir habilitar(true) o deshabilitar(false) el evento draw para todas las instancias del juego, dándote así el control sobre cómo y cuándo se dibujan las cosas, útil si deseas implementar una técnica de técnica de "salto de cuadro". Ten en cuenta que esto no sólo evita que las instancias dibujen en la pantalla, sino que suprime el evento draw por completo, lo que significa que hay que tener cuidado ya que cualquier lógica de juego que esté presente en ese evento no se ejecutará tampoco. Una cosa importante a entender sobre esta función es que si la llamas justo al comienzo del juego, antes de que el marco inicial sea renderizado (es decir: el evento Create del primer objeto en la primera habitación del juego), entonces la ventana del juego no será renderizada. Esto podría ser útil para proyectos online que requieren un servidor que no necesita renderizar nada.

Sintaxis:

draw_enable_drawevent(enable);

Argumento Descripción
Activar Establecer como verdadero o falso.

Devuelve:

N/A

Ejemplo:

frame_skip ++;
if (frame_skip mod 5) == 0
{
draw_enable_drawevent(true);
}
else
{
draw_enable_drawevent(false);
}

El código anterior comprueba una variable y si devuelve true, habilita el evento de sorteo, de lo contrario el evento de sorteo se desactiva.
