draw_set_colour

Con esta función puedes establecer el color de dibujo base para el juego. Este valor afectará a todos los dibujos posteriores cuando sea apropiado, incluyendo fuentes, formas, primitivas y 3D. Si alguno de esos activos se dibuja con su propio valor de color cambiado, este valor será ignorado.

Sintaxis:

draw_set_colour(col);

Argumento Descripción
col El color a establecer para el dibujo.

Devuelve:

N/A

Ejemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "NIVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "NIVEL 1");

El código anterior dibujará un texto en la posición especificada con un efecto de sombra creado por la modificación del alfa y el color del dibujo.
