fusionar_color

Con esta función se pueden tomar dos colores y fusionarlos para crear un nuevo color. La cantidad de cada uno de los colores componentes se puede definir cambiando el argumento "cantidad", donde un valor de 0 devolverá el primer color (col1), un valor de 1 devolverá el segundo color (col2) y un valor intermedio devolverá la mezcla correspondiente. Por ejemplo, un valor de 0,5 mezclará los dos colores por igual. La siguiente imagen ilustra el funcionamiento de la mezcla de los colores rojo y azul juntos:

Sintaxis:

fusionar_color(col1, col2, cantidad);

Argumento Descripción
col1 El primer color que se fusiona
col2 El segundo color para fusionar
cantidad La cantidad de cada color que se debe fusionar. Por ejemplo, 0 devolverá col1, 1 devolverá col2, y 0.5 devolverá una fusión de ambos colores por igual

Devuelve:

Int

Ejemplo:

col = merge_colour(c_lime, c_orange, 0.5);

El código anterior utiliza la función para crear un color mediante la fusión de lima y naranja 50/50 y luego almacena su valor en la variable "col" para su uso posterior.
