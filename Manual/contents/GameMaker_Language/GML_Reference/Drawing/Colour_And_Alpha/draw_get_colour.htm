dibujar_obtener_color

Esta función devuelve el color de dibujo actual que se utiliza para dibujar formas, texto, primitivas y modelos 3D sin textura. Se puede establecer con la función draw_set_colour() función.

Sintaxis:

dibujar_obtener_color()

Devuelve:

Real

Ejemplo:

si draw_get_colour() != c_black
{
draw_set_colour(c_black);
}

El código anterior comprueba primero el color de dibujo actual y, si no está establecido en c_negro, lo establece en c_negro.
