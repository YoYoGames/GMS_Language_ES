make_colour_rgb

GameMaker Studio 2 proporciona esta función (así como otras) para permitir al usuario crear sus propios colores. Esta función en particular toma tres componentes, el rojo, el verde y el azul del color que que se desea crear. Estos valores se toman entre 0 y 255, por lo que puedes hacer 16.777.216 (256*256*256) colores con esto. A continuación puedes ver una imagen de cómo se ven estos componentes cuando se separan:

La imagen de la izquierda es un desglose de los componentes individuales de la función, y luego a la derecha es una ilustración de cómo el cambio de estos componentes afecta al color final.

Sintaxis:

make_colour_rgb(rojo, verde, azul);

Argumento Descripción
rojo El componente rojo del color
verde El componente verde del color
azul El componente azul del color

Devuelve:

Int

Ejemplo:

col = make_colour_rgb(100, 145, 255);

El código anterior utiliza la función para crear un color y almacenar su valor en la variable "col" para su uso posterior.
