draw_get_alpha

Esta función devuelve el valor actual del alfa de dibujo, que oscilará entre 0 (totalmente transparente) y 1 (totalmente opaco). El alfa de dibujo afecta a la transparencia de todas las funciones de dibujo, y puede establecerse con la función draw_set_alpha() función.

Sintaxis:

draw_get_alpha()

Devuelve:

Real

Ejemplo:

si draw_get_alpha() != 1
{
draw_set_alpha(1);
}

El código anterior primero comprueba el alfa de dibujo actual y luego, si no está establecido en 1, lo establece en 1.
