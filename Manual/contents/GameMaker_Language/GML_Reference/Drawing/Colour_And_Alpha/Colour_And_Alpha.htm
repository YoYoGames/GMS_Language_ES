Color y Alfa

GameMaker Studio 2 tiene una amplia selección de funciones para manipular el color y el alfa de lo que se dibuja en la pantalla (para mezclar imágenes utilizando modos de mezcla especiales y otras funciones específicas de la PGU funciones específicas de la PGU, véase la sección sobre el Control de la GPU. Usando estas funciones puedes separar los colores en sus partes componentes, crear nuevos colores y manipular el alfa del dibujo para cambiar la forma en que se dibujan las diferentes cosas en tu juego. cosas en tu juego se dibujan juntas. Tenga en cuenta que también puede crear colores a partir de su valor hexadecimal utilizando el símbolo "$" de antemano, y estos valores también se pueden utilizar en las funciones de color para obtener matices de los componentes o para la mezcla, etc. El valor hexadecimal de estos colores se divide en tres partes con el formato Azul / Verde / Rojo así que, por ejemplo, para definir un color púrpura se podría utilizar

col = $983c95

Donde 98 es el componente azul, 3c es el componente verde y 95 es el componente rojo.

Las siguientes funciones se pueden utilizar para obtener las tonalidades de los componentes, el tono compuesto, la saturación y la luminosidad de un color seleccionado, así como el alfa y otras propiedades:

  • color_get_blue
  • color_get_verde
  • color_get_red
  • color_get_hue
  • color_get_saturation
  • color_get_value
  • dibujar_getpixel
  • draw_getpixel_ext
  • dibujar_obtener_color
  • draw_get_alpha

Estas funciones pueden utilizarse para crear colores a partir de los valores de entrada brutos de los componentes de color:

  • make_colour_hsv
  • make_colour_rgb
  • fusionar_color

Las siguientes funciones se pueden utilizar para establecer las diferentes opciones para dibujar en la pantalla, incluyendo el alfa, el color y la mezcla:

  • dibujar_despejar
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
