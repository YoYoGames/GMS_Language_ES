draw_set_alpha

Con esta función puedes establecer el alfa base de dibujo para el juego. Este valor puede establecerse de 0 a 1, siendo 0 totalmente transparente y 1 totalmente opaco (el valor por defecto), y afectará a todo el dibujo posterior, incluyendo fondos, sprites, fuentes primitivas y 3D.

Tenga en cuenta que si está dibujando cualquiera de los activos y suministrando un argumento alfa como parte de la función (por ejemplo, cuando se utiliza dibujar_espiga_ext()), entonces el valor alfa establecido aquí será anulado, por lo que dibujar un sprite usando draw_sprite_ext() con un argumento alfa de 1 dibujará el sprite completamente opaco independientemente del alfa establecido con draw_set_alpha() (esto funcionará igual cuando se use dibujar_self() y estableciendo el imagen_alfa de la instancia). También hay que tener en cuenta que cuando una instancia está dibujando por defecto (es decir, tiene un sprite pero no un evento de dibujo), entonces la variable image_alpha se ajustará al alfa establecido con draw_set_alpha(), de forma que un image_alpha de 1 y un draw alpha de 0.5 dibujarán el sprite a alfa 0.5, mientras que un image_alpha de 0.25 y un draw alpha de 0.5 dibujará el sprite a 0.25 alfa.

Sintaxis:

draw_set_alpha(alpha);

Argumento Descripción
alfa El alfa a establecer (entre 0 y 1)

Devuelve:

N/A

Ejemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "NIVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "NIVEL 1");

El código anterior dibujará un texto en la posición especificada con un efecto de sombra creado por la modificación del alfa y el color del dibujo.
