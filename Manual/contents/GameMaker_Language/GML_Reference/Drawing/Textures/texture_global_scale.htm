textura_escala_global

Esta función permite controlar el escalado de las páginas de texturas al cargarlas desde el archivo WAD que forma parte del ejecutable final del juego. El valor de entrada debe ser un valor de potencia de dos y funcionará así:

  • Para establecer que las páginas de textura sean 1:1 utiliza texture_global_scale(1)
  • Para que las texturas se reduzcan a la mitad todo el tiempo, utiliza texture_global_scale(2)
  • Para descuadrarlas utiliza texture_global_scale(4), etc...

De esta manera puedes controlar mejor el uso de la memoria de la página de texturas en plataformas con problemas de poca memoria. Tenga en cuenta que las páginas de texturas se crean bajo demanda del WAD, y por lo tanto puede llamar a esto al comienzo del juego para cargar todas las texturas a escala, o puede puede llamarlo en momentos específicos para cargar texturas específicas, en cuyo caso probablemente querrá llamar a draw_texture_flush() antes de llamar a esta función, y luego utilizar las funciones de pre-fetch para obtener la correcta páginas de textura en la memoria.

Sintaxis:

texture_global_scale(pow2integer);

Argumento Descripción
pow2integer El factor de escala a utilizar (1, sin escala, 2, media escala, 4, cuarto de escala, etc...)

Devuelve:

N/A

Ejemplo:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

El código anterior vaciará todas las texturas de la memoria, luego establecerá el escalado de las texturas a 2 (para que las páginas de texturas tengan la mitad de tamaño) y finalmente precargará la página de texturas que contiene el sprite "spr_Trees".
