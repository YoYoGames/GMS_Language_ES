texture_set_stage

Esta función establecerá la "ranura" del escenario dado una textura para ser utilizada. El número de "ranuras" de escenario disponibles dependerá de la plataforma en la que estés compilando, con un máximo de 8 disponibles para Windows, Mac y Linux, pero en dispositivos Android de gama baja (por ejemplo) este número puede ser de 2. dispositivos Android de gama baja (por ejemplo) este número puede ser tan bajo como 2. También vale la pena señalar que la primera etapa "ranura" siempre se utiliza automáticamente por GameMaker Studio 2.

NOTA: ¡Esta función no hará nada fuera del contexto de un shader en ejecución! Ver Sombreadores para más información.

Sintaxis:

texture_set_stage(stage, tex);

Argumento Descripción
escenario La "ranura" de la textura a utilizar.
tex La textura a utilizar.

Devuelve:

N/A

Ejemplo:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set( GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

El código anterior obtendrá el manejador del muestreador dentro del shader indexado como "GlassShader" y luego establecerá esa constante del shader a la textura del sprite dada.
