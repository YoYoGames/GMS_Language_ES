texturegroup_get_tilesets

Con esta función puedes recuperar el índice de tileset de cada uno de los tilesets asignados a las páginas de textura dentro del grupo de texturas dado. Se suministra la cadena de ID del grupo de texturas (como se define en el Editor de grupos de texturas) y la función devolverá una matriz 1D en el que cada entrada contiene el índice de tileset para un recurso de tileset. Si la función falla, es decir, si se proporciona un grupo no válido o si el grupo no tiene ninguna textura asignada, la matriz estará vacía (longitud 0).

Sintaxis:

texturegroup_get_tilesets(tex_id);

Argumento Descripción
tex El nombre del grupo de texturas a comprobar (una cadena)

Devuelve:

Matriz 1D

Ejemplo:

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i]));
}

El código anterior recuperará los índices del tileset para el grupo de texturas "MainMenu", y luego mostrará esos IDs en la ventana de salida de la consola.
