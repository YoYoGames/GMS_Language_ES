texture_get_height

Devuelve la altura de la textura con el id dado, que siempre es un valor dentro del rango 0 - 1. Esto puede ser usado cuando se mapean texturas a modelos o primitivas.

Sintaxis:

texture_get_height(tex);

Argumento Descripción
tex El puntero del activo de la página de la textura a utilizar

Devuelve:

Real

Ejemplo:

tex_h = texture_get_height(surface_get_texture(global.Surf));

El código anterior obtendrá la altura de la textura tomada de una superficie previamente creada.
