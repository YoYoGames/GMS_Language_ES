texturegroup_get_textures

Esta función se puede utilizar para recuperar los ID de las páginas de textura de las páginas individuales que componen un grupo de texturas. Se suministra la cadena de ID del grupo de texturas (como se define en el Editor de Grupos de Texturas), y la función devolverá una matriz 1D, donde cada entrada en la matriz es un ID de página de textura individual. Si la función falla - es decir, si se da un grupo no válido, o el grupo no tiene ninguna textura asignada - entonces la matriz estará vacía (longitud 0).

Sintaxis:

texturegroup_get_textures(tex_id);

Argumento Descripción
tex_id El nombre del grupo de texturas a comprobar (una cadena)

Devuelve:

Matriz 1D

Ejemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
if texture_is_ready(_tex_array[i])
{
texture_prefetch(_tex_array[i]);
}
}

El código anterior recuperará los IDs de las páginas de texturas para el grupo de texturas "MainMenu", luego comprobará si están desempaquetadas, y si lo están las colocará en la VRAM.
