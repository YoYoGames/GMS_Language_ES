texturegroup_get_fonts

Con esta función puedes recuperar el índice de fuente de cada una de las fuentes asignadas a las páginas de textura dentro del grupo de textura dado. Se suministra la cadena de ID del grupo de texturas (como se define en el Editor de grupos de texturas) y la función devolverá una matriz 1D donde cada entrada contiene el índice de la fuente para un recurso de la fuente. Si la función falla, es decir, si se proporciona un grupo no válido o si el grupo no tiene ninguna textura asignada, la matriz estará vacía (longitud 0).

Sintaxis:

texturegroup_get_fonts(tex_id);

Argumento Descripción
tex El nombre del grupo de texturas a comprobar (una cadena)

Devuelve:

Matriz 1D

Ejemplo:

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
show_debug_message("Fuente " + string(i) + " Índice:" + string(tex_array[i]));
}

El código anterior recuperará los índices de las fuentes para el grupo de texturas "MainMenu", y luego mostrará esos IDs en la ventana de salida de la consola.
