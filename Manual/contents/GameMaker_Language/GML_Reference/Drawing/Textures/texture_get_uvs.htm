texture_get_uvs

Esta función devuelve un array 1D con 4 elementos que representan las coordenadas UV para la imagen en la página de la textura, rellenando el array con los siguientes valores:

  • [0] = izquierda
  • [1] = arriba
  • [2] = derecho
  • [3] = fondo
  • 4] = cantidad de píxeles que el compilador de activos ha recortado del lado izquierdo de los sprites (sólo activos de sprites)
  • 5] = cantidad de píxeles que el compilador de activos ha recortado de la parte superior de los sprites (sólo activos de sprites)
  • 6] = porcentaje normalizado de datos de píxeles del ancho de los sprites originales que se ha guardado en la página de texturas (sólo activos de sprites)
  • 7] = porcentaje normalizado de datos de píxeles de la altura de los sprites originales que se ha guardado en la página de texturas (sólo activos de sprites)

Este valor puede ser usado en otras funciones de dibujo, particularmente en las funciones generales de 3D y algunas de las primitivas de 2D, así como en las funciones de Shader. Si necesitas el UVS para un sprite entonces puedes usar las funciones sprite_get_uvs() y para una fuente, font_get_uvs().

Sintaxis:

texture_get_uvs(texid)

Argumento Descripción
texid El activo de la imagen para obtener la textura de

Devuelve:

Matriz 1D (4 - 8 elementos)

Ejemplo:

var tex = texture_get_uvs(surf_back);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

El código anterior almacenará las coordenadas UV para la textura de la superficie dada en una matriz local y luego asignará los valores a las variables de instancia.
