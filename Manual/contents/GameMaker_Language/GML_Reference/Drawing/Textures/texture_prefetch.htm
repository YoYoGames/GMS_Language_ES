textura_prefetch

Esta función puede usarse para "precargar" una página de textura o un grupo de páginas de textura, es decir, cargarlas en la VRAM cuando sea necesario. Puedes suministrar el ID de la página de textura (tal y como lo devuelven las funciones texturegroup_* ) para precargar una sola página, o puedes suministrar el nombre de un grupo de texturas (tal y como se define en la función Texture Group). o puedes suministrar el nombre de un grupo de texturas (como se define en el Editor de Grupos de Texturas) para obtener todas las páginas de texturas del grupo.

Sintaxis:

texture_prefetch(tex_id);

Argumento Descripción
tex_id El ID de la página de la textura (un entero) o el ID de un grupo de texturas (una cadena)

Devuelve:

N/A

Ejemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
texture_prefetch(_tex_array[i]);
}

El código anterior precargará todas las páginas de texturas bajo el grupo de texturas "MainMenu".
