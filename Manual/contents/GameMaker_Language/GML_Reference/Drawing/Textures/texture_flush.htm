textura_de_lanzamiento

Esta función puede usarse para "vaciar" una página de textura o un grupo de páginas de textura, es decir, eliminarlas de la VRAM cuando ya no sean necesarias. Se suministra el ID de la página de textura única (tal y como lo devuelven las funciones texturegroup_* ) para vaciar una sola página, o puedes suministrar el nombre de un grupo de texturas (como se define en el Editor de Grupos de Texturas) para vaciar todas las páginas de texturas del grupo.

Sintaxis:

texture_flush(tex_id);

Argumento Descripción
tex_id El ID de la página de la textura (un entero) o el ID de un grupo de texturas (una cadena)

Devuelve:

N/A

Ejemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
texture_flush(_tex_array[i]);
}

El código anterior hará que se vacíen todas las páginas de texturas bajo el grupo de texturas "MainMenu".
