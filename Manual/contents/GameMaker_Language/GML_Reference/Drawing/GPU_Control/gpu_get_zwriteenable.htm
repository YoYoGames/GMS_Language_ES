gpu_get_zwriteenable

Esta función se puede utilizar para recuperar si la escritura en z está habilitada (la función devuelve true) o no (la función devuelve false). El valor por defecto es que la escritura z está habilitada, por lo que la función devolverá true.

Sintaxis:

gpu_get_zwriteenable();

Devuelve:

Booleano

Ejemplo:

si gpu_get_zwriteenable() == false
{
gpu_set_zwriteenable(true);
}

El código anterior comprueba si la escritura en z está activada o no y si está desactivada se vuelve a activar.
