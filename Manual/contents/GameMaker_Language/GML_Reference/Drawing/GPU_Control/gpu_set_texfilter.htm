gpu_set_texfilter

Esta función se puede utilizar para establecer el filtrado de texturas (interpolación lineal) de todas las imágenes dibujadas en la pantalla del juego. Cuando esto está habilitado(true) todas las texturas serán suavizadas cuando se dibujen (esto incluye a los sprites ya que también se consideran texturas), lo que significa que cuando se escalan o se mueven, si no hay una relación de píxeles de 1:1, habrá una "mancha" a través de varios píxeles que puede hacer que las imágenes aparezcan borrosas dependiendo del estilo artístico utilizado. Si esta opción está desactivada(false), las imágenes se dibujarán basándose en el píxel más cercano cuando se escalen o se muevan, lo que puede dar lugar a imágenes "en bloque". El valor por defecto es falso, y esto también se puede cambiar en las Opciones Globales del Juego para las plataformas individuales de destino.

NOTA: Este ajuste anulará cualquier interpolación de la fase de textura establecida para los sombreadores que utilicen la función gpu_set_texfilter_ext().

NOTA: En el objetivo HTML5, esta función sólo funcionará con WebGL activado.

Sintaxis:

gpu_set_texfilter(enable);

Argumento Descripción
activar Activar o desactivar el filtrado de texturas(verdadero / falso)

Devuelve:

N/A

Ejemplo:

si gpu_get_texfilter()
{
gpu_set_texfilter(false);
}
si no
{
gpu_set_texfilter(true);
}

El código anterior comprueba si el filtrado de texturas está activado o desactivado y lo cambia en consecuencia.
