gpu_set_texrepeat_ext

Esta función se puede utilizar para establecer si un "slot" de muestreo repite la textura dada cuando se utiliza Shaders en GameMaker Studio 2. Si se establece como verdadero, se repetirá la textura si las coordenadas uv están fuera del rango 0-1, mientras que un ajuste de falso significará que no se repite. El caso de uso probable para estas funciones es para la repetición de una textura en 3D, pero con el fin de que funcione y no tirar de las imágenes del resto de la página de texturas, el sprite utilizado tendrá que ser marcado como en una "Página de Textura Separada" en el Editor de Sprite.

NOTA: Este ajuste será anulado por el valor establecido al llamar a la función gpu_set_texrepeat().

Sintaxis:

gpu_set_texrepeat_ext(sampler_id, enable);

Argumento Descripción
sampler_id El id del muestreador del shader.
activar Activar o desactivar el filtrado de texturas(verdadero / falso)

Devuelve:

N/A

Ejemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
si no
{
gpu_set_texfilter_ext(s_tex, true);
}

El código anterior comprueba si el filtrado de textura está activado o desactivado para un ID de muestreador específico (almacenado en una variable local) y lo cambia en consecuencia.
