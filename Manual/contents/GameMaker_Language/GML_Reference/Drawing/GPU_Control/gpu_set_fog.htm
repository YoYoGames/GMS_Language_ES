gpu_set_fog

Esta función se puede utilizar para activar o desactivar el dibujo de la niebla. La niebla se puede utilizar en los juegos 3D para hacer que las instancias en la distancia se vean borrosas o incluso desaparezcan, lo que ayuda a crear una atmósfera, así como a enmascarar el hecho de que no se están dibujando instancias que están lejos. Puedes establecer si está activada(true) o desactivada(false), el color que la niebla debe utilizar para la mezcla, así como las distancias de dibujo inicial y final. Tenga en cuenta que el inicio de la niebla indica el radio relativo a la de la cámara donde comienza la niebla, y el final de la niebla indica en qué radio (también relativo) las instancias serán completamente invisibles.

La función puede tomar cuatro argumentos individuales (dados arriba) o un array con la siguiente estructura (el código de ejemplo que sigue muestra este método):

  • 0] = alternancia habilitada (un booleano, verdadero o falso), por defecto falso
  • 1] = color (real), por defecto c_black
  • 2] = distancia inicial (real), por defecto 0
  • 3] = distancia final (real), por defecto 1

Sintaxis:

gpu_set_fog(enable, color, start, end);

Argumento Descripción
activar Activar o desactivar la niebla
color El color de la niebla
iniciar La distancia a partir de la cual se aplica la niebla
fin La distancia para acabar con la niebla

Devuelve:

N/A

Ejemplo:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

El código anterior obtiene la configuración actual de la niebla y luego establece el elemento de color de la matriz a c_red antes de establecer la niebla de nuevo utilizando la matriz modificada.
