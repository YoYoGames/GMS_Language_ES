gpu_set_state

Esta función establecerá el estado actual de la GPU utilizando el mapa DS pasado. El mapa suministrado puede ser creado usando la función gpu_get_state().

Sintaxis:

gpu_set_state(ds_map);

Argumento Descripción
ds_map El estado de la GPU a establecer como un Mapa DS.

Devuelve:

N/A

Ejemplo:

gpu_set_state(gpu_map);

El código anterior establece el estado de la GPU utilizando el mapa suministrado en la variable "gpu_map".
