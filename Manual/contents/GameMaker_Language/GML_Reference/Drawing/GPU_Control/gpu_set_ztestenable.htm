gpu_set_ztestenable

Esta función puede usarse para activar o desactivar la prueba del búfer z, afectando a cómo se dibujarán las cosas (en general, sólo es útil cuando se trabaja con proyectos 3D). Esencialmente, por defecto cuando la prueba z está desactivada y tienes dos objetos dibujando en el mismo espacio, ambos objetos se renderizarán sin tener en cuenta si uno de ellos sobredibujará al otro, resultando en llamadas de dibujo innecesarias. Si activas esta opción, el buffer z se comprueba para ver si un objeto será "visible" y no se dibuja si no lo es. no lo es. Ten en cuenta que este es el comportamiento por defecto, pero puedes cambiarlo modificando el tipo de comparación utilizado para la comprobación del z-buffer (ver la función gpu_set_zfunc().

Por defecto, la comprobación del búfer z está desactivada(false).

Sintaxis:

gpu_set_ztestenable(enable);

Argumento Descripción
activar Habilitar o deshabilitar la prueba del búfer z(verdadero o falso).

Devuelve:

N/A

Ejemplo:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

El código anterior activa la prueba del buffer z para dibujar un sprite de fondo y luego la desactiva para continuar dibujando.
