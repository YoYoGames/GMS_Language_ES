gpu_get_fog

Esta función se puede utilizar para recuperar los ajustes de niebla. La función devuelve un array 1D de 4 elementos con la siguiente información:

  • 0] = alternancia habilitada (un booleano, verdadero o falso), por defecto falso
  • 1] = color (real), por defecto c_black
  • 2] = distancia inicial (real), por defecto 0
  • 3] = distancia final (real), por defecto 1

Tenga en cuenta que puede cambiar estos valores y pasar el array completo a la función gpu_set_fog() como un único argumento.

Sintaxis:

gpu_get_fog();

Devuelve:

Matriz 1D de 4 elementos (ver arriba los elementos de la matriz)

Ejemplo:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

El código anterior obtiene la configuración actual de la niebla y luego establece el elemento de color de la matriz a c_red antes de establecer la niebla de nuevo utilizando la matriz modificada.
