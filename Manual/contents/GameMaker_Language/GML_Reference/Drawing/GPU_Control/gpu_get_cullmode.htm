gpu_get_cullmode

Esta función se puede utilizar para recuperar el modo de culling de la cara posterior. El valor devuelto será una de las siguientes constantes (el valor por defecto es cull_noculling):

Constante Descripción
cull_noculling No se hará ningún sacrificio
cull_clockwise Todos los triángulos en el sentido de las agujas del reloj serán eliminados
cull_counterclockwise Todos los triángulos en sentido contrario a las agujas del reloj serán eliminados

Sintaxis:

gpu_get_cullmode();

Devuelve:

Constante (ver arriba las constantes)

Ejemplo:

si gpu_get_cullmode() != cull_clockwise
{
gpu_set_cullmode(cull_clockwise);
}

El código anterior obtiene el modo actual de cull y si no es cull_clockwise se ajusta a esa constante.
