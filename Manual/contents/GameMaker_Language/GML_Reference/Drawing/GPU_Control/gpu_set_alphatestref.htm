gpu_set_alphatestref

Esta función establecerá el valor de referencia para la prueba alfa (cuando ha sido habilitada usando el botón gpu_set_alphatestenable()). Este es el umbral de "corte" a partir del cual los píxeles con alfa no se dibujarán.

Básicamente, si tienes este valor a 0 y tienes la interpolación lineal activada (ya sea en las Opciones de Juego o usando gpu_set_texfilter()), todavía puede obtener errores visuales en los bordes de los objetos superpuestos objetos superpuestos. Esto se debe a que puede tener áreas casi completamente transparentes que bloquean las cosas dibujadas detrás de ellas, pero si establece este valor de referencia a, por ejemplo, 254 (es decir, uno por debajo del máximo), esto garantiza que sólo el 100% de los píxeles opacos van a ser dibujados. serán dibujados. También puedes desactivar la interpolación de texturas, lo que normalmente te permitiría utilizar los valores por defecto sin problemas y sin necesidad de utilizar esta función, sin embargo, depende mucho de la obra de arte de origen.

NOTA: Esta es una función de dibujo avanzada, así que si no estás seguro de si la necesitas o no, o no sabes exactamente lo que hace, probablemente sea mejor que no la uses, ya que puede corromper seriamente la forma en que se dibuja tu juego.

Sintaxis:

gpu_set_alphatestref(val);

Argumento Descripción
val Establece el valor de referencia para las pruebas alfa de 0 a 255 (el valor por defecto es 0)

Devuelve:

N/A

Ejemplo:

if !gpu_get_alphatestenable()
{
gpu_set_alphatestenable(true);
gpu_set_alphatestref(128);
}

El código anterior comprobará si la prueba de alfa está activada y, si no lo está, activará la prueba de alfa y establecerá el umbral de la prueba en 128 (sólo se dibujarán los píxeles con un alfa superior a 0,5).
