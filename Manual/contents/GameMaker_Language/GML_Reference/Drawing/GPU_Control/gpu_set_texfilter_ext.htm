gpu_set_texfilter_ext

Esta función se puede utilizar para establecer la interpolación lineal para una sola "ranura" del muestreador cuando se utiliza Sombreadores en GameMaker Studio 2. Cuando se activa(true) la textura del muestreador será suavizada y si se desactiva(false) las imágenes se dibujarán basándose en el píxel más cercano. El valor por defecto es el establecido por las Opciones Globales del Juego para su juego, o el establecido usando la función gpu_set_texfilter().

NOTA: Este ajuste será anulado por el valor establecido al llamar a la función gpu_set_texfilter().

NOTA: En el objetivo HTML5, esta función sólo funcionará con WebGL activado.

Sintaxis:

gpu_set_texfilter_ext(sampler_id, enable);

Argumento Descripción
sampler_id El id del muestreador del shader.
activar Activar o desactivar el filtrado de texturas(verdadero / falso)

Devuelve:

N/A

Ejemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
si no
{
gpu_set_texfilter_ext(s_tex, true);
}

El código anterior comprueba si el filtrado de textura está activado o desactivado para un ID de muestreador específico (almacenado en una variable local) y lo cambia en consecuencia.
