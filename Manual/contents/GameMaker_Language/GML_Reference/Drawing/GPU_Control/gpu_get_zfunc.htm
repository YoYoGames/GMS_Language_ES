gpu_get_zfunc

Esta función se puede utilizar para recuperar el modo de comparación z. El valor devuelto será una de las siguientes constantes (el valor por defecto es cmpfunc_lessequal):

  • cmpfunc_never
  • cmpfunc_menos
  • cmpfunc_equal
  • cmpfunc_lessequal
  • cmpfunc_grande
  • cmpfunc_notequal
  • cmpfunc_greaterequal
  • cmpfunc_always

Sintaxis:

gpu_get_zfunc();

Devuelve:

Constante (ver tabla anterior)

Ejemplo:

si gpu_get_zfunc() != cmpfunc_greater
{
gpu_set_zfunc(cmpfunc_greater);
}

El código anterior comprueba si el método de prueba z se establece en cmpfunc_greater y si no es así, se establece en esa constante.
