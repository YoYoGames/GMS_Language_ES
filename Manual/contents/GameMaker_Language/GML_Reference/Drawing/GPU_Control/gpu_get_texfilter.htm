gpu_get_texfilter

Con esta función puedes comprobar si el filtrado de texturas (interpolación lineal) está activado (devuelve true) o no (devuelve false). Para más información sobre el filtrado de texturas, véase la función gpu_set_texfilter().

NOTA: En el objetivo HTML5, esta función sólo funcionará con WebGL activado.

Sintaxis:

gpu_get_texfilter();

Devuelve:

Booleano

Ejemplo:

si gpu_get_texfilter()
{
gpu_set_texfilter(false);
}
si no
{
gpu_set_texfilter(true);
}

El código anterior comprueba si el filtrado de texturas está activado o desactivado y lo cambia en consecuencia.
