gpu_set_cullmode

Esta función se puede utilizar para establecer el modo de eliminación de la cara posterior. Un polígono (primitivo) tiene una cara frontal y otra trasera. Se dice que el lado frontal es el lado en el que los vértices están definidos en orden contrario a las agujas del reloj. Normalmente ambos lados se dibujan, pero si haces una forma cerrada (como una pirámide) entonces esto es un desperdicio de poder de procesamiento porque el lado posterior del triángulo nunca se puede ver. Es en estos casos cuando se puede activar la eliminación de la cara posterior, ya que esto ahorra aproximadamente la mitad del tiempo de dibujo, aunque la la mitad del tiempo de dibujo, aunque significa que tiene la tarea de definir sus polígonos de la manera correcta para evitar problemas.

El valor del modo de sacrificio utilizado debe ser una de las siguientes constantes (el valor por defecto es cull_noculling):

Constante Descripción
cull_noculling No se hará ningún sacrificio
cull_clockwise Todos los triángulos en el sentido de las agujas del reloj serán eliminados
cull_counterclockwise Todos los triángulos en sentido contrario a las agujas del reloj serán eliminados

Sintaxis:

gpu_set_cullmode(cullmode);

Argumento Descripción
modo de cullmode El modo de sacrificio a utilizar (véase la lista anterior)

Devuelve:

N/A

Ejemplo:

si gpu_get_cullmode() != cull_clockwise
{
gpu_set_cullmode(cull_clockwise);
}

El código anterior obtiene el modo actual de cull y si no es cull_clockwise se ajusta a esa constante.
