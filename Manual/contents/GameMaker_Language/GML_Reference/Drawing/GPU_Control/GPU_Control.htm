Control de la GPU

Al dibujar cosas en una habitación, a menudo necesitarás un control más granular sobre cómo se hará exactamente el dibujo. Es posible que desee que no haya mezcla alfa, o puede requerir la eliminación, o puede necesitar activar o desactivar la interpolación de la textura. desactivar la interpolación de texturas. Todo esto es posible - ¡y mucho más! - utilizando las funciones del controlador de la GPU que se enumeran a continuación. Ten en cuenta que la mayoría de estas funciones son de alcance global y una vez establecidas o modificadas afectarán a cómo se dibuja todo a partir de entonces.

Las siguientes funciones pueden utilizarse para obtener los valores de diferentes propiedades de la GPU:

  • gpu_get_blendenable
  • gpu_get_ztestenable
  • gpu_get_zfunc
  • gpu_get_zwriteenable
  • gpu_get_fog
  • gpu_get_cullmode
  • gpu_get_blendmode
  • gpu_get_blendmode_ext
  • gpu_get_blendmode_ext_sepalpha
  • gpu_get_blendmode_src
  • gpu_get_blendmode_dest
  • gpu_get_blendmode_srcalpha
  • gpu_get_blendmode_destalpha
  • gpu_get_colourwriteenable
  • gpu_get_alphatestenable
  • gpu_get_alphatestref
  • gpu_get_texfilter
  • gpu_get_texfilter_ext
  • gpu_get_texrepeat
  • gpu_get_texrepeat_ext

Las siguientes funciones pueden utilizarse para establecer los valores de diferentes propiedades de la GPU:

  • gpu_set_blendenable
  • gpu_set_ztestenable
  • gpu_set_zfunc
  • gpu_set_zwriteenable
  • gpu_set_fog
  • gpu_set_cullmode
  • gpu_set_blendmode
  • gpu_set_blendmode_ext
  • gpu_set_blendmode_ext_sepalpha
  • gpu_set_colourwriteenable
  • gpu_set_alphatestenable
  • gpu_set_alphatestref
  • gpu_set_texfilter
  • gpu_set_texfilter_ext
  • gpu_set_texrepeat
  • gpu_set_texrepeat_ext

Tenga en cuenta que antes de utilizar las funciones del modo de fusión es importante entender cómo funcionan para obtener los resultados correctos al aplicarlas. Para ello hemos creado la siguiente guía:

  • Guía de uso de Blendmodes

Por último, existen las siguientes funciones que pueden utilizarse para manipular la pila de la GPU:

  • gpu_push_state
  • gpu_pop_state
  • gpu_get_state
  • gpu_set_state
