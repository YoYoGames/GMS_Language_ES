gpu_set_colourwriteenable

Con esta función puedes activar o desactivar los canales de color y el canal alfa para todo el dibujo posterior. Por ejemplo, puede utilizar esta función para crear superficies sólo con canal alfa (desactivar la escritura RGB antes de utilizar la superficie), o para crear efectos especiales al dibujar en la pantalla. El valor por defecto para cada uno de los componentes es true, y puede ser suministrado como cuatro argumentos únicos o como un array 1D de 4 elementos que serán verdadero (activado) o falso (desactivado):

  • 0] = Canal rojo activado/desactivado
  • 1] = Canal verde activado/desactivado
  • 2] = Canal azul activado/desactivado
  • 3] = Canal alfa activado/desactivado

Sintaxis:

gpu_set_colorwriteenable(rojo, verde, azul, alfa);

Argumento Descripción
rojo Activar/desactivar el canal rojo
verde Activar/desactivar el canal verde
azul Activar/desactivar el canal azul
alfa Activar/desactivar el canal alfa

Devuelve:

N/A

Ejemplo:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

El código anterior obtiene los valores de escritura del color actual y luego establece el componente alfa en falso.
