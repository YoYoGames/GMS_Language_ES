gpu_get_blendenable

Esta función puede utilizarse para recuperar el estado de la mezcla alfa. Si devuelve true, la mezcla alfa está activada, y si devuelve false, está desactivada. Por defecto está activada, por lo que la función devolverá true

Sintaxis:

gpu_get_blendenable();

Devuelve:

Booleano

Ejemplo:

si gpu_get_blendenable() == false
{
gpu_set_blendenable(true);
}

El código anterior comprueba el estado de la mezcla alfa y si está desactivada se vuelve a activar.
