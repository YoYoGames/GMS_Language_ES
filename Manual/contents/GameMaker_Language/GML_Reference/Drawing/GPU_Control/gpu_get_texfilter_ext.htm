gpu_get_texfilter_ext

Con esta función puedes comprobar si la interpolación de la textura (interpolación lineal) está activada (devuelve true) o no (devuelve false) para una textura de muestreo del shader dada.

NOTA: En el objetivo HTML5, esta función sólo funcionará con WebGL activado.

Sintaxis:

gpu_get_texfilter_ext(sampler_id);

Argumento Descripción
sampler_id El id del muestreador del shader.

Devuelve:

Booleano

Ejemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
si no
{
gpu_set_texfilter_ext(s_tex, true);
}

El código anterior comprueba si el filtrado de textura está activado o desactivado para un ID de muestreador específico (almacenado en una variable local) y lo cambia en consecuencia.
