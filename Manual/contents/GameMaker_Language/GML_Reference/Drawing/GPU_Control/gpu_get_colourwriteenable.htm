gpu_get_colourwriteenable

Esta función se puede utilizar para recuperar los valores actuales de escritura del color. La función devuelve un array de 4 elementos 1D con los siguientes elementos que serán verdaderos (habilitados) o falsos (deshabilitados). Por defecto, todos los colores de color se establece en verdadero:

  • 0] = Canal rojo activado/desactivado
  • 1] = Canal verde activado/desactivado
  • 2] = Canal azul activado/desactivado
  • 3] = Canal alfa activado/desactivado

Sintaxis:

gpu_get_colorwriteenable();

Devuelve:

Matriz 1D de 4 elementos (ver detalles arriba)

Ejemplo:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

El código anterior obtiene los valores de escritura del color actual y luego establece el componente alfa en falso.
