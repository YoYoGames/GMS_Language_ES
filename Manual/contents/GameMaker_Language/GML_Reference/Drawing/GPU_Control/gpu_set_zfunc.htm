gpu_set_zfunc

Esta función puede ser usada para establecer el modo de comparación de la prueba del buffer z (ver gpu_set_ztestenable() para más información). Los valores disponibles para su uso son cualquiera de las siguientes constantes (el valor por defecto es cmpfunc_lessequal):

  • cmpfunc_never
  • cmpfunc_menos
  • cmpfunc_equal
  • cmpfunc_lessequal
  • cmpfunc_grande
  • cmpfunc_notequal
  • cmpfunc_greaterequal
  • cmpfunc_always

Sintaxis:

gpu_set_zfunc(cmp_func);

Argumento Descripción
cmp_func El modo de comparación a utilizar (véase la lista anterior)

Devuelve:

N/A

Ejemplo:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

El código anterior activa la prueba del buffer z y establece su modo de comparación antes de dibujar un sprite de fondo y luego lo desactiva de nuevo para continuar dibujando.
