dibujar_barra_de_salud

Con esta función puede dibujar una barra de color para mostrar un valor constante. Aunque la función utiliza la palabra "barra de salud", puede utilizarla para mostrar cualquier cosa que desee siempre que la cantidad a mostrar sea un valor porcentual entre 0 y 100 (un porcentaje de cualquier valor puede ser calculado con la fórmula (Valor_CURRENTE / Valor_MÁXIMO) * 100), así que, por ejemplo, puedes usar esto para mostrar la potencia, la salud, el maná, el tiempo o cualquier otra cosa que se beneficiaría de una barra de visualización.

Puede configurar varias cosas diferentes que cambian el aspecto visual de la barra y que se ilustran en la imagen siguiente (tenga en cuenta que los colores utilizados aquí son c_verde para el valor máximo y c_rojo para el valor mínimo, pero puede puede utilizar los colores que desee):

NOTA: Si quieres dibujar una forma usando un shader, debes tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vértice, textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vértice y color, por lo que el puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado con esto en mente.

Sintaxis:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Argumento Descripción
x1 La coordenada x de la izquierda de la barra de salud
y1 La coordenada y de la parte superior de la barra de salud
x2 La coordenada x de la derecha de la barra de salud
y2 La coordenada y de la parte inferior de la barra de salud
cantidad La variable que define la salud total (entre 0 y 100)
backcol El color del fondo de la barra
mincol El color de la barra cuando está sin salud (0)
maxcol El color de la barra cuando está a plena salud (100)
dirección Dónde se "ancla" la barra (0 = izquierda, 1 = derecha, 2 = arriba, 3 = abajo)
mostrar de nuevo Si se dibuja el fondo de la barra (true) o no (false). Si es falso, el argumento backcol no se tiene en cuenta.
mostrar frontera Si los elementos de la barra tienen un borde negro de 1px de ancho (true) o no (false).

Devuelve:

N/A

Ejemplo:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

El código anterior utiliza el valor porcentual que se obtiene de las variables "hp" y "hp_max" para dibujar una barra de salud roja/verde estándar.
