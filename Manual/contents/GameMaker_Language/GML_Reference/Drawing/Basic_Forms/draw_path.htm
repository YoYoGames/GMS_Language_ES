dibujar_trayectoria

Con esta función puedes hacer que GameMaker Studio 2 dibuje un camino en la pantalla. El camino se dibujará como una simple línea, y puede ser relativa a la instancia de llamada o en la posición absoluta que se creó en el editor de ruta o a través de código. Esta función es extremadamente útil para depurar rutas dinámicas (por ejemplo, las creadas para instancias con la función mp_grid_path() ).

NOTA: Si quieres dibujar una forma usando un shader, debes tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vértice, textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vértice y color, por lo que el puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado con esto en mente.

Sintaxis:

dibujar_camino(camino, x, y, absoluto);

Argumento Descripción
camino El camino a dibujar
x La coordenada x del lugar donde se dibuja la trayectoria
y La coordenada y del lugar donde se dibuja la trayectoria
absoluto Si la ruta se dibuja en la posición absoluta(true) o en la posición relativa(false)

Devuelve:

N/A

Ejemplo:

if mp_grid_path(grid, path, x, y, obj_Player.x, obj_Player.y, 1)
{
draw_path(path, x, y, false);
}

El código anterior utilizará la función mp_grid_path para generar un camino y almacenarlo en la variable "path". Si el camino se crea con éxito, entonces se dibuja en la pantalla en una posición relativa a la instancia que ejecuta el código.
