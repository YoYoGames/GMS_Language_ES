vertex_create_buffer

Con esta función puedes crear un nuevo búfer de vértices. Se trata de un búfer de crecimiento especial creado por GameMaker Studio 2 que está preformateado para su uso cuando se construyen primitivas (para su uso con shaders, por ejemplo). La función devolverá un "Pointer" (índice) para el búfer que luego debe ser utilizado en todas las llamadas posteriores a la misma.

Cuando se utiliza un búfer de vértices creado con esta función, simplemente se llama a vertex_begin() para empezar a asignar datos de vértices a ella para empezar a definir su primitiva personalizada, que luego se mantendrá en el buffer listo para para ser enviada al shader. El buffer puede ser reutilizado cuando sea necesario (a menos que hayas utilizado la función vertex_freeze() ), con cada llamada de vertex_begin() borrando los datos del datos del búfer listo para aceptar los nuevos datos.

Sintaxis:

vertex_create_buffer();

Devuelve:

Puntero

Ejemplo:

v_buff = vertex_create_buffer();

El código anterior creará un nuevo búfer de vértices y almacenará su manejador en la variable "v_buff".
