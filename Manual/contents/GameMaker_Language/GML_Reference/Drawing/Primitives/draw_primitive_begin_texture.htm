dibujar_primitivo_empezar_textura

Esta función debe ser llamada antes de definir los vértices de una primitiva con textura. Debe dar el tipo de primitiva a utilizar (ver draw_primitive_begin() para más información) y el id de una textura a utilizar, que puede ser un sprite o un activo de imagen de fondo. Este id de activo se puede obtener de las funciones sprite_get_texture()por ejemplo (utilice -1 si no hay textura).

NOTA: Para que una textura se repita debe ser una potencia de dos en tamaño, es decir: 32x32, 128x128, etc...

Sintaxis:

draw_primitive_begin_texture(kind, tex)

Argumento Descripción
amable El tipo de primitiva que vas a dibujar.
tex La textura a utilizar con la primitiva.

Devuelve:

N/A

Ejemplo:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertexture(0, 0, 0, 0);
draw_vertexture(640, 0, 1, 0);
draw_vertexture(0, 480, 0, 1);
draw_vertexture(640, 480, 1, 1);
draw_primitive_end();

El código anterior dibujará una tira de triángulos de 4 vértices (haciendo un rectángulo) texturizados con la textura mantenida en la variable "tex", y toda la textura se utilizará para cubrir la primitiva completada.
