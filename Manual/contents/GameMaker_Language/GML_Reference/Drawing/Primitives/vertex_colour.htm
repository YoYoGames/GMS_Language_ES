color_vértice

Si su formato de vértice definido toma un valor de color, puede utilizar esta función para añadir esos datos al vértice que se está definiendo para la primitiva actual. La función necesita un buffer para almacenar los datos y tomará una constante de color o un valor hexadecimal (utilizando el formato estándar de GameMaker Studio 2 de BGR, por ejemplo: $FF0000 para el azul), así como un valor alfa de 0 (transparente) a 1 (totalmente opaco).

Sintaxis:

vertex_colour(buffer, color, alpha);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
color El color de este vértice (puede ser una constante o un valor hexadecimal).
alfa El valor alfa del vértice (de 0 a 1).

Devuelve:

N/A

Ejemplo:

vertex_colour(b, c_white, 1);

El código anterior establecerá el vértice que se está definiendo como blanco con un valor alfa de 1.
