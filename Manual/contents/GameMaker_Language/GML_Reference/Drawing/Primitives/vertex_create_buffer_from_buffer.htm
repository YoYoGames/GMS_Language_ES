vertex_create_buffer_from_buffer

Al igual que la función vertex_create_buffer()esta función creará un nuevo búfer de vértices, sólo que ahora los datos de vértices que almacena se copian del búfer normal que se especifica como fuente. El búfer creado es un búfer de crecimiento especial que está preformateado con el formato de vértices para construir primitivas para su uso con (por ejemplo) shaders. Esta función requiere que se proporcione el puntero a un búfer regular previamente creado, y un formato de vértice que debe aplicarse a los datos copiados.

Sintaxis:

vertex_create_buffer_from_buffer(buffer, format);

Argumento Descripción
buffer El buffer para crear el buffer de vértices.
formato El formato de vértice primitivo a utilizar.

Devuelve:

Puntero

Ejemplo:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

El código anterior creará un nuevo formato de vértices y luego creará un nuevo búfer de vértices a partir de un búfer regular previamente creado, aplicándole el formato de vértices personalizado.
