vertex_get_number

Con esta función se puede averiguar el número de vértices individuales definidos en un búfer de vértices determinado.

Sintaxis:

vertex_get_number(buffer);

Argumento Descripción
buffer El buffer de vértices a comprobar.

Devuelve:

Real

Ejemplo:

v_num = vertex_get_number(v_buffer);

El código anterior almacenará en una variable el número de vértices almacenados en el buffer de vértices dado.
