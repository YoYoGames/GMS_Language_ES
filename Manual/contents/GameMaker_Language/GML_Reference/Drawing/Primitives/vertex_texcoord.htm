vertex_texcoord

Esta función establecerá las coordenadas de la textura a utilizar para el vértice actualmente definido para la primitiva personalizada. Se suministra el búfer para escribir los datos, así como la posición UV dentro de la textura a utilizar.

Sintaxis:

vertex_texcoord(buffer, u, v);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
u La primera coordenada de la textura a utilizar (0 - 1).
v La segunda coordenada de la textura a utilizar (0 - 1).

Devuelve:

N/A

Ejemplo:

vertex_texcoord(buff, 0, 0);

El código anterior establecerá los valores UV del vértice que se está definiendo.
