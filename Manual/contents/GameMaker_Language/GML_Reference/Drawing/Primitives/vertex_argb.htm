vertex_argb

Esta función establecerá los valores ARGB para el vértice que se está definiendo para la primitiva personalizada. Usted proporciona el búfer para escribir los datos, así como los valores rojo, verde, azul y alfa que desea utilizar como un único entero sin signo de 32 bits - la muestra de alfa en los 8 bits más altos, seguida por la muestra de rojo, la muestra de verde y finalmente la muestra de azul en los 8 bits más bajos. Puede utilizar la notación hexadecimal ($AARRGGBB) un número real o utilizar cualquiera de las funciones make_colour_*() para definir el valor del color.

Sintaxis:

vertex_argb(buffer, argb);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
a El valor alfa del color (0 - 255).
r La parte del componente rojo de un color (0 - 255).
g La parte del componente verde de un color (0 - 255).
b La parte del componente azul de un color (0 - 255).

Devuelve:

N/A

Ejemplo:

vertex_argb(buff, $FFFFFFFF);

El código anterior establecerá los valores ARGB del vértice que se está definiendo como blanco.
