vertex_create_buffer_ext

Al igual que la función vertex_create_buffer()esta función creará un nuevo búfer de vértices. Este es un búfer de crecimiento especial creado por GameMaker Studio 2 que está preformateado para su uso cuando construir primitivas para su uso con shaders. Puedes especificar un tamaño inicial para el buffer (en bytes) y devolverá un "handle" (índice) para el buffer que deberá ser utilizado en todas las llamadas posteriores al buffer.

Sintaxis:

vertex_create_buffer_ext(tamaño);

Argumento Descripción
tamaño El tamaño inicial del buffer (en bytes).

Devuelve:

Puntero

Ejemplo:

v_buff = vertex_create_buffer_ext(1024 * 1024);

El código anterior creará un nuevo búfer de vértices, inicialmente de 1MB de tamaño, y almacenará su manejador en la variable "v_buff".
