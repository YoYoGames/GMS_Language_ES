vertex_format_begin

Antes de que puedas definir tu nuevo formato de vértices debes decirle a GameMaker Studio 2 que lo estás haciendo usando esta función. Debes llamarla primero, luego definir los valores del formato usando las funciones apropiadas, y finalmente llamar a vertex_format_end() para terminar la definición y devolver el nuevo formato "handle".

Sintaxis:

vertex_format_begin();

Devuelve:

N/A

Ejemplo:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

El código anterior creará un nuevo formato de vértice con sólo los valores de color y posición y luego almacenará el id del formato en la variable "mi_formato".
