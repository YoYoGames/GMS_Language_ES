dibujar_vértice

Esta función define la posición de un vértice para una primitiva. El aspecto final de la primitiva dependerá del tipo de primitiva elegida para dibujar y del orden con el que se añadan los vértices a la misma. Véase dibujar_primitivo_inicio() para más información. Para terminar y dibujar la primitiva debes llamar a dibujar_primitiva_fin().

Sintaxis:

dibujar_vértice(x, y)

Argumento Descripción
x La coordenada x del vértice.
y La coordenada y del vértice.

Devuelve:

N/A

Ejemplo:

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

El código anterior dibujará una primitiva triangular simple.
