posición_vértice_3d

Esta función añadirá datos de posición 3D al vértice que se está definiendo para la primitiva personalizada. Usted proporciona el búfer en el que escribir los datos, así como la posición x, y y z para el dibujo.

Sintaxis:

vertex_position_3d(buffer, x, y, z);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
x La posición x.
y La posición y.
z La posición z.

Devuelve:

N/A

Ejemplo:

vertex_position_3d(buff, x - 100, y - 125, 0);

El código anterior establecerá la posición del vértice que se está definiendo.
