Primitivas y formatos de vértices

Una primitiva es un tipo de dibujo realizado mediante puntos para definir lo que se muestra en la pantalla. Pueden estar formadas por puntos simples, líneas o triángulos y pueden ser coloreadas, tener mezcla alfa e incluso recibir una textura, lo que significa que pueden ser utilizadas para crear algunos efectos muy emocionantes en un juego 2D. Por ejemplo, puedes crear un rectángulo usando múltiples primitivas de triángulo, luego texturizarlo con un sprite de bandera y en el evento de dibujo de una instancia hacer que esa bandera "ondee" en el viento y distorsionar el sprite que has utilizado simplemente modificando los puntos que forman la primitiva.

ATENCIÓN! Estas funciones no funcionan con el módulo HTML5 a menos que hayas habilitado OpenGL en las Opciones de Juego.

IMPORTANTE! Las primitivas que construyas deben seguir el formato que hayas establecido usando las funciones de Formato de Vértices, así que si has definido un formato de vértices con sólo datos posicionales, no tiene sentido construir tu primitiva con datos de color. Debes tener en cuenta que el orden en el que añades las propiedades a la primitiva que estás construyendo está definido por el orden en el que añadiste estas propiedades al crear el formato de vértice, así que si has definido el formato de vector con el orden posición, color y coordenada de textura, debes añadir estas propiedades a la primitiva que se está construyendo en el mismo orden, de lo contrario obtendrás errores.
