posición_vértice

Esta función añadirá datos de posición 2D al vértice que se está definiendo actualmente para la primitiva personalizada. El usuario suministra el búfer en el que se escribirán los datos, así como la posición x e y para el dibujo.

Sintaxis:

vertex_position(buffer, x, y);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
x La posición x.
y La posición y.

Devuelve:

N/A

Ejemplo:

vertex_position(buff, x - 100, y - 125);

El código anterior establecerá la posición del vértice que se está definiendo.
