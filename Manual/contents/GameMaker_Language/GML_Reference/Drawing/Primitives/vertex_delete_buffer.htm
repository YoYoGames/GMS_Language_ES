vertex_delete_buffer

Esta función puede utilizarse para eliminar un búfer de vértices previamente creado (véase vertex_create_buffer()) de la memoria del sistema. Esto también eliminará todos los datos de vértices para cualquier primitiva personalizada que contenga.

Sintaxis:

vertex_delete_buffer(buffer);

Argumento Descripción
buffer El búfer que se va a eliminar.

Devuelve:

N/A

Ejemplo:

vertex_delete_buffer(v_buff);

El código anterior borrará de la memoria el buffer indexado en la variable "v_buff".
