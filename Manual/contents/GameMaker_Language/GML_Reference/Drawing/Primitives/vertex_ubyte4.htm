vértice_ubyte4

Esta función añadirá cuatro valores de bytes sin signo (0 - 255) a los datos del vértice. El vértice debe haber sido formateado correctamente para aceptar esto usando la función vertex_format_add_custom() función.

Sintaxis:

vertex_ubyte4(buffer, byte, byte, byte, byte);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
byte El primer valor de entrada.
byte El segundo valor de entrada.
byte El tercer valor de entrada.
byte El cuarto valor de entrada.

Devuelve:

N/A

Ejemplo:

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

El código anterior añadirá cuatro valores a los datos de los vértices que se están definiendo.
