vertex_format_end

Esta función debe ser llamada después de definir cualquier nuevo formato de vértice. Devuelve el "handle" (índice) del nuevo formato que debe ser utilizado en todas las funciones de vértice posteriores que hagan referencia a este nuevo formato.

Sintaxis:

vertex_format_end();

Devuelve:

Real

Ejemplo:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

El código anterior creará un nuevo formato de vértice con sólo los valores de color y posición y luego almacenará el id del formato en la variable "mi_formato".
