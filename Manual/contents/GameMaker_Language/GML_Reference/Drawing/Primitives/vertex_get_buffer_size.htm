vertex_get_buffer_size

Con esta función puedes obtener el tamaño del buffer de vértices dado en bytes.

Sintaxis:

vertex_get_buffer_size(buffer);

Argumento Descripción
buffer El búfer para obtener el tamaño de.

Devuelve:

Real

Ejemplo:

bufferBytes = vertex_get_buffer_size(buff);

El código anterior obtendrá el número de bytes utilizados por el buffer de vértices dado y almacenará el valor en la variable "bufferBytes".
