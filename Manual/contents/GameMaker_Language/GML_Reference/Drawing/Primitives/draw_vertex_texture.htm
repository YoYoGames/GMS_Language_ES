dibujar_textura_de_vértice

Esta función define la posición de un vértice texturizado para una primitiva. El aspecto final de la primitiva dependerá del tipo de primitiva elegida para dibujar (Ver dibujar_primitivo_inicio() para más información), el orden con el que añadas los vértices a la misma y la posición del punto de inicio que establezcas para la textura. Para finalizar y dibujar la primitiva debes llamar a dibujar_primitiva_fin().

También puedes elegir dibujar sólo una parte de la textura sobre la superficie de la primitiva estableciendo la posición xtex/ytex apropiada para muestrear (estas coordenadas también se conocen como los puntos UV ). Este punto está definido por un valor entre 0 y 1, siendo la esquina superior izquierda de la imagen a utilizar el punto (0,0) y la esquina inferior derecha el punto (1,1). De esta manera se puede texturizar una primitiva con sólo una parte del activo de la imagen de origen.

Cuando se dibuja una primitiva texturizada, la textura se mezclará con el color establecido por draw_set_colour. Esto significa que para dibujar la textura sin mezcla, ésta debe establecerse en c_white, como se muestra en el siguiente ejemplo.

NOTA: Para que una textura se repita debe ser una potencia de dos en tamaño, es decir: 32x32, 128x128, etc...

Sintaxis:

draw_vertex_texture(x, y, xtex, ytex)

Argumento Descripción
x La coordenada x del vértice.
y La coordenada y del vértice.
xtex La anchura de la muestra de textura a utilizar.
ytex La altura de la muestra de textura a utilizar.

Devuelve:

N/A

Ejemplo:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

El código anterior dibujará una tira de triángulos de 4 vértices (haciendo un rectángulo) texturizados con la textura mantenida en la variable "tex", y toda la textura se utilizará para cubrir la primitiva completada.
