vertex_submit

Puedes utilizar esta función para enviar el contenido de un búfer de vértices a la línea de gráficos para su uso con un sombreador. Se suministra el índice del buffer a utilizar, el tipo de primitiva base a utilizar (ver las constantes más abajo) y la textura que se va a utilizar. El tipo de primitiva base tipo de primitiva base sólo se utiliza para asignar el orden en el que se dibujan y conectan los vértices almacenados en el búfer, pero los datos reales utilizados para cada uno de los vértices serán los que definiste al crear el búfer de vértices.

NOTA: Esta función sólo puede utilizarse en el evento de sorteo.

Para ver un ejemplo visual de las diferentes primitivas base disponibles, vea la imagen de abajo:

Constante Descripción
pr_pointlist Una primitiva que consiste en una lista de puntos.
pr_linelist Una primitiva formada por una línea individual en una lista.
pr_linestrip Una primitiva formada por una franja consecutiva de líneas.
pr_trianglelist Una primitiva formada por triángulos individuales en una lista.
pr_trianglestrip Una primitiva formada por una tira consecutiva de triángulos.

Sintaxis:

vertex_submit(buffer, primitive, texture);

Argumento Descripción
buffer El buffer a utilizar.
primitivo El tipo base primitivo.
textura La textura a utilizar (o -1 para ninguna textura).

Devuelve:

N/A

Ejemplo:

shader_set(shader_prim);
vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index));
shader_reset();

El código anterior enviará el buffer de vértices indexado en la variable "buff" para ser dibujado con el shader " shader_prim ",utilizando una lista de triángulos como primitiva base y la textura del sprite para la instancia que ejecuta el código.
