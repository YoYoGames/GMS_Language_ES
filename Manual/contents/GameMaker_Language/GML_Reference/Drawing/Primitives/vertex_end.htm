vertex_end

Con esta función se termina la construcción de la primitiva personalizada. Una vez que llames a este comando la primitiva puede ser usada en el vertex_submit() función para su uso en un shader o puedes congelar el buffer (haciendo que el búfer de vértices utilizado de sólo lectura y mucho más rápido) utilizando vertex_freeze().

Sintaxis:

vertex_end(buffer);

Argumento Descripción
buffer El búfer en el que se termina de escribir.

Devuelve:

N/A

Ejemplo:

vertex_end(buff);

El código anterior finalizará la definición de una primitiva personalizada.
