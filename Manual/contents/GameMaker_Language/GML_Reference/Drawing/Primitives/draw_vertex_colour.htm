dibujar_vértice_color

Esta función define la posición de un vértice para una primitiva, con su propio color y ajuste alfa. El aspecto final de la primitiva dependerá del tipo de primitiva elegida para dibujar y del orden con el que se añadan los vértices a la misma (ver dibujar_primitivo_inicio() para más información) y los vértices con diferentes colores y alfas se mezclarán suavemente de uno a otro. Para terminar y dibujar la primitiva debes llamar a dibujar_primitiva_fin().

Sintaxis:

dibujar_vértice_color(x, y, col, alpha)

Argumento Descripción
x La coordenada x del vértice.
y La coordenada y del vértice.
col El color con el que se dibujará este vértice.
alfa El alfa para dibujar este vértice con (0-1).

Devuelve:

N/A

Ejemplo:

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

El código anterior dibujará un triángulo semitransparente con cada vértice coloreado de un color diferente.
