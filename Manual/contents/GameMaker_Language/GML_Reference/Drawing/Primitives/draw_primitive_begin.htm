draw_primitive_begin

Esta función debe ser llamada antes de poder definir cualquier primitiva. Hay 6 tipos de primitivas que se pueden definir con las siguientes constantes:

  • pr_pointlist
  • pr_linelist
  • pr_linestrip
  • pr_trianglelist
  • pr_trianglestrip
  • pr_trianglefan

La siguiente imagen ilustra básicamente el aspecto que deben tener y también el orden en el que debes definir los vértices:

Tenga en cuenta que en algunas plataformas(Windows, UWP, XBox) el tipo pr_trianglefan no es soportado nativamente por lo que GameMaker Studio 2 realiza una conversión cuando se compila el juego para que funcione. Esto significa que en esas plataformas el tipo pr_trianglefan será mucho más lento de usar que los otros. También hay que tener en cuenta que para utilizar esta en HTML5, debes habilitar WebGL en las Opciones de Juego.

Sintaxis:

draw_primitive_begin(kind)

Argumento Descripción
amable El tipo de primitiva que vas a dibujar.

Devuelve:

N/A

Ejemplo:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
draw_vertex(_xx + lengthdir_x(_radio, 270 * i / _pasos), _yy + lengthdir_y(_radio, 270 * i / _pasos));
}
draw_primitive_end();

El código anterior dibujará tres cuartos de un círculo hecho con primitivas.
