vertex_format_delete

Esta función debe llamarse siempre que se termine de utilizar cualquier formato vectorial creado. Usted proporciona el valor del ID del formato (tal y como lo devuelve la función vector_format_end()), y esta función liberará la memoria memoria asociada a él. Tenga en cuenta que si intenta utilizar este formato de nuevo después de llamar a esta función, obtendrá un error.

Sintaxis:

vertex_format_delete(formatID);

Devuelve:

N/A

Ejemplo:

vertex_format_delete(myFormat);

El código anterior eliminará de la memoria el formato de vértice creado en la variable "myFormat".
