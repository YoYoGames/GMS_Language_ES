vertex_freeze

Esta función puede utilizarse para "congelar" un búfer de vértices. Este búfer pasa a ser de sólo lectura, lo que significa que si se necesita cambiar, habría que borrar todo el búfer y volver a crearlo. Un búfer congelado puede ser enviado al shader más rápido que un búfer normal y dinámico, y se recomienda para aquellos efectos que requieran una primitiva personalizada que no cambie durante un nivel o el juego.

Sintaxis:

vertex_freeze(buffer);

Argumento Descripción
buffer El buffer para congelar.

Devuelve:

N/A

Ejemplo:

vertex_freeze(buff);

El código anterior congelará el búfer de vértices dado.
