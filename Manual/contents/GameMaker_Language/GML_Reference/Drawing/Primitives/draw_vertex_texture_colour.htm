dibujar_textura_color

Esta función define la posición de un vértice texturizado para una primitiva. El aspecto final de la primitiva dependerá del tipo de primitiva elegida para dibujar (Ver dibujar_primitivo_inicio() para más información), el orden con el que añadas los vértices a la misma, la posición de los puntos de inicio y final que des para la muestra de la textura y los valores de color y alfa que hayas establecido. Para mantener la apariencia de la textura cambiando sólo el alfa, un valor de -1 (o c_white) para el argumento del color. Para finalizar y dibujar la primitiva debes llamar a dibujar_primitiva_fin().

NOTA: Para que una textura se repita debe ser una potencia de dos en tamaño, es decir: 32x32, 128x128, etc...

Sintaxis:

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

Argumento Descripción
x La coordenada x del vértice.
y La coordenada y del vértice.
xtex La coordenada x dentro de la textura.
ytex La coordenada y dentro de la textura.
col El color a mezclar con la textura en este vértice (-1 o c_white para no mezclar).
alfa El alfa para dibujar este vértice con (0-1).

Devuelve:

N/A

Ejemplo:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertexture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertexture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertexture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertexture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end();

El código anterior dibujará una tira de triángulos de 4 vértices (formando un rectángulo) texturizados con la textura mantenida en la variable "tex", y toda la textura se utilizará para cubrir la primitiva completada, y se mezclará con cuatro colores diferentes.
