vertex_float4

Esta función añadirá cuatro valores de punto flotante a los datos del vértice. El vértice debe haber sido formateado correctamente para aceptar esto usando la función vertex_format_add_custom() función.

Sintaxis:

vertex_float4(buffer, float, float, float);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
float El primer valor de entrada.
float El segundo valor de entrada.
float El tercer valor de entrada.
float El cuarto valor de entrada.

Devuelve:

N/A

Ejemplo:

vertex_float4(buff, 0.05, 0.01, room_width / x, room_height / y);

El código anterior añadirá cuatro valores de punto flotante a los datos de los vértices que se están definiendo.
