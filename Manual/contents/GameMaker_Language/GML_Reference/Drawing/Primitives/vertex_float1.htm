vertex_float1

Esta función añadirá un valor de punto flotante a los datos del vértice. El vértice debe haber sido formateado correctamente para aceptar esto usando la función vertex_format_add_custom() función.

Sintaxis:

vertex_float1(buffer, float);

Argumento Descripción
buffer El búfer en el que se escribirá la información.
float El valor de entrada.

Devuelve:

N/A

Ejemplo:

vertex_float1(buff, 0.05);

El código anterior añadirá un valor de punto flotante a los datos de los vértices que se están definiendo.
