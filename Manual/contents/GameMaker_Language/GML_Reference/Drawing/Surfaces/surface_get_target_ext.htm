superficie_obtener_objetivo_ext

Esta función recuperará el ID de la superficie asignada a uno de los 4 objetivos de renderización disponibles para las superficies. Se suministra el índice del objetivo de renderizado a comprobar, y la función devolverá -1 si no hay ninguna superficie asignada, o un valor entero >= 0, que representa el ID de la superficie asignada (devuelto por la función surface_create()).

Sintaxis:

surface_get_target_ext(index);

Argumento Descripción
índice El índice del objetivo de renderizado a comprobar (de 0 a 3).

Devuelve:

Entero (-1 para ningún objetivo de superficie, o un valor de ID de superficie)

Ejemplo:

if surface_get_target_ext(0) == -1
{
surface_set_target_ext(0, global.Surf);
}

El código anterior primero comprobará si el objetivo de renderizado del sombreador 0 ha sido establecido a una superficie, y si no, entonces se le asigna una.
