copia_de_superficie

Esta función simplemente toma la imagen de una superficie y la copia en otra en la posición local especificada dentro de esa superficie (donde la posición (0,0) es la esquina superior izquierda de la superficie de destino). Si la superficie de destino ya tiene información tiene información, ésta será sobrescrita por la copia, y la función no cambia la superficie de origen de ninguna manera.

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de texturas. Debe comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarla directamente.

Sintaxis:

copia_de_superficie(destino, x, y, origen);

Argumento Descripción
destino El ID de la superficie a la que se va a copiar la otra superficie.
x La posición x a la que copiar.
y La posición y a la que copiar.
fuente El ID de la superficie a copiar.

Devuelve:

N/A

Ejemplo:

si vista_actual == 0
{
surface_copy(surf, 0, 0, temp_surf);
}
si no
{
draw_surface(surf, 0, 0);
}

El código anterior comprobará la vista actual que se está dibujando y si es la vista[0] copiará la superficie indexada en la variable "temp_surf" en la superficie indexada en la variable "surf". Si la vista actual es otra que la vista[0] la superficie "surf" se dibuja en la pantalla.
