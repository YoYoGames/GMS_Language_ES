superficie_de_guardado

Esta función guardará una superficie en el disco utilizando el nombre de archivo dado. La superficie debe guardarse como un archivo con formato *.png.

Sintaxis:

superficie_guardar(superficie_id, fname);

Argumento Descripción
ID de superficie El ID de la superficie a establecer como objetivo de dibujo.
fname El nombre del archivo de imagen guardado.

Devuelve:

N/A

Ejemplo:

si keyboard_check_pressed(ord("S"))
{
surface_save(surf, "test.png");
}

El código anterior comprobará si el usuario pulsa la tecla "S" en el teclado y si lo hace guardará en el disco la superficie indexada en la variable "surf".
