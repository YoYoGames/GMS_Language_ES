superficie_reiniciar_objetivo

Con esta función se restablece todo el dibujo posterior de la superficie de destino a la pantalla. Tenga en cuenta que para empezar a dibujar en una superficie debe haber llamado primero a la función surface_set_target() y luego esta otra después de haber terminado, para cada superficie objetivo que hayas establecido o de lo contrario no se dibujará nada en la pantalla ya que todo el dibujo posterior (incluso en otras instancias) se hará en la superficie. También debe tener en cuenta que no se verá nada si la propia superficie no se dibuja en la pantalla en el evento de dibujo de una instancia.

NOTA: Si no ha establecido previamente un objetivo de renderizado con la función surface_set_target()usar esta función terminará silenciosamente (es decir, sin ningún mensaje de error) terminará toda la ejecución de código para el evento.

Sintaxis:

surface_reset_target();

Devuelve:

N/A

Ejemplo:

si view_current = 0
{
surface_set_target(surf);
con (obj_Effect)
{
draw_self();
}
surface_reset_target();
}
else
{
draw_surface(surf, 0, 0);
}

El código anterior comprobará qué vista se está dibujando actualmente, y si es la vista[0] establece el objetivo de dibujo a una superficie y dibuja todas las instancias del objeto "obj_Effect" antes de restablecer el objetivo de dibujo de nuevo. Si la vista no es view[0] la superficie se dibuja en la pantalla.
