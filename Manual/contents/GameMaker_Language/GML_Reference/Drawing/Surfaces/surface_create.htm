crear_superficie

Esta función se utiliza para crear una superficie y devolverá el índice de la superficie que debe ser almacenado en una variable para futuras llamadas a la función. Cuando la superficie se crea por primera vez, puede contener "ruido" ya que básicamente es sólo un área de de memoria que se pone a un lado para el propósito (y esa memoria todavía puede contener información), por lo que es posible que desee limpiar la superficie antes de usarla con una función como dibujar_claro_alfa().

Se recomienda encarecidamente que todas las superficies se creen con un tamaño que sea una potencia de 2, es decir: 16, 128, 512 o 1024 píxeles de tamaño, por ejemplo. Esto no es exactamente necesario en ciertas plataformas (como Windows y MacOS), pero sin duda aumentará la compatibilidad en esos objetivos, mientras que para HTML5 y los dispositivos es esencial y muy es importante que usted recuerde esto o puede tener problemas más adelante.

Sintaxis:

crear_superficie(w, h);

Argumento Descripción
w La anchura de la superficie a crear.
h La altura de la superficie a crear.

Devuelve:

Real

Ejemplo:

if !surface_exists(surf)
{
surf = surface_create(1024, 1024);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
surface_reset_target();
view_surface_id[0] = surf;
}

El código anterior comprueba si existe una superficie y, si no existe, crea una superficie de 1024 píxeles de ancho y 1024 píxeles de alto, asignando el índice a la variable "surf". El objetivo de dibujo se establece entonces en la nueva superficie, que se borra y se hace transparente antes de que el objetivo de dibujo se restablezca en la pantalla. Por último, la superficie se asigna a una vista.
