dibujar_superficie_estirada_ext

Esta función hace exactamente lo mismo que la función dibujar_superficie_extendida() con la capacidad añadida de establecer la mezcla de colores y el valor alfa para la superficie cuando se dibuja (similar a la función dibujar_superficie_ext()).

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de la textura. Debería comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarla directamente.

Sintaxis:

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

Argumento Descripción
id El valor de ID único de la superficie a dibujar.
x La posición x de donde dibujar la superficie.
y La posición y de dónde dibujar la superficie.
w La anchura a la que se debe dibujar la superficie.
h La altura a la que se dibujará la superficie.
color El color con el que colorear la superficie.
alfa El alfa con el que se mezcla la superficie.

Devuelve:

N/A

Ejemplo:

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

Esto dibujará la superficie dada con su esquina izquierda en la posición x/y de las instancias y se estirará para ocupar un área de 200x200 píxeles sin mezcla, pero con transparencia parcial.
