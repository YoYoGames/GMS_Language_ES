superficie_crear_ext

Esta función le permite adjuntar una superficie a un elemento de lienzo que ya existe en su página web, lo que significa que puede dividir eficazmente partes de su juego para ser dibujadas en varios lugares diferentes dentro de la página. Para ello, debes haber definido correctamente el elemento canvas dentro de la página *.html de tu juego utilizando los tamaños y nombres correctos que corresponden a las superficies que deseas crear. Por lo tanto, usted tendría su lienzo "principal", y luego su superficie secundaria que se asignarán mediante esta función a las superficies correctas. El diseño de la página puede ser complicado, especialmente si desea que todos los elementos se alineen correctamente, pero una vez que la ardua tarea de crear el diseño se ha completado, puede entonces añadir este archivo html como el archivo de página por defecto para el juego utilizando la pestaña HTML5 de las Opciones de Juego. Lo siguiente que debes hacer es configurar la sala y las vistas, ya que cada superficie deberá estar asociada a una vista específica para "capturar" las imágenes del juego (ver la variable view view_surface_id[0...7]). Por último, se utilizaría esta función para crear las superficies, con el mismo nombre que el utilizado para los elementos del lienzo y el tamaño correspondiente a ese mismo lienzo. utilizado para los elementos del lienzo y el tamaño correspondiente al tamaño de ese mismo lienzo. La función devolverá el índice de la superficie que deberá ser almacenado en una variable para futuras llamadas a la función. Cuando la superficie se crea por primera vez, puede contener "ruido", ya que básicamente es sólo un área de memoria que se pone a un lado para el propósito (y esa memoria todavía puede contener información), por lo que es posible que desee borrar la superficie antes de usarla con una función como dibujar_claro_alfa().

NOTA: Esta función sólo está disponible para su uso con el módulo HTML5.

Sintaxis:

surface_create_ext(nombre, w, h);

Argumento Descripción
nombre El nombre del elemento del lienzo al que se vincula la superficie.
w La anchura de la superficie a crear.
h La altura de la superficie a crear.

Devuelve:

Real

Ejemplo:

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

El código anterior crea dos superficies de diferentes tamaños, asignando cada una a un elemento del lienzo diferente, y luego esas superficies se asignan a dos vistas para que se capture la parte correcta de la habitación.
