superficie_obtener_textura

Esta función devuelve un puntero especial para la textura de la superficie. Este valor puede ser utilizado en otras funciones de dibujo, particularmente en las funciones generales de 3D y algunas de las funciones primitivas de 2D.

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de texturas. Debe comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarla directamente.

Sintaxis:

surface_get_texture(surface_id);

Argumento Descripción
ID de superficie El ID de la superficie para obtener el ancho.

Devuelve:

Puntero

Ejemplo:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertexture(0, 480, 0, 0);
draw_vertexture(640, 480, 1, 0);
draw_vertexture(640, 480, 1, 1);
draw_vertexture(0, 480, 0, 1);
draw_primitive_end();

El código anterior dibujará una tira de triángulos de 4 vértices texturizados con la textura mantenida en la variable "tex", que a su vez se toma de una superficie creada previamente.
