superficie_de_aplicación_está_habilitada

Esta función devolverá true si se está utilizando la superficie de la aplicación para dibujar, o false si se está utilizando el buffer de pantalla.

Sintaxis:

application_surface_is_enabled();

Devuelve:

Booleano

Ejemplo:

if keyboard_check_pressed(vk_space)
{
si application_surface_is_enabled()
{
application_surface_enable(false);
}
else
{
application_surface_enable(true);
}
}

El código anterior comprueba la pulsación de una tecla y activa o desactiva la superficie de la aplicación en función de su estado (como en un menú de opciones).
