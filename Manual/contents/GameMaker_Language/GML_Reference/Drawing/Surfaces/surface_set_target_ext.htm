surface_set_target_ext

Esta función se utiliza con las funciones de sombreado y establece el MRT (0 - 3) para los sombreadores de nivel nativo (DX9, DX11, OpenGL).

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de la textura. Debería comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarla directamente.

Sintaxis:

surface_set_target_ext(index, surface_id);

Argumento Descripción
índice El índice de destino de renderizado que se va a utilizar (de 0 a 3).
ID de superficie El ID de la superficie a utilizar.

Devuelve:

Booleano

Ejemplo:

surface_set_target_ext(0, surf);

El código anterior establecerá el objetivo de renderizado 0 al ID de la superficie indexada en la variable "surf".
