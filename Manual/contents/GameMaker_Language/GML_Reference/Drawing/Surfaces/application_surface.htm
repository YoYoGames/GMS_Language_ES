superficie_de_aplicación

Esta variable incorporada de ámbito global puede utilizarse para acceder a la superficie de la aplicación utilizando cualquiera de las funciones de superficie. Esta superficie está permanentemente disponible y es donde se realiza la mayor parte del dibujo en GameMaker Studio 2.

Sintaxis:

superficie_de_aplicación;

Devuelve:

Puntero

Ejemplo:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

El código anterior redimensionará la superficie de la aplicación para que tenga una relación de 1:1 con la capa de la GUI.
