superficie_obtener_ancho

Esta función simplemente devuelve la anchura, en píxeles, de la superficie indexada. Hay que tener en cuenta que si se llama a esta función para comprobar la superficie de la aplicación inmediatamente después de haber cambiado su tamaño usando redimensionar_superficie() no devolverá el nuevo valor ya que el cambio necesita uno o dos pasos para ser procesado completamente. Después de esperar un paso debería devolver el nuevo tamaño correctamente.

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de la textura. Debería comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarla directamente.

Sintaxis:

surface_get_width(surface_id);

Argumento Descripción
ID de superficie El ID de la superficie para obtener el ancho.

Devuelve:

Real

Ejemplo:

sw = surface_get_width(surf);

El código anterior almacenará el ancho de la superficie indexada en la variable "surf" en la variable "sw".
