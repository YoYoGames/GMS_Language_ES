dibujar_superficie_ext

Esta función dibujará la superficie dada como en la función dibujar_superficie() pero con opciones adicionales para cambiar la escala, la mezcla, la rotación y el alfa de la superficie dibujada. El cambio de estos valores no no modifica el recurso de ninguna manera (sólo cómo se dibuja).

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de texturas. Debe comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarla directamente.

Sintaxis:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Argumento Descripción
id El valor de ID único de la superficie a dibujar.
x La posición x de donde dibujar la superficie.
y La posición y de dónde dibujar la superficie.
xscale La escala horizontal.
yscale La escala vertical.
pudrirse La rotación o el ángulo para dibujar la superficie.
col El color con el que se mezcla la superficie.
alfa La transparencia alfa para dibujar la superficie.

Devuelve:

N/A

Ejemplo:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

El código anterior dibujará una superficie indexada en la variable "surf" en la posición (0,0) de la habitación, con el doble de la escala original, mezclada en rojo y semitransparente.
