superficie_obtener_profundidad_desactivar

Esta función comprueba si la generación automática de búferes de profundidad para las superficies está activada. Normalmente, todas las superficies tienen búferes de profundidad, por lo que si se dibujan objetos 3D en ellas, se ordenarán correctamente por profundidad, sin embargo, la asignación de búferes de profundidad esencialmente duplica el tamaño de las superficies, lo que podría ser una sobrecarga excesiva e innecesaria, especialmente si su juego requiere mucha memoria o es predominantemente 2D. En estos casos puedes comprobarlo usando esta función y desactivar el buffer de profundidad para las superficies si es necesario, utilizando la función surface_depth_disable().

Sintaxis:

surface_get_depth_disable();

Devuelve:

Booleano

Ejemplo:

if (!surface_exists(surf))
{
si surface_get_depth_disable() == false
{
surface_depth_disable(true);
}
surf = surface_create(room_width, room_height);
}

El código anterior comprobará si la superficie dada existe, y si no existe, entonces comprueba el estado actual del buffer de profundidad de la superficie y si está activado, lo desactivará en su lugar, antes de crear finalmente la superficie.
