superficie_guardar_parte

Esta función guardará una parte de una superficie en el disco utilizando el nombre de archivo dado. La superficie debe guardarse como un archivo en formato *.png, y la posición (x,y) debe darse como coordenadas locales a la superficie, teniendo en cuenta que la esquina superior izquierda de la superficie es siempre (0,0). de la superficie es siempre (0,0).

Sintaxis:

surface_save_part(surface_id, fname, x, y, width, height);

Argumento Descripción
ID de superficie El ID de la superficie a establecer como objetivo de dibujo.
fname El nombre del archivo de imagen guardado.
x La posición x inicial dentro de la superficie.
y La posición inicial y dentro de la superficie.
ancho El ancho de la pieza a guardar.
altura La altura de la pieza a guardar.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("S")
{
surface_save_part(surf, "test.png", 0, 0, 100, 100);
}

El código anterior comprobará si el usuario pulsa la tecla "S" en el teclado y si lo hace guardará en el disco una parte de la superficie indexada en la variable "surf".
