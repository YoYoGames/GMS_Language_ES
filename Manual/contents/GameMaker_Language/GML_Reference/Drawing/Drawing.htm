Dibujo

Esta sección contiene todas las funciones relacionadas con el dibujo dentro de la sala de juegos, así como para controlar cómo se dibujarán las cosas (mezcla, alfa, culling, etc...). Hay un gran número de funciones diferentes para dibujar, y están divididas en las siguientes categorías para que sea más fácil encontrar lo que necesitas:

  • Color y Alfa
  • Control de la GPU
  • Mipmapping
  • Formularios básicos
  • Sprites y baldosas
  • Texto
  • Primitivas y formatos de vértices
  • Superficies
  • Iluminación
  • Partículas
  • Texturas
  • Sombreadores

También hay dos funciones especiales disponibles, una que se utiliza para habilitar o deshabilitar el evento de dibujo para su proyecto, y la otra para limpiar toda la tubería de dibujo:

  • draw_enable_drawevent
  • draw_flush
