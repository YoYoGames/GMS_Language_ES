draw_get_font

Esta función obtendrá la fuente actualmente asignada para dibujar el texto. La función devolverá -1 si no se ha establecido ninguna fuente, o el valor de ID (un entero positivo) para el recurso de fuente asignado.

Sintaxis:

draw_get_font();

Devuelve:

Entero (-1 o valor de identificación de la fuente)

Ejemplo:

si draw_get_font() != fnt_MainMenu
{
draw_set_colour(c_blue);
draw_set_font(fnt_MainMenu);
}
draw_text(room_width / 2, 200, "MENU" );

El código anterior comprobará qué fuente se está utilizando para dibujar y si no es fnt_MainMenu , entonces la fuente y el color de dibujo se establecen antes de que se dibuje algún texto en la pantalla.
