draw_set_valign

Esta función se utiliza para alinear el texto a lo largo del eje vertical y el cambio de la alineación vertical cambiará la posición y la dirección en la que se dibuja todo el texto posterior, siendo el valor por defecto fa_top. Se aceptan las siguientes constantes:

Constante Alineación
fa_top
fa_middle
fa_bottom

Sintaxis:

draw_set_valign(valign);

Argumento Descripción
Valigno Alineación vertical.

Devuelve:

N/A

Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Puntuación: " + string(puntuación));

El código anterior dibujará la puntuación centrada en el centro del texto.
