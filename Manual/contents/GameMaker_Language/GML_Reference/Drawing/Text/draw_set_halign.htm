draw_set_halign

Esta función se utiliza para alinear el texto a lo largo del eje horizontal y el cambio de la alineación horizontal cambiará la posición y la dirección en la que se dibuja todo el texto posterior, siendo el valor por defecto fa_left. Se aceptan las siguientes constantes aceptadas:

Constante Alineación
fa_left
fa_center
fa_right

Sintaxis:

draw_set_halign(halign);

Argumento Descripción
halign Alineación horizontal.

Devuelve:

N/A

Ejemplo:

draw_set_halign(fa_left);
draw_text(100, 32, "Puntuación: " + string(puntuación));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

El código anterior dibujará dos cadenas en la misma línea, con la puntuación alineada a la izquierda y la salud alineada a la derecha.
