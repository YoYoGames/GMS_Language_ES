dibujar_texto_transformado

Esta función dibujará el texto de forma similar a dibujar_texto() sólo que ahora puede elegir escalar el texto a lo largo del eje horizontal o vertical (efectivamente estirándolo o encogiéndolo) y también hacer que GameMaker Studio 2 lo dibuje en un ángulo (donde 0 es normal y cada grado por encima de 0 gira el texto en sentido contrario a las agujas del reloj).

Sintaxis:

draw_text_transformed(x, y, string, xscale, yscale, angle);

Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cadena dibujada.
cadena El cordón a dibujar.
xscale La escala horizontal (por defecto 1).
yscale La escala vertical (por defecto 1).
ángulo El ángulo del texto.

Devuelve:

N/A

Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

El código anterior dibujará el texto dado en el centro de la habitación, dando vueltas y escalado al doble de su tamaño original.
