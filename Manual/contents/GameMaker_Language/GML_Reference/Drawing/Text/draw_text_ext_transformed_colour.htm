dibujar_texto_transformado_color

Esta función es una combinación de la función base dibujar_texto() con la función dibujar_texto_ext(), dibujar_texto_transformado() y dibujar_texto_color() . Como tal, es la más general de todas las funciones para dibujar texto y ofrece una gran flexibilidad a la hora de dibujar. Tenga en cuenta que el argumento "ancho" se basa en una escala de 1, por lo que si la escala es diferente, este valor debe cambiarse proporcionalmente. Por ejemplo, si el ancho base para un salto de línea es 300 y usted establece la escala en 2, entonces el texto aparecerá mal, sobrepasando el ancho dado. En su lugar, debería establecer el ancho a 150 para compensar la escala.

NOTA: La mezcla de gradientes no está disponible para el objetivo de HTML5 a menos que WebGL esté habilitado, aunque todavía puede establecer los colores de mezcla y se mezclará la fuente con el primer color dado. Sin embargo, todas las mezclas de este tipo crean un duplicado de la fuente que se almacena en la caché y se utiliza cuando se requiere, lo que está lejos de ser óptimo y si se utilizan múltiples cambios de color se ralentizará el rendimiento de los juegos. Puedes establecer el tamaño de la caché de la fuente para tratar de limitar esto si es necesario utilizando la función font_set_cache_size().

Sintaxis:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cadena dibujada.
cadena El cordón a dibujar.
sep La distancia en píxeles entre líneas de texto.
w La anchura máxima en píxeles de la cadena antes de un salto de línea.
xscale La escala horizontal.
yscale La escala vertical.
ángulo El ángulo del texto.
c1 El color para la parte superior izquierda del texto dibujado.
c2 El color para la parte superior derecha del texto dibujado.
c3 El color para la parte inferior derecha del texto dibujado.
c4 El color para la parte inferior izquierda del texto dibujado.
alfa El alfa para el texto.

Devuelve:

N/A

Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "Mi nombre es" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Esto dibujará el texto dado con un gradiente descendente de azul a azul oscuro, en la posición (200,200) de la habitación, girado para que esté al revés y con la mitad del tamaño de punto normal de la fuente. Habrá 20 píxeles entre cada línea si hay más de una línea dibujada y cada línea tendrá una anchura máxima de 200 píxeles.
