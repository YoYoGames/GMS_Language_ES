dibujar_obtener_alineación

Esta función se utiliza para obtener el ajuste de la alineación del texto a lo largo del eje horizontal, y devolverá una de las constantes indicadas a continuación.

Sintaxis:

draw_get_halign();

Devuelve:

Constante

Constante Alineación
fa_left
fa_center
fa_right

Ejemplo:

si draw_get_halign() != fa_left
{
draw_set_halign(fa_left);
}
draw_text(100, 32, "Puntuación: " + string(puntuación));

El código anterior comprobará si la alineación actual del texto está alineada a la izquierda, y si no lo está, se establecerá y entonces se dibujará algún texto.
