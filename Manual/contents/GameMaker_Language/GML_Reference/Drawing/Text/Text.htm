Texto

Hay una serie de funciones disponibles para dibujar texto de diversas formas. Hay que tener en cuenta que algunas de estas funciones cambian los colores y tamaños del texto de forma dinámica cuando se dibuja, pero cuando se trabaja con HTML5 o dispositivos, estas funciones pueden ralentizar tu juego o tener un aspecto pobre, por lo que puede ser mejor tener recursos de fuentes de varios tamaños en lugar de utilizar el dibujo a escala, o tener recursos de fuentes grandes y luego dibujarlos a escala en lugar de hacerlo a escala. Además, si necesitas colorear grandes bloques de texto es mejor utilizar las funciones dibujar_color() y draw_set_alpha() junto con la función simple draw_text, en lugar de esas funciones extendidas que tienen un componente de color.

Las siguientes funciones sirven para establecer el tipo de letra que se utilizará al dibujar, así como el posicionamiento del texto cuando se dibuja:

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • dibujar_obtener_alineación
  • draw_get_valign

Estas funciones pueden utilizarse para dibujar texto en la pantalla, ya sea de forma simple o con transformaciones y color:

  • dibujar_texto
  • dibujar_texto_texto
  • dibujar_texto_color
  • dibujar_texto_transformado
  • dibujar_texto_ext_color
  • dibujar_texto_transformado
  • dibujar_texto_transformado_color
  • dibujar_texto_transformado_color

También existe la siguiente función que es específica para dibujar las puntuaciones máximas creadas con el sistema de puntuaciones máximas incorporado:

  • draw_highscore
