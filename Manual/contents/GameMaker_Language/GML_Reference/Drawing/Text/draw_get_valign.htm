draw_get_valign

Esta función se utiliza para obtener el ajuste de la alineación del texto a lo largo del eje vertical, y devolverá una de las constantes indicadas a continuación.

Sintaxis:

draw_get_valign();

Devuelve:

Constante

Constante Alineación
fa_top
fa_middle
fa_bottom

Ejemplo:

si draw_get_valign() != fa_top
{
draw_set_valign(fa_top);
}
draw_text(100, 32, "Puntuación: " + string(puntuación));

El código anterior comprobará si la alineación actual del texto está alineada en la parte superior, y si no lo está, se establecerá y entonces se dibujará algún texto.
