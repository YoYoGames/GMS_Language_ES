dibujar_texto

Con esta función puedes dibujar cualquier cadena en cualquier posición dentro de la habitación (para dibujar números reales debes usar la función cadena() para convertirlos en texto). Para combinar cadenas puede utilizar el signo + (véase el ejemplo más abajo) y también puede utilizar \n dentro de una cadena para añadir un salto de línea para que se dibuje en varias líneas (para obtener información sobre cómo formatear correctamente una cadena y qué caracteres de escape puede utilizar, consulte aquí). El color del texto y el alfa se rigen por los valores actuales de alfa y color base establecidos por draw_set_alpha() y draw_set_colour().

NOTA: La posición real del texto estará influenciada por los valores de alineación establecidos por draw_set_halign() y draw_set_valign().

Sintaxis:

dibujar_texto(x, y, cadena);

Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cadena dibujada.
cadena El cordón a dibujar.

Devuelve:

N/A

Ejemplo:

draw_text(x, y, "¡Hola, " + global.Name + "!\NEspero que estés bien!");

El código anterior dibujará una cadena en la posición x/y de la instancia, que utilizará la cadena almacenada en la variable global "Nombre" y la dividirá en dos líneas.
