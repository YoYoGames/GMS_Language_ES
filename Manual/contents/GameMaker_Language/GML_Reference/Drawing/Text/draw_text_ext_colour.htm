dibujar_texto_ext_color

Esta función es una combinación de la función base dibujar_texto() con la función dibujar_texto_ext() y dibujar_texto_color() que le permite definir los colores del gradiente para el texto, así como el espacio entre líneas y el ancho máximo por línea, todo junto.

NOTA: La mezcla de gradientes no está disponible para el objetivo HTML5 a menos que WebGL esté habilitado, aunque todavía puede establecer los colores de mezcla y se mezclará la fuente con el primer color dado. Sin embargo, todas las mezclas de este modo crean un duplicado de la fuente que se almacena en la caché. que se almacena en la caché y se utiliza cuando se requiere, lo que está lejos de ser óptimo y si se utilizan múltiples cambios de color se ralentizará el rendimiento de los juegos. Puedes establecer el tamaño de la caché de la fuente para tratar de limitar esto si es necesario utilizando la función font_set_cache_size().

Sintaxis:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cadena dibujada.
cadena El cordón a dibujar.
sep La distancia en píxeles entre líneas de texto.
w La anchura máxima en píxeles de la cadena antes de un salto de línea.
c1 El color para la parte superior izquierda del texto dibujado.
c2 El color para la parte superior derecha del texto dibujado.
c3 El color para la parte inferior derecha del texto dibujado.
c4 El color para la parte inferior izquierda del texto dibujado.
alfa El alfa del texto.

Devuelve:

N/A

Ejemplo:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

El código anterior dibujará cualquier texto que el usuario escriba en el teclado, dividiéndolo en nuevas líneas cada vez que la longitud de la cadena para esa línea supere los 300 píxeles. el código también mantendrá una separación de 5 píxeles entre las líneas si esto ocurre. Cada línea del texto será coloreada usando un gradiente azul, con azul claro en la parte superior y azul oscuro en la parte inferior.
