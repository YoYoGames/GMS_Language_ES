draw_set_font

Esta función establecerá la fuente que se utilizará para todo el dibujo de texto posterior. Esta fuente debe haber sido añadida a los activos de fuentes del juego o haber sido creada utilizando las funciones font_add() o font_add_sprite().

Sintaxis:

draw_set_font(font);

Argumento Descripción
fuente El nombre de la fuente a utilizar.

Devuelve:

N/A

Ejemplo:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World" );

El código anterior dibujará el texto dado utilizando la fuente indexada en la variable "fnt_Game" y coloreada en azul.
