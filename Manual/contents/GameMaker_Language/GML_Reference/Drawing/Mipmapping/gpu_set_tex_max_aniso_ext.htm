gpu_set_tex_max_aniso_ext

Con esta función se puede establecer el nivel máximo de anisotropía cuando se utiliza el modo de filtro tf_anisotropic (ver gpu_get_tex_mip_filter() para más información) en un shader sampler. Se suministra el valor de del muestreador de sombreado (devuelto por la función shader_get_sampler_index()y un valor entre 1 y 16 para establecer el nivel.

Sintaxis:

gpu_set_tex_max_aniso_ext(sampler_index, maxaniso);

Argumento Descripción
índice_de_muestras El índice del shader sampler a obtener
maxaniso El nivel anisotrópico máximo a utilizar (por defecto: 16).

Devuelve:

N/A

Ejemplo:

var _sampleIndex = shader_get_sampler_index( shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set (shd_Glass);
si gpu_get_tex_max_aniso_ext(_sampleIndex) != 8
{
gpu_set_tex_max_aniso_ext(_sampleIndex, 8);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior establece el nivel máximo de anisotropía a 8 para el muestreador de textura del sombreador dado si no ha sido ya establecido a 8.
