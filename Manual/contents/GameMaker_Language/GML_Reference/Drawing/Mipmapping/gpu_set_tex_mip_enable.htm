gpu_set_tex_mip_enable

Con esta función puedes cambiar si el mapeo está desactivado, activado para todo, o activado sólo para los grupos de texturas seleccionados en el Gestor de Grupos de Texturas. La función requiere una de las constantes indicadas a continuación.

Constante Descripción
mip_off El mapeo está desactivado.
mip_on Se ha activado el mapeado de todas las texturas.
mip_markedonly El mapeo se activa para las texturas que lo tienen activado en las opciones del grupo de texturas (por defecto).

Sintaxis:

gpu_set_tex_mip_enable(setting);

Argumento Descripción
ajuste La configuración de mipmap (una constante, por defecto: mip_markedonly)

Devuelve:

N/A

Ejemplo:

si gpu_get_tex_mip_enable != mip_on
{
gpu_set_tex_mip_enable(mip_on);
}

El código anterior comprobará si el mipmapping está habilitado y, si no lo está, lo habilitará.
