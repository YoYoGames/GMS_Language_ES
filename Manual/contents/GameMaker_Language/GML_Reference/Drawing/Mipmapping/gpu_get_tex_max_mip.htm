gpu_get_tex_max_mip

Con esta función se puede obtener el nivel máximo de mipmap que se va a utilizar, donde un valor de 0 es la máxima resolución, 1 es utilizar el primer mipmap, 2 es el segundo, etc. Tenga en cuenta que esto puede ser muy útil para evitar los artefactos de sangrado al renderizar texturas, por ejemplo, si se establece el borde de la página de la textura a 8px y luego se establece el nivel máximo de mipmap a 3, se asegurará de que no se produzcan problemas de sangrado a distancias de renderizado mayores.

Sintaxis:

gpu_get_tex_max_mip();

Devuelve:

Real (por defecto: 16)

Ejemplo:

si gpu_get_tex_max_mip() != 4
{
gpu_set_tex_max_mip(4);
}

El código anterior comprobará el nivel máximo actual del mipmap y si no es 4 se pondrá a 4.
