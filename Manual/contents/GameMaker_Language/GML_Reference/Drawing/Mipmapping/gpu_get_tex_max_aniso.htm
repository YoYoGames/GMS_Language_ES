gpu_get_tex_max_aniso

Con esta función se puede obtener el nivel máximo de anisotropía actual cuando se utiliza el modo de filtro tf_anisotropic (ver gpu_get_tex_mip_filter() para más información). El valor devuelto estará entre 1 y 16.

Sintaxis:

gpu_get_tex_max_aniso();

Devuelve:

Real (por defecto: 16)

Ejemplo:

si gpu_get_tex_max_aniso() != 8
{
gpu_set_tex_max_aniso(8);
}

El código anterior comprobará el nivel máximo de filtrado anisotrópico actual y si no es 8 se pondrá a 8.
