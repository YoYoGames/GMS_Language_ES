gpu_get_tex_mip_bias_ext

Con esta función se puede recuperar el valor de sesgo del mipmap para un muestreador de sombreado dado. Se suministra el valor del índice del sombreador (devuelto por la función shader_get_sampler_index(), y la función devolverá un valor de 0.0 para ningún sesgo, o un valor mayor donde 1 equivale al primer mipmap, 2 equivale al segundo mipmap, etc... Esto controla la velocidad a la que el mapa mip se intercambia y generalmente hará que las texturas del shader sean más borrosas cuanto más alto sea el valor y mayor sea la "distancia" que se vea. Tenga en cuenta que esto puede devolver valores negativos también, en cuyo caso las texturas del sombreador serán más nítidas en una distancia mayor cuanto menor sea el valor.

Sintaxis:

gpu_get_tex_mip_bias_ext(sampler_index);

Argumento Descripción
índice_de_muestras El índice del shader sampler a obtener

Devuelve:

Real (por defecto: 0)

Ejemplo:

var _sampleIndex = shader_get_sampler_index( shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set (shd_Glass);
si gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
{
gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior establece el sesgo del filtro mip a 0 para el muestreador de textura del sombreador dado si no ha sido ya establecido a 0.
