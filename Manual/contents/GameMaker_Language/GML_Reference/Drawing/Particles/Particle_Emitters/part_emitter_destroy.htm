part_emitter_destroy

Esta función eliminará el emisor especificado del sistema dado y lo borrará de la memoria (esto también detendrá cualquier partícula que sea producida por el emisor dado, pero NO lo elimina de la sala). Esta función siempre debe cuando el emisor ya no sea necesario para el sistema, para evitar fugas de memoria y errores.

Sintaxis:

part_emitter_destroy( ps, ind );

Argumento Descripción
ps El sistema de partículas para destruir el emisor de.
ind El índice del emisor a destruir.

Devuelve:

N/A

Ejemplo:

if part_emitter_exists(global.Sname, p_emit)
{
part_emitter_destroy(global.Sname, p_emit1);
}

El código anterior comprobará si el emisor de partículas indexado en la variable "p_emit" existe en el sistema de partículas dado y si lo hace se destruye.
