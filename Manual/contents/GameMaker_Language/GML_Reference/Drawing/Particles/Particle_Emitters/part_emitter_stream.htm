part_emitter_stream

Esta función le permite establecer un emisor para transmitir un tipo específico de partícula y se utiliza típicamente el evento de creación, ya que es un código único que establece el emisor para generar el número de partículas especificado cada paso del juego a partir de entonces. Las partículas de partículas se emiten siguiendo la distribución, la forma y la posición establecidas por la función part_emitter_region(). Puedes poner a cero el número de partículas que se emiten y volver a llamar a esta función para "apagar" el flujo de partículas, y la función también aceptará números negativos para la cantidad, en cuyo caso el emisor producirá partículas basadas en el azar. Por ejemplo, si tienes el número de partículas establecido en -5, hay hay una probabilidad de 1:5 de que se produzca una partícula en cada paso. Si necesitas que las partículas aparezcan todas a la vez en lugar de cada paso, deberás utilizar la función part_emitter_burst().

Sintaxis:

part_emitter_stream(ps, ind, parttype, number);

Argumento Descripción
ps El sistema de partículas en el que se encuentra el emisor.
ind El índice del emisor desde el que se produce la explosión.
tipo de pieza El índice (tipo) de las partículas a crear.
número El número de partículas a crear por paso.

Devuelve:

N/A

Ejemplo:

part_emitter_stream(global. Sname, p_emit1, p1, 1);

El código anterior transmitirá 1 partícula en cada paso del juego hasta que el emisor sea destruido o el flujo se ponga a 0.
