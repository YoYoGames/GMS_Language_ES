part_emitter_exists

Con esta función puedes comprobar si el emisor de partículas indexado existe en el sistema dado o no. Tenga en cuenta que si la variable que se comprueba es una variable no inicializada (a la que un emisor de partículas debería tener asignado su índice) se producirá un error.

Sintaxis:

part_emitter_exists(ps, ind);

Argumento Descripción
ps El sistema de partículas para buscar un emisor.
ind El índice del emisor a buscar.

Devuelve:

Booleano

Ejemplo:

if part_emitter_exists(sname, p_emit1)
{
part_emitter_burst(sname, p_emit1, part_1, 30);
}

El código anterior buscará el emisor indexado en la variable "permiso" y si existe, hará estallar 30 partículas de él.
