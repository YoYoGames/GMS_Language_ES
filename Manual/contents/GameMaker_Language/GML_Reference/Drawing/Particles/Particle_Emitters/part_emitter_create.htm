part_emitter_create

Esta función debe utilizarse para crear un nuevo emisor y asignarlo a un sistema de partículas determinado. La función devolverá el valor del índice del nuevo emisor, que debe almacenarse en una variable y utilizarse en todas las funciones posteriores que hagan referencia al emisor, y el propio emisor debe ser destruido cuando ya no se utilice para evitar fugas de memoria (esto puede lograrse utilizando las funciones específicas de destrucción del emisor o destruyendo todo el sistema de partículas al que pertenece el emisor).

Sintaxis:

part_emitter_create(ps);

Argumento Descripción
ps El sistema de partículas para crear el emisor en.

Devuelve:

Real

Ejemplo:

p_emit1 = part_emitter_create(Sname);

Esto creará un nuevo emisor de partículas y almacenará su índice en la variable "p_emit".
