part_emitter_clear

Con esta función puedes borrar el emisor dado del sistema de partículas especificado y devolverlo a su estado por defecto. Esto también detendrá cualquier partícula que se esté transmitiendo desde el emisor en ese momento, y si desea utilizar el emisor de nuevo tendrá que Si desea utilizar el emisor de nuevo, tendrá que establecer la posición de la región y el tipo de partícula utilizando la función part_emitter_region() función.

Sintaxis:

part_emitter_clear(ps, ind);

Argumento Descripción
ps El sistema de partículas en el que se encuentra el emisor.
ind El índice del emisor que hay que borrar.

Devuelve:

N/A

Ejemplo:

part_emitter_clear(Sname, p_emit1 );

El código anterior borrará el emisor de partículas indexado en la variable "p_emit1".
