part_emitter_destroy_all

Esta función eliminará todos los emisores definidos del sistema dado y los borrará de la memoria (esto también detendrá cualquier partícula producida por el emisor dado, pero NO los elimina de la sala). Esta función siempre debe cuando los emisores ya no sean necesarios para el sistema, para evitar fugas de memoria y errores.

Sintaxis:

part_emitter_destroy_all( ps );

Argumento Descripción
ps El sistema de partículas para destruir todos los emisores de.

Devuelve:

N/A

Ejemplo:

si vidas = 0
{
part_emitter_destroy_all(global.Sname);
room_goto(rm_Menu);
}

El código anterior comprueba la variable global incorporada "vidas" y si es 0, destruye todos los emisores de partículas y luego cambia de sala.
