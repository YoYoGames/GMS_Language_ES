part_system_automatic_draw

Esta función puede utilizarse para desactivar el dibujo de un sistema de partículas, de modo que cualquier actualización realizada en el sistema (automática o de otro tipo) no se vea. Esta es una opción puramente visual y cuando se establece en falso no será capaz de ver las partículas ya que no se dibujan, pero siguen existiendo y cambiando de posición, color, etc... como lo harían normalmente. Cuando el dibujo automático está desactivado, puedes ordenar explícitamente a GameMaker Studio 2 que dibuje el estado actual del sistema de partículas utilizando la función part_system_drawit()y si vuelves a poner esta función en true puedes volver a activar el dibujo automático.

Hay que tener en cuenta que si se utilizan los efectos simples creados por las funciones effect_create_above() o effect_create_below() entonces puedes usar los valores de 0 (para los efectos de abajo) o 1 (para los efectos de arriba) como el índice del sistema de partículas y así alternar el dibujo automático para estos también (esto también funcionará para alternar el dibujo para los efectos de partículas DnD™).

Sintaxis:

part_system_automatic_draw(ind, automatic);

Argumento Descripción
ind El índice del sistema de partículas a cambiar.
automático Si el dibujo automático está activado (true) o no (false).

Devuelve:

N/A

Ejemplo:

part_system_automatic_draw(global.Sname, false);

El código anterior desactiva el dibujo automático para el sistema de partículas indexado en la variable global "Sname".
