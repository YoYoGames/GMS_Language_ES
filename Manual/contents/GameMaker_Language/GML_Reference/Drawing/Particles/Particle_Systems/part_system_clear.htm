part_system_clear

Con esta función puedes borrar el sistema indexado a su estado por defecto, eliminando todos los emisores y restableciendo la profundidad y la posición (si se habían cambiado). Tenga cuidado al usar esta función ya que si tiene algún ajuste de instancia o estallido o cualquier otra acción que utilice un emisor asociado a este sistema, obtendrá un error a menos que esté utilizando la función part_emitter_exists() para comprobarlo. Tampoco es necesario llamar a la part_emitter_destroy() función ya que también se encarga de ello automáticamente.

NOTA: esta función borrará las partículas visibles en la sala, pero no borra las propiedades de las partículas, ni las elimina de la memoria. Para ello debes utilizar las funciones part_type_clear() y part_type_destroy().

Sintaxis:

part_system_clear(ind);

Argumento Descripción
ind El índice del sistema de partículas que hay que limpiar.

Devuelve:

N/A

Ejemplo:

part_system_clear(global.Sname);

El código anterior borrará el sistema de partículas indexado en la variable global "Sname" a su estado por defecto.
