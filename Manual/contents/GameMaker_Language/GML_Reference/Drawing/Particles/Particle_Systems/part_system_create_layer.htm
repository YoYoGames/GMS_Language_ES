part_system_create_layer

Esta función creará un nuevo sistema de partículas en una capa dada. Se da el ID único de la capa devuelto por la función layer_create() o el nombre de la capa a utilizar como una cadena - por ejemplo "instance_layer" - y luego marcar el sistema como persistente o no. Si el sistema no está marcado como persistente, se destruirá automáticamente al final de la sala en la que fue creado (esto es lo mismo que si hubieras llamado a la función part_system_destroy() y también destruirá los emisores asociados al sistema). Sin embargo, cuando se marca como persistente, el sistema será llevado a la siguiente sala cuando se cambie de sala, y si la siguiente sala no tiene una capa con el mismo nombre o profundidad que la asignada, entonces se creará una nueva capa para el sistema que persiste a través de las salas, y será nombre de la capa original. Cuando se cambie de sala, si hay otra capa en las siguientes salas con el mismo nombre, entonces la instancia persistente se asignará a la capa con ese nombre, independientemente de la profundidad de la capa. Por último, si un sistema persistente se traslada a una sala con una capa con la misma profundidad que la capa en la que se creó el sistema, no se asignará a esta capa, sino que se creará una nueva capa con la misma profundidad (con el mismo nombre que la capa original). original).

La función devolverá un valor de ID único para el sistema de partículas que deberá utilizarse en todas las llamadas posteriores a la función en las que sea necesario dar un ID del sistema.

IMPORTANTE! Si marca el sistema de partículas como persistente, entonces (y cualquier emisor asignado a él) tendrá que ser limpiado manualmente usando las funciones de destrucción apropiadas cuando no esté en uso, de lo contrario se arriesga a una fuga de memoria que impactará negativamente negativamente en el juego final.

Sintaxis:

part_system_create_layer(layer_id, persistent);

Argumento Descripción
capa El valor del ID de la capa o la cadena a la que se asigna el sistema de partículas (puede ser cualquier tipo de capa)
persistente Marcar el sistema de partículas como persistente (establecer como verdadero) o no (establecer como falso)

Devuelve:

Real (valor único de identificación del sistema de partículas)

Ejemplo:

global.p_sys = part_system_create_layer("effects_layer", true);

El código anterior creará un nuevo sistema de partículas en la capa dada y lo marcará como persistente en las siguientes salas. El ID del sistema de partículas se almacena en una variable de ámbito global para futuras referencias.
