part_system_exists

Con esta función puedes comprobar si el sistema de partículas indexado existe en el juego o no. Tenga en cuenta que si la variable que se está comprobando es una variable no inicializada (a la que un sistema de partículas tendría asignado su índice) esto arrojará arrojará un error.

Sintaxis:

part_system_exists(ind);

Argumento Descripción
ind El índice del sistema de partículas a comprobar.

Devuelve:

Booleano

Ejemplo:

if !part_system_exists(global.Sname)
{
global.Sname = part_system_create();
}

El código anterior comprueba si el sistema de partículas al que se hace referencia en la variable global existe y, si no existe, se crea.
