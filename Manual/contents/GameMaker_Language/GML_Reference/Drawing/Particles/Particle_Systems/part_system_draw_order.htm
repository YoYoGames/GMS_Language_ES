part_system_draw_order

Con esta función puedes establecer la forma en que se dibujan las partículas cuando se crean en la pantalla. El sistema por defecto utiliza un aspecto viejo > nuevo (la función se establece en true), donde las partículas viejas se dibujan a una profundidad mayor que las nuevas y pero si estableces esta función en false puedes invertir este orden y hacer que las nuevas partículas se dibujen más arriba y aparezcan "debajo" de las más antiguas. Las imágenes de abajo ilustran esto, con la imagen de la izquierda con el valor por defecto de true y la imagen de la derecha con false:

NOTA: Cuando las partículas se dibujan con un modo de mezcla aditivo, el efecto de esta función puede no ser siempre obvio.

Sintaxis:

part_system_draw_order(ind, oldtonew);

Argumento Descripción
ind El índice del sistema de partículas a cambiar.
oldtonew Si las partículas antiguas deben dibujarse detrás de las nuevas (verdadero) o viceversa (falso).

Devuelve:

N/A

Ejemplo:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Esto creará un nuevo sistema de partículas con el índice " mysystem", y luego establece las partículas para dibujar las partículas más nuevas encima de las más antiguas.
