part_particles_create

Esta función es ideal para aquellos efectos que no requieren ninguna de las funcionalidades que ofrecen los emisores de partículas (por ejemplo, para crear humo de un misil, o un simple efecto de explosión) ya que le permite crear rápida y fácilmente partículas en cualquier posición de la sala de juego. Tenga en cuenta que debe haber creado el sistema de partículas y el tipo de partícula previamente para poder utilizar esta función.

Sintaxis:

part_particles_create(ind, x, y, parttype, number);

Argumento Descripción
ind El índice del sistema de partículas.
x La coordenada x de donde crear las partículas.
y La coordenada y de dónde crear las partículas.
tipo de pieza El índice (tipo) de las partículas a crear.
número El número de partículas a crear.

Devuelve:

N/A

Ejemplo:

if mouse_check_button(mb_left)
{
part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

El código anterior comprueba si el botón del ratón está pulsado y si devuelve true genera 5 partículas en la posición del ratón.
