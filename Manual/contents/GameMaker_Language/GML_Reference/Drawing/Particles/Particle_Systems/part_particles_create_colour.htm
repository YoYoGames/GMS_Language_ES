part_particles_create

Esta función es ideal para aquellos efectos que no requieren ninguna de las funcionalidades que ofrecen los emisores de partículas (por ejemplo, para crear humo de un misil, o un simple efecto de explosión) ya que permite crear rápida y fácilmente partículas en cualquier posición de la sala de juego. También puedes colorear las partículas "sobre la marcha" a medida que se crean con esta función, y este color anulará el color predefinido de la partícula, pero no se mezclará este color a lo largo de la vida de las partículas. Tenga en cuenta que debe haber creado el sistema de partículas y el tipo de partícula previamente para que esta función pueda ser utilizada.

Sintaxis:

part_particles_create_colour(ind, x, y, parttype, color, number);

Argumento Descripción
ind El índice del sistema de partículas.
x La coordenada x de donde crear las partículas.
y La coordenada y de dónde crear las partículas.
tipo de pieza El índice (tipo) de las partículas a crear.
color La mezcla de colores para las partículas.
número El número de partículas a crear.

Devuelve:

N/A

Ejemplo:

si (velocidad > 1)
{
var _c = choose(c_aqua, c_lime, c_fuschia, c_orange);
part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

El código anterior generará una sola partícula cada paso que la instancia con el código tenga una velocidad mayor que uno. Estas partículas serán de un color aleatorio.
