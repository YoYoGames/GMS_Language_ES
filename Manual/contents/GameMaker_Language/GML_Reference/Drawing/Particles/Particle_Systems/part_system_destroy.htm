part_system_destroy

Con esta función puedes destruir un sistema de partículas dado y eliminarlo de la memoria. Esto siempre debe ser llamado cuando el sistema ya no es necesario, como al final de una habitación, o en el evento de destrucción de una instancia, de lo contrario podría terminar con partículas que aparecen en salas posteriores y no hay forma de eliminarlas, así como una fuga de memoria que acabará por colapsar el juego.

NOTA: Esta función también destruirá y eliminará cualquier emisor que haya sido creado y asociado con el sistema que se está destruyendo.

Sintaxis:

part_system_destroy(ind);

Argumento Descripción
ind El índice del sistema de partículas a destruir.

Devuelve:

N/A

Ejemplo:

if (global.player_hp <= 0)
{
part_system_destroy(p_sys);
room_goto_next();
}

El código anterior comprueba si el valor de una variable global es menor o igual a cero, y si lo es entonces destruye el sistema de partículas referenciado en la variable dada y luego pasa a la siguiente sala.
