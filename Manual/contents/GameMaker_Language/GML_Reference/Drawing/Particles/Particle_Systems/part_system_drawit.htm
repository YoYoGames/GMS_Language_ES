part_system_update

Con esta función puedes forzar a un sistema de partículas a dibujar todas las partículas actuales en la pantalla. Si part_system_automatic_draw() está desactivada, esta función mostrará las partículas cuando se utiliza en el evento de dibujo de una instancia, o se puede utilizar cuando el objetivo de dibujo se establece en una superficie (utilizando surface_set_target()) para dibujar las partículas del sistema en esa superficie.

Sintaxis:

part_system_drawit(ind);

Argumento Descripción
ind El índice del sistema de partículas para dibujar las partículas de.

Devuelve:

N/A

Ejemplo:

if surface_exists(surf)
{
surface_set_target(surf);
part_system_drawit(global.psys);
surface_reset_target();
}

El código anterior comprueba si la superficie indexada en la variable "surf" existe, y si lo hace entonces establece el objetivo de dibujo a la superficie, dibuja el sistema de partículas con el ID almacenado en la variable global, y luego restablece el objetivo de dibujo para que todo el dibujo normal aparezca de nuevo en la pantalla.
