posición_sistema_parte

Con esta función puedes establecer la posición base para el sistema de partículas en relación con la posición (0,0) de la habitación, lo que significa que todas las funciones de partículas posteriores relacionadas con este sistema se dibujarán ahora en relación con la nueva posición. Por defecto esta posición es siempre (0,0), pero en algunas circunstancias muy especiales es posible que desee cambiar esto a otra cosa.

NOTA: Esta función cambiará todo dentro del sistema de partículas, así que si tienes un emisor en la posición (100,100) y luego estableces la posición del sistema de partículas en (0,100), el emisor ahora se dibujará en (100,200). Lo mismo ocurre si desplazas el sistema y luego creas el emisor, ya que aunque lo crees en (100,100) se dibujará en (100,200).

Sintaxis:

part_system_position(ind, x, y);

Argumento Descripción
ind El índice del sistema de partículas a cambiar.
x La nueva coordenada x del sistema de partículas.
y La nueva coordenada y del sistema de partículas.

Devuelve:

N/A

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
part_system_position(global.Sname, mouse_x, mouse_y);
}

El código anterior comprobará si se pulsa el botón del ratón y, si lo detecta, el sistema de partículas indexado en la variable global "Sname" se desplaza a la posición x/y del ratón
