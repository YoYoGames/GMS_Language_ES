conteo_de_partículas_de_parte

Con esta función puedes comprobar si un sistema de partículas tiene actualmente alguna partícula creada en la sala, y devolverá también el número de ellas.

Sintaxis:

part_particles_count(ind);

Argumento Descripción
ind El índice del sistema de partículas.

Devuelve:

Real

Ejemplo:

si (part_particles_count(Sname) == 0)
{
part_system_destroy(Sname);
instance_destroy();
}

El código anterior comprobará el número de partículas en el sistema de partículas local indexado en la variable "Sname" y si no hay ninguna, destruirá el sistema y luego la instancia.
