profundidad_sistema_parte

Con esta función puedes establecer la profundidad de dibujado para el sistema de partículas, de la misma manera que puedes establecer la profundidad de renderizado de las diferentes capas dentro de la habitación, donde una profundidad de dibujado baja significa que aparecerá encima de todas las cosas dibujadas con una profundidad mayor profundidad, y una profundidad de dibujado alta lo coloca debajo de todo lo que tiene una profundidad de dibujado más baja.

Sintaxis:

part_system_depth( ind, depth );

Argumento Descripción
ind El índice del sistema de partículas a cambiar.
profundidad La profundidad a la que se debe fijar el sistema de partículas.

Devuelve:

N/A

Ejemplo:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

El código anterior creará un sistema de partículas y almacenará su índice en la variable global "Sname". A este sistema se le da una profundidad baja de -1000, lo que significa que aparecerá por encima de todo lo que tenga una profundidad de dibujo mayor.
