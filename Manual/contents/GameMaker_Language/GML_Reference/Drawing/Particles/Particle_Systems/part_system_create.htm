part_system_create

Esta función se utiliza para crear un nuevo sistema de partículas y devolverá un número de índice único que debe ser almacenado y utilizado en todas las funciones posteriores relacionadas con ese sistema. Al sistema se le asignará una capa gestionada y se establecerá que tenga una profundidad de 0. Las capas gestionadas no son accesibles para el usuario y sólo se utilizan para la gestión interna cuando se utiliza la profundidad en lugar de las capas. Normalmente se utilizaría la función part_system_create_layer() en lugar de de ésta.

Sintaxis:

part_system_create();

Devuelve:

Real

Ejemplo:

global.Sname = part_system_create();

Esto creará un nuevo sistema de partículas y almacenará el índice en la variable global "Sname".
