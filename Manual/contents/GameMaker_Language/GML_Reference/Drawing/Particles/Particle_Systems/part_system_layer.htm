capa_sistema_parte

Esta función puede utilizarse para cambiar un sistema de partículas de su capa actual a una nueva. Se suministra el valor del ID del sistema de partículas (devuelto por la función part_system_create_layer()) y luego el ID único de la capa (devuelto por la función layer_create() o el nombre de la capa a utilizar como una cadena - por ejemplo "instance_layer" -. tal y como se define en el editor de salas), y el sistema se moverá a la nueva capa.

Sintaxis:

part_system_layer(ps, layer);

Argumento Descripción
ps El ID del sistema de partículas
capa El nuevo ID de la capa

Devuelve:

N/A

Ejemplo:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
part_system_layer(global.p_sys, "effects_layer";
}

El código anterior comprobará un sistema de partículas para ver en qué capa está y si no está en la capa dada se cambiará.
