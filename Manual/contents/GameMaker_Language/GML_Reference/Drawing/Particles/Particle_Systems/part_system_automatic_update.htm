part_system_automatic_update

Esta función controla si GameMaker Studio 2 debe actualizar el sistema de partículas de forma automática o no, y normalmente no sería necesario utilizar esta función como el valor predeterminado de true (actualización automática está en) es lo que desea ocurrir. Sin embargo, para efectos especiales o para pausar el juego, puede establecer esto en falso y evitará que se realice cualquier actualización en el sistema de partículas dado a menos que se ordene explícitamente mediante el uso de la función part_system_update() o que utilices esta función de nuevo para activar las actualizaciones automáticas. No actualizar significa que las partículas creadas en este sistema ya no cambiarán de posición, color o cualquier otro parámetro y que los emisores dejarán de funcionar también, "congelando" el sistema en el punto exacto en el que la actualización automática se estableció en falso.

Hay que tener en cuenta que si se utilizan los efectos simples creados por las funciones effect_create_above() o effect_create_below() entonces puede utilizar los valores de 0 (para los efectos inferiores) o 1 (para los efectos superiores) como el índice del sistema de partículas y así activar la actualización automática para estos también (esto también funcionará para activar la actualización para los efectos de partículas DnD™).

Sintaxis:

part_system_automatic_update(ind, automatic);

Argumento Descripción
ind El índice del sistema de partículas a cambiar.
automático Si la actualización automática está activada(true) o no (false).

Devuelve:

N/A

Ejemplo:

if global.Pause
{
part_system_automatic_update(global.Sname, false);
}

El código anterior desactivará las actualizaciones de las partículas si la variable global "Paused" es verdadera.
