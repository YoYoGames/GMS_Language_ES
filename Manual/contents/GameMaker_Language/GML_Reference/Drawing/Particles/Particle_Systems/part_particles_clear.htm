part_particles_clear

Con esta función puedes borrar de la sala todas las partículas creadas actualmente por el sistema. No restablece ni elimina los tipos de partículas en sí, sólo su representación visual, y si tiene algún objeto que emite partículas desde un emisor, estas partículas desaparecen pero comenzarán a aparecer de nuevo al siguiente paso después de llamar a este código.

Sintaxis:

part_particles_clear(ind);

Argumento Descripción
ind El índice del sistema de partículas.

Devuelve:

N/A

Ejemplo:

si (vidas <= 0)
{
part_particles_clear(global.Sname);
room_goto(rm_intro);
}

El código anterior comprobará el valor de la variable "vidas" y si es igual a 0, borra todas las partículas del sistema y luego cambia de sala.
