part_system_get_layer

Esta función se puede utilizar para recuperar el valor único de ID de capa para el sistema de partículas dado. Se suministra el valor del ID del sistema de partículas (devuelto por la función part_system_create_layer()) y la función devolverá el valor de ID de la capa.

Sintaxis:

part_system_get_layer(ind);

Argumento Descripción
ind El valor del ID del sistema de partículas para obtener el ID de la capa de

Devuelve:

Real (valor único de identificación de la capa o 0 si la capa es de gestión interna)

Ejemplo:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
part_system_layer(global.p_sys, "effects_layer";
}

El código anterior comprobará un sistema de partículas para ver en qué capa está y si no está en la capa dada se cambiará.
