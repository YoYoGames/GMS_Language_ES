effect_create_above

Con esta función puede crear un efecto simple sobre todas las instancias de su sala (en realidad se crea a una profundidad de -100000). Si el efecto es cualquier otra cosa que ef_rain o ef_snow entonces puede definir una posición x/y para crear el y el tamaño puede ser un valor de 0, 1 o 2, donde 0 es pequeño, 1 es mediano y 2 es grande.

Cabe destacar que estos efectos pueden activar y desactivar su dibujo, así como pausarlo, utilizando las funciones part_system_automatic_draw() y part_system_automatic_update() con el valor apropiado para el índice del sistema de partículas (donde 0 es para los efectos inferiores y 1 para los superiores).

Las constantes disponibles para los diferentes tipos de partículas son:

Constante Ejemplo Descripción
ef_cloud Un efecto que crea partículas de nubes aleatorias de diferentes tamaños
ef_ellipse Un efecto que crea elipses en expansión
ef_explosión Un efecto que crea explosiones expansivas que se desvanecen
ef_firework Un efecto que crea múltiples partículas pequeñas para generar una explosión de fuegos artificiales
ef_flare Un efecto que genera un punto brillante que se enciende y se apaga
ef_rain Un efecto que genera partículas de lluvia que bajan desde la parte superior de la pantalla
ef_ring Un efecto que genera círculos que se expanden y se desvanecen
ef_smoke Un efecto que genera pequeñas bocanadas de humo
ef_smokeup Un efecto que crea una columna de humo que sube por la pantalla
ef_snow Un efecto que genera múltiples partículas de nieve que caen por la pantalla
ef_spark Un efecto que genera una pequeña chispa
ef_star Un efecto que genera partículas estelares

Sintaxis:

effect_create_above(kind, x, y, size, colour);

Argumento Descripción
amable El tipo de efecto (utilice una de las constantes indicadas anteriormente).
x El posicionamiento x del efecto si es relevante.
y La posición y del efecto si es relevante.
tamaño El tamaño del efecto.
color El color del efecto.

Devuelve:

N/A

Ejemplo:

si salud <= 0
{
effect_create_above(ef_explosion, x, y, 1, c_yellow);
instance_destroy();
}

El código anterior creará una explosión mediana, amarilla, sobre la instancia y la destruirá si la variable "salud" es menor o igual a 0.
