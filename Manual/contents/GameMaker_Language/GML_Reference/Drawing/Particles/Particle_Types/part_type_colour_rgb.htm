part_type_colour_rgb

Con esta función puedes establecer la mezcla de colores rojo, verde y azul para todas las partículas creadas del tipo dado. Se proporciona un valor mínimo y un valor máximo para cada uno de los tres componentes y las partículas creadas tendrán un color aleatorio basado en el rango de parámetros dado. en el rango de parámetros dado. Todos los valores deben estar entre 0 y 255.

Sintaxis:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
rmin El mínimo que puede tener el componente rojo del color final.
rmax El máximo que puede tener el componente rojo del color final.
gmin El mínimo que puede tener el componente verde del color final.
gmax El máximo que puede tener el componente verde del color final.
bmin El mínimo que puede tener el componente azul del color final.
bmax El máximo que puede tener el componente azul del color final.

Devuelve:

N/A

Ejemplo:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

El código anterior establece que cada partícula emitida del tipo de partícula indexada en la variable global "Blood_Part" sea sólo de diferentes tonos de rojo.
