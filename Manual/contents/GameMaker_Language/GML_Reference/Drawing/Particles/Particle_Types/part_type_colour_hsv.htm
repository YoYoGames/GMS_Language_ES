part_type_colour_hsv

Con esta función puede establecer un rango de tono, saturación y valor para todas las partículas del tipo dado. Se proporciona un valor mínimo y un valor máximo para cada uno de los tres componentes y las partículas creadas tendrán un color aleatorio basado en el rango de parámetros dado. rango de parámetros dado. De esta manera puede crear partículas del mismo tono pero con diferentes saturaciones, o de diferentes tonos pero con el mismo valor (luminosidad), etc. Todos los valores deben estar entre 0 y 255.

Sintaxis:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
hmin El mínimo que puede tener el componente de tono del color final.
hmax El máximo componente de tono del color final puede ser.
smin El mínimo que puede tener el componente de saturación del color final.
smax El máximo componente de saturación del color final puede ser.
vmin El mínimo que puede tener el componente de valor del color final.
vmax El máximo que puede tener el componente de valor del color final.

Devuelve:

N/A

Ejemplo:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

El código anterior establece que cada partícula emitida del tipo de partícula indexada en la variable global "Estrellas" tenga diferentes colores y saturaciones, pero el mismo valor (luminosidad).
