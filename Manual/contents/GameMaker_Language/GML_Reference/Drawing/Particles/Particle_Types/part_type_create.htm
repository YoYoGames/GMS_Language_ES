part_type_create

Con esta función se puede crear un nuevo tipo de partícula y el valor de retorno debe ser almacenado en una variable para su uso en todas las funciones de partículas posteriores.

Sintaxis:

part_type_create();

Devuelve:

Entero (ID de la partícula)

Ejemplo:

mypart = part_type_create();

Esto creará un nuevo tipo de partícula, almacenando su índice en la variable "mypart".
