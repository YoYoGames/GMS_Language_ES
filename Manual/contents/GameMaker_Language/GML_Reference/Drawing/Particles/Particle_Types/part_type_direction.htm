tipo_parte_dirección

Esta función se utiliza para determinar la dirección de la partícula cuando se crea y también puede utilizarse para hacer que la partícula aumente o disminuya su dirección durante su vida. Los valores mínimo y máximo de la dirección son por defecto 0, lo que significa que la partícula se moverá hacia la derecha sin ninguna variación, pero estos valores pueden ser cambiados para limitar la dirección del movimiento siguiendo las direcciones estándar de GameMaker Studio 2 de 0 siendo la derecha, 90 siendo arriba, 180 siendo la izquierda y 270 siendo abajo. Si los estableces con el mismo valor las partículas sólo se moverán en una dirección.

También puede establecer un valor de incremento que sumará (si es un número positivo) o restará (si es un número negativo) una cantidad de grados a la dirección durante su vida. Este valor puede ser un mínimo de (+/-) 0,01.

Por último, puede establecer el factor de "meneo". Este es un valor que se sumará o restará aleatoriamente a la dirección en cada paso de la vida de las partículas. Obviamente, los valores más grandes son más pronunciados que los más pequeños, y este valor puede incluso ser negativo, siendo el rango máximo entre -20 y 20.

Sintaxis:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
dir_min La dirección mínima en la que puede empezar la partícula.
dir_max La dirección máxima en la que puede empezar la partícula.
dir_incr Cuánto debe aumentar o disminuir la dirección de las partículas por paso.
dir_wiggle Cuánto debe sumarse o restarse aleatoriamente de la dirección de la partícula por paso.

Devuelve:

N/A

Ejemplo:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

El código anterior establecerá varios valores de las partículas incluyendo la dirección que será un valor aleatorio entre 0 y 359 grados. También subirá 1 grado en cada paso y tendrá una cantidad aleatoria añadida entre 0 y 20 en cada paso también.
