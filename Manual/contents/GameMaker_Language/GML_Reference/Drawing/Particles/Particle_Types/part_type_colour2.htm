tipo_parte_color2

Esta función puede utilizarse para establecer un gradiente de dos colores para cada partícula creada del tipo dado. El primer color es aquel con el que comenzarán todas las partículas, y el segundo color es aquel con el que terminará la partícula, y se producirá un suave gradiente de un color a otro a lo largo de la vida de las partículas.

Sintaxis:

tipo_parte_color2(ind, color1, color2);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
color1 El color con el que comenzará la partícula.
color2 El color en el que terminará la partícula.

Devuelve:

N/A

Ejemplo:

part_type_colour2( part_Health, c_red, c_white);

El código anterior hará que todas las partículas creadas del tipo indexado en la variable "part_Health" tengan una mezcla de color de rojo a blanco durante su vida.
