parte_tipo_color1

Esta función se utiliza para establecer que un tipo de partícula sea de un solo color durante la duración total de la vida de cada partícula individual.

Sintaxis:

tipo_parte_color1(ind, color1);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
color1 El color único para hacer el tipo de partícula.

Devuelve:

N/A

Ejemplo:

part_type_colour1(global.Snow_Part, c_white);

El código anterior establecerá que todas las partículas creadas del tipo de partícula indexado en la variable global "Snow_Part" sean sólo blancas.
