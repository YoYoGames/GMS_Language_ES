part_type_colour_mix

Con esta función se puede establecer que el tipo de partícula dado sea una mezcla aleatoria de dos colores.

Sintaxis:

part_type_colour_mix(ind, color1, color2);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
color1 El primer color del que tomará la mezcla.
color2 El segundo color del que tomará la mezcla.

Devuelve:

N/A

Ejemplo:

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

El código anterior establecerá que el color de cada partícula emitida del tipo de partícula indexado en la variable global "P_Damage" sea una mezcla aleatoria entre los colores rojo y amarillo.
