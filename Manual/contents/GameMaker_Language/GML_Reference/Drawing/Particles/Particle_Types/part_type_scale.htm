tipo_parte_escala

Esta función establecerá la escala horizontal y vertical de la partícula antes de realizar cualquier otro cambio de tamaño. Esta función trabaja en la relación del tamaño del sprite, por lo que un valor de 1 (el valor por defecto) es una relación 1:1, lo que significa que 0,5 sería la mitad y 2 el doble.

Sintaxis:

part_type_scale(ind, xscale, yscale);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
xscale La escala horizontal de la partícula base.
yscale La escala vertical de la partícula base.

Devuelve:

N/A

Ejemplo:

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

El código anterior establecerá varios valores de las partículas, incluyendo la escala que establece la escala x a ocho veces el tamaño y la escala y a la mitad del tamaño.
