tipo_parte_claro

Con esta función puedes "resetear" una partícula, devolviendo todos los valores de cada una de las funciones relacionadas con la partícula (vida, color, alfa, orientación, etc...) a sus valores por defecto. Tenga en cuenta que esta función no elimina las partículas actualmente visibles en la sala de la pantalla, para eso deberías usar part_particles_clear().

Sintaxis:

part_type_clear(ind);

Argumento Descripción
ind El índice del tipo de partícula a limpiar.

Devuelve:

N/A

Ejemplo:

part_type_clear(global.explode_part);

El código anterior borrará el tipo de partícula indexado en la variable global "explode_part" a sus valores por defecto.
