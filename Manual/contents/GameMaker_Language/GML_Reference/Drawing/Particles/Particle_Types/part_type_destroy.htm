part_type_destroy

Con esta función puedes eliminar del juego el tipo de partícula especificado. Cuando se utiliza esta función, todas las partículas del tipo dado desaparecerán de la sala y la propia partícula se elimina de la memoria, por lo que esta función debe utilizarse sólo cuando ya no necesites la partícula.

Sintaxis:

part_type_destroy(ind);

Argumento Descripción
ind El índice del tipo de partícula a destruir.

Devuelve:

N/A

Ejemplo:

si part_particles_count(smoke_sys) = 0
{
part_type_destroy(smoke_part);
part_system_destroy(smoke_sys);
instance_destroy();
}
else alarma[0] = 1;

El código anterior comprueba un sistema de partículas para ver si hay partículas visibles en la sala y, si no, las partículas, el sistema y la instancia se destruyen.
