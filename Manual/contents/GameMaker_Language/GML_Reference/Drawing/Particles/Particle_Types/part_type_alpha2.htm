part_type_alpha2

Esta función se puede utilizar para establecer un gradiente de valor alfa (transparencia) para cada partícula creada del tipo dado. El primer alfa es con el que empezarán todas las partículas, y el segundo alfa es con el que terminará la partícula, y se producirá un suave cambio de gradiente en el alfa a lo largo de la vida de las partículas de uno a otro. Puede ser de 0 (transparente) a 1 (opaco).

Sintaxis:

part_type_alpha2(ind, alpha1, alpha2);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
alfa1 El alfa inicial de la partícula.
alfa2 El alfa final de la partícula.

Devuelve:

N/A

Ejemplo:

part_type_alpha2(global.Snow_Part, 0, 1);

El código anterior establecerá todas las partículas creadas del tipo de partícula indexado en la variable global "Snow_Part" para tener un valor alfa de 0 (transparente) y luego se desvanecerá para tener un alfa de 1 (opaco) al final de su vida.
