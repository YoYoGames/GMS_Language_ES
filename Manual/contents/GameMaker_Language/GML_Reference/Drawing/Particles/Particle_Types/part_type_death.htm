tipo_parte_muerte

Esta función se puede utilizar para hacer que su partícula estalle otro tipo de partícula al final de su vida.

NOTA: ¡Nunca utilices el mismo tipo de partícula ya que esto causará un bucle infinito y puede hacer que el juego se bloquee en segundos! Además, tenga mucho cuidado al usar esto, ya que puede aumentar en gran medida el número de partículas en la pantalla a la vez y hacer que sus juegos a ralentizar sus juegos notablemente.

Sintaxis:

part_type_death(ind, death_number, death_type);

Argumento Descripción
ind El índice del tipo de partícula a cambiar.
número_de_muerte La cantidad de la nueva partícula a crear en la muerte de la partícula. Si es un valor negativo, en su lugar se creará una fracción de probabilidad de que se cree una (por ejemplo, -3 significaría que se crea una cada 3 pasos más o menos).
tipo_de_muerte El índice del tipo de partícula a crear.

Devuelve:

N/A

Ejemplo:

particle1 = part_type_create();
part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 0.20, 0.30, 0.50, 0);
part_type_scale(particle1, 0.10, 0.20);
part_type_colour3(particle1, 33023, 8454143, 33023);
part_type_alpha3(particle1, 0,30, 0,50, 0);
part_type_speed(particle1, 0,40, 1, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_gravity(particle1, 0,20, 90);
part_type_orientation(particle1, 90, 90, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 1, 40);
particle2 = part_type_create();
part_type_shape(particle2, pt_shape_smoke);
part_type_size(particle2, 1, 1, 0, 0);
part_type_scale(particle2, 0.50, 0.50);
part_type_colour1(particle2, 12632256);
part_type_alpha3(particle2, 0.10, 0.40, 0);
part_type_speed(particle2, 0.50, 0, 0);
part_type_direction(particle2, 0, 359, 0, 0);
part_type_gravity(particle2, 0.20, 90);
part_type_orientation(particle2, 0, 359, 0, 1, 1);
part_type_blend(particle2, 0);
part_type_life(particle2, 60, 60);
part_type_death(particle1, 1, particle2);

El código anterior crea y define dos partículas y establece la partícula indexada en la variable "particle1" para crear 1 partícula de "particle2" al final de su vida.
