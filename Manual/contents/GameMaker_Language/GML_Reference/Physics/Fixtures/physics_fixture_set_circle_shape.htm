physics_fixture_set_circle_shape

Esta función define una forma de círculo para su accesorio con un radio definido por el argumento "rad".

Sintaxis:

physics_fixture_set_circle_shape(fixture, rad)

Argumento Descripción
accesorio el índice de la luminaria
rad radio del círculo

Devuelve:

N/A

Ejemplo:

physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2);

El código anterior aplicará una forma circular a la fijación indexada en la variable "fix_Ball" con un radio igual al del ancho del sprite "spr_Ball" dividido por 2.
