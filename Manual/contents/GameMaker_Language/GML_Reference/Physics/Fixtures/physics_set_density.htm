physics_set_density

Cuando se vincula un fixture a una instancia utilizando physics_fixture_bind() esto devuelve un "id" para el fixture vinculado. Puede utilizar este identificador para establecer el valor de la densidad del dispositivo vinculado, no del dispositivo "base", en cualquier momento utilizando esta función. "base", en cualquier momento utilizando esta función.

Sintaxis:

physics_set_density(fixture, density)

Argumento Descripción
accesorio el identificador del accesorio vinculado
densidad el nuevo valor de densidad a aplicar

Devuelve:

N/A

Ejemplo:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

El código anterior obtiene el valor de la densidad de corriente para las propiedades físicas vinculadas de la instancia y luego las establece en un valor diferente.
