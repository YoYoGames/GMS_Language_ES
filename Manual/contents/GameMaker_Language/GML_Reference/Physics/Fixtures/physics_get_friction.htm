physics_fixture_set_awake

Cuando se vincula un fixture a una instancia utilizando physics_fixture_bind() esto devuelve un "id" para el fixture vinculado. Puedes usar este id para obtener el valor de fricción del fixture ligado(no el fixture "base") en cualquier momento usando esta función.

Sintaxis:

physics_get_friction(fixture)

Argumento Descripción
accesorio el identificador del accesorio vinculado

Devuelve:

Real

Ejemplo:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

El código anterior obtiene el valor de fricción actual para las propiedades físicas vinculadas de la instancia y luego las establece a un valor diferente.
