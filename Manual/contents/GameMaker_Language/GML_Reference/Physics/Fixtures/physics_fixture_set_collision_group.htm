physics_fixture_set_collision_group

Una vez que hayas creado tu fixture, puedes definirlo como parte de un grupo de colisión. Esto significa que puedes decirle a las instancias con ese fixture que siempre colisionen o que nunca colisionen con otras instancias que tengan fixtures vinculados a ellas con con el mismo grupo de colisión. Los valores que se pueden asignar se basan en números positivos o negativos, siendo 0 el valor por defecto (lo que significa que la instancia con esta configuración de fixture se ignora a menos que se cree un evento de colisión para ella). A continuación se muestra una imagen para ayudar a explicar:

Como puede ver, la configuración de los grupos de colisión puede ser muy útil para crear complejos cuerpos físicos que interactúan con el mundo físico a su alrededor, pero no entre sí. Sin embargo, debe tenerse en cuenta que debido a la complejidad del cálculo de las colisiones basadas en grupos como este, es recomendable que trate de mantener el número de grupos en uso en cualquier sala a un mínimo absoluto.

Sintaxis:

physics_fixture_set_collision_group(fixture, group)

Argumento Descripción
accesorio el accesorio que va a ser atado
grupo el grupo al que se asigna esta luminaria (un valor positivo, 0, o un valor negativo)

Devuelve:

N/A

Ejemplo:

physics_fixture_set_collision_group(fix_Wheel, -1);

El código anterior establecerá la fijación indexada en la variable "fix_Wheel" para ser parte del grupo "-1".
