physics_fixture_set_edge_shape

Esta función define una forma de fijación de "borde". Una forma de borde es simplemente una línea que generará una colisión cuando otros accesorios la sobrepasen, y puede ser muy útil para generar (por ejemplo) terreno, o para crear bordes alrededor de una habitación. La posición del borde se define utilizando el espacio local, es decir, la posición x/y de la instancia se considera (0,0), por lo que debe tenerse en cuenta al crearlos (en el ejemplo de código siguiente, la instancia se habría colocado en (0,0) en la habitación para evitar complicaciones).

Sintaxis:

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

Argumento Descripción
accesorio el índice de la luminaria
local_x1 posición inicial x para el borde
local_y1 posición inicial y para el borde
local_x2 posición x final para el borde
local_y2 posición final y para la arista

Devuelve:

N/A

Ejemplo:

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
var fix = physics_fixture_create();
physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
physics_fixture_bind(fix, id);
physics_fixture_delete(fix);
xx += 50;
y1 = y2;
y2 = room_height - 50 - irandom(100);
}

El código anterior creará una línea de luminarias de "borde" con una variedad de alturas a lo largo de la habitación.
