physics_particle_group_circle

Esta función establecerá la forma del grupo de partículas que se está creando. Primero debe haber comenzado la definición del grupo utilizando la función physics_particle_group_begin()y luego Esta función se utiliza para establecer el radio aproximado del grupo en píxeles - aproximado porque el radio exacto dependerá del tamaño de las partículas base, definido por la función physics_particle_set_radius() ya que la simulación física intenta "encajar" el mayor número posible de partículas en la forma definida. Finalmente hay que llamar a physics_particle_group_end() para crear el grupo de partículas en la sala.

Sintaxis:

physics_particle_group_circle(radio)

Argumento Descripción
radio El radio (en píxeles) del círculo a crear.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

El código anterior almacena las banderas para el tipo de partícula y las propiedades del grupo de partículas en variables y luego las utiliza para crear un grupo de partículas circular con un radio de 100px en la posición del ratón.
