Partículas corporales blandas

GameMaker Studio 2 utiliza la biblioteca de física de cuerpos rígidos Box2D, y como tal no permite la física de cuerpos blandos. Sin embargo, la Diversión con los líquidos extensión añade partículas al el sistema de física, que permiten la creación de cuerpos blandos simulados utilizando la física de cuerpos rígidos. Básicamente, se puede crear una forma de partículas que mantienen la cohesión y así simular un cuerpo completo, que, debido a la interacción de las partículas, da la impresión de un cuerpo blando. También puede utilizar las partículas de física para simular líquidos y otros sistemas de física basados en partículas.

Antes de empezar a crear tus partículas de física necesitarás establecer las propiedades básicas de física para ellas, como lo harías con un accesorio en la física regular. Tenga en cuenta que la configuración de estas propiedades afectará a todas las partículas creadas anteriormente, así como a las nuevas partículas que cree. así como a las nuevas partículas que crees, por lo que normalmente sólo utilizarás estas funciones al principio del juego o del nivel. Sin embargo, hay funciones específicas disponibles para establecer las banderas de las partículas individuales (y grupos de partículas) que cambiarán sus propiedades como "rebote" o "viscosidad", permitiéndote crear diferentes tipos de partículas dentro de los rangos de estos ajustes globales.

Las siguientes funciones están disponibles para establecer y obtener los datos del sistema global de partículas:

  • physics_particle_count
  • physics_particle_group_count
  • physics_particle_set_max_count
  • physics_particle_set_radius
  • physics_particle_set_density
  • physics_particle_set_damping
  • physics_particle_set_gravity_scale
  • physics_particle_get_max_count
  • physics_particle_get_radius
  • physics_particle_get_density
  • physics_particle_get_damping
  • physics_particle_get_gravity_scale

También dispone de las siguientes funciones para establecer banderas en partículas individuales (o grupos de partículas), así como para obtener información, como la posición o el color, de una sola partícula o de un solo grupo:

  • physics_particle_set_flags
  • physics_particle_set_category_flags
  • physics_particle_set_group_flags
  • physics_particle_get_group_flags
  • physics_particle_group_get_inertia
  • physics_particle_group_get_centre_x
  • physics_particle_group_get_centre_y
  • physics_particle_group_get_vel_x
  • physics_particle_group_get_vel_y
  • physics_particle_group_get_ang_vel
  • physics_particle_group_get_x
  • physics_particle_group_get_y
  • physics_particle_group_get_angle
  • physics_particle_get_data
  • physics_particle_get_data_particle
  • physics_particle_group_get_data

Puede crear partículas de dos maneras: individualmente o en grupos. Las partículas individuales se comportarán exactamente como lo definen los ajustes globales y las banderas que se establecieron cuando se crearon, pero los grupos de partículas pueden mostrar diferentes comportamientos dependiendo de sus banderas de grupo. Estas banderas de grupo pueden establecerse en diferentes valores para dar al grupo diferentes propiedades, como la penetración y la rigidez.

Las siguientes funciones están disponibles para crear partículas:

  • physics_particle_create
  • physics_particle_group_begin
  • physics_particle_group_circle
  • physics_particle_group_box
  • physics_particle_group_polygon
  • physics_particle_group_add_point
  • physics_particle_group_end
  • physics_particle_group_join
  • physics_particle_group_delete

No sólo puede crear partículas, sino que también puede eliminarlas, ya sea individualmente o en una región. Las siguientes funciones están disponibles para eliminar partículas del sistema:

  • physics_particle_delete
  • physics_particle_delete_region_circle
  • physics_particle_delete_region_box
  • physics_particle_delete_region_poly

A menos que hayas activado el dibujo de depuración usando physics_draw_debug()no verás ninguna partícula en la pantalla cuando la crees. Por lo tanto, para ver realmente las partículas, necesitas dibujarlas en la pantalla en el evento Draw de una instancia usando las siguientes funciones (nota que también puedes dibujarlas usando las funciones regulares draw_sprite si obtienes los datos de cada partícula usando la función physics_particle_get_data()):

  • physics_particle_draw
  • physics_particle_draw_ext
