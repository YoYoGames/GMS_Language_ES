physics_particle_draw_ext

Esta función dibujará un sprite en la posición de todas las partículas que compartan el mismo valor de categoría definido por el usuario, y que tengan banderas que coincidan con las establecidas para la máscara. El valor de la máscara se define de la misma manera que se definen las banderas al crear las partículas (ver las funciones physics_particle_create() para ver un ejemplo), es decir, usando el bitwise o "|" para enmascarar los bits apropiados para cada bandera.

NOTA: Esta función dibujará todas las partículas de la simulación física con banderas que coincidan con las de la máscara, ya sean partículas individuales o grupos, si la categoría del usuario es la misma.

La partícula dibujada ignorará los valores de color y alfa establecidos cuando la creaste, y en su lugar se dibujará usando los parámetros de escala, ángulo, color y alfa dados en esta función. Tenga en cuenta que esto es puramente para dibujar un sprite en la posición de la partícula y no afecta en absoluto al comportamiento de la partícula.

Sintaxis:

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

Argumento Descripción
máscara de escritura La máscara con la que se comparan las banderas de las partículas.
categoría La categoría definida por el usuario para dibujar (use 0 para dibujar todas las categorías).
sprite El sprite a utilizar para cada partícula.
subimg El image_index del sprite a dibujar.
xscale El factor de escala a aplicar al sprite a lo largo del eje x.
yscale El factor de escala a aplicar al sprite a lo largo del eje y.
ang El ángulo (en grados) para dibujar el sprite.
col El color a mezclar con el sprite (por defecto es c_white).
alfa El valor alfa a utilizar al dibujar (0 - 1).

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

El código anterior dibujará un sprite para todas las partículas de la categoría definida por el usuario "2" que tengan las mismas banderas que la máscara dada, escaladas por 2 en ambos ejes y con un alfa de 0,5.
