physics_particle_group_join

Con esta función puede unir dos grupos de partículas, y los grupos unidos se comportarán como si fueran parte de una sola entidad. Los grupos deben haber sido creados con bordes superpuestos, ya que, si no se tocan, no se unirán. no se unirán. La función toma los ID de grupo únicos de los grupos a unir, tal y como los devuelve la función physics_particle_group_end()y puede utilizar la función cualquier número de veces para un solo grupo para unir varios cuerpos blandos.

Sintaxis:

physics_particle_group_join(to, from)

Argumento Descripción
a El primer grupo de partículas que se une.
de El segundo grupo de partículas para unirse.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x- 45, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g1 = physics_particle_group_end();
physics_particle_group_begin(flags, groupflags, mouse_x + 45, mouse_y, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g2 = physics_particle_group_end();
physics_particle_group_join(g1, g2);

El código anterior crea dos grupos de partículas circulares y los une.
