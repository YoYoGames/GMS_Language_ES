physics_particle_group_count

Esta función devolverá el número de partículas que están activas en un solo grupo. El índice del grupo (ID) es el valor que se devuelve cuando se llama a la función physics_particle_group_end().

Sintaxis:

physics_particle_group_count(grupo)

Argumento Descripción
grupo El índice de grupo (ID) del grupo de partículas a contar.

Devuelve:

Real

Ejemplo:

gp_num = physics_particle_group_count(group1);

El código anterior obtendrá el número de partículas utilizadas para hacer el grupo indexado en la variable "grupo1" y almacenará el valor en una variable.
