physics_particle_set_radius

Con esta función se puede establecer el radio (en píxeles) de las partículas de una simulación física. Esta función tiene un alcance global, en el sentido de que cambiará el radio no sólo para las nuevas partículas creadas después del cambio, sino también para las ya presentes en la simulación.

Sintaxis:

physics_particle_set_radius(radio)

Argumento Descripción
radio El radio (en píxeles) de la fijación de partículas.

Devuelve:

N/A

Ejemplo:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

El código anterior establecerá las propiedades base para todas las partículas de la simulación.
