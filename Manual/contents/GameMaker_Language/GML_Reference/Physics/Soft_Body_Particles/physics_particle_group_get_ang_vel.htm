physics_particle_group_get_ang_vel

Con esta función puedes recuperar la velocidad angular de un grupo de partículas. El valor del grupo es el que se devolvió al crear el grupo de partículas mediante la función physics_particle_group_end(), y la función devolverá un valor que es el valor combinado de las banderas actualmente establecidas.

Sintaxis:

physics_particle_group_get_ang_vel(grupo)

Argumento Descripción
grupo El grupo de partículas a conseguir.

Devuelve:

Real

Ejemplo:

ang_v = physics_particle_group_get_ang_vel(group1);

El código anterior obtendrá la velocidad angular del grupo de partículas indexado en la variable "grupo1" y la almacenará en una variable.
