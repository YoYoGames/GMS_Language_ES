physics_particle_get_group_flags

Con esta función puede recuperar los indicadores de grupo de un grupo de partículas. El valor del grupo es el que se devolvió al crear el grupo de partículas mediante la función physics_particle_group_end(), y la función devolverá un valor que es el valor combinado de las banderas actualmente establecidas.

Sintaxis:

physics_particle_get_group_flags(grupo)

Argumento Descripción
grupo El grupo de partículas a conseguir.

Devuelve:

Real

Ejemplo:

var flags = physics_particle_get_group_flags(group, flags);
if flags != phy_particle_group_flag_solid | phy_particle_group_flag_rigid
{
flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);
}

El código anterior creará una variable para almacenar el valor de las banderas y luego la utilizará para comprobar las banderas del grupo indexado en la variable "grupo". Si no son iguales, el grupo se establece con estas banderas.
