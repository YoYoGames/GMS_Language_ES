physics_particle_get_density

Con esta función puedes averiguar cuál es la densidad de corriente de las partículas en la simulación física (puedes establecer este valor con physics_particle_set_density()).

Sintaxis:

physics_particle_get_damping()

Devuelve:

Real

Ejemplo:

si (physics_particle_get_density() < 1)
{
physics_particle_set_density(physics_particle_get_density() + 0.01);
}

El código anterior comprobará el valor de la densidad actual para todas las partículas del sistema y si es menor que 1 entonces le añadirá 0,01.
