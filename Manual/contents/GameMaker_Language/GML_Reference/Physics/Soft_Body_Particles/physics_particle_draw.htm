physics_particle_draw

Esta función dibujará un sprite en la posición de todas las partículas que compartan el mismo valor de categoría definido por el usuario, y que tengan banderas que coincidan con las establecidas para la máscara. El valor de la máscara se define de la misma manera que se definen las banderas al crear la partícula (véase la función physics_particle_create() para ver un ejemplo), es decir, utilizando la función "|" para enmascarar los bits apropiados para cada bandera.

NOTA: Esta función dibujará todas las partículas de la simulación física con banderas que coincidan con las de la máscara, ya sean partículas individuales o grupos, si la categoría del usuario es la misma.

La partícula dibujada utilizará los valores de color y alfa establecidos cuando la creaste, y rotará con el momento angular de la partícula. Sin embargo, si deseas escalar el sprite o cambiar cualquier otra de sus propiedades dibujadas, debes utilizar la función physics_particle_draw_ext().

Sintaxis:

physics_particle_draw(typemask, category, sprite, subimg)

Argumento Descripción
máscara de escritura La máscara con la que se comparan las banderas de las partículas.
categoría La categoría definida por el usuario para dibujar (use 0 para dibujar todas las categorías).
sprite El sprite a utilizar para cada partícula.
subimg El image_index del sprite a dibujar.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

El código anterior dibujará un sprite para todas las partículas de la categoría "2" definida por el usuario que tengan las mismas banderas que la máscara dada.
