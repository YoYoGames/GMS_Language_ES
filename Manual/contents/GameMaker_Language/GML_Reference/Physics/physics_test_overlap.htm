physics_test_overlap

Esta función puede utilizarse para comprobar si un cuerpo físico (es decir, el accesorio de una instancia) se solapa, o se solapará, cuando se rote y se coloque en una posición determinada de la sala. el argumento "ángulo" es el es el ángulo de rotación que tiene (o tendrá) la instancia que se llama en la posición que se va a comprobar, y el argumento "obj" puede ser un único id de instancia, y el índice del objeto o el palabras clave todo u otro.

Sintaxis:

physics_test_overlap(xpos, ypos, angle, obj);

Argumento Descripción
xpos La posición x en la habitación para comprobar
ypos La posición y en la sala para comprobar
ángulo El ángulo a comprobar (de la instancia que llama)
obj El objeto a comprobar

Devuelve:

Booleano

Ejemplo:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
alarm[0] = room_speed;
ignited = true;
}

El código anterior comprobará una posición para un solapamiento de la fijación de la física, y si hay uno, establece una variable y una alarma.
