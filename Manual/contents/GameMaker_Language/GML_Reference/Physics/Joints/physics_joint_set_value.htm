physics_joint_set_value

Ciertas propiedades de la articulación pueden cambiarse y establecerse incluso después de la creación de la articulación. Hay una serie de constantes que se pueden utilizar en esta función y que se pueden encontrar aquí: Constantes de Articulación de Física.

Sintaxis:

physics_joint_set_value(joint, field, value)

Argumento Descripción
conjunta El índice de la articulación que desea modificar
campo La constante de la propiedad conjunta que desea modificar
valor El nuevo valor de la propiedad conjunta

Devuelve:

N/A

Ejemplo:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

El código anterior comprueba si el par máximo del motor de las articulaciones es inferior a 2 y, si lo es, lo ajusta a 2.
