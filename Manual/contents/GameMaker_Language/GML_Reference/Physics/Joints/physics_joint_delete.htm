physics_joint_delete

Una vez que dos instancias con representaciones físicas han sido unidas por una junta, ésta puede ser eliminada de nuevo en cualquier momento. Normalmente esto ocurrirá automáticamente cuando una de las dos instancias sea destruida, o cuando la sala termine, pero hay veces que puede desear hacer esto manualmente. En esos casos se utilizaría esta función.

También hay que tener en cuenta que siempre que se destruya una instancia que forme parte de una unión de engranajes, la unión de engranajes debe ser eliminada utilizando esta función antes que cualquiera de las instancias implicadas en la formación del (pero el resto de uniones se borrarán automáticamente), por ejemplo en el evento de destrucción de la instancia.

Sintaxis:

physics_joint_delete(joint)

Argumento Descripción
conjunta El índice de la junta que desea eliminar

Devuelve:

N/A

Ejemplo:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
{
fuerza-reacciónX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
fuerza-reacciónY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
fuerza-reacción = sqrt((fuerza-reacciónX + fuerza-reacciónX) + (fuerza-reacciónY + fuerza-reacciónY));
si fuerza-reacción > 2
{
physics_joint_delete(shipJoint);
shipJoint = -1;
}
}

El código anterior comprueba si la variable "shipJoint" contiene un índice de articulación y, si es así, calcula la fuerza que se aplica a esa articulación utilizando las dos constantes. Finalmente, si la fuerza total es mayor que 2, se elimina la articulación.
