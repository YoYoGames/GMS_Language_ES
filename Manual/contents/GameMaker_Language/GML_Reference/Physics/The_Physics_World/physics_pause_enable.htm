physics_pause_enable

Normalmente, para que una simulación de física funcione, debe ser continua y no puede detenerse e iniciarse, o que las instancias se muevan repentinamente de un lugar a otro de la sala. Sin embargo, hay momentos en los que se necesita pausar la simulación como (por por ejemplo) el dispositivo os ha pausado, y entonces utilizarías esta función. Pone en pausa la simulación si la bandera se establece en true y no se harán más cálculos de física hasta que la bandera se establezca en false de nuevo.

NOTA: Esta opción es especialmente útil si se desea desactivar todas las instancias de una sala, ya que incluso cuando se desactiva un cuerpo físico se sigue calculando y simulando en el mundo de la física.

Sintaxis:

physics_pause_enable(flag)

Argumento Descripción
bandera Se puede establecer como true para pausar la simulación, o false para iniciarla de nuevo.

Devuelve:

N/A

Ejemplo:

si os_is_paused()
{
physics_pause_enable(true);
instance_deactivate_all(true);
instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

El código anterior comprueba si el sistema operativo ha sido pausado y si lo ha sido entonces pausa el mundo de la física, desactiva todo excepto a sí mismo, y luego crea una instancia del menú de pausa.
