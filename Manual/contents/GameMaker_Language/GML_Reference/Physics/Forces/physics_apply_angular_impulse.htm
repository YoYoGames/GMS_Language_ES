physics_apply_angular_impulse

Esta función dará un impulso angular a una instancia de física habilitada. Este impulso establecerá la rotación angular por la cantidad dada, ignorando el par de torsión actual, esencialmente estableciendo la cantidad de "giro" que tiene un accesorio. Si desea aplicar una fuerza angular a una instancia utilizando el par de torsión, entonces debería utilizar la función physics_apply_torque().

Sintaxis:

physics_apply_angular_impulse(impulso)

Argumento Descripción
impulso El impulso (en Newton metros) a aplicar

Devuelve:

N/A

Ejemplo:

if keyboard_check(vk_up)
{
physics_apply_angular_impulse(10);
}

El código anterior dará al dispositivo de física habilitado un impulso angular si se pulsa una tecla.
