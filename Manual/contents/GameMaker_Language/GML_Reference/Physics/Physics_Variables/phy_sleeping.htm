phy_sleeping

Esta variable de sólo lectura devuelve si la instancia está actualmente "dormida" (true) ono (false), Una instancia"dormida" es aquella que no está activamente involucrada en ninguna simulación física. GameMaker Studio 2 pondrá los objetos en reposo para ahorrar ciclos de simulación cuando una instancia esté en reposo y no entre en colisión con otra instancia

Sintaxis:

phy_sleeping;

Devuelve:

Booleano (o indefinido si la instancia no tiene activada la física)

Ejemplo:

si phy_sleeping
{
instance_destroy();
}

El código anterior comprueba si el objeto está siendo simulado activamente o no y si no lo está se destruye.
