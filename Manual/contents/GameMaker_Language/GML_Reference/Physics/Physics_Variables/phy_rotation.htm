phy_rotation

Esta variable se puede utilizar para obtener (o establecer) el ángulo de la fijación de las instancias en grados, de forma similar a establecer u obtener la variable ángulo_de_la_imagen. Sin embargo, tenga en cuenta que en el mundo de la física las rotaciones se calculan de forma opuesta al mundo del juego normal de GameMaker Studio 2, lo que significa que las funciones vectoriales como punto_dirección()deben tienen sus valores de retorno modificados (simplemente haciendo positivo a negativo debería resolver esto).

Sintaxis:

phy_rotation;

Devuelve:

Real (valor de punto flotante de precisión única, o indefinido si la instancia no tiene activada la física)

Ejemplo:

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
phy_rotation += sqr(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

El código anterior comprueba la velocidad lineal y si alguno de los vectores no es 0, entonces calcula la velocidad real y la utiliza para establecer la rotación.
