phy_active

Esta variable controla si la instancia está actualmente "activa" o no. Si se establece en false, la instancia no participará en el mundo de la física, y si se establece en true, volverá a participar. Por favor, Tenga en cuenta que esto no es lo mismo que desactivar la instancia, ya que la instancia sigue siendo visible en la pantalla y todavía se puede cambiar a través de código, más bien esta función sólo impide que participe en la simulación física

Sintaxis:

phy_active;

Devuelve:

Booleano (o indefinido si la instancia no tiene activada la física)

Ejemplo:

if keyboard_check_pressed(ord "P")
{
global.Pause = !global.Pause
with (obj_Parent)
{
phy_active = !global.Pause;
}
}

El código anterior detectará una pulsación de la letra "P" y entonces cambiará la variable global "Pausa" de verdadero a falso y viceversa . Esta variable se utiliza entonces para establecer si la física está activa o no en los hijos del objeto indexado en la variable "obj_Parent".
