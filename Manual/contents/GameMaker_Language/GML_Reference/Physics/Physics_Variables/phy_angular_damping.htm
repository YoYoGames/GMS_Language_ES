phy_angular_damping

Esta variable se puede utilizar para establecer la amortiguación angular de la instancia, o se puede utilizar para obtener la amortiguación angular actual. La amortiguación es la cantidad de "resistencia" a la rotación angular que tiene la instancia habilitada por la física, con un valor menor permite que la instancia rote más rápido después de una colisión (por ejemplo) y un valor más alto hace que requiera un empuje más fuerte y rote más lentamente

Sintaxis:

phy_angular_damping;

Devuelve:

Real (valor de punto flotante de precisión única, o indefinido si la instancia no tiene activada la física)

Ejemplo:

si place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_angular_damping = 10;
}
else
{
phy_angular_damping = 3;
}

El código anterior comprobará si hay una colisión entre la instancia que llama y las instancias de "obj_Water" y cambiará la amortiguación angular en consecuencia.
