phy_kinematic

Esta variable de sólo lectura devolverá true si la instancia está clasificada como objeto cinemático, o false si no lo está. Una instancia cinemática es aquella que tiene una masa infinita (una densidad de 0) pero que puede moverse. Por lo tanto, para hacer que una instancia cinemática, primero hay que crear una instancia estática y luego establecer una o más de las variables de la instancia relacionadas con el movimiento (por ejemplo phy_speed_x, phy_speed_yo phy_velocidad_angular)

Sintaxis:

phy_kinematic;

Devuelve:

Booleano (o indefinido si la instancia no tiene activada la física)

Ejemplo:

if !phy_kinematic
{
phy_speed_x = 5;
}

El código anterior comprueba si la instancia es cinemática y si no lo es, establece la velocidad horizontal en 5.
