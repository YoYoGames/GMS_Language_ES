phy_dynamic

Una instancia dinámica es aquella que está totalmente simulada dentro del mundo de la física y esta variable de sólo lectura devolverá true si la instancia que se está comprobando está totalmente simulada o false si no lo está

Sintaxis:

phy_dynamic;

Devuelve:

Booleano (o indefinido si la instancia no tiene activada la física)

Ejemplo:

si other.phy_dynamic
{
con (otro)
{
var dir;
dir = point_direction(x, y, other.x, other.y);
physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
}
}

El código anterior crea una nueva instancia y luego la define como una "bala" en el mundo de la física antes de darle un impulso masivo a lo largo del eje y.
