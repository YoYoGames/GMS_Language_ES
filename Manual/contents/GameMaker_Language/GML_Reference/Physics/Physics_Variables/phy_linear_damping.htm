phy_linear_damping

Esta variable se puede utilizar para establecer la amortiguación lineal de la instancia, o se puede utilizar para obtener la amortiguación lineal actual. La amortiguación es la cantidad de "resistencia" al movimiento hacia delante que tiene la instancia habilitada para la física, con un valor menor un valor más bajo permite que la instancia se mueva y acelere más rápido y un valor más alto hace que requiera un empuje más fuerte

Sintaxis:

phy_linear_damping;

Devuelve:

Real (valor de punto flotante de precisión única, o indefinido si la instancia no tiene activada la física)

Ejemplo:

si place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_linear_damping = 10;
}
else
{
phy_linear_damping = 3;
}

El código anterior comprobará si hay una colisión entre la instancia que llama y las instancias de "obj_Water" y cambiará la amortiguación lineal en consecuencia.
