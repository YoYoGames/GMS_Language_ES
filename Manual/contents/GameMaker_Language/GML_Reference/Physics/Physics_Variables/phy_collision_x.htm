phy_collision_x

Esta matriz de sólo lectura devuelve la posición x de todos los puntos detectados en una colisión entre dos instancias habilitadas para la física.

NOTA: Esta variable sólo está disponible en el evento de colisión de una instancia con física habilitada.

Sintaxis:

phy_collision_x;

Devuelve:

Real (valor de punto flotante de precisión única, o indefinido si la instancia no tiene activada la física)

Ejemplo:

for(var i = 0; i < phy_collision_points; i += 1;)
{
part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5);
}

El código anterior crea partículas en todos los puntos definidos de una colisión entre dos instancias habilitadas para la física.
