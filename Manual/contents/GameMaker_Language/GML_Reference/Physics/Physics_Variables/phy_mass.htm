phy_mass

Esta variable de sólo lectura devuelve la masa de la instancia en kilogramos. Este valor se calcula automáticamente en función de la superficie de los artefactos asignados y sus valores de densidad, pero puede modificarse mediante la función physics_mass_properties().

Sintaxis:

phy_mass;

Devuelve:

Real (valor de punto flotante de precisión única, o indefinido si la instancia no tiene activada la física)

Ejemplo:

if (phy_mass < other.phy_mass)
{
instance_destroy();
}

El código anterior es del evento de colisión de la instancia con otra y compara la masa de cada instancia y destruye la que tiene menos masa.
