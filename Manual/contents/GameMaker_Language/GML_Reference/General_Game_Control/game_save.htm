guardar_juego

Esta es una función heredada que puede ser usada para guardar el estado actual del juego, y no se recomienda su uso. Utiliza las funciones de Archivo para crear un sistema de guardado personalizado en el que sólo guardes y cargar datos específicos del juego.

Esta función guardará el estado del juego tal y como está, pero no guardará ningún recurso dinámico que se esté utilizando en ese momento, como estructuras de datos, superficies, activos añadidos en tiempo de ejecución , etc. Si se carga un archivo de este tipo, cualquier actualización del juego que se que se hayan hecho después de guardar el juego no serán visibles, ya que se restaurará la versión del juego que se utilizó para crear el archivo guardado.

Tenga en cuenta que los archivos guardados creados con esta función pueden no ser compatibles con las versiones actualizadas Tiempo de ejecución por lo que reemplazar esta función con un nuevo sistema de guardado es esencial para asegurar la compatibilidad con futuras actualizaciones de GameMaker, y por las razones mencionadas anteriormente, las actualizaciones hechas a su propio juego.

Sintaxis:

game_save(filename);

Argumento Descripción
nombre de archivo El nombre del archivo para guardar el juego.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("S"))
{
global.Saved = true;
game_save("Save.dat");
}

El código anterior establecerá una variable global a true y luego guardará el juego en el archivo "Save.dat" cuando se presione la tecla "S".
