Estas funciones y variables están todas relacionadas con la funcionalidad específica del juego, como cargar/guardar o reiniciar, etc.

Las siguientes variables integradas de alcance global son útiles para depurar su juego - pueden ser usadas como la salida a un archivo de registro en el caso de problemas, por ejemplo - así como para mostrar el ID de compilación actual al usuario. así como para mostrar el ID de compilación actual, etc... al usuario:

  • ID_de_juego
  • game_save_id
  • nombre_del_juego
  • nombre_del_proyecto_del_juego

Para ayudarte a crear y mantener las puntuaciones máximas de tus juegos, GameMaker Studio 2 crea una matriz global de puntuaciones máximas de 10 lugares a la que puedes acceder, añadir y cambiar para crear tus propias listas de puntuaciones máximas personalizadas sin demasiadas complicaciones. personalizadas sin demasiadas complicaciones. Esta funcionalidad es local para el juego y funciona en todas las plataformas, por lo que puedes almacenar y mantener fácilmente una tabla de puntuaciones máximas básica utilizando estas funciones junto con las funciones de manejo de archivos.

Existen las siguientes funciones que se ocupan de la lista interna de puntuaciones altas.

  • highscore_add
  • nombre_de_la_punta_alta
  • valor_de_puntuación_alta
  • highscore_clear

GameMaker Studio 2 también incluye una función para dibujar las puntuaciones máximas en la pantalla como una lista, sin embargo, es muy básico y se recomienda que se utiliza para la depuración más que cualquier otra cosa como hacer su propia tabla de puntuación alta es es fácil y le dará mucho más control sobre la forma en que se muestra:

  • draw_highscore

Por último, tenemos una función que permite asignar un sprite al cursor del ratón:
