game_save_id

Esta variable de sólo lectura devolverá el ID de la ruta completa del directorio que utiliza tu juego para guardar los archivos. Este directorio puede o no ser visible para otras aplicaciones, dependiendo de la plataforma, y en el objetivo HTML5 devolverá una cadena vacía.

NOTA: Esta variable es meramente informativa y no debe utilizarse para obtener una ruta de acceso a un archivo o algo similar, ya que no funcionará correctamente en todas las plataformas.

Sintaxis:

game_save_id

Devuelve:

Cadena

Ejemplo:

save_dir = game_save_id;

Esto almacenará el directorio para guardar los archivos en una variable para su uso futuro.
