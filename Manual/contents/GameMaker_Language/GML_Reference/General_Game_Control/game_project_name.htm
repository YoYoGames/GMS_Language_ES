nombre_del_proyecto_del_juego

Esta variable de sólo lectura devuelve el nombre de pantalla de tu juego para la plataforma de destino en un formato "apto para guardar" para la plataforma de destino. Si el nombre del juego contiene caracteres no permitidos para un nombre de archivo, se sustituirán automáticamente por "_". serán sustituidos automáticamente por "_". El nombre de visualización puede ser configurado en las Opciones de Juego. Tenga en cuenta que, como no hay restricciones en los nombres de archivo para HTML5, esta cadena será probablemente la misma que la devuelta por nombre_de_display_del_juego.

Sintaxis:

nombre_del_proyecto_del_juego

Devuelve:

Cadena

Ejemplo:

var file = nombre_del_proyecto_de_juego + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

El código anterior obtiene el nombre del proyecto y lo utiliza para abrir (o crear) un archivo ini en el que se escribe.
