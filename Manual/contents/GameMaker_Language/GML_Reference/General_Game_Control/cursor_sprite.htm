cursor_sprite

Al establecer esta variable se le indicará a GameMaker Studio 2 que utilice el sprite designado como cursor (básicamente, ajustándolo a la posición actual del ratón x/y en cada paso). El valor por defecto es -1, que es ningún sprite para el cursor, pero puedes asignar cualquier índice de sprite de los activos del juego o que haya sido importado desde un recurso externo. Tenga en cuenta que no hay manera de controlar la velocidad de la animación o image_index, por lo que si el sprite tiene sub-imágenes, estas serán cicladas a la misma velocidad que la de la sala. Para eliminar el sprite del cursor, puedes volver a establecer esta variable a -1.

También hay que tener en cuenta que esta variable no sustituye al cursor de la ventana del juego, y que se seguirá dibujando de forma normal. Para evitar esto puedes utilizar la función window_set_cursor() y establecerla a la constante cr_none que hará invisible el cursor estándar.

Sintaxis:

cursor_sprite;

Devuelve:

Real

Ejemplo:

cursor_sprite = spr_CustomCursor;

El código anterior establecerá el sprite indexado en la variable "spr_CustomCursor" para que sea el sprite del cursor del juego.
