game_load_buffer

Con esta función puedes cargar un estado de juego que haya sido guardado previamente. El juego se carga desde un buffer "grow" creado previamente (ver Buffers) y el buffer debe haber tenido un estado de juego guardado en él utilizando game_save_buffer() función.

Sintaxis:

game_load_buffer(buffer);

Argumento Descripción
buffer El identificador del buffer desde el que se va a cargar.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("L"))
{
if global.Checkpoint game_load_buffer(save_buff);
}

El código anterior cargará un estado de juego previamente guardado desde el buffer indexado en la variable "save_buff", sólo si la variable global es verdadera, cuando el jugador presione la tecla "L".
