carga_de_juegos

Con esta función se puede cargar una partida previamente guardada mediante game_save(). Tenga en cuenta que restaurará la versión del juego que se utilizó para crear el guardado, por lo que cualquier actualización realizada después no será visibles. Para más información, lee la página de game_save().

Sintaxis:

game_load(filename);

Argumento Descripción
nombre de archivo El nombre del archivo a cargar.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("L"))
{
if global.Save game_load("Save.dat");
}

El código anterior cargará una partida previamente guardada si la variable global es verdadera cuando el jugador pulsa la tecla "L".
