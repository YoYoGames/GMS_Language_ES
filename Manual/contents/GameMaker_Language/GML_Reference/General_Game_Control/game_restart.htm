reinicio_del_juego

Con esta función puedes reiniciar el juego. Esto es esencialmente lo mismo que ejecutar el juego por primera vez y por lo tanto el Evento de Inicio de Juego se activará, así como el Evento de Fin de Juego. Sin embargo, debe tenerse en cuenta que ciertas cosas no se reiniciarán cuando se llame a esta función:

  • Las variables globales no se reiniciarán a menos que se codifiquen explícitamente como tales - por ejemplo, la variable global incorporada puntuación no comenzará a cero después de un reinicio del juego si ya ha sido modificada en el juego.
  • El estado de la GPU no se modificará (por lo que si has establecido el color de dibujo o el alfa, por ejemplo, se mantendrá en el valor modificado).
  • La velocidad del juego se mantendrá en lo que hayas establecido en tu código de juego (si lo has cambiado, este cambio se perpetuará).
  • Cualquier activo del Navegador de Activos que haya sido cambiado en tiempo de ejecución dentro del juego - por ejemplo si cambias el origen de un recurso de sprite o cambias la posición de un recurso de ruta - no será restablecido.
  • Los recursos dinámicos como los buffers, las superficies, las estructuras de datos o los sprites importados tampoco se limpiarán o eliminarán (aunque puede perder las referencias a ellos, así que tenga cuidado cuando utilice esta función de usar referencias globales para el recurso dinámico, o limpiarlas antes de llamar a la función).

Sintaxis:

game_restart();

Devuelve:

N/A

Ejemplo:

si keyboard_check_pressed(ord("R")) game_restart();

Esto reiniciaría el juego cuando el jugador pulsara la tecla "R".
