Constantes desombreado

Además de las funciones y constantes de sombreado definidas en las páginas de referencia de OpenGL ES Shading Language (GLSL ES), también hay una serie de de constantes de sombreado disponibles que son exclusivas de GameMaker Studio 2.

Las siguientes constantes de matriz de visualización pueden utilizarse como índices de matriz cuando se utiliza la constante de matriz de sombreado gm_Matrices:

Constante Descripción
MATRIX_VIEW Esta constante de índice de matriz mantiene el índice de la matriz de la vista actual. El índice devuelto se utilizará como un valor de matriz cuando se llame a la constante gm_Matrices dentro del código del shader.
PROYECCIÓN_DE_MATRIZ Esta constante de índice de matriz mantiene el índice de la matriz de proyección actual. El índice devuelto se utilizará como un valor de matriz cuando se llame a la constante gm_Matrices dentro del código del shader.
MATRIX_WORLD Esta constante de índice de matriz mantiene el índice de la matriz mundial actual. Esto puede ser usado para cosas como la iluminación si tienes información de luz en el espacio del mundo. El índice devuelto se utilizaría como un valor de matriz cuando se llame a la constante gm_Matrices dentro del código del shader.
VISTA_DEL_MUNDO_DE_LA_MATRIZ Esta constante de índice de matriz mantiene el índice del resultado de las matrices de mundo y vista multiplicadas juntas. Esto se utiliza a menudo para cosas como la niebla. El índice devuelto será utilizado como un valor de matriz cuando se llame a la constante gm_Matrices dentro del código del shader.
MATRIZ_VISTA_MUNDO_PROYECCIÓN Esta constante de índice de matriz mantiene el índice del resultado de las matrices de mundo, vista y proyección multiplicadas juntas. Esta es la matriz de transformación normal utilizada para las posiciones de los vértices. El índice devuelto se utilizará como un valor de matriz cuando llamar a la constante gm_Matrices dentro del código del shader.
MATRIX_MAX Esto no es un índice de matriz, sino que devuelve el tamaño de la matriz en el sombreador de vértices.
MAX_VS_LIGHTS Esto no es un índice de matriz, sino que devuelve el número de las luces en el sombreador de vértices.

La siguiente constante también está disponible para la iluminación:

Constante Descripción
MAX_VS_LIGHTS El número máximo de luces puntuales y direccionales disponibles en el shader

Los siguientes uniformes y constantes de matriz predefinidos pueden utilizarse en su sombreador para acceder a los valores específicos de GameMaker Studio 2:

Constante Descripción
gm_Matrices[matriz] Esta constante de matriz devuelve una matriz de transformación y es uno de los uniformes predefinidos disponibles que GameMaker Studio 2 crea para que usted pueda utilizar dentro del editor de código de sombreado. El índice de la matriz se elige de una de las constantes enumeradas anteriormente.
gm_BaseTexture Se trata de una constante de muestreo 2D que devuelve la textura del objeto actual, tal y como lo establece GameMaker Studio 2. Así que sería el sprite actual, la superficie o la textura que normalmente se utilizaría al dibujar sin que se llame al shader.
gm_LightingEnabled Esto se puede utilizar para obtener o establecer la iluminación de GameMaker Studio 2 cuando se utiliza el 3D.
gm_FogStart Esto se puede utilizar para obtener la distancia en la que los polígonos comienzan a mezclarse con el color de la niebla.
gm_RcpFogRange Esto se puede utilizar para obtener la distancia a la que la niebla es máxima y ya no se puede ver nada.
gm_PS_FogEnabled Esto devolverá true o false si la GPU tiene activada la niebla de píxeles o no.
gm_FogColour Esto se puede utilizar para obtener el color de la niebla utilizado por GameMaker Studio 2.
gm_VS_FogEnabled Esto devolverá true o false si la GPU tiene activada la niebla de vértices o no.
gm_AlphaTestEnabled Esto puede ser usado para obtener pruebas alfa en el shader. Ver gpu_set_alphatestenable() para más información sobre las pruebas alfa.
gm_AlphaRefValue Se puede utilizar para obtener el valor de referencia de la prueba alfa actual. Consulte gpu_set_alphatestref() para más información sobre la referencia de prueba alfa.
