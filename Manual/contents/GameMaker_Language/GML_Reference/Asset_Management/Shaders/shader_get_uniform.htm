shader_get_uniform

Dado que no se puede cambiar el valor de una constante del sombreador dentro del propio sombreador, hay que establecerla antes de llamar al sombreador utilizando una de las funciones de establecimiento de uniformes disponibles. Sin embargo, para poder hacerlo, primero debes llamar a esta función para obtener el "handle" de la constante del sombreador que quieres cambiar.

NOTA: Aunque un shader se compone de dos programas discretos (vértice y fragmento) esta función no diferenciará entre los dos y devolverá el handle de la constante del shader de cualquiera de ellos.

Sintaxis:

shader_get_uniform(shader, uniform);

Argumento Descripción
sombreador El índice del shader a utilizar.
uniforme La constante del sombreador para obtener el manejador de (una cadena).

Devuelve:

Real (manija uniforme)

Ejemplo:

shader_params= shader_get_uniform(shd_glass, "u_vRefractColour");

El código anterior obtendrá el control de la constante del shader "u_vRefractColour".
