shader_set_uniform_f_array

Con esta función puedes establecer una constante de sombreado para mantener un array de valores. Previamente debes haber obtenido el "handle" de la constante utilizando la función shader_get_uniform()y tendrás que haber inicializado previamente el array.

NOTA: Todos los uniformes deben establecerse después de llamar a la función shader_set()y antes de llamar a la función shader_reset().

Sintaxis:

shader_set_uniform_f_array(handle, array);

Argumento Descripción
manija El manejador de la constante del sombreador a establecer.
matriz Una matriz de valores de punto flotante previamente inicializada.

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
tex_array[0] = 0.5;
tex_array[1] = 0.1;
tex_array[2] = 0.25;
shader_params= shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f_array(shader_params, tex_array);
draw_self();
shader_reset();

El código anterior obtendrá el control de la constante del sombreador "u_vParams" y luego establecerá esa constante en la matriz dada.
