shader_set

Con esta función puedes establecer el objetivo de dibujo al shader dado y todo el dibujo posterior se hará usando eso. Puedes terminar el uso del sh ader con la función shader_reset().

NOTA: Sólo se puede establecer un sombreador en el Evento de Dibujo de una instancia.

Sintaxis:

shader_set(shader);

Argumento Descripción
sombreador El índice del shader a utilizar.

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
draw_self();
shader_reset();

El código anterior establecerá un shader top que se utilizará para dibujar, y luego dibujará el sprite actual utilizado para la instancia que lo utiliza.
