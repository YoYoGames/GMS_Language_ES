shader_set_uniform_matrix_array

Esta función restablece el objetivo de dibujo y debe ser llamada cuando ya no se desea utilizar el sombreador actual (establecido con shader_set()).

Sintaxis:

shader_reset();

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
draw_self();
shader_reset();

El código anterior establecerá un shader top que se utilizará para el dibujo, y luego dibujará el sprite actual utilizado para la instancia que lo utiliza.
