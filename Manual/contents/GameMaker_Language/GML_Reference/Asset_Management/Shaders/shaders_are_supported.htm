shaders_are_supported

Esta función hará una comprobación para ver si la plataforma de destino elegida soporta shaders, devolviendo true si lo hacen, y false si no lo hacen. Es importante tener en cuenta que en Android, si el proyecto no tiene ningún recurso de sombreado definido, entonces la función siempre devolverá false, independientemente de si el dispositivo soporta sombreadores o no.

Sintaxis:

shaders_are_supported();

Devuelve:

Booleano

Ejemplo:

global.GFX = false;
if shaders_are_supported()
{
if shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)
{
global.GFX = true;
}
}

El código anterior establecerá una variable global a false, y luego si la plataforma soporta shaders y los dos shaders que se están comprobando han compilado correctamente, se establecerá a true.
