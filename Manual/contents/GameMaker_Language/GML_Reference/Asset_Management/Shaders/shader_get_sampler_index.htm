shader_get_sampler_index

Dado que no se puede cambiar el valor de un muestreador de sombreado dentro del propio sombreador, hay que establecerlo antes de llamar al sombreador utilizando una de las funciones de ajuste de uniformes disponibles. Sin embargo, para poder hacer esto, primero debe llamar primero a esta función para obtener el "handle" del muestreador del sombreador que querrá establecer.

NOTA: Aunque un shader se compone de dos programas discretos (vértice y fragmento) esta función no diferenciará entre los dos y devolverá el handle de la muestra del shader de cualquiera de ellos.

Sintaxis:

shader_get_sampler_index(shader, uniforme);

Argumento Descripción
sombreador El índice del shader a utilizar.
uniforme El muestreador de sombreado del que se obtendrá el control (una cadena).

Devuelve:

Real

Ejemplo:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set( GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

El código anterior obtendrá el manejador del muestreador dentro del shader indexado como "GlassShader" y luego establecerá esa constante del shader a la textura del sprite dada.
