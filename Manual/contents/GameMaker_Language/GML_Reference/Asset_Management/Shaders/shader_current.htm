shader_actualidad

Esta función devolverá el valor del ID del índice del sombreador que se está utilizando actualmente para el renderizado, o devolverá -1 si no se está utilizando ningún sombreador.

Sintaxis:

shader_current();

Devuelve:

Real (valor de identificación del shader, o -1 si no hay shader)

Ejemplo:

if(shader_current() == -1)
{
shader_set(sh_warp)
}

El código anterior comprobará cuál es el shader actual y si devuelve -1 (no se está utilizando ningún shader ) entonces se establece un shader.
