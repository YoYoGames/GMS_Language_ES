shader_set_uniform_matrix

Con esta función puedes establecer el valor (o los valores) de una constante del sombreador a la matriz de transformación actual (tal y como se establece utilizando las Funciones de Matriz). Para ello debe haber obtenido previamente el "handle" de la constante utilizando la función shader_get_uniform().

NOTA: Todos los uniformes deben establecerse después de llamar a la función shader_set()y antes de llamar a la función shader_reset().

Sintaxis:

shader_set_uniform_matrix(handle);

Argumento Descripción
manija El manejador de la constante del sombreador a establecer.

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
shader_matrix= shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);
draw_self();
shader_reset();

El código anterior obtendrá el control de la constante del shader "u_vMatrix" y luego establecerá esa constante a la matriz de transformación actual.
