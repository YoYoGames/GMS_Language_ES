shader_is_compiled

Esta función comprobará un sombreador en tiempo de ejecución para asegurarse de que ha sido compilado con éxito. Si lo ha hecho, la función devuelve true, de lo contrario devuelve false. Esta función debe ser utilizar al inicio del juego para asegurarse de que la plataforma que ejecuta el juego ha compilado correctamente los sombreadores utilizados (en particular en Windows, donde algunos ordenadores pueden estar utilizando DX9 con el nivel de sombreado 2.0 y no una versión posterior que utiliza el nivel de sombreado 3.0).

Si tu shader NO ha sido compilado y llamas a shader_set() el juego se bloqueará, por lo que vale la pena tener algún tipo de comprobación siempre que se utilice cualquier cosa que no sea un simple sombreador GLSL ES.

Sintaxis:

shader_is_compiled(shader);

Argumento Descripción
sombreador El sombreado a comprobar.

Devuelve:

N/A

Ejemplo:

global.GFX = false;
if(shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
global.GFX = true;
}

El código anterior establecerá una variable global a false, y luego si los dos shaders que se están comprobando han compilado correctamente, se establecerá a true.
