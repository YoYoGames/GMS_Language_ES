shader_set_uniform_i

Con esta función puedes establecer el valor (o los valores) de una constante del shader. Para ello debes haber obtenido previamente el "handle" de la constante mediante la función shader_get_uniform()y tendrás que saber qué tipo de constante es para pasarle el número correcto de valores enteros, es decir: si tienes una vec2 tendrás que pasar dos valores a la función.

NOTA: Todos los uniformes deben ser establecidos después de llamar a la función shader_set()y antes de llamar a shader_reset().

Sintaxis:

shader_set_uniform_i(handle, value1 [, value2, value3, value4]);

Argumento Descripción
manija El manejador de la constante del sombreador a establecer.
valor1 ... valor4 El valor entero (o los valores) para establecer la constante del sombreador.

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
shader_params= shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

El código anterior obtendrá el control de la constante del sombreador "u_vParams" (un vec3), y luego establecerá esa constante a los valores enteros dados.
