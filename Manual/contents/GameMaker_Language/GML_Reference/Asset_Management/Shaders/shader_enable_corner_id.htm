shader_enable_corner_id

Con esta función se puede establecer un estado global para todos los sombreadores que se utilicen en el que, cuando se activa, el sombreador "roba" 2 bits de los valores de color de entrada. El primero es del bit inferior del valor de color rojo, y el segundo es del bit inferior del valor del color azul. Estos valores pueden ser recuperados en el sombreador para saber con qué vértice se está tratando (es decir, qué esquina).

Sintaxis:

shader_enable_corner_id(enable);

Argumento Descripción
activar Activar(true) o desactivar(false) esta función.

Devuelve:

N/A

Ejemplo:

shader_enable_corner_id(true);

El código anterior habilitará el uso de bits de color para el id de esquina para todos los shaders.
