shader_get_name

Con esta función puedes recuperar el nombre de un recurso de sombreado. Usted proporciona el valor de ID único para el sombreador para obtener el nombre de y la función devolverá el nombre del recurso como una cadena.

Sintaxis:

shader_get_name(shader);

Argumento Descripción
sombreador El índice del sombreador para obtener el nombre.

Devuelve:

Cadena

Ejemplo:

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

El código anterior obtendrá el nombre del shader dado y lo dibujará en la pantalla.
