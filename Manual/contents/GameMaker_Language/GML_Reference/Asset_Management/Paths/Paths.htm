Caminos

GameMaker Studio 2 cuenta con una amplia colección de funciones y variables incorporadas para utilizar las rutas dentro de su juego. Esta sección trata de las básicas que se utilizan cuando se trata de caminos que han sido previamente creados en el Path Editor, pero existen otras funciones que te permiten crear tus propias rutas usando código, y manipularlas de varias maneras.

Existen las siguientes funciones básicas que se ocupan de las rutas:

  • inicio_de_la_caminata
  • camino_final
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • ruta_obtener_número
  • ruta_obtener_velocidad_de_punto
  • path_get_point_x
  • path_get_point_y
  • ruta_obtener_precisión
  • ruta_obtener_velocidad
  • ruta_obtener_x
  • path_get_y

Para ayudar a la depuración de las rutas, tiene una función para dibujarla en la pantalla también:

  • dibujar_trayectoria

Cada instancia de un objeto también tiene una serie de variables incorporadas diseñadas para facilitar el uso de las rutas, y estas variables se enumeran en la siguiente sección:

  • Variables de trayectoria

Por último, es posible manipular el contenido de una ruta, así como crear nuevas rutas cuando el juego se está ejecutando. Las funciones que permiten estas cosas y más están disponibles desde la siguiente sección:

  • Manipulación de la trayectoria
