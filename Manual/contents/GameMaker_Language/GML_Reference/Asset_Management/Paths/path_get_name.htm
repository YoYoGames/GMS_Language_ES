path_get_name

Esta función devolverá el nombre de la ruta a la que se hace referencia como una cadena. El nombre es el que se haya utilizado para definir la ruta en el editor o (si la ruta se ha creado a través de una función de código) devolverá una cadena con el formato " _newpathXX" donde "XX" es el número de la ruta generada, empezando por 0 e incrementando en uno cada vez que se crea una nueva ruta. Tenga en cuenta que esto es sólo una y no puede utilizarse para referirse a la ruta directamente - para ello necesitaría el ID del índice de la ruta. Sin embargo, puede utilizar esta cadena para obtener el ID del índice de la ruta utilizando la cadena devuelta junto con con la función asset_get_index().

Sintaxis:

path_get_name(index);

Argumento Descripción
índice El índice de la ruta a comprobar.

Devuelve:

Cadena

Ejemplo:

nombre_de_la_ruta = nombre_de_la_ruta(matriz_de_la_ruta[0]);

Esto establecerá "nombre_de_la_pista" al nombre de la ruta indexada en el array dado en la posición 0.
