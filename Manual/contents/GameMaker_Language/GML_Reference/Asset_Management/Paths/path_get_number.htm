ruta_obtener_número

Esta función puede utilizarse para devolver el número de puntos de un camino.

Sintaxis:

path_get_number(index);

Argumento Descripción
índice El índice de la ruta a comprobar.

Devuelve:

Real

Ejemplo:

si (path_get_number(pth_AI) > 1)
{
path_start( pth_AI, 4, 3, 0);
}

El código anterior comprueba si un camino tiene más de un punto en él y, si es así, inicia el movimiento de la instancia a lo largo de ese camino.
