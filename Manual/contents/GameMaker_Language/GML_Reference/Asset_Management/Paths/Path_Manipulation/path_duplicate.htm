ruta_duplicada

Esta función toma una ruta y la copia en una nueva ruta. La nueva ruta se crea en el proceso, y el índice se devuelve para ser utilizado en todas las llamadas posteriores para utilizar esta nueva ruta.

Sintaxis:

ruta_duplicada(índice);

Argumento Descripción
índice El índice de la ruta existente a duplicar.

Devuelve:

Real

Ejemplo:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

El código anterior elige uno de los cuatro recursos de ruta y lo duplica, almacenando el índice de la nueva ruta en la variable "mypath".
