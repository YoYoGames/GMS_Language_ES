ruta_suprimir

Puede utilizar este código para eliminar una ruta de la memoria. Si esta ruta ha sido creada dinámicamente usando path_add()la variable que contiene el índice de la ruta ya no será válida para acceder a la ruta porque ya no existe. existe, y si la ruta fue creada usando el Editor de rutas, ya no se puede acceder a esa ruta en todo el juego, ya que la estás borrando permanentemente.

Sintaxis:

path_delete(index);

Argumento Descripción
índice El índice de la ruta a eliminar.

Devuelve:

N/A

Ejemplo:

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
{
path_assign(mypath, t_path);
}
path_delete(t_path);

El código anterior creará una ruta y almacenará su índice en una variable local. Esta ruta se utiliza entonces para almacenar una mp_grid_path() ruta generada que, si tiene éxito en encontrar su camino hacia el objetivo, se asigna a la ruta indexada en "mypath". Por último, se elimina "t_path".
