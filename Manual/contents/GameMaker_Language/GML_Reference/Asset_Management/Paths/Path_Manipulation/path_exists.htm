ruta_existe

Esta función devuelve si existe o no un camino con el índice dado. Tenga en cuenta que si comprueba la existencia de un camino a través de una variable que aún no ha sido declarada, esto arrojará un error.

Sintaxis:

ruta_existe(índice);

Argumento Descripción
índice El índice de la ruta a comprobar.

Devuelve:

Booleano

Ejemplo:

if (path_exists(my_path))
{
path_delete(my_path);
}

Este código comprueba si la variable dada almacena un índice de ruta y si lo hace, entonces la ruta se borra de la memoria.
