punto_de_cambio_de_la_senda

Con esta función puedes cambiar la posición y/o el factor de velocidad de cualquier punto previamente definido para una trayectoria (la trayectoria puede haber sido creada en el editor de trayectorias o a través de código usando path_add()). Si se utiliza en una del Navegador de Activos, tenga en cuenta que la función cambiará el activo actual, y por lo tanto afectará permanentemente a la forma en que el camino es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear primero una copia de la ruta, y luego llamar a esta función en el activo duplicado (no te olvides de llamar a path_delete() en el activo cuando ya no sea necesario).

Sintaxis:

camino_cambio_punto(índice, n, x, y, velocidad);

Argumento Descripción
índice El índice de la trayectoria para cambiar un punto de.
n El punto de definición para cambiar los atributos de.
x La nueva coordenada x (relativa a la trayectoria) del punto.
y La nueva coordenada y (relativa a la trayectoria) del punto.
velocidad El nuevo factor de velocidad del punto.

Devuelve:

N/A

Ejemplo:

for (var i = 0; i < path_get_number(path0); ++i;)
{
var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
path_change_point(pth_Patrol, i, px, py, 100);
}

El código anterior recorre todos los puntos de la ruta indexada como "ruta0" y recoloca todos los puntos en una posición aleatoria dentro de un área de 128x128 píxeles.
