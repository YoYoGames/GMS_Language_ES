ruta_borrar_punto

Con esta función puedes eliminar un punto de la ruta especificada. Los puntos de la ruta están numerados desde 0, por lo que el primer punto es siempre el punto 0 y el último punto de la ruta será camino_obtener_número() - 1. Si usted Si llama a esta función en un activo de la ruta, la eliminación del punto será permanente y todas las instancias asignadas a la ruta en el futuro carecerán del punto eliminado. Si esto no es lo que necesita, entonces debería utilizar una función como path_duplicate() para crear primero una copia de la ruta, y luego llamar a esta función en el activo duplicado (no olvide llamar a path_delete() en el activo cuando ya no sea necesario).

Sintaxis:

camino_borrar_punto(ind, n);

Argumento Descripción
índice El índice de la ruta de la que se va a borrar un punto.
n El punto definitorio a eliminar.

Devuelve:

N/A

Ejemplo:

var _col = true;
while (_col)
{
var _num = path_get_number(mypath) - 1;
if (_num > 2)
{
var _x1 = path_get_point_x(my_path, _num);
var _y1 = path_get_point_y(my_path, _num);
var _x2 = path_get_point_x(my_path, _num -1);
var _y2 = path_get_point_y(my_path, _num -1);
if collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false)
{
path_delete_point(mypath, _num);
_num -= 1;
}
else _col = false;
}
else _col = false;
}

El código anterior genera un bucle que comprueba los puntos de una trayectoria hacia atrás desde el final para buscar una colisión con el objeto dado. Si se encuentra una colisión, se elimina un punto de la ruta, hasta que no haya colisión, o la ruta tenga sólo 2 puntos.
