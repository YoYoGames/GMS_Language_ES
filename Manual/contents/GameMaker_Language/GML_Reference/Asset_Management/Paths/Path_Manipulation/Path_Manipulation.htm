Manipulación de la trayectoria

En GameMaker Studio 2 no se limita a las trayectorias que se hacen con el Path Editor. Hay una serie de funciones que se pueden utilizar para manipular y cambiar esas rutas en tiempo de ejecución, o incluso para crear otras nuevas y luego cambiarlas y manipularlas.

Las siguientes funciones se utilizan para cambiar las rutas:

  • ruta_existe
  • ruta_add
  • ruta_añadir_punto
  • punto_de_cambio_de_la_senda
  • punto_de_inserción_de_la_senda
  • ruta_borrar_punto
  • ruta_de_punto_de_claro
  • path_append
  • ruta_de_asignación
  • ruta_suprimir
  • ruta_duplicada
  • camino_inclinado
  • ruta_espejo
  • camino_inverso
  • ruta_rotación
  • ruta_recalada
  • path_set_closed
  • path_set_kind
  • path_set_precision
  • desplazamiento_de_trayectoria
