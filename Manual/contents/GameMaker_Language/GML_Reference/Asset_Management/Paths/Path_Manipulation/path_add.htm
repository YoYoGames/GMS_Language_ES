ruta_add

Con esta función puedes crear una ruta en GameMaker Studio 2 sin usar el editor de rutas. Esta función devolverá el índice de la ruta que debe ser almacenado en una variable y usado como referencia para esa ruta a partir de entonces. Tenga en cuenta que la ruta creada está vacía, es decir, no tiene puntos definidos, por lo que tendrá que utilizar las otras funciones disponibles para añadir puntos a la ruta o estar utilizando MP rejillas para para generar la trayectoria. Una vez que haya terminado de utilizar la trayectoria, o desee crear una nueva y almacenar su índice en la misma variable, deberá eliminar primero la antigua trayectoria con path_delete para evitar fugas de memoria que que pueden llegar a bloquear el juego.

Sintaxis:

path_add();

Devuelve:

Real

Ejemplo:

global.newpath = path_add();

Esto creará una nueva ruta y asignará su índice a global.newpath.
