path_get_kind

Las trayectorias pueden ser suaves o rectas (una trayectoria suave calcula una trayectoria curva alrededor de los puntos definidos, mientras que una trayectoria recta simplemente va de un punto a otro). Esta función puede utilizarse para averiguar si la trayectoria dada es suave(verdadero) o no(falso).

Sintaxis:

path_get_kind(index);

Argumento Descripción
índice El índice de la ruta a comprobar.

Devuelve:

Booleano

Ejemplo:

path_kind = path_get_kind(pth_Patrol);

Esto establecerá la variable "path_kind" como verdadera o falsa dependiendo del tipo de ruta que sea el índice de ruta dado.
