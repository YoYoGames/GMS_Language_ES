ruta_obtener_velocidad_de_punto

Con esta función puedes obtener la velocidad del punto (tal y como se define en el Path Editor o cuando añades dinámicamente un punto de la ruta usando path_add_point()) expresado como un porcentaje. Así, si tienes un punto de ruta establecido en 50 en el editor de rutas, esta función devolvería 50 cuando se utilice.

Sintaxis:

camino_obtener_punto_velocidad(índice, n);

Argumento Descripción
índice El índice de la ruta a comprobar.
n La identidad de punto para comprobar.

Devuelve:

Real

Ejemplo:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
var _px = path_get_point_x(pth_Patrol, 0);
var _py = path_get_point_x(pth_Patrol, 0);
path_change_point(pth_Patrol, 0, _px, _py, 100);
}

El código anterior comprueba el punto "0" de la ruta indexada por "pth_Patrol" para ver si la velocidad no es igual a 100. Si no lo es, entonces establece la velocidad de ese punto a 100.
