path_get_point_y

Esta función devolverá la posición y (en coordenadas de habitación) del punto que usted introdujo para la trayectoria que usted indexa. Si el punto está fuera del rango de la trayectoria (por ejemplo: una trayectoria tiene 8 puntos y usted pide la posición y del punto 10) entonces se devolverá un valor de 0 será devuelto.

Sintaxis:

path_get_point_y(index, n);

Argumento Descripción
índice El índice de la ruta a comprobar.
n La identidad de punto para comprobar.

Devuelve:

Real

Ejemplo:

si (posición_de_la_pista == 1)
{
var _num = path_get_number(pth_Patrol);
var _pos = floor(random(_num));
x = path_get_point_x(pth_Patrol, _pos);
y = path_get_point_y(pth_Patrol, _pos);
path_position = (1 / _num) * _pos;
}

El código anterior comprobará si una instancia está al final de un camino. Si lo está, elegirá un punto aleatorio en el camino y moverá la instancia a ese punto.
