Variables de trayectoria

Las siguientes variables están incorporadas a todas las instancias cuando se crean y pueden utilizarse para obtener información sobre una ruta, o para cambiar ciertas propiedades:

  • índice_de_camino
  • posición_de_camino
  • posición_de_camino_anterior
  • velocidad_de_camino
  • escala_de_trayectoria
  • orientación_de_la_vía
  • ruta_de_actuación

Tenga en cuenta que puede encontrar más información sobre las variables de instancia aquí: Variables de instancia.
