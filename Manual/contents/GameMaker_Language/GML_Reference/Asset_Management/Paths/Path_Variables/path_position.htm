posición_de_camino

Esta función puede utilizarse para obtener o establecer la posición de una instancia a lo largo de un camino. El valor se normaliza de 0 a 1, por lo que si se establece, por ejemplo, en 0,5, la instancia se moverá exactamente a la mitad de la ruta.

Sintaxis:

posición_de_camino;

Devuelve:

Real

Ejemplo:

posición_de_la_senda = random(1);

El código anterior establece la instancia en una posición aleatoria en cualquier lugar de la ruta actual.
