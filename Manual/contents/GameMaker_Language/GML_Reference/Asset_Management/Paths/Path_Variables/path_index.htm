índice_de_camino

Todos los recursos en GameMaker Studio 2 tienen un número de identificación único, y el path_index es una variable de sólo lectura que contiene el valor del índice para un determinado que se ha asignado a una instancia mediante la función path_start(). Si la instancia no tiene ninguna ruta asignada, la variable se establecerá en -1.

Sintaxis:

path_index;

Devuelve:

integer (ID de la ruta)

Ejemplo:

si (path_index == -1)
{
path_start(pth_enemy3, 4, path_action_reverse, 0);
}

El código anterior comprueba si se ha asignado una ruta a la instancia, y si no es así, inicia una nueva ruta (asignándola al mismo tiempo al path_index ).
