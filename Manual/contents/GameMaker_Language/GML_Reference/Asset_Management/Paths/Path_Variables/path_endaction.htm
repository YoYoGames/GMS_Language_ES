ruta_de_actuación

Esta variable se puede utilizar para obtener o cambiar la reacción de una instancia cuando llega al final de la ruta actual. Normalmente se establece cuando se inicia la ruta utilizando camino_inicio() pero en puede desear cambiar este comportamiento dependiendo de cualquier número de eventos en su juego. Los valores disponibles se expresan mediante las siguientes constantes:

Constante Descripción
ruta_accion_stop detener el camino
path_action_restart continuar desde la posición inicial, saltando al principio si el camino no está cerrado
path_action_continue volver a iniciar el camino desde la posición actual
ruta_accion_inversa invertir la velocidad de la trayectoria (ejecutar la trayectoria en sentido inverso)

Sintaxis:

path_endaction;

Devuelve:

Constante (ver descripción)

Ejemplo:

if (path_endaction == path_action_stop)
{
path_endaction = path_action_reverse;
}

El código anterior comprobará la acción final de la ruta y si se establece como parada, entonces la acción final se cambiará a inversa.
