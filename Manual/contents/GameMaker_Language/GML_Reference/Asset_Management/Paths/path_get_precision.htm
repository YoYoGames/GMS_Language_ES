ruta_obtener_precisión

Esta función devuelve con qué precisión se ha "suavizado" la trayectoria dada, y será un valor entero de 1 a 8. Aunque se puede obtener (y establecer) este valor para una trayectoria en línea recta, no influirá en la forma en que una instancia utiliza la ruta, ya que sólo es relevante cuando el tipo de ruta se establece como "suave".

Sintaxis:

path_get_precision(index);

Argumento Descripción
índice El índice de la ruta a comprobar.

Devuelve:

Real

Ejemplo:

if (path_get_kind(pth_Patrol))
{
if (path_get_precision(pth_Patrol)) != 8
{
path_set_precision(pth_Patrol, 8);
}
}

El código anterior comprueba el camino indexado en "pth_Patrol" para ver qué tipo es y si es suave, entonces comprueba la precisión del suavizado. Si es inferior a 8, se establece un valor de 8.
