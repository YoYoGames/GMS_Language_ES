script_existe

Esta función devolverá true o false dependiendo de si el script o función de script con el índice dado existe. Tenga en cuenta que no se trata de una cadena, sino del nombre del activo que contiene el índice único para cada script (tal y como aparecería en el IDE) o la variable del nombre de la función del script, tal y como se define dentro del activo del script (tenga en cuenta que esto no funcionará para las variables de método). Para más información sobre los scripts, véase El editor de scripts.

Sintaxis:

script_exists(scr);

Argumento Descripción
scr El nombre de la secuencia de comandos que desea comprobar.

Devuelve:

Booleano

Ejemplo:

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
script_execute(script[script_num]);
}

El ejemplo anterior añade dos funciones de script y un valor en un array, luego procede a obtener un número aleatorio y lo utiliza para elegir una función de script para ejecutar, a menos que se elija el -1 en cuyo caso no ocurrirá nada.
