script_get_name

Esta función devolverá el nombre en forma de cadena del script especificado. Este nombre es el que se ha especificado para el guión en el navegador de activos de la ventana principal de GameMaker Studio 2. Para obtener más información sobre los scripts, consulte El editor de scripts.

Sintaxis:

script_get_name(scr);

Argumento Descripción
scr El índice de la secuencia de comandos de la que se desea obtener el nombre.

Devuelve:

Cadena

Ejemplo:

scr_name = script_get_name(Archivo_de_ayuda);

El código de ejemplo anterior almacenará el nombre del índice del script indicado en la variable "scr_name".
