script_execute_ext

Esta función funciona de manera similar a la función script_execute() sólo que usted puede suministrar un array que contenga los argumentos necesarios para la función/script que se llama. Usted también puede suministrar dos argumentos opcionales a la función para especificar un desplazamiento en la matriz para obtener los argumentos, así como el número de argumentos a utilizar de la matriz (esto debe ser un máximo de array_length - offset).

Sintaxis:

script_execute(scr, array_args, [offset], [num_args]);

Argumento Descripción
scr El nombre de la función/script que quiere llamar
array_args La matriz que contiene los argumentos de la función/script
[offset] OPCIONAL! El desplazamiento en la matriz de argumentos
[num_args] OPCIONAL! El número de argumentos para (a partir del desplazamiento)

Devuelve:

Dependerá del valor de retorno de la función/script que se llame

Ejemplo:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

El código de ejemplo anterior utilizará script_execute_ext para llamar a la función dada, suministrando un array de cuatro argumentos, pero utilizando sólo uno de ellos al azar.

NOTA: Esta función no se puede utilizar con las funciones incorporadas.
