script_execute

Esta función se diseñó originalmente para llamar a los activos de los scripts de forma dinámica, pero con la introducción de las funciones definidas por el usuario ahora puede utilizarse de dos maneras:

  • La primera forma es apoyar los scripts heredados, donde cada script era una sola función y el título del script era también el nombre de la función. Usado de esta manera, funcionará como antes y ejecutará un activo de script creado por el usuario y pasará los argumentos dados en él. Tenga en cuenta que en realidad no está llamando al script en este caso sino, a la función del script del mismo nombre dentro del script.
  • La segunda forma es llamar a una función desde un activo de script que contiene múltiples funciones o definiciones de variables. En este caso, deberá proporcionar el nombre de la función y los argumentos que requiere en lugar del nombre del activo de script. Tenga en cuenta que esto sólo funcionará para las funciones de script y no para los métodos, ya que sólo las funciones de script tienen un ID de script asignado.

Aunque esta función está diseñada principalmente para el soporte de legado, no está obsoleta, ya que todavía tiene su lugar en situaciones de red y de llamadas a procedimientos remotos. También hay que tener en cuenta que esta función es lenta, y usarla mucho puede afectar negativamente al rendimiento.

NOTA: Esta función no se puede utilizar con las funciones incorporadas.

Sintaxis:

script_execute(scr, arg0, arg1, arg2..., etc...);

Argumento Descripción
scr El nombre de la función/script que quiere llamar
arg0, arg1, arg2..., etc... Los diferentes argumentos que quieres pasar a la función/script

Devuelve:

Dependerá del valor de retorno del script/función que se llame

Ejemplo:

script_execute(choose(move_up, move_down, move_left, move_right), irandom(5));

El código de ejemplo anterior utilizará script_execute para elegir una de las cuatro funciones definidas por el usuario (en este caso relacionadas con el movimiento) y le pasará un número entero aleatorio como primer argumento.
