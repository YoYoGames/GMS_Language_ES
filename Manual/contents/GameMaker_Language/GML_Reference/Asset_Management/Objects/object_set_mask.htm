object_set_mask

Con esta función puede establecer el índice_de_máscara de un objeto específico (para más información sobre las máscaras vea El Editor de Objetos). Esto significa que todas las instancias de este objeto que se creen después de cambiar la más cara se crearán con esta nueva máscara, mientras que las instancias que ya están en la sala pueden no verse afectadas. Puede establecer esto a -1 para eliminar un sprite de máscara y así por defecto la máscara definida para el sprite del objeto (o ninguna máscara si no se ha elegido ningún sprite elegido).

Tenga en cuenta que ésta no es una función de instancia. Puede establecer el índice de máscara de instancias individuales utilizando la variable máscara_índice y así tener diez instancias, todas con una máscara diferente diferente a la del objeto del que han sido creadas, e incluso si cambias el índice de máscara del objeto usando esta función, todas las instancias que están actualmente en la sala permanecerán como estaban, y sólo las instancias creadas después de llamar a la función comenzarán con la nueva máscara.

Sintaxis:

object_set_mask(index, spr);

Argumento Descripción
índice El índice del objeto a modificar.
spr El nuevo sprite a utilizar como máscara del objeto.

Devuelve:

N/A

Ejemplo:

if (global.level == 10)
{
object_set_mask(obj_Platform, spr_mask_10);
}

El código anterior comprueba el valor de la variable global y luego cambia el índice de la máscara del objeto "obj_Platform" si es igual a diez.
