object_get_sprite

Esta función le dirá si el objeto que está comprobando tiene un sprite o no, y si lo tiene entonces devolverá el índice de ese sprite, o -1 si no lo tiene. Tenga en cuenta que esta no es una función de instancia. Puedes tener un objeto sin mientras que una instancia de ese mismo objeto puede tener uno y viceversa, o incluso pueden tener diferentes sprites. Puedes establecer el sprite de una instancia individual usando la variable sprite_index variable de instancia.

Sintaxis:

object_get_sprite(obj);

Argumento Descripción
obj El índice del objeto a comprobar

Devuelve:

Real (ID del activo de sprite)

Ejemplo:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
sprite_index = _spr;
}

El ejemplo anterior comprobará el sprite_index de la instancia con el sprite del object_index de la instancia. Si no son iguales, entonces asignará el mismo sprite que el del índice del objeto a la instancia.
