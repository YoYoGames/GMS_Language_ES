object_set_solid

Con esta función puede marcar un objeto específico como sólido o no (para más información sobre el indicador "sólido", consulte la sección sobre el Editor de objetos ). Esto significa que todas las instancias de este objeto que se creen después de cambiar el estado sólido se crearán con este nuevo estado, mientras que las instancias que ya están en la sala no se verán afectadas.

Tenga en cuenta que ésta no es una función de instancia. Puede establecer la bandera sólida de instancias individuales usando la variable sólido y así tener diez instancias marcadas como sólidas aunque el objeto en cuestión tenga la bandera establecida en falso en el IDE, e incluso si cambia la bandera de sólido del objeto a verdadero usando esta función, todas las instancias que actualmente están en la habitación permanecerán como estaban, y sólo las instancias creadas después de llamar a la función comenzarán con el sólido marcado como verdadero.

Sintaxis:

object_set_solid( index, solid );

Argumento Descripción
índice El índice del objeto a modificar.
sólido La nueva solidez del objeto (true=sólido, false=no sólido).

Devuelve:

N/A

Ejemplo:

if !object_get_solid(obj_Block)
{
object_set_solid(obj_Block, true);
}

El código anterior comprueba el indicador de solidez del objeto "obj_Block" y, si es falso, se cambia para que se marque como verdadero.
