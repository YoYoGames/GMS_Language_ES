evento_realizar_objeto

Esta función funciona igual que event_perform() excepto que esta vez puede especificar eventos de otro objeto. Hay muchas opciones aquí que permiten la simulación completa de todos posibles eventos, pero tenga en cuenta que esto literalmente sólo realiza todo el código del evento y el juego no modificará nada para que se dispare manualmente, por ejemplo si elige realizar un evento de pulsación de teclado, el evento se disparará pero no se reconocerá que se ha pulsado la tecla correspondiente. O si realizas un evento de alarma, el contador de alarmas no se pondrá a -1 sino que continuará la cuenta atrás. Puede encontrar una lista completa de las constantes disponibles que esta función en la página de la función event_perform().

NOTA: Las acciones del evento llamado con esta función se aplican a la instancia actual, y no a las instancias del objeto dado.

Sintaxis:

event_perform_object(obj, type, numb);

Argumento Descripción
obj El objeto que debe tener su evento disparado.
tipo El tipo de evento a realizar. Véase más abajo.
numb El evento específico, si es necesario (de lo contrario, sólo use 0).

Devuelve:

N/A

Ejemplo:

event_perform_object(obj_Player, ev_keypress, ord("W"));

Esto realizaría el evento asociado a la comprobación del teclado pulsado > tecla W del objeto "obj_Player" en la instancia actual.
