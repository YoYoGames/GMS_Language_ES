event_action (obsoleto)

Esta variable de sólo lectura devolvía el índice de la acción que se estaba ejecutando en ese momento, empezando por 0 en versiones anteriores de GameMaker. Sin embargo, esta es ahora una variable obsoleta en GameMaker Studio 2. Se ha dejado en el apoyo de legado sólo, y siempre devolverá 0 como todas las acciones se concatenan juntos para mejorar la velocidad de ejecución.

Sintaxis:

acción_de_evento;

Devuelve:

Real

Ejemplo:

num = evento_acción;

El código anterior almacena el número de acción actual en la variable "num".
