número_de_evento

Esta variable de sólo lectura devuelve el número del evento que se está llamando actualmente, donde el número se refiere en realidad al "subevento" del evento, es decir: para el evento de paso el número de evento podría ser cualquiera de las constantes ev_step_normal , ev_step_begin, o ev_step_end. Para una lista completa de las constantes que están disponibles para los sub-acontecimientos específicos ver event_perform()y si necesita conocer el evento principal en sí, debería consultar el archivo tipo_de_evento.

Sintaxis:

número_de_evento;

Devuelve:

Real

Ejemplo:

switch (event_number)
{
case ev_step_normal: show_debug_message("¡Evento de paso!"); break;
case ev_game_start: show_debug_message("Inicio del juego""); break;
case ev_room_start: show_debug_message("¡Inicio de la sala!"); break;
}

El código anterior podría llamarse desde un script y usarse para mostrar mensajes de depuración informando de qué evento se está activando en ese momento.
