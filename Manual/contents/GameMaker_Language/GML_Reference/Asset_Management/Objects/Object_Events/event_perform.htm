evento_performance

Esta función realizará el evento especificado, con el argumento designado, para la instancia que ejecuta el código. Hay muchas opciones aquí que permiten la simulación completa de todos los eventos posibles, pero tenga en cuenta que esto literalmente sólo realiza todo el código en el evento y el juego no modificará nada para que se dispare manualmente, por ejemplo, si eliges realizar un evento de pulsación de teclado, el evento se disparará pero no se reconocerá que la tecla relevante ha sido pulsada. O si usted realiza un evento de alarma, el contador de alarmas no se pondrá a -1 sino que continuará la cuenta atrás.

Sintaxis:

event_perform(type, numb);

Argumento Descripción
tipo El tipo de evento a realizar. Véase más abajo.
numb La constante o el valor del evento específico, si es necesario (de lo contrario, sólo use 0).

Devuelve:

N/A

Ejemplo ampliado


Para ayudar a entender cómo se puede utilizar esta función, observe los siguientes ejemplos:

event_perform(ev_keypress, ord("W"));

Esto realizaría el evento asociado a la comprobación de teclado tecla "W" pulsada (sin generar realmente una pulsación de teclado).

event_perform(ev_step, ev_step_begin);

Esto realizaría el evento Begin Step (si se llama desde cualquiera de los eventos de paso causaría que el código del evento begin step se ejecute dos veces).

event_perform(ev_create, 0);

Esto realizaría el eventoCreate de la instancia.
