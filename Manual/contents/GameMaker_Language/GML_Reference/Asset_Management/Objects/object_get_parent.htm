object_get_parent

Esta función obtendrá el object_index de cualquier padre que haya sido asignado al objeto especificado, o bien devolverá -100 para mostrar que el objeto no tiene ningún padre asignado, o -1 si el objeto que se está comprobando no existe. Para más información sobre los padres vea la sección sobre el Editor de Objetos.

Sintaxis:

object_get_parent(obj);

Argumento Descripción
obj El índice del objeto a comprobar

Devuelve:

Real

Ejemplo:

par = object_get_parent(object_index);

El código anterior obtendrá el padre del índice del objeto para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "par".
