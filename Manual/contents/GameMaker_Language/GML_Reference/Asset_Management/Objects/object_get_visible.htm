object_get_visible

Esta función le dirá si el objeto que está comprobando ha sido marcado como "visible" (ejecuta su evento de dibujo) o no (no ejecuta su evento de dibujo). Tenga en cuenta que esta no es una función de instancia. Por lo tanto, puedes tener un objeto visible y una instancia invisible del mismo objeto y viceversa. Puedes establecer la visibilidad de una instancia individual usando la variable visible variable de instancia.

Sintaxis:

object_get_visible(obj);

Argumento Descripción
obj El índice del objeto a comprobar.

Devuelve:

Booleano

Ejemplo:

if (!visible) && (object_get_visible(object_index))
{
visible = true;
}

El código anterior comprobará la instancia que se está ejecutando para ver si es visible o no, así como comprobará el índice del objeto de la instancia para ver si está marcado como visible o no. Si la instancia no es visible todavía el índice del objeto está marcado como on, establecerá "visible" como verdadero para esa instancia.
