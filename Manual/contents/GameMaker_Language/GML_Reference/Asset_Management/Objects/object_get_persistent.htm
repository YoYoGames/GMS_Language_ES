object_get_persistent

Esta función le dirá si el objeto que está comprobando ha sido marcado como "persistente" o no. Un objeto persistente es uno que hará que cualquier instancia del mismo sea llevada de sala en sala a menos que sea explícitamente destruida. Tenga en cuenta que esta no es una función de instancia. Por lo tanto, puedes tener un objeto persistente y una instancia no persistente del mismo objeto y viceversa. Puedes establecer la bandera de persistencia de una instancia individual utilizando la función persistente instancia .

Sintaxis:

object_get_persistent(obj);

Argumento Descripción
obj El índice del objeto a comprobar.

Devuelve:

Booleano

Ejemplo:

if (!persistent) && (object_get_persistent(object_index))
{
persistent = true;
}

El código anterior comprobará la instancia que se está ejecutando para ver si es persistente o no, así como comprobará el índice del objeto de la instancia para ver si está marcado como persistente o no. Si la instancia no es persistente pero el índice del objeto está marcado como persistente, establecerá "persistente" como verdadero para esa instancia.
