Objetos

En GameMaker Studio 2 tienes objetos y tienes instancias. Los objetos son esencialmente la plantilla base para una instancia, y como tal nunca están presentes en una sala directamente... sólo se colocan instancias del objeto en la sala. Esto significa que si quieres cambiar algo para todas las instancias que vas a crear, entonces puedes hacerlo cambiando el objeto.

IMPORTANTE! Si se cambia algo de un objeto, no se modificarán las instancias presentes en la sala, sólo las que se creen después del cambio.

También puedes acceder a información sobre un objeto base y utilizarla para dictar comportamientos o cambios en el juego. Por ejemplo, puedes hacer una comprobación del ID de un objeto padre, y en el siguiente código si se encuentra una instancia con ese padre puedes comprobar su object_index para decidir qué debe hacerse.

GameMaker Studio 2 le ofrece una serie de funciones que le permiten obtener varios detalles sobre un objeto. Tenga en cuenta que un objeto no es una instancia (una entidad en el juego), es puramente un recurso a partir del cual se crean todas las instancias. Esto significa que que los valores de retorno de estas funciones pueden ser diferentes a los valores reales mostrados por las instancias en la sala, ya que una vez que se crea una instancia se puede cambiar a través de código y acciones de DnD™.

  • objeto_índice
  • objeto_existe
  • object_get_name
  • object_get_mask
  • object_get_parent
  • object_get_persistent
  • object_get_solid
  • object_get_sprite
  • object_get_visible
  • object_get_physics
  • object_is_ancestor

También hay una serie de funciones que permiten establecer las propiedades de un objeto. Hay que tener en cuenta que las instancias de este objeto que ya existen en la sala pueden no verse afectadas por estas funciones, pero todas las nuevas instancias de este objeto creadas en la sala, por lo que se recomienda no cambiar nunca las propiedades de un objeto cuando haya instancias de ese objeto en la sala actual.

  • object_set_mask
  • object_set_persistent
  • object_set_solid
  • object_set_sprite
  • object_set_visible
