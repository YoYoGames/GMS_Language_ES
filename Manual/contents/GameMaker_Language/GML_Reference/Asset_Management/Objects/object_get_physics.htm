object_get_physics

Esta función le dirá si el objeto que está comprobando ha sido marcado como "física habilitada" - en cuyo caso devolverá true, - o no - en cuyo caso devolverá false.

Sintaxis:

object_get_physics(obj);

Argumento Descripción
obj El índice del objeto a comprobar.

Devuelve:

Booleano

Ejemplo:

if object_get_physics(object_index)
{
phy_active = true;
}

El código anterior comprobará la instancia que lo ejecuta para ver si el objeto del que se crea está habilitado para la física, y si lo está, activa la simulación de la física para la instancia.
