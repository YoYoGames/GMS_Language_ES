object_get_mask

Esta función le dirá si el objeto que está comprobando tiene un índice de máscara o no, y si lo tiene entonces devolverá el índice de esa máscara, o -1 si no lo tiene. Tenga en cuenta que ésta no es una función de instancia. Puedes tener un objeto sin sin máscara mientras que una instancia de ese mismo objeto puede tener una y viceversa, o incluso pueden tener diferentes máscaras. Puede establecer el índice de máscara de una instancia individual utilizando la variable índice_de_máscara variable de instancia.

Sintaxis:

object_get_mask(obj);

Argumento Descripción
obj El índice del objeto a comprobar

Devuelve:

real

Ejemplo:

if (mask_index != object_get_mask(object_index))
{
mask_index = object_get_mask(object_index);
}

El ejemplo anterior comprobará el índice de máscara de la instancia con la máscara del índice_objeto de la instancia. Si no son iguales, entonces asignará a la instancia la misma máscara que tiene el índice objeto.
