object_set_visible

Con esta función puede marcar un objeto específico como visible o no (para más información sobre la bandera "visible" vea la sección sobre el Editor de Objetos ). Esto significa que todas las instancias de este objeto que se creen después de que se haya cambiado el estado visible se crearán con este nuevo estado, mientras que las instancias que ya están en la sala no se verán afectadas.

Tenga en cuenta que ésta no es una función de instancia. Puede establecer la bandera visible de instancias individuales utilizando la variable visible y así tener diez instancias marcadas como visibles incluso aunque el objeto en cuestión tenga la bandera establecida en falso en el IDE, e incluso si cambia la bandera visible del objeto a verdadera usando esta función, todas las instancias que actualmente están en la sala permanecerán como estaban, y sólo las instancias creadas después de de llamar a la función comenzarán con la bandera visible como verdadera.

Sintaxis:

object_set_visible( index, vis );

Argumento Descripción
índice El índice del objeto a modificar.
vis La nueva visibilidad del objeto (true=visible, false=invisible).

Devuelve:

N/A

Ejemplo:

if !object_get_visible(obj_Block)
{
object_set_visible(obj_Block, true);
}

El código anterior comprueba la bandera visible del objeto "obj_Block" y si es falsa entonces se cambia para ser marcada como verdadera.
