object_get_solid

Esta función le dirá si el objeto que está comprobando ha sido marcado como "sólido" o no. Un objeto sólido genera un evento de colisión especial cuando se utiliza el sistema de colisión tradicional (es decir, el mundo de la física está desactivado). Tenga en cuenta que esta no es una función de instancia! Por lo tanto, puedes tener un objeto sólido y una instancia normal del mismo objeto y viceversa. Puedes establecer la bandera de sólido de una instancia individual usando la función sólido instancia de la instancia.

Sintaxis:

object_get_solid(obj);

Argumento Descripción
obj El índice del objeto a comprobar.

Devuelve:

Booleano

Ejemplo:

if (!solid) && (object_get_solid(object_index))
{
solid = true;
}

El código anterior comprobará la instancia que se está ejecutando para ver si es sólida o no, así como comprobar el índice del objeto de la instancia para ver si está marcado como sólido o no. Si la instancia no es sólida pero el índice del objeto está marcado como sólido, establecerá se establecerá el valor de "sólido" como verdadero para esa instancia.
