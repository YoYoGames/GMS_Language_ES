objeto_existe

Esta función devuelve si un objeto con el índice especificado existe o no. Tenga en cuenta que esto comprueba si un objeto está presente en el Navegador de Activos, y no realmente en la sala de juego. Para ello debe utilizar la función instance_exists().

Sintaxis:

object_exists(obj);

Argumento Descripción
obj El índice del objeto a comprobar.

Devuelve:

Booleano

Ejemplo:

if object_exists(obj_temp)
{
with (obj_Temp) scr_add_event();
}

El código anterior comprueba la variable "obj_temp" para asegurarse de que almacena el índice de un objeto y entonces, si lo hace, hace que el objeto ejecute un script.
