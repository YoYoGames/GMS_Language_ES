objeto_índice

Esta variable de sólo lectura devuelve el índice del objeto del que se ha creado la instancia. Esto no es lo mismo que el nombre del objeto, que es una cadena y puede encontrarse usando object_get_name(), ya que esta función devuelve el número de índice, que es un valor único que GameMaker Studio 2 asigna a cada objeto en el momento de su creación.

Sintaxis:

object_index;

Devuelve:

Real (entero - valor del índice de recursos)

Ejemplo:

obj_name = object_get_name(object_index);

El código anterior utilizará el object_index para encontrar el nombre del objeto del que se ha creado la instancia actual.
