object_set_sprite

Con esta función puedes establecer el índice de sprite de un objeto específico. Esto significa que todas las instancias de este objeto que se creen después de cambiar el sprite_index se crearán con este nuevo sprite_index, mientras que las instancias que ya están en la sala no se verán afectadas.

Tenga en cuenta que esta no es una función de instancia! Puedes establecer el índice de sprite de instancias individuales usando la variable sprite_index y así tener diez instancias todas con un diferente al objeto del que se crean, e incluso si cambias el índice de sprite del objeto usando esta función, todas las instancias que están actualmente en la sala permanecerán como estaban, y sólo las instancias creadas después de llamar a la función comenzarán con el nuevo sprite.

Sintaxis:

object_set_sprite( index, spr );

Argumento Descripción
índice El índice del objeto a modificar.
spr El sprite a asignar al objeto.

Devuelve:

N/A

Ejemplo:

switch (sala)
{
case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

El código anterior establecerá el índice del sprite del objeto "obj_Player" a diferentes valores dependiendo de la habitación en la que se encuentre la instancia que ejecuta el código. A continuación, crea una instancia de "obj_Player".
