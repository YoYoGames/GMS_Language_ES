object_get_name

Esta función devolverá el nombre en forma de cadena del objeto especificado. Este nombre es el que se ha especificado para el objeto en el Navegador de Activos de la ventana principal de GameMaker Studio 2. Tenga en cuenta que esto es sólo una cadena y no se puede utilizar para referenciar el objeto directamente - para eso necesitaría el índice del objeto. Sin embargo, puede utilizar esta cadena para obtener el índice del objeto utilizando la cadena devuelta junto con la función asset_get_index().

Sintaxis:

object_get_name(obj);

Argumento Descripción
obj El índice del objeto a comprobar.

Devuelve:

Cadena

Ejemplo:

str = object_get_name(object_index);

El código anterior obtendrá el nombre del índice del objeto para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "str".
