font_add_enable_aa

Esta función puede utilizarse para activar o desactivar el antialiasing (AA) de las fuentes añadidas mediante font_add(). Esta función debe ser llamada antes de añadir cualquier fuente y al establecerla como a true habilitará el AA, y si se establece a false lo deshabilitará. Por defecto, el AA está activado. Tenga en cuenta que esto no tendrá efecto en las fuentes que se hayan añadido antes de que se llame a la función y la función sólo necesita ser invocada una vez cuando la fuente es añadida, y no cada dibujo/paso en que la fuente es usada.

Sintaxis:

font_add_enable_aa(enable);

Argumento Descripción
activar Si se activa(true) o desactiva(false) el AA para las fuentes añadidas.

Devuelve:

N/A

Ejemplo:

if !font_add_get_enable_aa()
{
font_add_enable_aa(true);
}

El código anterior comprueba el estado del antialiasing para las fuentes añadidas y, si no está activado, lo activamos.
