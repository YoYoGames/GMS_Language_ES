font_replace_sprite_ext

Esta función reemplazará una fuente de sprite que hayas creado previamente usando la función font_add_sprite_ext(). Utiliza una "tira de sprites", donde cada sub-imagen es un símbolo o letra individual, y las imágenes son mapeadas a letras basándose en el argumento "string_map" de la función. Este argumento es una cadena que puede usar para decirle a GameMaker Studio 2 en qué orden están las sub-imágenes de la fuente sprite y las mapeará automáticamente al escribir el texto. Así, por ejemplo, si tienes una cadena-mapa de "AaBbCcDdEeFfGgHh", tu fuente sprite debe tener las sub-imágenes ordenadas de esta manera.

También puede establecer que el espaciado de la fuente sea proporcional(verdadero) o no(falso), donde una fuente proporcional se espacia en base a la anchura real de las letras (por lo que la letra "i" ocupa menos espacio que la letra "w", por ejemplo) mientras que una fuente no proporcional se espaciará en base al ancho de la sub-imagen, como una fuente monoespaciada. Por último, puede definir el espacio que se dejará entre cada letra al escribir, y esto puede ser cualquier valor entero, siendo 0 sin espacio (las letras se "tocarán" si son proporcionales). La función devuelve un valor de índice que debe almacenarse en una variable, ya que se necesitará en todos los códigos posteriores que hagan referencia a esta fuente.

Si la función falla, entonces fallará silenciosamente sin reemplazar la fuente sprite dada.

NOTA: Esta función requiere que se haya añadido previamente una fuente de sprite utilizando font_add_sprite()ya que la reemplazará y no se puede utilizar en fuentes que se incluyen como parte de los recursos del juego.

Sintaxis:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Argumento Descripción
fuente El tipo de letra a sustituir.
spr El sprite en el que se basará la fuente.
mapa_de_cadenas Cadena de la que se toma el orden de las subimágenes de los sprites.
puntal Establecer como fuente proporcional o no.
sep El espacio que hay que dejar entre cada letra.

Devuelve:

N/A

Ejemplo:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

El código anterior reemplazará un activo de fuente sprite que está indexado en la variable "global.Font" con imágenes sprite indexadas en la variable "spr_CalcFont".
