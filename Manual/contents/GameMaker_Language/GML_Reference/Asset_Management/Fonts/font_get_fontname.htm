font_get_fontname

Con esta función puede obtener el nombre real del sistema del activo de fuente seleccionado. Esta función devuelve una cadena y no un índice, y el nombre devuelto dependerá de la fuente que se esté utilizando. Por ejemplo puede tener un activo de fuente llamado "fnt_Main" en el navegador de activos de GameMaker Studio 2, y la fuente en sí puede ser la fuente del sistema de Windows Arial. En este caso, la función devolverá "Arial" ya que ese es el nombre del sistema de la fuente. Si necesita el nombre tal y como aparece en el Navegador de Activos, deberá utilizar font_get_name().

Sintaxis:

font_get_fontname(ind);

Argumento Descripción
ind Índice de la fuente a comprobar.

Devuelve:

Cadena

Ejemplo:

fnt_Name = font_get_fontname(font0);

El código anterior obtendrá el nombre del sistema de un recurso de fuente y lo almacenará como una cadena en la variable "fnt_Name".
