font_get_texture

Esta función devuelve un puntero especial para la página de textura de la fuente. Este valor puede ser utilizado en otras funciones de dibujo, particularmente en el dibujo general cuando se utilizan primitivas así como así como en las funciones Shader funciones de dibujo. Puede obtener más información sobre la página de textura devuelta utilizando las diferentes funciones texture_ que se encuentran aquí.

Sintaxis:

font_get_texture(fuente);

Argumento Descripción
fuente El índice de la fuente a utilizar.

Devuelve:

Puntero (ID de la página de textura)

Ejemplo:

tex = font_get_texture(fnt_Main);

El código anterior obtendrá el puntero de textura para la fuente indexada como "fnt_Main".
