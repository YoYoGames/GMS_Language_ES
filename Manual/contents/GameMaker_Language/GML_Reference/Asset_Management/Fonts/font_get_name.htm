font_get_name

Esta función se puede utilizar para obtener el nombre (en forma de cadena) que se le dio a la fuente cuando se añadió como activo al navegador de activos de GameMaker Studio 2. Tenga en cuenta que esto es sólo una cadena y no se puede utilizar para hacer referencia a la fuente directamente, para lo que necesitaría el índice de la fuente. Sin embargo, puede utilizar esta cadena para obtener el índice de la fuente utilizando la cadena devuelta junto con la función función asset_get_index().

Sintaxis:

font_get_name(ind);

Argumento Descripción
ind Índice de la fuente a comprobar.

Devuelve:

Cadena

Ejemplo:

fnt_Name = font_get_name(font0);

El código anterior obtendrá el nombre de un recurso de fuente tal como aparece en el Navegador de Activos y lo almacenará como una cadena en la variable "fnt_Name".
