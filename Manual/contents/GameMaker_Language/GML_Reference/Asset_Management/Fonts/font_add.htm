añadir_fuente

Esta función puede utilizarse para añadir un tipo de letra a su juego a partir de los tipos de letra que están instalados en el sistema en el que se está ejecutando. Puede definir el tamaño de la fuente (en puntos), así como si la fuente debe estar en negrita o en cursiva, y puede También puede definir el rango de caracteres a incluir. La función devuelve un valor de índice que debe almacenarse en una variable, ya que se necesitará en todos los códigos posteriores que hagan referencia a esta fuente, o devolverá -1 si la función falla por cualquier razón.

Para el módulo de destino de HTML5, esta función sólo puede utilizarse para añadir fuentes web, y sólo funcionará si WebGL no está habilitado - si WebGL está habilitado, deberá utilizar un recurso de fuente general (es decir, utilizando draw_set_font() junto con una fuente del navegador de activos) o fuentes de mapa de bits (véase font_add_sprite()).

IMPORTANTE! Cuando se dirige a HTML5, el texto en cursiva y en negrita sólo se admite si el activo de la fuente o la propia fuente web tiene estos caracteres incluidos. No se sintetizarán a partir de la fuente base.

En todos los demás objetivos (no web), puede utilizar esta función para añadir una fuente desde un archivo. El archivo debe incluirse en el paquete del juego mediante la función Archivos incluidos de GameMaker Studio 2, y debe ser un archivo de fuente en formato *.ttf o *.otf, útil para añadir fuentes no estándar como las asiáticas o las árabes.

ADVERTENCIA Si incluye un archivo de fuentes de este tipo con un juego, debe tener una licencia para su distribución con el juego.

Cuando se carga una fuente desde un archivo de esta manera, el tamaño de la fuente está en píxeles y los valores primero y último se ignoran, lo que significa que se añadirán todos los glifos de la fuente. Las fuentes añadidas de esta manera se asignan a su propia textura por lo que hay que tener cuidado al utilizar esta función, ya que incrementará el número de intercambios de texturas al dibujar. También hay que tener en cuenta que las fuentes pueden aparecer ligeramente más grandes cuando se dibujan, ya que los glifos pueden tener partes que se dibujan fuera del de la caja definida para la fuente. También debes tener en cuenta que hay una limitación de alrededor de 200 glifos que pueden ser renderizados en un solo cuadro para una fuente en particular.

Por defecto, las fuentes añadidas tendrán el antialiasing activado. Sin embargo, puede comprobar el estado del AA en cualquier momento utilizando la función font_add_get_enable_aa() y puede activar/desactivar el AA utilizando la función font_add_enable_aa().

NOTA: Cuando cargues una fuente en GameMaker Studio 2: Studio debes recordar eliminarla de nuevo (con font_delete()) cuando ya no la necesites, de lo contrario existe el riesgo de una fuga de memoria que ralentizará y eventualmente bloqueará su juego.

Sintaxis:

font_add(name, size, bold, italic, first, last);

Argumento Descripción
nombre El nombre de la fuente que se va a añadir (por ejemplo, 'Arial'), o la ruta del archivo si la fuente es un archivo incluido *. ttf o *.otf .
tamaño El tamaño de la fuente: puntos para las fuentes web, píxeles para las fuentes de archivo.
negrita Si la fuente está en negrita (true) o no (false).
cursiva Si la fuente es cursiva (true) o no (false).
primero El primer carácter que debe incluirse (si no está seguro, opte por el 32).
último El último carácter que debe incluirse (si no está seguro, opte por el 128).

Devuelve:

Real

Ejemplo:

newfont = font_add("Arial", 24, true, true, 32, 128);

Esto creará una nueva fuente que tiene un tamaño de 24pt, utiliza "Arial" que es negrita y cursiva y el índice para esta nueva fuente se almacena en la variable newfont. El rango de fuentes incluye letras mayúsculas y minúsculas, números y toda la puntuación común. puntuación común.
