fuente_existe

Esta función devuelve si existe o no una fuente con el índice especificado. Puede comprobar los índices de las fuentes definidos desde el Navegador de Activos, o las fuentes que han sido añadidas utilizando funciones como font_add().

Sintaxis:

font_exists(ind);

Argumento Descripción
ind Índice de la fuente a comprobar.

Devuelve:

Booleano

Ejemplo:

si font_exists(fnt_Main)
{
draw_set_font(fnt_Main);
}

Esto establecerá la fuente de dibujo activa en fnt_Main si existe.
