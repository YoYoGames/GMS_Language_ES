font_replace_sprite

Con esta función se puede utilizar una "tira de sprites" para reemplazar un activo de fuente de sprites previamente creado, donde cada sub-imagen sería un símbolo o letra individual. El índice es el número de mapa UTF8 para el primer carácter a utilizar y los rangos a utilizar son los mismos que para una fuente normal, empezando por "!", pero recuerde que sólo debe incluir lo que necesita y que debe utilizar la función ord() para obtener el valor correcto de UTF8 para la letra inicial.

También puede establecer que el espaciado de la fuente sea proporcional(verdadero) o no(falso), donde una fuente proporcional se espacia en base a la anchura real de las letras (por lo que la letra "i" ocupa menos espacio que la letra "w", por ejemplo) mientras que una fuente no proporcional se espaciará en base al ancho de la sub-imagen, como una fuente monoespaciada. Por último, puede definir el espacio que se dejará entre cada letra al escribir, y esto puede ser cualquier valor entero, siendo 0 sin espacio (las letras se "tocarán" si son proporcionales).

Si la función falla, entonces fallará silenciosamente sin reemplazar la fuente sprite dada.

También hay que tener en cuenta que no puede haber espacios en el mapa de caracteres, de modo que si, por ejemplo, utiliza '.' y '0' pero no '/', deberá incluir '/' de todos modos o dejar una subimagen en blanco entre los dos caracteres utilizados. La imagen siguiente muestra un diseño típico de fuente de sprites:

Sintaxis:

font_replace_sprite(ind, spr, first, prop, sep);

Argumento Descripción
ind El índice de la fuente que será reemplazada (es decir, todas las referencias a este recurso después de esta función se referirán a la nueva fuente, no a la original).
spr El sprite en el que se basará la fuente.
primero El índice del primer carácter disponible.
puntal Si se debe eliminar el espacio en blanco alrededor de cada letra (si es falso, la fuente creada será monoespaciada, de lo contrario será como la mayoría de las fuentes regulares).
sep El espacio entre cada letra, 0 significa que la derecha de una letra apenas toca la izquierda de la siguiente.

Devuelve:

N/A

Ejemplo:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

El código anterior sustituirá el activo de fuente indexado en la variable global "Font" por un nuevo activo de fuente de un archivo de imagen externo.
