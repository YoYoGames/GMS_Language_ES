tamaño_de_página_de_textura_de_la_fuente

Esta variable incorporada se puede utilizar para obtener o establecer el tamaño de la página de la textura cuando se utiliza la función font_add(). Al añadir una fuente utilizando esa función, GameMaker Studio 2 creará una caché de página de textura de los glifos requeridos hasta el tamaño definido por esta variable. El proceso es el siguiente:

  • Cada carácter que se utiliza se almacena en caché de la fuente en una página de textura
  • Cuando la página de texturas está llena (porque utilizas caracteres grandes o muchos caracteres diferentes) GameMaker Studio 2 eliminará los caracteres de la caché y los sustituirá por los nuevos que necesites
  • GameMaker Studio 2 trata de no hacer la textura demasiado grande porque no todos los dispositivos pueden soportar texturas grandes, y trata de no hacer la textura demasiado pequeña porque eliminar y volver a añadir caracteres a la textura lleva tiempo de la CPU y hace que su juego se ejecute lentamente

Si cree que necesita un tamaño de página de textura mayor o menor que el predeterminado de 1024px que utiliza GameMaker Studio 2, puede establecer el valor utilizando esta variable incorporada. Si no está utilizando la función font_add(), entonces la configuración de este no tendrá ningún efecto sobre el rendimiento de sus juegos como las fuentes añadidas en el IDE se almacenan en caché en las páginas de textura regular (que se puede controlar desde el Editor de Fuentes y las opciones de juego para la plataforma de destino).

Sintaxis:

tamaño_de_página_de_textura_de_fuente;

Devuelve:

Real (anchura/altura máxima en píxeles)

Ejemplo:

font_texture_page_size = 512;
newfont = font_add("Arial", 24, true, true, 32, 128);

El código anterior establece la página de textura de la caché de la fuente a un tamaño máximo de 512px y luego añade una nueva fuente al juego.
