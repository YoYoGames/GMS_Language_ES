habitación_siguiente

Con esta función puede recuperar el índice de la habitación después de la entrada de la habitación en la función. Por ejemplo, puede utilizar la variable sala para obtener el índice de la sala actual y luego utilizar esta función para encontrar la sala que le sigue, tal y como aparece en el Gestor de Salas. Si no hay ninguna sala después de la introducida, se devuelve -1. Tenga en cuenta que esta función no reconocerá ni tendrá en cuenta las habitaciones que han sido añadidas dinámicamente usando room_add() o room_duplicate().

Sintaxis:

room_next(numb);

Argumento Descripción
numb El índice de la sala para obtener la siguiente después.

Devuelve:

Índice

Ejemplo:

si room_next(room) != -1
{
room_goto_next());
}

El código anterior comprobará si existe la siguiente sala y, en caso afirmativo, irá a ella.
