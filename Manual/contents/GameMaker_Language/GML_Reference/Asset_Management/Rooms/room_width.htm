ancho_de_habitación

Esta variable contiene la anchura de la sala actual en píxeles. Puedes modificar esta variable para cambiar el ancho de la sala en cualquier momento.

Sintaxis:

ancho_de_habitación;

Devuelve:

Real

Ejemplo:

si bbox_right > room_width
{
x += room_width - bbox_right;
}

El código anterior comprueba si la caja delimitadora del sprite de la instancia actual es mayor que el ancho de la habitación, y si lo es, mueve la instancia hacia arriba.
