room_get_camera

Con esta función puedes obtener el índice único de la cámara asignada a una vista específica en una sala distinta a la actual. Se da la sala a utilizar, el puerto de vista a utilizar (de 0 a 7) y la función devolverá un índice de cámara.

Sintaxis:

room_get_camera(rm, vind);

Argumento Descripción
rm El índice de la habitación para obtener la cámara de vista de
vind El índice del puerto de vista para obtener la cámara de

Devuelve:

Real (índice de la cámara)

Ejemplo:

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
{
room_set_camera(rm_Game, 0, global.MainCam);
}

El código anterior asigna una cámara en una sala recién creada al puerto de vista [0].
