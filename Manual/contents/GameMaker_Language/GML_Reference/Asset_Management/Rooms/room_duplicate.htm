habitación_duplicada

Esto duplicará una habitación dada y devolverá el índice de duplicados que se utilizará en todas las llamadas posteriores para hacer referencia a la nueva habitación.

NOTA: Las salas nuevas no forman parte del ordenamiento habitual de las salas, por lo que no tienen una sala "anterior" o "siguiente" (lo que significa que las funciones room_next() y room_previous() no funcionarán). Para saltar de una sala añadida a otra, debe utilizar el índice de la propia sala.

Sintaxis:

room_duplicate(index);

Argumento Descripción
índice El índice de la sala original que se va a duplicar.

Devuelve:

Índice

Ejemplo:

global.myroom = room_duplicate(rm_level);

Esto duplicará la sala indexada por "rm_level" y asignará el índice de esta nueva sala a la variable "global.myroom".
