layer_sprite_get_xscale

Esta función puede utilizarse para obtener el multiplicador de escala actual a lo largo del eje x para el elemento sprite. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y la función devolverá un valor real para la escala x, donde 1 es sin escala.

Sintaxis:

layer_sprite_get_xscale(sprite_element_id);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento sprite del que se va a obtener la información

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_xscale(spr_id) != 1
{
layer_sprite_xscale(spr_id, 1);
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza entonces para comprobar el valor de la escala x y, si es distinto de 1, se establece en 1.
