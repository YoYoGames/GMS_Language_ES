ángulo_de_espiga_de_capa

Con esta función puede cambiar el ángulo del elemento sprite dado en una capa. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o cuando utilizar la función layer_sprite_get_id()), y luego establece el valor del ángulo, de 0 a 359 en sentido contrario a las agujas del reloj, donde 0 es a la derecha, 90 es a la parte superior, 180 es a la izquierda y 270 es a la parte inferior. Si establece un valor superior a 360, éste será puesto en bucle para situarlo en el rango de 0 a 359.

Sintaxis:

layer_sprite_angle(sprite_element_id, angle);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento del sprite que se va a cambiar
ángulo El ángulo del sprite (por defecto es 0)

Devuelve:

N/A

Ejemplo:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
layer_sprite_angle(asset_sprite, 0);
}

El código anterior comprobará el elemento sprite asignado a la capa en la que se encuentra la instancia que ejecuta el código y si su ángulo no es 0 lo pondrá a 0.
