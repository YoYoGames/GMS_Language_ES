cambio_de_capa_de_espectro

Con esta función puedes cambiar el recurso sprite asignado a un elemento sprite dado en una capa. Se da el ID del elemento sprite (que se obtiene al crear un elemento sprite con layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y luego suministrar un índice de elemento sprite activo será cambiado. Tenga en cuenta que si el sprite tiene sub-imágenes entonces también se animará (esto puede ser controlado usando los botones layer_sprite_index() y layer_sprite_speed() ). Tenga en cuenta que puede asignar un valor de -1 como el nuevo índice de sprite y no se mostrará ningún sprite, aunque el elemento seguirá existiendo y se podrá volver a cambiar más tarde.

Sintaxis:

layer_sprite_change(sprite_element_id, sprite_index)

Argumento Descripción
sprite_element_id El valor de identificación único del elemento del sprite que se va a cambiar
sprite_index El nuevo índice de sprite a utilizar

Devuelve:

N/A

Ejemplo:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
layer_sprite_change(s, spr_nighttime);
}

El código anterior comprueba el índice del sprite del elemento con el ID almacenado en la variable global global.Asset_sprite y si no es "spr_Nighttime" entonces ese sprite se asigna al elemento.
