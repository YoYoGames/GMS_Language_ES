layer_sprite_create

Con esta función puede asignar un recurso de sprite a una capa que se utilizará en su proyecto. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create() o cuando utiliza el nombre de la capa junto con layer_get_id()), una posición dentro de la sala, y un índice de sprite (que sería el nombre del sprite como se muestra en el Navegador de Activos), y se se añadirá a la capa. La función devuelve el valor de ID único para el elemento, que puede ser utilizado en otras funciones de capa para los sprites.

Sintaxis:

layer_sprite_create(layer_id, x, y, sprite)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige
x La posición x a utilizar
y La posición y a utilizar
sprite El índice del sprite a utilizar

Devuelve:

ID del elemento Sprite (Real)

Ejemplo:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
var _x = random(room_width);
var _y = room_height - 100;
global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

El código anterior crea una nueva capa y luego añade 10 nuevos elementos de sprite a ella, almacenando el ID de cada elemento en un array.
