layer_sprite_get_sprite

Esta función se puede utilizar para obtener el índice actual del elemento sprite. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o cuando se utiliza la función layer_sprite_get_id()), y la función devolverá un valor real que representa el índice del sprite que se muestra. Si el elemento no tiene ningún sprite asignado, la función devolverá -1.

Sintaxis:

layer_sprite_get_sprite(sprite_element_id);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento sprite del que se va a obtener la información

Devuelve:

Índice de Sprite (Real) o -1

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
layer_sprite_change(spr_id, spr_Clouds);
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza entonces para comprobar el sprite asignado al elemento, estableciéndolo en el sprite "spr_Clouds" si no lo está ya.
