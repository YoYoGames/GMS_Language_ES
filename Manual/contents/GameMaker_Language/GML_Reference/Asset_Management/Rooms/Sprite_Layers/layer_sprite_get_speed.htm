layer_sprite_get_speed

Esta función se puede utilizar para obtener el valor actual del multiplicador de velocidad del elemento sprite. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o cuando se utiliza la función layer_sprite_get_id()), y la función devolverá un valor real que representa el multiplicador de velocidad que se está utilizando para animar el sprite. El valor por defecto es 1.

Sintaxis:

layer_sprite_get_speed(sprite_element_id);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento sprite del que se va a obtener la información

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_speed(spr_id) > 0
{
layer_sprite_speed(spr_id, 0);
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza para comprobar la velocidad de animación del elemento y si es mayor que 0, se pone a 0. se establece en 0.
