layer_sprite_destroy

Esta función destruirá el elemento sprite dado. Se proporciona el ID del sprite (que se obtiene al crear el sprite con layer_sprite_create() o cuando se utiliza el ID de la capa junto con layer_get_sprite_id()) y esto lo eliminará. Tenga en cuenta que esto no elimina la capa, sólo el sprite de la misma, y si el sprite es uno que ha sido añadido en el editor de la sala, entonces la próxima vez que salga de la sala y luego regrese, el sprite será recreado de nuevo. Sin embargo, si la sala es persistente, el sprite será eliminado a menos que la persistencia de la sala se desactive de nuevo.

Sintaxis:

layer_sprite_destroy(sprite_element_id)

Argumento Descripción
sprite_element_id El valor de identificación único del sprite que se va a destruir

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
layer_sprite_destroy(lay_id);
}

El código anterior comprueba la capa "Asset_Trees" para ver si el elemento sprite dado existe y si lo hace, entonces se destruye (pero no la capa).
