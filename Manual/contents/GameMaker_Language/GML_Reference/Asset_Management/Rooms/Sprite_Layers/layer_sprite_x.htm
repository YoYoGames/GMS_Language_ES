capa_espiga_x

Esta función controla la posición a lo largo del eje x de la habitación del elemento sprite del activo en la capa. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y luego se establece el valor x a utilizar (basado en las coordenadas de la habitación).

Sintaxis:

layer_sprite_x(sprite_element_id, x);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento del sprite que se va a cambiar
x La posición x del sprite

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

El código anterior obtiene el valor del ID del activo sprite "Clouds" asignado a la capa "Asset_sky" y luego establece su posición x a un valor aleatorio entre 0 y el ancho de la habitación.
