layer_sprite_get_blend

Esta función puede utilizarse para obtener el color de mezcla del elemento sprite. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o cuando se utiliza la función layer_sprite_get_id()), y la función devolverá un valor real que representa el color asignado.

Sintaxis:

layer_sprite_get_blend(sprite_element_id);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento sprite del que se va a obtener la información

Devuelve:

Valor del color (real)

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_blend(back_id) == c_white
{
layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza para comprobar el color de mezcla del elemento y si es equivalente a la constante c_white, entonces la mezcla del elemento se establece en un color aleatorio.
