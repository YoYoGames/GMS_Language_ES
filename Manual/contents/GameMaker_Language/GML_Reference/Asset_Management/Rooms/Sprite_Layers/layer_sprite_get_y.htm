layer_sprite_get_y

Esta función se puede utilizar para obtener la posición y del elemento sprite en la sala. Se da el ID del elemento sprite (que se obtiene al crear un elemento sprite con layer_sprite_create() o cuando utilizar la función layer_sprite_get_id()), y la función devolverá el valor de la posición y.

Sintaxis:

layer_sprite_get_y(sprite_element_id);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento sprite del que se va a obtener la información

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_y(spr_id) < 0
{
layer_sprite_y(spr_id, room_height);
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza para comprobar la posición y del elemento y si es menor que 0, entonces el elemento de la capa se mueve a una posición y diferente.
