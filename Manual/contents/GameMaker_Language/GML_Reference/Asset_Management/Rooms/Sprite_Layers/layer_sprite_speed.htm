velocidad_de_espionaje_de_capa

Esta función puede ser utilizada para establecer la velocidad de una imagen animada que ha sido asignada a un elemento sprite. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y luego establece la velocidad de la animación. Si el IDE ha establecido que la velocidad de la animación para el sprite utilizado se calcule como Cuadros por Cuadro de Juego, entonces el valor de la velocidad es un multiplicador, siendo 1 el valor por defecto, y si se establece en 0.5 se reducirá a la mitad la velocidad de la animación, mientras que si se establece en 1.5 se duplicará. Sin embargo, si el IDE ha establecido que la velocidad de la animación para el sprite utilizado se calcule como Fotogramas por segundo, entonces el valor de la velocidad dada es un valor literal de fotogramas por segundo, por lo que si se establece en 5, por ejemplo, se establecerá la animación para hacer un bucle de 5 fotogramas cada segundo que el juego se ejecute. Si el sprite utilizado no tiene sub-imágenes, esta función no tendrá efecto. no tendrá ningún efecto.

Sintaxis:

layer_sprite_speed(sprite_element_id, speed);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento del sprite que se va a cambiar
velocidad El multiplicador/valor de velocidad que se utilizará para el elemento

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

El código anterior obtendrá el ID de la capa llamada "Activos_agua" y luego lo utilizará para obtener el ID del elemento sprite dado en esa capa. Este ID se utiliza entonces para cambiar la velocidad de la imagen del elemento.
