índice_de_espiga_de_capa

Esta función se puede utilizar para establecer el índice de imagen de un activo sprite que tiene múltiples sub-imágenes en una capa. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o cuando se utiliza la función layer_sprite_get_id()), y luego establece el índice de la imagen a utilizar. Si estableces un valor fuera del rango de sub-imágenes, entonces el índice de la imagen dará vueltas.

Sintaxis:

layer_sprite_index(sprite_element_id, image_index);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento sprite que se va a establecer
índice El índice de la imagen a utilizar para el sprite

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

El código anterior obtendrá el ID de la capa llamada "sprite_trees" y luego lo utilizará para obtener el ID del elemento sprite dado en esa capa. Este ID se utiliza entonces para cambiar el índice de la imagen del elemento.
