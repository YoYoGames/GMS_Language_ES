layer_sprite_exists

Puede utilizar esta función para comprobar si existe un elemento sprite en una capa determinada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o al utilizar la función layer_sprite_get_id()) y la función devolverá true si el elemento existe, o false si no existe.

NOTA: Esta función trabaja dentro del ámbito de la sala de destino actual -por defecto la sala en la que se llama a la función- que puede establecerse mediante la función layer_set_target_room().

Sintaxis:

layer_sprite_exists(layer_id, sprite_element_id)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)
sprite_element_id El valor de ID único del elemento de sprite a comprobar

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
layer_sprite_destroy(lay_id, global.TreesSprites);
}

El código anterior comprueba la capa "Asset_trees" para ver si el elemento sprite dado existe y si lo hace, entonces se destruye (pero no la capa).
