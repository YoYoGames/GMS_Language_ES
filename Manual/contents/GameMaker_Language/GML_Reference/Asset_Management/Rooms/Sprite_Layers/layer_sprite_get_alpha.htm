layer_sprite_get_alpha

Esta función puede utilizarse para obtener el valor alfa del elemento sprite. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o cuando se utiliza la función layer_sprite_get_id()), y la función devolverá un valor entre 0 (totalmente transparente) y 1 (totalmente opaco).

Sintaxis:

layer_sprite_get_alpha(sprite_element_id);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento sprite del que se va a obtener la información

Devuelve:

Real (de 0 a 1)

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_alpha(spr_id) < 0.1
{
layer_sprite_destroy(spr_id);
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza para comprobar el alfa del elemento y si es menor de 0,1, el elemento de la capa se destruye. destruido.
