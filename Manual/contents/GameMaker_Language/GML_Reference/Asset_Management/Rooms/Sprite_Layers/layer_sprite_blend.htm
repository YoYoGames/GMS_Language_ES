capa_espiga_mezcla

Esta función controla la mezcla (o "tintado") del sprite y el valor por defecto es -1 (que representa la constante c_white, que también puede utilizarse). Cualquier otro valor (incluyendo las constantes de color internas como c_red, o c_aqua) mezclará el color especificado con el sprite original. Le das el ID del elemento sprite (que obtienes cuando creas un elemento sprite usando layer_sprite_create() o cuando usas la función layer_sprite_get_id()), y luego establece el color de mezcla a utilizar. A continuación puede ver un ejemplo de un sprite que ha sido mezclado con diferentes colores:

Tenga en cuenta que debe intentar limitar la mezcla en la plataforma HTML5 (a menos que utilice WebGL), ya que cada sprite mezclado tiene que ser almacenado en caché por separado, por lo que tener muchos sprites mezclados puede afectar negativamente al rendimiento (también puedes establecer el tamaño de la caché utilizando la función sprite_set_cache_size()).

Sintaxis:

layer_sprite_blend(sprite_element_id, blend);

Argumento Descripción
sprite_element_id El valor de identificación único del elemento del sprite que se va a cambiar
mezcla El color a mezclar con el sprite sprite (por defecto es c_white)

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

El código anterior obtiene el valor del ID del sprite llamado "Clouds" asignado a la capa "Asset_sky" y luego lo tiñe de un color.
