layer_sprite_get_id

Esta función puede utilizarse para recuperar el valor de identificación único de un elemento de sprite en una capa. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create() o cuando se utiliza el nombre de la capa junto con layer_get_id()) y el nombre del elemento sprite tal y como está definido en el Room Editor. La función devolverá el valor del ID asociado a ese elemento sprite en la capa.

Tenga en cuenta que esta función está diseñada específicamente para su uso con sprites que han sido añadidos en el IDE en una capa de activos, y si ha añadido un sprite a una capa a través de código utilizando la función layer_sprite_create(), entonces el ID devuelto por esa función debe ser utilizado para todas las referencias futuras (ya que ese elemento sprite no tendrá ningún nombre para ser pasado a esta función).

Si la capa especificada no existe, o no se puede encontrar el elemento sprite dado, la función devolverá -1.

Sintaxis:

layer_sprite_get_id(layer_id, sprite_element_name)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige
sprite_element_name El nombre único del elemento sprite en la capa tal y como se define en el Editor de Salas

Devuelve:

Valor del ID del elemento del sprite (Real)

Ejemplo:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

El código anterior obtendrá el ID de la capa llamada "Activos_árboles" y luego lo utilizará para recuperar el ID del elemento sprite "graphic_254367CB" en esa capa. El ID del elemento sprite recuperado se utilizará para cambiar el elemento sprite del elemento.
