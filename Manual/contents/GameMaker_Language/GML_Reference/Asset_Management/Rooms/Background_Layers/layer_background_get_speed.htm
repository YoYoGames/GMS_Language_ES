layer_background_get_speed

Esta función puede utilizarse para obtener el valor actual del multiplicador de velocidad del elemento de fondo. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando se utiliza la función layer_background_get_id()), y la función devolverá un valor real que representa el multiplicador de velocidad que se está utilizando para animar el sprite. El valor por defecto es 1.

Sintaxis:

layer_background_get_speed(background_element_id);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo para obtener la información

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_speed(back_id) > 0
{
layer_background_speed(back_id, 0);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza para comprobar la velocidad de animación del elemento y si es mayor que 0, se establece en 0.
