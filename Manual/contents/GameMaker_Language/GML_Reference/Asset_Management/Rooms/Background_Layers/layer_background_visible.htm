layer_background_visible

Con esta función se puede cambiar la visibilidad de un elemento de fondo. Se suministra el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando utiliza la función layer_background_get_id()), y luego establece el argumento visible como verdadero o falso. Cuando se establece en true el elemento se mostrará, y cuando se establece en false el elemento no se mostrará. Tenga en cuenta que esto depende de la visibilidad de la capa, e incluso si el elemento de fondo está marcado como visible, no se dibujará si la capa en la que se encuentra está marcada como no visible.

Sintaxis:

layer_background_visible(background_element_id, visible)

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
visible El conmutador de visibilidad, que puede ser verdadero o falso

Devuelve:

N/A

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_visible(back)
{
layer_background_visible(back, false);
}
si no
{
layer_background_visible(back, true);
}
}

El código anterior comprueba si se ha pulsado el botón del ratón y, si se detecta alguno, cambiará la visibilidad del elemento de fondo asignado a la capa en la que se encuentra la instancia que ejecuta el código.
