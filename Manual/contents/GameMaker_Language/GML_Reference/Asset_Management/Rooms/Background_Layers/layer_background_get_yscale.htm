layer_background_get_yscale

Esta función puede utilizarse para obtener el valor del multiplicador de escala actual del elemento de fondo. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando se utiliza la función layer_background_get_id()), y la función devolverá un valor real que representa el multiplicador de escala que se está utilizando para dibujar el sprite. El valor por defecto es 1.

Sintaxis:

layer_background_get_yscale(background_element_id);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo para obtener la información

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_yscale(back_id) != -1
{
layer_background_yscale(back_id, -1);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para comprobar la escala y del elemento y si no es -1, se establece en -1.
