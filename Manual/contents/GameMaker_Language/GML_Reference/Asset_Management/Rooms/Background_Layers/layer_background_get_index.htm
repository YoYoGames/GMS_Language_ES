layer_background_get_index

Esta función se puede utilizar para obtener el valor del índice de la imagen actual del elemento de fondo. Se da el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()), y la función devolverá un valor real que representa el índice de la imagen que se muestra para el sprite. La función devolverá -1 si el elemento de fondo no existe o el elemento no tiene un sprite válido asignado.

Sintaxis:

layer_background_get_index(background_element_id);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo para obtener la información

Devuelve:

Real (el índice de la imagen del sprite actual o -1)

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_index(back_id) < 4
{
layer_background_index(back_id, 4);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza para comprobar si el índice de la imagen del elemento es menor que 4, y si es así se establece en 4.
