layer_background_get_id

Esta función puede utilizarse para recuperar el valor de identificación único del elemento de fondo en una capa. Se suministra el ID de la capa (que se obtiene cuando se utiliza el nombre de la capa junto con layer_get_id()) y la función devolverá el valor de ID asociado con el elemento de fondo en la capa. Tenga en cuenta que esta función está diseñada específicamente para su uso con fondos que han sido añadidos en el IDE, ya que si añade un fondo a una capa a través de código utilizando la función layer_background_create()devolverá el ID único del elemento de fondo añadido.

Sintaxis:

layer_background_get_id(layer_id)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige

Devuelve:

Valor de identificación del elemento de fondo (Real)

Ejemplo:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

El código anterior obtendrá el ID de la capa llamada "Background_trees" y lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para cambiar el sprite del elemento.
