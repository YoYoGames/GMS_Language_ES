layer_background_index

Esta función se puede utilizar para establecer el índice de la imagen del sprite de fondo que tiene múltiples sub-imágenes. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando se utiliza la función layer_background_get_id()), y luego se establece el índice de la imagen a utilizar. Si estableces un valor fuera del rango de sub-imágenes, entonces el índice de la imagen dará vueltas.

Sintaxis:

layer_background_index(background_element_id, image_index);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
índice El índice de la imagen a utilizar para el fondo

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

El código anterior obtendrá el ID de la capa llamada "Árboles_de_fondo" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para cambiar el índice de la imagen del elemento.
