layer_background_stretch

Usando esta función puedes cambiar el sprite de un elemento de fondo para que se estire y se ajuste a la habitación o para que permanezca a 1:1 con la resolución. Se suministra el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo utilizando layer_background_create() o al utilizar la función layer_background_get_id()), y luego establece el argumento de estiramiento como verdadero o falso. Cuando se establece como verdadero, el sprite del elemento se estirará para ajustarse a la habitación (ya sea escalado hacia arriba o hacia abajo dependiendo de los tamaños del sprite y de la habitación), y cuando se establece en falso, el sprite del elemento se mostrará en su resolución nativa.

Sintaxis:

layer_background_stretch(background_element_id, stretch)

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
estiramiento La palanca de estiramiento, que puede ser verdadera o falsa

Devuelve:

N/A

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_stretch(back)
{
layer_background_stretch(back, false);
}
si no
{
layer_background_stretch(back, true);
}
}

El código anterior comprueba si se pulsa el botón del ratón y, si se detecta alguno, conmuta el estiramiento del sprite del elemento de fondo asignado a la capa en la que se encuentra la instancia que ejecuta el código.
