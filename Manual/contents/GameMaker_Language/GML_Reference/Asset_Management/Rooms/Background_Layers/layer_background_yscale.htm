capa_fondo_escala

Esta función puede utilizarse para establecer la escala a lo largo del eje Y de un elemento de fondo. Se da el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()) y, a continuación, se establece el valor de la escala. El valor de la escala que se da es un multiplicador que se utilizará para cambiar la forma en que se muestra el elemento de fondo, donde un valor de 0,5 mostraría el elemento a media escala, y un valor de 2 lo mostraría a doble escala. Tenga en cuenta que los valores negativos son válidos, y "voltearán" el elemento alrededor de la posición (0,0), por lo que una escala x de -1 mostraría la imagen invertida.

Sintaxis:

layer_background_yscale(background_element_id, yscale);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
velocidad El multiplicador/valor de velocidad que se utilizará para el elemento

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_yscale(back_id, -1);

El código anterior obtendrá el ID de la capa llamada "Agua_de_fondo" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para cambiar la escala de y del elemento.
