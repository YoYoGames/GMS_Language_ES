layer_background_get_visible

Esta función se puede utilizar para obtener el estado visible del elemento de fondo. Se da el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o cuando se utiliza la función layer_background_get_id()), y la función devolverá true si el elemento es actualmente visible, o false si no lo es. Tenga en cuenta que este valor de retorno no se ve afectado por si la capa en la que se encuentra el elemento es visible o no.

Sintaxis:

layer_background_get_visible(background_element_id);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo para obtener la información

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_visible(back_id)
{
layer_background_visible(back_id, false);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza para comprobar la visibilidad del elemento y, si es visible, esta propiedad se establece en falso.
