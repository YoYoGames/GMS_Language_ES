layer_background_vtiled

Con esta función puede cambiar si el elemento de fondo dado en una capa debe ser embaldosado verticalmente o no. Se da el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()), y luego se establece el valor del mosaico. Si se establece como verdadero, el elemento se mostrará en mosaico verticalmente en la habitación, y cuando se establece como falso el elemento no será embaldosado. Esta función es sólo para proyectos 2D, y no funcionará correctamente cuando se utilice una proyección de cámara 3D.

Sintaxis:

layer_background_vtiled(background_element_id, tiled);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
visible La alternancia de mosaico vertical, que puede ser verdadera o falsa

Devuelve:

N/A

Ejemplo:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
layer_background_htiled(back, true);
layer_background_vtiled(back, true);
}

El código anterior comprobará el elemento de fondo asignado a la capa en la que se encuentra la instancia que ejecuta el código y, si no está alicatado en ninguna dirección, lo pondrá alicatado vertical y horizontalmente.
