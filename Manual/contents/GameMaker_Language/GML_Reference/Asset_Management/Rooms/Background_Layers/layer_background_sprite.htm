layer_background_sprite

Con esta función puedes establecer el índice del sprite del elemento de fondo. Se suministra el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o cuando se utiliza la función layer_background_get_id()), y a continuación se da un índice de sprite a utilizar. La imagen del elemento de fondo será reemplazada por el nuevo sprite. Si da un valor de -1, el elemento no tendrá ningún sprite asignado (pero seguirá existiendo y se le podrá asignar un sprite más adelante).

Sintaxis:

layer_background_sprite(background_element_id, sprite_index)

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
sprite_index El índice del sprite a utilizar para el elemento de fondo

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
layer_background_sprite(back_id, spr_Clouds);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para comprobar el sprite asignado al elemento, estableciéndolo en el sprite "spr_Clouds" si no lo está ya.
