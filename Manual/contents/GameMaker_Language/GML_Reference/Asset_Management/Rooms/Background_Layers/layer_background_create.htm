layer_background_create

Con esta función puedes asignar un recurso de sprite a una capa para que sea utilizada como fondo en tu proyecto. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y un índice de sprite (que sería el nombre del sprite como se muestra en el Navegador de Activos), y se añadirá a la capa. La función devuelve el valor de ID único para el elemento, que puede utilizarse en otras funciones de la capa para los fondos.

Sintaxis:

layer_background_create(layer_id, sprite)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)
sprite El índice del sprite a utilizar

Devuelve:

ID del elemento de fondo (Real)

Ejemplo:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

El código anterior crea una nueva capa y luego añade un nuevo elemento de fondo a la misma, estableciendo un sprite para ser la imagen de fondo utilizada.
