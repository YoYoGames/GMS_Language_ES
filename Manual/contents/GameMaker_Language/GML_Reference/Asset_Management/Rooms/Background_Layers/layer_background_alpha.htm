layer_background_alpha

Esta función controla el alfa (transparencia) del sprite de fondo. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando se utiliza la función layer_background_get_id()), y luego se establece el valor alfa a utilizar. El valor alfa puede estar entre 0 (totalmente transparente) y 1 (totalmente opaco), siendo el valor alfa por defecto para el elemento de fondo es 1. Tenga en cuenta que si la capa a la que se ha asignado el elemento de fondo no es visible -o el propio elemento se ha hecho invisible- no verá ninguna diferencia con esta función hasta que la capa o el elemento se hayan hecho visibles de nuevo.

Sintaxis:

layer_background_alpha(background_element_id, alpha);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
alfa El alfa para el sprite de fondo, de 0 a 1 (por defecto es 1)

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

El código anterior obtiene el valor ID del fondo asignado a la capa "Background_sky" y luego establece su alfa a un valor aleatorio entre 0 y 1.
