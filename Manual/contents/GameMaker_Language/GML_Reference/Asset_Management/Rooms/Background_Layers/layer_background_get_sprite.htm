layer_background_get_sprite

Esta función puede utilizarse para obtener el valor del índice del sprite actual del elemento de fondo. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o al utilizar la función layer_background_get_id()), y la función devolverá un valor real que representa el índice del sprite que se está mostrando. Si el elemento no tiene ningún sprite asignado, la función devolverá -1.

Sintaxis:

layer_background_get_sprite(background_element_id);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo para obtener la información

Devuelve:

Índice de Sprite (Real) o -1

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
layer_background_sprite(back_id, spr_Clouds);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para comprobar el sprite asignado al elemento, estableciéndolo en el sprite "spr_Clouds" si no lo está ya.
