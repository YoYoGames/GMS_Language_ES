layer_background_get_stretch

Esta función puede utilizarse para obtener el estado de estiramiento del sprite del elemento de fondo. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando se utiliza la función layer_background_get_id()), y la función devolverá true si el sprite del elemento está actualmente estirado para ajustarse a la habitación, o false si no lo está. no lo está.

Sintaxis:

layer_background_get_stretch(background_element_id);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo para obtener la información

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_stretch(back_id)
{
layer_background_stretch(back_id, false);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza para comprobar si el sprite del elemento será estirado para encajar en la habitación o no y si se estira, entonces esta propiedad se establece en false.
