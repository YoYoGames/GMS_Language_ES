layer_background_exists

Puede utilizar esta función para comprobar si existe un elemento de fondo en una capa determinada. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando utilice la función layer_background_get_id()) y la función devolverá true si el elemento existe, o false si no existe.

NOTA: Esta función trabaja dentro del ámbito de la sala de destino actual -por defecto la sala en la que se llama a la función- que puede establecerse mediante la función layer_set_target_room().

Sintaxis:

layer_background_exists(layer_id, background_element_id)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)
background_element_id El valor de identificación único del elemento de fondo a comprobar

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
layer_background_destroy(lay_id, global.TreesBackground);
}

El código anterior comprueba la capa "Árboles_de_fondo" para ver si el elemento de fondo dado existe y si lo hace, entonces se destruye (pero no la capa).
