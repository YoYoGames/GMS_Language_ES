layer_background_destroy

Esta función destruirá el elemento de fondo dado. Se suministra el ID del fondo (que se obtiene cuando se crea el fondo usando layer_background_create() o cuando se utiliza el ID de la capa junto con con layer_get_background_id()) y esto lo eliminará. Tenga en cuenta que esto no elimina la capa, sólo el fondo de la misma, y si el fondo es uno que se ha añadido en la sala la próxima vez que salga de la sala y vuelva, el fondo se volverá a crear. Sin embargo, si la sala es persistente, el fondo se eliminará a menos que la persistencia de la sala se desactive de nuevo.

Sintaxis:

layer_background_destroy(background_element_id)

Argumento Descripción
background_element_id El valor de identificación único del fondo que se va a destruir

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if layer_background_exists(lay_id, bck_id)
{
layer_background_destroy(bck_id);
}

El código anterior comprueba la capa "Árboles_de_fondo" para ver si el elemento de fondo dado existe y si lo hace, entonces se destruye (pero no la capa).
