mezcla_de_fondo_de_capas

Esta función controla la mezcla (o "tintado") del sprite de fondo y el valor por defecto es -1 (que representa la constante c_white, que también puede utilizarse). Cualquier otro valor (incluyendo las constantes de color internas como c_red, o c_aqua) mezclará el color especificado con el sprite original. Le das el ID del elemento de fondo (que obtienes cuando creas un elemento de fondo usando layer_background_create() o al utilizar la función layer_background_get_id()), y luego establece el color de mezcla a utilizar. A continuación puedes ver un ejemplo de un sprite que ha sido mezclado con diferentes colores:

Tenga en cuenta que debe intentar limitar la mezcla en la plataforma HTML5 (a menos que utilice WebGL), ya que cada sprite mezclado tiene que ser almacenado en caché por separado y, por lo tanto, tener muchos sprites mezclados puede afectar negativamente al rendimiento (también se puede establecer el tamaño de la caché utilizando la función sprite_set_cache_size()).

Sintaxis:

layer_background_blend(background_element_id, blend);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
mezcla El color a mezclar con el sprite de fondo (por defecto es c_white)

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_blend(back_id, c_aqua);

El código anterior obtiene el valor del ID del fondo asignado a la capa "Background_sky" y luego lo tiñe de un color.
