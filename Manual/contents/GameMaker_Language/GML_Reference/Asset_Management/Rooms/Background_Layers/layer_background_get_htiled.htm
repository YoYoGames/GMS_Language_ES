layer_background_get_htiled

Esta función puede utilizarse para averiguar si el elemento de fondo está en mosaico horizontal o no. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o cuando se utiliza la función layer_background_get_id()), y la función devolverá true si el elemento está actualmente en mosaico, o false si no lo está.

Sintaxis:

layer_background_get_htiled(background_element_id);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo para obtener la información

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_htiled(back_id)
{
layer_background_htiled(back_id, true);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza para comprobar si el elemento está en mosaico a través del eje horizontal, y si si no lo está, se ajusta para que lo haga.
