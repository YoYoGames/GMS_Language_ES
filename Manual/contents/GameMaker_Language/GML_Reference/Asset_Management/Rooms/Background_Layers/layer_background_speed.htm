capa_fondo_velocidad

Esta función puede utilizarse para establecer la velocidad de un sprite animado que ha sido asignado a un elemento de fondo. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o al utilizar la función layer_background_get_id()), y luego se establece la velocidad de la animación. Si el IDE ha establecido que la velocidad de animación para el sprite utilizado se calcule como Fotogramas por Fotograma de Juego, entonces el valor de la velocidad que se da es un multiplicador, siendo 1 el valor por defecto, y fijándolo en 0.5 se reducirá a la mitad la velocidad de la animación, mientras que fijándolo en 2 se duplicará. Sin embargo, si el IDE ha establecido que la velocidad de la animación para el sprite utilizado sea calculada como fotogramas por segundo, entonces el valor de la velocidad dada es un valor literal de fotogramas por segundo, por lo que si se establece en 5, por ejemplo, se establecerá la animación para hacer un bucle de 5 fotogramas cada segundo que el juego se ejecute. Si el elemento de fondo no tiene un sprite adjunto o el sprite utilizado no tiene sub-imágenes, esta función no tendrá efecto.

Sintaxis:

layer_background_speed(background_element_id, speed);

Argumento Descripción
background_element_id El valor de identificación único del elemento de fondo que se va a cambiar
velocidad El multiplicador/valor de velocidad que se utilizará para el elemento

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

El código anterior obtendrá el ID de la capa llamada "Agua_de_fondo" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para cambiar la velocidad de la imagen del elemento.
