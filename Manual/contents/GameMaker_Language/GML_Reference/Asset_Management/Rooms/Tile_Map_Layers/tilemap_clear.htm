tilemap_clear

Con esta función puedes borrar/configurar todos los azulejos de un mapa de azulejos determinado. Se le da el ID del elemento del mapa de azulejos (que se obtiene cuando se crea un elemento del mapa de azulejos usando layer_tilemap_create() o cuando se utiliza la función la función layer_tilemap_get_id()), y luego suministrar los datos del mosaico que desea limpiar la capa. Un valor por defecto de 0 borrará todos los azulejos de la capa (esencialmente haciendo que todos los azulejos estén "vacíos"), mientras que puede utilizar las funciones dedicadas de tile_* para crear sus propios datos de azulejos para limpiar el mapa de azulejos.

Sintaxis:

tilemap_clear(tilemap_element_id, tiledata)

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos que se va a cambiar
tiledata Los datos del azulejo que se utilizarán para borrar la capa

Devuelve:

N/A

Ejemplo:

var layer_id = layer_get_id("Bosque");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

El código anterior obtiene el valor de la ID de un mapa de baldosas creado en el editor de salas, y luego lo borra usando baldosas "vacías".
