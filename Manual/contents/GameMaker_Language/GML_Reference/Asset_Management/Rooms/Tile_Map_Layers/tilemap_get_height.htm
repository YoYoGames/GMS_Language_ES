tilemap_get_height

Con esta función se puede obtener la altura (en celdas) del elemento del mapa de azulejos. Se le da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos usando layer_tilemap_create() o cuando se utiliza la función layer_tilemap_get_id()), y la función devolverá la altura (número de celdas de ancho).

Sintaxis:

tilemap_get_height(tilemap_element_id);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener la altura de

Devuelve:

Entero

Ejemplo:

var lay_id = layer_get_id("alpargatas");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

El código anterior utiliza el ID del mapa de azulejos recuperado para obtener la anchura y la altura del mapa de azulejos y luego utiliza esos valores para establecer una variable de instancia.
