tilemap_x

Esta función controla la posición a lo largo del eje x de la habitación del elemento de mapa de azulejos del activo en la capa. Se le da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y luego establece el valor x a utilizar (basado en las coordenadas de la habitación).

Sintaxis:

tilemap_x(tilemap_elemento_id, x);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos que se va a cambiar
x La posición x para el mapa de azulejos

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

El código anterior obtiene el valor del ID del activo del mapa de azulejos asignado a la capa "Asset_sky" y luego establece su posición x a un valor aleatorio entre 0 y el ancho de la habitación.
