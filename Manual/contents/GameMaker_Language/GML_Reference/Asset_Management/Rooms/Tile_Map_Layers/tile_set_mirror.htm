tile_set_mirror

Esta función se puede utilizar para establecer un conjunto dado de datos de azulejos para reflejar el azulejo o no. Se dan los datos del mosaico, que pueden ser recuperados usando la función tilemap_get()y, a continuación, se establece el argumento de reflejo como verdadero si se desea que el mosaico se refleje, o falso si se desea que el mosaico esté en su estado predeterminado, sin reflejo. La función devolverá un conjunto de datos de mosaico modificado que puede aplicarse con la función tilemap_set() función.

Sintaxis:

tile_set_mirror(tiledata, mirror)

Argumento Descripción
tiledata los datos de los azulejos que hay que fijar
espejo Establecer en true para reflejar y false para dejarlo como está

Devuelve:

Real (tiledata)

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID del mapa de azulejos de la capa dada y luego obtiene la posición de la celda x e y para el azulejo bajo el ratón. Esta posición se utiliza para obtener los datos del mosaico, que se reflejan y se utilizan para establecer el mosaico de nuevo.
