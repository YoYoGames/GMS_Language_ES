tilemap_get_at_pixel

Con esta función puedes recuperar los datos de los azulejos de una posición (dentro de la habitación) del elemento del mapa de azulejos. Se le da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), así como la posición x e y en la sala de la que se desea obtener los datos de los azulejos, y la función devolverá el "blob" de datos de los azulejos. Estos datos es esencialmente una máscara de bits que contiene el índice de la baldosa, los booleanos de volteo/rotación/espejo y cualquier dato de máscara que se haya aplicado (ver tilemap_set_mask() para más detalles), y el valor de los datos resultantes puede puede utilizarse en las funciones de tile_* para cambiar las propiedades de un mosaico. Si necesita obtener los datos de un mosaico específico, debería utilizar la función tilemap_get() en su lugar.

IMPORTANTE! Si los azulejos del mapa de azulejos no se han modificado (es decir, no se han girado o volteado, etc.), entonces el valor de retorno de los datos del conjunto de azulejos será exactamente igual al índice del azulejo en el conjunto de azulejos. Por lo tanto, puede crear "mapas de colisión de azulejos utilizando un azulejo en el índice 1 del conjunto de azulejos - por ejemplo - y luego utilizar esta función para comprobar 1 o 0 (un azulejo vacío) para calcular las colisiones.

Sintaxis:

tilemap_get_at_pixel(tilemap_element_id, x, y);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener los datos de los azulejos
x La posición a lo largo del eje x para obtener los datos del mosaico (en píxeles de la habitación)
y La posición a lo largo del eje y para obtener los datos del mosaico (en píxeles de la habitación)

Devuelve:

Real (-1 si hay un error)

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

El código anterior obtiene el ID del mapa de azulejos "Nubes" en la capa "Tiles_Sky" y luego lo utiliza para obtener los datos del azulejo en la posición (64, 128). Estos datos del azulejo se voltean antes de ser utilizados para establecer el azulejo en el mapa de azulejos de nuevo.
