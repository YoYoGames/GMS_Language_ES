tilemap_get_cell_at_pixel

Con esta función puede recuperar la posición del eje x de una celda individual del mapa de azulejos dando la posición relativa del eje x dentro de la habitación. Se da el ID del elemento del mapa de azulejos (que se obtiene cuando se crea un elemento del mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), así como la posición x e y dentro de la habitación y la función devolverá la posición x de la celda dentro del mapa de azulejos para ese punto. Tenga en cuenta que si el valor está fuera del área del mapa de azulejos, y no hay ninguna celda disponible, devolverá -1.

Sintaxis:

tilemap_get_cell_at_pixel(tilemap_element_id, x, y);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener la posición x de la celda
x La posición x dentro de la habitación que se utilizará para obtener la célula
y La posición y dentro de la habitación que se utilizará para obtener la célula

Devuelve:

Entero (posición de la celda del eje x o -1 si hay un error)

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

El código anterior utiliza el ID del mapa de baldosas recuperado para obtener la posición de la celda a partir de un conjunto de coordenadas de la habitación y, a continuación, almacena los datos de cualquier baldosa que se encuentre allí en una variable de instancia.
