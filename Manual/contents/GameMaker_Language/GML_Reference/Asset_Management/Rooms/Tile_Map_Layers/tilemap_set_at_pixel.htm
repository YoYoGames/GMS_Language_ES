tilemap_set_at_pixel

Esta función se puede utilizar para establecer una celda dentro del elemento de mapa de azulejos en la capa a un nuevo azulejo utilizando la posición real del azulejo para cambiar dentro de la habitación. Se da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y luego los datos de la baldosa a establecer, así como la posición dentro de la habitación. Usted puede recuperar los datos de las baldosas utilizando la función tilemap_get_at_pixel() y luego usar las funciones tile_ para cambiarlos antes de establecerlos de nuevo usando esta función. La función devolverá true si el azulejo se ha establecido con éxito y false si ha habido un problema y no se ha establecido.

Sintaxis:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos que se va a cambiar
tiledata Los datos del conjunto de azulejos a establecer
xcell La posición x (en la sala)
ycell La posición y (en la sala)

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

El código anterior obtiene el ID del mapa de azulejos en la capa "Tiles_Sky" y luego lo utiliza para obtener los datos del azulejo en la posición (4, 4). Estos datos del azulejo se voltean antes de ser utilizados para establecer el azulejo en el mapa de azulejos de nuevo.
