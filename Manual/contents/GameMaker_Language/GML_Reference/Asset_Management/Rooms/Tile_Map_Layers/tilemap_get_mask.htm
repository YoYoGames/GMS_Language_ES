tilemap_get_mask

Esta función puede utilizarse para recuperar el valor de la máscara de bits para el mapa de azulejos dado. Se le da el ID del elemento del mapa de azulejos (que se obtiene cuando se crea un elemento del mapa de azulejos usando layer_tilemap_create() o cuando se utiliza la función layer_tilemap_get_id()), y la función devolverá el valor de la máscara actual o -1 si hay un error o 0 si no se especifica ninguna máscara. Para más información sobre las máscaras de bits del mapa de azulejos consulte la función tilemap_set_mask().

Sintaxis:

tilemap_get_mask(tilemap_element_id);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener la máscara de

Devuelve:

Real (0 para ninguna máscara, -1 para un error)

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
tilemap_set_mask(map_id, new_mask);
}

El código anterior obtiene el ID del mapa de azulejos de la capa dada y luego comprueba el valor de la máscara asociada a ella. Si no es el mismo que el valor definido en la variable "new_mask", entonces se establece a ese valor.
