tilemap_set

Esta función puede utilizarse para establecer cualquier celda (cuadrado de la cuadrícula) dentro del elemento de mapa de azulejos en la capa a un nuevo azulejo. Se le da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()) y, a continuación, los datos del mosaico que hay que establecer, así como la posición dentro del mapa de mosaicos. Puede recuperar los datos del mosaico utilizando la función tilemap_get() y luego usar las funciones tile_ para cambiarlos antes de establecer la celda usando esta función. La función devolverá true si la baldosa se ha establecido con éxito y false si ha habido un problema y no se ha establecido.

Sintaxis:

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos que se va a cambiar
tiledata Los datos de los azulejos que hay que fijar
xcell La posición de la celda (rejilla) a establecer a lo largo del eje x
ycell La posición de la celda (rejilla) a establecer a lo largo del eje y

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

El código anterior obtiene el ID del mapa de azulejos en la capa "Tiles_Sky" y luego lo utiliza para obtener los datos del azulejo en la celda (0, 0). Estos datos del azulejo se voltean antes de ser utilizados para establecer el azulejo en el mapa de azulejos de nuevo.
