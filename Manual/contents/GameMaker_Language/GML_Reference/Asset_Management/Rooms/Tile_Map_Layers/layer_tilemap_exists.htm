layer_tilemap_exists

Puede utilizar esta función para comprobar si existe un elemento de mapa de azulejos en una capa determinada. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()) y la función devolverá true si el elemento existe, o false si no existe.

NOTA: Esta función trabaja dentro del ámbito de la sala de destino actual -por defecto la sala en la que se llama a la función- que puede establecerse mediante la función layer_set_target_room().

Sintaxis:

layer_tilemap_exists(layer_id, tilemap_element_id)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos que se va a comprobar

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("tilemap_trees");
if layer_tilemap_exists(lay_id, global.Treestilemap)
{
layer_tilemap_destroy(lay_id, global.Treestilemap);
}

El código anterior comprueba la capa "tilemap_trees" para ver si el elemento de mapa de azulejos dado existe y si lo hace, entonces se destruye (pero no la capa).
