tile_set_index

Esta función puede utilizarse para establecer el índice de los azulejos (la posición del azulejo dentro de la imagen del conjunto de azulejos) para un conjunto de datos de azulejos. Se dan los datos de los mosaicos, que se pueden recuperar con la función tilemap_get()y el valor entero que representa la posición de la baldosa (su índice) dentro del conjunto de baldosas. La función devolverá un conjunto de datos de baldosas modificado que puede aplicarse con la función tilemap_set() función.

Sintaxis:

tile_set_index(tiledata, index)

Argumento Descripción
tiledata los datos de los azulejos que hay que fijar
índice El valor del índice entero del conjunto de azulejos a utilizar

Devuelve:

Real (tiledata)

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID del mapa de azulejos de la capa dada y luego lo utiliza para obtener los datos de los azulejos de la celda bajo la posición del ratón. Estos datos se utilizan para establecer el índice de la baldosa con un número aleatorio.
