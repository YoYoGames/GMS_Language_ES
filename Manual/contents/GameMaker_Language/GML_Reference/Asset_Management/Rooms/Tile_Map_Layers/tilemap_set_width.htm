tilemap_set_width

Esta función puede utilizarse para cambiar el tamaño de un elemento de mapa de azulejos. Se le da el ID del elemento del mapa de azulejos (que se obtiene cuando se crea un elemento del mapa de azulejos utilizando layer_tilemap_create() o cuando se utiliza la función layer_tilemap_get_id()), y el nuevo ancho del mapa de azulejos en celdas de azulejos.

Sintaxis:

tilemap_set_width(tilemap_element_id, width)

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para establecer el ancho de
ancho El valor de la anchura (en "celdas")

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_width(map_id) != room_width div 16
{
tilemap_set_width(map_id, room_width div 16);
}

El código anterior comprueba la anchura de un mapa de azulejos específico y si no es el tamaño correcto, entonces se establece la anchura.
