tilemap_set_height

Esta función puede utilizarse para cambiar el tamaño de un elemento de mapa de azulejos. Se le da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos utilizando layer_tilemap_create() o cuando se utiliza la función layer_tilemap_get_id()), y la nueva altura del mapa de azulejos en celdas de azulejos.

Sintaxis:

tilemap_set_height(tilemap_element_id, height)

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para establecer la altura de
altura El valor de la altura (en "celdas")

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_height(map_id) != room_height div 16
{
tilemap_set_height(map_id, room_height div 16);
}

El código anterior comprueba la anchura de un mapa de azulejos específico y si no es el tamaño correcto, entonces se establece la anchura.
