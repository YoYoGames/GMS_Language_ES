tilemap_get

Con esta función puede recuperar los datos de los azulejos de una celda del elemento de mapa de azulejos. Se da el ID del elemento de mapa de azulejos (que se obtiene al crear un elemento de mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), así como la posición x e y de la cel da, y la función devolverá el "blob" de datos del mosaico. Estos datos son esencialmente una máscara de bits que contiene el índice del mosaico, los booleanos de volteo/rotación/espejo y cualquier dato de máscara que se haya aplicado (véase tilemap_set_mask() para más detalles), y el valor de los datos resultantes puede utilizarse en las funciones de tile_* para cambiar las propiedades de los azulejos. Tenga en cuenta que estamos utilizando las posiciones de las celdas x/y basadas en el tamaño de la cuadrícula del mapa de azulejos y no como una posición en la habitación, por lo que "cell_x" es un valor de 0 a la anchura del mapa de azulejos , y "cell_y" es un valor de 0 a la altura del mapa de azulejos. Si necesitas obtener los datos de una baldosa en una posición específica de la sala, entonces debes utilizar la función tilemap_get_at_pixel().

IMPORTANTE! Si los azulejos del mapa de azulejos no se han modificado (es decir, no se han girado o volteado, etc.), entonces el valor de retorno de los datos del conjunto de azulejos será exactamente igual al índice del azulejo en el conjunto de azulejos. Por lo tanto, puede crear "mapas de colisión de azulejos utilizando un azulejo en el índice 1 en el conjunto de azulejos - por ejemplo - y luego utilizar esta función para comprobar 1 o 0 (un azulejo vacío) para calcular las colisiones.

Sintaxis:

tilemap_get(tilemap_element_id, cell_x, cell_y);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener los datos de los azulejos
x_celda La celda a lo largo del eje x para obtener los datos del mosaico
y_cell La celda a lo largo del eje y para obtener los datos del mosaico

Devuelve:

Real (-1 si hay un error)

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

El código anterior obtiene el ID del mapa de azulejos en la capa "Tiles_Sky" y luego lo utiliza para obtener los datos del azulejo en la celda (0, 0). Estos datos del azulejo se voltean antes de ser utilizados para establecer el azulejo en el mapa de azulejos de nuevo.
