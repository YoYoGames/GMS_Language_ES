tilemap_tileset

Con esta función se puede cambiar el recurso del conjunto de azulejos asignado a un determinado elemento del mapa de azulejos en una capa. Se da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y luego suministrar un índice de conjunto de azulejos y el mapa de azulejos recibirá el nuevo sprite.

Sintaxis:

tilemap_tileset(tilemap_elemento_id, tileset_index)

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos que se va a cambiar
tileset_index El nuevo índice del conjunto de azulejos a utilizar

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(tile_id) != ts_Nighttime
{
tilemap_tileset(tile_id, ts_Nighttime);
}

El código anterior comprueba el conjunto de azulejos actual asignado al mapa de azulejos en la capa "Tiles_trees" y si no es "ts_Nighttime" entonces ese conjunto de azulejos se asigna al mapa de azulejos.
