tilemap_get_tile_width

Con esta función se puede obtener la anchura (en píxeles) de cada celda del elemento del mapa de azulejos. Se le da el ID del elemento de mapa de azulejos (que se obtiene cuando se crea un elemento de mapa de azulejos usando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y la función devolverá la anchura de la celda del conjunto de azulejos.

Sintaxis:

tilemap_get_tile_width(tilemap_element_id);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener el ancho de la celda del azulejo

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

El código anterior utiliza el ID del mapa de azulejos recuperado para obtener la anchura y la altura de las celdas de los azulejos y los utiliza para establecer dos variables globales.
