tilemap_get_tileset

Con esta función se puede recuperar el valor del índice del recurso del conjunto de azulejos asignado a un elemento de mapa de azulejos determinado en una capa. Se da el ID del elemento de mapa de azulejos (que se obtiene al crear un elemento de mapa de azulejos con layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y la función devolverá el valor del índice del conjunto de azulejos.

Sintaxis:

tilemap_get_tileset(tilemap_element_id);

Argumento Descripción
tilemap_element_id El valor de identificación único del elemento del mapa de azulejos para obtener el conjunto de azulejos

Devuelve:

Azulejo Establecer valor de índice (Real)

Ejemplo:

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(map_id) != ts_Nighttime
{
tilemap_tileset(map_id, ts_Nighttime);
}

El código anterior comprueba el conjunto de azulejos actual asignado a la capa "Tiles_trees" y si no es "ts_Nighttime" entonces ese conjunto de azulejos se asigna al mapa de azulejos.
