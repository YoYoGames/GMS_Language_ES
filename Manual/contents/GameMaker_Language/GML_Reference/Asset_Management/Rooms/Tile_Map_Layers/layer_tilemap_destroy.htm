layer_tilemap_destroy

Esta función destruirá el elemento del mapa de azulejos dado. Se suministra el ID del mapa de azulejos (que se obtiene cuando se crea el mapa de azulejos utilizando layer_tilemap_create() o cuando se utiliza el ID de la capa junto con layer_get_tilemap_id()) y esto lo eliminará. Tenga en cuenta que esto no elimina la capa, sólo el mapa de azulejos de la misma, y si el mapa de azulejos es uno que ha sido añadido en el editor de la sala, entonces la próxima vez que salga de la sala y luego regrese, el mapa de azulejos será recreado de nuevo. Sin embargo, si la habitación es persistente, el mapa de azulejos se eliminará a menos que la persistencia de la habitación se desactive de nuevo.

Sintaxis:

layer_tilemap_destroy(tilemap_element_id)

Argumento Descripción
tilemap_element_id El valor de identificación único del mapa de azulejos que se va a destruir

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if layer_tilemap_exists(lay_id, tile_id)
{
layer_tilemap_destroy(tile_id);
}

El código anterior comprueba la capa "Tiles_trees" para ver si el elemento del mapa de azulejos dado existe y si lo hace, entonces se destruye (pero no la capa).
