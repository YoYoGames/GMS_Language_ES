layer_tilemap_get_id

Esta función se puede utilizar para recuperar el valor de ID único del elemento de mapa de azulejos en una capa. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y la función devolverá el valor de ID asociado con el elemento del mapa de azulejos en la capa. Tenga en cuenta que esta función está diseñada específicamente para su uso con mapas de azulejos que han sido añadidos en el IDE, ya que si se añade un mapa de azulejos a una capa a través de código utilizando la función layer_tilemap_create()devolverá el ID único del elemento de mapa de azulejos añadido. Si el ID del mapa de azulejos dado es incorrecto o el tilemap no existe, la función devolverá -1.

Sintaxis:

layer_tilemap_get_id(layer_id)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)

Devuelve:

valor del ID del elemento del mapa de azulejos (Real) o -1

Ejemplo:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

El código anterior obtendrá el ID de la capa llamada "tilemap_trees" y luego lo utilizará para obtener el ID del elemento de mapa de azulejos en esa capa. Este ID se utilizará para eliminar el mapa de azulejos de la capa.
