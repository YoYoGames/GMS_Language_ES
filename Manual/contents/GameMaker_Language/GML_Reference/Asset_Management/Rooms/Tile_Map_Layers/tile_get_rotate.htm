tile_get_rotate

Esta función puede utilizarse para comprobar si en un conjunto dado de datos de baldosas la baldosa ha sido girada 90 grados o no. Se dan los datos de la baldosa, que se pueden recuperar con la función tilemap_get()y la función devolverá true si el azulejo está rotado, o false si no lo está.

Sintaxis:

tile_get_rotate(tiledata)

Argumento Descripción
tiledata los datos de las baldosas que se van a comprobar

Devuelve:

Booleano

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID del mapa de azulejos de la capa dada y luego obtiene la posición de la celda x e y para el azulejo bajo el ratón. Esta posición se utiliza para obtener los datos del mosaico, que se gira y se utiliza para establecer el mosaico de nuevo.
