room_get_name

Esta función puede utilizarse para devolver el nombre de la sala especificada como una cadena. Tenga en cuenta que esto es sólo una cadena y no se puede utilizar para hacer referencia a la habitación directamente - para que usted necesita el índice de la habitación. Sin embargo, puede utilizar esta cadena para obtener el índice de la sala utilizando la cadena devuelta junto con la función asset_get_index().

Sintaxis:

room_get_name(index);

Argumento Descripción
índice El índice de la sala para comprobar el nombre.

Devuelve:

Cadena

Ejemplo:

var roomname = room_get_name(room);
draw_text(32, 32, roomname);

El código anterior obtendrá el nombre de la habitación actual y lo dibujará en la pantalla.
