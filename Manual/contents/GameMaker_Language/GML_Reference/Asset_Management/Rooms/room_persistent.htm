habitación_persistente

Esta variable se puede utilizar para obtener y establecer la bandera persistente para la sala actual. Si se establece en true la sala se considera persistente, en cuyo caso cada vez que se abandona la sala y se vuelve a ella el estado de las instancias dentro de esa sala se habrán mantenido. Sin embargo, si se marca como falso, cada vez que vuelvas a la sala se restablecerá su estado inicial. Debe tener en cuenta que una sala persistente utiliza considerablemente más memoria que una sala normal y no es recomendable tener muchas de ellas en tu juego.

Sintaxis:

room_persistent;

Devuelve:

Booleano

Ejemplo:

si vidas < 1
{
room_persistent = false;
room_goto(rm_start);
}

El código anterior comprueba la variable "vidas" y si es menor que 1, establecerá la persistencia de la habitación en falso y luego cambiará de habitación.
