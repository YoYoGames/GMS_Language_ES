habitación_última

Esta variable de sólo lectura devuelve el índice de la última sala del juego (esto se define por el orden en que aparecen las salas en el Gestor de Salas y no por el orden en que fueron creadas). Tenga en cuenta que esta variable no reconocerá ni tendrá en cuenta las salas que se hayan añadido dinámicamente utilizando room_add() o room_duplicate().

Sintaxis:

room_last;

Devuelve:

Índice

Ejemplo:

if keyboard_check_pressed(ord("Q"))
{
room_goto(room_last);
}

El código anterior comprueba si se ha pulsado una tecla y, si es así, va a la última habitación del juego.
