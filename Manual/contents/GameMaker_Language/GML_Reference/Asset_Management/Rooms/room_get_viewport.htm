room_get_viewport

Con esta función puedes recuperar los detalles de un puerto de vista en una sala distinta a la actual. Se da el ID de la sala y el índice del puerto de vista a recuperar (de 0 a 7) y la función devolverá un array de 5 índices, donde:

  • [0] = visible (booleano: verdadero / falso)
  • [1] = posición x (real)
  • [2] = posición y (real)
  • [3] = ancho (real)
  • [4] = altura (real)

Sintaxis:

room_get_viewport(rm, vind);

Argumento Descripción
rm El índice de la sala para obtener los datos de la ventana gráfica
vind El índice del puerto de la vista a obtener

Devuelve:

Array (5 elementos: visible, x, y, anchura, altura)

Ejemplo:

v_vals = room_get_viewport(rm_Game, 0);
if v_vals[0] == false
{
room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

El código anterior recupera los datos del puerto de vista para la habitación dada y luego comprueba si el puerto está marcado como visible. Si no lo está, los datos del puerto de vista se establecen para hacerlo visible.
