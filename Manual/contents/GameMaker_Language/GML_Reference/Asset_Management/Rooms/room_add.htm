room_add

Esta función creará una nueva sala vacía y la añadirá a su juego, devolviendo su índice para ser almacenado en una variable para todos los códigos posteriores que traten con esta sala. Tenga en cuenta que cada sala se añade permanentemente al juego hasta que se cierre el ejecutable, es decir, las habitaciones añadidas mediante código no pueden volver a borrarse. Esto tiene importantes implicaciones para el uso de la memoria, por lo que debe utilizar esta función con cuidado.

NOTA: Las salas nuevas no forman parte del ordenamiento habitual de las salas, por lo que no tienen una sala "anterior" o "siguiente" (lo que significa que las funciones room_next() y room_previous() no funcionarán). Para saltar de una sala añadida a otra, debe utilizar el índice de la propia sala.

Sintaxis:

room_add();

Devuelve:

Índice

Ejemplo:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Esto creará una nueva sala y almacenará su índice en la variable "global.myroom". A continuación, establecerá su anchura a 640 píxeles, su altura a 480 píxeles y su persistencia a false.
