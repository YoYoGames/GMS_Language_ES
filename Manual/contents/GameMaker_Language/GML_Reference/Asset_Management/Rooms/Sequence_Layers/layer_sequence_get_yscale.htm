layer_sequence_get_yscale

Con esta función se suministra el ID del elemento de la secuencia - como el devuelto por layer_sequence_create() o por una de las funciones del elemento de la capa - y devolverá la escala actual a lo largo del eje Y del elemento de la secuencia en la sala de juego.

Sintaxis:

layer_sequence_get_yscale(sequence_element_id)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

Real

Ejemplo:

var _ys = layer_sequence_get_yscale(title_sequence)
if _ys < 1
{
_ys += 0.01;
layer_sequence_yscale(title_sequence, _ys);
}

El código anterior recupera la escala actual a lo largo del eje Y del elemento de la secuencia con el ID almacenado en la variable "title_sequence", y si es menor que 1, entonces se añade 0,01 a la escala Y actual.
