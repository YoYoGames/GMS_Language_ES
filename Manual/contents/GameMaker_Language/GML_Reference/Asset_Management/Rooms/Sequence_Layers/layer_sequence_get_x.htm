secuencia_de_capas_obtener_x

Con esta función se suministra el ID del elemento de la secuencia - como el devuelto por layer_sequence_create() o por una de las funciones de elemento de capa - y devolverá la posición X actual en la sala de juego para la secuencia.

Sintaxis:

layer_sequence_get_x(sequence_element_id)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

Real

Ejemplo:

if layer_sequence_get_x(title_sequence) != room_width / 2
{
layer_sequence_x(title_sequence, room_width / 2);
}

El código anterior comprueba la posición X del ID del elemento de la secuencia almacenado en la variable "title_sequence", y si no está ajustado a la mitad de la anchura de la sala, se ajusta a este valor.
