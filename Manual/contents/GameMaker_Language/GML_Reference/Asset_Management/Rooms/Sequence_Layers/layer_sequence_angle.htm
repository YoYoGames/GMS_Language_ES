ángulo_de_secuencia_de_capa

Con esta función se puede establecer el ángulo de rotación para el elemento de secuencia dado. Se suministra el ID del elemento de la secuencia devuelto por layer_sequence_create() o por una de las funciones del elemento de la capa junto con el nuevo ángulo a establecer y la secuencia será girada sobre su origen a la nueva posición. Los ángulos en GameMaker se calculan con 0º a la derecha, y van en sentido contrario a las agujas del reloj - por lo que 90º es arriba, 180º es izquierda y 270º es abajo - y el ángulo por defecto para una secuencia sería 0º.

Tenga en cuenta que no debe cambiar las variables image_xscale \ image_yscale \ image_angle \ x \ y para cualquier instancia en una secuencia que utilice esta función, ya que se sobrescribirán tan pronto como la secuencia se actualice después de llamar a la función.

Sintaxis:

capa_secuencia_ángulo(secuencia_elemento_id, ángulo)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige
ángulo El nuevo ángulo al que se girará el elemento de la secuencia

Devuelve:

N/A

Ejemplo:

si ángulo_actual < 90
{
ángulo_actual += 1;
ángulo_secuencia_capa(mi_seq, ángulo_actual);
}

El código anterior comprueba el valor mantenido en la variable current_angle, y si es menor que 90 entonces lo suma y luego utiliza el valor para establecer el ángulo del elemento de la secuencia referenciado en la variable my_seq.
