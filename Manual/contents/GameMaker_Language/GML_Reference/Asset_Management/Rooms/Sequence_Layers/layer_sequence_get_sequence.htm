secuencia_de_capa_obtener_secuencia

Con esta función se suministra el ID del elemento de la secuencia - como el devuelto por layer_sequence_create() o por una de las funciones del elemento de la capa - y devolverá la estructura del objeto de la secuencia. Esta función evita la necesidad de obtener primero la estructura de la instancia de la secuencia y le permite acceder a los datos de la secuencia directamente. Puede obtener más información sobre el formato de la estructura del objeto de secuencia en esta página.

Sintaxis:

layer_sequence_get_sequence(sequence_element_id)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

Estructurar

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

El código anterior crea un nuevo elemento de secuencia y luego recupera su estructura de datos de secuencia. Esto se utiliza entonces para cambiar la velocidad de reproducción y el tipo de la secuencia.
