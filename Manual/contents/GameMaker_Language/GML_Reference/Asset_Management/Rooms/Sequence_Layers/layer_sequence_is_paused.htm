layer_sequence_is_paused

Con esta función se suministra el ID del elemento de la secuencia -como el devuelto por layer_sequence_create() o por una de las funciones del elemento de la capa - y comprobará si la secuencia está actualmente en pausa o no, devolviendo true si está en pausa, y false si no lo está.

Sintaxis:

layer_sequence_is_paused(sequence_element_id)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

Booleano

Ejemplo:

if layer_sequence_is_paused(title_sequence) != 0
{
layer_sequence_play(title_sequence);
}

El código anterior comprueba si el ID del elemento de la secuencia almacenado en la variable "title_sequence" está en pausa, y si lo está, inicia su reproducción.
