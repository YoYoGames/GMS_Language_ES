ruta_de_cabecera_de_la_capa

Con esta función se puede establecer la dirección del cabezal de reproducción de la secuencia dada. Se proporciona el ID del elemento de la secuencia devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa, y luego da la dirección del cabezal de reproducción que debe ser una de las siguientes constantes:

Constante Descripción Valor
seq_dir_right La secuencia reproducirá los fotogramas en un orden incremental de izquierda a derecha 1
seq_dir_left La secuencia reproducirá los fotogramas en un orden decreciente de derecha a izquierda -1

Sintaxis:

layer_sequence_headdir(sequence_element_id, direction)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige
dirección La dirección del cabezal de reproducción, una constante, indicada anteriormente

Devuelve:

N/A

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

El código anterior crea una nueva secuencia y almacena su ID en una variable local. Este ID se utiliza entonces para establecer la dirección del cabezal de reproducción para disminuir los fotogramas (reproducción de derecha a izquierda).
