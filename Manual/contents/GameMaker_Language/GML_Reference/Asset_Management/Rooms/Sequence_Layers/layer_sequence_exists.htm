secuencia_de_capas_existe

Con esta función puede comprobar si existe un elemento de secuencia en la capa dada. Se suministra el ID de la capa, que puede ser una cadena con el nombre de la capa -como se define en el editor de salas- o el ID único de la capa -como devuelve la función layer_get_id(), así como el ID del elemento de la secuencia - tal y como lo devuelve layer_sequence_create() o por una de las funciones del elemento de la capa - y devolverá true si el elemento dado existe o false en caso contrario.

Sintaxis:

layer_sequence_exists(layer_id, sequence_element_id)

Argumento Descripción
ID_capa El ID o nombre único de la capa a comprobar
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

Booleano

Ejemplo:

if !layer_sequence_exists("Background", my_seq)
{
my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(my_seq);
}

El código anterior comprueba si el elemento de la secuencia dada existe, y si no existe entonces crea una nueva secuencia en la capa dada y luego la pausa.
