layer_sequence_get_headpos

Con esta función se suministra el ID del elemento de la secuencia - como el devuelto por layer_sequence_create() o por una de las funciones del elemento de la capa - y devolverá la posición actual del cabezal de reproducción (el fotograma actual en el que se encuentra el cabezal de reproducción).

Sintaxis:

layer_sequence_get_headpos(sequence_element_id)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

Real

Ejemplo:

if layer_sequence_get_headpos(title_sequence) != 0
{
layer_sequence_headpos(title_sequence, 0);
}

El código anterior comprueba la posición actual del cabezal de reproducción del ID del elemento de la secuencia almacenado en la variable "title_sequence", y si no está a 0, se pone a este valor.
