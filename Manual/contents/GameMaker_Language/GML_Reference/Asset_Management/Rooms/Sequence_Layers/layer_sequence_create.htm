crear_secuencia_de_capa

Con esta función puede crear una instancia de un activo de secuencia en la capa dada. Se suministra el ID de la capa, que puede ser una cadena con el nombre de la capa -como se define en el editor de salas- o el ID único de la capa -como devuelve la función layer_get_id(), así como la posición X e Y en la sala para crear la secuencia, y finalmente el ID de la secuencia a crear. El ID de la secuencia es la constante del nombre que usted definió en el Navegador de Activos para la secuencia. La función devolverá el ID único del elemento de la secuencia, que se puede utilizar en todas las funciones de capa posteriores para las secuencias, o se puede utilizar para recuperar la estructura de la instancia de la secuencia utilizando la función layer_sequence_get_instance().

Sintaxis:

layer_sequence_create(layer_id, x, y, sequence_id)

Argumento Descripción
layer_id El valor de identificación único de la capa de instancia a la que se dirige
x La posición x en la sala para crear la secuencia en
y La posición y en la sala para crear la secuencia en
sequence_id El ID de la secuencia que se va a crear, tal y como se define en el Navegador de Activos

Devuelve:

Entero (ID del elemento)

Ejemplo:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

El código anterior crea una nueva secuencia en la capa "Fondo" y luego la pausa.
