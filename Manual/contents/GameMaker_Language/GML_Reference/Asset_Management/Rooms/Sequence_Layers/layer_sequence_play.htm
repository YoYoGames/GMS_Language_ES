secuencia_de_capa_de_reproducción

Con esta función puede iniciar la reproducción de la secuencia dada. Se suministra el ID del elemento de la secuencia devuelto por layer_sequence_create() o por una de las funciones de elementos de capa y la función reproducirá la secuencia, que luego podrá pausar si lo desea mediante la función layer_sequence_pause().

IMPORTANTE! Si su secuencia tiene algunas instancias en ella, estas instancias no deben cambiar su image_xscale / image_yscale / ángulo_de_imagen / x / y ya que se sobrescribirán cuando la secuencia actualice cada paso después de empezar a reproducirse.

Sintaxis:

layer_sequence_play(sequence_element_id)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("P"))
{
global.Pause = !global.Pause;
var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sequence
{
if global.Pause
{
layer_sequence_pause(a[i]);
}
else
{
layer_sequence_play(a[i]);
}
}
}
}

El código anterior comprueba si el juego ha sido pausado o no cuando se pulsa una tecla. Si el juego está en pausa, entonces recorre todos los elementos de la secuencia en la capa actual (la capa de la instancia que llama) y pausa su reproducción, y si el si el juego no está en pausa, entonces el bucle iniciará su reproducción de nuevo.
