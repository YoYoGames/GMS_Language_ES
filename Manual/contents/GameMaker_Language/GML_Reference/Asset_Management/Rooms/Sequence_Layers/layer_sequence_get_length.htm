secuencia_de_capa_obtener_longitud

Con esta función se suministra el ID del elemento de la secuencia -como el devuelto por layer_sequence_create() o por una de las funciones del elemento de la capa - y devolverá la longitud de la secuencia. Este es el número de fotogramas que durará la secuencia.

Sintaxis:

secuencia_de_capa_obtener_longitud(elemento_de_secuencia_id)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige

Devuelve:

Entero

Ejemplo:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

El código anterior recupera el número de fotogramas que ejecutará una secuencia y luego utiliza este valor para establecer una alarma.
