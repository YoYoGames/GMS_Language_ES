secuencia_de_capas_x

Con esta función puede establecer la posición a lo largo del eje X (horizontal) de la sala para el elemento de secuencia dado. Se suministra el ID del elemento de secuencia devuelto por layer_sequence_create() o por una de las funciones de las funciones del elemento de capa junto con la posición X a establecer y la secuencia se moverá a la nueva posición.

Sintaxis:

capa_secuencia_x(secuencia_elemento_id, pos_x)

Argumento Descripción
sequence_element_id El valor de identificación único del elemento de la secuencia a la que se dirige
pos_x La posición X a la que mover el elemento de la secuencia

Devuelve:

N/A

Ejemplo:

if layer_sequence_exists(my_seq)
{
layer_sequence_x(my_seq, x);
layer_sequence_y(my_seq, y);
}

El código anterior comprueba si el elemento de la secuencia al que se hace referencia en la variable "mi_secuencia" existe, y si lo hace establece la posición x/y a la de la instancia que ejecuta el código.
