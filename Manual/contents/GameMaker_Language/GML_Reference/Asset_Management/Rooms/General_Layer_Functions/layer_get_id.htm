layer_get_id

Esta función se puede utilizar para obtener el valor de ID único de una capa determinada. En el IDE, todas las capas tienen un nombre y un tipo, y para poder editarlas o cambiarlas a través del código hay que dar el valor de ID de la capa. Esta función se utiliza para recuperar este ID utilizando el nombre (una cadena) de la capa (tal y como está escrita en el IDE). Si crea una nueva capa a través de código utilizando la función layer_create() entonces esa función devolverá el valor de ID único en su lugar (las capas creadas dinámicamente no reciben nombres). Tenga en cuenta que si da el nombre de una capa que no existe en la sala actual, entonces obtendrá un error y el proyecto se bloqueará.

Sintaxis:

layer_get_id(nombre_capa)

Argumento Descripción
nombre_de_capa El nombre de la capa (una cadena)

Devuelve:

Valor del ID de la capa (real), o -1 si la capa especificada no existe

Ejemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

El código anterior obtendrá primero el índice de la instancia más cercana a la posición x/y dada y lo almacenará en una variable local. A continuación, obtiene el ID de la capa de instancia única para la capa llamada "Frente de Instancias", y mueve la instancia encontrada a a esa capa.
