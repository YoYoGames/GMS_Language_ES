capa_hspeed

Puede utilizar esta función para establecer la velocidad horizontal (en píxeles por fotograma del juego) de la capa dentro de la sala actual. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y el valor de la velocidad a establecer, donde un valor positivo es a la derecha y un valor negativo a la izquierda.

Sintaxis:

layer_hspeed(layer_id, hspd)

Argumento Descripción
layer_id El valor de identificación único de la capa para establecer la velocidad horizontal de
hspd La velocidad horizontal (en píxeles por fotograma del juego) a establecer

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

El código anterior comprueba las velocidades horizontales y verticales de la capa dada y si no están ambas a 0 entonces las pone a 0.
