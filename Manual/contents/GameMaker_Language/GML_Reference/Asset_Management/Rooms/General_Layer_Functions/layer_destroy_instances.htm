layer_destroy_instances

Esta función puede utilizarse para destruir todas las instancias asignadas a la capa dada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento), y entonces todas las instancias que están en la capa serán eliminadas del juego, activando sus eventos Destroy y Clean Up.

Sintaxis:

layer_destroy_instances(layer_id)

Argumento Descripción
layer_id El valor de identificación único de la capa para establecer la profundidad (o el nombre de la capa como una cadena)

Devuelve:

Real (valor de identificación único para una instancia)

Ejemplo:

si global.game_over
{
layer_destroy_instances(layer);
}

El código anterior comprobará una variable global y si es verdadera entonces todas las instancias que están en la capa de la instancia que llama serán destruidas (incluyendo la instancia que llama).
