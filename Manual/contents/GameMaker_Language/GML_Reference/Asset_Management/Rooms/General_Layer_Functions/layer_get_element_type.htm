layer_get_element_type

Puede utilizar esta función para obtener el tipo de elemento para el elemento dado. Se suministra el valor único del ID del elemento (por ejemplo, el devuelto por la función que creó el elemento o por el editor de salas) y la función devolverá una de las siguientes constantes (o -1 si el elemento no existe o el valor de ID es erróneo):

Constante Descripción
layerelementtype_background El elemento es un fondo.
layerelementtype_instance El elemento es una instancia.
layerelementtype_sprite El elemento es un activo sprite.
layerelementtype_tilemap El elemento es un tilemap.
layerelementtype_particlesystem El elemento es un sistema de partículas.
elemento de capa tipo baldosa El elemento es un azulejo de fondo heredado (esto sólo es válido para los proyectos que se han importado de versiones anteriores de GameMaker).
secuencia_de_tipos_de_capas El elemento es un activo de la secuencia.

Tenga en cuenta que esta función es más útil cuando tiene varios tipos de elementos diferentes asignados a una capa, y que puede obtener una lista de todos los elementos de cualquier capa utilizando la función layer_get_all_elements().

Sintaxis:

layer_get_element_type(element_id)

Argumento Descripción
capa El valor de identificación único del elemento para obtener el tipo de

Devuelve:

Constante (ver tabla, arriba) o -1 si el elemento no existe o es inválido

Ejemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

El código anterior obtiene los IDs de todos los elementos de la instancia asignados a la capa de la instancia que ejecuta el código. A continuación, el código comprueba si alguno de los elementos devueltos es un activo de sprite y, si lo es, se destruye.
