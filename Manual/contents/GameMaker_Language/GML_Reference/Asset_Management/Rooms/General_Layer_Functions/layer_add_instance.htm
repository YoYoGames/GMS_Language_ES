layer_add_instance

Esta función se puede utilizar para mover una instancia dada de la capa en la que se encuentra actualmente a otra capa. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y el ID de la instancia a mover entre capas. Por ejemplo, digamos que su reproductor está en una capa que está a una profundidad menor que otra capa y quiere que aparezca detrás de las capas entre las dos. Puede utilizar esta función para cambiar el reproductor de la capa de primer plano a la capa de fondo y hacer que aparezca detrás de las otras capas que se están dibujando.

NOTA: Esta función no puede utilizarse para añadir una nueva instancia a una capa. Debe haber creado la instancia previamente y haber almacenado su ID en una variable.

NOTA: Esta función sólo es válida dentro del ámbito de la sala actual y no puede utilizarse cuando el ámbito ha sido modificado mediante la función layer_set_target_room().

Sintaxis:

layer_add_instance(layer_id, instance_id)

Argumento Descripción
layer_id El valor de ID único de la capa de instancia a la que se dirige (o el nombre de la capa como cadena)
instance_id El valor del ID de instancia único de la instancia que se va a trasladar a la capa de destino

Devuelve:

N/A

Ejemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

El código anterior obtendrá primero el índice de la instancia más cercana a la posición x/y dada y lo almacenará en una variable local. A continuación, obtiene el ID de la capa de instancia única para la capa llamada "Frente de Instancias", y mueve la instancia encontrada a a esa capa.
