layer_has_instance

Esta función puede utilizarse para comprobar si una instancia dada está asignada actualmente a la capa dada. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y el ID de la instancia a comprobar. También puede dar un object_index (es decir, el nombre del objeto en el Navegador de Activos) y la función comprobará si hay alguna instancia de ese objeto en la capa dada. La función devolverá true si la instancia está en la capa y false si no lo está.

Sintaxis:

layer_has_instance(layer_id, instance_id)

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)
instance_id El ID único de la instancia o el índice del objeto de la instancia a comprobar

Devuelve:

Booleano

Ejemplo:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
layer_destroy(global.Bullet_Layer);
}

El código anterior comprobará si la capa dada contiene alguna instancia del objeto "obj_Bullet_Parent" y en caso contrario destruirá la capa.
