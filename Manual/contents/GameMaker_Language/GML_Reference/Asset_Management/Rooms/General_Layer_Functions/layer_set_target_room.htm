layer_set_target_room

Cuando se llama a esta función se está indicando a GameMaker Studio 2 que todas las funciones de capas posteriores deben aplicarse a la sala dada. De este modo, puede cambiar o generar capas y contenidos de capas de forma procedimental en una sala que no sea la sala actual. Cuando haya terminado de añadir capas o elementos de capa a una sala, llame a la función layer_reset_target_room() para restablecer el objetivo de la sala (o llame a esta función de nuevo con un argumento de sala de -1). Tenga en cuenta que esta función sólo puede utilizarse en salas distintas de la sala actual, y está diseñada para que pueda añadir/eliminar capas y elementos de capa a salas distintas de la sala que se está ejecutando en ese momento.

ADVERTENCIA Mientras se dirige a otra sala, puede utilizar todas las funciones regulares de la capa, excepto que no puede crear instancias utilizando instance_create_layer() o instance_create_depth()ni la función de capa layer_add_instance() estará disponible.

Sintaxis:

layer_set_target_room(room)

Argumento Descripción
habitación La sala de destino para todas las funciones de la capa posterior

Devuelve:

N/A

Ejemplo:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
var _x = random(room_get_width(rm_Game));
var _y = random(room_get_height(rm_Game));
layer_sprite_create(l, _x, _y, spr_Trees);
}
layer_reset_target_room();

El código anterior establece la sala de destino a la sala "rm_Game" y luego obtiene el ID de la capa llamada "SpriteAssets" en esa sala. Este ID de capa se utiliza para añadir 50 activos de sprites al azar a la capa, antes de que el objetivo de la capa se restablece a la sala actual.
