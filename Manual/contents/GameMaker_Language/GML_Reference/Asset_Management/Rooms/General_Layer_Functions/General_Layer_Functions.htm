Funciones generales de la capa

El editor de salas de GameMaker Studio 2 te permite añadir capas de diferentes activos a tu sala. Estos pueden ser mapas de azulejos (utilizando conjuntos de azulejos), fondos, sprites o secuencias, e instancias, pero tenga en cuenta que, a diferencia del editor de salas, no está limitado a un solo tipo de activo por capa, y puede añadir varios tipos de activos a la misma capa - por ejemplo, puede tener un fondo y un mapa de azulejos en la misma capa. Tenga en cuenta que cada recurso que se añade a una capa se considera un elemento de esa capa.

En esta sección se describen las funciones generales que se utilizan al trabajar con las capas de la sala, así como algunas funciones específicas para trabajar con capa de instancia elementos:

También tenemos un par de funciones adicionales relacionadas con la orientación de las capas en una habitación diferente. Puede ser que quieras cambiar algo en una habitación que no es la actual, o tal vez quieras generar habitaciones procedimentalmente, o cualquier número de cosas. Para permitir esto usamos las siguientes funciones:

  • layer_set_target_room
  • layer_get_target_room
  • layer_reset_target_room

Lo que se hace aquí es establecer la sala de destino utilizando la función apropiada y luego llamar a las funciones de capa son normales. Estas funciones de capa se aplicarán ahora sólo a la sala de destino y no a la actual. Cuando haya terminado, llame a la función función reset para devolver el ámbito del código a la sala actual de nuevo.

ADVERTENCIA Mientras se dirige a otra sala, puede utilizar todas las funciones regulares de la capa, excepto que no puede crear instancias utilizando instance_create_layer() o instance_create_depth()ni la función de capa layer_add_instance() estará disponible.

Por último, y esto es exclusivamente para usar con instancias en capas (aunque la capa en sí puede tener otros elementos, no se verán afectados por las siguientes funciones), tienes un par de funciones que se pueden usar para desactivar o reactivar todas las instancias:

  • instancia_desactivar_capa
  • instancia_activar_capa
