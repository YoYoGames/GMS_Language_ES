layer_get_element_layer

Puede utilizar esta función para obtener el ID de la capa en la que se encuentra el elemento dado. Se suministra el valor único del ID del elemento (por ejemplo, tal como lo devuelve la función que creó el elemento o el editor de salas) y la función devolverá el ID único de la capa en la que se encuentra el elemento. de la capa en la que se encuentra el elemento. Si el ID del elemento dado no es válido, la función devolverá -1.

Sintaxis:

layer_get_element_layer(element_id)

Argumento Descripción
element_id El valor de identificación único del elemento para obtener la capa de

Devuelve:

ID de la capa o -1 si el elemento no es válido (es decir, no existe)

Ejemplo:

element = layer_get_element_layer(asset_1);

El código anterior obtiene el ID de la capa para el elemento con el ID almacenado en la variable "asset_1" y lo almacena en una variable.
