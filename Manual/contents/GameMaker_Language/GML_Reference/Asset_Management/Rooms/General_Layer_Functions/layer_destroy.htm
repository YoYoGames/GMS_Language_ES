destruir_capas

Esta función destruirá la capa dada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y esto la eliminará de la sala actual. Si la capa ha sido diseñada en el editor de salas, la próxima vez que abandone la sala y regrese, la capa se volverá a crear con el contenido original, sin embargo, si la sala es persistente, la capa será eliminada a menos que la persistencia de la sala sea desactivada de nuevo. Cuando se destruye una capa de esta manera, todo su contenido será eliminado también, por lo que cualquier ID de referencia para fondos o mapas de azulejos, etc... ya no será válido y cualquier instancia asignadas a la capa serán destruidas (realizando su Evento de Destrucción al mismo tiempo, si tienen uno, así como el Evento de Limpieza).

Sintaxis:

layer_destroy(layer_id)

Argumento Descripción
layer_id El valor de identificación único de la capa a destruir (o el nombre de la capa como cadena)

Devuelve:

N/A

Ejemplo:

if !instance_exists(obj_Bullet_Parent)
{
layer_destroy(global.Bullet_Layer);
}

El código anterior comprobará si existe alguna instancia del objeto "obj_Bullet_Parent", y si no existe destruirá la capa con el ID almacenado en la variable global.
