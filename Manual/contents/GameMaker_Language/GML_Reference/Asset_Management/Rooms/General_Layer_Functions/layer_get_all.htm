layer_get_all

Esta función devolverá un array poblado con los valores de ID únicos de cada capa en la sala.

Sintaxis:

layer_get_all()

Devuelve:

Array (1D, poblado con IDs de capas)

Ejemplo:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
layer_destroy(a[i]);
}

El código anterior recupera todas las capas de una habitación y añade sus valores de ID a un array. Este array es entonces analizado para destruir o las capas de la sala.
