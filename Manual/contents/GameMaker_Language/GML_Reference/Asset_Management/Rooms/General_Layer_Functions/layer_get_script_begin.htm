layer_get_script_begin

Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y esta función devolverá el índice de la función de script de la función asignada para ejecutarse al principio de la renderización para esa capa, o devolverá -1 si no hay ninguna función asignada. Puede asignar funciones de script a una capa con layer_script_begin() y layer_script_end().

Sintaxis:

layer_get_script_begin(layer_id);

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)

Devuelve:

Índice de la función de script (real) o -1 si no se asigna ningún script

Ejemplo:

if layer_get_script_begin(layer) == -1
{
layer_script_begin(layer, scr_SetShaderValues);
}

El código anterior comprobará si la capa que la instancia que ejecuta el código tiene una función de script asignada y si no la tiene se le asigna una.
