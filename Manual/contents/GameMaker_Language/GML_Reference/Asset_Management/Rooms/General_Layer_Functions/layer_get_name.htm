nombre_de_capa

Puede utilizar esta función para obtener el nombre de la capa dada. Usted suministra el valor único del ID de la capa y si la capa es una de las capas con nombre creadas en el editor de salas, entonces la función devolverá una cadena con el nombre de la capa. Si la capa no es una de las creadas por el editor de salas (es decir, fue creada con layer_create()) entonces se devolverá una cadena vacía.

Sintaxis:

layer_get_name(layer_id)

Argumento Descripción
layer_id El valor de identificación único de la capa para obtener el nombre de

Devuelve:

Cadena

Ejemplo:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
if layer_get_name(a[i]) != ""
{
ds_list_add(layer_list, a[i])
}
}

El código anterior obtiene los IDs de todas las capas de la sala y luego hace un bucle para comprobar si alguna es una capa con nombre. Si lo son, su ID se añade a una lista.
