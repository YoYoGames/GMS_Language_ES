layer_get_vspeed

Puede utilizar esta función para obtener la velocidad vertical (en píxeles por fotograma de juego) de la capa dentro de la sala actual. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y la función devuelve un número real para la velocidad vertical, donde un valor positivo es hacia abajo y un valor negativo hacia arriba. El valor por defecto es 0 (a menos que se establezca en el editor de la sala).

Sintaxis:

layer_get_vspeed(layer_id)

Argumento Descripción
layer_id El valor de identificación único de la capa para obtener la velocidad vertical de

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

El código anterior comprueba las velocidades horizontales y verticales de la capa dada y si no están ambas a 0 entonces las pone a 0.
