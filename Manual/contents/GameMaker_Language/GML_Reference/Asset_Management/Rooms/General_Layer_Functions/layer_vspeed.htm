capa_velocidad

Puede utilizar esta función para establecer la velocidad vertical (en píxeles por fotograma de juego) de la capa dentro de la sala actual. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y el valor de la velocidad a establecer, donde un valor positivo es hacia abajo y un valor negativo hacia arriba.

Sintaxis:

layer_vspeed(layer_id, vspd)

Argumento Descripción
layer_id El valor de identificación único de la capa para establecer la velocidad vertical de
vspd La velocidad vertical (en píxeles por fotograma del juego) a establecer

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

El código anterior comprueba las velocidades horizontales y verticales de la capa dada y si no están ambas a 0 entonces las pone a 0.
