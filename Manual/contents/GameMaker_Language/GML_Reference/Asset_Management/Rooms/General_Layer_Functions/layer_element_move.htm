capa_elemento_mover

Puede utilizar esta función para mover un elemento de una capa a otra. Se da el ID del elemento, devuelto por la función utilizada para crear el elemento o el editor de salas o la función layer_get_all_elements(), y luego le das el ID de la capa (que obtienes cuando creas la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento).

Sintaxis:

layer_element_move(element_id, layer_id)

Argumento Descripción
element_id El valor de identificación único del elemento a mover
layer_id El valor de ID único de la capa a la que se va a mover el elemento (o el nombre de la capa como cadena)

Devuelve:

N/A

Ejemplo:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_element_move(a[i], asset_layer)
}
}

El código anterior obtiene los elementos de la capa a la que está asignada la instancia que ejecuta el código, luego los comprueba para ver si son activos de sprite, y si lo son entonces se mueven a la capa con el ID almacenado en la variable "asset_layer".
