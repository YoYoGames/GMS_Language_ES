layer_get_all_elements

Puede utilizar esta función para obtener los ID de los elementos de la capa dada. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y la función devolverá una matriz de IDs, donde cada entrada en la matriz es un ID único para un elemento en esa capa. Por ejemplo, si la capa es una capa de activos, la matriz se llenará con los valores de ID para cada activo de sprite que esté asignado a la capa. Tenga en cuenta que el uso de código para trabajar con capas significa que puede asignar diferentes tipos de elementos a la misma capa - por lo que puede tener activos sprite junto con instancias, por ejemplo, en cuyo caso puede utilizar la función layer_get_element_type() para obtener el tipo de elemento al que se refiere el ID.

Sintaxis:

layer_get_all_elements(layer_id)

Argumento Descripción
layer_id El valor de identificación único de la capa de la que se van a obtener los elementos (o el nombre de la capa como cadena)

Devuelve:

Array (1D, poblado con IDs de elementos)

Ejemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

El código anterior obtiene los IDs de todos los elementos de la instancia asignados a la capa de la instancia que ejecuta el código. A continuación, el código comprueba si alguno de los elementos devueltos es un activo de sprite y, si lo es, se destruye.
