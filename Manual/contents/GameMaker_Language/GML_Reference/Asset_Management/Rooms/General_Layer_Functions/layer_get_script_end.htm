layer_get_script_end

Esta función devuelve el índice de función de script de la función asignada para ejecutarse al final de la renderización de la capa dada, o devolverá -1 si no se asigna ninguna función. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento). Puede asignar funciones de script a una capa con layer_script_begin() y layer_script_end() .

Sintaxis:

layer_get_script_end(layer_id);

Argumento Descripción
layer_id El valor de identificación único de la capa a la que se dirige (o el nombre de la capa como cadena)

Devuelve:

Índice de la función de script (real) o -1 si no se asigna ningún script

Ejemplo:

if layer_get_script_end(layer) == -1
{
layer_script_end(layer, scr_ResetShaderValues);
}

El código anterior comprobará si la capa que la instancia que ejecuta el código tiene una función de script asignada y si no la tiene se le asigna una.
