capa_existe

Esta función puede utilizarse para comprobar si la capa dada existe. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento ) y la función devolverá un valor booleano de true si existe o false si no existe.

NOTA: Esta función trabaja dentro del ámbito de la sala de destino actual -por defecto la sala en la que se llama a la función- que puede establecerse mediante la función layer_set_target_room().

Sintaxis:

layer_exists(layer_name)

Argumento Descripción
nombre_de_capa El nombre de la capa (una cadena o un valor de identificación)

Devuelve:

Booleano

Ejemplo:

if !layer_exists(global.tileLayer)
{
global.tileLayer = layer_create(1000);
}

El código anterior comprobará si la capa almacenada en la variable global existe realmente, y si no existe entonces se crea.
