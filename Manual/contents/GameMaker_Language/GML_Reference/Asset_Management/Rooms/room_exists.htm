habitación_existe

Con esta función se puede comprobar si la habitación especificada existe o no. Esta función toma el índice de la sala (un número real) y no el nombre de la sala (una cadena). Esta función es muy útil cuando se crean salas dinámicamente utilizando la función room_add()pero también puede utilizar las variables de sólo lectura room_first y room_last o las funciones room_next() y room_previous() para obtener el índice de una sala específica, o proporcionar una variable que haya almacenado el índice de cualquier otra sala de tu juego.

Sintaxis:

room_exists(index);

Argumento Descripción
índice El índice de la sala a comprobar.

Devuelve:

Booleano

Ejemplo:

if room_exists(global.rm[0])
{
room_goto(global.rm[0]);
}

El código anterior comprueba si la sala indexada en el array "global.rm[]" existe y, si lo hace, va a esa sala.
