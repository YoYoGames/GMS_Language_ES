room_speed (obsoleto)

Esta variable contiene la velocidad de ejecución de todas las habitaciones (y del juego) en cuadros de juego por segundo. Tenga en cuenta que esto NO es el FPS (cuadros por segundo), sino más bien el número de pasos de juego que GameMaker Studio 2 tratará de mantener cada segundo.

IMPORTANTE! Esta variable se mantiene sólo para el soporte de Legacy, y no debe ser utilizada ya que no establece la velocidad para una sola sala, sino para TODAS las salas del juego. Para cambiar la velocidad del juego debes utilizar la función game_set_speed().

Sintaxis:

velocidad_de_habitación;

Devuelve:

Real

Ejemplo:

alarma[0] = velocidad_habitación * 10;

El código anterior utiliza room_speed para establecer que la alarma se ejecute durante 10 segundos. El uso de la variable de esta manera asegurará que la alarma se ejecute durante 10 segundos, independientemente de la velocidad que se establezca para la habitación.
