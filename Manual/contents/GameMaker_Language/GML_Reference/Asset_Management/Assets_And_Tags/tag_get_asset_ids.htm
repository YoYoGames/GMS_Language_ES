tag_get_asset_ids

Con esta función puede obtener todos los activos de un tipo determinado que tengan asignadas las etiquetas indicadas. Puede proporcionar una sola etiqueta (como una cadena) o una matriz, donde cada elemento de la matriz es una etiqueta (como una cadena), así como el tipo de activo a comprobar. El tipo de activo debe ser una de las siguientes constantes:

Constante Descripción
objeto_activo El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
sala_de_activos El nombre de pila se refiere a una habitación.
activos_tiles El nombre dado se refiere a un conjunto de baldosas.
ruta_de_activo El nombre dado se refiere a una ruta.
script_activo El nombre dado se refiere a un guión.
asset_font El nombre dado se refiere a una fuente.
línea_de_tiempo_de_activos El nombre de pila se refiere a una línea de tiempo.
asset_shader El nombre se refiere a un shader.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
secuencia_de_activos El nombre dado se refiere a una Secuencia.

La función devolverá una matriz, donde cada elemento de la matriz será un único valor de índice de activo. Si no hay activos del tipo que tienen la(s) etiqueta(s) dada(s), se devolverá un array vacío.

Sintaxis:

tag_get_asset_ids(tags, asset_type);

Argumento Descripción
tags Una sola cadena de etiquetas de activos o una matriz con varias etiquetas de activos.
tipo_de_activo Una constante de tipo de activo (enumerada anteriormente)

Devuelve:

Array

Ejemplo:

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

El código anterior utiliza la etiqueta "enemigo" para encontrar todos los activos de la ruta con esa etiqueta, antes de elegir uno al azar y asignarlo a la instancia que ejecuta el código.
