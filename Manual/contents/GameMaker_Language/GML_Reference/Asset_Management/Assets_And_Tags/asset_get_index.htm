asset_get_index

Puedes utilizar esta función para obtener el índice de identificación único de un activo del juego a partir de su nombre. Si el activo no se encuentra, la función devolverá un valor de -1, de lo contrario devolverá el índice único de identificación para el activo que se está comprobando. Este identificador puede puede utilizarse en otras funciones como cualquier otro valor de índice, como el sprite_index o el path_index, por ejemplo. Por favor, tenga en cuenta que aunque esta función se puede utilizar para referenciar activos desde cadenas (ver ejemplo más abajo) siempre debe asegurarse de que el activo existe antes de utilizarlo, de lo contrario puede obtener errores que bloquearán su juego.

Sintaxis:

asset_get_index(nombre);

Argumento Descripción
nombre El nombre del activo del juego para obtener el índice (una cadena).

Devuelve:

Índice de activos (entero)

Ejemplo:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

El código anterior obtendrá un índice de objeto de una cadena, y si ese índice existe, creará una instancia del objeto en el juego.
