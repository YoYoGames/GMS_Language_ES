asset_get_tags

Con esta función puede recuperar todas las etiquetas asignadas a un activo desde el Navegador de Activos. Usted proporciona el nombre del activo (como una cadena) o su índice de activo, y la función devolverá una matriz de etiquetas para ese activo. Si no se encuentra ninguna etiqueta o hay un error (es decir, la cadena de nombre dada no existe), el array devuelto estará vacío. Si suministra un valor de índice de activo, tendrá que suministrar el argumento opcional tipo de activo (una constante), ya que los activos de diferentes tipos pueden tener el mismo índice, aunque no puedan tener el mismo nombre. Los tipos de activos disponibles se enumeran en la siguiente tabla:

Constante Descripción
objeto_activo El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
sala_de_activos El nombre de pila se refiere a una habitación.
activos_tiles El nombre dado se refiere a un conjunto de baldosas.
ruta_de_activo El nombre dado se refiere a una ruta.
script_activo El nombre dado se refiere a un guión.
asset_font El nombre dado se refiere a una fuente.
línea_de_tiempo_de_activos El nombre de pila se refiere a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
secuencia_de_activos El nombre dado se refiere a una Secuencia.

Sintaxis:

asset_get_tags(name_or_index, [asset_type]);

Argumento Descripción
nombre_o_índice El nombre del activo (una cadena) o su valor de índice (un entero).
[asset_type] OPCIONAL! El tipo de activo para el que se obtienen las etiquetas, sólo se utiliza cuando se suministra un índice para el primer argumento.

Devuelve:

Array

Ejemplo:

mis_etiquetas = asset_get_tags(object_get_name(object_index));

El código anterior recuperará todas las etiquetas asignadas al objeto del que se ha creado la instancia que ejecuta el código.
