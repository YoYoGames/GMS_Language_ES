asset_has_tags

Con esta función puede comprobar si una o más cadenas de etiquetas están asignadas a cualquier activo del navegador de activos. Se proporciona el nombre del activo (como una cadena) o su índice de activo, así como una sola cadena de etiquetas o una matriz en la que cada elemento es una sola cadena de etiquetas. Si se suministra un valor de índice de activo, será necesario suministrar el argumento opcional de tipo de activo (una constante), ya que los activos de diferentes tipos pueden tener el mismo índice, aunque no puedan tener el mismo nombre. Los tipos de disponibles se enumeran en la siguiente tabla:

Constante Descripción
objeto_activo El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
sala_de_activos El nombre de pila se refiere a una habitación.
activos_tiles El nombre dado se refiere a un conjunto de baldosas.
ruta_de_activo El nombre dado se refiere a una ruta.
script_activo El nombre dado se refiere a un guión.
asset_font El nombre dado se refiere a una fuente.
línea_de_tiempo_de_activos El nombre de pila se refiere a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
secuencia_de_activos El nombre dado se refiere a una Secuencia.

Si la función tiene éxito y una o más de las etiquetas están presentes para el activo, entonces devolverá true, de lo contrario, devolverá false. Si necesita comprobar una coincidencia precisa con cualquier etiqueta o conjunto de etiquetas, utilice la función asset_has_tags().

Sintaxis:

asset_has_any_tag(name_or_index, tags, [asset_type]);

Argumento Descripción
nombre_o_índice El nombre del activo (una cadena) o su valor de índice (un entero).
tags Una sola cadena de etiquetas de activos o una matriz con varias etiquetas de activos.
[asset_type] OPCIONAL! El tipo de activo para comprobar las etiquetas, sólo se utiliza cuando se suministra un índice para el primer argumento.

Devuelve:

Booleano

Ejemplo:

var _a = array_create(3);
_a[0] = "boss";
_a[1] = "main_boss";
_a[2] = "final_boss";
if asset_has_any_tag(object_index, _a, asset_object)
{
instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

El código anterior creará un array de etiquetas y luego comprobará si alguna de ellas se aplica al objeto dado, y si lo hace creará una instancia de otro objeto.
