tag_get_assets

Con esta función puede recuperar los nombres de todos los activos a los que se ha asignado la etiqueta o etiquetas dadas. Puede suministrar una sola cadena de etiquetas o una matriz, donde cada elemento de la matriz es una cadena de etiquetas. La función devolverá una matriz en la que cada entrada es el nombre (como cadena) del activo con la etiqueta dada. Si necesita el índice único del activo, puede utilizar la función asset_get_index() junto con con el nombre devuelto. Si no hay activos con la(s) etiqueta(s) dada(s), o si hay un error (por ejemplo: las etiquetas dadas no existen), entonces se devolverá un array vacío.

Sintaxis:

tag_get_assets(tags);

Argumento Descripción
tags Una sola cadena de etiquetas de activos o una matriz con varias etiquetas de activos.

Devuelve:

Array

Ejemplo:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
if asset_get_type(_assets[i]) == asset_sprite
{
ds_list_add(backgrounds, asset_get_index(_assets[i]));
}
}

El código anterior crea una lista y luego recupera los nombres de todos los activos con la etiqueta "background". Recorre la matriz de nombres devueltos, comprobando si alguno de ellos es un activo de sprite, y si lo es, el valor de índice único del activo se añade a la lista para su uso futuro.
