asset_get_type

Con esta función puede obtener el tipo de activo al que se hace referencia a partir de su nombre (una cadena). Se devolverá una de las constantes enumeradas a continuación.

Sintaxis:

asset_get_type(nombre);

Argumento Descripción
nombre El nombre del activo del juego para obtener el tipo de (una cadena).

Devuelve:

Constante

Constante Descripción
objeto_activo El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
sala_de_activos El nombre de pila se refiere a una habitación.
activos_tiles El nombre dado se refiere a un conjunto de baldosas.
ruta_de_activo El nombre dado se refiere a una ruta.
script_activo El nombre dado se refiere a un guión.
asset_font El nombre dado se refiere a una fuente.
línea_de_tiempo_de_activos El nombre de pila se refiere a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
secuencia_de_activos El nombre dado se refiere a una Secuencia.
activo_desconocido El nombre dado se refiere a un bien que o bien no existe, o bien no es uno de los enumerados anteriormente.

Ejemplo:

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
show_debug_message("¡La ruta no existe!");
}
else
{
path_index = asset_get_index("pth_Path_" + string(global.Game));
}

El código anterior comprueba un nombre de activo creado dinámicamente para ver si el activo es del tipo correcto. Si no lo es, entonces se mostrará un mensaje de depuración, de lo contrario el nombre del activo se utiliza para asignar el activo a la instancia.
