asset_clear_tags

Con esta función puede borrar todas las etiquetas presentes en cualquier activo desde el navegador de activos. Se suministra el nombre del activo (como una cadena) o su índice de activo, y si se suministra un valor de índice de activo, será necesario suministrar el argumento opcional tipo de activo (una constante), ya que los activos de diferentes tipos pueden tener el mismo índice, aunque no puedan tener el mismo nombre. Los tipos de activos disponibles se enumeran en la siguiente tabla:

Constante Descripción
objeto_activo El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
sala_de_activos El nombre de pila se refiere a una habitación.
activos_tiles El nombre dado se refiere a un conjunto de baldosas.
ruta_de_activo El nombre dado se refiere a una ruta.
script_activo El nombre dado se refiere a un guión.
asset_font El nombre dado se refiere a una fuente.
línea_de_tiempo_de_activos El nombre de pila se refiere a una línea de tiempo.
asset_shader El nombre se refiere a un shader.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
secuencia_de_activos El nombre dado se refiere a una Secuencia.

Si la función consigue borrar las etiquetas, devolverá true, de lo contrario, devolverá false.

Sintaxis:

asset_clear_tags(name,[asset_type]);

Argumento Descripción
nombre_o_índice El nombre del activo (una cadena) o su valor de índice (un entero).
[asset_type] OPCIONAL! El tipo de activo del que se van a borrar las etiquetas, sólo se utiliza cuando se proporciona un índice para el primer argumento.

Devuelve:

Booleano

Ejemplo:

asset_clear_tags(obj_Enemy_Parent, asset_object);

El código anterior borrará todas las etiquetas del objeto activo dado.
