tamaño de la línea de tiempo

Con esta función puedes obtener el número total de momentos activos para una línea de tiempo (un momento "activo" es aquel que tiene código o DnD™ añadido). Esto puede ser útil cuando se crean líneas de tiempo dinámicas utilizando las funciones timeline_moment_add_script() y timeline_moment_clear() ya que puedes comprobar si una línea de tiempo dada tiene el número correcto de momentos activos o ninguno.

Sintaxis:

timeline_size(ind);

Argumento Descripción
ind El índice de la línea de tiempo de la que se obtiene la información.

Devuelve:

N/A

Ejemplo:

si timeline_size(global.tl) == 0
{
timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

El código anterior comprueba el tamaño de la línea de tiempo dada, y si devuelve 0 (es decir, la línea de tiempo no tiene momentos activos) añadirá una función de script que se utilizará en un momento aleatorio dentro de la línea de tiempo indexada en la variable global "tl".
