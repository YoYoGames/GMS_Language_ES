añadir_temporización

Con esta función puede añadir una nueva línea de tiempo (vacía) en su juego. La función devuelve el índice de esta nueva línea de tiempo que debe ser almacenada en una variable para su uso en todas las llamadas de función posteriores que involucren esta nueva línea de tiempo. También debe asegurarse de asegurarse de utilizar la función timeline_delete() siempre que no desee utilizar la línea de tiempo (como al final de la sala) para evitar posibles fugas de memoria que podrían ralentizar ralentizar o bloquear el juego. Para añadir momentos a una línea de tiempo creada de esta manera, vea la función timeline_moment_add_script().

Sintaxis:

timeline_add();

Devuelve:

Real

Ejemplo:

global.tl = timeline_add();

El código anterior crea una nueva línea de tiempo y almacena su índice en la variable "global.tl".
