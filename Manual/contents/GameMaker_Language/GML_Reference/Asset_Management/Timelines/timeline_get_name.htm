timeline_get_name

Esta función puede utilizarse para obtener el nombre de una línea de tiempo en forma de cadena. si la línea de tiempo ha sido creada dinámicamente utilizando la función timeline_add() el nombre devuelto tendrá el formato formato "__newtimelineN" donde " N" es elnúmero de la línea de tiempo (empezando por 0). Tenga en cuenta que esto es sólo una cadena y no se puede utilizar para hacer referencia a la línea de tiempo directamente - para para ello necesitaría el índice de la línea de tiempo. Sin embargo, puede utilizar esta cadena para obtener el índice de la línea de tiempo utilizando la cadena devuelta junto con la función asset_get_index().

Sintaxis:

timeline_get_name(ind);

Argumento Descripción
ind El índice de la línea de tiempo para comprobar el nombre de.

Devuelve:

Cadena

Ejemplo:

var _str = timeline_get_name(timeline_index);
show_debug_message("Timeline actual = " + _str);

El código anterior obtiene el nombre de la línea de tiempo asignada actualmente y lo muestra en la consola.
