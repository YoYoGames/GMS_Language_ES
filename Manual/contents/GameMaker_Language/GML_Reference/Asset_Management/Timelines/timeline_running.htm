cronograma_de_ejecución

Esta variable mantiene el estado actual de la línea de tiempo asignada y devolverá true si se está ejecutando y false si no. También puede establecer esta variable como verdadera o falsa para Hay que tener en cuenta que una línea de tiempo detenida no se restablece, por lo que si se inicia de nuevo en un momento posterior, se iniciará desde el momento exacto en que se detuvo.

Sintaxis:

timeline_running;

Devuelve:

Booleano

Ejemplo:

if !timeline_running
{
timeline_position = 0;
timeline_running = true;
}

El código anterior comprobará si la instancia está ejecutando una línea de tiempo, y si no lo está entonces restablece la línea de tiempo asignada para que comience en el primer momento y luego la inicia.
