borrar_temporal

Con esta función puede eliminar cualquier línea de tiempo de su juego. si esta línea de tiempo fue creada en el Navegador de Activos de GameMaker Studio 2, por favor tenga en cuenta que se elimina completamente del juego y tratar de llamar a la línea de tiempo indexada después de usar esta función causará un error. Si la línea de tiempo fue creada dinámicamente utilizando la función timeline_add() entonces esta función debe usarse en el momento que la línea de tiempo ya no es necesaria para evitar cualquier fuga de memoria que podría ralentizar (y eventualmente colapsar) su juego.

Sintaxis:

timeline_delete(ind);

Argumento Descripción
ind El índice de la línea de tiempo a eliminar.

Devuelve:

N/A

Ejemplo:

if timeline_exists(global.tl)
{
timeline_delete(global.tl);
}

El código anterior comprueba si existe una línea de tiempo y está indexada en la variable global "tl" y, si es así, borra esa línea de tiempo.
