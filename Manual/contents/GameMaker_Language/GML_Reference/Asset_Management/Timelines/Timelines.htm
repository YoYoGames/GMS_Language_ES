Líneas de tiempo

Las líneas de tiempo son un poderoso mecanismo para controlar los eventos en su juego, y se basan en "momentos", donde un "momento" es simplemente un solo paso del juego. Estas líneas de tiempo se crean generalmente desde el Navegador de Activos utilizando el Editor de Líneas de Tiempo de forma similar a un objeto, es decir: creas una línea de tiempo, luego en cada momento que requieras que realice una acción añades algún código o un icono DnD™para que la realice. Una vez creada, puedes detener, iniciar y manipular las líneas de tiempo a través del código. Incluso puedes modificar momentos individuales definiendo una función de script y añadiéndola a la línea de tiempo de forma dinámica desde un objeto controlador (por ejemplo), lo que hace que sea una herramienta muy potente y versátil herramienta a la hora de crear tus juegos.

Las siguientes funciones sirven para añadir y manipular líneas de tiempo desde el juego:

  • existe la línea de tiempo
  • timeline_get_name
  • añadir_temporización
  • borrar_temporal
  • timeline_moment_add_script
  • timeline_moment_clear
  • borrar la línea de tiempo
  • tamaño de la línea de tiempo
  • timeline_max_moment

Las siguientes variables están "incorporadas" a todas las instancias para facilitar el trabajo con las Líneas de Tiempo y darle control sobre ellas:

  • índice_de_línea_de_tiempo
  • cronograma_de_ejecución
  • velocidad de la línea de tiempo
  • posición_temporal
  • bucle_de_tiempo

Tenga en cuenta que puede encontrar más información sobre las variables de instancia aquí: Variables de instancia.
