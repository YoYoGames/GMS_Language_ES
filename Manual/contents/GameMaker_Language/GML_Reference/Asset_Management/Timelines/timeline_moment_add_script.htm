timeline_moment_add_script

Con esta función puedes añadir dinámicamente una función de script a las líneas de tiempo en cualquier "momento" dentro de esa línea de tiempo, donde un "momento" es el equivalente a un tick de juego (o paso). De este modo, puede crear una nueva línea de tiempo utilizando la función timeline_add() y añadir diferentes comportamientos en cualquier punto, o simplemente modificar un recurso de línea de tiempo previamente creado con nuevos comportamientos. Tenga en cuenta que la función no puede requerir ningún argumento adicional cuando se utiliza esta función, y si la utiliza para modificar un recurso de línea de tiempo presente en el Navegador de Activos, entonces todas las instancias que utilizan esta línea de tiempo se verán afectadas, y el cambio durará hasta el final del juego (llamando a game_restart() tampoco se restablecerá el cambio).

Sintaxis:

timeline_moment_add_script(ind, step, script);

Argumento Descripción
ind El índice de la línea de tiempo para añadir un momento.
paso El momento (paso) a añadir.
guión El índice de la función de script a añadir en el momento.

Devuelve:

N/A

Ejemplo:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
i += room_speed * 60;
}

El código anterior creará una nueva línea de tiempo y almacenará su índice en la variable "global.tl". A continuación, añadirá tres scripts a la línea de tiempo, elegidos al azar, a intervalos de un minuto.
