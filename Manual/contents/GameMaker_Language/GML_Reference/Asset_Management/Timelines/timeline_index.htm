índice_de_línea_de_tiempo

Esta variable contiene el índice de la línea de tiempo actualmente asociada a la instancia. Puede establecerla a una línea de tiempo en particular para usarla, o establecerla a -1 para dejar de usar una línea de tiempo para la instancia (si no se define ninguna línea de tiempo para la instancia, también se devuelve -1). Tenga en cuenta que esto no inicia la línea de tiempo - para ello utilice la variable línea_de_tiempo_en_curso.

Sintaxis:

timeline_index;

Devuelve:

Real (entero - valor del índice de recursos)

Ejemplo:

if timeline_exists(global.tl)
{
timeline_index = global.tl;
}

El código anterior asignará a la instancia que ejecuta el código una línea de tiempo indexada en la variable "global.tl" si esa línea de tiempo existe.
