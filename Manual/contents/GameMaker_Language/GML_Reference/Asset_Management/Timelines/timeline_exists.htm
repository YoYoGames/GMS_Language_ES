existe la línea de tiempo

Con esta función puede comprobar si una línea de tiempo existe (devuelve true) o no (devuelve false). Esto es especialmente útil cuando se crean líneas de tiempo de forma dinámica utilizando la función timeline_add() función, pero debe tener en cuenta que si busca una variable no inicializada (que de otro modo se asignaría al índice de una línea de tiempo) se lanzará un error.

Sintaxis:

timeline_exists(ind);

Argumento Descripción
ind El índice de la línea de tiempo a comprobar.

Devuelve:

Booleano

Ejemplo:

if timeline_exists(global.tl)
{
timeline_delete(global.tl);
}

El código anterior comprueba si existe una línea de tiempo y está indexada en la variable global "tl" y, si es así, borra esa línea de tiempo.
