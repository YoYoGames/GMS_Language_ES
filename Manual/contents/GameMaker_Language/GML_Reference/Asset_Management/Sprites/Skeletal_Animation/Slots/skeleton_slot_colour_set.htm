skeleton_slot_colour_set

Con esta función puedes establecer el color y el alfa de una ranura adjunta para que todos los sprites adjuntos a ella se dibujen con estos valores de mezcla. Tenga en cuenta que el sprite de instancia puede tener un color de mezcla y un ajuste de alfa (mezcla_de_imagen y image_alpha), al igual que el adjunto (ver la función esqueleto_adjunto_crear_color()), por lo que el color y alfa finales que tendrá el sprite adjunto asignado para la ranura serán un compuesto de todos estos valores.

Sintaxis:

skeleton_slot_colour_set(slot, color, alpha);

Argumento Descripción
ranura El nombre de la ranura a establecer, una cadena
color El color a establecer, ya sea un entero, una constante o un valor hexadecimal.
alfa El alfa a establecer de 0 a 1.

Devuelve:

N/A

Ejemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
for (var i = 0; i < ds_list_size(slot_list); ++i;)
{
if skeleton_slot_colour_get(slot_list[| i]) != c_white
{
skeleton_slot_colour_set(slot_list[| i], c_white, 1);
}
}
}

El código anterior crea una lista DS y luego la rellena con los nombres de las ranuras encontradas en la posición del ratón en la sala. Luego recorre la lista de ranuras y restablece el color de las ranuras encontradas a blanco con un alfa de 1 si el color no es ya es blanco.
