lista_de_ranuras_del_esqueleto

Con esta función puedes rellenar una lista DS (precreada) con todos los nombres de las ranuras creadas como parte del sprite de animación del esqueleto. Los nombres serán cadenas y podrán ser utilizados en las otras funciones de ranuras de animación del esqueleto para estos tipos de sprites.

Sintaxis:

skeleton_slot_list(sprite, list);

Argumento Descripción
sprite El índice del sprite de la animación del esqueleto de la columna vertebral para obtener la lista.
lista El ID de la lista de DS a rellenar con los nombres de los huesos.

Devuelve:

N/A

Ejemplo:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

El código anterior crea una lista DS y la rellena con los nombres de las ranuras para su uso posterior.
