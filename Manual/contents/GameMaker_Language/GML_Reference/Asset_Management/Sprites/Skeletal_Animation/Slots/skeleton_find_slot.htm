skeleton_find_slot

Con esta función puedes encontrar qué ranuras se encuentran en una posición de espacio de habitación especificada en el sprite de Spine asociado a la instancia actual. Se crea una lista de DS y se suministra su ID junto con con una posición x/y para comprobar y la lista se rellenará con una cadena de nombres para cada una de las ranuras de fijación disponibles para el sprite (incluyendo cualquier modificación de fijación). Tenga en cuenta que la lista siempre se ordena en orden descendente empezando por la ranura más alta.

Sintaxis:

esqueleto_encontrar_ranura(x, y, lista);

Argumento Descripción
x La posición x en la habitación para comprobar.
y La posición y en la sala para comprobar.
lista El ID de la lista de DS a rellenar con los mapas de DS.

Devuelve:

N/A

Ejemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

El código anterior crea una lista DS y luego la rellena con los nombres de las ranuras encontradas en la posición del ratón en la sala.
