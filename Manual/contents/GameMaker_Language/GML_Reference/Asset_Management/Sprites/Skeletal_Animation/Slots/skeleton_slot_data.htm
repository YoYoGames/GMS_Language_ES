datos_del_esqueleto_de_la_ranura

Con esta función se puede rellenar una lista DS (pre-creada) con los datos de cada una de las ranuras de adjuntos disponibles que contiene una animación. Estos datos se devuelven en el foro de un mapa DS que contiene los siguientes pares clave/valor:

  • "nombre": El nombre de la ranura.
  • "hueso": El nombre del hueso.
  • "adjunto": El nombre del adjunto si se utiliza uno o "(ninguno)" si no hay ninguno asignado.

Los valores de cada clave serán cadenas y podrán ser utilizados en las otras funciones de fijación del esqueleto para estos tipos de sprite. Tenga en cuenta que los mapas DS creados no se destruyen, por lo que tendrá que hacer un bucle a través de la lista de DS y destruir cada uno de los mapas creados.

Sintaxis:

skeleton_slot_data(sprite, list);

Argumento Descripción
sprite El índice del sprite de la animación del esqueleto de la columna vertebral para obtener los datos.
lista El ID de la lista de DS a rellenar con los mapas de DS.

Devuelve:

N/A

Ejemplo:

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data(sprite_index, list);
for (var i = 0; i < ds_list_size(list); i++;)
{
var map = list[| i];
if open
{
if map[? "attachment"] == "(none)"
{
open = false;
slot_name = map[? "name"];
}
}
ds_map_destroy(map);
}
ds_list_destroy(list);

El código anterior crea una lista de DS y luego la rellena con los datos de las ranuras para el sprite de la instancia. Estos datos se analizan para extraer los mapas DS individuales con los datos de las ranuras. A continuación se comprueba si hay una ranura vacía, y si es así se asigna a la variable "slot_name" se le asigna el nombre de la ranura vacía antes de destruir el mapa DS. Finalmente, destruimos la lista de DS porque ya no es necesaria.
