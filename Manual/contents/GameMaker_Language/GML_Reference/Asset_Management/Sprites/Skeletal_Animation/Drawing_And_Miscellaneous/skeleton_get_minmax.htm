esqueleto_obtener_mínimo

Esta función devolverá un array con el valor total de los cuadros delimitadores para todos los cuadros delimitadores individuales asignados a un sprite de animación de esqueleto. El array devuelto tendrá 4 elementos:

  • 0] - la posición x mínima para todas las cajas delimitadoras
  • 1] - la posición y mínima de todas las cajas delimitadoras
  • 2] - la posición x máxima de todas las cajas delimitadoras
  • 3] - la posición máxima en y de todas las cajas delimitadoras

Sintaxis:

skeleton_get_minmax();

Devuelve:

Array (4 elementos: xMin, yMin, xMax, yMax)

Ejemplo:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
{
var box = skeleton_get_minmax();
var ww = (box[2] - box[0]) / 2;
var hh = (box[3] - box[1]) / 2;
part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

El código anterior comprobará el fotograma actual de la animación asignada a la pista 0 y luego hará estallar algunas partículas desde un punto en el centro de la caja delimitadora total para el sprite.
