dibujar_esqueleto_colisión

Esta función dibujará las máscaras de colisión asociadas con la animación esquelética dada. Se suministra el sprite base, el conjunto de animación a utilizar y el fotograma del que se obtendrá la información, y también se pueden establecer las propiedades de transformación para adaptarse.

Sintaxis:

draw_skeleton_collision(sprite, animname, frame, x, y, xscale, yscale, rot, color)

Argumento Descripción
sprite El índice del sprite a dibujar.
animname El nombre de la animación de la que se va a obtener el fotograma (una cadena).
marco El cuadro de animación a dibujar (de 0 a número_de_imagen - 1).
x La coordenada x de donde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
yscale La escala vertical del sprite, como un multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
pudrirse La rotación del sprite. 0=normal, 90=girado 90 grados en sentido contrario a las agujas del reloj, etc.
color El color con el que se mezclará el sprite.

Devuelve:

N/A

Ejemplo:

draw_skeleton_collision(sprite_index, "jump", image_index, x, y, image_xscale, image_yscale, image_angle, c_white);

El código anterior dibujará los datos de la máscara de colisión para el sprite actual, utilizando las transformaciones actuales, para el conjunto de animación "salto".
