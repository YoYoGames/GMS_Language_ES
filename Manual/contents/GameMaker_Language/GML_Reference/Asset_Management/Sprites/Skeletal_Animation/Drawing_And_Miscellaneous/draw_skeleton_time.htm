draw_skeleton_time

Esta función dibujará la animación dada usando el skin dado en un momento específico de la animación. El valor del tiempo debe estar entre 0 (el comienzo) y la duración final de la animación, que puedes encontrar usando la función esqueleto_animación_obtener_duración(). Puedes establecer el valor del tiempo en valores superiores a la duración total de la animación y la animación volverá al principio, pero corres el riesgo de perder precisión en punto flotante a medida que el tiempo acumulado se hace más grande.

Sintaxis:

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, color)

Argumento Descripción
sprite El índice del sprite a dibujar.
animname El nombre de la animación de la que se va a obtener el fotograma (una cadena).
nombre de la piel El nombre de la piel a utilizar para el dibujo (una cadena).
tiempo El marco de tiempo para dibujar la animación (de 0 a la duración final, en segundos).
x La coordenada x de donde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
yscale La escala vertical del sprite, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
pudrirse La rotación del sprite. 0=normal, 90=girado 90 grados en sentido contrario a las agujas del reloj, etc.
color El color con el que se mezclará el sprite.

Devuelve:

N/A

Ejemplo:

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

El código anterior dibujará el sprite de animación esquelético dado utilizando el tiempo delta para establecer el cuadro que se dibuja.
