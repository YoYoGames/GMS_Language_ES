Dibujo y otros

Las siguientes funciones son herramientas adicionales para obtener diferentes conjuntos de datos sobre un sprite de animación de esqueleto:

  • skeleton_get_minmax
  • skeleton_get_num_bounds
  • skeleton_get_bounds

También puedes utilizar varias funciones de dibujo diseñadas específicamente para este tipo de sprites (aparte de las funciones regulares draw_sprite() ). Estas funciones le dan más control sobre lo que se está dibujando y le permiten cambiar las características específicas de la animación, como el tiempo y las pieles:

  • skeleton_collision_draw_set
  • dibujar_esqueleto
  • dibujar_esqueleto_instancia
  • dibujar_esqueleto_colisión
  • draw_skeleton_time
