skeleton_collision_draw_set

Con esta función, puedes activar(true) o desactivar(false) el dibujo de los datos de colisión para el sprite de animación actual que está siendo utilizado por la instancia. Si está activada, la caja delimitadora y la máscara de colisión precisa se dibujarán como contornos alrededor del sprite. se dibujarán como contornos alrededor del sprite.

Sintaxis:

skeleton_collision_draw_set(flag);

Argumento Descripción
bandera Establecer en true para activar el dibujo, y en false para desactivarlo.

Devuelve:

N/A

Ejemplo:

si debug_mode == true
{
skeleton_collision_draw_set(true);
}

El código anterior comprueba si el juego se está ejecutando en modo de depuración y, si lo está, activa los datos de colisión del esqueleto para que se muestren.
