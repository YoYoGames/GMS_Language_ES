lista_de_pieles_del_esqueleto

Con esta función puedes rellenar una lista (pre-creada) de DS con todos los nombres de las pieles incluidas como parte del sprite de la animación esquelética. Los nombres serán cadenas y podrán ser utilizados en las usarse en las otras funciones de skeleton animation skin para estos tipos de sprite.

Sintaxis:

skeleton_skin_list(sprite, list);

Argumento Descripción
sprite El índice del sprite de la animación del esqueleto de la columna vertebral para obtener la lista.
lista El ID de la lista de DS a rellenar con los nombres de las animaciones.

Devuelve:

N/A

Ejemplo:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

El código anterior crea una lista DS y la rellena con los nombres de las pieles. Luego se elige uno al azar y se aplica a la instancia antes de destruir la lista.
