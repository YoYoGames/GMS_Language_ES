skeleton_skin_get

Con los sprites de animación esquelética, puedes asignar texturas separadas (llamadas "skins") a la animación, usando así una animación para múltiples cosas diferentes. Esta función devolverá el nombre de la piel (como una cadena) que está actualmente asignado al sprite de la animación esquelética que está utilizando su instancia. El nombre devuelto es el que usted estableció cuando creó el sprite en su programa de animación.

Sintaxis:

skeleton_skin_get();

Devuelve:

Cadena

Ejemplo:

if skeleton_skin_get() == "skin_Enemy1"
{
skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

El código anterior comprobaría la piel actualmente asignada para la animación y si es "skin_Enemy1", elige y establece una nueva piel de una de las tres opciones.
