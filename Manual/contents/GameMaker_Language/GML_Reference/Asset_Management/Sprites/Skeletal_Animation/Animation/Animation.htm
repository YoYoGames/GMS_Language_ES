Animación

Cuando se trabaja con sprites de animación esquelética creados con Spine, se pueden tener múltiples animaciones por sprites, y como tal se necesita poder cambiar entre ellas en los momentos apropiados del juego. Para ayudarte a hacerlo, GameMaker Studio 2 ofrece las siguientes funciones:

  • skeleton_animation_get
  • skeleton_animation_set
  • skeleton_animation_get_ext
  • skeleton_animation_set_ext
  • skeleton_animation_get_duration
  • skeleton_animation_mix
  • lista_de_animación_del_esqueleto
  • esqueleto_animación_claro
  • skeleton_animation_get_frames
  • skeleton_animation_get_frame
  • skeleton_animation_set_frame
