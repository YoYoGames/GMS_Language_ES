lista_de_animación_del_esqueleto

Con esta función puedes rellenar una lista (pre-creada) de DS con todos los nombres de las animaciones incluidas como parte del sprite de animación esquelética. Los nombres serán cadenas y podrán ser utilizados ser utilizados en las otras funciones de animación para estos tipos de sprites.

Sintaxis:

skeleton_animation_list(sprite, list);

Argumento Descripción
sprite El índice del sprite de la animación del esqueleto de la columna vertebral para obtener la lista.
lista El ID de la lista de DS a rellenar con los nombres de las animaciones.

Devuelve:

N/A

Ejemplo:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);
for (var i = 1; i < ds_list_size(list); i++;)
{
skeleton_animation_mix(list[| 0], list[| i], 0.5);
}
ds_list_destroy(list);

El código anterior crea una lista DS de todos los nombres de animación para el sprite que está siendo utilizado por la instancia. A continuación, hace un bucle a través de ellos y establece el valor de la mezcla para todos ellos con la primera animación a 0,5.
