skeleton_animation_set_frame

Esta función establecerá la animación asignada a la pista dada en el fotograma que proporciones. El índice de fotogramas debe ser un valor real entre 0 y el número de fotogramas de la animación (que puedes obtener usando la función skeleton_animation_get_frames). Si proporcionas un valor fuera de este rango, la animación se repetirá para compensar la diferencia, por ejemplo, si la animación tiene 5 fotogramas y estableces el fotograma en 8, el fotograma real mostrado será 3.

Sintaxis:

skeleton_animation_set_frame(track, index);

Argumento Descripción
pista La pista de animación para establecer el marco de.
índice El nuevo índice del fotograma de la animación.

Devuelve:

N/A

Ejemplo:

si global.pause
{
skeleton_animation_set_frame(0, -image_speed);
}

El código anterior establecerá el índice de fotogramas para la animación en la pista 0 a la velocidad negativa de la imagen cuando la variable global "pause" sea verdadera (esencialmente pausando la animación).
