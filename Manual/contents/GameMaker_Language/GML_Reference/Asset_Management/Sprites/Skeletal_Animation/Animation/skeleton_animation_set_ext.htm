skeleton_animation_set_ext

Cuando estableces un set de animación para tu sprite de animación esquelética, se le asigna la pista 0 internamente. Sin embargo, puede asignar otros conjuntos a otras pistas (1 y superiores) y de esta manera hacer que su sprite reproduzca dos (o más) animaciones al mismo tiempo. Tenga en cuenta que el sprite debe haber sido diseñado en su programa de animación con esto en mente, de tal manera que la animación "base" en la pista 0 sólo tiene fotogramas clave para ciertos huesos, mientras que las siguientes pistas tienen fotogramas clave para los demás. Por ejemplo, si tienes una animación de "caminar" y sus fotogramas clave sólo afectan a las piernas del sprite, puedes tener un conjunto de animaciones de la cabeza y de los brazos que puedes establecer en las pistas 1 y 2 respectivamente, permitiéndote tener un movimiento base y adaptarlo en tiempo real mientras se desarrolla el juego.

Sintaxis:

skeleton_animation_set_ext(animname, track);

Argumento Descripción
animname El nombre (una cadena) del conjunto de animaciones a utilizar.
pista La pista del set de animación a utilizar.

Devuelve:

N/A

Ejemplo:

if mouse_check_button(mb_left)
{
skeleton_animation_set_ext("bodyfight", 1);
}
else
{
skeleton_animation_set_ext("bodywalk", 1);
}

El código anterior cambiará el conjunto de animación que está siendo utilizado por la pista 1 al conjunto "bodyfight" o al conjunto "bodywalk" cuando se presione el botón izquierdo del ratón.
