skeleton_animation_get_frames

Esta función puede utilizarse para obtener el número de fotogramas que tiene una determinada animación de esqueleto. Se proporciona el nombre de la animación del esqueleto (como una cadena, tal y como se define en el programa utilizado para hacer la animación, o como se devuelve utilizando la función esqueleto_animación_gety la función devuelve los fotogramas que tiene como un valor entero. La función devolverá 0 si la animación especificada no existe.

Sintaxis:

skeleton_animation_get_frames(anim_name);

Argumento Descripción
nombre_de_animación El nombre de la animación para obtener los fotogramas.

Devuelve:

Entero

Ejemplo:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

El código anterior obtendrá el número de fotogramas de la animación y luego pondrá el sprite en el último fotograma y dejará de animar.
