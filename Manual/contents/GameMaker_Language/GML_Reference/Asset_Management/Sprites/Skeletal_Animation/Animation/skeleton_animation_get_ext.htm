skeleton_animation_get_ext

Un solo sprite de animación de esqueleto puede tener varios conjuntos de animación, y a estos conjuntos se les pueden asignar diferentes pistas para que puedas "mezclar y combinar" conjuntos de animación. Esta función devolverá el nombre del conjunto de animación actualmente utilizado por el número de pista dado (como se establece en la función skeleton_animation_set_ext).

Sintaxis:

skeleton_animation_get_ext(pista);

Argumento Descripción
pista El número de pista para obtener el nombre del conjunto de animación.

Devuelve:

Cadena

Ejemplo:

if skeleton_animation_get_ext(1) != "bodyfight"
{
skeleton_animation_set_ext("bodyfight", 1);
}

El código anterior cambiará el conjunto de animación que está siendo utilizado por la pista 1 a "bodyfight" si no lo está ya.
