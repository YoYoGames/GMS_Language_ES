skeleton_animation_set

Una vez que hayas asignado un sprite de animación esquelética a tu instancia, puedes definir qué conjunto de animación debe utilizar en cada momento con esta función. Cuando creaste tu sprite (en Spine) habrás definido múltiples conjuntos de animación para los diferentes estados y les habrás dado el nombre adecuado. Estos nombres son los que utilizarás para decirle a GameMaker Studio 2 qué conjunto de animación debe utilizar en cualquier momento de tu juego. Por ejemplo, si su sprite es para un juego de plataformas, entonces usted puede tener un conjunto de "salto", un conjunto de "carrera" y un conjunto de "inactividad", todos contenidos dentro de un sprite y puede cambiar entre ellos en su juego utilizando esta función.

Tenga en cuenta que al establecer una nueva animación de esta manera se restablecerá el valor de image_index a 0, iniciando el nuevo conjunto de animación desde el principio de su bucle de animación. Si este no es el comportamiento comportamiento deseado, entonces puedes establecer un valor de "mezcla" usando skeleton_animation_mix() y cuando establezca una nueva animación se interpolará suavemente con el conjunto anterior. También debes asegurarte de asegurarse de tener un fotograma clave al principio de cada animación para evitar que los datos se retengan de las animaciones anteriores, a menos que planee usar pistas de animación separadas.

Sintaxis:

skeleton_animation_set(animname);

Argumento Descripción
animname El nombre (una cadena) del conjunto de animaciones a utilizar.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(vk_space)
{
if skeleton_animation_get() != "jump"
{
skeleton_animation_set("jump");
}
}

El código anterior cambiará el set de animación que se está utilizando al set de "salto" cuando se pulse la tecla espacio, pero sólo si el set actual que se está utilizando no es ya "salto".
