skeleton_animation_get_duration

Esta función devolverá el tiempo necesario para que el conjunto de animación dado se ejecute antes de volver al principio. El valor de retorno está en segundos.

Sintaxis:

esqueleto_animación_obtener_duración(nombre_de_animación);

Argumento Descripción
animname El nombre (una cadena) del conjunto de animaciones a utilizar.

Devuelve:

Real

Ejemplo1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0,5) % número_de_imagen;
índice_de_imagen = marco;
draw_self();

El código anterior establecerá el image_index al valor correcto para el sprite de animación esquelético asignado actualmente.

Ejemplo2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

El código anterior dibujará el sprite de animación esquelético dado utilizando el tiempo delta para establecer el cuadro que se dibuja.
