skeleton_attachment_create

Cuando creas tu sprite de animación esquelético, puedes asignar ranuras de adición y archivos adjuntos para ir en ellas. Estas son simplemente imágenes que están separadas de la animación, pero cuando se adjuntan se moverán junto con el hueso al que están unidos. Normalmente asignarías estos adjuntos en tu programa de animación (Spine), pero con esta función puedes crear los tuyos propios en tiempo de ejecución usando un activo sprite de tu juego. la función requiere que le des un nombre al adjunto (como una cadena) y luego establezcas el sprite_index y el image_index a utilizar, así como el origen x e y (que puede ser diferente al definido por el sprite en las propiedades del sprite), y también puedes establecer cualquier transformación que desee que se apliquen a la imagen cuando se adjunte. Si el archivo adjunto se ha creado con éxito, la función devolverá 1, y si no lo ha hecho (ha proporcionado un índice de sprite no válido, o el sprite base no es un sprite de Spine ) entonces devolverá -1.

Sintaxis:

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

Argumento Descripción
nombre El nombre (en forma de cadena) del archivo adjunto a crear.
sprite El sprite_index para obtener la imagen adjunta.
ind El image_index para obtener la imagen adjunta.
xorigin El origen x de la imagen utilizada.
yorigen El origen y de la imagen utilizada.
xscale La escala horizontal de la imagen, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
yscale La escala vertical de la imagen, como multiplicador: 1 = escala normal, 0,5 es la mitad, etc.
pudrirse La rotación de la imagen. 0=normal, 90=girada 90 grados en sentido contrario a las agujas del reloj, etc.

Devuelve:

1 si tiene éxito, -1 si no

Ejemplo:

skeleton_attachment_create("espada", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("cuchillo", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("ballesta", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

El código anterior comprobaría el nombre de adjunto asignado actualmente para la ranura llamada "slot_leftHand" y si se devuelve una cadena vacía, se adjunta un nuevo sprite.
