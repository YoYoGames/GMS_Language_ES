skeleton_attachment_get

Un sprite de animación esquelética puede tener otros sprites añadidos como adjuntos, siendo estos sprites añadidos a una ranura con nombre (el nombre se da cuando se crea la ranura del adjunto en su programa de animación) y serán dibujados junto con la animación del sprite actual. Con esta función puedes obtener el nombre (como una cadena) del adjunto para la ranura dada del sprite actualmente asignado. Ten en cuenta que los sprites adjuntos se referencian a través de su cadena de nombre tal y como se asigna en Spine, o cuando se llama a skeleton_attachment_create().

Sintaxis:

skeleton_attachment_get(slot);

Argumento Descripción
ranura El nombre de la ranura (una cadena) para obtener el archivo adjunto.

Devuelve:

Cadena

Ejemplo:

if skeleton_attachment_get("slot_leftHand") == ""
{
skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

El código anterior comprobaría el nombre de adjunto asignado actualmente para la ranura llamada "slot_leftHand" y si se devuelve una cadena vacía, se adjunta un nuevo sprite.
