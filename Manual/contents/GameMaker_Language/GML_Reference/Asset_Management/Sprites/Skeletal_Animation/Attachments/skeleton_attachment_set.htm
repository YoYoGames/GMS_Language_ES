esqueleto_adherido_set

Un sprite de animación esquelética puede tener otras imágenes añadidas como adjuntos, siendo estas imágenes añadidas a una ranura con nombre (el nombre se da cuando se crea la ranura de adjuntos en su programa de animación) y serán dibujadas junto con la animación de el sprite actual. Con esta función se puede establecer un adjunto a una ranura determinada, donde se requiere dar los nombres (como cadenas) de la ranura y el adjunto. Estos nombres son definidos por el programa de animación utilizado, o (en el caso del adjunto) cuando se llama a skeleton_attachment_create().

Ten en cuenta que también puedes pasar un sprite_index como adjunto, y ese sprite se utilizará, o puedes utilizar -1 para eliminar el adjunto de la ranura. Cuando pasas un índice de sprite como argumento, se creará una ranura de adjunto utilizando el nombre del sprite como la cadena para nombrar la ranura (así que usando spr_sword, por ejemplo, creará una ranura de adjunto "spr_sword"), y la ranura utilizará el primer índice de imagen (0) del sprite, así como su desplazamiento de origen x e y, con una escala de (1,1) y una rotación de 0.

Sintaxis:

skeleton_attachment_set(slot, attachment);

Argumento Descripción
ranura El nombre de la ranura (una cadena) para obtener el archivo adjunto.
archivo adjunto El nombre (como una cadena o un sprite_index) de la imagen adjunta.

Devuelve:

Cadena

Ejemplo:

if skeleton_attachment_get("slot_leftHand") == ""
{
skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

El código anterior comprobaría el nombre de adjunto asignado actualmente para la ranura llamada "slot_leftHand" y si se devuelve una cadena vacía, se adjunta un nuevo sprite.
