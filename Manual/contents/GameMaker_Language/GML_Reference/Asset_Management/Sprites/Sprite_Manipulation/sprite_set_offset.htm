sprite_set_offset

Esta función se puede utilizar para establecer el origen x e y de un sprite, y toma valores relativos basados en que la posición (0,0) es la esquina superior izquierda del sprite. La siguiente imagen lo ilustra:

NOTA: Esta función afecta al activo del sprite para que todas las instancias posteriores con este sprite tengan el mismo desplazamiento.

Sintaxis:

sprite_set_offset(ind, xoff, yoff);

Argumento Descripción
ind El índice del sprite para cambiar el desplazamiento.
xoff La posición x del origen.
yoff La posición y del origen.

Devuelve

N/A

Ejemplo:

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

El código anterior asigna el sprite indexado en "spr_Base" al sprite indexado en "spr_Custom" y luego utiliza los valores de desplazamiento x e y de "spr_Base" para establecer el origen del nuevo sprite.
