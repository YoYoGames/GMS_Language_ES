sprite_flush_multi

Con esta función puedes eliminar de la memoria de textura cualquier número de páginas de textura para los sprites dados. Se suministran los índices de los sprites como una matriz y las páginas de textura a las que están asignadas se eliminarán de la memoria de textura. Tenga en cuenta que si una de las páginas de textura se utiliza en otra parte de la sala (por otro sprite de instancia o un fondo, etc...) se puede obtener un pequeño impacto en el rendimiento ya que la página se vuelve a cargar inmediatamente en la memoria, por lo que se debe tener cuidado al utilizar esta función. Tenga en cuenta que la función devolverá -1 si la descarga no es compatible con los recursos elegidos, o devolverá 0 si todo ha funcionado correctamente.

Sintaxis:

sprite_flush_multi(array);

Argumento Descripción
matriz Matriz con los índices de los sprites que se van a descargar

Devuelve:

Int (-1 o 0)

Ejemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

El código anterior crea un array donde cada elemento contiene un índice de sprite. Este array se utiliza para borrar las texturas de los sprites de la memoria.
