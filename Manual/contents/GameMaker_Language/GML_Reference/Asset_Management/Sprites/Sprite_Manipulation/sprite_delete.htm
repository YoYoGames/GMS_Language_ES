sprite_delete

Esta función borrará un sprite del juego, liberando cualquier memoria que estuviera reservada para él. Esta es una función muy importante para aquellos momentos en los que necesitas crear y cambiar sprites desde fuentes externas (como cargar un sprite desde un archivo con sprite_add() , o duplicar un sprite usando sprite_duplicate()) y siempre debe ser usada para eliminar aquellos activos que ya no son que ya no son necesarios para el juego, o para borrar un activo indexado de una variable antes de reasignar otro activo a esa variable. Esto no puede ser usado para borrar sprites que están incluidos en el juego como parte de los activos en el Navegador de Activos.

Sintaxis:

sprite_delete(índice);

Argumento Descripción
índice El índice del sprite a eliminar.

Devuelve

N/A

Ejemplo:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

El código anterior crea una variable local y luego almacena el índice del sprite creado desde la superficie de la aplicación. Este sprite se fusiona entonces con el activo indexado en la variable "spr_Player" antes de volver a ser eliminado de la memoria.
