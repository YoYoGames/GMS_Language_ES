sprite_prefetch_multi

Esta función se puede utilizar para pre-obtener (colocar en la memoria de la textura) un número de páginas de textura que contienen los sprites dados. Se suministra un array con los índices de los sprites (tal y como se definieron al crear el activo de sprites) y las páginas de textura en las que se encuentran se cargarán en la memoria. se cargarán en la memoria. Tenga en cuenta que la función devolverá -1 si la precarga no es compatible con el recurso elegido o la plataforma de destino es HTML5, o devolverá 0 si todo ha funcionado correctamente.

NOTA: Hay un impacto en el rendimiento cuando la textura se carga en la memoria de textura en la mayoría de los dispositivos, por lo que se recomienda llamar a sprite_prefetch_multi() en cualquier gráfico necesario al comienzo de un nivel para evitar cualquier bloqueo.

Sintaxis:

sprite_prefetch_multi

Argumento Descripción
matriz Matriz con los índices de los sprites a recuperar

Devuelve:

Int (-1 o 0)

Ejemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

El código anterior crea un array donde cada elemento contiene un índice de sprite. Esta matriz se utiliza para colocar esas texturas de sprites en la memoria.
