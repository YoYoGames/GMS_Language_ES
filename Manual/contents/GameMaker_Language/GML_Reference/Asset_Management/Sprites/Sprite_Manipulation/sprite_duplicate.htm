sprite_duplicate

Esta función devolverá el índice de un sprite recién creado que es un duplicado (copia) del introducido como argumento "índice". Si la función falla, se devuelve -1. Esta función debe utilizarse para copiar cualquier sprites de los activos originales antes de poder realizar cualquier transformación en ellos. Por ejemplo, si desea cambiar el cuadro delimitador de un sprite, o establecer su alfa desde otro sprite, primero debe duplicarlo, luego realizar la operación en el duplicado y utilizarlo. Un sprite duplicado se colocará en su propia página de textura única cuando se cree, lo que significa que la duplicación de múltiples sprites creará múltiples páginas de textura y tendrá un impacto en el rendimiento, por lo que utilice esta función sólo cuando sea necesario.

NOTA: Cuando duplicas un sprite en GameMaker Studio 2 debes recordar eliminarlo de nuevo (con sprite_delete()) cuando ya no lo necesites, de lo contrario existe el riesgo de que se produzca una fuga de memoria que ralentizará y eventualmente bloqueará tu juego.

Sintaxis:

sprite_duplicate(índice);

Argumento Descripción
índice El índice del sprite a duplicar.

Devuelve

Real (valor único de identificación del sprite)

Ejemplo:

new_spr = sprite_duplicate(sprite_index)

El código anterior duplica el sprite que se está utilizando actualmente como índice del sprite de la instancia y almacena el índice de este nuevo sprite en la variable "new_spr".
