sprite_set_bbox_mode

Esta función se puede utilizar para establecer el modo de caja delimitadora para un sprite. Se proporciona el índice del sprite y el modo a utilizar, que debe ser una de las siguientes constantes:

Constante Descripción
bboxmode_automatic Automático - El cuadro delimitador se calculará automáticamente, basándose en la configuración de la tolerancia del sprite
bboxmode_fulllimage Imagen completa: el cuadro delimitador se establecerá para utilizar toda la anchura y la altura del sprite, independientemente de la tolerancia y los píxeles "vacíos".
bboxmode_manual Manual - El cuadro delimitador se establece manualmente con valores definidos por el usuario, lo que requiere el uso de la función sprite_set_bbox()

NOTA: Esta función afecta al activo sprite de manera que todas las instancias posteriores con este sprite tendrán el mismo modo de caja delimitadora.

Sintaxis:

sprite_set_bbox_mode(ind, mode);

Argumento Descripción
ind El índice del sprite para cambiar el modo.
modo El modo que se debe establecer (una constante).

Devuelve

N/A

Ejemplo:

if sprite_get_bbox_mode(sprite_index) != 0
{
sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Lo anterior comprueba el modo bbox para el sprite actual y si no es automático, entonces se establece en ese valor.
