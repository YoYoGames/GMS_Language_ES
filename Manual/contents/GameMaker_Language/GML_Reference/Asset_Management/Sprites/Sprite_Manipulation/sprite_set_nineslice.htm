sprite_set_nineslice

Esta función se utiliza para aplicar una estructura Nine Slice a un sprite. Se suministra el índice del sprite y la estructura Nine Slice a aplicar, que puede ser recuperada usando sprite_nineslice_create() o sprite_get_nineslice().

NOTA: Esta función afecta al activo sprite, por lo que cualquier cambio que se haga con esta función afectará a todas las instancias que estén usando este sprite y a todas las instancias futuras también.

Sintaxis:

sprite_set_nineslice(ind, nineslice);

Argumento Descripción
ind El índice del sprite a modificar
nineslice La estructura de nueve rodajas para aplicar

Devuelve:

N/A

Ejemplo:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

El código anterior crea una nueva estructura Nine Slice, la habilita y establece los desplazamientos de las guías. La estructura se aplica entonces a un sprite, cambiando sus propiedades de Nine Slice.
