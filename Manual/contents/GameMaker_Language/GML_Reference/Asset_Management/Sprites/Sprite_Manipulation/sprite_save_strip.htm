sprite_save_strip

Esta función creará una imagen de tira a partir de todas las subimágenes de un sprite, guardándola en el disco con el nombre de archivo especificado. El sprite debe haber sido añadido en tiempo de ejecución (no puede guardar sprites añadidos a través del IDE) y el archivo debe guardarse con una extensión *.png.

NOTA: Dependiendo de la plataforma de destino que se elija, usted está limitado en cuanto a dónde puede guardar y cargar archivos. Consulte la sección Manejo de archivos para obtener más información.

Sintaxis:

sprite_save_strip(ind, filename);

Argumento Descripción
ind El índice del sprite para cambiar el mapa de transparencia.
nombre de archivo El nombre del archivo para la tira de sprites guardada.

Devuelve

N/A

Ejemplo:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);
for (i = 1; i < 8; i +=1)
{
sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}
sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

El código anterior crea un sprite a partir de la superficie indexada en la variable "surf", asignando su índice a la variable local "spr_Custom", y luego utiliza un bucle for para moverse por la superficie y capturar varias secciones que Este nuevo sprite se guarda como una tira de png antes de ser eliminado de la memoria.
