sprite_set_bbox

Esta función se puede utilizar para establecer los valores de la caja delimitadora de un sprite. Se suministra el índice del sprite que se va a utilizar y luego los valores izquierdo, superior, derecho e inferior para las posiciones del cuadro delimitador. Las posiciones son valores absolutos, donde la posición (0, 0) corresponde a la esquina superior izquierda del sprite, sin tener en cuenta el desplazamiento del sprite, los píxeles "vacíos" que pueda tener el sprite o el lugar de la sala en el que se esté dibujando.

NOTA: Esta función afecta al activo sprite de manera que todas las instancias posteriores con este sprite tendrán la misma caja delimitadora.

Sintaxis:

sprite_set_bbox(ind, left, top, right, bottom);

Argumento Descripción
ind El índice del sprite para establecer la caja delimitadora.
a la izquierda El lado izquierdo del cuadro delimitador
top La parte superior del cuadro delimitador.
a la derecha El lado derecho del cuadro delimitador
fondo La parte inferior del cuadro delimitador.

Devuelve

N/A

Ejemplo:

si sprite_get_bbox_mode(sprite_index) == 2
{
left = irandom(sprite_width / 2);
right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
top = irandom(sprite_height / 2);
bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
sprite_set_bbox(sprite_index, izquierda, arriba, derecha, abajo);
}

El código anterior comprobará el modo de caja delimitadora del sprite asignado al sprite_index, y si está establecido como manual entonces tendrá su caja delimitadora cambiada.
