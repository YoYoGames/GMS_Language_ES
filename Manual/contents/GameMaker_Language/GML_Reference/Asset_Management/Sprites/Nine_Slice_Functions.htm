Nueve funciones de corte

Además de la sección de Nine Slice en el Editor de Sprite, GameMaker Studio 2 proporciona funciones para establecer las propiedades de Nine Slice para los sprites en tiempo de ejecución. Antes de leer sobre estas funciones, se recomienda leer esta página para obtener una comprensión de la técnica Nine Slice.

Las siguientes funciones se pueden utilizar para recuperar y cambiar las propiedades de Nine Slice para un activo de sprite en tiempo de ejecución:

  • sprite_nineslice_create
  • sprite_get_nineslice
  • sprite_set_nineslice

Estas funciones hacen uso de una estructura Nine Slice, y puedes leer sobre el contenido de esa estructura en esta página.
