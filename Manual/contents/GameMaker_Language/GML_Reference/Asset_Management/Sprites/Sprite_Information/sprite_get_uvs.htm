sprite_get_uvs

Esta función devuelve un array con las coordenadas UV y otros datos de la textura de la subimagen del sprite en la página de textura. La función devuelve un array con los siguientes 8 elementos:

  • [0] = izquierda
  • [1] = arriba
  • [2] = derecho
  • [3] = fondo
  • 4] = cantidad de píxeles que el compilador de activos ha recortado del lado izquierdo de los sprites
  • 5] = cantidad de píxeles que el compilador de activos ha recortado de la parte superior de los sprites
  • 6] = porcentaje normalizado de datos de píxeles del ancho original de los sprites que se ha guardado en la página de la textura
  • 7] = porcentaje normalizado de datos de píxeles de la altura de los sprites originales que se ha guardado en la página de la textura

Esta matriz puede ser utilizada en otras funciones de dibujo, particularmente cuando se trabaja en 3D o se utilizan las funciones primitivas 2D, así como cuando se trabaja con las funciones Shader funciones.

NOTA: Esta función no funcionará con sprites vectoriales o sprites de animación de esqueleto.

Sintaxis:

sprite_get_uvs(sprite, subimagen);

Argumento Descripción
sprite El índice del sprite a utilizar.
subimagen La sub-imagen del sprite a utilizar.

Devuelve

Matriz (1D, 8 elementos)

Ejemplo:

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

El código anterior almacenará las coordenadas UV para el sprite dado en un array local y luego asignará los valores a las variables de instancia.
