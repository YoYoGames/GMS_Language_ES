sprite_get_name

Esta función devolverá el nombre en forma de cadena del sprite especificado. Este nombre es el que se ha especificado para el sprite en el árbol de recursos de la ventana principal de GameMaker Studio 2. Tenga en cuenta que esto es sólo una cadena y no se puede usar para referenciar el sprite directamente - para eso necesitarías el índice del sprite. Sin embargo, puedes utilizar esta cadena para obtener el índice del sprite utilizando la cadena devuelta junto con la función asset_get_index().

Sintaxis:

sprite_get_name(índice);

Argumento Descripción
índice El índice del sprite para obtener el nombre.

Devuelve

Cadena

Ejemplo:

str = sprite_get_name(sprite_index);

El código anterior obtendrá el nombre del índice del sprite para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "str".
