sprite_get_nineslice

Esta función devuelve la estructura Nine Slice de un sprite, que contiene todas sus propiedades Nine Slice tal y como se establecen en el Editor de Sprite o utilizando sprite_set_nineslice(). El contenido de esta estructura se detalla en esta página.

Si el sprite suministrado no tiene una estructura de Nine Slice asignada, se creará una nueva estructura con las propiedades predeterminadas de Nine Slice para el sprite y se devolverá.

El cambio de cualquier valor en esta estructura modificará las propiedades de Nine Slice del sprite original, afectando a cualquier llamada de dibujo futura realizada con ese sprite.

Sintaxis:

sprite_get_nineslice(ind);

Argumento Descripción
ind El índice del sprite del que se recuperará la estructura Nine Slice

Devuelve:

Struct (o -1 si el sprite no existe)

Ejemplo:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

El código anterior recupera la estructura Nine Slice de un sprite, activa el Nine Slicing para él y establece los desplazamientos de la guía.
