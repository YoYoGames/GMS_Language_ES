Información sobre Sprite

Las siguientes funciones se pueden utilizar para obtener información sobre un activo sprite. Tenga en cuenta que estas funciones obtienen la información del sprite base (es decir, el activo sprite, tal y como se creó en el navegador de activos) y no del sprite tal y como se está utilizando en una instancia. Por ejemplo, puede utilizar la variable sprite variable de instancia ancho_de_sprite para obtener el ancho del sprite que está utilizando una instancia y esta anchura tendrá en cuenta cualquier escalado que se haya aplicado. Sin embargo, si necesitas saber la anchura de un sprite tal y como sería normalmente, sin transformaciones aplicadas, necesitarías utilizar la función sprite_get_width() que aparece a continuación.

Para más información sobre las variables de instancia de los sprites, consulte la sección de Variables de Instancia.
