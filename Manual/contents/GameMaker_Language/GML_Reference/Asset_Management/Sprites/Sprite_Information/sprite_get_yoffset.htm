sprite_get_yoffset

Cuando defines un sprite en el editor de sprites, necesitas establecer el origen para ese sprite. Este es el punto en el que el sprite será "fijado" o "dibujado" cuando se utilice con una instancia. Esta función devuelve el desplazamiento relativo para el eje y del origen basado en la esquina superior izquierda que es la posición 0,0, con un +y que es hacia abajo y un -y que es hacia arriba de esto. La siguiente imagen lo ilustra:

Sintaxis:

sprite_get_yoffset(índice);

Argumento Descripción
índice El índice del sprite para encontrar el desplazamiento del yo.

Devuelve

Real

Ejemplo:

si y - sprite_get_yoffset(sprite_index) < 0
{
y = sprite_get_yoffset(sprite_index);
}

El código anterior asegurará que se mantenga una instancia dentro de la habitación basada en el sprite (para que el sprite sea siempre visible).
