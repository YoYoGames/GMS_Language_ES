sprite_get_tpe

Esta función devolverá un valor especial que se relaciona con la posición del sprite y la subimagen dados en la página de texturas utilizada por tu juego. De esta manera usted puede pasar el valor a las extensiones para su uso fuera del juego compilado, evitando efectivamente las funciones de dibujo de GameMaker Studio 2 y permitiendo que el sprite sea utilizado en el contenido del DOM que puede ser dibujado en cualquier lugar dentro de la ventana que contiene el lienzo del juego. Esta función es de especial interés para aquellos que deseen crear botones y otros medios interactivos fuera del elemento del lienzo de GameMaker Studio 2 en su página anfitriona utilizando la función clickable_add().

NOTA: Esta función es sólo para HTML5.

Sintaxis:

sprite_get_tpe(sprite, index);

Argumento Descripción
sprite El índice del sprite para encontrar la entrada de la página de textura.
índice El índice de la imagen del sprite.

Devuelve

Real

Ejemplo:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

El código anterior crea un botón DOM en la posición (32, 32) de la página en la que se está ejecutando el lienzo del juego. El botón utiliza el sprite referenciado desde la página de textura como "spr_MS_Home" y cuando se hace clic en el botón se abrirá una nueva ventana para la URL especificada y con las propiedades definidas para esa ventana.
