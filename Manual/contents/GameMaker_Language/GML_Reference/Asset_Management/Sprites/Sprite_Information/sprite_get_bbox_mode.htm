sprite_get_bbox_mode

Esta función devolverá el "modo" actual para los cálculos de la caja delimitadora. Usted proporciona el índice del sprite a comprobar, y la función devolverá una de las constantes que se muestran a continuación.

Sintaxis:

sprite_get_bbox_mode(ind);

Argumento Descripción
ind El índice del sprite a comprobar.

Devuelve

Constante

Constante Descripción
bboxmode_automatic Automático - El cuadro delimitador se calculará automáticamente, basándose en la configuración de la tolerancia del sprite
bboxmode_fulllimage Imagen completa: el cuadro delimitador se establecerá para utilizar toda la anchura y la altura del sprite, independientemente de la tolerancia y los píxeles "vacíos"
bboxmode_manual Manual - El cuadro delimitador se ha ajustado manualmente a los valores definidos por el usuario (ya sea en el editor de sprites, o utilizando la función sprite_set_bbox())

Ejemplo:

if sprite_get_bbox_mode(sprite_index) != 0
{
sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Lo anterior comprueba el modo bbox para el sprite actual y si no es automático, entonces lo establece a ese valor.
