sprite_get_info

Esta función se utiliza para recuperar la información del sprite dado. Se proporciona un índice de sprite (que puede ser un activo añadido a través del navegador de activos o un sprite añadido en tiempo de ejecución) y la función devuelve una estructura con las siguientes variables:

Nombre de la variable Tipo de datos Descripción
ancho real La anchura del sprite (en píxeles)
altura real La altura del sprite (en píxeles)
xoffset real El desplazamiento/origen X del sprite (en píxeles)
yffset real El desplazamiento/origen Y del sprite (en píxeles)
transparente booleano true si el sprite está marcado como transparente, en caso contrario false
(Esto sólo puede ser especificado a través de sprite_add() o funciones similares, y será falsopara los sprites creados en el IDE)
suave booleano true si el sprite está marcado como suave, en caso contrario false
(Esto sólo se puede especificar a través de sprite_add() o funciones similares, y será falso para los sprites creados en el IDE)
tipo real El tipo de sprite:
0 - Mapa de bits (sprites normales)
1 - SWF
2 - Lomo
bbox_left real Posición del borde izquierdo del cuadro delimitador (en píxeles)
bbox_top real Posición del borde superior del cuadro delimitador (en píxeles)
bbox_right real Posición del borde derecho del cuadro delimitador (en píxeles)
bbox_bottom real Posición del borde inferior del cuadro delimitador (en píxeles)
nombre cadena El nombre del sprite
número_subimágenes real El número de sub-imágenes (o cuadros) en el sprite
usar_máscara booleano true si este sprite utiliza una máscara de colisión, en caso contrario false
número_de_máscaras real El número de máscaras en este sprite (será mayor que 1 si el sprite utiliza máscaras por fotograma)
nineslice struct La estructura de nueve cortes para este sprite, o indefinida si no tiene datos de nueve cortes
mensajes matriz Matriz de mensajes de difusión para este sprite, donde cada mensaje de difusión es una estructura que contiene información sobre el mensaje (más información en "Datos generales del sprite")
frame_info matriz Matriz de fotogramas para este sprite, donde cada fotograma es una estructura que contiene información sobre su temporización (más información en "Datos generales del sprite")

Esta variable adicional sólo está disponible para los sprites Bitmap (regulares):

Nombre de la variable Tipo de datos Descripción
marcos matriz Matriz de fotogramas para este sprite, donde cada fotograma es una estructura que contiene información sobre su textura (más información en "Datos del sprite de mapa de bits")

Estas variables adicionales sólo están disponibles para los sprites de la columna vertebral:

Nombre de la variable Tipo de datos Descripción
num_atlas real El número de texturas del atlas utilizadas
atlas_textura matriz Matriz de identificadores de textura utilizados para el atlas
premultiplicado booleano true si este sprite está marcado como premultiplicado, en caso contrario false
nombres_de_animación matriz Array que contiene los nombres de cada animación en este sprite
nombres_de_piel matriz Matriz que contiene los nombres de cada piel de este sprite
huesos matriz Matriz que contiene los structs de cada hueso de este sprite (más información en "Datos del sprite de la columna vertebral")
ranuras matriz Matriz que contiene structs para cada ranura de este sprite (más información en "Datos del sprite de la columna vertebral")

La función devolverá undefined si el sprite dado no existe. También hay que tener en cuenta que la información devuelta en esta estructura debe considerarse de sólo lectura, ya que la modificación de cualquiera de estas variables no afectará al sprite.

Las secciones siguientes contienen información sobre las matrices y los structs incluidos en la estructura devuelta en función del tipo de sprite:

Datos generales de los sprites

Esta sección contiene información sobre las variables incluidas en la estructura para todos los tipos de sprites.

La variable messages es un array que contiene información sobre los mensajes de difusión que existen en el sprite dado. Cada mensaje de difusión en esta matriz es una estructura que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
marco real El tiempo de este mensaje de difusión desde el inicio de la animación (en fotogramas)
mensaje cadena La cadena de mensajes de difusión

La variable frame_info es una matriz que contiene información sobre los tiempos de los fotogramas del sprite. Cada cuadro de esta matriz es una estructura que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
marco real El tiempo para el inicio de este cuadro (en cuadros)
índice_de_imagen real El índice de la imagen de este fotograma

Datos de Sprite de mapa de bits

Esta sección contiene información sobre las variables incluidas en la estructura para los sprites de tipo Bitmap (sprites que no son de tipo Spine o SWF).

La variable frames es un array que contiene información sobre las texturas de los frames del sprite. Cada cuadro de este array es una estructura que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
x real La posición X de este marco en su página de textura (en píxeles)
y real La posición Y de este marco en su página de textura (en píxeles)
w real La anchura lógica de la trama (en píxeles) utilizada internamente
h real La altura lógica del cuadro (en píxeles) utilizada internamente
textura real El ID de la página de la textura para este marco
ancho_original real La anchura original del cuadro (en píxeles)
altura_original real La altura original del cuadro (en píxeles)
ancho_de_recorte real La anchura real del marco en la página de la textura después de recortar y escalar (ya que GMS2 recorta automáticamente el espacio vacío alrededor de una imagen, y también la reduce si no cabe)
altura_de_cultivo real La altura real del marco en la página de la textura después de recortar y escalar
x_offset real El desplazamiento X desde el borde izquierdo de la imagen original hasta el borde izquierdo de la sección recortada
y_offset real El desplazamiento Y desde el borde superior de la imagen original hasta el borde superior de la sección recortada

Datos del Sprite de la columna vertebral

Esta sección contiene información sobre las variables incluidas en la estructura de los sprites de Spine.

La variable bones es una matriz que contiene información sobre todos los huesos del sprite de la columna vertebral. Cada hueso de esta matriz es una estructura que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
padre cadena El nombre del hueso padre, o indefinido si este hueso no tiene padre
nombre cadena El nombre de este hueso
índice real El índice de este hueso
longitud real La longitud de este hueso
x real La posición X de este hueso
y real La posición Y de este hueso
rotación real La rotación de este hueso
escala_x real (Interna a la columna vertebral) Valor de la escala en X
escala_y real (Interna a la columna vertebral) Valor de la escala en Y
shear_x real (Interna a la columna vertebral) Valor de cizallamiento en X
shear_y real (Interna a la columna vertebral) Valor de cizallamiento en Y
modo_de_transformación real (Interno a la columna vertebral) El modo de transformación

NOTA: Consulte la documentación de Spine para obtener más información sobre las variables internas de Spine.

La variable slots es una matriz que contiene información sobre todas las ranuras del sprite de Spine dado. Cada ranura de esta matriz es una estructura que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
nombre cadena El nombre de la ranura
índice real El índice de la ranura
hueso cadena El nombre del hueso de la ranura, o "( ninguno)" si no hay hueso para esta ranura
archivo adjunto cadena Nombre del archivo adjunto
rojo real Componente rojo del color de la ranura (0-1)
verde real Componente verde del color de la ranura (0-1)
azul real Componente azul del color de la ranura (0-1)
alfa real Componente alfa del color de la ranura (0-1)
blendMode real (Interno a la columna vertebral) Modo de mezcla para la ranura
dark_red* real Componente rojo del color oscuro de la ranura (0-1)
dark_green* real Componente verde del color oscuro de la ranura (0-1)
dark_blue* real Componente azul del color oscuro de la ranura (0-1)
dark_alpha* cadena Componente alfa del color oscuro de la ranura (0-1)

NOTA: Las variables de color oscuro sólo están disponibles si la ranura tiene activado el Tinte Negro.

Sintaxis:

sprite_get_info(índice);

Argumento Descripción
índice El índice del sprite para obtener la información.

Devuelve

Struct (o indefinido)

Ejemplo:

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
var _messages = _info.messages;
var _messageCount = array_length(_messages);

for (var i = 0; i < _messageCount; i ++)
{
var _message = _messages[i];
show_debug_message("Mensaje en el marco " + string(_message.frame) + ": " + _message.message);
}
}

El código anterior obtiene la estructura de información para el sprite de la instancia, y luego comprueba si no está indefinido y si su tipo es 0 (lo que significa que es un sprite de mapa de bits). En ese caso, obtiene la matriz de mensajes de difusión para ese sprite y luego ejecuta un bucle for para imprimir cada mensaje de difusión (junto con su marco) en el registro de salida.
