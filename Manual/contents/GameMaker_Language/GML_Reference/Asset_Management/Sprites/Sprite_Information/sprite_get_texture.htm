sprite_get_texture

Esta función devuelve un puntero especial para la página de textura del sprite. Este valor puede ser utilizado en otras funciones de dibujo, particularmente en las funciones de primitivas 2D, así como en las funciones Shader de la función de dibujo. Puedes obtener más información sobre la página de textura devuelta usando las diferentes funciones texture_ que se encuentran aquí.

Sintaxis:

sprite_get_texture(spr, subimg);

Argumento Descripción
spr El índice del sprite a utilizar.
subimg La sub-imagen del sprite a utilizar.

Devuelve

Puntero (ID de la página de textura)

Ejemplo:

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertexture(0, 0, 0, 0);
draw_vertexture(480, 0, 1, 0);
draw_vertexture(480, 640, 1, 1);
draw_vertexture(0, 640, 0, 1);
draw_primitive_end();

El código anterior dibujará una tira de triángulos de 4 vértices texturizados con la textura mantenida en la variable "tex".
