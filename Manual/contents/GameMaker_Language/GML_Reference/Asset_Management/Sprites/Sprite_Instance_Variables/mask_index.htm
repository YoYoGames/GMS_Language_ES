índice_de_máscara

Cuando se define un objeto en GameMaker Studio 2 se le puede asignar una máscara que se utilizará para las colisiones en lugar de la que corresponde al sprite definido. Esta variable se puede utilizar para encontrar el sprite_index de esa máscara (o devolverá -1 si no se ha asignado ningún sprite) o para establecer la máscara de una instancia al sprite elegido. Establecer el índice de la máscara significa que puedes tener, por ejemplo, un sprite para la instancia con una forma irregular, pero darle una máscara de colisión circular que se obtiene de un sprite diferente.

Sintaxis:

mask_index;

Devuelve:

Real (entero - valor del índice de recursos)

Ejemplo:

mask_index = spr_Round;

El código anterior establece la máscara de la instancia a la del sprite "spr_Round".
