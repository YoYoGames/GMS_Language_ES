altura_de_sprites

Esta variable de sólo lectura devuelve la altura del sprite que se ha asignado a la instancia. Esta altura se devuelve en píxeles y dependerá de la variable image_yscale. Si necesitas la altura sin escalar debes usar sprite_get_height().

Sintaxis:

altura_de_sprites;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si sprite_height != sprite_get_height(sprite_index)
{
image_yscale = 1;
}

El código anterior comprueba la altura del sprite asignado a la instancia que ejecuta el código contra la altura del recurso sprite y si no es la misma, restablece el image_yscale a 1,
