número_de_imagen

Esta variable de sólo lectura puede utilizarse para obtener el número de subimágenes de un sprite que ha sido asignado a una instancia (si necesita el número de subimágenes de un sprite distinto del asignado a la instancia debe utilizar debe utilizar sprite_get_number()). Tenga en cuenta que cuando hay (por ejemplo) 1 sub-imagen esta variable devolverá "1" pero el image_index de esa sub-imagen es 0.

Sintaxis:

número_de_imagen;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si velocidad_de_imagen > 0
{
si image_index > image_number - 1 instance_destroy();
}

El código anterior comprueba si el sprite se está animando, y si lo está, comprueba si el image_index actual es mayor que el número de subimágenes y si lo es, destruye la instancia.
