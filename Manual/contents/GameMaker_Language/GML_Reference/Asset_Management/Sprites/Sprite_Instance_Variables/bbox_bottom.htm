bbox_bottom

Esta variable de sólo lectura devuelve la posición y (dentro de la habitación) de la parte inferior del cuadro delimitador para la instancia, donde el cuadro delimitador está definido por la anchura y la altura máximas de la máscara para la instancia (como definidos por sprite_index o por mask_index). Incluso cuando un sprite tiene una máscara de colisión precisa, el cuadro delimitador existe y se utiliza para ciertas cosas, por lo que puedes usar esta variable para encontrarla. Ten en cuenta que cuando la instancia no tiene ningún sprite asignado el valor devuelto será el mismo que la posición Y de la instancia.

Sintaxis:

bbox_bottom;

Devuelve:

Real (entero)

Ejemplo:

si bbox_bottom > room_height
{
y = room_height - sprite_yoffset;
}

El código anterior comprueba si el cuadro delimitador de la instancia está fuera de la habitación y, si lo está, coloca la instancia en una nueva posición.
