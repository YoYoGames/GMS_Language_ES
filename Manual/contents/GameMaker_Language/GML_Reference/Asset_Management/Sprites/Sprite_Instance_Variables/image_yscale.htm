imagen_escala

Este valor establece la escala vertical (a lo largo del eje y) aplicada al sprite que ha sido asignado a la instancia actual. Una escala de 1 indica que no hay escala (1:1), los valores más pequeños reducirán la escala (0.5, por ejemplo, reducirá la altura del sprite a la mitad), los valores mayores aumentarán la escala y los valores negativos reflejarán el sprite y lo escalarán, a menos que el valor utilizado sea exactamente -1 (en cuyo caso el sprite se reflejará a lo largo del eje y sin escalar).

Sintaxis:

image_yscale;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si image_xscale < 5
{
image_xscale += 0.2;
image_yscale = image_xscale;
}
si no
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy();
}

El código anterior escala el sprite y una vez que se escala a 5 veces su tamaño original, se crea una nueva instancia de otro objeto y se destruye la instancia.
