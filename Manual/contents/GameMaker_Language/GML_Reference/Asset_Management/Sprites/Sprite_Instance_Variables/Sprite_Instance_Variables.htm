Variables de instancia de sprites

Estas variables se incluyen por defecto con los objetos de GameMaker Studio 2 y están todas relacionadas con el sprite asignado a la instancia en la sala. Estas variables se pueden acceder en cualquier momento dentro del código o acciones para una instancia y se utilizan para cambiar lo que se dibuja y cómo.

  • sprite_index
  • ancho_de_sprites
  • altura_de_sprites
  • sprite_xoffset
  • sprite_yoffset
  • imagen_alfa
  • ángulo_de_imagen
  • mezcla_de_imágenes
  • índice_de_imagen
  • número_de_imagen
  • velocidad_de_imagen
  • imagen_escala_x
  • imagen_escala
  • índice_de_máscara
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
