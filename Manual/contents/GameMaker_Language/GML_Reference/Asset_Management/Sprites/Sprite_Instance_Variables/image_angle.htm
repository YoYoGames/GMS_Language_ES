ángulo_de_imagen

Este valor establece el ángulo (rotación) del sprite y se mide en grados, siendo la derecha 0º, arriba 90º, izquierda 180º y abajo 270º. Establezca esta variable a 0 para restablecer el sprite a ser dibujado como fue definido en el editor de sprites. Por favor, tenga en cuenta que para que los cambios en esta variable sean visibles, la instancia no debe tener ningún evento de dibujo (y así GameMaker Studio 2 dibujará por defecto el sprite) o se dibujará utilizando una de las funciones de dibujo extendidas como dibujar_self() o dibujar_sprite_ext().

Sintaxis:

ángulo_de_imagen;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

ángulo_de_imagen = punto_dirección(x, y, mouse_x, mouse_y);

El código anterior girará el sprite de la instancia para que siempre apunte a la posición del ratón.
