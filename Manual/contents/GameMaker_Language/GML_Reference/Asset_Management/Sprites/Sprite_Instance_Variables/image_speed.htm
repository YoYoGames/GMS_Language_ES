velocidad_de_imagen

Esta variable determina la velocidad a la que GameMaker Studio 2 recorrerá las subimágenes del sprite de instancia actual. El valor de la velocidad dada es un multiplicador, con 1 es el valor por defecto, y el establecimiento de 0,5 será la mitad de la velocidad de la animación - como se establece en el Editor de Sprite o en el Editor de Imágenes - mientras que si se establece en 2 se duplicará. Si el sprite utilizado no tiene no tiene subimágenes, esta variable no tendrá efecto.

Sintaxis:

velocidad_de_imagen;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
velocidad_de_imagen = 0.5;
}

El código anterior crea una instancia del objeto "obj_Explosion" y establece su image_speed en 0,5 (reduciendo efectivamente la velocidad de ejecución a la mitad).
