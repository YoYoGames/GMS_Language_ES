ancho_de_sprites

Esta variable de sólo lectura devuelve el ancho del sprite que ha sido asignado a la instancia. Esta anchura se devuelve en píxeles y dependerá de la variable image_xscale. Si necesita la anchura sin escalar debe utilizar sprite_get_width().

Sintaxis:

sprite_width;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

if sprite_width != sprite_get_width(sprite_index)
{
image_xscale = 1;
}

El código anterior comprueba el ancho del sprite asignado a la instancia que ejecuta el código contra el ancho del recurso sprite y si no es el mismo, restablece el image_xscale a 1,
