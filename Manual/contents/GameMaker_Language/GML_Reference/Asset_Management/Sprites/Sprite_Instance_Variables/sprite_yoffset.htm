sprite_yoffset

Esta variable de sólo lectura devuelve el yffset local (el componente y del origen definido en el editor de sprites) del sprite que ha sido asignado a la instancia. Este yffset se devuelve en píxeles y dependerá de la variable imagen_escala, de modo que (por ejemplo) si un sprite cuadrado de 32x32 se define con el origen y en 16, cuando ese sprite se escale en 2 esta variable devolverá 32. Si necesitas el origen y sin escalar debes usar sprite_get_yoffset().

Sintaxis:

sprite_yoffset;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si y - sprite_yoffset < 0
{
y = sprite_yoffset;
}

El código anterior comprueba si el borde superior del sprite de la instancia está fuera de la habitación y si lo está mueve la instancia para que todo el sprite sea visible.
