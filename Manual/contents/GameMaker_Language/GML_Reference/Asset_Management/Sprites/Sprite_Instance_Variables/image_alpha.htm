imagen_alfa

Esta variable se utiliza para obtener o establecer el valor alfa del sprite. El alfa siempre se calcula como un valor entre 0 y 1, donde 0 es completamente transparente y 1 es completamente opaco. Tenga en cuenta que para que los cambios en esta variable sean visibles, la instancia no debe tener ningún evento de dibujo (por lo que GameMaker Studio 2 dibujará por defecto el sprite) o se dibujará utilizando una de las funciones de dibujo extendidas como draw_self() o draw_sprite_ext().

Sintaxis:

image_alpha;

Devuelve:

Real (entre 0 y 1 - valor de punto flotante de precisión única)

Ejemplo:

imagen_alfa = clamp(imagen_alfa - 0,01, 0, 1);

El código anterior reducirá lentamente el image_alpha hasta que llegue a 0.
