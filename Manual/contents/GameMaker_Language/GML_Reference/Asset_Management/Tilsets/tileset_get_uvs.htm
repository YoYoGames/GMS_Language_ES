tileset_get_uvs

Esta función devuelve un array con las coordenadas UV y otros datos de la textura del mosaico dado en la página de la textura. La función devuelve un array con los siguientes 8 elementos:

  • [0] = izquierda
  • [1] = arriba
  • [2] = derecho
  • [3] = fondo
  • 4] = cantidad de píxeles que el compilador de activos ha recortado del lado izquierdo del conjunto de azulejos
  • 5] = cantidad de píxeles que el compilador de activos ha recortado de la parte superior del conjunto de azulejos
  • 6] = porcentaje normalizado de datos de píxeles de la anchura del conjunto de azulejos original que se ha guardado en la página de la textura
  • 7] = porcentaje normalizado de datos de píxeles de la altura del conjunto de azulejos original que se ha guardado en la página de la textura

Esta matriz puede ser utilizada en otras funciones de dibujo, particularmente cuando se trabaja en 3D o se utilizan las funciones primitivas 2D, así como cuando se trabaja con las funciones Shader funciones.

Sintaxis:

tileset_get_uvs(tileset);

Argumento Descripción
conjunto de azulejos El índice del conjunto de azulejos a utilizar.

Devuelve

Matriz (1D, 8 elementos)

Ejemplo:

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

El código anterior almacenará las coordenadas UV para el conjunto de azulejos dado en una matriz local y luego asignará los valores a las variables de instancia.
