tileset_get_name

Esta función devolverá el nombre en forma de cadena del conjunto de baldosas especificado. Este nombre es el que se ha especificado para el conjunto de baldosas en el navegador de activos de la ventana principal de GameMaker Studio 2. Tenga en cuenta que esto es sólo una cadena y no se puede utilizar para hacer referencia al conjunto de baldosas directamente - para eso necesitarías el índice del conjunto de baldosas. Sin embargo, puede utilizar esta cadena para obtener el índice del conjunto de baldosas utilizando la cadena devuelta junto con la función asset_get_index().

Sintaxis:

tileset_get_name(index);

Argumento Descripción
índice El índice del conjunto de baldosas para obtener el nombre.

Devuelve

Cadena

Ejemplo:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_tileset(_m);
tileset_name = tileset_get_name(_t);

El código anterior obtendrá el nombre del índice del conjunto de azulejos para la capa dada, almacenando la cadena de retorno en la variable tileset_name.
