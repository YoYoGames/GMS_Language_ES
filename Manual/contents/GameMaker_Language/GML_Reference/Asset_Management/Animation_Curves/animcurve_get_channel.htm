animcurve_get_channel

Esta función devolverá la estructura que contiene los datos del canal especificado en un activo de curva de animación. Usted suministra el ID o la estructura de la curva de animación, así como el nombre o el índice del canal, y la función devolverá una estructura con el siguiente formato:

Nombre de la variable Tipo de datos Descripción
nombre cadena Este es el nombre del canal.
tipo constante Será una de las constantes animcurvetype_linear para la interpolación lineal entre puntos, o animcurvetype_catmullrom para la interpolación "suave" entre los puntos utilizando la interpolación catmull-rom.
iteraciones entero Si el canal está utilizando interpolación catmull-rom ("suave"), este valor contiene cuántos puntos se han generado para cada segmento de la curva (tenga en cuenta que estos puntos extra son internos a la función y sólo se utilizan para los cálculos en tiempo de ejecución). Si el canal utiliza la interpolación lineal, este valor seguirá existiendo pero puede ser ignorado ya que no tiene relación con la forma en que se interpola la curva.
puntos puntero de matriz Se trata de un array, donde cada elemento del array es una estructura de puntos.


El ID de la curva de animación es el nombre de la curva de animación tal y como se definió en el Navegador de Activos, o el puntero de la estructura devuelto por la función animcurve_create(). El nombre del canal es una cadena que El nombre del canal es una cadena que se refiere al canal tal y como se definió en el activo de la curva de animación, o puede proporcionar un valor de índice, que va de 0 a n, donde n es el último canal en el activo de la curva (por ejemplo: si una curva de animación tiene 4 canales, se indexarán de 0 a 3). Tenga en cuenta que pasar un valor de índice requerirá menos procesamiento que pasar un nombre de canal. Si la función falla (es decir, si no existe ningún canal con el nombre o el índice dado) entonces obtendrá un error.

Los puntos de un mismo canal se almacenan como structs en un array, donde cada elemento del array es un struct de punto único. La estructura de puntos tiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
posx real La posición en el tiempo (normalizada de 0 a 1) del punto.
valor real El valor del punto.

Sintaxis:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

Argumento Descripción
estructura_curva_o_id El ID o puntero de estructura de la curva de animación a la que se dirige
nombre_canal_o_índice El nombre del canal (una cadena) o el índice del canal (un entero).

Devuelve:

Estructurar

Ejemplo:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
_points[i].value += 1;
}

El código anterior recupera la estructura del canal 0 en el activo de la curva "ac_ButtonTween", luego hace un bucle a través de los puntos de la curva del canal y añade uno a su valor.
