animcurve_get_channel_index

Esta función devolverá el valor del índice para cualquier canal de la curva de animación. Usted suministra el ID de la curva de animación o la estructura, donde el ID de la curva es el nombre de la curva de animación tal como fue definida en el Navegador de Activos, o el puntero de la estructura tal como fue devuelto por la función animcurve_create(). A continuación, se suministra el nombre del canal, como una cadena, y la función devolverá el valor del índice asociado a ese canal. Tenga en cuenta que si la curva o el canal no existe, obtendrá un error.

Sintaxis:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

Argumento Descripción
estructura_curva_o_id El ID o puntero de estructura de la curva de animación a la que se dirige
nombre_del_canal El nombre del canal (una cadena).

Devuelve:

Estructurar

Ejemplo:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
_points[i].value += 1;
}

El código anterior recupera la estructura del canal llamado "x_pos" en el activo de la curva "ac_ButtonTween", luego hace un bucle a través de los puntos de la curva del canal y añade uno a su valor.
