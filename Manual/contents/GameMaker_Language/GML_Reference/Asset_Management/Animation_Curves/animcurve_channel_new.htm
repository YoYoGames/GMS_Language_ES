animcurve_channel_new

Esta función se puede utilizar para crear una nueva estructura de canal de curva de animación. Una estructura de canal tiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
nombre cadena Este es el nombre del canal.
tipo constante Esta será una de las constantes animcurvetype_linear - para la interpolación lineal entre puntos - o animcurvetype_catmullrom - para la interpolación "suave" entre los puntos utilizando la interpolación catmull-rom.
iteraciones entero Si el canal está utilizando interpolación catmull-rom ("suave"), este valor contiene cuántos puntos se han generado para cada segmento de la curva (tenga en cuenta que estos puntos adicionales son internos a la función y sólo se utilizan para los cálculos en tiempo de ejecución). Si el canal utiliza la interpolación lineal, este valor seguirá existiendo pero puede ser ignorado ya que no tiene relación con la forma en que se interpola la curva.
puntos puntero de matriz Se trata de un array, donde cada elemento del array es una estructura de puntos.


Por defecto cuando se crea una nueva estructura de canal, la variable "name" será una cadena vacía, el "type" será animcurvetype_linear y las "iteraciones" serán 16. Todas estas variables pueden ajustarse a los valores valores que usted requiera (tenga en cuenta que el valor de "iteraciones" no tiene efecto en los tipos de curvas lineales). Una vez creada la curva, hay que añadir puntos a la misma, lo que se hace añadiendo diferentes estructuras de puntos a la matriz "points". Estos structs de puntos se pueden crear con la función animcurve_point_new()y contienen las siguientes variables:

Nombre de la variable Tipo de datos Descripción
posx real La posición en el tiempo (normalizada de 0 a 1) del punto.
valor real El valor del punto.

Sintaxis:

animcurve_channel_new();

Devuelve:

Referencia de la estructura

Ejemplo:

mi_curva = animcurve_create();
mi_curva.name = "Mi_Curva";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iteraciones = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_puntos[2] = animcurve_point_new();
_puntos[2].posx = 1;
_puntos[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

El código anterior crea una nueva estructura de curva de animación y la almacena en la variable "mi_curva". Esta estructura se rellena con un nombre y una matriz de canales. La matriz de canales contiene un único canal con tres puntos.
