animcurve_get

Esta función devolverá una estructura que contiene todos los datos de la curva de animación dada. Usted proporciona el ID del activo de la curva de animación (como se define en el Navegador de Activos), y la función devolverá una estructura con las siguientes variables:

Nombre de la variable Tipo de datos Descripción
nombre cadena Este es el nombre de la curva de animación.
canales puntero de matriz Se trata de un array, donde cada elemento del array es una estructura de canal.


La variable de canales en la estructura es un array donde cada elemento del array es un struct con datos relativos a un canal dentro de la curva. La estructura de canales tendrá las siguientes variables:

Nombre de la variable Tipo de datos Descripción
nombre cadena Este es el nombre del canal.
tipo constante Será una de las constantes animcurvetype_linear para la interpolación lineal entre puntos, o animcurvetype_catmullrom para la interpolación "suave" entre los puntos utilizando la interpolación catmull-rom.
iteraciones entero Si el canal está utilizando interpolación catmull-rom ("suave"), este valor contiene cuántos puntos se han generado para cada segmento de la curva (tenga en cuenta que estos puntos extra son internos a la función y sólo se utilizan para los cálculos en tiempo de ejecución). Si el canal utiliza la interpolación lineal, este valor seguirá existiendo pero puede ser ignorado ya que no tiene relación con la forma en que se interpola la curva.
puntos puntero de matriz Se trata de un array, donde cada elemento del array es una estructura de puntos.


Al igual que en el caso de los canales, los puntos de un mismo canal se almacenan como structs en un array, donde cada elemento del array es un struct de puntos. La estructura de puntos tiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
posx real La posición en el tiempo (normalizada de 0 a 1) del punto.
valor real El valor del punto.


Tenga en cuenta que si la función falla (es decir, el ID de la Curva de Animación dado no existe) entonces la función devolverá -1.

Sintaxis:

animcurve_get(curve_id);

Argumento Descripción
curve_id El ID del navegador de activos (índice) de la curva de animación a obtener.

Devuelve:

Estructura o -1

Ejemplo:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if _channel.type != animcurvetype_linear
{
_channel.type = animcurvetype_linear;
}

El código anterior recupera la estructura de la curva de animación para el activo de la curva "ac_ButtonTween", luego comprueba el tipo de curva para el primer canal, y si no es lineal lo establece como lineal.
