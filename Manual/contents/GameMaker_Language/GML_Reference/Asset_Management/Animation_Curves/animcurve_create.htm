animcurve_create

Esta función creará una estructura de curva de animación vacía, lista para que usted la rellene con los datos de los canales. La función devolverá un puntero a la estructura y ésta se utilizará para añadir canales y otros datos a la curva de animación. También se puede suministrar a funciones como animcurve_get() para obtener posteriormente los datos de la curva.

Cuando se utiliza esta función, la estructura creada no tendrá datos en ella, y para utilizarla se debe añadir al menos un canal y el canal debe tener puntos añadidos. Para añadir un canal, vea la función animcurve_channel_new(), y para añadir puntos al canal, vea la función animcurve_point_new(). Además, puede dar un nombre a la estructura de la curva de animación estableciendo la variable "name" (como se muestra en el código de ejemplo, más abajo). Los canales añadidos a una nueva curva de animación deben ser añadidos como una matriz, donde cada elemento de la matriz hace referencia a una estructura de canal, y los puntos de un canal también deben ser añadidos como una matriz, donde cada elemento de la matriz hace referencia a una estructura de puntos.

Tenga en cuenta que las curvas de animación creadas de esta manera deben ser eliminadas cuando ya no se necesiten, ya que de lo contrario ocuparán espacio en la memoria, lo que puede conducir a una fuga de memoria, ralentizando y eventualmente estrellando su juego. Puedes eliminar una curva de animación cuando ya no la necesites utilizando la función animcurve_destroy(). No es necesario limpiar ningún canal o punto de datos dentro de la curva, ya que esto se hará ya que el recolector de basura lo hará automáticamente cuando la curva sea destruida.

Sintaxis:

animcurve_create();

Devuelve:

Referencia de la estructura

Ejemplo:

mi_curva = animcurve_create();
mi_curva.name = "Mi_Curva";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iteraciones = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_puntos[2] = animcurve_point_new();
_puntos[2].posx = 1;
_puntos[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

El código anterior crea una nueva estructura de curva de animación y la almacena en la variable "mi_curva". Esta estructura se rellena con un nombre y una matriz de canales. La matriz de canales contiene un único canal con tres puntos.
