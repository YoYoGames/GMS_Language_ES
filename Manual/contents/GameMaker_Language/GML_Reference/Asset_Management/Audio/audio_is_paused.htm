audio_is_paused

Esta función comprobará el sonido dado para ver si está actualmente en pausa. El sonido puede ser una instancia única de un sonido (el índice de los sonidos individuales que se reproducen puede almacenarse en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() ) o un activo de sonido, en cuyo caso se comprobarán todas las instancias del sonido dado y, si alguna de ellas está en pausa, la función devolverá true; en caso contrario devolverá false.

Sintaxis:

audio_is_paused(index);

Argumento Descripción
índice El índice del sonido a comprobar.

Devuelve:

Booleano

Ejemplo:

if audio_is_paused(snd_Waterfall)
{
audio_resume_sound(snd_Waterfall);
}

El código anterior comprueba si el sonido indexado en la variable "snd_Waterfall" está actualmente en pausa y si devuelve true se reanudará la reproducción del sonido.
