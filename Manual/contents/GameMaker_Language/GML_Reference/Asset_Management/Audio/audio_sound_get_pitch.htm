audio_sound_get_pitch

Esta función puede utilizarse para obtener el tono de un sonido dado. El sonido puede ser uno referenciado desde un índice para un sonido individual que se está reproduciendo y que ha sido almacenado en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() o un activo de sonido real del Navegador de Activos.

Sintaxis:

audio_sound_get_pitch(index);

Argumento Descripción
índice El índice del sonido para obtener el tono.

Devuelve:

Real

Ejemplo:

if audio_sound_get_pitch(snd_Explode) != 1
{
audio_sound_pitch(snd_Explode, 1);
}

El código anterior cambiará el tono del audio reproducido del sonido indexado como "snd_Explode" si su tono no es igual a 1.
