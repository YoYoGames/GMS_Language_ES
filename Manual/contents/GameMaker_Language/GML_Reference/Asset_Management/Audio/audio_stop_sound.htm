audio_stop_sound

Esta función detendrá el sonido dado si se está reproduciendo actualmente. El sonido puede ser una sola instancia de un sonido (el índice de los sonidos individuales que se están reproduciendo puede almacenarse en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() ) o un activo de sonido, en cuyo caso se detendrán todas las instancias del sonido dado.

Sintaxis:

audio_stop_sound(index);

Argumento Descripción
índice El índice del sonido a detener.

Devuelve:

N/A

Ejemplo:

if !global.SFX
{
audio_stop_sound(snd_Waterfall);
}
else
{
audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

El código anterior comprueba la variable global "SFX" y si devuelve false , detendrá el sonido indexado en la variable "snd_Waterfall" que se está reproduciendo actualmente, y si devuelve true, el sonido se reproducirá en bucle.
