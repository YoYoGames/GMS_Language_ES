audio_sound_set_listener_mask

Esta función puede utilizarse para establecer la máscara de bits de un sonido, de forma que sólo se reproduzca desde los oyentes especificados. Se puede crear una máscara de bits utilizando los parámetros audio_get_listener_info() y y luego usando el bitwise o ("|") para crear una máscara personalizada con aquellos oyentes de los que se requiere que el sonido se reproduzca, y luego aplicar esta máscara personalizada al sonido después de haberlo reproducido. Esta máscara anulará la máscara por defecto o la que haya establecido mediante la función audio_set_listener_mask().

NOTA: Esta función sólo se puede utilizar en un sonido después de que se haya configurado para reproducirlo y requiere el soundID devuelto por la función utilizada para reproducir el sonido.

Sintaxis:

audio_sound_set_listener_mask(soundID, mask);

Argumento Descripción
soundID El ID único del sonido para establecer la máscara de
máscara Los datos de la máscara de bits que hay que establecer para el sonido

Devuelve:

N/A

Ejemplo:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

El código anterior reproduce un sonido y luego establece la(s) escucha(s) que debe(n) reproducir utilizando los datos de la máscara almacenada en una variable global.
