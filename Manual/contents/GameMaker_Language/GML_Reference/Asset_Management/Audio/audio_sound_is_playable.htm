audio_sound_is_playable

Esta función se puede utilizar para comprobar si el índice de sonido dado se puede reproducir actualmente. Esto es necesario debido a las diferentes formas en que se maneja la reproducción de sonido en flujo y sin flujo en la plataforma de destino HTML5, y devolverá devolverá true si el sonido puede ser reproducido y false si no puede. Tenga en cuenta que en todas las demás plataformas que no sean HTML5, la función siempre devolverá true..

Sintaxis:

audio_sound_is_playable(index);

Argumento Descripción
índice El índice del sonido a comprobar.

Devuelve:

Real

Ejemplo:

if audio_sound_get_pitch(snd_Explode) != 1
{
audio_sound_pitch(snd_Explode, 1);
}

El código anterior cambiará el tono del audio reproducido del sonido indexado como "snd_Explode" si su tono no es igual a 1.
