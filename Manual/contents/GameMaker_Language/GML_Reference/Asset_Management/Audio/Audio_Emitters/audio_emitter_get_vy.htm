audio_emitter_get_vy

Esta función devuelve la velocidad actual a lo largo del eje y para el emisor de audio dado.

Sintaxis:

audio_emitter_get_vy(emisor);

Argumento Descripción
emisor El índice del emisor a utilizar.

Devuelve:

Real

Ejemplo:

if audio_emitter_get_vy(emitter_player) != 0
{
audio_emitter_velocity(emitter_player, 0, 0, 0);
}

El código anterior comprueba la velocidad actual en y de un emisor determinado y si no es igual a 0, se pone a 0.
