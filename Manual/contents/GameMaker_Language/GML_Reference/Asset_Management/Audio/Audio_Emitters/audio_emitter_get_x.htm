audio_emitter_get_x

Esta función devuelve la posición x actual del emisor de audio dado.

Sintaxis:

audio_emitter_get_x(emisor);

Argumento Descripción
emisor El índice del emisor a utilizar.

Devuelve:

Real

Ejemplo:

if audio_emitter_get_x(emitter_player) != x
{
audio_emitter_position(emitter_player, x, y, 0);
}

El código anterior comprueba la posición x actual de un emisor dado y si no es igual a la posición x de la instancia, se establece en la posición de la instancia.
