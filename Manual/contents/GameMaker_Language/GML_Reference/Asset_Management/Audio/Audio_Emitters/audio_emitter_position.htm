posición_emisora_de_audio

Con esta función puede cambiar la posición de un emisor de audio dentro del espacio de audio 3D. La posición afectará al sonido de diferentes maneras dependiendo de dónde se sitúe el oyente dentro del espacio de audio (la posición por defecto es (0, 0, 0). Consulte audio_listener_position() para más información), así que, por ejemplo, si la posición del emisor se establece en (100, 0, 0) y el oyente actual está en (200, 0, 0) el audio transmitido desde el emisor aparecerá a la izquierda del campo de audio. La imagen siguiente muestra una representación visual de los emisores y sus posiciones relativas respecto al oyente:

Sintaxis:

audio_emitter_position(emisor, x, y, z);

Argumento Descripción
emisor El índice del emisor a cambiar.
x La posición x.
y La posición y.
z La posición z.

Devuelve:

N/A

Ejemplo:

si velocidad > 0
{
audio_emitter_position(s_emit, x, y, 0);
}

El código anterior comprueba si la velocidad de la instancia es superior a 0 y si lo es actualiza el emisor de audio indexado en la variable "s_emit" a la posición x/y actual.
