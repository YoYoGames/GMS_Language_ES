audio_emitter_get_vx

Esta función devuelve la velocidad actual a lo largo del eje x para el emisor de audio dado.

Sintaxis:

audio_emitter_get_vx(emisor);

Argumento Descripción
emisor El índice del emisor a utilizar.

Devuelve:

Real

Ejemplo:

if audio_emitter_get_vx(emitter_player) != 0
{
audio_emitter_velocity(emitter_player, 0, 0, 0);
}

El código anterior comprueba la velocidad x actual de un emisor determinado y si no es igual a 0, se pone a 0.
