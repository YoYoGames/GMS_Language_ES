audio_emitter_set_listener_mask

Esta función puede utilizarse para establecer la máscara de bits de un emisor, de modo que todos los sonidos reproducidos a través del emisor se reproduzcan sólo desde los oyentes especificados. Se puede crear una máscara de bits utilizando los parámetros audio_get_listener_info() y y luego utilizando el bitwise o ("|") para crear una máscara personalizada con los oyentes desde los que quieres que se reproduzca el sonido, y luego aplicar esta máscara personalizada al emisor. Esta máscara anulará la máscara por defecto por defecto o la que haya establecido con la función audio_set_listener_mask().

Sintaxis:

audio_emitter_set_listener_mask(emitterID, mask);

Argumento Descripción
emitterID El ID único del emisor para establecer la máscara de
máscara Los datos de la máscara de bits que hay que establecer para el sonido

Devuelve:

N/A

Ejemplo:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

El código anterior crea un emisor de audio y luego establece la(s) escucha(s) para reproducir usando los datos de la máscara almacenados en una variable global.
