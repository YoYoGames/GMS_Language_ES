audio_emisor_existe

Esta función devuelve si un emisor de audio existe(true) o no(false). Tenga en cuenta que si el índice que busca no ha sido inicializado previamente, esta función causará un error ya que está buscando índices inexistentes.

Sintaxis:

audio_emitter_exists(index);

Argumento Descripción
índice El índice del emisor para comprobar la existencia.

Devuelve:

Booleano

Ejemplo:

if audio_emitter_exists(s_emit)
{
audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
s_emit = audio_emitter_create();
audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

El código anterior comprueba si existe un emisor, indexado en la variable (previamente inicializada) "s_emit". Si existe, entonces se reproduce un sonido a través de él, pero si no existe, se crea y entonces se reproduce el sonido.
