audio_emitter_create

Esta función crea un nuevo emisor de audio y devuelve su índice. Este índice debe almacenarse en una variable para todas las funciones posteriores que se relacionen con este emisor, y luego, cuando ya no se necesite, debe eliminarse de la memoria mediante la función función audio_emitter_free() para evitar fugas de memoria que puedan llegar a colapsar el juego.

Sintaxis:

audio_emitter_create();

Devuelve:

Entero (ID del emisor)

Ejemplo:

s_emit = audio_emitter_create();

El código anterior crea un nuevo emisor de audio y asigna su índice a la variable "s_emit".
