ganancia_emisora_de_audio

Esta función establece la ganancia máxima (volumen) del sonido. El volumen percibido para un sonido puede cambiar dependiendo del valor de caída y de la posición que tenga con respecto al oyente, pero al ajustar la ganancia con esta función, el volumen total nunca superará el valor de ganancia especificado. La imagen siguiente ilustra cómo la ganancia afecta al volumen del emisor cuando el fall-off es mayor que 0:

Esta función cambiará el volumen del sonido mientras se está reproduciendo, así como todos los sonidos posteriores reproducidos a través del emisor dado. Tenga en cuenta que en algunas plataformas puede tener una ganancia mayor que 1, aunque un valor de 1 se considera "volumen completo" y cualquier cosa mayor puede introducir recortes de audio o distorsión.

NOTA: el volumen final también se verá influenciado por la ganancia de audio global que se haya establecido mediante la función audio_master_gain().

Sintaxis:

audio_emitter_gain(emisor, ganancia);

Argumento Descripción
emisor El índice del emisor a cambiar.
ganar La ganancia máxima (por defecto 1).

Devuelve:

N/A

Ejemplo:

si up
{
ganancia += 0.05;
si ganancia >= 1 up = false;
}
si no
{
gain += 0.05;
if gain <= 0 up = true;
}
audio_emitter_gain(s_emit, gain);

El código anterior establece la variable "gain" a diferentes valores y luego utiliza esa misma variable para establecer la ganancia del emisor indexado en la variable "s_emit".
