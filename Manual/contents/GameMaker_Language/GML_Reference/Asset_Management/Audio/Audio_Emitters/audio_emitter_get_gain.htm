audio_emitter_get_gain

Esta función devuelve la ganancia actual (volumen) establecida para el emisor de audio dado, normalmente entre 0 y 1, donde 0 es silencio y 1 es volumen completo. Tenga en cuenta que en algunas plataformas puede tener una ganancia superior a 1, aunque un valor de 1 se considera "volumen completo" y cualquier valor mayor puede introducir recortes de audio.

Sintaxis:

audio_emitter_get_gain(emisor);

Argumento Descripción
emisor El índice del emisor a utilizar.

Devuelve:

Real

Ejemplo:

si audio_emitter_get_gain(emitter_player) < 1
{
audio_emitter_gain(emitter_player, 1);
}

El código anterior comprueba la ganancia actual de un determinado emisor y si es menor que 1 se pone a 1.
