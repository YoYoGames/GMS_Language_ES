audio_sound_set_track_position

Esta función establecerá la posición (en segundos) dentro del archivo de sonido para que el sonido se reproduzca. El sonido puede ser una instancia única de un sonido (el índice de los sonidos individuales que se reproducen puede almacenarse en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() ) o un activo de sonido. Si se trata de un ID de sonido único que se utiliza, entonces la posición cambiará inmediatamente al nuevo tiempo dado, y si es un activo del Navegador de Activos entonces todas las reproducciones posteriores del sonido dado comenzarán en el nuevo tiempo. La función ignorará cualquier valor dado que sea más largo que la longitud del sonido dado para el valor "index" del sonido.

Sintaxis:

audio_sound_set_track_position(index, time);

Argumento Descripción
índice El índice del sonido a cambiar.
tiempo El tiempo (en segundos) al que se debe fijar el punto de inicio.

Devuelve:

N/A

Ejemplo:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

El código anterior reproduce un sonido y luego utiliza el valor de ID devuelto para establecer la posición de inicio del sonido a 32 segundos.
