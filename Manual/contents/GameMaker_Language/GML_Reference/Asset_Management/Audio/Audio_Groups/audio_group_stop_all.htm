audio_group_stop_all

Esta función detendrá todos los sonidos del grupo de audio dado que se están reproduciendo actualmente.

Sintaxis:

audio_group_stop_all(groupID);

Argumento Descripción
groupID El índice del grupo de audio que se va a detener (como se define en la ventana de grupos de audio)

Devuelve:

N/A

Ejemplo:

si keyboard_check_pressed(vk_space)
{
audio_group_stop_all(audiogroup_level1);
}

El código anterior comprueba la tecla "espacio" y entonces detiene toda la reproducción de audio del grupo dado.
