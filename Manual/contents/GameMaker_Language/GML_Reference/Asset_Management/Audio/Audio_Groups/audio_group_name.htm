nombre_del_grupo_de_audio

Esta función devolverá una cadena que contiene el nombre del grupo de audio dado para su visualización o comprobación. Cuando se define un grupo de audio en las Opciones de Juego, se le da un "nombre" único que es realmente una constante para usar como valor de identificación para el grupo. Todo lo que hace esta función es tomar el ID y devolver una cadena con el nombre del ID que le diste.

Sintaxis:

audio_group_name(groupID);

Argumento Descripción
groupID La constante del valor del índice del grupo de audio que se va a comprobar (como se define en la ventana de grupos de audio)

Devuelve:

Cadena

Ejemplo:

var name= nombre_del_grupo_de_audio(nivel_del_grupo_de_audio1);
draw_text(32, 32, "Grupo en reproducción: " + nombre);

El código anterior recupera el nombre de la constante del grupo de audio dado y lo muestra en la pantalla.
