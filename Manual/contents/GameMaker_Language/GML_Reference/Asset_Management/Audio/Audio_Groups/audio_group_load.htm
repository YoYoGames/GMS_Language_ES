audio_group_load

Esta función cargará en la memoria todos los sonidos marcados como pertenecientes al Grupo de Audio dado. La función devolverá true si se inicia la carga y false si el ID del grupo es inválido, o ya ha sido marcado para la carga. La función es asíncrona, por lo que su juego continuará ejecutándose mientras se carga el audio en segundo plano. Esto significa que también activará un evento asíncrono de carga/guardado para para informarle de que todo el grupo se ha cargado en la memoria y que los sonidos ya pueden ser utilizados.

Sintaxis:

audio_group_load(groupID);

Argumento Descripción
groupID El índice del grupo de audio que se va a cargar (como se define en la ventana de grupos de audio)

Devuelve:

Booleano

Ejemplo:

if !audio_group_is_loaded(audiogroup_level1)
{
audio_group_load(audiogroup_level1);
}

El código anterior comprueba si se ha cargado un grupo de audio y, si no, lo carga.
