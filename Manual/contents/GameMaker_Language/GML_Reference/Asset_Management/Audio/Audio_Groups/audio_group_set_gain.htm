audio_group_set_gain

Con esta función se puede hacer un fundido de entrada o salida de un grupo de sonidos durante un tiempo determinado, o se puede utilizar para ajustar la ganancia del grupo de forma instantánea. El tiempo se mide en milisegundos, y la función requiere que introduzca un nivel final de ganancia para que el para que el grupo haya alcanzado al final de ese tiempo. Esta ganancia puede estar entre 0 (silencio) y 1 (volumen total) y la escala es lineal, de forma que un valor de 0,5 sería la mitad del volumen. Para cambiar instantáneamente la ganancia, basta con poner el argumento del tiempo a 0. Tenga en cuenta que en algunas plataformas se puede tener una ganancia superior a 1, aunque un valor de 1 se considera "volumen completo" y cualquier valor superior puede introducir recortes de audio.

Sintaxis:

audio_group_set_gain(groupID, volume, time);

Argumento Descripción
groupID El índice del grupo de audio que se va a detener (como se define en la ventana de grupos de audio)
volumen El valor final del volumen del grupo.
tiempo La duración del cambio de ganancia en milisegundos.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(vk_space)
{
audio_group_set_gain(audiogroup1, 0, 5000);
}

El código anterior comprueba la tecla "espacio" y luego desvanece todo el audio para "audiogroup1" a 0 durante 5 segundos.
