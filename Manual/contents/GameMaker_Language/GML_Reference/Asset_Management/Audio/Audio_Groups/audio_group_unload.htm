audio_group_unload

Esta función descargará en la memoria todos los sonidos marcados como pertenecientes al Grupo de Audio dado. La función devolverá true si se inicia la descarga y false si el ID del grupo no es válido, o ya ha sido marcado para su descarga. Tenga en cuenta que cualquier audio que se esté reproduciendo cuando se llame a esta función se detendrá.

Sintaxis:

audio_group_unload(groupID);

Argumento Descripción
groupID El índice del grupo de audio que se va a descargar (como se define en la ventana de grupos de audio)

Devuelve:

Booleano

Ejemplo:

if audio_group_is_loaded(audiogroup_level1)
{
audio_group_unload(audiogroup_level1);
}

El código anterior comprueba si se ha cargado un grupo de audio y si lo ha hecho lo descarga.
