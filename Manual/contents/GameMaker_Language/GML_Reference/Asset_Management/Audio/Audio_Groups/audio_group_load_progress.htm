audio_group_load_progress

Esta función comprobará el progreso de carga de un grupo de audio y devolverá un valor (aproximado) entre 0 y 100.

Sintaxis:

audio_group_load_progress(groupID);

Argumento Descripción
groupID El índice del grupo de audio que se va a comprobar (como se define en la ventana de grupos de audio)

Devuelve:

Real

Ejemplo:

si carga
{
var pc = audio_group_load_progress(audiogroup_level1);
draw_text(32, 32, "Loading: " + string(pc));
}

El código anterior comprueba una variable y si devuelve true entonces se dibujará un texto mostrando el progreso de la carga del audio.
