audio_listener_set_orientation

Con esta función puede cambiar la orientación del oyente dado dentro del espacio de audio 3D. El índice del oyente por defecto es 0, pero puede utilizar la función audio_get_listener_info() para obtener los diferentes índices disponibles para la plataforma de destino.

El vector de observación y el vector ascendente se basan en los valores que se resuelven a partir de las posiciones relativas x, y y z dadas, y por defecto son (0, 0, -1) para el vector de observación y (0, 1, 0) para el vector superior, como se muestra en la siguiente ilustración:

Cambiar la orientación del oyente con esta función cambiará la forma en que el sonido creado por los emisores de audio alrededor de la sala de juego es percibido por el jugador de su juego. En el ejemplo siguiente, los sonidos creados por el emisor cuando el oyente está en la posición posición por defecto parecerían venir de abajo y a la derecha del oyente, pero con la nueva posición y orientación del oyente ahora se percibirán como procedentes de arriba y a la derecha.

Sintaxis:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Argumento Descripción
índice El oyente para establecer la orientación de.
lookat_x El vector de apariencia x (por defecto 0).
mirar_y El vector de apariencia y (por defecto 0).
lookat_z El vector de apariencia z (por defecto -1).
up_x El vector x up (por defecto 0).
up_y El vector y up (por defecto 1).
up_z El vector z hacia arriba (por defecto 0).

Devuelve:

N/A

Ejemplo:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

El código anterior recupera la matriz de vista para la vista de cámara [0] y luego la utiliza para establecer la posición y la orientación del oyente de audio con el ID almacenado en la variable global "Player_Listener".
