audio_listener_position

Con esta función puede cambiar la posición del oyente dentro del espacio de audio 3D. La imagen de ejemplo siguiente muestra la posición por defecto del oyente en el espacio de audio:

NOTA: si tiene varios oyentes debe utilizar la función audio_listener_set_position().

Como puedes ver, la posición por defecto es (0, 0, 0) pero normalmente usarás esta función para mover el listener con el objeto jugador dentro de tu juego y así cambiar la forma en que el audio creado por los emisores es escuchado por el jugador, por ejemplo, en la imagen imagen de abajo de un juego de arriba hacia abajo, la instancia del jugador establece el oyente que hará que el audio de los diversos emisores de "cambiar" como el jugador se mueve alrededor del nivel:

Sintaxis:

audio_listener_position(x, y, z);

Argumento Descripción
x La posición x del oyente (por defecto 0).
y La posición y del oyente (por defecto 0).
z La posición z del oyente (por defecto 0).

Devuelve:

N/A

Ejemplo:

si velocidad > 0
{
audio_listener_position(x, y, 0);
}

El código anterior comprueba si la velocidad de la instancia del reproductor es superior a 0 y, si lo es, actualiza el oyente de audio a la posición x/y actual.
