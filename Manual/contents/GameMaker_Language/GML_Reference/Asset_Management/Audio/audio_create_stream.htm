audio_create_stream

Con esta función se puede crear un nuevo índice de sonido que luego se puede utilizar en las funciones regulares de audio para transmitir el audio directamente desde una fuente externa de archivos OGG. La función requiere el nombre del archivo (que puede ser un archivo incluido, por ejemplo) y devolverá el nuevo índice de sonido para su uso. Tenga en cuenta que cuando ya no necesite el sonido deberá llamar a la función audio_destroy_stream() con el índice de sonido para eliminarlo de de la memoria, de lo contrario puede obtener una fuga de memoria que ralentizará y eventualmente bloqueará su juego.

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_create_stream(filename);

Argumento Descripción
nombre de archivo El archivo (sólo OGG) para transmitir el audio.

Devuelve:

Índice de sonido

Ejemplo:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

El código anterior crea un nuevo índice de sonido en la variable "snd" a partir del archivo dado, y luego reproduce este sonido.
