audio_get_name

Esta función devolverá el nombre de un recurso de audio dado como una cadena. El valor del "índice" puede ser el del propio recurso (como se ve en el Navegador de Activos) o el valor de ID único que se da cuando se reproduce el sonido usando, por ejemplo, audio_play_sound(). Tenga en cuenta que la cadena devuelta no es la misma que el ID del recurso y no puede utilizarse para acceder al recurso en sí, por lo que sólo debe utilizarse para la visualización o la comprobación de errores.

Sintaxis:

audio_get_name(index);

Argumento Descripción
índice El índice del sonido a comprobar.

Devuelve:

Cadena

Ejemplo:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

El código anterior reproduce un sonido aleatorio elegido entre tres recursos de sonido diferentes y luego muestra un mensaje de depuración con su nombre.
