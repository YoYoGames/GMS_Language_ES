audio_destroy_stream

Si ha creado previamente un flujo de audio desde un archivo utilizando la función audio_create_stream() y ya no necesita ese sonido, puede llamar a esta función para borrarlo de memoria. Cualquier otra llamada al índice de sonido después de que haya sido destruido dará un error. Hay que tener en cuenta que esto liberará el flujo, pero en la plataforma de destino esto puede no aparecer en un gestor de memoria. Esto se debe a que GameMaker Studio 2 agrupa para evitar la sobrecarga de asignación de memoria, por lo que la memoria permanecerá asignada hasta que se requiera para otra cosa o se reutilice para un nuevo flujo.

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_destroy_stream(filename);

Argumento Descripción
nombre de archivo El archivo desde el que se transmite el audio.

Devuelve:

NA

Ejemplo:

audio_destroy_stream(snd);

El código anterior elimina de la memoria el sonido indexado en la variable "snd".
