audio_sync_group_is_playing

Esta función puede utilizarse para comprobar si se está reproduciendo algún audio de un grupo sincronizado. Es necesario proporcionar el ID del grupo de sincronización devuelto por la función audio_create_sync_group().

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_sync_group_is_playing(group_index);

Argumento Descripción
grupo_índice El índice del grupo a comprobar.

Devuelve:

N/A

Ejemplo:

if audio_sync_group_is_playing(group_one)
{
audio_stop_sync_group(grupo_uno);
}

El código anterior comprueba si el grupo de sincronización group_one se está reproduciendo en ese momento y, si es así, el grupo se detiene.
