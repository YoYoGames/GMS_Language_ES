audio_play_in_sync_group

Con esta función puedes asignar un sonido a un grupo de sincronización previamente creado. El índice del grupo es el valor devuelto al crear el grupo mediante la función audio_create_sync_group()y el índice del sonido es el índice de cualquier sonido comprimido(*.ogg o *.mp3) del navegador de activos. Esta función sólo añade el sonido al grupo, pero no reproduce el sonido, y tendrá un valor > -1 si tiene éxito y -1 si falla, donde un valor exitoso indica el orden de los sonidos en el grupo de sincronización (así que 0 sería el primer sonido añadido, 1 sería el segundo, etc...).

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_play_in_sync_group(group_index, sound_index);

Argumento Descripción
grupo_índice El índice del grupo al que se asigna el sonido.
índice_de_sonido El índice de sonido que se asignará al grupo.

Devuelve:

Índice

Ejemplo:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_play_in_sync_group(sg, sound3);
audio_play_in_sync_group(sg, sound4);
audio_start_sync_group(sg);

Lo anterior crea un nuevo grupo de sincronización y asigna el índice del grupo a la variable "sg". A continuación, se añaden cuatro sonidos al grupo, con la ganancia de tres de ellos ajustada a 0. Por último, se reproduce el grupo de sincronización.
