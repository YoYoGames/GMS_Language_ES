audio_resume_sync_group

Esta función reanudará el grupo de sincronización dado si se está reproduciendo y ha sido pausado previamente (utilizando la función audio_pause_sync_group). El índice del grupo es el valor devuelto al crear el grupo utilizando la función audio_create_sync_group().

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_resume_sync_group(group_index);

Argumento Descripción
grupo_índice El índice del grupo para reanudar.

Devuelve:

N/A

Ejemplo:

if (keyboard_check_pressed((ord) "P"))
{
global.Pause = !global.Pause
if global.Pause
{
audio_pause_sync_group(sg);
}
else
{
audio_resume_sync_group(sg);
}
}

El código anterior comprueba si se ha pulsado la tecla "P", y si se detecta una, activa la variable "global.Pause" y la comprueba para pausar o reanudar el grupo de sincronización indexado en la variable "sg".
