audio_sync_group_get_track_pos

Esta función devuelve la posición de reproducción actual del grupo de sincronización dado. El índice del grupo es el valor devuelto al crear el grupo mediante la función audio_create_sync_group()y el valor de retorno es el tiempo en segundos que las pistas han estado reproduciéndose.

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_sync_group_get_track_pos(group_index);

Argumento Descripción
grupo_índice El índice del grupo para obtener la posición de.

Devuelve:

Real

Ejemplo:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
secs = string(secs);
}
else
{
secs = "0" + string(secs);
}
draw_text(32, 32, "Time = " + mins + ":" + secs);

El código anterior obtiene la posición de tiempo para el grupo de sincronización indexado en la variable "sg", y luego utiliza este valor para dibujar el tiempo reproducido en la pantalla.
