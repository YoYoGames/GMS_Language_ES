audio_pause_sync_group

esta función pondrá en pausa el grupo de sincronización dado si está reproduciendo, siendo el índice del grupo el valor devuelto al crear el grupo mediante la función audio_create_sync_group(). Esto no detiene el sonido, y al llamar a audio_resume_sync_group()se iniciará la reproducción desde la misma posición en la que se detuvo.

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_pause_sync_group(group_index);

Argumento Descripción
grupo_índice El índice del grupo para hacer una pausa.

Devuelve:

N/A

Ejemplo:

if (keyboard_check_pressed((ord) "P"))
{
global.Pause = !global.Pause
if global.Pause
{
audio_pause_sync_group(sg);
}
else
{
audio_resume_sync_group(sg);
}
}

El código anterior comprueba si se ha pulsado la tecla "P", y si se detecta una, activa la variable "global.Pause" y la comprueba para pausar o reanudar el grupo de sincronización indexado en la variable "sg".
