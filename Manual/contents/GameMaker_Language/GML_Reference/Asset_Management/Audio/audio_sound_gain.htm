audio_sound_gain

Con esta función se puede hacer un fundido de entrada o salida de un sonido durante un tiempo determinado, o se puede utilizar para ajustar la ganancia del sonido de forma instantánea. El tiempo se mide en milisegundos, y la función requiere que se introduzca un nivel final de ganancia para que el sonido haya que el sonido haya alcanzado al final de ese tiempo. Esta ganancia puede estar entre 0 (silencio) y cualquier valor superior a 0, aunque normalmente se consideraría que el volumen máximo es 1. Se puede utilizar cualquier valor superior a 1 pero, dependiendo del sonido utilizado y de la plataforma a la que se compila puede producirse una distorsión o recorte cuando se reproduce el sonido. Tenga en cuenta que la escala de ganancia es lineal, y para cambiar instantáneamente la ganancia, simplemente ponga el argumento de tiempo a 0.

Esta función afectará a todas las instancias del sonido que se estén reproduciendo actualmente en la sala si el índice es un recurso sonoro, y el volumen final será el volumen al que se reproducirán todas las instancias posteriores del sonido sonido. Sin embargo, si ha utilizado el índice devuelto por una función como audio_play_sound() sólo afectará a esa única instancia del sonido.

Sintaxis:

audio_sound_gain(índice, volumen, tiempo);

Argumento Descripción
índice El índice del sonido para el que se va a ajustar la ganancia.
volumen Valor para el volumen de la música.
tiempo La duración del cambio de ganancia en milisegundos.

Devuelve:

N/A

Ejemplo:

si val
{
var snd = audio_play_sound(snd_fountain);
audio_sound_gain(snd, 0, 0);
audio_sound_gain(snd, 1, 5000);
}

El código anterior comprueba una variable y si devuelve true asigna el índice de un sonido a reproducir a la variable local "snd". Esta variable se utiliza entonces para reducir el volumen de ese sonido sonido a 0 y subir el volumen al máximo durante 5 segundos.
