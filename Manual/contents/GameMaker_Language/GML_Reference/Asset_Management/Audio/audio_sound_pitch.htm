audio_sound_pitch

Esta función puede utilizarse para cambiar el tono de un sonido determinado. El sonido puede ser uno referenciado a partir de un índice para un sonido individual que se está reproduciendo y que ha sido almacenado en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() o un activo de sonido real del Navegador de Activos. Si se trata de un índice de un sonido que se está reproduciendo, entonces sólo se cambiará esa instancia, sin embargo, cuando se utiliza un activo de sonido del Navegador de Activos, todas las las demás instancias que se reproduzcan de ese sonido se modificarán.

El argumento de tono es un multiplicador de tono, en el sentido de que el valor de entrada multiplica el tono actual por esa cantidad, por lo que el valor por defecto de 1 no supone ningún cambio de tono, mientras que un valor menor que 1 bajará el tono y mayor que 1 mayor que 1. Es mejor utilizar incrementos pequeños para esta función, ya que cualquier valor inferior a 0 o superior a 5 puede no ser audible de todos modos. Hay que tener en cuenta que el cambio total de tono permitido está fijado en (1/256) - 256 octavas, por lo que cualquier valor por encima o por debajo de no se registrará.

NOTA: El valor indicado arriba es lo que GameMaker Studio 2 intenta fijar el rango, pero este valor no está garantizado en todas las plataformas de destino , se ajusta a (1/256) - 8, por lo que es posible que tenga que experimentar en cada plataforma de destino y tener diferentes versiones de un recurso de sonido, cada uno pre-desplazado, en caso de necesitar valores de octava más altos o más bajos.

Sintaxis:

audio_sound_pitch(index, pitch);

Argumento Descripción
índice El índice del sonido a cambiar.
paso El multiplicador de paso (por defecto 1).

Devuelve:

N/A

Ejemplo:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
caso 1: audio_sound_pitch(s_engine, 0.8); break;
caso 2: audio_sound_pitch(s_engine, 0.9); break;
caso 3: audio_sound_pitch(s_engine, 0.95); break;
caso 4: audio_sound_pitch(s_engine, 1); break;
caso 5: audio_sound_pitch(s_engine, 1.2); break;
}

El código anterior cambiará el tono del audio reproducido del sonido indexado en la variable "s_engine" en función del valor de la variable "gear".
