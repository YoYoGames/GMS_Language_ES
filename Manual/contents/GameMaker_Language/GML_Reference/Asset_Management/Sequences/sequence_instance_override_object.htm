secuencia_instancia_sobrecarga_objeto

Con esta función puede anular (reemplazar) todas las instancias de un objeto utilizado en una secuencia por otro. Debe proporcionar el ID de la estructura de la instancia de la secuencia (tal como se devolvió cuando se creó la instancia de la secuencia en la sala o utilizando una de las funciones de la capa de la sala véase aquí), así como el índice del objeto (tal y como se define en el navegador de activos) para el objeto que desea sustituir. Por último, se da un índice de objeto o un ID de instancia para utilizar como el objeto que va a anular la secuencia (si se proporciona un ID de instancia, se utilizará simplemente el objeto del que se creó la instancia como anulación). Tenga en cuenta que esto sólo puede hacerse en las instancias de la secuencia (no en los objetos de la secuencia) y debe hacerse antes de que la secuencia comience a reproducirse, de lo contrario no funcionará.

Sintaxis:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Argumento Descripción
estructura_de_secuencia La estructura de la instancia de la secuencia a modificar.
object_id El índice del objeto dentro de la secuencia a anular.
instance_or_object_id El índice del objeto o el ID de la instancia que se utilizará para anular los objetos de la secuencia.

Devuelve:

N/A

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq); sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

El código anterior crea una nueva instancia de secuencia en la capa dada y luego la modifica para que todas las instancias del objeto "obj_Árboles_Invierno" sean reemplazadas por instancias del objeto "obj_Árboles_Verano".
