La estructura de los fotogramas clave

Cada pista (como se define en la página que cubre las estructuras de pista) tendrá una o más estructuras de fotogramas clave asignadas a ella (que puede obtener de la propiedad fotogramas clave propiedad de la estructura de pista), y cada una tendrá las siguientes propiedades:

  • fotograma: La posición (en fotogramas) a lo largo de la línea de tiempo para el fotograma clave. El valor por defecto es 0.
  • longitud: La longitud del fotograma clave. El valor por defecto es 1, y cuando se establece a valores mayores entonces la propiedad de la pista a la que se refiere el keyframe se mantendrá en el valor inicial durante la duración de la longitud dada. Tenga en cuenta que la propiedad stretch anulará esto si se establece como verdadero.
  • estirar: Si esta propiedad se establece en true, el fotograma clave se estira hasta el siguiente fotograma clave de la pista o hasta el final de la pista si es el último fotograma clave . Puede obtener o establecer este valor, y el valor por defecto es falso.
  • canales: Esta propiedad permite acceder a la lista de structs de datos de fotogramas clave para los canales de la pista. Cuando se obtiene esta propiedad se devuelve un array de Cuando se obtiene esta propiedad se devuelve una matriz de estructuras de datos de fotogramas clave, y cuando se establece esta propiedad se debe especificar una matriz de estructuras de datos de fotogramas clave.
