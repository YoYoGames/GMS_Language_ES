secuencia_obtener

Con esta función se puede recuperar la estructura del objeto de la secuencia a partir de un índice de activos de la secuencia. Se suministra el índice de activos para la secuencia como se define en el navegador de activos, y la función devolverá la estructura del objeto de la secuencia a la que se puede acceder, o -1 si la secuencia no existe o el índice de activos dado no es una secuencia.

Sintaxis:

sequence_get(sequence_index);

Argumento Descripción
índice_de_secuencia El índice de la secuencia como se define en el navegador de activos.

Devuelve:

estructura o -1

Ejemplo:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

El código anterior recupera la estructura del objeto secuencia del objeto secuencia "seq_Loop" y luego establece el modo de bucle para la secuencia (todas las instancias de secuencia creadas posteriormente utilizarán ahora este modo de bucle).
