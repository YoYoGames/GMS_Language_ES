sequence_get_objects

Con esta función se puede recuperar una matriz de todos los índices de objetos que tienen instancias creadas dentro de la secuencia dada. Se suministra la estructura del objeto de la secuencia (devuelta por la función sequence_create() o sequence_get()) o el ID de la secuencia (tal como lo devuelve la función layer_sequence_get_sequence() o de la secuencia de la propiedad struct de la instancia de la secuencia) y la función devolverá un array, donde cada elemento del array es un object_index para los diferentes objetos que utiliza la secuencia para crear instancias.

Sintaxis:

sequence_get_objects(sequence_struct_or_id);

Argumento Descripción
estructura_secuencia_o_id La estructura o ID del objeto de la secuencia para obtener los objetos.

Devuelve:

Array

Ejemplo:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

El código anterior obtiene la estructura de un objeto de secuencia y luego recupera los objetos que utiliza y almacena la matriz en una variable.
