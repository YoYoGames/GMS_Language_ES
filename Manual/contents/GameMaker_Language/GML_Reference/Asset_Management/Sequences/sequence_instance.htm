secuencia_instancia

Esta es una variable incorporada que forma parte de las variables de instancia creadas para cada instancia de objeto en tu juego. Si la instancia está siendo controlada por una secuencia, esta variable contendrá la estructura de la instancia de la secuencia que controla la instancia, de lo contrario será indefinida. Esta es una variable de sólo lectura y no puede ser modificada.

Tenga en cuenta que esta variable se volverá indefinida después de que la Secuencia de control haya terminado, incluso si su Elemento de Secuencia todavía existe, y obtendrá la estructura de nuevo si ese elemento de Secuencia se reproduce.

Sintaxis:

secuencia_instancia

Devuelve:

Booleano

Ejemplo:

if (in_sequence)
{
sequence_instance.speedScale = 2;
}

El código anterior comprueba la variable in_sequence y si es true (lo que significa que la instancia está siendo controlada por una secuencia) entonces cambiará la escala de velocidad de esa Secuencia a 2.
