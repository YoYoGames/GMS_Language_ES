secuencia_in

Esta es una variable incorporada que forma parte de las variables de instancia creadas para cada instancia de objeto en tu juego. Si la instancia está siendo controlada por una secuencia esta variable devolverá true, de lo contrario devolverá false. Esta es una variable de sólo lectura y no puede ser modificada.

Tenga en cuenta que esta variable se convertirá en falsa después de que la Secuencia que la controla haya terminado, incluso si su Elemento de Secuencia todavía existe, y se convertirá en verdadera si ese elemento de Secuencia se reproduce de nuevo.

Puedes usar esta variable en tus objetos controlados por el jugador (o por la CPU) para asegurarte de que sólo se mueven por código cuando no están en una Secuencia. Esto puede ser útil para las cutscenes, cuando necesitas usar las mismas instancias en una Secuencia (usando la función sequence_instance_override_object() ) y necesitas asegurarte de que no pueden ser movidos por su código habitual mientras esa escena está activa.

Sintaxis:

secuencia_in

Devuelve:

Booleano

Ejemplo:

if (!in_sequence)
{
x += move_x;
y += move_y;
}

El código anterior comprueba la variable in_sequence, y si es false (lo que significa que la instancia no está siendo controlada por una secuencia) entonces añade move_x y move_y a la posición de la instancia de la instancia, asegurándose de que sólo se mueve cuando no está en una secuencia.
