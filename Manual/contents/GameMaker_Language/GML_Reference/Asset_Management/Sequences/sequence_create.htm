crear_secuencia

Con esta función puede crear un nuevo objeto de secuencia al que puede añadir pistas. La función devuelve una estructura a la que puede acceder para configurar la nueva secuencia que ha creado. El contenido de esta estructura se detalla en esta página. La estructura del objeto de secuencia puede utilizarse para crear instancias de la secuencia en una capa de la sala utilizando la función layer_sequence_create(). Tenga en cuenta que al crear objetos de secuencia de esta manera debe eliminarlos de nuevo llamando a la función sequence_destroy() cuando ya no se necesiten, de lo contrario se producirá una fuga de memoria que puede ralentizar y eventualmente bloquear su juego.

Sintaxis:

sequence_create();

Devuelve:

Estructurar

Ejemplo:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

El código anterior crea una nueva estructura de objeto de secuencia y establece su longitud de reproducción y el modo de bucle.
