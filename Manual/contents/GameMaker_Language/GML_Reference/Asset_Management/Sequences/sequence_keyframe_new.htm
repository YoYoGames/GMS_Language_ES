sequence_keyframe_new

Con esta función puedes crear una nueva estructura de fotogramas clave de pista, suministrando el tipo de pista al que se aplicará el fotograma clave, que será una de las siguientes constantes:

Constante Descripción Valor
seqtracktype_graphic Esta es una pista de activos gráficos (sprite). 1
seqtracktype_audio Esta es una pista de activos de audio. 2
seqtracktype_instance Esta es una pista de activos de instancia. 14
seqtracktype_sequence Se trata de una pista de activos secuenciales. 7
seqtracktype_clipmask Esta es una pista de activos de grupo de máscara de clip. 8
seqtracktype_clipmask_mask Esta es una pista de activos de máscara de clip utilizada para generar la máscara de clip. 9
seqtracktype_clipmask_subject Esta es una pista de activos de sprite de máscara de clip que se está enmascarando. 10
seqtracktype_group Este seguimiento de los activos de la carpeta de grupo. 11
seqtracktype_colour Esta es una pista de parámetros de datos de color. 4
seqtracktype_real Se trata de una pista de parámetros de valor numérico real. 3
seqtracktype_message Esta es una pista de mensajes de difusión. 15
seqtracktype_moment Esta es una pista de eventos/momentos. 16
seqtracktype_bool No se utiliza actualmente. 5
seqtracktype_string No se utiliza actualmente. 6
seqtracktype_spriteframes No se utiliza actualmente. 13
seqtracktype_empty No se utiliza actualmente. 12

La función devolverá una estructura de fotogramas clave de pista a la que se le pueden añadir datos antes de asignarlos a una estructura de pista.

Sintaxis:

sequence_keyframe_new(type);

Argumento Descripción
tipo El tipo de fotograma clave a crear, una constante, listada arriba.

Devuelve:

Estructurar

Ejemplo:

myseq = sequence_create();
var mytracks = array_create(1);
mytracks[0] = sequence_track_new(seqtracktype_graphic);
var graphickeys = array_create(1);
graphickeys[0] = sequence_keyframe_new(seqtracktype_graphic);
graphickeys[0].frame = 0;
graphickeys[0].length = 1;
graphickeys[0].stretch = true;
graphickeys[0].disabled = false;
var graphickeydata = array_create(1);
graphickeydata[0] = sequence_keyframedata_new(seqtracktype_graphic);
graphickeydata[0].spriteIndex = spr_Platform;
graphickeydata[0].channel = 0;
graphickeys[0].channels = graphickeydata;
mytracks[0].name = "TestGraphicTrack";
mytracks[0]. keyframes= graphickeys;
myseq.tracks = mytracks;

El código anterior crea una nueva secuencia y luego crea una pista de activos gráficos y establece algunos datos de fotogramas clave en la pista. Esta pista se asigna a la instancia.
