secuencia_existe

Con esta función puede comprobar si un objeto de secuencia existe o no. Se suministra la estructura del objeto secuencia (devuelta por la función sequence_create() o sequence_get()) o el ID de la secuencia (devuelto por la función layer_sequence_get_sequence() o de la estructura de la instancia de la secuencia, o el índice del navegador de activos ) y la función devolverá true si el objeto de secuencia existe o false si no existe.

Sintaxis:

sequence_exists(sequence_struct_or_id);

Argumento Descripción
estructura_secuencia_o_id La estructura o ID del objeto de la secuencia que se va a comprobar

Devuelve:

Bool

Ejemplo:

if sequence_exists(my_seq)
{
sequence_destroy(my_seq);
}

El código anterior comprueba si el objeto de la secuencia dada existe y si lo hace se destruye.
