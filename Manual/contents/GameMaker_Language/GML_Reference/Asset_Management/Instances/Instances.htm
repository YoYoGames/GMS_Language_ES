Instancias

Las inst ancias se crean a partir de Objetos y, una vez creadas, pueden ser modificadas y manipuladas mediante código GML y acciones DnD™. Las principales funciones relacionadas con las instancias que están disponibles y más utilizadas son:

  • instance_create_layer
  • instance_create_depth
  • instance_destroy
  • instancia_existe
  • cambio_de_instancia
  • copia_de_instancia
  • buscar_instancias
  • instancia_más_alejada
  • instancia_más_cercana
  • número_de_instancia
  • lugar_de_instancia
  • lista_de_lugares_de_instancia
  • posición_de_instancia
  • lista_de_posición_de_instancia

Tenga en cuenta que hay dos variables adicionales incorporadas (son variables de ámbito global ) y una función relacionada con el uso de instancias que puede ser útil para recuperar ciertos detalles:

  • instance_id_get
  • instance_id
  • conteo_de_instancias

Hay un número de variables disponibles dentro de cada instancia de un objeto que pueden ser usadas para controlar diferentes aspectos de cómo se muestra la instancia, cómo usa otros activos, y cómo reacciona a las colisiones, etc... que están todas listadas aquí:

  • Variables de instancia

Las instancias tienen una variable de matriz que se puede utilizar para obtener o establecer los valores del evento de alarma, pero también hay dos funciones que se pueden utilizar para hacer lo mismo:

  • Alarma_obtener
  • alarm_set

Por último, también tienes la posibilidad de desactivar instancias, lo que significa que no procesarán ninguno de sus eventos, aunque seguirán existiendo dentro del juego. Las funciones relacionadas con esto están disponibles en la siguiente página:

  • Desactivación de instancias
