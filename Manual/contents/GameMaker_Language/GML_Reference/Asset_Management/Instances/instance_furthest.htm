instancia_más_alejada

Esta función comprobará todas las instancias del objeto dado para ver cuál está más lejos del punto x/y dado. Todas las comprobaciones se harán desde la posición x/y dada hasta el origen (la posición x/y) de las instancias del objeto especificado. Si no existen instancias del objeto, la función devolverá la palabra clave noone, pero si hay instancias entonces devolverá el id de la instancia encontrada. Tenga en cuenta que si la instancia que ejecuta el código tiene el mismo índice de objeto que el objeto que se está comprobando, entonces se incluirá en la comprobación (esto incluye la comprobación de los objetos padre si la instancia que llama es también un hijo del padre).

Sintaxis:

instancia_más_alejada(x, y, obj);

Argumento Descripción
x La posición x para comprobar las instancias lejos de.
y La posición y para comprobar las instancias lejos de.
obj El objeto que se va a comprobar para las instancias de.

Devuelve:

Real (un valor de ID de instancia o la palabra clave noone)

Ejemplo:

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
{
draw_line(x, y, inst.x, inst.y);
}

El código anterior encontrará la instancia más lejana que comparta el mismo índice de objeto que la instancia que ejecuta el código y almacenará el id en una variable. Esta variable se comprueba para ver si es la misma que la de la instancia que llama y, si no lo es no lo es, se traza una línea entre las dos instancias.
