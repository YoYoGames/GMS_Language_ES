posición_de_instancia

Con esta función puede comprobar una posición para una colisión con otra instancia o con todas las instancias de un objeto. Cuando se utiliza esta función, se comprueba un único punto de la sala para una instancia o un objeto. Esta comprobación se hará contra el cuadro delimitador de la instancia o contra la máscara de la instancia. de la instancia o contra la máscara de la instancia si esa instancia tiene colisiones precisas comprobadas y devolverá la instancia única id. Si no necesita el identificador de la instancia que colisiona, debería debería considerar el uso de posición_encuentro() en su lugar. Esta función también acepta las palabras claveespeciales todos y other y devolverá la palabra clave noone si no se produce ninguna colisión o el valor de ID único de la instancia encontrada si se produce una colisión.

Sintaxis:

instance_position( x, y, obj );

Argumento Descripción
x La posición x para comprobar las instancias.
y La posición y para comprobar las instancias.
obj El objeto que se va a comprobar para las instancias de.

Devuelve:

Real (un valor de ID de instancia o la palabra clave noone)

Ejemplo:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
with (inst) image_index=1;
instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

El código anterior comprobará si hay una colisión con una instancia de "obj_Pause_Button" en la posición del ratón, y si la hay utilizará el id devuelto para establecer su image_index a un nuevo valor antes de crear una nueva instancia del objeto "obj_Menú".
