número_de_instancia

Con esta función se puede averiguar cuántas instancias activas del objeto especificado existen en la sala. Al realizar la comprobación mediante esta función, si el objeto es un padre, todos los objetos hijos se incluirán también en el valor de retorno, y tenga en cuenta también que aquellas instancias que hayan sido desactivadas con las funciones de desactivación de instancias no se incluirán en esta comprobación.

Sintaxis:

número_de_instancia(obj);

Argumento Descripción
obj El objeto para totalizar el número de instancias.

Devuelve:

Real

Ejemplo:

if instance_number(object_index) < 50
{
instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

El código anterior comprobará el número de instancias que se crean del mismo objeto que la instancia actual y, si hay menos de 50, creará otra en una posición aleatoria dentro de la habitación.
