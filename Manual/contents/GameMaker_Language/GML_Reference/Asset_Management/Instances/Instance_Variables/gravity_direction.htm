dirección_de_gravedad

gravity_direction es una de las propiedades "incorporadas" que tienen todas las instancias y se puede utilizar para establecer la dirección del movimiento cuando las instancias gravedad es mayor que 0. Tenga en cuenta que las direcciones en GameMaker Studio 2 se calculan normalmente como 0° a la derecha, 90° hacia arriba, 180° a la izquierda y 270° hacia abajo.

Sintaxis:

dirección_de_gravedad;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si place_meeting(x, y, obj_switch)
{
gravity_direction += 180;
}

El código anterior cambiará la dirección de la gravedad si el objeto jugador se encuentra con una instancia de "obj_switch".
