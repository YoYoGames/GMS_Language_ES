id

Esta variable de sólo lectura contiene el número de identificación único de la instancia. Cada instancia que se crea - ya sea a través de código o añadiéndola a una sala en el Editor de Salas - se le da un número que se utiliza internamente para identificar esta instancia y la variable id es la que puedes utilizar para referenciarla. El id también es devuelto (y puede ser almacenado en una variable) cuando se crea una instancia usando instance_create_layer() o instance_create_depth() , así como otras funciones de instancia.

NOTA: El valor de esta variable no es el mismo que el identificador que se le da a la instancia en el editor de salas y también es diferente al instance_id matriz que contiene todos los ids de todas las instancias actualmente activas.

Sintaxis:

id;

Devuelve:

Real (entero - valor de identificación de la instancia)

Ejemplo:

for(var i = 0; i < instance_count; i++;)
{
if instance_id[i] != id
{
instance_id[i].scr += 5;
}
}

El código anterior añade 5 a la variable "scr" para cada instancia activa en la sala excepto la que ejecuta el código. Lo hace recorriendo TODAS las instancias activas (usando el array "instance_id" para devolver el ID de cada instancia activa ID) y comparándolos con la variable "id" incorporada, que es el ID de la instancia original que ejecuta el código.
