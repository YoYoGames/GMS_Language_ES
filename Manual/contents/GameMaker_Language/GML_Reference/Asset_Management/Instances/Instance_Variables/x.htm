x

El valor x de una instancia es la posición horizontal en la habitación actual, medida en píxeles. Este valor puede ser 0, positivo o negativo, donde 0 es el lado izquierdo de la habitación y moverse a la derecha aumenta x, moverse a la izquierda disminuye x (un valor negativo de significa que la instancia ha salido del lado izquierdo de la habitación). Incluso puede dar a x un valor real como 12,345, pero como no se puede obtener 0,345 de un píxel, la instancia puede parecer que no se ha movido, aunque estos valores se mantendrán y utilizados internamente por GameMaker Studio 2 y son perfectamente válidos.

Estableciendo los valores x (e y) puedes hacer que el objeto salte por la habitación hasta la posición que elijas, o puedes sumar y restar a ellos cantidades menores para dar la ilusión de movimiento sin usar realmente los velocidad y dirección incorporadas. Esto debe tenerse en cuenta al crear tus juegos, ya que a menudo un tipo de movimiento se presta a un estilo particular de juego más que otro.

Sintaxis:

x;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

El código anterior utilizará las teclas de flecha para sumar o restar a la posición x y así crear la ilusión de movimiento.
