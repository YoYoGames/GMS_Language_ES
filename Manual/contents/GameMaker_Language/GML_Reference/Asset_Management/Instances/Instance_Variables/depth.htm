profundidad

Cuando creas un objeto puedes asignarle una profundidad inicial que define cómo se dibujarán las instancias de ese objeto en la sala cuando el juego se esté ejecutando y esta variable se puede utilizar para obtener y cambiar ese valor de profundidad mientras el juego se está ejecutando. Normalmente no es necesario utilizar esta variable, ya que se debe establecer que las instancias se añadan a capas discretas, que a su vez se establecen a una profundidad específica, pero puede ser que quieras cambiar la profundidad de una instancia utilizando este valor, en cuyo caso se creará una "capa temporal" específicamente para la instancia en la profundidad dada. Tenga en cuenta que cuando no haya instancias en la misma profundidad, esta capa temporal se eliminará de la memoria (a diferencia de las capas regulares que permanecerán aunque no tengan nada en ellas).

IMPORTANTE! Cuando se modifica la variable de profundidad y GameMaker Studio 2 gestiona las capas, la variable incorporada capa devolverá -1 y no el ID de la capa, ya que las capas gestionadas no pueden ser manipuladas mediante código.

En GameMaker Studio 2 cuanto menor sea el valor de profundidad de una instancia, más "cerca de la cámara" se dibujará esa instancia, mientras que un valor de profundidad mayor significa que la instancia se dibujará "más lejos de la cámara", es decir -1000 se dibuja encima de -100, que se dibuja encima de 0, que se dibuja encima de 100, que se dibuja encima de 1000.

Tenga en cuenta que las instancias de un objeto que tienen la misma profundidad pueden dibujarse por encima o por debajo de las demás, incluso cuando parecen estar colocadas una encima de la otra en el editor de espacios. Si quieres garantizar que algo se dibuje por encima o por debajo de todo lo demás, siempre debes establecer la profundidad (o capa) explícitamente. Tenga en cuenta también que hay un mínimo (-16000) y un valor máximo (+16000) de profundidad fuera de los cuales las instancias no se dibujarán, aunque seguirán existiendo y procesando eventos.

NOTA: No se puede establecer la profundidad de una instancia en su evento de dibujo (todos los demás eventos están bien).

Sintaxis:

profundidad;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si y != yprevio
{
profundidad = -y;
}

El código anterior comprobará si la posición y ha cambiado y, si lo ha hecho, la profundidad también se ajustará para que se corresponda con ella.
