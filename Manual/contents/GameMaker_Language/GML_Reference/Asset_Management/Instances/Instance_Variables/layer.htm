capa

Esta variable incorporada se crea para cada instancia de una sala y contiene el valor del ID de la capa a la que está asignada la instancia. Este valor puede utilizarse en otras funciones como layer_get_depth() o se puede cambiar para mover la instancia a otra capa, pero ten en cuenta que si la capa que se está asignando no existe en la sala actual, entonces obtendrás un error que obligará a cerrar tu juego. Cuando se asigna una capa, se debe proporcionar el único ID de la capa devuelto por la función layer_get_id() (cuando se utilizan capas con nombre en la sala), o como lo devuelve la función layer_create() (cuando se crean capas propias en tiempo de ejecución).

IMPORTANTE: si ha creado la instancia con la función instance_create_depth() o ha cambiado manualmente la variable profundidad la capa asignada a la instancia se convierte en una capa "gestionada", es decir, una capa que GameMaker Studio 2 controla y gestiona automáticamente. En estos casos la variable de capa devolverá -1.

Sintaxis:

capa;

Devuelve:

Real (entero - valor del índice de recursos)

Ejemplo:

capa = layer_create(-1000);

El código anterior creará una nueva capa con una profundidad de -1000 y luego establecerá la variable de capa de la instancia al ID de la capa devuelta, moviendo la instancia de la capa en la que se encuentra actualmente a la nueva capa que se está creando.
