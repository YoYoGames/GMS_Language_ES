visible

Una instancia puede ser marcada como visible o no estableciendo esta variable como verdadera (visible) o falsa (invisible). Esto funciona diciéndole a GameMaker Studio 2 que omita el evento de dibujo para esta instancia, por lo que se debe tener cuidado cuando se utiliza esto, ya que significa que ningún código colocado en el evento de dibujo se ejecutará mientras la instancia no es visible. También hay que tener en cuenta que si la capa a la que está asignada la instancia está marcada como invisible, establecer esta variable a true no tendrá ningún efecto hasta que la propia capa sea marcada como visible también.

Sintaxis:

visible;

Devuelve:

Booleano

Ejemplo:

if other.visible = true
{
x = xprevious;
y = yprevious;
}

El código anterior comprobará la bandera visible de la "otra" instancia en un evento de colisión y, si es verdadera, moverá esa instancia a su posición anterior.
