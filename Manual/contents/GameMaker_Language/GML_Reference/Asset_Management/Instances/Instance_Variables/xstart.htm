xstart

Esta variable almacena la posición x inicial de la instancia cuando se crea por primera vez en la sala. No es una variable de sólo lectura y puede ser establecida así como leída.

Sintaxis:

xstart;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si place_meeting(x, y, obj_spike)
{
score -= 100;
x = xstart;
y = ystart;
}

El código anterior comprobará si hay una colisión con una instancia de "obj_spike" y, si la hay, descuenta 100 de la puntuación y devuelve la instancia a su posición inicial.
