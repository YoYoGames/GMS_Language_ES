fricción

La fricción es una de las propiedades "incorporadas" que pueden tener las instancias y puede utilizarse para ralentizar la instancia en el tiempo cuando el velocidad es distinto de cero. Funciona simplemente restando una cantidad de la velocidad a cada paso hasta que el objeto tenga una velocidad de 0, por lo que si la fricción se establece, por ejemplo, en 0,1 y la velocidad de la instancia es de 1 (1 píxel por paso), se ralentizará y se detendrá después de que hayan pasado 10 pasos. Tenga en cuenta también que la fricción se aplica se aplica a las velocidades positivas y negativas por igual, con el resultado neto de que el objeto tiene una velocidad de 0 después de un tiempo determinado.

Sintaxis:

fricción;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo:

si abs(velocidad) > 0
{
fricción = 0,05;
}
si no
{
fricción = 0;
}

El código anterior sólo aplicará la fricción si la velocidad absoluta de la instancia es superior a 0.
