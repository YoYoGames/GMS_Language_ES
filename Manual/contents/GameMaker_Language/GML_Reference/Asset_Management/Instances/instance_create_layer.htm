instance_create_layer

Con esta función se puede crear una nueva instancia del objeto especificado en cualquier punto de la sala y en la capa especificada. La capa puede identificarse mediante el valor de ID de la capa (devuelto por la función layer_create()) o por el nombre de la capa (como una cadena, por ejemplo "instance_layer") tal y como se define en el editor de la sala.

Esta función devuelve el id de la nueva instancia que puede ser almacenada en una variable o usada para acceder a esa instancia. Tenga en cuenta que esta función también llamará al evento Create de la instancia que se está creando antes de continuar con el código o las acciones del evento que llamó a la función.

IMPORTANTE! Hay una profundidad de capa mínima y máxima de -16000 a 16000. Todo lo que se coloque en una capa fuera de ese rango no se dibujará aunque todos los eventos se seguirán ejecutando normalmente.

Sintaxis:

instance_create_layer(x, y, layer_id, obj);

Argumento Descripción
x La posición x en la que se creará el objeto
y La posición y en la que se creará el objeto
layer_id El ID (o nombre) de la capa a la que se asigna la instancia creada
obj El índice del objeto para crear una instancia de

Devuelve:

Real (valor de identificación de la instancia)

Ejemplo:

var inst = instance_create_layer(x, y, "instance_layer", obj_Bullet);
with (inst)
{
velocidad = otra.velocidad_de_disparo;
dirección = otra.ángulo_de_imagen;
}

El código anterior crea una nueva instancia del objeto "obj_Bullet" y almacena el id de la instancia en una variable. Esta variable se utiliza entonces para asignar la velocidad y la dirección a la nueva instancia.
