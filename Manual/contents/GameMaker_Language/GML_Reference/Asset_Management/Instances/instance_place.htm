lugar_de_instancia

Con esta función puede comprobar una posición para una colisión con otra instancia o todas las instancias de un objeto utilizando la máscara de colisión de la instancia que ejecuta el código para la comprobación. Cuando usted usa esto, está pidiendo a GameMaker Studio 2 que mueva la instancia a la nueva posición, compruebe si hay una colisión, vuelva a moverse y le diga si se encontró una colisión o no. Esto funcionará para colisiones precisas, pero sólo si tanto la instancia como el objeto que se está comprobando tienen máscaras de colisión precisas esta función devolverá la instancia única del objeto con el que se ha colisionado, pero no se ha encontrado ninguna colisión. id del objeto con el que se está colisionando, pero si no se necesita es ligeramente más rápido utilizar la función función lugar_colisión(). Esta función también acepta las palabras clave especiales all y other y devolverá la palabra clave noone si no se produce ninguna colisión, o el valor del ID de la instancia única encontrada si se produce una colisión.

Tenga en cuenta que las coordenadas x/y dadas se redondearán al número entero más cercano antes de que se realice la comprobación, por lo que si esto no es lo que usted necesita (o ha estado utilizando un producto GameMaker heredado), deberá soltar las coordenadas en el campo comprobación:

instance_place_list(floor(x), floor(y), obj_Enemy, list, true).

Sintaxis:

instance_place(x, y, obj);

Argumento Descripción
x La posición x para comprobar las instancias.
y La posición y para comprobar las instancias.
obj El objeto que se va a comprobar para las instancias de.

Devuelve:

Real (un valor de ID de instancia o la palabra clave noone)

Ejemplo:

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
{
hp -= _inst.dmg;
instance_destroy(_inst);
}

El código anterior comprobará si hay una colisión con instancias de "obj_Enemigo" y si la hay, reducirá la variable "hp" en la cantidad almacenada en la variable "dmg" de la instancia que colisiona y luego destruirá la instancia que colisiona la instancia que colisiona.
