instancia_existe

Esta función se puede utilizar de dos maneras, dependiendo de lo que desee comprobar. Puedes darle un object_index para comprobar, en cuyo caso esta función devolverá true si existe alguna instancia activa del objeto especificado en la sala actual, o también puede proporcionarle un id de instancia, en cuyo caso esta función devolverá true si esa instancia específica existe y está activa en la sala actual.

Tenga en cuenta que esta función no tiene en cuenta las instancias que han sido desactivadas mediante las funciones de desactivación de instancias.

Sintaxis:

instance_exists(obj);

Argumento Descripción
obj El objeto o la instancia cuya existencia se va a comprobar.

Devuelve:

Booleano

Ejemplo:

if !instance_exists(obj_Enemy)
{
score += 200;
room_goto(rm_hiscores);
}

El código anterior comprueba si existe alguna instancia del objeto "obj_Enemigo" y si no es así lo añade a la variable "puntuación" y cambia de sala.
