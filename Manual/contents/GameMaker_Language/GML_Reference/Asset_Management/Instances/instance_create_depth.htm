instance_create_depth

Con esta función puedes crear una nueva instancia del objeto especificado en cualquier punto de la habitación y a la profundidad especificada. La profundidad puede ser cualquier valor, donde cuanto menor sea la profundidad más "cerca" de la cámara se dibujarán las cosas y cuanto más alta sea la profundidad, más lejos, por lo que una instancia en la profundidad -200 se dibujará sobre una instancia en la profundidad +300 (por ejemplo).

Tenga en cuenta que esta función realmente creará una capa de la sala para la instancia, ya que todas las instancias deben estar en una capa de la sala, pero como esta es una capa gestionada (es decir, no una que haya creado a través de código o en la sala, sino una que GameMaker Studio 2 ha creado automáticamente) no se puede acceder a ella, y la capa variable de instancia devolverá -1.

IMPORTANTE! Hay una profundidad de capa mínima y máxima de -16000 a 16000. Todo lo que se coloque en una profundidad fuera de ese rango no se dibujará aunque todos los eventos se seguirán ejecutando normalmente.

Esta función devuelve el id de la nueva instancia que puede ser almacenada en una variable o usada para acceder a esa instancia. Tenga en cuenta que esta función también llamará al evento Create de la instancia que se está creando antes de continuar con el código o las acciones del evento que llamó a la función.

Sintaxis:

instance_create_depth(x, y, depth, obj);

Argumento Descripción
x La posición x en la que se creará el objeto
y La posición y en la que se creará el objeto
profundidad La profundidad a la que se asigna la instancia creada
obj El índice del objeto para crear una instancia de

Devuelve:

Real (valor de identificación de la instancia)

Ejemplo:

var inst = instance_create_depth(x, y, -10000, obj_Bullet);
with (inst)
{
velocidad = otra.velocidad_de_disparo;
dirección = otra.ángulo_de_imagen;
}

El código anterior crea una nueva instancia del objeto "obj_Bullet" y almacena el id de la instancia en una variable. Esta variable se utiliza entonces para asignar la velocidad y la dirección a la nueva instancia.
