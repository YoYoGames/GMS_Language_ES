instance_id_get

Con esta función se puede obtener el valor de ID único de cualquier instancia de la lista de instancias actualmente activa. Se da el índice en la lista de instancias para obtener el ID y la función devolverá el valor para almacenarlo en una variable.

Sintaxis:

instance_id_get(index);

Argumento Descripción
índice El índice dentro de la lista de instancias de 0 - (conteo de instancias - 1).

Devuelve:

Valor del ID de la instancia (real)

Ejemplo:

for (var i = 0; i < instance_count; ++i;)
{
var temp_id = instance_id_get(i);
with (temp_id)
{
speed += 0.1;
}
}

El código anterior recorrerá todas las instancias de la sala y añadirá 0,1 a su velocidad.
