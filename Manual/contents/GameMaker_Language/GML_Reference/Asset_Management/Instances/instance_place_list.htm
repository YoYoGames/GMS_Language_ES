lista_de_lugares_de_instancia

Con esta función puede comprobar una posición para una colisión con todas las instancias de un objeto utilizando la máscara de colisión de la instancia que ejecuta el código para la comprobación. Cuando usted utiliza esto usted está pidiendo efectivamente GameMaker Studio 2 para mover la instancia a la nueva posición, comprobar si hay una colisión, a continuación, mover de nuevo y le dirá si se encontró una colisión o no. Esto funcionará para colisiones precisas, pero sólo si tanto la instancia como las instancias del objeto que se está comprobando tienen máscaras de colisión precisa seleccionadas, de lo contrario sólo se aplican las colisiones de caja delimitadora. La función rellenará una lista DS (creada previamente) con los valores únicos id de todas las instancias del objeto que se consideran en colisión con la instancia que llama, y tiene la opción de ordenar la lista en función de la distancia desde el origen de la instancia que realiza la comprobación hasta el origen de las instancias que se encuentran en colisión. Tenga en cuenta que la función también acepta la palabra clave especial todoen cuyo caso se listarán todas las instancias que se encuentren en colisión. La función devuelve el número de instancias encontradas, o 0 si no se encuentra ninguna.

Tenga en cuenta que las coordenadas x/y dadas se redondearán al número entero más cercano antes de que se realice la comprobación, por lo que si esto no es lo que usted necesita (o ha estado utilizando un producto GameMaker heredado), deberá soltar las coordenadas en el campo comprobación:

instance_place_list(floor(x), floor(y), obj_Enemy, list, true).

Sintaxis:

instance_place_list(x, y, obj, list, ordered);

Argumento Descripción
x La posición x para comprobar las instancias.
y La posición y para comprobar las instancias.
obj El objeto que se va a comprobar para las instancias de.
lista La lista DS que se utilizará para almacenar los ID de las instancias que colisionan.
ordenado Si la lista debe ser ordenada por distancia(true) o no(false).

Devuelve:

Int (El número de instancias que se encuentran en colisión)

Ejemplo:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);
if _num > 0
{
for (var i = 0; i < _num; ++i;)
{
instance_destroy(_list[| i]);
}
}
ds_list_destroy(_list);

El código anterior buscará una colisión con todas las instancias encontradas en la posición de la instancia que llama. Éstas se añadirán a una lista DS, que luego se recorrerá para destruir cada una de las instancias en colisión.
