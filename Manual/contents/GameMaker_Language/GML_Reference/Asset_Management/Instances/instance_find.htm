buscar_instancias

Todas las instancias tienen un identificador único (id) que se puede utilizar para modificar y manipular mientras se ejecuta un juego, pero no siempre se puede saber cuál es el identificador de una instancia específica, por lo que esta función puede ayudarte, ya que puedes usarla para iterar a través de todas ellas para encontrar lo que necesitas. Especifica el objeto del que quieres encontrar la instancia y un número, y si hay una instancia en esa posición en la lista de instancias entonces la función devuelve el id de esa instancia, y si no, devuelve la palabra clave especial noone. También puede utilizar la palabra clave all para iterar a través de todas las instancias de una habitación, así como un objeto padre para iterar por todas las instancias que forman parte de esa jerarquía padre / hijo, e incluso puede especificar una instancia (si tiene su id) como una comprobación para ver si existe realmente en la sala actual. Tenga en cuenta que, como las instancias se ordenan de forma arbitraria, no hay un orden específico para la comprobación de las instancias por parte de esta función, y cualquier instancia puede estar en cualquier posición.

El valor máximo de "n" en esta función sería

  • Para la palabra clave all: conteo_de_instancias - 1
  • Para un índice de objeto: número_de_instancia(OBJ) - 1

Sintaxis:

instance_find(obj, n);

Argumento Descripción
obj El objeto para encontrar la enésima instancia de
n El número de la instancia a encontrar.

Devuelve:

Real (un valor de ID de instancia o la palabra clave noone)

Ejemplo:

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
enemy[i] = instance_find(obj_Enemy,i);
}

El código anterior utilizará un bucle for para iterar a través de todas las instancias de "obj_Enemigo" y almacenar su id en el array "enemigo[]".
