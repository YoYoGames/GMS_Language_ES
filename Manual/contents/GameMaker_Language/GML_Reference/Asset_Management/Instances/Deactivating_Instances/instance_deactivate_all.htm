instance_deactivate_all

Esta función desactivará todas las instancias que estén en la sala en el momento en que se ejecute el código. Esto puede incluir a la instancia que ejecuta el código si la bandera "notme" se establece en falso, pero normalmente querrás que esa instancia que esté activa, en cuyo caso la bandera "notme" debería ser verdadera. Tenga en cuenta que la desactivación no es instantánea, y una instancia que ha sido desactivada de esta manera no se considerará inactiva hasta el final del evento en el que se llamó a la función.

NOTA: Si desactiva una instancia al iniciar la sala (es decir, desde el código de creación de la sala, o desde un evento de creación de una instancia dentro de la sala) todas las instancias que se coloquen dentro de la sala desde el editor de salas seguirán ejecutando su evento de creación antes de ser desactivadas.

ADVERTENCIA: Desactivar las instancias que tienen la física activada NO impedirá que sus accesorios interactúen dentro de la simulación de la física. Para ello, debe establecer su phy_active variable a true o false al activar/desactivar las instancias.

Sintaxis:

instance_deactivate_all(notme);

Argumento Descripción
notme Si se mantiene la instancia de llamada activada (true) o no (false).

Devuelve:

N/A

Ejemplo:

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

El código anterior desactiva todas las instancias excepto la que está ejecutando el código y luego activa una región dentro de la sala.
