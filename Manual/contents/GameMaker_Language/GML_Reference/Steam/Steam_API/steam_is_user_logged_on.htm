steam_is_user_logged_on

Esta función devolverá true si el cliente de Steam tiene actualmente una conexión activa con los servidores de Steam. Si devuelve false, significa que no hay una conexión activa debido a un problema de red en la máquina local, o a que el servidor de Steam no funciona o está ocupado.

Sintaxis:

steam_is_user_logged_on();

Devuelve:

Cadena

Ejemplo:

if steam_is_user_logged_on()
{
global.user_id = steam_get_user_account_id();
}

El código anterior comprobará si el usuario ha iniciado sesión en el servidor de Steam y si almacena el ID de usuario en una variable global.
