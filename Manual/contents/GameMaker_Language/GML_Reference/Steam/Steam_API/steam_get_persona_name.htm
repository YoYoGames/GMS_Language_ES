steam_get_persona_name

Puedes utilizar esta función para devolver el nombre de usuario del usuario actualmente conectado al cliente de Steam. Este es el nombre de pantalla visible y no el id de usuario único (este puede ser encontrado usando la función steam_get_user_steam_id()).

Sintaxis:

steam_get_persona_name();

Devuelve:

Cadena

Ejemplo:

if steam_initialised()
{
global.p_name = steam_get_persona_name();
}

El código anterior establecerá una variable global con el nombre de pantalla del usuario actual si la API del cliente de Steam está correctamente inicializada.
