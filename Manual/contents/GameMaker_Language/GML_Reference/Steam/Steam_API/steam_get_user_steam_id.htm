steam_get_user_steam_id

Puedes utilizar esta función para devolver el identificador único de usuario de Steam del usuario actualmente conectado al cliente de Steam. Se trata de un valor único de 64 bits que se puede utilizar para identificar a los usuarios individuales y no su nombre de usuario en pantalla (esto se puede encontrar utilizando la función steam_get_persona_name()). Como se trata de un int64, es posible que tenga que convertirlo en tal cuando lo utilice en funciones posteriores usando la función int64() función.

Sintaxis:

steam_get_user_steam_id();

Devuelve:

Entero (int64)

Ejemplo:

si steam_initialised()
{
global.u_id = steam_get_user_steam_id();
}

El código anterior establecerá una variable global con el ID único de Steam del usuario actual si la API del cliente de Steam está correctamente inicializada.
