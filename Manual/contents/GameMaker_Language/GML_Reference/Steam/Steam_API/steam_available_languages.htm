steam_available_languages

Esta función se puede utilizar para recuperar una lista de todos los idiomas disponibles para Steam. El valor devuelto es simplemente una lista de idiomas separada por comas.

Sintaxis:

steam_available_languages();

Devuelve:

Cadena

Ejemplo:

idioma = steam_available_languages();

Lo anterior obtiene los idiomas disponibles para Steam como una cadena y la almacena en una variable.
