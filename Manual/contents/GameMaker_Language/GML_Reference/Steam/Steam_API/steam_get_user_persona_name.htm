steam_get_user_persona_name

Esta función se puede utilizar para recuperar el nombre de usuario (nombre de pantalla) para cualquier ID de usuario específico. Usted da el ID de usuario entonces la función devuelve un valor único de ID asíncrono que puede ser utilizado para comprobar los detalles correctamente en el eventoSteam Async. Este evento tendrá el async_load mapa DS que se rellenará con los siguientes pares clave/valor:

  • " id" - El ID asíncrono devuelto por la función que desencadenó el evento
  • " event_type" - La cadena " user_persona_name"
  • " steamid""- El valor del ID de Steam utilizado para obtener el nombre de usuario.
  • "Nombre_de_persona" - El nombre de usuario para el ID de Steam dado

Sintaxis:

steam_get_user_persona_name(steamID);

Argumento Descripción
steamID El ID de Steam único para un usuario.

Devuelve:

Real

Ejemplo:

request = steam_get_user_persona_name(global.UGC_UserID);

El código anterior solicitará el nombre de usuario del ID de usuario almacenado en la variable global "UGC_UserID", almacenando el valor devuelto en una variable para su análisis en el evento asíncrono.
