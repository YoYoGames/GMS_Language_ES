steam_current_game_language

Esta función se utiliza para recuperar el idioma actual que Steam está utilizando (como una cadena), por ejemplo "inglés".

Sintaxis:

steam_current_game_language();

Devuelve:

Cadena

Ejemplo:

idioma = steam_current_game_language();

El código anterior obtiene el idioma utilizado para el juego actual.
