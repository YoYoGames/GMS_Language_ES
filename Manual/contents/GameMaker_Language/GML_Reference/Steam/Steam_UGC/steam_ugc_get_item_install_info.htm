steam_ugc_get_item_install_info

Esta función se puede utilizar para recuperar información sobre cualquier elemento de archivo publicado que se haya suscrito y descargado en el área de almacenamiento local de Steam para tu juego. Se da el ID del elemento y se suministra el índice a un mapa DS vacío que se rellenará con los siguientes pares clave/valor si el elemento existe:

  • " size_on_disk" - El tamaño del archivo en el disco (en bytes)
  • "legacy_item" - Será verdadero o falso dependiendo de si es un archivo heredado o no.
  • "Carpeta" - Esta es la ruta completa del contenido instalado (por favor, consulta "Instalación de elementos" en los documentos del SDK de Steam, ya que los elementos " heredados" cargados con el método antiguo se tratan de forma diferente).

Si el elemento existe, la función devolverá true y rellenará el mapa, de lo contrario devolverá false y el mapa permanecerá vacío.

Sintaxis:

steam_ugc_get_item_install_info(published_file_id, info_map);

Argumento Descripción
archivo_publicado_id El identificador único para el UGC que se va a actualizar.
info_mapa Un índice de mapa DS (creado previamente).

Devuelve:

Booleano

Ejemplo:

var item_map = ds_map_create();
steam_ugc_get_item_install_info(global.fileID, item_map);

El código anterior consultará el estado de instalación del elemento indexado en la variable global "fileID", utilizando un Mapa DS para almacenar la información.
