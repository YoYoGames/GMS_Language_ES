steam_ugc_download

Con esta función puede descargar una imagen de vista previa para cualquier elemento UGC dado. El ugc_handle es el valor de identificación único de la imagen (que puede obtener con la función steam_ugc_send_query()), y el nombre del archivo de destino es el nombre (y la ruta local dentro de la caja de arena de Steam ) que desea dar al archivo de imagen cuando la descarga se haya completado.

Cuando se utiliza esta función, se desencadena un evento asíncronode Steam para informar de los detalles del archivo de imagen solicitado que contiene los siguientes pares clave/valor en el async_load ds_map:

  • " id" - El ID asíncrono devuelto por la función de llamada
  • "resultado" - El resultado de la operación (un valor real). Será la constante GML ugc_result_success o algún otro número real. Por lo tanto, debe comprobar esta constante para asegurarse de que la llamada ha tenido éxito, y si de lo contrario algo no ha funcionado correctamente. El resto de los posibles valores devueltos se muestran como el resultado del valor "EResult" de Steam y debería ver steamclientpublic .h en las cabeceras del SDK para los 89 valores posibles.
  • " event_type" - Esta clave tendrá el valor " ugc_download"
  • "original_filename" - Esta clave contiene el nombre original del archivo de imagen en el servidor (una cadena )
  • " dest_filename" - Esta clave contiene el nombre del archivo de imagen que se ha pasado (una cadena)
  • " ugc_handle" - Esta clave contiene el valor de ugc_handle que se pasó a la función de llamada

Sintaxis:

steam_ugc_download(ugc_handle, dest_filename);

Argumento Descripción
ugc_handle El identificador único de la vista previa que se va a descargar.
nombre_destino El nombre del archivo con el que se guardará la vista previa.

Devuelve:

Identificación asíncrona

Ejemplo ampliado:

En este ejemplo, primero llamamos a la función y almacenamos el valor del ID asíncrono en una variable:

steam_get= steam_ugc_download(steam_handle, "\UGC\Preview_file.png");

Esto enviaría una solicitud de archivo a la API de Steam, generando un evento asíncrono que trataríamos de la siguiente manera:

var event_id = async_load[? "id"];
if event_id == steam_get
{
var type = async_load[? "event_type"];
if type == "ugc_download"
{
sprite_delete(preview_sprite); preview_sprite = sprite_add(async_load[? "dest_filename"], 0, false, false, 0, 0);
}
}

El código anterior comprueba el tipo de evento y luego crea un sprite a partir de la imagen descargada.
