steam_ugc_start_item_update

Esta función debe llamarse antes de añadir o actualizar la información de un elemento UGC. Debe suministrar el ID único de la aplicación de su juego en Steam, junto con el ID único del archivo publicado que se devolvió para el elemento cuando lo creó utilizando la función steam_ugc_create_item(). La función devolverá un manejador de actualización único para el elemento, que luego puedes utilizar en las funciones del elemento UGC para actualizar (o añadir) información para la carga.

Sintaxis:

steam_ugc_start_item_update(consumer_app_id, published_file_id);

Argumento Descripción
consumer_app_id El ID de aplicación único para su juego en Steam.
archivo_publicado_id El valor único del ID del archivo publicado para el artículo.

Devuelve:

Mango

Ejemplo:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "¡Mi artículo del taller(3)!");
steam_ugc_set_item_description( updateHandle, "probando taller...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags( updateHandle, tagArray);
steam_ugc_set_item_preview (updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

El código anterior obtiene el ID del juego y lo utiliza junto con el ID del archivo publicado previamente para generar un controlador de actualización para el elemento. Este manejador se utiliza entonces para actualizar varias piezas de información antes de que la actualización sea enviada a los servidores de Workshop del taller.
