steam_ugc_get_item_update_info

Esta función se puede utilizar para recuperar información sobre el estado actual de la descarga para el ID de archivo dado. Se da el ID del elemento y se proporciona el índice de un mapa DS vacío que se rellenará con los siguientes pares clave/valor si el elemento existe:

  • " needs_update" - Devolverá true o false dependiendo de si el elemento necesita una actualización o no.
  • " is_downloading" - Devolverá true o false dependiendo de si el elemento se está descargando o no.
  • "bytes_downloaded" - El número de bytes que se ha descargado.
  • " bytes_total" - El tamaño total (número de bytes) requerido para el elemento en el disco.

Si el elemento existe, la función devolverá true y rellenará el mapa, de lo contrario devolverá false y el mapa permanecerá vacío.

Sintaxis:

steam_ugc_get_item_update_info(published_file_id, info_map);

Argumento Descripción
archivo_publicado_id El ID de archivo único para el UGC que se va a comprobar.
info_mapa Un índice de mapa DS (creado previamente).

Devuelve:

Booleano

Ejemplo:

var info_map = ds_map_create();
var info = steam_ugc_get_item_update_info(global.fileID, info_map);
if info
{
draw_text(32, 15, "needs_update: " + string(info_map[? "needs_update"]));
draw_text(32, 30, "is_downloading: " + string(info_map[? "is_downloading"]));
draw_text(32, 45, "bytes_downloaded: " + string(info_map[? "bytes_downloaded"]));
draw_text(32, 60, "bytes_total: " + string(info_map[? "bytes_total"]));
}

El código anterior consultará el estado de la descarga del elemento indexado en la variable global "fileID", utilizando un Mapa DS para almacenar la información.
