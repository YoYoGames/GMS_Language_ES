steam_is_screenshot_requested

Esta función sondeará la API de Steam para ver si se ha pulsado la tecla para tomar una captura de pantalla del juego. La función sólo devolverá true durante un paso (game tick) si la tecla ha sido pulsada, y devolverá false en todas las demás tiempos.

NOTA: esto no hace una captura de pantalla por ti. Esto sólo indica que la tecla ha sido presionada y debes usar las funciones de GameMaker Studio 2 screen_save() o screen_save_part() para guardar una copia local del archivo que se va a cargar.

Sintaxis:

steam_is_screenshot_requested();

Devuelve:

Booleano

Ejemplo:

if steam_is_screenshot_requested()
{
var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
screen_save(file)
steam_send_screenshot(file, window_get_width(), window_get_height());
global.scrn_num += 1;
}

El código anterior sondeará la API de Steam en busca de una solicitud de captura de pantalla y, si lo ha sido, se generará un nombre único para el archivo de imagen, se tomará una captura de pantalla y se enviará el archivo a la página de la Comunidad de Steam para el usuario.
