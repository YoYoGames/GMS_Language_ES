steam_ugc_query_set_cloud_filename_filter

Esta función se puede utilizar para filtrar aún más cualquier consulta UGC dada, específicamente para comprobar y ver si un nombre de archivo de elemento de Taller debe coincidir o no. El manejador de la consulta es el valor devuelto al crear la consulta (usando, por ejemplo, steam_ugc_create_query_user()) y el segundo argumento es true o false, dependiendo de si requiere que los nombres de los archivos coincidan. La función devolverá true si el filtro de la consulta se ha establecido correctamente, o false en caso contrario.

Sintaxis:

steam_ugc_query_set_cloud_filename_filter(ugc_query_handle , match_cloud_filename);

Argumento Descripción
ugc_query_handle El mango de la consulta que se va a utilizar.
match_cloud_filename Establece si el nombre del archivo del elemento UGC debe coincidir o no.

Devuelve:

Booleano

Ejemplo:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_cloud_filename_filter(query_handle, true);
steam_ugc_query_add_excluded_tag(query_handle, "nasty chips");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

El código anterior crea una solicitud de consulta y almacena su manejador en una variable local para su uso futuro en el resto de las funciones que definen aún más la solicitud de consulta antes de enviarla.
