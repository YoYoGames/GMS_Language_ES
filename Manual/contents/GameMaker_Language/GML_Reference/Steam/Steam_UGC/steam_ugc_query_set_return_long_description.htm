steam_ugc_query_set_return_long_description

Esta función se puede utilizar para filtrar aún más cualquier consulta UGC dada, específicamente para recuperar el texto de la descripción larga en el evento de devolución de llamada que se disparó cuando se envió la consulta. El manejador de la consulta es el valor devuelto cuando se creó la consulta (usando, por ejemplo ejemplo, steam_ugc_create_query_user()) y el segundo argumento es true o false. La función devolverá true si el filtro de la consulta se estableció correctamente, o false en caso contrario.

Sintaxis:

steam_ugc_query_set_return_long_description(ugc_query_handle , long_description);

Argumento Descripción
ugc_query_handle El mango de la consulta que se va a utilizar.
descripción_larga Si se desea que la consulta devuelva el texto de la descripción larga.

Devuelve:

Booleano

Ejemplo:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

El código anterior crea una solicitud de consulta y almacena su manejador en una variable local para su uso futuro en el resto de las funciones que definen aún más la solicitud de consulta antes de enviarla.
