steam_ugc_query_add_required_tag

Esta función puede utilizarse para filtrar aún más cualquier consulta de CGU, concretamente para buscar sólo aquellos elementos de CGU con la etiqueta dada. El manejador de la consulta es el valor devuelto al crear la consulta (utilizando, por ejemplo, steam_ugc_create_query_user()) y el segundo argumento es una cadena que se quiere utilizar como etiqueta a incluir. La función devolverá true si el filtro de consulta se ha establecido correctamente, o false en caso contrario.

Sintaxis:

steam_ugc_query_add_required_tag(ugc_query_handle , tag_name);

Argumento Descripción
ugc_query_handle El mango de la consulta que se va a utilizar.
nombre_de_la_etiqueta El nombre de la etiqueta a incluir (una cadena).

Devuelve:

Booleano

Ejemplo:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByTrend, ugc_match_Items, 1);
steam_ugc_query_add_required_tag(query_handle, "RPG");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

El código anterior crea una solicitud de consulta y almacena su manejador en una variable local para su uso futuro en el resto de las funciones que definen aún más la solicitud de consulta antes de enviarla.
