steam_ugc_query_set_search_text

Esta función puede utilizarse para filtrar aún más cualquier consulta de CGU dada, concretamente para buscar la cadena dada en el título y la descripción de los elementos de CGU. El manejador de la consulta es el valor devuelto cuando se creó la consulta (usando, por ejemplo, steam_ugc_create_query_user()) y el segundo argumento es una cadena que se desea utilizar como término de búsqueda. La función devolverá true si el filtro de la consulta se ha establecido correctamente, o false en caso contrario.

Sintaxis:

steam_ugc_query_set_search_text(ugc_query_handle , search_text);

Argumento Descripción
ugc_query_handle El mango de la consulta que se va a utilizar.
texto_de_busqueda El texto de búsqueda a utilizar (una cadena) para la consulta.

Devuelve:

Booleano

Ejemplo:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_search_text(query_handle, "texture");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

El código anterior crea una solicitud de consulta y almacena su manejador en una variable local para su uso futuro en el resto de las funciones que definen aún más la solicitud de consulta antes de enviarla.
