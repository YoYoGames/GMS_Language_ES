steam_file_share

Con esta función puedes forzar a tu juego a sincronizar el archivo dado con la Nube de Steam. Esto no es normalmente necesario debido a que el juego se sincronizará automáticamente al final de la sesión del jugador, ni es recomendado por Steam, pero puede ser útil para asegurar que la información sensible se sincroniza inmediatamente. La función devolverá un valor 0 si falla por cualquier motivo y un valor mayor que 0 si tiene éxito.

Sintaxis:

steam_file_share(nombre de archivo);

Argumento Descripción
nombre de archivo El nombre de la sincronización del archivo.

Devuelve:

Real

Ejemplo:

if !steam_file_persisted("Save.txt")
{
steam_file_share("Guardar.txt");
}

El código anterior comprobará si un archivo ha sido almacenado en la nube de Steam, y si no lo ha hecho lo sincronizará.
