tamaño_de_archivo_de_vapor

Con esta función puedes comprobar el tamaño de un archivo almacenado en Steam Cloud. El número real devuelto es el tamaño, en bytes, del archivo.

Sintaxis:

steam_file_size(filename);

Argumento Descripción
nombre de archivo El nombre del archivo para comprobar el tamaño.

Devuelve:

Real

Ejemplo:

file_bytes = steam_file_size("Save.txt");

El código anterior almacena el tamaño de un archivo de Steam Cloud en la variable "file_bytes".
