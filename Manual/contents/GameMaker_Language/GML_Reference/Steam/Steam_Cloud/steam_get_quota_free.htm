steam_get_quota_free

Con esta función puedes saber cuánto espacio libre le queda al usuario de la cuota de Steam Cloud. El valor devuelto está en bytes.

Sintaxis:

steam_get_quota_free();

Devuelve:

Real

Ejemplo:

if steam_is_cloud_enabled_for_app()
{
quota = steam_get_quota_free();
}

El código anterior comprueba si la nube de vapor está habilitada para el juego y si es así obtiene el tamaño del espacio de almacenamiento libre y lo almacena en una variable.
