steam_file_persisted

Con esta función puedes comprobar el archivo dado para ver si se ha sincronizado con la nube de Steam. Un valor de retorno de true significa que lo está, mientras que false significa que no lo está.

Sintaxis:

steam_file_persisted(filename);

Argumento Descripción
nombre de archivo El nombre del archivo a comprobar.

Devuelve:

Booleano

Ejemplo:

if !steam_file_persisted("Save.txt")
{
steam_file_share("Guardar.txt");
}

El código anterior comprobará si un archivo ha sido almacenado en la nube de Steam, y si no lo ha hecho lo sincronizará.
