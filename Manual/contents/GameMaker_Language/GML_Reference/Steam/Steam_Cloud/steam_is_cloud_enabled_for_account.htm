steam_is_cloud_enabled_for_account

Con esta función puedes comprobar que el servicio Steam Cloud está habilitado por el usuario en la configuración de su Cliente Steam. Devolverá true si lo está y false en caso contrario. Ten en cuenta que esto no significa automáticamente que puede almacenar datos en la Nube, ya que también tendrá que haber sido habilitado para su juego (puede comprobarlo utilizando la función steam_is_cloud_enabled_for_app()). Si la Nube de Steam está habilitada para tu juego, pero el usuario la tiene desactivada localmente, puedes seguir utilizando las funciones de la Nube para almacenar y recuperar datos de una copia local de todos los archivos, sólo que no los subirá a la Nube al finalizar el juego, ni se sincronizará al iniciarlo.

Sintaxis:

steam_is_cloud_enabled_for_account();

Devuelve:

Booleano

Ejemplo:

si steam_is_cloud_enabled_for_account
{
steam_file_share("Save.txt");
}

El código anterior comprueba si el usuario tiene activada la nube de Steam y, si devuelve true, sincronizará el archivo dado.
