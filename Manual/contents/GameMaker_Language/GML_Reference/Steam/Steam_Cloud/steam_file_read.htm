steam_file_read

Esta función leerá el contenido del archivo dado en una cadena que luego puede ser analizada en su juego.

Sintaxis:

steam_file_read(filename);

Argumento Descripción
nombre de archivo El nombre del archivo del que se va a leer.

Devuelve:

Cadena

Ejemplo:

if steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt");
}

El código anterior comprueba si existe un archivo en la Nube de Steam y, si es así, lo abre y lee su contenido en la variable "save_str".
