Nube devapor

La Nube de Steam proporciona un sistema de almacenamiento de archivos remoto fácil y transparente para tu juego. Todos los archivos que se escriban en el disco utilizando las funciones de la nube de GameMaker Studio 2 (que se enumeran a continuación) se replicarán en los servidores de Steam una vez que se cierre el juego. Si en el usuario cambia de ordenador, los archivos se descargarán en el nuevo ordenador antes de que se inicie el juego, lo que significa que el juego puede acceder a los archivos leyéndolos mediante las funciones de Steam adecuadas. El cliente de Steam se encarga de de asegurar que los archivos se mantienen sincronizados en todos los ordenadores a los que el usuario pueda acceder.

NOTA: Por defecto, la Nube no está habilitada para un juego en Steamworks. Debe habilitarse previamente desde la pestaña 'Nube' del administrador del juego de Steamworks, donde debes establecer la cuota de bytes y archivos. La próxima vez que publiques tu juegos la configuración de Steamworks, el almacenamiento en la Nube estará listo para ser utilizado.

Las siguientes funciones pueden utilizarse para acceder a Steam Cloud desde GameMaker Studio 2

  • steam_is_cloud_enabled_for_app
  • steam_is_cloud_enabled_for_account
  • steam_get_quota_total
  • steam_get_quota_free
  • steam_file_exists
  • tamaño_de_archivo_de_vapor
  • steam_file_persisted
  • steam_file_write
  • steam_file_write_file
  • steam_file_read
  • steam_file_share
  • steam_file_delete
