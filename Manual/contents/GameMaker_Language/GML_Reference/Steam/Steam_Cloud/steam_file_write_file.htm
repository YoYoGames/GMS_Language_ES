steam_file_write_file

Con esta función puedes copiar el contenido de un archivo guardado localmente a un archivo sincronizado con la Nube de Steam. El archivo local debe existir antes de usar esta función, y devolverá un valor de 0 si falla por cualquier razón y un valor mayor que 0 si tiene éxito.

Sintaxis:

steam_file_write_file(steam_filename, local_filename);

Argumento Descripción
nombre_de_vapor El archivo de Steam Cloud a copiar.
nombre_de_archivo_local El archivo local que se utilizará para copiar.

Devuelve:

real

Ejemplo:

steam_file_write_file("rm_koala.png", "Koala2.png");

El código anterior copiará el contenido del archivo "Koala2.png" al archivo de Steam Cloud "rm_koala.png".
