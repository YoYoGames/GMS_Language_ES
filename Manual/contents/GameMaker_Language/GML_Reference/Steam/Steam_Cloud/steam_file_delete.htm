steam_file_delete

Esta función eliminará el archivo dado de la nube de Steam. La función devolverá un valor de 0 si falla por cualquier razón y un valor mayor que 0 si tiene éxito.

Sintaxis:

steam_file_delete(filename);

Argumento Descripción
nombre de archivo El nombre de la eliminación del archivo.

Devuelve:

Real

Ejemplo:

if steam_file_exists("Save.txt")
{
steam_file_delete("Save.txt");
}

El código anterior comprobará si un archivo existe, y si lo hace, elimina el archivo de la nube de Steam.
