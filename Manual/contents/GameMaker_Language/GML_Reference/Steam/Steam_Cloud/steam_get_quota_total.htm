steam_get_quota_total

Cuando utilices Steam Cloud para almacenar y sincronizar archivos, debes establecer la cuota de espacio que necesitará tu juego. Esta cuota se aplica a cada juego habilitado para la Nube, en función de cada usuario por juego, así que, por ejemplo, si la cuota para el Juego X es de 1 megabyte, entonces cada cuenta de Steam que posea el Juego X podrá almacenar, como máximo, 1 megabyte de datos asociados a ese juego en la Nube. Cualquier otro juego habilitado para la Nube que posea el usuario (por ejemplo, el Juego Y) no se verá afectado por los datos almacenados por el Juego X. La cuota por defecto para los nuevos juegos de Steamworkses de un gigabyte, pero puedes cambiarla desde el panel de control de Steamworkspara tu juego.

NOTA: Una vez agotada la cuota, las escrituras de archivos fallarán. Si crees que es posible que la cuota se agote para el usuario de tu juego, debes crear un código para manejarlo, ya que al no hacer nada dejas a los usuarios en una situación en la que no pueden arreglar las cosas y eso llevará a una mala experiencia de su juego.

Sintaxis:

steam_get_quota_total();

Devuelve:

Real

Ejemplo:

if steam_is_cloud_enabled_for_app()
{
quota = steam_get_quota_total();
}

El código anterior comprueba si la nube de vapor está habilitada para el juego y si es así obtiene el tamaño de la cuota de almacenamiento y lo almacena en una variable.
