steam_file_exists

Con esta función puedes comprobar si un archivo de la nube de Steam existe o no, con un valor de retorno de true si existe, o false en caso contrario.

Sintaxis:

steam_file_exists(filename);

Argumento Descripción
nombre de archivo El nombre del archivo a comprobar.

Devuelve:

Booleano

Ejemplo:

if steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt");
}

El código anterior comprueba si existe un archivo en la Nube de Steam y, si es así, lo abre y lee su contenido en la variable "save_str".
