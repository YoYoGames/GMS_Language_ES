steam_is_overlay_activated

Esta función se puede utilizar para averiguar si el usuario tiene la superposición de vapor activa o no. Si la superposición está activa y visible para el usuario la función devolverá true, y si no lo está, entonces devolverá false. Un ejemplo de lo que esta función puede ser utilizada para el sondeo de la API de Steam para la superposición de modo que usted puede pausar su juego, mientras que la superposición se muestra.

Sintaxis:

steam_is_overlay_activated();

Devuelve:

Booleano

Ejemplo:

if steam_is_overlay_activated()
{
global.Pause = true;
}

El código anterior comprobará si la superposición de Steam está activa y, si lo está, establecerá la variable global "Pausa" en true.
