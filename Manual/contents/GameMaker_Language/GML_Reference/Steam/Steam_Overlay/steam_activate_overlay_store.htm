steam_activate_overlay_store

Con esta función puedes abrir la superposición de Steam en la página de la tienda de un juego para que los usuarios puedan comprar o descargar DLC (por ejemplo). Es necesario suministrar el ID de la aplicación única para el juego o DLC que se obtiene del panel de control de Steam cuando se configura. de Steam al configurarlo.

Sintaxis:

steam_activate_overlay_store(app_id);

Argumento Descripción
app_id El identificador único de la aplicación para su juego.

Devuelve:

N/A

Ejemplo:

si keyboard_check_pressed(vk_f1)
{
steam_activate_overlay_store(global.DLC_id);
}

El código anterior sondea el teclado en busca de la tecla F1 y, si lo hace, se abrirá la superposición de Steam en la página para el contenido del juego utilizando el ID de la aplicación almacenado en la variable global.
