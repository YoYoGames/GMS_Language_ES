steam_activate_overlay_user

Esta función abrirá la superposición de Steam a uno de los diálogos elegidos relacionados con el ID de usuario dado. Las posibles cadenas de diálogo disponibles son:

  • " steamid" - abre el navegador web de la Comunidad Steam a la página del usuario o grupo
  • " chat"- abre una ventana de chat al usuario especificado

Tenga en cuenta que los ID de vapor pueden ser números grandes, por lo que es posible que tenga que convertir su valor de ID en un int64() antes de proporcionarlo a la función.

Sintaxis:

steam_activate_overlay_user(dialog_name, steamid);

Argumento Descripción
nombre_diálogo El diálogo en el que se abre la superposición (véase más abajo).
steamid El ID de usuario de Steam o el ID de grupo a utilizar.

Devuelve:

N/A

Ejemplo:

var key = keyboard_lastkey;
switch (key)
{
case vk_f1: steam_activate_overlay_user(" steamid",global.GameGroupID); break;
case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

El código anterior sondea la última tecla del teclado presionada y si es la tecla de función 1 o la tecla de función 2, abrirá la superposición de Steam para ver el grupo de Steam almacenado en la variable global "GamegroupID", o abrirá la ventana de chat para chatear con el usuario almacenado en la variable global "FriendID".
