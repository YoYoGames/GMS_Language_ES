Vapor

Esta sección es para aquellos usuarios a los que se les ha dado acceso a la API de Steam para publicar su juego en dicha plataforma. Para poder utilizar estas funciones debes haber sido aceptado en Steam previamente, ya sea a través de un editor o mediante el sistema de autopublicación, y haber configurado correctamente las Preferencias deSteam y las Opciones de Juegode Steam.

Hay un gran número de funciones diferentes relacionadas con la API de Steam, por lo que las hemos dividido en las siguientes secciones para facilitar la navegación:

  • API general de vapor
  • La superposición de vapores
  • Tablas de clasificación
  • Logros y estadísticas
  • Nube devapor
  • DLC (contenido descargable)
  • UGC (Contenido Generado por el Usuario)
