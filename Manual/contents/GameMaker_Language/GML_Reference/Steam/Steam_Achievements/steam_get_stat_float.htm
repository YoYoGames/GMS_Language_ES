steam_get_stat_float

Con esta función puedes obtener el valor de una estadística específica de punto flotante. La estadística debe haber sido definida previamente en la página de cuentas del panel de control de Steamworkspara tu juego y la cadena que se pasa a la función debe coincidir con la utilizada como nombre de la API en el panel de control.

Sintaxis:

steam_get_stat_float(stat_name);

Argumento Descripción
nombre_de_estado El nombre de la estadística a obtener (una cadena).

Devuelve:

Real

Ejemplo:

var dist_pc = (dist / dist_max) * 100;
if steam_get_stat_float("Travelled") < dist_pc
{
steam_set_stat_int("Travelled", dist_pc);
}

El código anterior calcula un porcentaje basado en la variable de distancia recorrida "dist" y la distancia máxima que se puede recorrer "dist_max". A continuación, sondea el valor actual de la estadística "Recorrido" y si es menor que el valor calculado, establece la estadística de nuevo.
