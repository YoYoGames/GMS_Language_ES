steam_get_stat_avg_rate

Con esta función puedes obtener el valor de una estadística media específica. La estadística debe haber sido definida previamente en la página de cuentas del panel de control de Steamworkspara tu juego y la cadena que se pasa a la función debe coincidir con con la utilizada como nombre de la API en el panel de control.

Sintaxis:

steam_get_stat_avg_rate(stat_name);

Argumento Descripción
nombre_de_estado El nombre de la estadística a obtener (una cadena).

Devuelve:

Real

Ejemplo:

var avg = steam_get_stat_avg_rate("PointsPerHour");
draw_text(8, 8, "PPH = " + string(avg);

El código anterior obtiene el valor actual de la estadística media "PointsPerHour" y lo dibuja en la pantalla.
