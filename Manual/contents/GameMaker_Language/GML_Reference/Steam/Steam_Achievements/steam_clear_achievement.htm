steam_clear_achievement

Con esta función puedes decirle a la API de Steam que borre (restablezca) un logro específico. El logro debe haber sido definido previamente en la página de cuentas del panel de control de Steamworkspara tu juego y la cadena que se pasa a la función debe coincidir con la utilizada como nombre de la API en el panel de control.

Sintaxis:

steam_clear_achievement(ach_name);

Argumento Descripción
nombre_de_la_actividad El nombre del logro a borrar (cadena).

Devuelve:

N/A

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
steam_clear_achievement("Ach_Game_Win");
steam_clear_achievement("Ach_Died_10_Times");
steam_clear_achievement("Ach_Killed_100_Enemies");
steam_clear_achievement("Ach_Beat_Boss_Level_1");
}

El código anterior restablecerá los logros del juego cuando el usuario haga clic en el botón izquierdo del ratón.
