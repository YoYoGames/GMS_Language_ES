steam_set_achievement

Con esta función puedes indicar a la API de Steam que conceda ("establezca") un logro para el jugador. Estos logros deben haber sido definidos en la página de cuentas del panel de control de Steamworkspara tu juego y la cadena que se pasa a la función debe coincidir con la utilizada como nombre de la API en el panel de control. El Steam Game Overlay mostrará un panel de notificación al usuario informándole del logro que ha recibido, a menos que el logro ya haya sido concedido, en en cuyo caso no ocurrirá nada.

Sintaxis:

steam_set_achievement(ach_name);

Argumento Descripción
nombre_de_la_actividad El nombre del logro a establecer (cadena).

Devuelve:

N/A

Ejemplo:

si hp <= 0
{
global.Muertes += 1;
if global.Muertes == 10
{
if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
}
}

El código anterior premiará al jugador con un logro si la variable global "Muertes" es igual a 10 y si el logro no ha sido ya concedido.
