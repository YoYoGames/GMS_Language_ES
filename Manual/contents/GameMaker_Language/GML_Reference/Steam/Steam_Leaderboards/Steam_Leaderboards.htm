La API de Steam soporta tablas de clasificación persistentes con entradas ordenadas automáticamente. Estas tablas de clasificación pueden utilizarse para mostrar tablas de clasificación globales y de amigos en tu juego y en la página web de la comunidad de tu juego. Cada juego puede tener hasta 10.000 tablas de clasificación, y cada tabla de clasificación puede ser recuperada inmediatamente después de que la puntuación de un jugador haya sido insertada en ella, pero tenga en cuenta que para cada tabla de clasificación, un jugador sólo puede tener una entrada, aunque no hay límite en el número de jugadores por tabla de clasificación.

Cada entrada de la tabla de clasificación contiene un nombre, una puntuación y un rango para la tabla de clasificación, y estos datos serán reemplazados cuando se cree una nueva entrada de la tabla de clasificación para el usuario, y las siguientes funciones se pueden utilizar para añadir y recuperar estos datos de las tablas de clasificación de tu juego:

  • steam_create_leaderboard
  • steam_upload_score
  • steam_upload_score_buffer
  • steam_download_scores
  • steam_download_scores_around_user
  • steam_download_friends_scores
