steam_user_owns_dlc

Si tu juego tiene DLC creado para él, puedes utilizar esta función para comprobar si el usuario lo ha comprado antes de acceder a cualquier archivo asociado a él. La función devuelve true si el jugador posee el contenido, y false si el no lo tiene, pero tenga en cuenta que aunque el usuario sea propietario del DLC, puede que no lo tenga instalado, por lo que debe utilizar la función adicional de steam_user_installed_dlc() para comprobar que está instalado también antes de usarlo. Tenga en cuenta que los IDs de Steam pueden ser números grandes, por lo que puede ser necesario lanzar el valor de su ID como un int64() antes de proporcionarlo a la función.

Sintaxis:

steam_user_owns_dlc(dlc_id);

Argumento Descripción
dlc_id El identificador único del DLC que se va a comprobar (un entero int64).

Devuelve:

Booleano

Ejemplo:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
if steam_user_installed_dlc(10354)
{
global.Level_max = 200;
}
}

El código anterior comprobará si el usuario ha comprado, e instalado, el DLC con el id 10354, y si es así establecerá una variable global con un valor diferente.
