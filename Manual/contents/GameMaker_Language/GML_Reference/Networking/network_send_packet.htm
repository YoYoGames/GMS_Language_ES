network_send_packet

Con esta función puedes enviar un "paquete" de datos a través de la red. La función toma el id del socket a través del cual se conecta y luego debe suministrar el id del buffer que contiene los datos a enviar (para más información sobre los buffers ver Referencia - Bufferes) y finalmente el tamaño (en bytes) del paquete de datos. Los paquetes enviados con esta función tienen un formato tal que el juego GameMaker Studio 2 que recibe los datos puede "dividir" los paquetes correctamente, y la función devolverá el número de bytes de datos enviados, o un número menor que 0 si el envío ha fallado. Cabe destacar que el tamaño final de los datos enviados que devuelve esta función también incluirá la información de la cabecera de información de la cabecera de GameMaker, que son 12 bytes adicionales.

Sintaxis:

network_send_packet(socket, buffer, size);

Argumento Descripción
toma de corriente El id del socket a utilizar.
buffer El id del buffer del que se van a obtener los datos.
tamaño El tamaño (en bytes) de los datos.

Devuelve:

Real

Ejemplo:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

La información anterior carga los datos de un buffer previamente guardado en la memoria y devuelve el id del buffer para ser almacenado en la variable "buff". Este buffer completo se envía entonces como un paquete a través de la red utilizando el socket identificado por la variable "sock".
