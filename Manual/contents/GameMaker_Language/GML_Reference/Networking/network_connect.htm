conexión_de_red

Con esta función se puede enviar una petición para conectarse a un servidor. La función toma el id del socket para conectarse (ver network_create_socket()) y requiere que se le dé la dirección IP a la que conectarse (una cadena) así como el puerto a través del cual conectarse, y si la conexión falla se devolverá un valor inferior a 0. La conexión utiliza un protocolo especial que asegura que sólo los juegos de GameMaker Studio 2 se conectan entre sí, sin embargo, si usted necesita para conectarse a un servidor que no es un juego de GameMaker Studio 2, puede utilizar network_connect_raw(). Tenga en cuenta que por defecto la función es sincrónica, lo que significa que su juego puede parecer "colgado" mientras se realiza la conexión. Puede establecer un valor de tiempo de espera para la conexión, o alternativamente hacer la creación asíncrona, utilizando la función network_set_config()o, alternativamente, utilizar la función network_connect_async() en su lugar.

Sintaxis:

network_connect(socket, url, port);

Argumento Descripción
toma de corriente El id del socket a utilizar.
url La URL o IP a la que conectarse (una cadena).
puerto El puerto al que hay que conectarse.

Devuelve:

Real

Ejemplo:

cliente = network_create_socket(network_socket_tcp);
network_connect(cliente, "192.134.0.1", 6510);

El código anterior creará un nuevo socket TCP e intentará conectarse a través de él a la dirección IP indicada en el puerto 6510.
