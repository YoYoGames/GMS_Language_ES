network_set_timeout

Con esta función puedes establecer el tiempo de espera para leer y escribir datos a/desde un servidor a través del socket dado. Tenga en cuenta que el tiempo de espera no genera ningún tipo de evento, por lo que tendrá que ocuparse usted mismo de los tiempos de espera utilizando alarmas (por ejemplo). Tenga en cuenta que este valor sólo afecta al envío y la recepción de datos, y si desea cambiar el valor del tiempo de espera de la conexión, deberá utilizar la función network_set_config().

Sintaxis:

network_set_timeout(socket, read_timeout, write_timeout);

Argumento Descripción
toma de corriente El id del socket a utilizar.
read_timeout Los milisegundos en los que se agotará el tiempo de espera de una transferencia desde un servidor.
write_timeout Los milisegundos en los que se agotará el tiempo de transferencia a un servidor.

Devuelve:

N/A

Ejemplo:

network_set_timeout(sock, 3000, 3000);

El código anterior establecerá el tiempo de espera para leer y escribir datos a través del socket indexado en la variable "sock" a 3 segundos.
