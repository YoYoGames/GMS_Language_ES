network_create_socket_ext

Esta función se utiliza para crear un nuevo socket de cliente para que su juego se comunique a través de la red. Debes definir el protocolo del socket (ver la lista de constantes más abajo) y dar un puerto a utilizar, y la función devolverá un id único que debe que debe ser utilizado en todas las llamadas de función posteriores para ese socket, o un valor inferior a 0 si la conexión falla.

Constante Descripción
network_socket_tcp Crear un socket usando TCP.
network_socket_udp Crear un socket usando UDP.
network_socket_ws Crear un socket web(sólo para proyectos que se ejecutan en HTML5), utilizando TCP.

Sintaxis:

network_create_socket_ext(protocolo, puerto);

Argumento Descripción
protocolo El protocolo de red a utilizar
puerto El puerto a utilizar

Devuelve:

Real

Ejemplo:

cliente = network_create_socket_ext(network_socket_udp, 6510);

El código anterior creará un nuevo socket UDP en el puerto 6510.
