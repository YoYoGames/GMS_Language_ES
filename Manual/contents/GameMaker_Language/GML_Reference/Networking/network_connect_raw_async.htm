network_connect_raw_async

Con esta función se puede enviar una petición para conectarse a un servidor. La función toma el id del socket para conectarse (ver network_create_socket()) y requiere que le dé la dirección IP a la que se va a conectar (una cadena) así como el puerto para conectarse, y si la conexión falla se devolverá un valor inferior a 0. La diferencia entre esta función y network_connect_async()es que esta función puede conectarse a cualquier servidor y no hace nada con los datos en bruto, lo que significa que tienes que implementar los protocolos tú mismo en el extremo del servidor. Tenga en cuenta que esta función es asíncrona, generando un evento de red asíncrono del tipo network_type_non_blocking_connect.

Sintaxis:

network_connect_raw_async(socket, url, port);

Argumento Descripción
toma de corriente El id del socket a utilizar.
url La URL o IP a la que conectarse (una cadena).
puerto El puerto al que hay que conectarse.

Devuelve:

Real

Ejemplo:

cliente = network_create_socket(network_socket_tcp);
network_connect_raw_async(cliente, "www.macsweeneygames.com", 6510);

El código anterior creará un nuevo socket TCP y luego intentará crear una conexión asíncrona "cruda" a través de él a la URL dada en el puerto 6510.
