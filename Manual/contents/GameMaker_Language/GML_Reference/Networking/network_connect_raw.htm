network_connect_raw

Con esta función se puede enviar una petición para conectarse a un servidor. La función toma el id del socket para conectarse (ver network_create_socket()) y requiere que se le dé la dirección IP a la que conectarse (una cadena) así como el puerto a través del cual conectarse, y si la conexión falla se devolverá un valor inferior a 0. La diferencia entre esta función y network_connect() es que esta función puede conectarse a cualquier servidor y no hace nada con los datos en bruto, lo que significa que tienes que implementar los protocolos tú mismo en el extremo del servidor. Tenga en cuenta que por defecto la función es sincrónica, lo que significa que su juego puede parecer que se "cuelga" mientras se realiza la mientras se realiza la conexión. Puedes establecer un valor de tiempo de espera para la conexión, o alternativamente hacer la creación asíncrona, usando la función network_set_config()o, alternativamente, utilizar la función network_connect_async_raw() en su lugar.

Sintaxis:

network_connect_raw(socket, url, port);

Argumento Descripción
toma de corriente El id del socket a utilizar.
url La URL o IP a la que conectarse (una cadena).
puerto El puerto al que hay que conectarse.

Devuelve:

Real

Ejemplo:

cliente = network_create_socket(network_socket_tcp);
network_connect_raw(cliente, "www.macsweeneygames.com", 6510);

El código anterior creará un nuevo socket TCP y luego intentará crear una conexión "cruda" a través de él a la URL dada en el puerto 6510.
