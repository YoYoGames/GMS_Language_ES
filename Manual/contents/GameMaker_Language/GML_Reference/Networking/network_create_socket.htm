network_create_socket

Esta función se utiliza para crear un nuevo socket de cliente para que su juego se comunique a través de la red. Debes definir el tipo de socket (ver la lista de constantes más abajo) y la función devolverá un id único para ese socket, que debe ser usado en todas las llamadas posteriores a la función para ese socket, o un valor inferior a 0 si la conexión falla.

NOTA: Puedes utilizar la IP "127.0.0.1" para conectarte de nuevo al mismo dispositivo que está ejecutando el juego.

Constante Descripción
network_socket_tcp Crear un socket usando TCP.
network_socket_udp Crear un socket usando UDP.
network_socket_ws Crear un socket web(sólo para proyectos que se ejecutan en HTML5), utilizando TCP.
network_socket_bluetooth Crear una toma bluetooth(¡actualmente no disponible!).

Sintaxis:

network_create_socket(type);

Argumento Descripción
tipo El tipo de conexión de socket que se va a crear (ver las constantes que se indican a continuación).

Devuelve:

Real

Ejemplo:

si os_browser == browser_not_a_browser
{
cliente = network_create_socket(network_socket_tcp);
network_connect( cliente, "192.134.0.1", 6510 );
}
else
{
client = network_create_socket(network_socket_ws);
network_connect( client, "192.134.0.1", 6520 );
}

El código anterior comprobará si el juego se está ejecutando en un navegador o no y creará un nuevo socket TCP o Web antes de intentar conectarse a través de él a la dirección IP dada en el puerto dado.
