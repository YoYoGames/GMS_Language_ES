red_resuelta

Esta función devolverá la dirección IP de la URL dada.

Sintaxis:

network_resolve(url);

Argumento Descripción
url La URL para obtener la IP de (una cadena).

Devuelve:

Cadena

Ejemplo:

game_ip = network_resolve("www.macsweeneygames.com");

El código anterior devolverá la dirección IP de la URL dada y la almacenará en la variable "game_ip".
