network_send_raw

Con esta función puedes enviar un paquete de datos "en bruto" a través de la red. La función toma el id de socket para conectarse y luego debe suministrar el id de buffer que contiene los datos a enviar (para más información sobre buffers ver Referencia - Bufferes) y finalmente el tamaño (en bytes) del paquete de datos. Los datos enviados no son formateados por GameMaker Studio 2 de ninguna manera y los dispositivos receptores obtendrán los datos como un flujo que significa que tendrá que manejarlo usted mismo. La función devolverá el número de bytes de datos enviados, o un número menor que 0 si el envío ha fallado.

Sintaxis:

network_send_raw(socket, buffer, size);

Argumento Descripción
toma de corriente El id del socket a utilizar.
buffer El id del buffer del que se van a obtener los datos.
tamaño El tamaño (en bytes) de los datos.

Devuelve:

Real

Ejemplo:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

La información anterior carga los datos de un búfer previamente guardado en la memoria y devuelve el id del búfer para ser almacenado en la variable "buff". Este búfer completo se envía entonces como un paquete de datos en bruto a través de la red utilizando el socket identificado por la variable "sock".
