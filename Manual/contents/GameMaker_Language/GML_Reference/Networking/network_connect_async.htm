network_connect_async

Con esta función puedes enviar una petición para conectarte a un servidor. La función toma el id del socket para conectarse (ver network_create_socket()) y requiere que se le dé la dirección IP a la que se va a conectar (una cadena) así como el puerto a través del cual conectarse, y si la conexión falla se devolverá un valor inferior a 0. La conexión utiliza un protocolo especial que asegura que sólo los juegos de GameMaker Studio 2 se conectan entre sí, sin embargo, si usted necesita para conectarse a un servidor que no es un juego de GameMaker Studio 2, puede utilizar network_connect_async_raw(). Tenga en cuenta que esta función es asíncrona, generando un evento de red asíncrona del tipo network_type_non_blocking_connect.

Sintaxis:

network_connect_async(socket, url, port);

Argumento Descripción
toma de corriente El id del socket a utilizar.
url La URL o IP a la que conectarse (una cadena).
puerto El puerto al que hay que conectarse.

Devuelve:

Real

Ejemplo:

cliente = network_create_socket(network_socket_tcp);
network_connect_async(cliente, "192.134.0.1", 6510);

El código anterior creará un nuevo socket TCP y luego intentará hacer una conexión asíncrona a través de él a la dirección IP dada en el puerto 6510.
