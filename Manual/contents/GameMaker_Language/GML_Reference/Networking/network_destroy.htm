red_destrucción

Con esta función puedes eliminar una conexión de red de tu juego.

NOTA: Esta función no funcionará cuando se utilice en un proyecto que se ejecute en el objetivo HTML5.

Sintaxis:

network_destroy(socket);

Argumento Descripción
toma de corriente El id del enchufe a eliminar.

Devuelve:

N/A

Ejemplo:

if !os_is_network_connected()
{
network_destroy(sock);
}

El código anterior comprobará si hay una conexión de datos y si no se encuentra ninguna, destruirá el socket con el id "sock".
