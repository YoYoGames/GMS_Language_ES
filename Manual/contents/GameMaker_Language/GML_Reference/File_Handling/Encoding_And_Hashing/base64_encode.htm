base64_encode

Esta función convertirá una cadena en una cadena codificada en formato base64. Este es un esquema de codificación comúnmente utilizado para cualquier medio que necesite ser almacenado o transferido a través de Internet como texto, y hace que la salida sea ilegible para el ojo humano. el ojo humano.

Sintaxis:

base64_encode(string)

Argumento Descripción
cadena La cadena a codificar.

Devuelve:

Cadena

Ejemplo:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

El código anterior convertirá la cadena almacenada en "game_data" en una cadena codificada en base64 que se almacenará en un archivo de texto externo.
