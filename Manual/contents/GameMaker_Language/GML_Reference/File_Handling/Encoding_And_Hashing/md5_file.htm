archivo_md5

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se utiliza habitualmente para comprobar la integridad de archivos y cadenas de caracteres. Esta función tomará el archivo dado y generará un MD5 único para ese archivo que puede ser almacenado para su uso posterior.

NOTA: MD5 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

md5_file(nombre de archivo)

Argumento Descripción
nombre de archivo El archivo para el que se generará el hash MD5.

Devuelve:

Cadena

Ejemplo:

hash = md5_file(directorio_de_trabajo + "game_data.ini")

El código anterior generará un hash MD5 para el archivo especificado y almacenará el valor devuelto en la variable "hash".
