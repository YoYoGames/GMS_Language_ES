zip_unzip

Esta función abrirá un archivo ZIP almacenado y extraerá su contenido en el directorio dado. Tenga en cuenta que si no proporciona una ruta completa al directorio ZIP, se utilizará la raíz de la unidad actual, y si desea colocarlo en una ruta relativa al directorio de trabajo del paquete del juego, deberá utilizar la variable directorio_de_trabajo variable como parte de la ruta (las rutas relativas que utilizan "." o ".." no funcionarán como se espera por lo que deben evitarse). Tenga en cuenta también que el zip debe ser parte del paquete del juego (es decir, un archivo incluido) o haber sido descargado en el área de almacenamiento utilizando http_get_file().

La función devolverá un valor que indica el número de archivos extraídos, o devolverá 0 o menos si la extracción ha fallado.

Sintaxis:

zip_unzip(archivo_zip, directorio_destino)

Argumento Descripción
archivo_de_cierre El archivo zip a abrir
directorio_de_objetivo El directorio de destino para extraer los archivos

Devuelve:

Real

Ejemplo:

var num = zip_unzip("/descargas/nivel_datos.zip", directorio_de_trabajo + "extraído/");
if num <= 0
{
show_debug_message("¡Fallo en la extracción!");
}

El código anterior abrirá el archivo zip almacenado en el directorio "descargas" y extraerá su contenido al directorio "extraído" (creando ese directorio si no existe ya) y luego comprobará que la extracción ha ha sido correcta, mostrando un mensaje de depuración en caso de que falle.
