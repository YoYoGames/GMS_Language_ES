md5_string_unicode

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se utiliza habitualmente para comprobar la integridad de archivos y cadenas de caracteres. Esta función tomará una cadena unicode de entrada (que tiene 16bits por cada char) y devolverá el hash MD5 de 32 caracteres hexadecimales que es único para esa cadena. De esta manera se puede generar una clave segura que puede ser almacenada y utilizada para comprobar la integridad de la información que se envía a (o se recibe de) un servidor externo (por ejemplo).

NOTA: Hay dos formatos para la codificación MD5, UTF-8 y unicode. Ambos se proporcionan para facilitar la comunicación con diferentes configuraciones de servidor, pero el más común de usar es unicode.

NOTA: MD5 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

md5_string_unicode(cadena)

Argumento Descripción
cadena La cadena a hash.

Devuelve:

Cadena

Ejemplo:

var hash, str;
str = base64_encode(game_data);
hash = md5_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

El código anterior codificará en base64 una cadena y luego generará un hash MD5. Finalmente, tanto el hash como la cadena codificada se envían a un servidor.
