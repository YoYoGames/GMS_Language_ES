ini_read_real

Puede utilizar esta función para leer un número de un archivo de datos ini. Los archivos ini se dividen en secciones y luego cada sección se divide en pares clave-valor. Por lo tanto, un archivo ini típico sería algo así:

Sintaxis:

ini_read_real(section, key, default);

Argumento Descripción
sección La sección del .ini para leer.
clave La clave dentro de la sección relevante del .ini para leer.
por defecto El valor a devolver si no se encuentra un valor en el lugar definido (o el archivo .ini no existe). Debe ser un número real.

Devuelve:

Real

Ejemplo:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Esto abrirá "savedata.ini" y establecerá la puntuación al valor bajo "save1" > "score" en él, luego cerrará el .ini de nuevo. Si no hay ningún valor en "save1" > "score", o no hay ningún archivo "savedata.ini" la puntuación se ajustará a 0.
