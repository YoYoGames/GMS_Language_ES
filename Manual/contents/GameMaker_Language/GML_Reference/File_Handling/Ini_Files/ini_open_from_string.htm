ini_open_from_string

Esta función creará un archivo ini a partir de una cadena y lo abrirá para su lectura/escritura. La cadena debe estar correctamente formateada como un archivo ini (es decir, con secciones, claves de saltos de línea y valores), de lo contrario el archivo ini no se creará correctamente. Tenga en cuenta que este archivo ini es temporal y se eliminará de la memoria en el momento en que se cierre, perdiendo cualquier información que se haya almacenado en él, sin embargo la función ini_close() función devuelve una cadena del archivo ini completo que puede ser guardado en un servidor o en el disco.

Sintaxis:

ini_open_from_string(cadena);

Argumento Descripción
cadena La cadena que contiene toda la información ini.

Devuelve:

N/A

Ejemplo:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

El código anterior establece un ini temporal para mantener la información de la cadena "str", luego lee del ini antes de cerrarlo de nuevo.
