ini_write_string

Puede utilizar esta función para escribir una cadena (texto) en un archivo de datos ini. Los archivos ini se dividen en secciones y luego cada sección se divide en pares clave - valor. Por lo tanto, un archivo ini típico se vería algo así:

Vale la pena señalar que el uso de comillas y caracteres de escape al escribir cadenas ini puede causar problemas cuando más tarde leer los datos, ya que pueden ser truncados prematuramente. Por ejemplo, escribir esto será un problema:

ini_write_string("test2", "section", "hello \"Fritz\");

Sintaxis:

ini_write_string(section, key, value);

Argumento Descripción
sección La sección del .ini en la que se debe escribir.
clave La clave dentro de la sección relevante del .ini para escribir.
valor La cadena a escribir en el destino correspondiente.

Devuelve:

N/A

Ejemplo:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

El código anterior abre un archivo ini para leer y escribir, luego escribe la cadena almacenada en la variable global "Nombre" en la sección "Guardar" con la clave "Reproductor" antes de cerrar el archivo de nuevo.
