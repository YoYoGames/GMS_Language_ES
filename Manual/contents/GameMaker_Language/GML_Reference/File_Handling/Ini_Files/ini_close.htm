ini_close

Esta función debe ser llamada en el momento en que se termina de leer o escribir en cualquier archivo ini abierto. Si no utiliza la función después de haber utilizado cualquiera de las funciones de escritura ini, entonces no se escribirá nada en el disco, ya que la información del archivo se mantiene en la memoria hasta que se llama a esta función, que fuerza la escritura. Si intentas abrir un ini sin haber cerrado previamente otro (o el mismo) también obtendrás un error.

La función también devolverá una cadena con el archivo ini codificado en ella. Esta cadena puede guardarse en un servidor y/o utilizarse de nuevo junto con la función ini_open_from_string() para volver a crear el ini.

Sintaxis:

ini_close();

Devuelve:

Cadena

Ejemplo:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Esto abrirá 'savedata.ini' y establecerá la puntuación al valor bajo la sección "save1" con la clave "score" en ella, luego cerrará el .ini de nuevo. Si no hay ningún valor en "save1", "score" o no hay "savedata.ini" presente, la puntuación se establecerá en 0 y se creará un nuevo archivo ini.
