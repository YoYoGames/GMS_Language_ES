ini_section_exists

Esta función comprueba si una sección existe en el ini actualmente abierto y devolverá true si existe o false en caso contrario. Esta comprobación no es necesaria para evitar errores ya que, cuando una sección no existe, la lectura de una sección inexistente devolverá un valor por defecto, pero puede ser útil para ver si un archivo ini ha guardado datos específicos.

Sintaxis:

ini_section_exists(section);

Argumento Descripción
sección La sección en el archivo .ini abierto para comprobar.

Devuelve:

Booleano

Ejemplo:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
global.savegame = "1";
}
ini_close();

Esto pondrá la variable global.savegame en "1" si NO se encuentra "save1". Con un bucle for, esto podría ser utilizado como una forma básica de calcular cuántas partidas guardadas hay para un juego (es decir, la creación de un sistema para múltiples partidas guardadas simultáneas para un juego). simultáneos para un juego).
