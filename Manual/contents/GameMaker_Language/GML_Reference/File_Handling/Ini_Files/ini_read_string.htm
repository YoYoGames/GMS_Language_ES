ini_read_string

Puede utilizar esta función para leer una cadena (texto) de un archivo de datos ini. Los archivos ini se dividen en secciones y luego cada sección se divide en pares clave - valor. Por lo tanto, un archivo ini típico se vería algo así:

Sintaxis:

ini_read_string(section, key, default);

Argumento Descripción
sección La sección del .ini para leer.
clave La clave dentro de la sección relevante del .ini para leer.
por defecto La cadena a devolver si no se encuentra una cadena en el lugar definido (o el archivo .ini no existe). Debe ser una cadena.

Devuelve:

Cadena

Ejemplo:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

Esto abrirá "savedata.ini" y establecerá global.name a la cadena bajo "save1" > "Name" en él, luego cerrará el .ini de nuevo. Si no hay ninguna cadena en "save1" > "Nombre", o no hay ningún archivo "savedata.ini" presente, global.name se ajustará a "Player1".
