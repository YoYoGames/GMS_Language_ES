ini_key_exists

Esta función comprueba si una clave existe en el ini actualmente abierto y devolverá true si existe o false en caso contrario. Esta comprobación no es necesaria para evitar errores ya que, cuando una clave no existe, la lectura de una clave inexistente devolverá un valor por defecto. Sin embargo, puede ser útil para ver si un archivo ini ha guardado datos específicos y algunas otras cosas.

Sintaxis:

ini_key_exists(section, key);

Argumento Descripción
sección La sección en el archivo .ini abierto para comprobar una clave en.
clave La clave para comprobarlo.

Devuelve:

Booleano

Ejemplo:

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
global.name = "Player1";
}
ini_close();

Esto establecerá la variable global.name como "Player1" si no existe tal clave como "name".
