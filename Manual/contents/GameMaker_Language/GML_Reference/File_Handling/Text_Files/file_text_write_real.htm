file_text_write_real

Con esta función se puede escribir un número en el archivo de texto previamente abierto. Tenga en cuenta que como el valor a escribir puede ser un número real, todos los decimales se escribirán con un punto "." como separador. Si el archivo ya contiene información esta información se borrará y la cadena se escribirá al principio del archivo, a menos que haya abierto el archivo con el botón file_text_open_append(). También puede evitar esto utilizando el botón file_text_readln() junto con la función file_text_eof() para recorrer el contenido del archivo hasta llegar al final y luego comenzar a escribir.

Es importante tener en cuenta que cuando se escriben números muy grandes en un archivo de texto mediante esta función, es posible que se traduzcan a un formato estándar simplificado, como "6,6624e+003", que no puede leerse correctamente en GameMaker Studio 2. Para evitar este tipo de problemas, debería convertir el valor en una cadena y utilizar la función file_text_write_string() en su lugar.

Sintaxis:

file_text_write_real(fileid, val);

Argumento Descripción
fileid El id del archivo a editar.
val El valor real para escribir en el archivo.

Devuelve:

Real

Ejemplo:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
file_text_write_real(file, scr[i]);
file_text_writeln(file);
file_text_write_string(file, scr_name[i]);
file_text_writeln(file);
}
file_text_close(file);

El código anterior abre un archivo para escribir y luego hace un bucle a través de dos matrices, escribiendo cada valor de la matriz en una nueva línea del archivo. El archivo se cierra cuando el bucle ha terminado.
