file_text_write_string

Con esta función se puede escribir una cadena en un archivo de texto previamente abierto. Si el archivo ya contiene información, esta información se borrará y la cadena se escribirá al principio del archivo, a menos que haya abierto el archivo con la función file_text_open_append(). También puede evitar esto utilizando la función file_text_readln() junto con la función file_text_eof() para recorrer el contenido del archivo hasta llegar al final y luego comenzar a escribir.

Sintaxis:

file_text_write_string(fileid, str);

Argumento Descripción
fileid El id del archivo a editar.
str La cadena a escribir en el archivo.

Devuelve:

Real

Ejemplo:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
file_text_write_real(file, scr[i]);
file_text_writeln(file);
file_text_write_string(file, scr_name[i]);
file_text_writeln(file);
}
file_text_close(file);

El código anterior abre un archivo para escribir y luego hace un bucle a través de dos matrices, escribiendo cada valor de la matriz en una nueva línea del archivo. El archivo se cierra cuando el bucle ha terminado.
