file_text_writeln

Con esta función puede escribir una nueva línea en un archivo de texto abierto. De este modo, puede omitir líneas o escribir información línea por línea (véase el código de ejemplo siguiente).

Sintaxis:

file_text_writeln(fileid);

Argumento Descripción
fileid El id del archivo a editar.

Devuelve:

Real

Ejemplo:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
file_text_write_real(file, scr[i]);
file_text_writeln(file);
file_text_write_string(file, scr_name[i]);
file_text_writeln(file);
}
file_text_close(file);

El código anterior abre un archivo para escribir y luego hace un bucle a través de dos matrices, escribiendo cada valor de la matriz en una nueva línea del archivo. El archivo se cierra cuando el bucle ha terminado.
