file_text_eof

Esta función devuelve true cuando se ha alcanzado el final de un archivo de texto abierto dado o false en caso contrario.

Sintaxis:

file_text_eof(fileid);

Argumento Descripción
fileid El id del archivo a comprobar.

Devuelve:

Booleano

Ejemplo:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
str[num++] = file_text_readln(file);
}
file_text_close(file);

El código anterior abre un archivo para escribirlo y luego recorre las líneas de texto ya escritas en el archivo hasta llegar al final, almacenando cada línea en la matriz "str".
