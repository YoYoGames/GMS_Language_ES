file_text_eoln

Con esta función puede hacer que GameMaker Studio 2 compruebe el archivo actualmente abierto para ver si la línea que se está leyendo ha terminado. La función devuelve true si se ha alcanzado el final de la línea y false en caso contrario.

Sintaxis:

file_text_eoln(fileid);

Argumento Descripción
fileid El id del archivo a comprobar.

Devuelve:

Booleano

Ejemplo:

var file = file_text_open_read(working_directory + "Game_Data.txt");
var num = 0; while (!file_text_eoln(file))
{
score_array[num] = file_text_read_real(file);
num++;
}
file_text_close(file);

El código anterior abre un archivo para su lectura y luego lee los valores de una sola línea en un array.
