file_bin_seek

Esta función desplaza la posición de lectura actual dentro del archivo a la posición indicada. Se suministra el valor del ID del archivo, tal y como lo devuelve la función file_bin_open()y para añadir un archivo, mueve la posición hasta el tamaño del archivo antes de escribir.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_seek(binfile, pos);

Argumento Descripción
binfile El ID del archivo del que se va a leer.
pos La posición en el archivo a la que hay que desplazarse.

Devuelve:

N/A

Ejemplo:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

Esto abriría un archivo del mismo directorio que el juego, y asignaría su índice a la variable "archivo", luego obtendría el tamaño del archivo y establecería la siguiente posición de escritura a ese tamaño.
