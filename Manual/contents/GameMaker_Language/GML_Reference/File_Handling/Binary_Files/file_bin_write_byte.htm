file_bin_write_byte

Esta función escribirá un byte de datos en el archivo identificado por el ID del archivo en la posición de escritura actual. Usted proporciona el valor del ID del archivo, tal como lo devuelve la función file_bin_open() y el byte de datos a escribir.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_write_byte(binfile, byte);

Argumento Descripción
binfile El ID del archivo en el que se va a escribir.
byte Los datos a escribir.

Devuelve:

real

Ejemplo:

file_bin_write_byte(archivo, datos);

Esto escribiría un byte en el archivo seleccionado.
