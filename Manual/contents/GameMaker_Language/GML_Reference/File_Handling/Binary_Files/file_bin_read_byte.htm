file_bin_read_byte

Esta función devolverá un byte de datos desde la posición actual dentro del archivo con el ID de archivo dado. Se suministra el valor del ID del archivo, tal y como lo devuelve la función file_bin_open().

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_read_byte(binfile);

Argumento Descripción
binfile El ID del archivo del que se van a obtener los datos.

Devuelve:

Real

Ejemplo:

file = file_bin_open("miarchivo.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Esto abriría un archivo del mismo directorio que el juego, y obtendría un byte de datos de él antes de cerrarlo de nuevo.
