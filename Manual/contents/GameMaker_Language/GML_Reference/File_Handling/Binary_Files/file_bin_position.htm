file_bin_position

Esta función devuelve la posición actual en bytes, donde 0 es la primera posición, del archivo con el id de archivo dado. Se suministra el valor del ID del archivo, tal y como lo devuelve la función file_bin_open().

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_position(binfile);

Argumento Descripción
binfile El ID del archivo para obtener la posición.

Devuelve:

Real

Ejemplo:

pos = file_bin_position(file);

Esto almacenaría la posición actual en la variable "pos".

Volver: Binary Files
