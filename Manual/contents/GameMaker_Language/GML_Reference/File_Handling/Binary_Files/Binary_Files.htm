Archivos binarios

En raras situaciones puede ser necesario leer datos de archivos binarios. Estas funciones pueden usarse para ese propósito, aunque en general, le recomendamos que use las funciones dedicadas al buffer para crear, guardar y cargar datos binarios en lugar de estas funciones.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Para ello existen las siguientes rutinas de bajo nivel:

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
