archivo_existe

Esta función devolverá true si el archivo especificado existe y false si no existe. Tenga en cuenta que la función sólo puede utilizarse para comprobar los archivos locales, pero no cualquier archivo almacenado en un servidor remoto.

Sintaxis:

file_exists(fname);

Argumento Descripción
fname El nombre del archivo a comprobar.

Devuelve:

Booleano

Ejemplo:

if file_exists("nivel.txt")
{
file = file_text_open_read("level.txt");
}

Esto comprobaría la existencia de un archivo y, si existe, se abre para su lectura.
