nombre_de_archivo

Esta función devuelve la información de la unidad de disco del nombre del archivo.

Sintaxis:

nombre_de_archivo(nombre_f);

Argumento Descripción
fname El archivo a utilizar.

Devuelve:

Cadena

Ejemplo:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

El código anterior obtiene la información de la unidad (como una cadena) del archivo especificado.
