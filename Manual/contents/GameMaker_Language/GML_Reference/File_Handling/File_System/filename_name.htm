nombre_de_archivo

Esta función devuelve la parte del nombre del archivo indicado, con la extensión pero sin la ruta

Sintaxis:

nombre_de_archivo(nombre_f);

Argumento Descripción
fname El archivo a utilizar.

Devuelve:

Cadena

Ejemplo:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

El código anterior obtiene el nombre (en forma de cadena) del primer archivo de tipo "doc" encontrado en el directorio especificado.
