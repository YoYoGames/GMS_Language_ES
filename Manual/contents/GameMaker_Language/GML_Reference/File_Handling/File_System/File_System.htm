Sistema de archivos

En GameMaker Studio 2 existen varias funciones que permiten manipular archivos de todo tipo de diferentes maneras:

  • archivo_existe
  • archivo_eliminar
  • renombrar archivo
  • copia_de_archivo
  • file_find_first
  • file_find_next
  • file_find_close
  • atributos_de_archivo

Las siguientes funciones se pueden utilizar para cambiar los nombres de los archivos:

NOTA: ¡Estas funciones no trabajan con los archivos reales! Sólo se ocupan de las cadenas relacionadas con los diferentes elementos de la ruta y deben utilizarse junto con las otras funciones de archivo.

  • nombre_de_archivo
  • ruta_de_archivo
  • nombre_de_archivo
  • nombre_de_archivo
  • nombre_de_archivo_ext
  • cambio_de_archivo_ext

Por último, hay algunas funciones especiales para obtener nombres de archivos que pueden utilizarse para cargar y guardar archivos con filtros de tipo de archivo. Estas funciones abrirán una ventana del explorador y, por tanto, harán que el juego pierda el foco. Esto significa que cualquier entrada de la que dependa el Ello significa que cualquier entrada de la que dependa el juego puede no ser registrada correctamente, por lo que recomendamos que (si se utilizan estas funciones en un evento de teclado) se utilice el evento de tecla arriba y no los eventos de pulsación o abajo.

NOTA: Estas funciones son Windows y macOS sólo.

ATENCIÓN! Es posible que estas funciones no funcionen como se espera debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, consulte la sección sobre el Sistema de Archivos para más información.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
