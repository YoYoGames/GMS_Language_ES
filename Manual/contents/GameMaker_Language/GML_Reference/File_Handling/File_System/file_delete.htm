archivo_eliminar

Esta función borrará el archivo especificado del sistema. Hay que tener en cuenta que esta función sólo eliminará los archivos que GameMaker Studio 2 es capaz de crear y analizar: archivos ini, archivos de texto y archivos binarios, o aquellos archivos hechos para almacenar recursos creados por el juego, como sprites o superficies. Sin embargo, no eliminará ningún otro archivo. La función también devolverá true si el archivo ha sido eliminado con éxito, o false en cualquier otra circunstancia.

ATENCIÓN! Esta función puede no funcionar como se espera debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, consulte la sección sobre los límites del sistema de archivos para obtener más información.

Sintaxis:

file_delete(fname);

Argumento Descripción
fname El nombre del archivo a eliminar.

Devuelve:

Booleano

Ejemplo:

if file_exists("nivel.txt")
{
file_delete("level.txt");
}

Esto comprobaría la existencia de un archivo y si existe se elimina.
