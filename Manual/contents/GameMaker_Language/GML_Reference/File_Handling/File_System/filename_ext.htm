nombre_de_archivo_ext

Esta función devuelve la parte de la extensión del nombre del archivo indicado, incluyendo el punto inicial.

Sintaxis:

filename_ext(fname);

Argumento Descripción
fname El archivo a utilizar.

Devuelve:

Cadena

Ejemplo:

ext = filename_ext(file_find_first("*.*", 0));

El código anterior obtiene la extensión (como cadena) del archivo especificado.
