get_save_filename

Esta función abre un diálogo y pide al jugador un nombre de archivo para guardar con el filtro dado. El filtro tiene la forma " nombre1|máscara1|nombre2|máscara2|... ", donde la máscara puede contener las diferentes opciones con un punto y coma entre ellas y también se puede utilizar un "*" para significar cualquier cadena. Por ejemplo: " bitmaps|*.bmp;*.wmf", pediría al usuario que seleccionara el nombre de un archivo de mapa de bits del sistema, pero sólo los que tienen las dos extensiones especificadas (bmp ypng ). ¡esta función no guarda el archivo en sí! Sólo devuelve una cadena con el nombre completo y la ruta del archivo. Si el usuario pulsa "Cancelar" se devuelve una cadena vacía "".

Una cosa importante a tener en cuenta cuando se utiliza esta función es que le otorga ciertos permisos para ese archivo, durante la duración de su juego. Así, una vez que obtengas la ruta del archivo del usuario, podrás acceder a él una y otra vez sin tener que preguntar, y el permiso se aplica a la lectura y la escritura por igual.

NOTA: Esta función sólo es válida en los objetivos de Windows y macOS.

Sintaxis:

get_save_filename(filter, fname);

Argumento Descripción
filtrar El filtro del tipo de archivo y/o el nombre.
fname El nombre de archivo que se sugiere utilizar.

Devuelve:

Cadena

var file;
file = get_save_filename("screenshot|*.png", "");
