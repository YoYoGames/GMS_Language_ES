atributos_de_archivo

Puede utilizar esta función para comprobar los atributos de un archivo determinado. Puede sumar las siguientes constantes para ver el tipo de archivos que desea:

Constante Descripción
fa_readonly Archivos de sólo lectura
fa_hidden Archivos ocultos
fa_sysfile Archivos del sistema
fa_volumenid Archivos de volumen-identificación
fa_directorio Directorios
fa_archivo Ficheros archivados

NOTA: Esta es una función sólo para Windows.

Sintaxis:

file_attributes(fname, attr);

Argumento Descripción
fname El nombre del archivo a comprobar.
attr Los atributos que hay que comprobar.

Devuelve:

Booleano

Ejemplo:

if !file_attributes(file, fa_hidden)
{
file_delete(file);
}

Esto comprobaría un archivo para ver si está oculto o no, y si no se elimina.
