file_find_next

Esta función devuelve el nombre del siguiente archivo que satisface la máscara dada previamente y los atributos (definidos por file_find_first()). Si no existe tal archivo, se devuelve la cadena vacía.

¡AVISO! Esta función puede no funcionar como esperas debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

NOTA: Esta función funciona en todas las plataformas de destino de C++(Windows, Mac, iOS, Android y WinPhone), PERO los indicadores de filtro sólo funcionan en Windows.

Sintaxis:

file_find_next();

Devuelve:

Cadena

Ejemplo:

archivo = file_find_next();

Esto devolverá el siguiente archivo que coincida con los atributos dados previamente.
