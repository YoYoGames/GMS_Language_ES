renombrar archivo

Esta función renombrará el archivo especificado con el nombre especificado. La función devolverá true si el archivo ha sido renombrado con éxito, o false en cualquier otra circunstancia.

¡AVISO! Esta función puede no funcionar como esperas debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

file_rename(oldname, newname);

Argumento Descripción
oldname El nombre del archivo a modificar.
nuevo nombre El nuevo nombre que se le dará al archivo.

Devuelve:

Booleano

Ejemplo:

if file_exists("nivel1.txt")
{
file_rename("nivel1.txt", "nivel.txt");
}

Esto comprobaría la existencia de un archivo y si existe se le cambia el nombre.
