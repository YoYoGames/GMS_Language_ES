copia_de_archivo

Esta función copiará el archivo especificado, dándole el nuevo nombre que usted elija. Si está copiando un archivo en Android desde el área de Archivos Incluidos (que estará dentro de su APK) se copiará primero en la memoria, así que, tenga en cuenta que copiar archivos muy grandes desde el APK puede hacer que los dispositivos se queden sin memoria. En el objetivo de HTML5, utilizamos la API de almacenamiento local de HTML5 para guardar en el lado del cliente, y las funciones no admiten operaciones con archivos binarios, por lo que se supone que estás que estás copiando un archivo de texto.

ATENCIÓN! Esta función puede no funcionar como se espera debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, consulte la sección sobre los límites del sistema de archivos para obtener más información.

Sintaxis:

file_copy(fname, newname);

Argumento Descripción
fname El nombre del archivo a copiar.
nuevo nombre El nombre del nuevo archivo copiado.

Devuelve:

N/A

Ejemplo:

if file_exists("nivel1.txt")
{
file_copy("nivel1.txt", "nivel2.txt");
}

Esto comprobaría un archivo y si existe se copia.
