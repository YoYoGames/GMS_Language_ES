file_find_first

Esta función devolverá el nombre del primer archivo que satisfaga la máscara y los atributos. Si no existe tal archivo, se devuelve una cadena vacía. La máscara puede contener una ruta y puede contener caracteres comodín, por ejemplo "C:\temp\*.doc". Los atributos dan los archivos adicionales que se quieren ver, por lo que los archivos normales se devuelven siempre que satisfagan la máscara. Puede sumar las siguientes constantes para ver el tipo de archivos que desea (si no desea añadir ningún atributo, utilice 0):

Constante Descripción
fa_readonly Archivos de sólo lectura
fa_hidden Archivos ocultos
fa_sysfile Archivos del sistema
fa_volumenid Archivos de volumen-identificación
fa_directorio Directorios
fa_archivo Ficheros archivados

¡AVISO! Esta función puede no funcionar como esperas debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

NOTA: Esta función no funcionará en absoluto en el objetivo HTML5, y las banderas de filtro sólo funcionarán en Windows (utilice 0 para el resto de plataformas).

Sintaxis:

file_find_first(mask, attr);

Argumento Descripción
máscara La máscara a utilizar para la búsqueda.
attr El atributo específico del archivo a buscar.

Devuelve:

Cadena

Ejemplo:

if directory_exists("\User Content")
{
file = file_find_first("/Contenido de usuario/*.doc", fa_readonly);
}

Esto comprobará si el directorio especificado existe y, si lo hace, irá allí y devolverá el primer archivo doc de "sólo lectura" encontrado.
