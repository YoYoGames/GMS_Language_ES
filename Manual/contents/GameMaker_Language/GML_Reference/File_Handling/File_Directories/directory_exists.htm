directorio_existe

Esta función devolverá true si el directorio indicado existe o false si no existe. El nombre especificado debe incluir la ruta completa, no una ruta relativa y por defecto no se puede acceder a ningún directorio desde fuera del paquete de juegos ya que todos los juegos están protegidos por una caja de arena (véase la sección sobre el sistema de archivos para más información).

Sintaxis:

directorio_existe(nombre_d)

Argumento Descripción
dname El nombre del directorio a buscar.

Devuelve:

Booleano

Ejemplo:

if directory_exists(directorio_de_trabajo + "Saves/")
{
file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Esto comprobará si el directorio especificado existe y, si lo hace, irá allí y devolverá el primer archivo doc de "sólo lectura" encontrado.
