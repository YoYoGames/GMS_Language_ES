crear_directorio

Esta función creará un directorio con el nombre dado en el área de guardado.

NOTA: Esta función no funcionará en HTML5 ya que no se pueden crear o eliminar directorios en el almacenamiento local del navegador.

¡AVISO! Esta función puede no funcionar como esperas debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

directory_create(dname)

Argumento Descripción
dname El nombre del directorio a crear.

Devuelve:

N/A

Ejemplo:

if !directory_exists("Juegos")
{
directory_create("Juegos");
}

Esto comprobará si el directorio especificado existe en la carpeta de datos local y, si no existe, lo crea por ti.
