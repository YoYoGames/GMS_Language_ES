directorio_de_programas

Esto devolverá el directorio donde se almacena el ejecutable del juego. Sin embargo, esto puede no ser siempre útil, particularmente porque algunos dispositivos ejecutan el exe desde un archivo *.zip, por lo que esto devolvería lo mismo sin importar desde dónde se esté ejecutando el juego.

¡AVISO! Esta función puede no funcionar como esperas debido a que GameMaker Studio 2 está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

NOTA: Si desactiva la caja de arena, no podrá escribir en este directorio para evitar cualquier posible problema con el borrado de los archivos necesarios del juego.

Sintaxis:

directorio_de_programas

Devuelve:

Cadena

Ejemplo:

dir = directorio_de_programa;

Esto almacenará en una variable el directorio donde se encuentra el ejecutable.
