Directorios de archivos

Al trabajar con archivos, habrá momentos en los que necesites hacer referencia al directorio en el que están almacenados los archivos y GameMaker Studio 2 tiene algunas funciones y variables especiales relacionadas con los diferentes directorios posibles que puedes necesitar:

  • directorio_existe
  • crear_directorio
  • directorio_destrucción
  • directorio_temporal
  • directorio_de_trabajo
  • directorio_de_programas

NOTA: No se pueden crear ni destruir directorios en HTML5.
