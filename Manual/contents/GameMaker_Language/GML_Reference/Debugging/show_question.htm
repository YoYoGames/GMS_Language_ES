mostrar_pregunta

Esta función crea un cuadro de mensaje emergente con dos botones para "Sí" y "No". Devuelve verdadero o falso dependiendo de cuál de los dos botones pulse el usuario.

NOTA: Esta función es para uso de depuración sólo en objetivos de escritorio, pero es obsoleta en todos los demás objetivos.

Sintaxis:

mostrar_pregunta(str);

Argumento Descripción
str La cadena a mostrar en la pregunta emergente.

Devuelve:

Booleano

Ejemplo:

if (score > 500) && debug_mode
{
if show_question("¿Continuar a la siguiente sala?")
{
room_goto(rm_Level2);
}
else game_end();
}

El código anterior comprobará la puntuación y si es superior a 500, preguntará al usuario si desea continuar o no y si se pulsa el botón "sí" se irá a otra sala, pero si se selecciona el botón "no" terminará el juego.
