get_integer

Esto crea una ventana emergente que muestra un mensaje personalizado, con un botón llamado "Ok", y pide al usuario que introduzca un valor entero. La función devolverá el número entero introducido, o el valor por defecto si no se ha introducido nada.

NOTA: Esta función es para uso de depuración sólo en objetivos de escritorio, pero es obsoleta en todos los demás objetivos.

Sintaxis:

get_integer(str, def);

Argumento Descripción
str La cadena a mostrar en el mensaje emergente.
def El valor por defecto del cuadro de texto.

Devuelve:

Real

Ejemplo:

global.level = get_integer("¿Nivel a probar?", 1);

El código anterior mostrará un mensaje pidiendo al usuario que seleccione un nivel para la prueba. El valor de retorno se almacenará en la variable global "global.level".
