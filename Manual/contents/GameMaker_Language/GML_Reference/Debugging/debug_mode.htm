modo_depuración

Esta variable de sólo lectura devuelve true cuando el juego se está jugando en modo de depuración y false cuando se está jugando de forma normal.

Sintaxis:

modo_depuración

Devuelve:

Booleano

Ejemplo:

si debug_mode
{
ini_open("Cheats.ini");
}
else
{
ini_open("Game.ini");
}

El código anterior abre un archivo ini diferente dependiendo del valor de la variable de sólo lectura debug_mode.
