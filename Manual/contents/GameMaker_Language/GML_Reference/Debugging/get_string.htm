get_string

Esto crea una ventana emergente que muestra un mensaje estándar, con un botón llamado "Ok", que pide al usuario que introduzca una cadena. La función devolverá la cadena introducida, o el valor por defecto si no se ha introducido nada.

NOTA: Esta función es para uso de depuración sólo en objetivos de escritorio, pero es obsoleta en todos los demás objetivos.

Sintaxis:

get_string(str, def);

Argumento Descripción
str La cadena a mostrar en el mensaje emergente.
def La cadena por defecto en el cuadro de texto.

Devuelve:

Cadena

Ejemplo:

global.nombre_de_prueba = get_string("Nombre de la puntuación máxima de la prueba:", "Anónimo");

El código anterior pedirá al usuario que dé un nombre que se almacenará en la variable global "nombre_de_prueba". Si no se introduce nada y el usuario se limita a pulsar "Ok", se devolverá el valor por defecto, "Anónimo".
