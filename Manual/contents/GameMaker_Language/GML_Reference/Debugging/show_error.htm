mostrar_error

Esta función mostrará una cadena personalizada como mensaje de error y dará la opción de abortar el juego después si es necesario.

NOTA: Esta función es sólo para uso de depuración.

Sintaxis:

show_error(str, abort);

Argumento Descripción
str La cadena a mostrar en el mensaje emergente.
abortar Si se debe abortar (true) o no (false).

Devuelve:

N/A

Ejemplo:

if (room != rm_Dungeon)
{
show_error("Error: Went to wrong area. Aborting game.", true);
}

El código anterior comprobará si la sala en la que se encuentra el jugador es "rm_Dungeon" y, en caso contrario, mostrará un mensaje de error y obligará al usuario a abortar la partida.
