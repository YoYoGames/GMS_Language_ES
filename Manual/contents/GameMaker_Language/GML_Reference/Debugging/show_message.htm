mostrar_mensaje

Esta función crea un cuadro de mensaje emergente que muestra la cadena dada y un botón "Ok" para cerrarlo.

NOTA: Esta función es para uso de depuración sólo en objetivos de escritorio y UWP, pero es obsoleta en todos los demás objetivos.

Sintaxis:

mostrar_mensaje(str);

Argumento Descripción
str La cadena a mostrar en el mensaje emergente.

Devuelve:

N/A

Ejemplo:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
tot += inv[i];
}
show_message("Total = " + string(tot));

El código anterior recorrerá los valores almacenados en la matriz "inv" y los añadirá a la variable "tot" antes de mostrar un mensaje con el total.
