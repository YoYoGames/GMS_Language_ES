show_debug_message

Esta función creará un mensaje de depuración personalizado que se mostrará en la ventana del compilador en tiempo de ejecución. Tenga en cuenta que el mensaje puede ser una cadena o un número real, pero si necesita ambos, el número tendrá que ser convertido a cadena primero usando la función cadena() (véase el ejemplo siguiente) y que si el número tiene más de dos decimales, deberá utilizar string_format() para mostrarlos ya que, por defecto, los decimales se redondean a los dos más cercanos (por lo que "1,2468" se mostraría como "1,25" en la ventana de salida).

Los mensajes de depuración mostrados con esta función se mostrarán en la ventana de salida del compilador en la parte inferior del IDE, así como en la vista gráfica del del depurador cuando se ejecuta el juego en modo de depuración. Si sólo desea ver los mensajes en el modo de depuración, entonces probablemente debería utilizar debug_event() en su lugar.

Sintaxis:

show_debug_message(string);

Argumento Descripción
cadena El mensaje de depuración personalizado a mostrar.

Devuelve:

N/A

Ejemplo:

if !instance_exists(obj_Exit)
{
show_debug_message("¡Salida no creada!");
show_debug_message("Número de error: " + string(global.error));
game_end();
}

El código anterior comprueba si existe una instancia y, si no es así, se muestra un mensaje de depuración en la ventana de compilación y se termina el juego.
