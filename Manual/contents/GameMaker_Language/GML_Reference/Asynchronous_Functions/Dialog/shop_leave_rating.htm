shop_leave_rating

Esta función abre un diálogo dependiente del sistema operativo en el que puede pedir al usuario que publique una valoración o un comentario en una página concreta. Puede definir el texto que debe aparecer en el diálogo, así como el texto que desea que aparezca en los dos botones y la URL donde debe publicarse el comentario.

NOTA: Esta función es sólo para objetivos Android e iOS.

IMPORTANTE! En iOScuando se utiliza TestFlight, en el desarrollo se mostrará el aviso pero no podrá hacer nada realmente. Una vez que su aplicación está en la App Store, este botón se activa y el usuario tiene la opción de escribir una reseña o dejar una calificación. Ten en cuenta que si distribuyes el juego a través de TestFlight, es posible que esta ventana ni siquiera se muestre.

Sintaxis:

shop_leave_rating(text, yes_string, no_string, url)

Argumento Descripción
texto Texto que aparece en el diálogo.
sí_cadena Texto que aparece en el botón "sí".
no_string Texto que aparece en el botón "no".
url La URL completa a la que debe enviarse el comentario.

Devuelve:

N/A

Ejemplo:

if timer <= 0
{
shop_leave_rating("¡Gracias por jugar al juego! ¿Por qué no dejas un comentario?", "¡Bien!", "¡Hoy no!", "http://MacSweeney/comments");
timer = 10000000;
}
else
{
temporizador -= 1;
}

El código anterior pedirá al usuario que deje un comentario si la variable "timer" ha llegado a 0.
