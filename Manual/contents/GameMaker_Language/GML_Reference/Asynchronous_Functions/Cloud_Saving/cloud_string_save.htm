cloud_string_save

Esta función enviará una cadena al servicio de nube elegido para su almacenamiento. La función devolverá un valor de identificación único que deberá ser utilizado en el evento asíncrono apropiado para identificar el mapa DS que se devuelve como una "llamada de vuelta" del servicio en la nube. La cadena debe contener toda la información que necesites guardar para tu juego, ya que sólo puedes almacenar un único "blob de datos" en la nube, y ejecutar esta función de nuevo sobrescribirá cualquier valor previamente almacenado (al igual que el uso de la función cloud_file_save() ). La descripción debe ser una cadena corta de información que describa el guardado, por ejemplo: "Nivel2, Etapa2".

Para más información sobre los datos asíncronos devueltos, consulte la función cloud_synchronise().

Sintaxis:

cloud_string_save(string, description);

Argumento Descripción
cadena La cadena de datos que se va a cargar.
descripción Una breve descripción de los datos almacenados.

Devuelve:

Real

Ejemplo:

var t_str = "";
for (var i = 0; i < 10; i++;)
{
t_str += string(global.Highscore[i]) + "|"
}
save_check = cloud_string_save(t_str, "Current Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);

El código anterior crea una cadena a partir de los valores almacenados en la matriz global "Highscores" y luego escribe esta cadena en el servicio en la nube para su almacenamiento, además de escribirla en un archivo para su almacenamiento local.
