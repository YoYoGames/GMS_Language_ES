http_get_request_crossorigin

Esta función se puede utilizar para obtener el conjunto de tipos de origen cruzado para los juegos HTML5 y devolverá una cadena (en todas las demás plataformas se devolverá una cadena vacía "").

Sintaxis:

http_get_request_crossorigin();

Devuelve:

Cadena

Ejemplo:

if string_lower(http_get_request_crossorigin()) != "use-credentials"
{
http_set_request_crossorigin("use-credentials");
}
sprite_add("sprites/player_outfit_1.png", 0, false, false, 0, 0);

El código anterior comprobará primero el tipo de origen cruzado actualmente establecido y si no es "use-credentials" entonces se establece como "use-credentials" y luego se añade un sprite desde un archivo.

Volver: HTTP
Siguiente: http_set_request_crossorigin
