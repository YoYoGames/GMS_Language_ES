push_get_first_local_notification

Esta función rellenará un mapa DS prefabricado con una serie de pares clave/valor para que se muestre la primera notificación push local en la cola. Devuelve -1 si no hay notificaciones en cola, o un valor real en caso contrario que representa el ID de la notificación. Este ID puede utilizarse para cancelar la notificación mediante la función push_cancel_local_notification().

NOTA: Esta función está limitada al iOS y Android módulos de destino.

El mapa DS contendrá los siguientes pares clave/valor:

  • " título" - El título de la notificación (sólo estará disponible en la Android plataforma, ya que iOS no almacena la información del título)
  • "mensaje" - El texto del cuerpo del mensaje
  • " data"- La cadena del paquete de datos

Sintaxis:

push_get_first_local_notification(map);

Argumento Descripción
mapa El ID del mapa DS a utilizar.

Devuelve:

Real

Ejemplo :

var map = ds_map_create();
var ntf = push_get_first_local_notification(map);
while(ntf >= 0)
{
var data = ds_map_find_value(map, "data");
if data == "Daily_Reward"
{
push_cancel_local_notification(ntf);
}
ntf = push_get_next_local_notification(map);
}
ds_map_destroy(map);

El código anterior crea un mapa DS y luego solicita los datos de la primera notificación disponible en la cola. Si existe una notificación, el mapa se rellena y el código comprueba la clave "data" para una cadena específica. Si la cadena se cancela la notificación y se pasa a comprobar la siguiente notificación hasta que se hayan comprobado todas las de la cola.
