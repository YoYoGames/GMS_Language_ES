os_get_region

Esta función devuelve una cadena con el código regional de dos o tres letras para el sistema operativo que está ejecutando el juego, tal y como establece el estándar ISO3166-1. Si la información no está disponible se mantendrá simplemente una cadena vacía "".

NOTA: Este no es el código regional de la ubicación que se devuelve, sino el código regional del idioma del sistema operativo.

Sintaxis:

os_get_region()

Devuelve:

Cadena

Ejemplo:

switch (os_get_language())
{
case "zh":
var region = os_get_region();
if (region == "HK" || region == "MO" || region == "TW")
{
ini_open("chinese_traditional.ini");
}
si no
{
ini_open("chinese_simplified.ini");
}
break;
caso "fr":
ini_open("french.ini");
break;
case "it":
ini_open("italian.ini");
break;
por defecto:
ini_open("english.ini");
break;
}

El código anterior comprueba el idioma del sistema operativo y si es chino, entonces comprueba la región del sistema operativo, abriendo un archivo *.ini diferente dependiendo de los valores devueltos.
