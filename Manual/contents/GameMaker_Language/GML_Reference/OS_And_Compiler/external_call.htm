llamada_externa

Si ha creado una llamada a una función externa a una dll o dylib utilizando external_define()puede utilizar esta función para llamarla. Debe proporcionar el nombre de la función previamente definida, así como así como cada uno de los argumentos que requiere (cada argumento debe ser del tipo correcto, ya sea real o cadena) y la función devuelve el resultado de la llamada externa.

Sintaxis:

llamada_externa(id, args[0...15]);

Argumento Descripción
id El nombre de la función que se quiere llamar
args[0...10] Los diferentes argumentos que quieres pasar a la función externa

Devuelve:

Valor (el tipo de valor devuelto dependerá de la función definida)

Ejemplo:

mi_función = external_define("MiDLL.dll", "MiMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(mi_función, x, y);

El código de ejemplo anterior llama a una función externa previamente definida y almacena el valor devuelto en una variable local.
