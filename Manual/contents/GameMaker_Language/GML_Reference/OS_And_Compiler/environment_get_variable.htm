environment_get_variable

Esta función devuelve el valor (una cadena) de la variable de entorno con el nombre dado (también una cadena). Puede obtener las variables de entorno disponibles en macOS y Ubuntu(Linux) escribiendo " env" en la aplicación de terminal, y para información sobre las variables de entorno de Windows, si estás usando el símbolo del sistema, escribe " echo%PATH%", y usando PowerShell es " ls env". Tenga en cuenta que tanto en macOS como en Ubuntu(Linux) la variable de entorno "HOME" devolverá la ruta "~/", que corresponde a "/Users/<nombredeusuario>" en macOS y a "/home/<nombredeusuario >" en Ubuntu (Linux).

NOTA: Esta función es sólo para Windows, macOS y Ubuntu(Linux).

Sintaxis:

environment_get_variable(nombre);

Argumento Descripción
nombre El nombre (una cadena) de la variable de entorno a comprobar.

Devuelve:

Cadena

Ejemplo:

e_str = environment_get_variable("APPDATA");

El código anterior devolverá la ruta completa de la variable de entorno "%appdata%", que normalmente es "C:\N-usuario\N-AppData\N-Roaming".
