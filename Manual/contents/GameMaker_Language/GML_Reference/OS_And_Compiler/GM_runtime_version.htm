GM_runtime_version

Esta constante mantiene el número de versión del tiempo de ejecución definido en las Preferencias como el tiempo de ejecución que se está utilizando para construir el proyecto. El valor se almacena como una cadena.

Sintaxis:

GM_runtime_version;

Devuelve:

Cadena

Ejemplo:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, " Runtime " + GM_runtime_version);

El código anterior toma las constantes de fecha, versión y tiempo de ejecución de GM y las dibuja en la pantalla.
