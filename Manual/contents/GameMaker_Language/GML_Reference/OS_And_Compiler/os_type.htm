tipo_os

Esta variable de sólo lectura contiene una de las varias constantes que tiene GameMaker Studio 2 para decirle para qué sistema operativo ha sido creado el juego. Tenga en cuenta que esto no es necesariamente el mismo que el sistema operativo del dispositivo que se ejecuta ya que, por ejemplo, tu juego podría ejecutarse en un Amazon Fire OS, pero habrá sido creado para la plataforma Android (en cuyo caso os_type será os_android).

Se pueden devolver las siguientes constantes:

Constante Descripción
os_windows Sistema operativoWindows
os_uwp Plataforma universal deWindows 10
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos Apple tvOS
os_android Android
os_ps4 Sony PlayStation 4
os_xboxone Microsoft XBox One
os_switch Nintendo Switch
os_unknown SO desconocido

Sintaxis:

os_type;

Devuelve:

Constante

Ejemplo:

switch (os_type)
{
case os_windows: global.Config = 0; break;
case os_android: global.Config = 1; break;
case os_linux: global.Config = 2; break;
case os_macosx: global.Config = 3; break;
case os_ios: global.Config = 4; break;
case os_winphone: global.Config = 5; break;
}

El código anterior comprueba el sistema operativo que ejecuta el juego y establece una variable global en consecuencia.
