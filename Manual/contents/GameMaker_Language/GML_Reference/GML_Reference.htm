Referenciadel lenguaje GameMaker

Esta sección del manual es una guía de referencia para el GameMaker Studio 2(GML). Aquí encontrará todas las funciones disponibles documentadas junto con los argumentos necesarios y ejemplos de código para mostrar cómo se pueden utilizar. Para obtener información sobre el uso de GML, consulte la sección Visión general deGML del manual.

NOTA: En cualquier lugar del manual donde aparezca "N/A" como valor de retorno de una función, significa que la función no está destinada a devolver ningún valor como parte de su operación. Si intenta comprobar el valor de retorno de estas funciones, obtendrá el valor obtendrá el valor undefined.
