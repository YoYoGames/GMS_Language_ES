motion_set

Esta función establece una nueva dirección de movimiento y una nueva velocidad a la instancia que ejecuta el código. Tenga en cuenta que esto no se suma a la velocidad y la dirección actuales de la instancia (para eso se usaría motion_add()) sino que la fuerza a la nueva configuración.

Sintaxis:

motion_set(dir, speed);

Argumento Descripción
dir La nueva dirección.
velocidad La nueva velocidad.`

Devuelve:

N/A

Ejemplo:

si irandom(9) = 1
{
motion_set(random(360), 1 + random(3));
}

Este código hará que la instancia cambie de velocidad y dirección a intervalos aleatorios.
