lugar_secuestrado

Con esta función puedes comprobar y ver si el origen de una instancia (su posición x e y) está alineado a una cuadrícula con los valores hsnap y vsnap especificados por ti.

Sintaxis:

place_snapped(hsnap, vsnap);

Argumento Descripción
hsnap El encaje horizontal para comprobar.
vsnap El encaje vertical para comprobar.

Devuelve:

Booleano

Ejemplo:

con (obj_Piezas)
{
if !place_snapped(32, 32)
{
move_snap(32, 32);
}
}

El código anterior comprueba todas las instancias de "obj_Pieces" para ver si están ajustadas a una cuadrícula de 32x32 píxeles, y si no lo están las ajusta a la posición más cercana de esa cuadrícula.
