move_outside_solid

Con esta función se puede indicar a una instancia que salga de una colisión con cualquier instancia marcada como sólido en cualquier dirección y cualquier número de píxeles en cada paso, con un valor de -1 o 0 para la distancia máxima que es de 1000px por defecto, es decir: GameMaker Studio 2 moverá la instancia continuamente hacia arriba 1000 píxeles hasta que esté fuera de la colisión.

Sintaxis:

move_outside_solid(dir, maxdist);

Argumento Descripción
dir La dirección a seguir.
maxdist La distancia máxima que puede recorrer el objeto (-1 o 0 un valor por defecto de 1000 píxeles).

Devuelve:

N/A

Ejemplo:

si otro.sólido
{
var pdir;
pdir = point_direction(other.x, other.y, x, y);
move_outside_solid(pdir, -1);
}

El código anterior entraría en un evento de colisión y comprobaría si la "otra" instancia en la colisión está marcada como "sólida". Si lo es, moverá la instancia fuera de la colisión.
