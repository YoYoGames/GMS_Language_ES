move_snap

Esta función se utiliza para "ajustar" la instancia a una cuadrícula de un tamaño determinado. Se ajustará a la posición correspondiente más cercana en la cuadrícula "invisible" que los valores hsnap y vsnap definen.

Sintaxis:

move_snap( hsnap, vsnap );

Argumento Descripción
hsnap El ajuste horizontal (el tamaño en píxeles entre "celdas").
vsnap El ajuste vertical (el tamaño en píxeles entre "celdas").

Devuelve:

N/A

Ejemplo:

con (obj_Piezas)
{
if !place_snapped(32, 32)
{
move_snap(32, 32);
}
}

El código anterior comprueba todas las instancias de "obj_Pieces" para ver si están ajustadas a una cuadrícula de 32x32 píxeles, y si no lo están las ajusta a la posición más cercana de esa cuadrícula.
