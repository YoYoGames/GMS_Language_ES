Movimiento

En cualquier juego, el movimiento y la posición son de vital importancia, por lo que GameMaker Studio 2 dispone de una completa selección de funciones para hacer frente a cualquier situación. Las dos formas principales de mover una instancia es establecer la posición real o establecer un vector de velocidad/dirección, y esto puede hacerse utilizando las variables de instancia incorporadas o utilizando funciones de movimiento específicas. Ambas opciones se explican en las secciones que aparecen a continuación:

Las siguientes variables se incluyen "integradas" en cada instancia y pueden ser leídas y modificadas para cambiar diferentes comportamientos cuando la instancia se está moviendo (afectarán a los vectores de velocidad y dirección):

  • dirección
  • fricción
  • gravedad
  • dirección_de_gravedad
  • hspeed
  • vspeed
  • velocidad

Aparte de las variables mencionadas anteriormente, también puede establecer la posición de la instancia directamente utilizando lo siguiente:

  • x
  • y
  • xprevio
  • yprevio
  • xstart
  • ystart

Todas las funciones siguientes pueden utilizarse para mover una instancia de alguna manera, algunas de las cuales afectan a los vectores de velocidad/dirección y otras afectan directamente a la posición real x/y dentro de la sala:

  • motion_add
  • motion_set
  • mover_hacia_el_punto
  • move_bounce_all
  • move_bounce_solid
  • move_contact_all
  • mover_contacto_sólido
  • move_outside_all
  • move_outside_solid
  • move_random
  • move_snap
  • move_wrap
  • lugar_secuestrado
