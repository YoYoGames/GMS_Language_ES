move_bounce_all

Con esta función puedes ordenar a una instancia que rebote contra todas las instancias dentro de la sala, con la única excepción de aquellas que no tienen asignado ningún índice de sprite o máscara. También puedes decirle que utilice la comprobación de colisión precisa cuando esté activada, pero ten en cuenta que esto requiere que todas las instancias tengan máscaras precisas habilitadas y ralentizará mucho tu juego cuando haya muchas instancias involucradas debido a la cantidad de procesamiento que hay que hacer. Esto debería ir normalmente en el evento de paso de una instancia pero también puede usarse de forma selectiva en un evento de colisión, como se ilustra en el siguiente ejemplo de código.

Sintaxis:

move_bounce_all( adv );

Argumento Descripción
adv Si se activa el rebote avanzado (true) o no (false).

Devuelve:

N/A

Ejemplo:

si otro.visible
{
move_bounce_all(false);
}

El código anterior se colocaría en un evento de colisión con otro objeto. Primero comprueba si el objeto es visible y, si lo es, realiza la acción move_bounce_all(). Tenga en cuenta que en este caso el rebote es selectivo y sólo se calculará para esta colisión, en lugar de para cada instancia, cada paso.
