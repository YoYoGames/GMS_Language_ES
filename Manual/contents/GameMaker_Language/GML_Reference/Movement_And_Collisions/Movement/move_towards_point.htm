mover_hacia_el_punto

Esta función simplemente le dice a la instancia que ejecuta el código que se mueva hacia un punto establecido a una velocidad determinada. Ten en cuenta que eres tú quien tiene que decirle qué hacer una vez que llegue a ese punto, ya que no se detendrá por sí mismo. Esto significa que puedes obtener una instancia "vibrante" mientras se mueve (por ejemplo) 3 píxeles hacia adelante, sobrepasando la posición del objetivo x/y en 2, luego se mueve 3 píxeles hacia atrás de nuevo, sobrepasando el punto objetivo en uno y así sucesivamente... Hay varias maneras de evitar esto, por ejemplo, utilizando distancia_al_punto() para ver a qué distancia de la ubicación especificada se encuentra la instancia antes de moverse.

NOTA: esta función establece la velocidad de la instancia, por lo que incluso si se detiene utilizando este código, la instancia seguirá moviéndose en la dirección anterior, por lo que es necesario establecer la instancia velocidad a 0 si desea que se detenga. Vea el ejemplo de código más abajo.

Sintaxis:

mover_hacia_el_punto( x, y, sp );

Argumento Descripción
x La posición x del punto hacia el que hay que moverse.
y La posición y del punto hacia el que hay que moverse.
sp La velocidad a la que se debe mover en píxeles por paso.

Devuelve:

N/A

Ejemplo:

si punto_distancia(x, y, objetivo.x, objetivo.y) > 5
{
mover_hacia_el_punto(objetivo.x, objetivo.y, 5);
}
else speed = 0;

El código anterior moverá la instancia hacia la posición de la instancia indexada en la variable "target" a una velocidad de 5 píxeles por paso. Una vez que alcanza esa posición la velocidad se pone a 0.
