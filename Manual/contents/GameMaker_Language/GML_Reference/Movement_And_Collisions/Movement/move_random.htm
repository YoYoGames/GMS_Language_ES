move_random

Con esta función puedes establecer la instancia en una posición en cualquier lugar de la habitación, pero alineada a una cuadrícula "invisible". Así que un valor de 32 para el hsnap y el vsnap establecerá la instancia a una posición aleatoria que está alineada a una cuadrícula de 32x32 cuadrados. puedes establecer estos valores a 1 para obtener una posición en cualquier lugar de la habitación.

NOTA: Esta función se ajustará a las mismas posiciones cada vez que el juego se ejecute de nuevo debido a que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que la depuración del código sea una tarea mucho más fácil. Para evitar este comportamiento utilice aleatorizar() al inicio de su juego. Esto sólo es cierto cuando las pruebas y la depuración del juego, como el final paquete ejecutable no mostrará este comportamiento y será aleatorio cada juego.

Sintaxis:

move_random(hsnap, vsnap);

Argumento Descripción
hsnap El ajuste horizontal (el tamaño en píxeles entre "celdas").
vsnap El ajuste vertical (el tamaño en píxeles entre "celdas").

Devuelve:

N/A

Ejemplo:

if keyboard_check(vk_space)
{
move_random(1,1);
}

Esto moverá la instancia a una posición aleatoria en cualquier parte de la habitación cuando se pulse la tecla espacio.
