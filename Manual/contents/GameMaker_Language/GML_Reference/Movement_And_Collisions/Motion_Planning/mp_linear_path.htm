mp_linear_path

Esta función calcula una trayectoria en línea recta desde la posición actual de la instancia hasta el punto especificado por los valores "xgoal" y "ygoal". Utiliza el tamaño de paso indicado, el mismo que en la función mp_paso_lineal(). El camino indicado debe existir ya y será sobrescrito por el nuevo camino y la función devolverá si se encontró un camino completo (true) o no (false). Si se devuelve false, se sigue generando un camino, pero sólo llegará hasta la posición en la que se bloqueó la ruta.

NOTA: Esta función no mueve la instancia. Sólo establece una ruta, y debe utilizar las funciones de ruta para el movimiento.

Sintaxis:

mp_linear_path(path, xgoal, ygoal, stepize, checkall);

Argumento Descripción
camino El índice de la ruta a utilizar.
xgoal La posición x del objetivo.
ygoal La posición y del objetivo.
tamaño del paso La velocidad a la que se mueve la instancia en píxeles por paso.
comprobar todo Si se comprueban las colisiones con todas las instancias (true) o sólo con las sólidas (false).

Devuelve:

Booleano

Ejemplo:

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
path_start(path, 2, 0, 0);
}

Esto hace que el objeto compruebe si hay un camino lineal desde su posición actual hasta la posición de "obj_Player". Si lo hay, se inicia el camino.
