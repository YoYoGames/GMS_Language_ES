mp_grid_destroy

Puede utilizar esta función para destruir el mp_grid indicado (como se define en la función mp_grid_create()) y liberar la memoria asignada. Es esencial que llames a esta función siempre que la rejilla de MP se termine o usted podría potencialmente obtener una fuga de memoria, lo que significa que su juego se ralentizará con el tiempo y eventualmente se bloqueará.

NOTA: El uso de las funciones mp_grid_* en una rejilla después de que haya sido destruida dará lugar a un error.

Sintaxis:

mp_grid_destroy(id);

Argumento Descripción
id Índice del mp_grid que se va a destruir

Devuelve:

N/A

Ejemplo:

si timer = 0
{
mp_grid_destroy(grid);
room_goto(rm_Menu);
}

El código anterior destruirá el mp_grid indexado en la variable "grid" si la variable "timer" es igual a 0, y luego pasará a otra sala.
