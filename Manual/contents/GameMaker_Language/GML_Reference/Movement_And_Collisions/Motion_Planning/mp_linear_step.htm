mp_linear_step

Con esta función le dices a una instancia que dé un "paso" hacia un punto específico, especificado por los valores xgoal y ygoal. El tamaño del paso (es decir, cuántos píxeles debe mover la instancia en cada paso) se indica mediante el tamaño del paso, y si la instancia ya está en la posición, se detendrá y no se moverá más, al contrario que otras funciones más sencillas como mover_hacia_un_punto(). El tamaño del paso es también la distancia "por delante" que el objeto comprobará en cada paso en busca de una colisión, y se puede elegir si la instancia se detiene ante una colisión con cualquier instancia o sólo con aquellas que estén marcadas como sólidas. La función devolverá si ha alcanzado la posición de meta (true) o si ha fallado (false).

NOTA: Esta función no intenta dar un rodeo si encuentra un obstáculo, simplemente falla y deja de moverse.

Sintaxis:

mp_linear_step(xobjetivo, yobjetivo, tamaño del paso, checkall);

Argumento Descripción
xgoal La posición x del objetivo.
ygoal La posición y del objetivo.
tamaño del paso La velocidad a la que se mueve la instancia en píxeles por paso.
comprobar todo Si se comprueban todas las instancias (true) o sólo las sólidas (false).

Devuelve:

Booleano

Ejemplo:

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy();
}

El código anterior hará que el objeto se mueva hacia el ratón a una velocidad de 5 píxeles por paso. Una vez que alcance la posición del ratón creará un objeto "obj_Explosion" y se destruirá a sí mismo.
