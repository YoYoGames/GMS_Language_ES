mp_linear_step_object

Con esta función le dices a una instancia que dé un "paso" hacia un punto específico, especificado por los valores xgoal y ygoal. El tamaño del paso (es decir, cuántos píxeles debe mover la instancia en cada paso) se indica mediante el tamaño del paso, y si la instancia ya está en la posición, se detendrá y no se moverá más, al contrario que otras funciones más sencillas como mover_hacia_un_punto(). El tamaño del paso es también la distancia "por delante" que el objeto comprobará en cada paso en busca de una colisión, y puedes elegir si la instancia se detiene en una colisión con un objeto o instancia de tu elección.

Sintaxis:

mp_linear_step_object(xgoal, ygoal, stepize, obj);

Argumento Descripción
xgoal La posición x del objetivo.
ygoal La posición y del objetivo.
tamaño del paso La velocidad a la que se mueve la instancia en píxeles por paso.
obj El objeto que debe bloquear la ruta. Puede ser un índice de objeto, un id de instancia o la palabra clave especial, todos

Devuelve:

Booleano

Ejemplo:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy();
}

El código anterior hará que el objeto se mueva hacia el ratón a una velocidad de 5 píxeles por paso, comprobando únicamente las colisiones con el objeto "obj_Wall". Una vez que alcance la posición del ratón creará un objeto "obj_Explosion" y se destruirá a sí mismo.
