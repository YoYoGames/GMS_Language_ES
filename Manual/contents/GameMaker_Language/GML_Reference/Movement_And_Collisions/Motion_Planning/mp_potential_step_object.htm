mp_potencial_paso_objeto

Esta función permite a la instancia dar un paso hacia una posición determinada definida por xgoal/ygoal, intentando al mismo tiempo evitar los obstáculos. Cuando la instancia se encuentre con una instancia del objeto especificado por el argumento "obj", cambiará la dirección del movimiento para tratar de evitar esa instancia y moverse alrededor de ella. No está garantizado que este enfoque funcione, pero en la mayoría de los casos fáciles hará que la instancia se mueva efectivamente hacia la meta. La función devuelve si el objetivo fue alcanzado o no.

Sintaxis:

mp_potencial_paso_objeto(xobjetivo, yobjetivo, tamaño del paso, objeto)

Argumento Descripción
xgoal La posición x del objetivo.
ygoal La posición y del objetivo.
tamaño del paso La velocidad a la que se mueve la instancia en píxeles por paso.
obj El objeto que debe bloquear la ruta de la instancia que ejecuta la función. Puede ser un índice de objeto, un id de instancia o la palabra clave especial,all.

Devuelve:

Booleano

Ejemplo:

if instance_exists(obj_Enemy)
{
var inst;
inst = instance_nearest(x, y, obj_Enemy);
mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

El código anterior primero comprueba si hay alguna instancia de "obj_Enemigo" en la sala. Si la hay, encuentra la más cercana y almacena su id en una variable. Esta variable se utiliza entonces para decirle a mp_potential_step_object que mueva la instancia con el código hacia la posición x/y del objeto que se encontró a una velocidad de 5 píxeles por paso mientras se evitan sólo las instancias del objeto "obj_Wall".
