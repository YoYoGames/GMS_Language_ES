mp_grid_clear_all

Con esta función puede limpiar una cuadrícula MP de todas las celdas "prohibidas".

Sintaxis:

mp_grid_clear_all(id);

Argumento Descripción
id Índice del mp_grid que se va a utilizar

Devuelve:

Booleano

Ejemplo:

if !instance_exists(obj_Player)
{
mp_grid_clear_all(grid);
}

El código anterior borrará el mp_grid indexado en la variable "grid", marcando todas las celdas como libres, si una instancia del objeto "obj_Player" ya no existe en la sala.
