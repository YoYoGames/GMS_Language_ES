mp_grid_get_cell

Con esta función se puede comprobar cualquier celda del mp_grid para ver si ha sido marcada como ocupada o no, dando el índice del mp_grid (previamente creado) y las coordenadas x e y de la celda a comprobar. Si ha sido ocupada o la posición que se está comprobando está fuera de los límites de la cuadrícula, la función devolverá -1, de lo contrario devolverá 0.

Sintaxis:

mp_grid_get_cell(id, x , y);

Argumento Descripción
id Índice del mp_grid que se va a utilizar
x1 La coordenada x de la cuadrícula a comprobar.
y1 La coordenada y de la cuadrícula a comprobar.

Devuelve:

Real

Ejemplo:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
image_blend = c_red;
}
else
{
image_blend = c_lime;
}

El código anterior comprobará la celda mp_grid que corresponde a la posición del ratón y si está ocupada pone la variable image_blend en rojo, y si no está ocupada la pone en verde.
