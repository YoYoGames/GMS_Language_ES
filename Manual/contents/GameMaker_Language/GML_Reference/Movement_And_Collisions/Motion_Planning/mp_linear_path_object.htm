mp_linear_path_object

Esta función calcula una trayectoria en línea recta desde la posición actual de la instancia hasta el punto especificado por los valores "xgoal" y "ygoal". Utiliza el tamaño de paso indicado, el mismo que en la función mp_paso_lineal(). El camino indicado debe existir ya y será sobrescrito por el nuevo camino y la función devolverá si se encontró un camino completo (true) o no (false). Si se encuentra una ruta completa sólo es que no hubo colisión con el objeto o instancia especificada y si se devuelve false, se sigue generando una trayectoria, pero sólo llegará hasta la posición en la que se bloqueó la trayectoria.

NOTA: Esta función no mueve la instancia. Sólo establece una ruta, y debe utilizar las funciones de ruta para el movimiento.

Sintaxis:

mp_linear_path_object(path, xgoal, ygoal, stepize, obj);

Argumento Descripción
camino El índice de la ruta a utilizar
xgoal La posición x del objetivo.
ygoal La posición y del objetivo.
tamaño del paso La velocidad a la que se mueve la instancia en píxeles por paso.
obj El objeto que debe bloquear la ruta. Puede ser un índice de objeto, un id de instancia o la palabra clave especial,all

Devuelve:

Booleano

Ejemplo:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
path_start(path, 4, 0, 0);
}

El código anterior comprueba si hay una colisión con "obj_Wall" a lo largo de la trayectoria entre el objeto que ejecuta el código y la posición x/y del ratón. Si no hay colisión y se genera la trayectoria completa, entonces iniciará el objeto a lo largo de la ruta generada.
