punto_en_triángulo

Cuando se utiliza esta función, se define un área triangular y GameMaker Studio 2 calculará si el punto dado cae dentro de sus límites o no. Si el punto cae dentro del triángulo definido, la función devolverá true, de lo contrario la función devolverá false.

Sintaxis:

punto_en_triángulo(px, py, x1, y1, x2, y2, x3, y3);

Argumento Descripción
px La coordenada x del punto a comprobar.
py La coordenada y del punto a comprobar.
x1 La coordenada x de la primera esquina del triángulo a comprobar.
y1 La coordenada y de la primera esquina del triángulo a comprobar.
x2 La coordenada x de la segunda esquina del triángulo a comprobar.
y2 La coordenada y de la segunda esquina del triángulo a comprobar.
x3 La coordenada x de la tercera esquina del triángulo a comprobar.
y3 La coordenada y de la tercera esquina del triángulo a comprobar.

Devuelve:

Booleano

Ejemplo:

var inst = instance_nearest(x, y, obj_Player);
if instance_exists(inst)
{
var x1 = x + lengthdir_x(100, image_angle - 45);
var y1 = y + lengthdir_y(100, image_angle - 45);
var x2 = x + lengthdir_x(100, image_angle + 45);
var y2 = y + lengthdir_y(100, image_angle + 45);
if point_in_triangle(inst.x, inst.y, x, y, x1, y1, x2, y2)
{
can_see = true;
}
}

El código anterior utiliza la función point_in_triangle como un "cono de visión" para buscar una instancia de "obj_player", estableciendo una variable a true si la posición de los objetos x/y cae dentro del triángulo definido.
