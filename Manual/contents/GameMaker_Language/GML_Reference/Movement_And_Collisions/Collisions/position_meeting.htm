posición_reunión

Con esta función puedes comprobar una posición para una colisión con otra instancia o con todas las instancias de un objeto. Cuando se utiliza esta función se comprueba un único punto en la sala para una instancia o un objeto. Esta comprobación se hará contra el cuadro delimitador de la instancia o contra la máscara. de la instancia o contra la máscara de la instancia si esa instancia tiene colisiones precisas comprobadas. Si necesita obtener la instancia única id si el objeto con el que se está colisiona con usted debe utilizar posición_de_instancia(). Esta función también acepta las palabras clave especiales all y other.

Sintaxis:

posición_reunión(x, y, obj);

Argumento Descripción
x La posición x a comprobar.
y La posición y a comprobar.
obj El objeto (o id de instancia, o las palabras clave "todos" u "otros") con el que se comprobará la colisión.

Devuelve:

Booleano

Ejemplo:

if mouse_check_button(mb_left)
{
if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
}

El código anterior comprueba el botón izquierdo del ratón, y si está pulsado comprueba la posición x/y del ratón para ver si hay una colisión con alguna instancia. Si no hay ninguna, se crea una instancia de "obj_Wall".
