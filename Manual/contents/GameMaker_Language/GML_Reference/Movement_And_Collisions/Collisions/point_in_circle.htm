punto_en_círculo

Cuando se utiliza esta función, se define un área circular y GameMaker Studio 2 calculará si el punto dado cae dentro de sus límites o no. Si el punto cae dentro del círculo definido, la función devolverá true; en caso contrario, la función función devolverá falso.

Sintaxis:

punto_en_circulo(px, py, x1, y1, rad);

Argumento Descripción
px La coordenada x del punto a comprobar.
py La coordenada y del punto a comprobar.
x1 La coordenada x del centro del círculo.
y1 La coordenada y del centro del círculo.
rad El radio del círculo.

Devuelve:

Booleano

Ejemplo:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
over = true;
}
si no
{
over = false;
}

El código anterior utiliza la función point_in_circle para comprobar si la posición del ratón cae dentro del área circular definida, estableciendo la variable "over" como verdadera en caso de que así sea, o falsa en caso contrario.
