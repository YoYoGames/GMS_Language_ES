lista de líneas de colisión

Esta función es la misma que la función línea_de_colisión() sólo que en lugar de detectar una instancia en colisión a la vez, detectará múltiples instancias. Usted suministra las posiciones x/y del inicio y final de la línea a comprobar junto con el objeto a comprobar, y puede establecer que la comprobación sea precisa (en cuyo caso todas las instancias que se comprueben tendrán que tener máscaras de colisión precisas ) y si la comprobación debe incluir la instancia que llama o no.

También se suministra una lista de DS, por lo que los valores de id valores de cualquier instancia que colisione con la instancia que llama se se añadirán al final de la lista dada. Puede borrar la lista antes de llamar a esta función para que sólo contenga los resultados de esta llamada a la función. También tiene la opción de ordenar las instancias en base a sus distancias desde el centro de la línea a sus orígenes. La función devuelve el número de instancias encontradas, o 0 si no se encuentra ninguna.

Tenga en cuenta que en lugar de un índice de objeto puede suministrar la palabra clave de instancia all, para buscar todas las instancias en la sala actual, que puede incluir la instancia que ejecuta el código (dependiendo del argumento notme ).

Sintaxis:

lista_de_colisiones(x1, y1, x2, y2, obj, prec, notme, list, ordered);

Argumento Descripción
x1 La coordenada x del inicio de la línea.
y1 La coordenada y del inicio de la línea.
x2 La coordenada x del final de la línea.
y2 La coordenada y del final de la línea.
obj El objeto para comprobar las colisiones de instancia.
prec Si la comprobación se basa en colisiones precisas(verdadero, que es más lento) o en su caja delimitadora en general(falso, más rápido).
notme Si la instancia que llama, si es relevante, debe ser excluida(true) o no(false).
lista La lista DS que se utilizará para almacenar los ID de las instancias que colisionan.
ordenado Si la lista debe ser ordenada por distancia(true) o no(false).

Devuelve:

Entero (El número de instancias que se encuentran en colisión)

Ejemplo:

var _list = ds_list_create();
var _num = collision_line_list(x - 100, y, x + 100, y, obj_Enemy, false, true, _list, false);
if _num > 0
{
for (var i = 0; i < _num; ++i;)
{
instance_destroy(_list[| i]);
}
}
ds_list_destroy(_list);

El código anterior comprobará una línea de 100 píxeles a cada lado de la posición de la instancia de llamada para las colisiones con las instancias de "obj_Enemy". Si hay alguna colisión, se recorre la lista precreada y se destruye cada instancia que haya estado en la colisión. colisión es destruida.
