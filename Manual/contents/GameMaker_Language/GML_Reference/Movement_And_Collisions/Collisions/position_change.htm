cambio_de_posición

Esta función comprobará una posición en busca de una colisión con cualquier instancia en el punto dado, y si hay una, cambiará todas las instancias en colisión para que sean instancias del objeto elegido. Puede establecer el argumento "perf" a true que forzará a GameMaker Studio 2 a realizar los eventos Destroy y Clean Up de la instancia encontrada así como el evento Create de la nueva instancia, o false para no realizar ninguno de estos eventos. Por favor, tenga en cuenta que si elige no realizar los eventos Destroy, Clean Up y Create, cualquier instancia creada que utilice una variable normalmente definida en el evento create, bloqueará el juego ya que esa variable ya no existirá. existirá.

Sintaxis:

position_change(x, y, obj, perf);

Argumento Descripción
x La coordenada x de dónde cambiar las instancias de colisión.
y La coordenada y de dónde cambiar las instancias de colisión.
obj El nuevo objeto en el que se convertirá el objeto llamado.
perf Si se realiza el evento Create de ese nuevo objeto (true) o no (false).

Devuelve:

N/A

Ejemplo:

position_change(200, 200, obj_Bird, true);

Esto cambiará todas las instancias que colisionan en el punto (200,200) en una instancia de "obj_Bird", realizando el evento Create de "obj_Bird" para cada una de ellas en el proceso.
