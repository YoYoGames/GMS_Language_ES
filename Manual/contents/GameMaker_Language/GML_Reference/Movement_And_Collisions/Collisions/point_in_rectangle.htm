punto_en_rectángulo

Cuando se utiliza esta función, se define un área rectangular y GameMaker Studio 2 calculará si el punto dado cae dentro de sus límites o no. Si el punto cae dentro del rectángulo definido, la función devolverá true, de lo contrario la función devolverá false.

Sintaxis:

punto_en_rectángulo(px, py, x1, y1, x2, y2);

Argumento Descripción
px La coordenada x del punto a comprobar.
py La coordenada y del punto a comprobar.
x1 La coordenada x del lado izquierdo del rectángulo a comprobar.
y1 La coordenada y del lado superior del rectángulo a comprobar.
x2 La coordenada x del lado derecho del rectángulo a comprobar.
y2 La coordenada y del lado inferior del rectángulo a comprobar.

Devuelve:

Booleano

Ejemplo:

if point_in_rectangle(mouse_x, mouse_y, x -10, y - 10, x + 10, y + 10)
{
audio_play_sound(snd_click, 0, false);
}

Este breve código comprueba la posición del ratón respecto al área rectangular definida y reproduce un sonido si cae dentro de los límites.
