lugar_vacío

Puedes utilizar esta función para comprobar si la instancia que llama colisiona con cualquier otra instancia de un objeto o con todas las instancias de tu juego. Para que esta colisión se resuelva correctamente, la instancia que ejecuta el código debe tener una colisión válida válida (ya sea para el propio sprite, o a través del índice_de_máscara) y sólo registrará colisiones con aquellas instancias que también tengan una máscara válida.

La función comprueba si no hay colisiones en caso de que la instancia que llama se coloque en una posición específica, y puede proporcionar un argumento opcional para refinar la comprobación para que sólo compruebe si una posición está libre de colisiones con instancias del tipo dado dado. Tenga en cuenta que si no se suministra un ID de objeto opcional, la comprobación se realizará con todas las instancias de la sala. La comprobación de las colisiones será precisa o se basará en el cuadro delimitador de la instancia, dependiendo del tipo de máscara de colisión pero para que las colisiones precisas funcionen correctamente, todas las instancias en la comprobación deben tener máscaras de colisión precisas.

Tenga en cuenta que las coordenadas x/y dadas se rebajarán al número entero más cercano antes de realizar la comprobación.

Sintaxis:

place_empty(x, y, [object_id]);

Argumento Descripción
x La posición x a comprobar.
y La posición y a comprobar.
[object_id] Opcional! El objeto a comprobar.

Devuelve:

Booleano

Ejemplo:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
x = mouse_x;
y = mouse_y;
}

El código anterior comprobará si hay una colisión con cualquier otra instancia del objeto "obj_Enemigo", como si la instancia que llama se colocara en la misma posición que el ratón. Si no se detecta ninguna colisión, la instancia tiene sus coordenadas x/y a las del ratón.
