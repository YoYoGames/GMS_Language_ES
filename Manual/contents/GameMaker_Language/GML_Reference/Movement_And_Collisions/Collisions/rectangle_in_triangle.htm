rectángulo_en_triángulo

Esta función comprobará un área rectangular que usted defina para ver si no está en colisión, si está completamente dentro de los límites de destino, o si simplemente está tocando, un área triangular definida. Si no se tocan en absoluto, la función devolverá devolverá 0, si el origen está completamente dentro del destino devolverá 1, y si simplemente se superponen entonces devolverá 2. La imagen siguiente lo ilustra:

Sintaxis:

rectángulo_en_triángulo(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2, dx3, dy3);

Argumento Descripción
sx1 La coordenada x del lado izquierdo del rectángulo de origen.
sy1 La coordenada y del lado superior del rectángulo de origen.
sx2 La coordenada x del lado derecho del rectángulo de origen.
sy2 La coordenada y del lado inferior del rectángulo de origen.
x1 La coordenada x del primer lado del triángulo a comprobar.
y1 La coordenada y del primer lado del triángulo a comprobar.
x2 La coordenada x del segundo lado del triángulo a comprobar.
y2 La coordenada y del segundo lado del triángulo a comprobar.
x3 La coordenada x del tercer lado del triángulo a comprobar.
y3 La coordenada y del tercer lado del triángulo a comprobar.

Devuelve:

Real

Ejemplo:

var inst = instance_nearest(x, y, obj_Player);
if instance_exists(inst)
{
var x1 = x + lengthdir_x(100, image_angle - 45);
var y1 = y + lengthdir_y(100, image_angle - 45);
var x2 = x + lengthdir_x(100, image_angle + 45);
var y2 = y + lengthdir_y(100, image_angle + 45);
if rectangle_in_triangle(inst.bbox_left, inst.bbox_top, inst.bbox_right, inst.bbox_bottom, x, y, x1, y1, x2, y2) == 1
{
can_see = true;
}
}

El código anterior utiliza la función rectángulo_en_triángulo como un "cono de visión" para buscar una instancia de "obj_player". Si el cuadro delimitador completo está dentro del área triangular dada, establecerá una variable como verdadera.
