ds_existe

Con esta función puede comprobar si existe una estructura de datos del tipo dado. Se suministra el valor del "índice" (como se mantiene en una variable) y el "tipo" del DS, que puede ser cualquiera de las constantes que se enumeran a continuación, y la función devolverá true si la estructura de datos existe y false en caso contrario.

Constante Descripción
ds_type_map Una estructura de datos de mapa
ds_type_list Una estructura de datos de lista
ds_type_stack Una estructura de datos de pila
ds_type_grid Una estructura de datos de cuadrícula
ds_type_queue Una estructura de datos de cola
ds_type_priority Una estructura de datos prioritarios

Sintaxis:

ds_exists(ind, type);

Argumento Descripción
ind El índice de la variable para comprobar la estructura de datos
tipo El tipo de estructura de datos que se va a comprobar (véase la lista de constantes más abajo)

Devuelve:

Booleano

Ejemplo:

if !ds_exists(ai_grid, ds_type_grid)
{
ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

El código anterior comprueba la variable (previamente inicializada) "ai_grid" para ver si indexa una estructura de datos de tipo grid DS, y si no lo hace entonces crea una y almacena su índice en la variable.
