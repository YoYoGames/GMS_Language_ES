ds_grid_value_disk_x

Con esta función se puede obtener la coordenada x (dentro de la región en forma de disco de la cuadrícula dada) del valor que se busca. Se da el índice de la cuadrícula DS (devuelto por ds_grid_create()) junto con las posiciones x/y para la celda central del disco. A continuación, se da el radio (como un valor entero) alrededor de la celda central a buscar, antes de suministrar el valor a buscar. Si el valor buscado no existe, la función devolverá -1.

Sintaxis:

ds_grid_value_disk_x(index, xm, ym, r, val);

Argumento Descripción
índice El índice de la cuadrícula.
xm La posición x del disco en la cuadrícula.
ym La posición y del disco en la cuadrícula.
r El radio del disco en la cuadrícula.
val El valor a encontrar.

Devuelve:

Real

Ejemplo:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

El código anterior comprueba un ds_grid para un valor específico dentro de una región del disco. si se encuentra, entonces almacena la posición x e y del valor en dos variables para su uso posterior.
