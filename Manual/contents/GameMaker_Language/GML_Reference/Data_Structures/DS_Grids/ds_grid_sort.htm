ds_grid_sort

Esta función se puede utilizar para ordenar una cuadrícula DS en base a los valores de una columna determinada (de forma parecida a como se ordenan los archivos por fecha, tamaño, etc. en el explorador de archivos del sistema operativo). La siguiente imagen muestra un ejemplo:

Sintaxis:

ds_grid_sort(índice, columna, ascendente);

Argumento Descripción
índice El índice de la cuadrícula a ordenar.
columna La columna que se utilizará para ordenar las filas
ascendente Si se ordena de menor a mayor(true), o de mayor a menor(false).

Devuelve:

N/A

Ejemplo:

ds_grid_sort(grid, 3, false)

Esto tomaría todos los valores de la rejilla DS indexados en la variable "rejilla" y los ordenaría según los valores encontrados en la 3ª columna de la rejilla (como se muestra en la imagen anterior).
