ds_grid_copy

Con esta función se puede copiar el contenido de una rejilla en otra. Ambas rejillas deben haber sido creadas previamente con la función ds_grid_create() función.

Sintaxis:

ds_grid_copy(destino, origen);

Argumento Descripción
destino Este índice de la cuadrícula a copiar.
fuente Este índice de la rejilla para copiar.

Devuelve:

N/A

Ejemplo:

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

El código anterior crea una nueva rejilla DS, basada en la anchura y la altura de una rejilla creada anteriormente, y luego copia la información de la rejilla anterior a la nueva. Finalmente, borra la rejilla antigua para que todas las celdas tengan el valor -1.
