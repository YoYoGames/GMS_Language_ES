ds_grid_destroy

Esta función eliminará de la memoria la estructura de datos de la rejilla dada, liberando los recursos que estaba utilizando y eliminando todos los valores que contenía. Esta función debe usarse siempre cuando se termine de usar la rejilla DS para evitar fugas de memoria que pueden ralentizar y bloquear el juego.

IMPORTANTE! Cuando se crea una estructura de datos, el valor del índice para identificarla es un valor entero que comienza en 0. Esto significa que las estructuras de datos de diferentes tipos pueden tener el mismo valor de índice, por lo que en caso de duda se debe utilizar la función ds_exists() antes de acceder a ellas. También hay que tener en cuenta que los índices se reutilizan, por lo que el valor del índice de una estructura de datos destruida puede ser utilizado por otra recién creada posteriormente, por lo que recomendamos siempre establecer la variable que contenía el índice DS a -1 después de la destrucción.

Sintaxis:

ds_grid_destroy(índice);

Argumento Descripción
índice Este índice de la cuadrícula a destruir.

Devuelve:

N/A

Ejemplo:

if (lives == 0)
{
ds_grid_destroy(Wall_Grid);
Wall_Grid = -1;
room_goto(rm_Menu);
}

El código anterior comprobará el valor de la variable global incorporada "lives" y si es 0, destruye la rejilla DS indexada en la variable "Wall_Grid" y luego cambia de habitación.
