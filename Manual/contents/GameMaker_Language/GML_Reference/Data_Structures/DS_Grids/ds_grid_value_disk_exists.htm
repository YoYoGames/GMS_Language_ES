ds_grid_value_disk_exists

Con esta función se puede comprobar si un valor específico (real o cadena) está presente dentro de un área circular de una cuadrícula DS dada. Si está presente la función devolverá true, de lo contrario devolverá false.

Sintaxis:

ds_grid_value_disk_exists(index, xm, ym, r, val);

Argumento Descripción
índice El índice de la cuadrícula.
xm La posición x del disco en la cuadrícula.
ym La posición y del disco en la cuadrícula.
r El radio del disco en la cuadrícula.
val El valor a encontrar.

Devuelve:

Booleano

Ejemplo:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

El código anterior comprueba una cuadrícula DS para un valor específico dentro de una región del disco. si se encuentra, entonces almacena la posición x e y del valor en dos variables para su uso posterior.
