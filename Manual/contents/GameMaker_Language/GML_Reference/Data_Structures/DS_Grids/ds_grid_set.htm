ds_grid_set

Esta función se puede utilizar para establecer una celda dada dentro de la cuadrícula DS dada a cualquier valor, que puede ser un número real o una cadena. La imagen siguiente lo ilustra:

Sintaxis:

ds_grid_set(índice, x, y, valor);

Argumento Descripción
índice Este índice de la red.
x La posición x de la celda a fijar.
y La posición y de la celda a fijar.
valor El valor con el que se va a fijar la celda.

Devuelve:

N/A

Ejemplo:

grid = ds_grid_create(5, 5);
var i = 0;
var j = 0;
repeat (ds_grid_width(grid))
{
repeat (ds_grid_height(grid))
{
ds_grid_set(grid, i, j, irandom(9));
j += 1;
}
j = 0;
i += 1;
}

El código anterior crea una cuadrícula y almacena su índice en la variable "grid". A continuación, rellena esta cuadrícula con enteros aleatorios de 0 a 9.
