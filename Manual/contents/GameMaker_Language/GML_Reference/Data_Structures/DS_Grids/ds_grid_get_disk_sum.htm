ds_grid_get_disk_sum

Esta función puede utilizarse para sumar todos los valores de todas las celdas que se encuentran dentro del área de disco definida de una cuadrícula, como se muestra en la imagen siguiente:

Sintaxis:

ds_grid_get_disk_sum(index, xm, ym, r);

Argumento Descripción
índice El índice de la cuadrícula.
xm La posición x del disco en la cuadrícula.
ym La posición y del disco en la cuadrícula.
r El radio del disco en la cuadrícula.

Devuelve:

Real o Cadena

Ejemplo:

val = ds_grid_get_disk_sum(grid, 5, 5, 2)

El código anterior establecerá la variable "val" a la suma de todos los valores contenidos dentro del disco dado de la cuadrícula DS indexada en la variable "grid".
