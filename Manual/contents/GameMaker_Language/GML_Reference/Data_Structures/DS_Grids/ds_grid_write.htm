ds_grid_write

Esta función se puede utilizar para convertir el ds_grid dado en una cadena, que luego se puede almacenar en un archivo externo (por ejemplo). Puede leer la cadena devuelta por esta función de nuevo en un ds_grid utilizando la función ds_grid_read().

NOTA: La cadena devuelta no es una cadena legible, sino un volcado del contenido de la estructura de datos.

Sintaxis:

ds_grid_write(índice);

Argumento Descripción
índice El índice de la cuadrícula a escribir.

Devuelve:

Cadena

Ejemplo:

ini_open("Guardar.ini");
ini_write_string("Guardar", "0", ds_grid_write(mygrid));
ini_close()

El código anterior abrirá un archivo ini (creándolo si no existe ya) y luego escribirá la ds_grid dada como una cadena en ese archivo.
