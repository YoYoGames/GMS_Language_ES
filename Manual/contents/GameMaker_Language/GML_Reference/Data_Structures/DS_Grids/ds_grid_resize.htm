ds_grid_resize

Con esta función puedes redimensionar la rejilla DS dada para que tenga una anchura y/o altura diferente. Si el tamaño de la rejilla es mayor que el de la rejilla actual, las nuevas celdas tendrán un valor base de 0, y si el tamaño es menor entonces los valores mantenidos en las celdas que ya no están dentro del nuevo tamaño se perderán. El resto de las celdas no se modificarán.

Sintaxis:

ds_grid_resize(index, w, h);

Argumento Descripción
índice Este índice de la cuadrícula a redimensionar.
w La nueva anchura de la cuadrícula.
h La nueva altura de la cuadrícula.

Devuelve:

N/A

Ejemplo:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

El código anterior redimensionará la cuadrícula DS indexada en la variable global "Grid" y luego la borrará para que cada celda tenga el valor -1.
