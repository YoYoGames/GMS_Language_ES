ds_grid_value_exists

Con esta función puede comprobar si un valor específico (real o cadena) está presente dentro de un área rectangular de una cuadrícula DS dada. Si está presente la función devolverá true, de lo contrario devolverá false.

Sintaxis:

ds_grid_value_exists(index, x1, y1, x2, y2, val);

Argumento Descripción
índice El índice de la cuadrícula.
x1 La posición x de la izquierda de la región en la cuadrícula.
y1 La posición y de la parte superior de la región en la cuadrícula.
x2 La posición x de la derecha de la región en la cuadrícula.
y2 La posición y del fondo de la región en la cuadrícula.
val El valor a encontrar.

Devuelve:

Booleano

Ejemplo:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

El código anterior comprueba una cuadrícula DS en busca de un valor específico dentro de una región rectangular. Si lo encuentra, entonces almacena la posición x e y del valor en dos variables para su uso posterior.
