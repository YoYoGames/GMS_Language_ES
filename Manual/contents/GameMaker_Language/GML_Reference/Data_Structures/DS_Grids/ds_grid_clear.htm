ds_grid_clear

Esta función se puede utilizar para borrar una cuadrícula DS dada a un valor específico. Todas las celdas de la rejilla contendrán este valor, que puede ser un número real o una cadena. La imagen siguiente ilustra cómo funciona esto:

Sintaxis:

ds_grid_clear(index, val);

Argumento Descripción
índice Este índice de la rejilla a limpiar.
val El nuevo valor para todas las celdas de la cuadrícula.

Devuelve:

N/A

Ejemplo:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

El código anterior redimensionará la cuadrícula DS indexada en la variable global "Grid" y luego la borrará para que cada celda tenga el valor -1.
