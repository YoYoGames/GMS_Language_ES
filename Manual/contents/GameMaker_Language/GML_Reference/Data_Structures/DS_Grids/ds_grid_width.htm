ds_grid_width

Esta función devolverá el ancho de la cuadrícula dada. Este valor es el número de celdas que tiene la cuadrícula a lo largo del eje x y siempre es un número entero, como se muestra en la imagen siguiente:

Sintaxis:

ds_grid_width(index);

Argumento Descripción
índice Este índice de la cuadrícula para encontrar el ancho de.

Devuelve:

Real

Ejemplo:

for (var i = 0; i < ds_grid_width(grid); ++i;)
{
for (var j = 0; j < ds_grid_height(grid); ++j;)
{
if (ds_grid_get(grid, i, j) == 1)
{
instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
}
}
}

El código anterior hará un bucle a través de la cuadrícula DS indexada en la variable "grid" y si el valor encontrado en cualquier celda específica es igual a 1, entonces creará una instancia de "obj_Wall" en la posición apropiada dentro de la habitación.
