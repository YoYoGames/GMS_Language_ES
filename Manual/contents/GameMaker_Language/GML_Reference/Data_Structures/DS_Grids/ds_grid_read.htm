ds_grid_read

Esta función puede ser utilizada para convertir una cadena que ha sido creada previamente por la función ds_grid_write() en una cuadrícula DS. La rejilla DS debe haber sido creada previamente (ver el ejemplo de abajo). Tenga en cuenta que si la estructura de datos fue creada con versiones anteriores de GameMaker debe añadir el argumento opcional "legacy", poniéndolo a true ya que el formato de la cadena ha cambiado con esta versión.

Sintaxis:

ds_grid_read(index, string [, legacy]);

Argumento Descripción
índice El índice de la cuadrícula a leer.
cadena La cadena a leer en la rejilla DS.
legado(opcional) Puede ser verdadero o falso u omitirse por completo.

Devuelve:

N/A

Ejemplo:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close()

El código anterior crea una cuadrícula DS basada en el tamaño de la habitación (cada cuadrado de 32x32 píxeles representa una celda de la cuadrícula) y luego lee un conjunto de datos de la cuadrícula previamente guardados desde un archivo ini en la nueva cuadrícula DS.
