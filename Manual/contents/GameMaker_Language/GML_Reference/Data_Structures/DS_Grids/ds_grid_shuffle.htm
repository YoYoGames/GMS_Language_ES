ds_grid_shuffle

Esta función puede utilizarse para aleatorizar las posiciones de todos los valores en todas las celdas de una cuadrícula.

NOTA: Esta función barajará las celdas de la rejilla en las mismas posiciones cada vez que el juego se ejecute de nuevo debido a que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que la depuración del código sea una tarea mucho más fácil. Para evitar este comportamiento utilice aleatorizar() al inicio de su juego. Esto sólo es cierto cuando las pruebas y la depuración del juego, como el ejecutable final no mostrará este comportamiento y será aleatorio cada vez que se juegue.

Sintaxis:

ds_grid_shuffle(índice);

Argumento Descripción
índice El índice de la cuadrícula a barajar.

Devuelve:

N/A

Ejemplo:

ds_grid_shuffle(grid)

Esto tomaría todos los valores de la cuadrícula DS indexados en la variable "grid" y los barajaría para obtener un orden diferente.
