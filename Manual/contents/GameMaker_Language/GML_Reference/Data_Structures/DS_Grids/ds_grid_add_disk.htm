ds_grid_add_disk

Esta función puede utilizarse para añadir un valor dado (real o de cadena) a todos los valores de las celdas que se encuentran dentro del área de disco definida de una cuadrícula. El valor que se añada debe ser del mismo tipo que el de las celdas de la cuadrícula, es decir, no se puede añadir una cadena a un real o viceversa, y para las cadenas esto corresponde a la concatenación.

Sintaxis:

ds_grid_add_disk(index, xm, ym, r, val);

Argumento Descripción
índice El índice de la cuadrícula.
xm La posición x del disco en la cuadrícula.
ym La posición y del disco en la cuadrícula.
r El radio del disco en la cuadrícula.
val El valor a añadir a las celdas dentro del disco.

Devuelve:

N/A

Ejemplo:

ds_grid_add_disk(grid, 7, 6, 5, 2)

Esto sumaría 2 a todos los valores contenidos en las celdas dentro del área de disco definida de la cuadrícula DS referenciada por la variable "grid".
