ds_map_secure_load

Esta función cargará un mapa DS previamente guardado desde el archivo dado. Este archivo debe haber sido creado previamente con la función ds_map_secure_save() y, una vez cargado, la función devolverá el índice del mapa DS que se ha creado a partir de los datos cargados. Este índice del mapa DS debe almacenarse en una variable y utilizarse para todas las llamadas posteriores de la función a este mapa. Tenga en cuenta que si el mapa DS que se está cargando se guardó con un array como uno de los valores clave valores clave, este array se habrá convertido en una lista DS en la carga.

IMPORTANTE! Una de las características de un archivo guardado de forma segura es que está bloqueado para el dispositivo en el que fue creado, por lo que no se puede cargar un archivo guardado en un dispositivo en un proyecto que se ejecuta en otro dispositivo.

Sintaxis:

ds_map_secure_load(nombre de archivo);

Argumento Descripción
nombre de archivo El nombre del archivo para cargar los datos del mapa.

Devuelve:

Entero (índice del mapa DS)

Ejemplo:

p_map = ds_map_secure_load("p_data.dat");

El código anterior cargará un mapa DS guardado de forma segura y almacenará su valor de índice en una variable para su uso futuro.
