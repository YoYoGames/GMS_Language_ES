ds_map_replace_map

Con esta función se puede reemplazar un mapa DS que ha sido almacenado en la "clave" dada por otro mapa que ha sido creado previamente. Esta función está diseñada para crear mapas compatibles con JSON que luego codificarías utilizando json_encode() y sólo debe utilizarse junto con esa función.

Sintaxis:

ds_map_replace_map(id, key, value)

Argumento Descripción
id El id del ds_map a utilizar.
clave La clave para reemplazar.
valor El id del ds_map a utilizar para reemplazar el almacenado previamente en la clave dada.

Devuelve:

N/A

Ejemplo:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

El código anterior creará un Mapa DS y lo poblará con un array y una Lista DS antes de reemplazar un mapa previamente almacenado en el Mapa DS "j_map".
