ds_map_secure_load_buffer

Esta función cargará un mapa DS guardado de forma segura desde un buffer. Debe haber cargado previamente el buffer en la memoria (usando buffer_load()) y luego pasar eso a esta función devolverá un mapa DS con el contenido del buffer. Tenga en cuenta que el buffer debe haber sido creado usando la función ds_map_secure_save_buffer()para que esto funcione correctamente, y también tenga en cuenta que si el mapa DS que se está cargando contiene un array, éste se convertirá en una lista DS cuando se cargue.

IMPORTANTE! Una de las características de un archivo guardado de forma segura es que está bloqueado para el dispositivo en el que fue creado, por lo que no se puede cargar un archivo guardado en un dispositivo en un proyecto que se ejecuta en otro dispositivo.

Sintaxis:

ds_map_secure_load_buffer(buffer);

Argumento Descripción
nombre de archivo El ID del búfer del que se cargarán los datos del mapa.

Devuelve:

Entero (índice del mapa DS)

Ejemplo:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

El código anterior cargará un mapa DS guardado de forma segura desde un buffer y almacenará su valor de índice en una variable para su uso futuro.
