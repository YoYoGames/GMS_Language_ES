ds_map_find_value

Con esta función se puede obtener el valor de una clave especificada. Los valores de entrada de la función son el mapa DS (previamente creado) a utilizar y la clave a comprobar.

NOTA: Si no existe dicha clave, la función devolverá undefined. Siempre debe comprobarlo utilizando la función is_undefined() función

Sintaxis:

ds_map_find_value(id, key)

Argumento Descripción
id El id del mapa a utilizar.
clave La clave para encontrar.

Devuelve:

Real, cadena o indefinido

Ejemplo:

cantidad = ds_map_find_value(inventory, "food");

O bien, utilizando el accesorio de mapa "?":

cantidad = inventario[? "comida"];

El código anterior obtendrá el valor de la clave "comida" y lo almacenará en la variable "cantidad".
