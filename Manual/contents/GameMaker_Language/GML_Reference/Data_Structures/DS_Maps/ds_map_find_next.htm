ds_map_find_next

Esta función devuelve la siguiente clave almacenada en el mapa DS después de la especificada en la función. Esto puede ser útil si tiene que iterar a través del mapa DS buscando algo, pero debe evitarse si es posible ya que puede ser lento. Si no existe dicha la función devolverá undefined. Siempre debe comprobarlo utilizando la función is_undefined() función

Sintaxis:

ds_map_find_next(id, key);

Argumento Descripción
id El id del mapa a utilizar.
clave La clave para encontrar el siguiente a.

Devuelve:

Real, cadena o indefinido

Ejemplo:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if (key != "gold")
{
key = ds_map_find_next(inventory, key);
}
else break;
}

El código anterior crea algunas variables temporales y luego obtiene el tamaño del mapa DS y encuentra la primera clave almacenada por el ordenador en el mapa. Luego utiliza un bucle for para iterar a través del mapa DS buscando el valor de la clave "oro". Si lo encuentra, rompe el bucle.
