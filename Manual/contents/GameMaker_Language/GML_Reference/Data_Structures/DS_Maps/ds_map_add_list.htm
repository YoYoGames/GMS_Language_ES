ds_map_add_list

Con esta función puedes asignar una lista DS (previamente creada) a una clave dentro del mapa DS dado. Esta función está diseñada para crear mapas compatibles con JSON que luego se codificarían utilizando json_encode() y sólo debe usarse junto con esa funcionalidad. Si un mapa DS tiene una lista añadida de esta manera, al destruir el mapa padre también se destruirán las listas contenidas y se liberará su memoria, y al llamar a ds_map_clear() en el mapa padre también destruirá y limpiará cualquier lista marcada.

Sintaxis:

ds_map_add_list(id, key, value)

Argumento Descripción
id El id del mapa a utilizar.
clave La clave para la lista añadida.
valor El id de la lista a añadir.

Devuelve:

N/A

Ejemplo:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

El código anterior creará una lista y la rellenará con los distintos valores de las variables globales. Esta lista es entonces "anidada" dentro de un mapa DS, y el mapa es entonces codificado en una cadena JSON, antes de que el mapa sea antes de que el mapa sea destruido, eliminando de la memoria el mapa y cualquier lista que contenga.
