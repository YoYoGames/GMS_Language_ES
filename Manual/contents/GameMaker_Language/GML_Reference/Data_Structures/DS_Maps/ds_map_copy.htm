ds_map_copy

Puede utilizar esta función para copiar el contenido de un mapa en otro que haya creado previamente utilizando ds_map_create(). Si el mapa DS al que se está copiando no está vacío entonces esta función lo borrará primero antes de copiarlo. El mapa DS original permanece inalterado por este proceso.

Sintaxis:

ds_map_copy(id, source);

Argumento Descripción
id El identificador del mapa que se está copiando
fuente El id del mapa del quese está copiando

Devuelve:

N/A

Ejemplo:

inventario_2 = ds_map_create();
ds_map_copy(inventario_2, inventario_1);

El código anterior creará un nuevo mapa y lo asignará a la variable "inventory_2". Luego copiará el contenido del mapa DS indexado en la variable "inventario_1" a este nuevo mapa.
