ds_map_write

Esta función convertirá los datos del mapa DS del mapa especificado en un formato de cadena que puede escribirse en un archivo *.ini o *.txt para facilitar su almacenamiento. Esta cadena puede ser leída posteriormente en un nuevo ds_map usando ds_map_read().

NOTA: La cadena devuelta no es una cadena legible, sino un volcado del contenido de la estructura de datos.

Sintaxis:

ds_map_write(id);

Argumento Descripción
id El id de la estructura de datos a utilizar

Devuelve:

Cadena

Ejemplo:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

El código anterior abre un archivo ini listo para ser escrito. Luego utiliza ds_map_write() para generar una cadena que se almacena en la variable temporal "t_string". Finalmente, escribe esa cadena en el archivo ini antes de cerrarlo.
