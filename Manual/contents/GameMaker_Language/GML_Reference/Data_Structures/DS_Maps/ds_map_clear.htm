ds_map_clear

Esta función borrará el mapa DS (previamente creado) de todos los pares clave/valor, pero no borrará el propio mapa DS de la memoria. Para ello debe utilizar la función ds_map_destroy(). Tenga en cuenta que al borrar un mapa desreferenciará cualquier estructura de datos almacenada en el mapa dando lugar a una fuga de memoria, por lo que necesitarás recorrerlo y destruir todos los elementos de la estructura de datos manualmente antes de borrar el mapa para evitar esto. La única vez que esto no es necesario es cuando se ha marcado algún elemento del mapa como lista DS o como otro mapa DS, en cuyo caso estos elementos se destruirán (¡no se borrarán!) y su memoria se limpiará automáticamente cuando se borre el cuando se borre el mapa principal.

Sintaxis:

ds_map_clear(id);

Argumento Descripción
id El id de la estructura de datos que se va a borrar.

Devuelve:

N/A

Ejemplo:

si global.new_game
{
ds_map_clear(inventario);
}

La función anterior comprobará una variable global para ver si es verdadera y, si lo es, entonces el mapa DS indexado en la variable de instancia "inventario" será limpiado de todos los pares clave/valor (pero no es destruido ).
