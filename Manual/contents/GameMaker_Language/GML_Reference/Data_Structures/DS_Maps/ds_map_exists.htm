ds_map_exists

Esta función devolverá true si la clave especificada existe en el mapa DS (previamente creado), y false si no existe.

Sintaxis:

ds_map_exists(id, key);

Argumento Descripción
id el id de la estructura de datos a comprobar
clave la clave para comprobar

Devuelve:

Booleano

Ejemplo:

if !ds_map_exists(inventory, "potions")
{
ds_map_add(inventario, "pociones", 1);
}

El código anterior buscará en el mapa de DS indexado en la variable "inventario" la clave "pociones" y si no existe la añadirá al mapa.
