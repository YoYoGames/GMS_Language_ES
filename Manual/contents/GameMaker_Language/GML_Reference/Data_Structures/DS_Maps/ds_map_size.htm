ds_map_size

Con esta función puede encontrar cuántos pares clave/valores contiene el mapa DS (previamente creado).

Sintaxis:

ds_map_size(id);

Argumento Descripción
id El id de la estructura de datos a comprobar.

Devuelve:

Real

Ejemplo:

si (ds_map_size(inventario) > 49)
{
full = true;
}

El código anterior comprobará el tamaño del mapa DS (es decir, el número de pares clave/valor) y si es mayor que 49, establece la variable "full" en true.
