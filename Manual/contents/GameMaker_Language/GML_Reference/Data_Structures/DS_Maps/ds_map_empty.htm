ds_map_empty

Esta función simplemente devolverá false si el mapa DS especificado (previamente creado) contiene algún par clave/valor, o true si no lo contiene.

Sintaxis:

ds_map_empty(id);

Argumento Descripción
id El id de la estructura de datos a comprobar.

Devuelve:

Booleano

Ejemplo:

si ds_map_empty(inventario)
{
peso = 0;
}

El código anterior comprueba si el mapa DS indexado en la variable "inventario" tiene algún par clave/valor y, si no lo tiene, pone la variable "peso" a 0.
