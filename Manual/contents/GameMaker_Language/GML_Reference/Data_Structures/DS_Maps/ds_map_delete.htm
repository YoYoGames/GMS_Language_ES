ds_map_delete

Con esta función se puede eliminar cualquier clave dada (y su correspondiente valor) del mapa DS dado, previamente creado.

Sintaxis:

ds_map_delete(id, key);

Argumento Descripción
id El id del mapa a cambiar.
clave La clave (junto con su valor asociado) a eliminar.

Devuelve:

N/A

Ejemplo:

ds_map_delete(inventory, "shield");

El código anterior borrará la clave "escudo" (y el valor con el que está emparejado) del mapa DS (inventario).
