ds_map_values_to_array

Con esta función puede recuperar todos los valores que contiene un mapa DS para cada clave del mapa. Se suministra el ID del mapa DS del que se van a obtener las claves (devuelto por ds_map_create()) y la función devolverá un array en el que cada entrada del array es un valor del mapa DS (por ejemplo, si el mapa tiene 10 claves, entonces se devolverá un array de 10 entradas con cada entrada correspondiente a un valor de una clave). a un valor de una clave). La función tiene un segundo argumento opcional en el que puede suministrar un array que haya creado, en cuyo caso los datos del valor de la clave del mapa se añadirán a cualquier dato existente en el array. Tenga en cuenta que la función modificará la matriz suministrada directamente, pero también devolverá una referencia a ella (o una referencia a una nueva matriz si no se suministra ninguna).

Sintaxis:

ds_map_values_to_array(id, [array])

Argumento Descripción
id El id del mapa a utilizar.
[array] OPCIONAL! El array al que añadir los valores del mapa DS.

Devuelve:

Array
