ds_map_find_last

Esta función devuelve la última clave almacenada en el mapa DS. No se trata de la última clave en el orden en que se han añadido. Los mapas DS no se almacenan de forma lineal, para ello se utilizan listas DS, así que todo lo que esto hace es encontrar la última clave tal y como la almacena el ordenador. Esto puede ser útil si tienes que iterar a través del mapa DS buscando algo, pero debe ser evitado si es posible ya que puede ser lento.

Tenga en cuenta que esta función devolverá undefined si el mapa DS dado está vacío.

Sintaxis:

ds_map_find_last(id);

Argumento Descripción
id El id del mapa a utilizar.

Devuelve:

Real, cadena o indefinido

Ejemplo:

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
if (key != "gold")
{
key = ds_map_find_previous(inventory, key);
}
else break;
}

El código anterior crea algunas variables temporales y luego obtiene el tamaño del mapa DS y encuentra la última clave almacenada por la computadora en el mapa. A continuación, utiliza un bucle for para iterar a través del mapa DS buscando el valor de la clave "oro". Si lo encuentra, rompe el bucle.
