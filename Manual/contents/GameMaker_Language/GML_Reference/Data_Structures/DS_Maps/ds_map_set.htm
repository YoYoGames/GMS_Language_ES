ds_map_set

Con esta función se puede establecer el valor de una clave dentro de un mapa DS dado. Se proporciona el valor del ID del mapa DS (devuelto por la función ds_map_create()), y a continuación la clave que se desea establecer y el valor que se desea asignar a. Las claves pueden ser números enteros o cadenas, y si la clave dada no existe, se creará para usted y se establecerá el valor. Esta función es lo mismo que usar el accesorio de mapa DS para establecer/crear un par un par clave/valor del mapa. La función no devuelve nada, por lo que si necesita comprobar si el valor de la clave ha sido reemplazado o si se ha creado una nueva clave, deberá utilizar la función ds_map_replace().

Sintaxis:

ds_map_set(id, key, value)

Argumento Descripción
id El id del mapa a utilizar.
clave La clave de la fijación.
valor El valor para establecer la clave.

Devuelve:

N/A

Ejemplo:

if is_undefined(ds_map_find_value(map, "score"))
{
ds_map_set(map, "score", 0);
}

El código anterior comprobará si la clave dada existe y si no existe entonces se crea y se establece.
