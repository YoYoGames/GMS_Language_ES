ds_map_is_map

Con esta función se puede comprobar si un mapa DS está almacenado en la clave de mapa dada. Si la clave dada contiene un ID de mapa DS, entonces la función devolverá true, de lo contrario devolverá false.

Sintaxis:

ds_map_is_map(id, key)

Argumento Descripción
id El id del ds_map a utilizar.
clave La clave para reemplazar.

Devuelve:

Booleano

Ejemplo:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if ds_map_is_map(inventory, key)
{
ds_map_destroy(key);
}
key = ds_map_find_next(inventory);
}
ds_map_destroy(inventario);

El código anterior hace un bucle a través de un mapa DS y comprueba si alguna de las claves dentro de él son para otros mapas DS. Si lo son, entonces el mapa DS almacenado es destruido, y al final del bucle el mapa DS principal es destruido también.
