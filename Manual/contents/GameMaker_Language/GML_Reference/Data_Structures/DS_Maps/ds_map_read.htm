ds_map_read

Esta función tomará una cadena que ha sido creada previamente por la función ds_map_write() y la leerá en un mapa DS previamente creado. Si el mapa en el que se lee la cadena contiene algún par clave/valor, éste se borrará antes de reconstruir el mapa guardado. Tenga en cuenta que si la estructura de datos fue creada con versiones anteriores de GameMaker Studio 2 debe añadir el argumento opcional "legacy", poniéndolo a true ya que el formato de la cadena ha cambiado con esta versión.

Sintaxis:

ds_map_read(id, str [, legacy]);

Argumento Descripción
id El id de la estructura de datos para leer la cadena en
str La cadena a leer
legado(opcional) Puede ser verdadero o falso u omitirse por completo.

Devuelve:

N/A

Ejemplo:

inventario = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
ds_map_read(inventory, t_string);
}
ini_close();

El código anterior crea un nuevo mapa DS y almacena su índice de identificación en la variable "inventario". Luego abre un archivo ini y lee una cadena de ese archivo en la variable temporal "t_string". Finalmente, comprueba que la cadena es válida (no el valor ini por defecto de "") y si lo es, lee la cadena en el mapa DS recién creado antes de cerrar el ini de nuevo.
