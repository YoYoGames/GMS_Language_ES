ds_map_replace_list

Con esta función se puede reemplazar una lista DS que ha sido almacenada en la "clave" dada por otra lista que ha sido creada previamente. Esta función está diseñada para crear mapas compatibles con JSON que que luego codificarás con json_encode() y sólo debe usarse junto con esa funcionalidad.

Sintaxis:

ds_map_replace_list(id, key, value)

Argumento Descripción
id El id del mapa a utilizar.
clave La clave para reemplazar.
valor El id de la ds_list que se utilizará para reemplazar la almacenada anteriormente en la clave dada.

Devuelve:

N/A

Ejemplo:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

El código anterior creará una Lista DS y la poblará con los valores de varias variables globales antes de reemplazar una lista previamente almacenada en el Mapa DS "j_map".
