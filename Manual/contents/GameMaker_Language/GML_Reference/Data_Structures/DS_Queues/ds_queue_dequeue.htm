ds_queue_dequeue

Esta función retirará el valor de la cabeza de la cola DS, eliminándolo de la cola y devolviendo el valor para ser almacenado en una variable. Si la cola está vacía, la función devolverá la constante indefinida, de lo contrario devolverá el valor real o de cadena contenido en la cola.

Sintaxis:

ds_queue_dequeue(id);

Argumento Descripción
id El identificador de la cola de la que se va a retirar.

Devuelve:

Valor del tipo de datos almacenado en la cola o indefinido

Ejemplo:

if !ds_queue_empty(move_queue)
{
var xx = ds_queue_dequeue(move_queue);
var yy = ds_queue_dequeue(move_queue);
move_towards_point(xx, yy, 4);
}

El código anterior comprueba la cola DS indexada en la variable "move_queue" para ver si está vacía, y si no lo está, entonces dequeues los dos valores de la cabeza de la cola y los utiliza para establecer una dirección para el movimiento.
