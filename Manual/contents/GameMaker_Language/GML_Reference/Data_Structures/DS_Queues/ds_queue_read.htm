ds_queue_read

Con esta función puedes recrear una cola DS guardada (una que ha sido previamente escrita como una cadena usando ds_queue_write()). Primero debe crear una nueva cola DS para leer la cadena, y si la cola DS ya existe y tiene información almacenada en ella, ésta se borrará antes de la lectura. existe y tiene información almacenada en ella, ésta se borrará antes de la lectura. Esta función es de vital importancia a la hora de crear mecanismos de guardado/carga para tu juego. Ten en cuenta que si la estructura de datos fue creada con versiones anteriores de GameMaker debes añadir el argumento opcional "legacy", poniéndolo a true ya que el formato de la cadena ha cambiado con esta versión.

Sintaxis:

ds_queue_read(id, str [, legacy]);

Argumento Descripción
id El id de la estructura de datos a leer.
str La cadena a leer.
legado(opcional) Puede ser verdadero o falso u omitirse por completo.

Devuelve:

N/A

Ejemplo:

queue = ds_queue_create();
ini_open("save.ini");
var str = ini_read_string("Queues", "0", "");
if (str != "")
{
ds_queue_read(queue, str);
}
ini_close();

El código anterior crea una cola y almacena el índice en la variable "cola". A continuación, abre un archivo ini y lee una cadena desde él, comprobando primero que la cadena no se devuelva vacía. Esta cadena se lee entonces en la cola DS recién creada.
