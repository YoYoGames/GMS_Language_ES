ds_queue_clear

Con esta función puede borrar todos los datos de la estructura de datos de la cola dada. Esto NO destruye la estructura de datos (para eso debe usar ds_queue_destroy()), sólo borra todos los datos de la misma y devuelve una cola vacía.

Sintaxis:

ds_queue_clear(id);

Argumento Descripción
id El id de la estructura de datos que se va a borrar.

Devuelve:

N/A

Ejemplo:

if (count = 15) && (!ds_queue_empty(command_queue))
{
ds_queue_clear(command_queue);
alarm[0] = room_speed;
ai_count = 0;
}

El código anterior comprueba una variable para ver si ha alcanzado un valor específico y si lo ha hecho borra la cola DS indexada en la variable "command_queue", establece una alarma y reinicia la variable a 0.
