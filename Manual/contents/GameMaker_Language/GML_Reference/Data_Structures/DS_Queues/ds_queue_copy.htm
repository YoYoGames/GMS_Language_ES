ds_queue_copy

Esta función puede utilizarse para copiar el contenido de una cola en otra. Tenga en cuenta que esto NO elimina el contenido de la cola original, ni destruye la cola original. Cuando se utiliza esta función, la cola a la que se copia debe haber sido haber sido creada previamente y si contenía algún elemento antes de la copia, éste se borrará primero (lo que significa que esta información se perderá).

Sintaxis:

ds_queue_copy(id, source);

Argumento Descripción
id El id de la NUEVA cola.
fuente La cola ORIGINAL a copiar en el id.

Devuelve:

N/A

Ejemplo:

with (instance_create_layer(x, y, "Enemigos", obj_Enemy))
{
queue = ds_queue_create();
ds_queue_copy(queue, other.queue);
}

La función anterior crea una nueva instancia y luego en esa instancia crea una nueva cola DS y copia el contenido de la cola en la instancia que ejecuta el bloque de código, en la cola de la instancia recién creada.
