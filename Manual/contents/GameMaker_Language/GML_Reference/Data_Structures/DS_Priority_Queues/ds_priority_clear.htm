ds_priority_clear

Con esta función puede borrar todos los datos de la estructura de datos de la cola de prioridad dada. Esto NO destruye la estructura de datos (para eso debe usar ds_priority_destroy()), sólo borra todos los datos datos de la misma y devuelve una cola de prioridad vacía.

Sintaxis:

ds_priority_clear(id);

Argumento Descripción
id El id de la estructura de datos que se va a borrar.

Devuelve:

N/A

Ejemplo:

if (count = 15) && (!ds_priority_empty(command_queue))
{
ds_priority_clear(command_queue);
alarm[0] = room_speed;
ai_count = 0;
}

El código anterior comprueba una variable para ver si ha alcanzado un valor específico y si lo ha hecho borra la cola de prioridad DS indexada en la variable "command_queue", establece una alarma y reinicia la variable a 0.
