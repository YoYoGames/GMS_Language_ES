ds_priority_find_priority

Con esta función se puede recuperar la prioridad de cualquier valor dado. Si el valor no existe en la cola de prioridades, se devolverá undefined.

Sintaxis:

ds_priority_find_priority(id, val);

Argumento Descripción
id El id de la cola de prioridad a utilizar.
val El valor para encontrar la prioridad de.

Devuelve:

Real o indefinido

Ejemplo:

p = ds_priority_find_priority(ai_priority, "inteligencia");

El código anterior almacenará la prioridad devuelta para el valor dado en la variable de instancia "p".
