ds_priority_find_min

Con esta función se puede encontrar el valor almacenado en la cola de prioridad con la prioridad más baja, y si más de un valor tiene la misma prioridad entonces cualquiera de ellos podría ser devuelto en cualquier orden. Sin embargo, a diferencia de ds_priority_delete_min(), esta función no eliminará el valor de la cola.

Sintaxis:

ds_priority_find_min(id);

Argumento Descripción
id El id de la cola de prioridad a utilizar.

Devuelve:

Real o Cadena

Ejemplo:

si ai_move
{
script_execute(ds_priority_find_min(ai_priority));
}

El código anterior comprueba una variable de instancia y si devuelve true ejecutará una función de script indexada en la cola de prioridad con el valor de prioridad más bajo.
