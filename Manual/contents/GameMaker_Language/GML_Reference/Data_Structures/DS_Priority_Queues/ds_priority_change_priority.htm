ds_priority_change_priority

Esta función tomará un valor dado y cambiará su prioridad dentro de la cola de prioridad referenciada.

Sintaxis:

ds_priority_change_priority(id, val, priority);

Argumento Descripción
id El identificador de la cola prioritaria a cambiar.
val El valor para cambiar la prioridad de.
prioridad La nueva prioridad del valor.

Devuelve:

N/A

Ejemplo:

if (global.Game_Time < 1000)
{
ds_priority_change(ai_priority, AI_Search, 1);
}

El código anterior comprueba una variable global y, si está por debajo de un determinado valor, cambiará la prioridad del índice de la función de script mantenida en la cola de prioridades.
