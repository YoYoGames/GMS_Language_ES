ds_priority_add

Con esta función puede añadir un valor (ya sea un número real o una cadena) a una cola de prioridad, asignándole al mismo tiempo un valor de prioridad.

Sintaxis:

ds_priority_add(id, val, priority);

Argumento Descripción
id El id de la cola prioritaria a la que se va a añadir.
val El valor a añadir a la cola de prioridad.
prioridad La prioridad del valor a añadir.

Devuelve:

N/A

Ejemplo:

ds_priority_add(ai_priority, AI_Search, 5);

El código anterior añade una función de script a la cola de prioridad indexada en la variable "ai_priority" y le asigna una prioridad de 5.
