ds_priority_copy

Esta función puede utilizarse para copiar el contenido de una cola de prioridad en otra. Tenga en cuenta que esto NO elimina el contenido de la cola de prioridad original, ni destruye la cola de prioridad original. Cuando se utiliza esta función, la cola prioritaria a la que se está copiando debe haber sido creada previamente y si contenía algún elemento antes de la copia, éste se borrará primero (lo que significa que esta información se perderá).

Sintaxis:

ds_priority_copy(id, source);

Argumento Descripción
id El id de la cola prioritaria a la que se va a copiar.
fuente El id de la cola prioritaria de la que se va a copiar.

Devuelve:

N/A

Ejemplo:

with (instance_create_layer(x, y, "Enemigos", obj_Enemy))
{
p_queue = ds_priority_create();
ds_priority_copy(p_queue, other.p_queue);
}

La función anterior crea una nueva instancia y luego en esa instancia crea una nueva cola de prioridad DS y copia el contenido de la cola de prioridad en la instancia que ejecuta el bloque de código, en la cola de prioridad de la instancia recién creada.
