ds_priority_delete_max

Esta función devolverá el valor que tenga la mayor prioridad en la cola y luego eliminará el valor (y la prioridad) de la estructura de datos. Si más de un valor tiene la misma prioridad, entonces cualquiera de ellos podría ser devuelto en cualquier orden, pero todos otros valores con la misma prioridad seguirán en la cola.

Sintaxis:

ds_priority_delete_max(id);

Argumento Descripción
id El id de la cola de prioridad a utilizar.

Devuelve:

Tipo de datos almacenados en la prioridad

Ejemplo:

si ai_move
{
script_execute(ds_priority_delete_max(ai_priority));
}

El código anterior comprueba una variable de instancia y si devuelve true ejecutará un script indexado en la cola de prioridad con el valor de prioridad más alto y luego eliminará ese script de la cola.
