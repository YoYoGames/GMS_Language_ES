ds_priority_read

Con esta función puedes recrear una prioridad DS guardada (una que haya sido previamente escrita como una cadena usando ds_priority_write()). Primero debes crear una nueva prioridad DS para leer la cadena, y si la prioridad DS ya existe y tiene información almacenada en ella, ésta se borrará antes de la lectura. Esta función es de vital importancia a la hora de crear mecanismos de guardado/carga para tu juego. Ten en cuenta que si la estructura de datos fue creada con versiones anteriores de versiones de GameMaker debes añadir el argumento opcional "legacy", poniéndolo a true ya que el formato de la cadena ha cambiado con esta versión.

Sintaxis:

ds_priority_read(id, str [, legacy]);

Argumento Descripción
id El id de la estructura de datos a escribir.
str La cadena desde la que escribir.
legado(opcional) Puede ser verdadero o falso u omitirse por completo.

Devuelve:

N/A

Ejemplo:

p_queue = ds_priority_create();
ini_open("save.ini");
var str = ini_read_string("P_Queues", "0", "");
if (str != "")
{
ds_priority_read(p_queue, str);
}
ini_close();

El código anterior crea una cola de prioridad y almacena el índice en la variable "p_queue". A continuación, abre un archivo ini y lee una cadena de texto, comprobando primero que la cadena no se devuelva vacía. Esta cadena es entonces Esta cadena se lee en la prioridad DS recién creada.
