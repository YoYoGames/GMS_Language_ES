ds_priority_delete_value

Esta función simplemente borrará el valor dado, junto con su prioridad, de la cola de prioridad indexada.

Sintaxis:

ds_priority_delete_value(id,val);

Argumento Descripción
id El id de la cola de prioridad a utilizar.
val El valor a eliminar de la cola de prioridad.

Devuelve:

N/A

Ejemplo:

si (ai_move == false)
{
ds_priority_delete_value(ai_priority, AI_Move);
}

El código anterior comprueba una variable de instancia y si devuelve false eliminará la función de script indexada de la cola de prioridades.
