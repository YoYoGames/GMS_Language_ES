ds_list_sort

Con esta función puede ordenar todos los valores de una lista, ya sea en orden ascendente o descendente. Si la lista contiene cadenas, éstas se ordenarán alfabéticamente, basándose en el alfabeto inglés de 26 letras.

Sintaxis:

ds_list_sort(id, ascendente);

Argumento Descripción
id El id de la lista a ordenar.
ascender Si los valores deben estar en orden ascendente(verdadero) o descendente(falso).

Devuelve:

N/A

Ejemplo:

si newgame
{
ds_list_sort(name_list, true);
}

El código anterior ordenará la lista indexada en la variable "name_list" si la variable "newgame" está marcada como verdadera.
