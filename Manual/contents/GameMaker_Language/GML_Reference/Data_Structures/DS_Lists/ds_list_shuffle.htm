ds_list_shuffle

Con esta función puede barajar una lista, lo que reordenará todos los valores de los componentes en posiciones aleatorias respecto a las que se añadieron originalmente a la lista.

NOTA: Esta función barajará los elementos de la lista en las mismas posiciones cada vez que el juego se ejecute de nuevo debido a que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que la depuración del código sea una tarea mucho más fácil. Para evitar este comportamiento utilice aleatorizar() al inicio de su juego. Esto sólo es cierto cuando se prueba y depura el juego, ya que el paquete ejecutable final no mostrará este comportamiento y será aleatorio cada vez que se juegue.

Sintaxis:

ds_list_shuffle(id);

Argumento Descripción
id El id de la lista a barajar.

Devuelve:

N/A

Ejemplo:

si se reinicia
{
ds_list_shuffle(card_list);
}

El código anterior barajará la lista indexada en la variable "card_list" si la variable "restart" está marcada como verdadera.
