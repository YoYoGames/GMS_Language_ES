ds_list_find_value

Con esta función se puede comprobar la posición de la lista dada y se devolverá el valor mantenido dentro de la lista para esa posición. Tenga en cuenta que si da una posición que está fuera del tamaño de la lista dada (por ejemplo, la posición 11 en una lista de 10 valores) entonces la función puede devolver undefined o 0. Esto se debe a que cuando se crea la lista, internamente las primeras entradas de la lista se ponen a 0 para minimizar los problemas de rendimiento al añadir inicialmente elementos a la lista (aunque la función ds_list_size() devolverá 0 en una lista recién creada). Si desea asegurarse de que la lista está "realmente" vacía al crearla, debe llamar a ds_list_clear() después de crear la lista, con lo que significa que cualquier valor devuelto para las ranuras de la lista sin rellenar será indefinido, lo que puede comprobar de forma consistente utilizando la función is_undefined() función.

Sintaxis:

ds_list_find_value(id, pos);

Argumento Descripción
id El id de la lista a utilizar.
pos La posición en la que buscar, donde 0 corresponde al principio de la lista y la posición final es ds_list_size(id)-1.

Devuelve:

Real, Cadena o Indefinido

Ejemplo:

val = ds_list_find_value(list, ds_list_size(list) - 1);

El código anterior comprueba la lista indexada en la variable "list" en la última posición de la lista y almacena el valor devuelto en la variable "val".
