ds_list_is_list

Con esta función puedes comprobar si otra lista DS está almacenada en la posición dada dentro de una lista DS. Si la posición dada contiene un ID de lista DS, entonces la función devolverá true, de lo contrario devolverá false.

Sintaxis:

ds_list_is_list(id, pos);

Argumento Descripción
id El id de la lista a comprobar.
pos La posición dentro de la lista a comprobar.

Devuelve:

Booleano

Ejemplo:

var size = ds_list_size(ships) ;
for (var i = 0; i < size; i++;)
{
if ds_list_is_list(ships, i)
{
ds_list_destroy(i);
}
}
ds_list_destroy(ships);

El código anterior hace un bucle a través de una lista DS y comprueba si alguna de las entradas contiene otros ID de lista. Si lo hacen, entonces estas listas son destruidas, y luego la lista principal es destruida después de que el bucle haya terminado.
