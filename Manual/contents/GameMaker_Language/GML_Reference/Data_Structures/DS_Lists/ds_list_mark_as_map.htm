ds_list_mark_as_map

Esta función "marcará" (o "marcará") una posición dada dentro de una lista DS previamente creada como si tuviera un mapa DS. Esta funcionalidad es necesaria cuando se codifican cadenas JSON (que se pueden crear utilizando json_encode()), pero también puede ser útil cuando se anidan estructuras de datos, ya que los elementos marcados de esta manera serán automáticamente recogidos de la basura (destruidos) cuando la lista padre DS se destruye. Esto significa que no tienes que recorrer manualmente el contenido de la lista y destruir las estructuras de datos marcadas individualmente antes de destruir la lista "padre". Sin embargo, si elimina la posición de la lista individualmente la estructura de datos que contenía no será recolectada por la basura y deberá llamar a la función apropiada de destrucción del mapa DS antes de borrar la posición de la lista padre. Tenga en cuenta también que si llama a la función ds_list_clear() en una lista, cualquier elemento marcado como mapa se destruirá también cuando se borre la lista.

También hay que tener en cuenta que si la posición que se ha marcado cambia dentro de la lista, la "marca" se moverá con ella, por lo que si ha marcado la posición 3 en la lista (por ejemplo) y luego inserta 2 elementos más por debajo para que se mueva hasta la posición 5, seguirá marcada como mapa.

Sintaxis:

ds_list_mark_as_map(id, pos);

Argumento Descripción
id El id de la lista a marcar.
pos La posición dentro de la lista a marcar.

Devuelve:

Real (entero - ID del mapa DS si tiene éxito o -1 si falla)

Ejemplo:

var sub_map = ds_map_create();
ds_map_add(sub_map, "player", player_array);
ds_map_add(sub_map, "enemy", enemy_array);
ds_list_add(j_list, sub_map);
ds_list_mark_as_map(j_list, 0);

El código anterior crea un mapa DS y luego lo rellena con dos claves, cada una de las cuales contiene una matriz de valores. Este mapa se añade entonces a la lista DS dada, y la posición se "marca" como tal para que pueda ser codificada correctamente más tarde.
