ds_list_size

Esta función devolverá el "tamaño" de la lista, es decir, el número de elementos que se han añadido a ella.

Sintaxis:

ds_list_size(id);

Argumento Descripción
id El id de la estructura de datos a comprobar.

Devuelve:

Real

Ejemplo:

if !ds_list_empty(control_list)
{
num = ds_list_size(control_list);
}

El código anterior comprueba una lista DS para ver si está vacía o no, y si no lo está, obtiene el número de elementos que contiene y almacena el valor en una variable.
