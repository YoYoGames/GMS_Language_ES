ds_list_write

Esta función devuelve una cadena que puede ser almacenada o transferida a otra estructura de datos utilizando la función ds_list_read() función.

NOTA: La cadena devuelta no es una cadena legible, sino un volcado del contenido de la estructura de datos.

Sintaxis:

ds_list_write(id);

Argumento Descripción
id El id de la estructura de datos a escribir.

Devuelve:

Cadena

Ejemplo:

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

El código anterior abre un archivo ini y luego escribe una cadena que contiene la información almacenada en la lista DS indexada en la variable "lista". A continuación se borra la lista y se cierra el archivo ini.
