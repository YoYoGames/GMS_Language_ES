ds_list_insert

Esta función añadirá el valor dado a la lista en la posición dada. Si la lista contiene más valores después de la posición dada, su posición se desplazará uno hacia arriba para hacer espacio haciendo la lista más grande en uno.

Sintaxis:

ds_list_insert(id, pos, val);

Argumento Descripción
id El identificador de la lista a la que hay que añadir.
pos La posición para añadir el valor, donde 0 corresponde al principio de la lista y la posición final es ds_list_size(id)-1.
val El valor a añadir a la lista.

Devuelve:

N/A

Ejemplo:

ds_list_insert(list, 9, score);

El código anterior añadirá el valor almacenado en la variable "score" en la décima posición de la lista indexada por la variable "list" (las listas se cuentan desde 0, por lo que una lista de diez valores se numera de 0 a 9).
