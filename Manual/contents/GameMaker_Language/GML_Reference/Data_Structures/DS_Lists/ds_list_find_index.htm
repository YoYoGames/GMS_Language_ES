ds_list_find_index

Con esta función se puede comprobar la lista dada para un valor y se devolverá la posición dentro de la lista para ese valor. Tenga en cuenta que si hay más de una entrada en la lista con el mismo valor, se puede devolver la posición de cualquiera de ellas, y que si el valor no existe, se devolverá -1. Tenga en cuenta que el valor puede ser un array, que puede comprobar con la función is_array().

Sintaxis:

ds_list_find_index(id, val);

Argumento Descripción
id El id de la lista a utilizar.
val El valor a encontrar.

Devuelve:

Real

Ejemplo:

pos = ds_list_find_index(list, "Player1");

El código anterior comprueba la lista indexada en la variable "list" para el valor "Player1" y almacena la posición devuelta en la variable "pos".
