ds_list_add

Esta función puede utilizarse para añadir un nuevo valor (real o de cadena) a la lista, que se añadirá al final. La función puede recibir más argumentos opcionales (tantos como necesite), lo que le permite añadir varios valores consecutivos a la lista en una sola llamada.

Sintaxis:

ds_list_add(id, val1 [, val2, ... max_val]);

Argumento Descripción
id El identificador de la lista a la que hay que añadir.
val El valor a añadir a la lista.
[val2, ... max_val] Valores opcionales que se añaden a la lista.

Devuelve:

N/A

Ejemplo:

ds_list_add(sc_list, score);

El código anterior añadirá el valor almacenado en la variable "score" a la lista indexada en la variable "sc_list".
