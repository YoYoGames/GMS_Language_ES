ds_list_replace

Esta función reemplazará el valor en la posición dada por otro.

Sintaxis:

ds_list_replace(id, pos, val);

Argumento Descripción
id El id de la lista a cambiar.
pos La posición para reemplazar el valor, donde 0 corresponde al principio de la lista y la posición final es ds_list_size(id)-1.
val El nuevo valor por el que se sustituye el valor dado.

Devuelve:

N/A

Ejemplo:

ds_list_replace(n_list, 3, name);

El código anterior sustituirá el valor almacenado en la posición 3 de la lista por el almacenado en la variable "nombre".
