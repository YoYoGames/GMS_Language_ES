ds_list_copy

Con esta función puede copiar el contenido de una lista en otra. Ambas listas deben haber sido creadas previamente y si la lista a la que se está copiando ya tiene información dentro, esta lista se borrará primero. El resultado final son dos listas independientes independientes que contienen la misma información.

Sintaxis:

ds_list_copy( id, source );

Argumento Descripción
id El id de la lista que se está copiando.
fuente El id de la lista de la que se va a copiar.

Devuelve:

N/A

Ejemplo:

if !ds_list_empty(main_list)
{
old_list = ds_list_create();
ds_list_copy(old_list, main_list);
ds_list_clear(main_list);
}

El código anterior comprobará una lista DS para ver si está vacía. Si no está vacía se copia en otra lista DS (que ha sido creada previamente) y luego se borra la lista original.
