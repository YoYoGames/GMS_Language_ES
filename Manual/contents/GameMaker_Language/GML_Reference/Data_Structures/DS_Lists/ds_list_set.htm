ds_list_set

Esta función puede utilizarse para fijar una entrada de la lista previamente añadida. Se da el ID de la lista (tal y como se devolvió cuando se creó la lista) y la posición dentro de la lista que se va a establecer, así como el valor que se va a establecer. Tenga en cuenta que si la entrada que se establece está fuera de los límites de la lista (por ejemplo, usted establece la entrada 20 de la lista, pero la lista actual sólo contiene 10 entradas), entonces la lista se llenará hasta la posición dada y cada entrada se establecerá en 0. Esta función es la misma que el uso del accesorio de lista DS.

Sintaxis:

ds_list_set(id, pos, val);

Argumento Descripción
id El identificador de la lista a la que hay que añadir.
pos La posición dentro de la lista a establecer.
[val2, ... max_val] El valor a establecer en la lista.

Devuelve:

N/A

Ejemplo:

for (var i = 0; i < ds_list_size(list); i++;)
{
ds_list_set(list, i, -1);
}

El código anterior añadirá el valor almacenado en la variable "score" a la lista indexada en la variable "sc_list".
