Pilas DS

Una pila DS es una estructura de datos denominada last-in-first-out (LIFO), similar a una cola DS. Puedes introducir valores en una pila y luego eliminarlos de nuevo sacándolos de la pila, donde el valor que fue introducido en la pila más recientemente es el primero en ser sacado de ella de nuevo (piensa en una pila de monedas, donde cada moneda que añades tiene que ser sacada de nuevo antes de que puedas llegar al resto de las monedas que hay debajo). Las pilas se utilizan a menudo cuando hay que manejar interrupciones, o cuando se tienen funciones recursivas, o incluso cuando se construye una IA rudimentaria para tus juegos.

NOTA: Como todos los recursos dinámicos, las estructuras de datos ocupan memoria, por lo que siempre deben ser destruidas cuando ya no se necesiten para evitar fugas de memoria que ralentizarán y eventualmente bloquearán tu juego.

NOTA: Si necesita comprobar si una estructura de datos existe, puede utilizar la función ds_exists() función

Existen las siguientes funciones que se ocupan de las pilas DS.

  • ds_stack_create
  • ds_stack_destroy
  • ds_stack_clear
  • ds_stack_empty
  • ds_stack_size
  • ds_stack_copy
  • ds_stack_top
  • ds_stack_pop
  • ds_stack_push
  • ds_stack_read
  • ds_stack_write
