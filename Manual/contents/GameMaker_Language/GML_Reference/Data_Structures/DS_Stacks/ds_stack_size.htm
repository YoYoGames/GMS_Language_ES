ds_stack_size

Esta función devolverá el "tamaño" de la pila, es decir, el número de elementos que se han introducido en ella.

Sintaxis:

ds_stack_size(id);

Argumento Descripción
id El id de la estructura de datos a comprobar.

Devuelve:

Real

Ejemplo:

if !ds_stack_empty(control_stack)
{
num = ds_stack_size(control_stack);
}

El código anterior comprueba una pila DS para ver si está vacía o no, y si no lo está, obtiene el número de elementos que contiene y almacena el valor en una variable.
