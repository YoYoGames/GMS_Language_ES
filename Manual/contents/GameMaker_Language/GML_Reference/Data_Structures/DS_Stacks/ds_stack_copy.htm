ds_stack_copy

Esta función puede utilizarse para copiar el contenido de una pila en otra. Tenga en cuenta que esto NO elimina el contenido de la pila original, ni destruye la pila original. Cuando se utiliza esta función, la pila a la que se copia debe haber sido haber sido creada previamente y si contenía algún elemento antes de la copia, éste se borrará primero (lo que significa que esta información se perderá).

Sintaxis:

ds_stack_copy(id, source);

Argumento Descripción
id El id de la pila NUEVA.
fuente La pila ORIGINAL a copiar en el id.

Devuelve:

N/A

Ejemplo:

with (instance_create_layer(x, y, "Enemigos", obj_Enemy))
{
stack = ds_stack_create();
ds_stack_copy(stack, other.stack);
}

La función anterior crea una nueva instancia y luego en esa instancia crea una nueva pila DS y copia el contenido de la pila en la instancia que ejecuta el bloque de código, en la pila de la instancia recién creada.
