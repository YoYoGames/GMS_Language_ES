ds_stack_read

Con esta función puedes recrear una pila DS guardada (una que haya sido previamente escrita como una cadena usando ds_stack_write()). Primero debe crear una nueva pila DS para leer la cadena, y si la pila DS ya existe y tiene información almacenada en ella, entonces ésta se borrará antes de la lectura. Esta función es de vital importancia a la hora de crear mecanismos de guardado/carga para tu juego. Ten en cuenta que si la estructura de datos fue creada con versiones anteriores de GameMaker debes añadir el argumento opcional "legacy", poniéndolo a true ya que el formato de la cadena ha cambiado con esta versión.

Sintaxis:

ds_stack_read(id, str [, legacy] );

Argumento Descripción
id El id de la estructura de datos a leer.
str La cadena a leer.
legado(opcional) Puede ser verdadero o falso u omitirse por completo.

Devuelve:

N/A

Ejemplo:

stack = ds_stack_create();
ini_open("save.ini");
var str = ini_read_string("Stacks", "0", "");
if (str != "")
{
ds_stack_read(stack, str);
}
ini_close();

El código anterior crea una pila y almacena el índice en la variable "stack". A continuación, abre un archivo ini y lee una cadena desde él, comprobando primero que la cadena no se devuelva vacía. Esta cadena se lee entonces en la pila DS recién creada.
