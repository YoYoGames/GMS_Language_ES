ds_stack_clear

Con esta función puedes borrar todos los datos de la estructura de datos de la pila dada. Esto NO destruye la estructura de datos (para eso debe usar ds_stack_destroy()) sólo borra todos los datos de la misma y devuelve una pila vacía.

Sintaxis:

ds_stack_clear(id);

Argumento Descripción
id El id de la estructura de datos que se va a borrar.

Devuelve:

N/A

Ejemplo:

if (ai_count = 15) && (!ds_stack_empty(AI_stack))
{
ds_stack_clear(AI_stack);
alarm[0] = room_speed;
ai_count = 0;
}

El código anterior comprueba una variable para ver si ha alcanzado un valor específico y, si lo ha hecho, borra la pila de DS indexada en la variable "AI_stack", establece una alarma y reinicia la variable a 0.
