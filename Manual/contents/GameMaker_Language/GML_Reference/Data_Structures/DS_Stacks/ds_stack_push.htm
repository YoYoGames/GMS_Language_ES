ds_stack_push

Esta función empuja (añade) un valor, que puede ser de cualquier tipo de datos, a la parte superior de la pila. La función puede tomar otros 14 argumentos opcionales (haciendo un total de 15 posibles adiciones), permitiéndole empujar múltiples valores consecutivamente a la pila en una sola llamada.

Sintaxis:

ds_stack_push(id, val [, val2, ... val15]);

Argumento Descripción
id El id de la estructura de datos a la que se va a empujar.
val El valor que se va a introducir en la pila.
[val2, ... val13] Valores opcionales que se añaden a la pila.

Devuelve:

N/A

Ejemplo:

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

El código anterior crea una nueva pila DS y almacena su índice en la variable "move_stack". A continuación, empuja un número de valores en la pila para su uso futuro.
