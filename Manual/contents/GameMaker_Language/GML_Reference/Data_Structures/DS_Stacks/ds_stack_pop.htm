ds_stack_pop

Esta función sacará el valor superior de la pila DS, eliminándolo de la pila y devolviendo el valor para ser almacenado en una variable. Si la pila está vacía, la función devolverá la constante indefinida, de lo contrario devolverá el valor real o de cadena contenido en la pila.

Sintaxis:

ds_stack_pop(id);

Argumento Descripción
id El id de la estructura de datos de la que se va a extraer.

Devuelve:

Valor del tipo de datos que se almacena en la pila o indefinido

Ejemplo:

if !ds_stack_empty(move_stack)
{
var xx = ds_stack_pop(move_stack);
var yy = ds_stack_pop(move_stack);
move_towards_point(xx, yy, 4);
}

El código anterior comprueba la pila DS indexada en la variable "move_stack" para ver si está vacía, y si no lo está, entonces saca los dos valores superiores de la pila y los utiliza para establecer una dirección para el movimiento.
