ds_stack_empty

Con esta función puedes comprobar la pila DS dada para ver si está vacía (devuelve true) o no (devuelve false).

Sintaxis:

ds_stack_empty(id);

Argumento Descripción
id El id de la estructura de datos a comprobar.

Devuelve:

Booleano

Ejemplo:

if (ai_count == 15) && (!ds_stack_empty(AI_stack))
{
ds_stack_clear(AI_stack);
alarm[0] = room_speed;
ai_count = 0;
}

El código anterior comprueba una variable para ver si ha alcanzado un valor específico y si lo ha hecho borra la pila de DS indexada en la variable "AI_stack", establece una alarma y restablece la variable a 0.
