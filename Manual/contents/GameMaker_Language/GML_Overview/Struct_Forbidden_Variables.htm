Variables prohibidas de la estructura

Al crear una estructura, es posible utilizar ciertas variables incorporadas como nombres de variables miembros, por ejemplo:

mystruct = {
object_index : obj_Player,
speed : 3,
image_blend : c_red
}

Sin embargo, sólo las variables incorporadas de ámbito local y de instancia pueden usarse de esta manera y asignarles valores como si fueran variables miembro regulares de la estructura. El uso de variables incorporadas de ámbito global está prohibido y causará problemas en tu juego. A continuación puedes encontrar una lista completa de estas variables para que sepas cuáles debes evitar.
