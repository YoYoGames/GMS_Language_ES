devolver

La sentencia return tiene la siguiente sintaxis:

return (<expresión>)

La sentencia return sólo se utiliza en las funciones y métodos de los scripts, y se usa para devolver un valor de la función que se utilizará en otro código o llamada a una función de la función. Hay que tener en cuenta que la ejecución de la función termina en la sentencia return , lo que significa que cualquier código que venga después de la llamada a return no se ejecutará. A continuación se muestra un breve de ejemplo llamado sqr_calc que calcula el cuadrado de cualquier valor que se le pase, e incluye la captura de errores en caso de que el parámetro que se le pase no sea un número real:

/// @function sqr_calc(val);
/// @param {real} val El valor a calcular el cuadrado de
/// @description Calcula el cuadrado del valor dado

function sqr_calc(value)
{
if !is_real(valor)
{
return 0;
}
else
{
return (valor * valor);
}
}

Para llamar a una función desde un trozo de código, basta con utilizarla de la misma manera que cuando se llama a funciones en tiempo de ejecución, es decir, escribir el nombre de la función con los valores de los parámetros entre paréntesis. Así, la función anterior se llamaría así:

si keyboard_check_pressed(vk_enter)
{
val = scr_sqr(amount);
}

Aquí la variable val será o bien 0 (porque la variable amount no es un número real) o bien el total de value * value.
