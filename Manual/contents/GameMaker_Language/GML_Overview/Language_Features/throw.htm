tirar

La sentencia de ejecución throw te permite generar tus propias excepcionesen tiempo de ejecución, terminando el juego y mostrando un mensaje de error, utilizando la siguiente sintaxis:

throw (<expresión>);

La expresión utilizada puede ser un valor o una cadena o cualquier otro tipo de datos, y esto generará un error de excepción que se muestra -por defecto- en la pantalla y al cerrar el mensaje de error el juego terminará. Por ejemplo llamando a esto:

throw ("¡Hola Mundo!");

provocará que se muestre el siguiente error de excepción no controlada:

No obstante, puede hacerse cargo de este mensaje de error y utilizar su propio código de gestión llamando a la función la función exception_unhandled_handler(). Esta funciónen tiempo de ejecución te permite suministrar un método personalizado que será llamado cada vez que se produzcan excepciones no manejadas en tu juego.
