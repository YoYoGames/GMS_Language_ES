continuar

La sentencia continue tiene la siguiente sintaxis básica:

continuar;

Si se utiliza dentro de una sentencia que forma un bucle (para, repetir, mientras o do / hasta), terminará inmediatamente la iteración actual y saltará al principio del bucle, comenzando una nueva iteración y omitiendo cualquier código que venga después de continue dentro del bucle. También puede puede utilizarse dentro de la sentencia con donde hará que el código salte a la siguiente instancia y se ejecute de nuevo. Tenga en cuenta que si continue se utiliza fuera de cualquiera de estos contextos dará un error.

A continuación se muestra un ejemplo de uso en un bucle for:

var _val = 0;
for (var i = 0; i < 10; i += 1)
{
if (val_array[i] <= 0)
{
continue;
}
_val += val_array[i];
}
draw_text(32, 32, "Total de valores positivos = " + string(_val));

A continuación se muestra un ejemplo de uso en un bucle de while:

var file = file_text_open_read("Game_Data.txt");
var _num = 0;
while (!file_text_eof(file))
{
var _s = file_text_readln(file);
if (_s == "")
{
continue;
}
str[num++] = _s;
}
file_text_close(file);

A continuación se muestra un ejemplo de uso en un bucle de do / until:

do
{
var _x = random(room_width);
var _y = random(room_height);
if (instance_position(_x, y, obj_Enemy)
{
continúa;
}
instance_create_layer(_x, _y, "Instances", obj_Enemy);
}
hasta que (instance_count(obj_Enemy) >= 10);

with (obj_Enemy_Parent)
