borrar

El operador delete se utiliza para desreferenciar una estructura, y tiene la siguiente sintaxis:

borrar <variable>;

Esto significa que se llama al operador delete junto con la variable que contiene una estructura y se eliminará la referencia específica a la estructura almacenada en la variable dada (una referencia es simplemente un valor que apunta al área de memoria que almacena la estructura y su contenido). Si esta referencia fue la última referencia a la estructura en tu juego, entonces el recolector de basura eliminará la estructura de la memoria en una siguiente iteración, normalmente al comienzo del siguiente paso.

NOTA: la supresión sólo puede utilizarse junto con variables, y no con expresiones

Por defecto, los structs que no tengan más referencias en el código serán recolectados automáticamente en los pasos siguientes a su uso, pero es una buena práctica utilizar este operador para marcarlos explícitamente para que el recolector de basura los elimine de la memoria al al inicio del siguiente paso, liberando la memoria de forma más rápida y eficiente.

El siguiente ejemplo muestra la creación de una estructura, por ejemplo en el evento de creación de una instancia:

mystruct =
{
a : 0,
b : 100,
c : "Hello World"
}

Esta estructura se utilizará entonces en los eventos de instancia como de costumbre, antes de ser finalmente marcada para la recolección de basura en el evento de limpieza, así:

borrar mystruct;
