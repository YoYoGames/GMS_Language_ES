romper

La sentencia break se utiliza para finalizar prematuramente un para, repetir, mientras o do / until de algún tipo, o para indicar a una sentencia cambiar que termine en ese punto, o para terminar prematuramente una con llamada. Por favor, consulte las páginas individuales de estas diferentes funciones para obtener una explicación más para obtener una explicación más detallada de cómo se puede utilizar en cada circunstancia. Tenga en cuenta que si break se utiliza fuera de cualquiera de estos contextos dará un error.

A continuación puede ver algunos ejemplos básicos de cómo se puede utilizar break, cuya sintaxis es sencilla:

romper;

romper en un bucle for:

for (var i = 0; i < 10; i += 1)
{
if (array[i] == 234)
{
break;
}
}
num = i;

romper en un bucle de repetición:

var i = 0;
var temp = 0;
repeat (10)
{
temp += array[i];
if (temp > max_total)
{
break;
}
else
{
i += 1;
}
}

romper en un bucle while:

var i = 0;
while (!place_free(x, y))
{
x = random(room_width);
y = random(room_height);
if (i > 50)
{
break;
}
else
{
i+=1;
}
}

break en un bucle do / until:

var _id = noone;
do
{
_id = list[| 0];
if instance_exists(_id)
{
_break;
}
ds_list_delete(list, 0);
}
until (ds_list_empty(list));
target = _id;

romper cuando se utiliza con:

var count = 0;
with (obj_Enemy)
{
count++;
if (count > 10)
{
break;
}
hp = 100;
}

break en un switch:

switch (tecla_del_teclado)
{
case vk_left:
case ord("A"):
x -= 4;
break;
case vk_right:
case ord("D"):
x += 4;
break;
case vk_up:
case ord("W"):
y -= 4;
break;
case vk_down:
case ord("S"):
y += 4;
break;
}
