El operador new puede ser utilizado para crear una estructura con una función que es llamada con los argumentos dados. La función puede entonces rellenar la estructura con las variables de los argumentos - al igual que el evento de creación de una instancia establecerá las variables iniciales para la instancia - y luego el operador new devolverá la estructura. Antes de usar este operador es importante tener en cuenta que la función dada debe ser marcada como una función constructora, de lo contrario el operador new no creará la estructura (el ejemplo de código de abajo muestra esto, y para más información, por favor vea la página sobre Estructuras).

NOTA: Puedes comprobar una estructura para saber qué función se utilizó para crearla utilizando la función de tiempo de ejecución instanceof().

Este operador tiene la siguiente sintaxis:

<variable> = nueva <función> (<argumento>, <argumento>, ...);

Cuando se utiliza el operador new junto con una función, devolverá una referencia a la estructura que se almacena en la variable dada, permitiéndole acceder a la estructura y a cualquier variable que haya sido creada dentro de ella por la función.

El siguiente ejemplo muestra una función que ha sido definida para ser utilizada como constructor:

function init_struct(_a, _b, _c) constructor
{
a = _a;
b = _b;
c = _c;
}

Esta función se puede utilizar junto con el operador new para crear una estructura y rellenarla con las variables ajustadas a los valores de los argumentos utilizados en la función, así:

mystruct = new init_struct(10, 100, "Hello World");

Es importante tener en cuenta que llamar a new en cualquier función que no haya sido marcada como constructor provocará una excepción en tiempo de ejecución.
