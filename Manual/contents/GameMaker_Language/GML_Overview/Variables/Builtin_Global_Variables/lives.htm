vidas

Esta variable es de alcance global y se utiliza para mantener un valor numérico que normalmente se utiliza para las vidas de los jugadores. Esta variable sólo está diseñada para soportar proyectos heredados de versiones anteriores de GameMaker y debería no se debe usar en proyectos nuevos ya que puede ser obsoleta en el futuro.

Sintaxis:

vidas;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo

si (vidas < 5)
{
vidas += 1;
}

El código anterior comprueba la variable lives y si es menor que 5, se le añade 1.
