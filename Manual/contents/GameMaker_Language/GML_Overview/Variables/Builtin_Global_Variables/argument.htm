argumento

Esta variable contiene un array que se utiliza junto con la variable de sólo lectura argument_count en funciones o métodos de script. Cada posición de la matriz contiene un valor de entrada para la función y se utiliza específicamente con funciones de argumento variable .

Obsérvese que también hay una serie de variables independientes de ámbito global que también pueden utilizarse en las funciones definidas por el usuario para hacer referencia a los diferentes argumentos de entrada: argument0, argument1, argument2, etc... hasta .argument15

NOTA: Esto ya no es estrictamente necesario y se proporciona como soporte de legado más que nada. Todas las funciones definidas por el usuario aceptarán argumentos variables, y cualquier argumento que se pase a una función que no forme parte del nombre será inicializado a indefinido por defecto, lo que puede ser comprobado usando la función is_undefined().

Sintaxis:

argumento[n]
argumento1
argumento2
...
argumento15

Devuelve:

Valor (puede ser de cualquier tipo de datos suministrados a la función)

Ejemplo:

function mi_matriz()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
si argument_count > i
{
arg[i] = argument[i]
}
else
{
arg[i] = -1;
}
}
}

El código anterior utiliza la variable argument_count junto con la matriz argument para inicializar las variables, y es un ejemplo de cómo permitir que una función definida por el usuario acepte un número variable de argumentos.
