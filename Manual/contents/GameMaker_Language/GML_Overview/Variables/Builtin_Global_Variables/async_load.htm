async_load

Esta variable es de ámbito global y se utiliza para mantener un mapa DS cuando se utiliza en los Eventos Asíncronos, y -1 en todas las demás ocasiones. El contenido real del mapa DS dependerá del tipo de evento asíncrono, así como de la función que se utilizó para desencadenar el evento, así que consulte las páginas individuales de esos eventos para obtener detalles completos de todos los posibles contenidos del mapa DS.

Sintaxis:

async_load;

Devuelve:

Real (valor entero)

Ejemplo ampliado:

sprite = sprite_add("sitio.com/ruta/imagen.png", 16, true, true, 0, 0);

El código anterior sería llamado en un evento para cargar un sprite desde una URL externa. Esto desencadenaría el evento asíncrono Image Loaded, donde se analizaría el mapa async_load:

if ds_map_find_value(async_load, "id") == sprite
{
if (ds_map_find_value(async_load, "status") >= 0)
{
sprite_index = sprite;
}
}

El código anterior comprobará primero el ID del mapa async_load y luego el estado de la llamada de retorno. Si el valor es mayor o igual a 0 (señalando el éxito), el resultado de la llamada de retorno se utilizará para establecer el índice del sprite de la instancia a la imagen recién cargada.
