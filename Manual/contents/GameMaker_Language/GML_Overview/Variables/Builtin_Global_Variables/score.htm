puntuación

Esta variable es de alcance global y se utiliza para mantener un valor numérico que normalmente se utiliza para la puntuación del jugador. Esta variable sólo está diseñada para soportar proyectos heredados de versiones anteriores de GameMaker y debería no debe utilizarse en proyectos nuevos ya que puede quedar obsoleta en el futuro.

Sintaxis:

puntuación;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo

switch (object_index)
{
case obj_Enemy_Fighter:
score += 10;
break;
case obj_Enemy_Mage:
score += 25;
break;
case obj_Enemy_Boss:
score += 100;
break;
}
instance_destroy();

El código anterior comprueba el índice del objeto de la instancia que ejecuta el código utilizando un interruptor, y luego añade diferentes cantidades a la variable score dependiendo de qué objeto sea.
