salud

Esta variable es de alcance global y se utiliza para mantener un valor numérico que normalmente se utiliza para la salud del jugador. Esta variable sólo está diseñada para soportar proyectos heredados de versiones anteriores de GameMaker y debería no ser utilizado en los nuevos proyectos ya que puede ser obsoleta en el futuro.

Sintaxis:

salud;

Devuelve:

Real (valor de punto flotante de precisión única)

Ejemplo

if (health <= 0)
{
global.state = "Game Over";
instance_destroy();
}

El código anterior comprueba la variable health y si es menor o igual a 0, se establece una variable global y se destruye la instancia.
