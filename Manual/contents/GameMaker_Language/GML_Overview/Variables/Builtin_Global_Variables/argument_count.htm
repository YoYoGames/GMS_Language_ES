conteo_de_argumentos

Esta variable de sólo lectura devuelve el número de "argumentos" que se pasan a una función de script o a un método. Normalmente se utiliza junto con una matriz de argumentos (argumento[0...15]) para permitir variar los argumentos de entrada de una función determinada.

NOTA: Esto ya no es estrictamente necesario y se proporciona como soporte de legado más que nada. Todas las funciones definidas por el usuario aceptarán argumentos variables, y cualquier argumento que se pase a una función que no forme parte del nombre será inicializado a indefinido por defecto, lo que puede ser comprobado usando la función is_undefined().

Sintaxis:

cuenta_de_los_argumentos;

Devuelve:

Entero

Ejemplo:

function mi_matriz()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
si argument_count > i
{
arg[i] = argument[i]
}
else
{
arg[i] = -1;
}
}
}

El código anterior utiliza la variable argument_count junto con la matriz argument para inicializar las variables, y es un ejemplo de cómo permitir que una función definida por el usuario acepte un número variable de argumentos.
