Visión generaldel lenguaje GameMaker

Esta sección del manual contiene toda la información necesaria para entender y utilizar el GameMaker Studio 2 lenguaje de scripting GML. Las funciones de tiempo de ejecución en este lenguaje se puede utilizar para crear sus juegos y se añade en los objetos de la Editor de Objetos, normalmente cuando has elegido hacer un nuevo Proyecto deLenguaje GameMaker aunque también se puede utilizar junto con DnD™(arrastrar y soltar). A continuación puedes ver una imagen típica de un objeto con el editor de código abierto en un evento:

Cada evento tiene su propia pestaña en el editor y puede añadir, editar o eliminar código de ellos en cualquier momento (para más información sobre los eventos, consulte Eventos de objetos). El código en sí debe tener una estructura básica y puede contener índices de recursos, variables, funciones, expresiones, palabras clave... etc... todo lo cual se explica en las secciones siguientes. Si eres un novato en la programación o estás haciendo el cambio desde DnD™, se recomienda que empieces con la página sobre Estructura básica del código y luego lea todas las demás páginas de esta sección, probando el código de cada una dentro de GameMaker Studio 2 mismo.
