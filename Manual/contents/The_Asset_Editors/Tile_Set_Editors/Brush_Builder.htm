El constructor de pinceles para conjuntos de azulejos

Por defecto, cuando se "pintan" azulejos en una capa de mapa de azulejos en el editor de habitaciones, se selecciona un solo azulejo y se pinta con él. Sin embargo, los conjuntos de azulejos están casi siempre diseñados para tener secciones que encajan de diferentes maneras para formar áreas o elementos completos. Por ejemplo, un juego de baldosas de RPG puede tener baldosas de características del paisaje que pueden conectarse para crear características más grandes o más pequeñas - como árboles y edificios - dependiendo del número de baldosas utilizadas. Ahora, la colocación de múltiples características como esta en una capa de habitación requeriría que usted vaya y venga muchas veces para cambiar de baldosa, lo cual no es bueno para su flujo de trabajo. Para resolver esto, GameMaker Studio 2 le permite crear pinceles de azulejos en el editor de conjuntos de azulejos, disponible al hacer clic en en el botón Brush Builder:

En el Brush Builder tienes el conjunto de azulejos originales a la izquierda y un "lienzo" en blanco a la derecha. Ahora puede seleccionar cualquier baldosa de la izquierda y pintarla en la derecha para crear "pinceles" personalizados que luego podrás utilizar en el editor de habitaciones. Ten en cuenta que puedes mantener pulsado el botón izquierdo del ratón y luego arrastrar el conjunto de baldosas para seleccionar varias baldosas para pintar en el lienzo del pincel y también puedes mantener pulsado / + para añadir baldosas individuales específicas o mantener pulsado + para eliminar baldosas específicas.

La siguiente imagen muestra nuestras características de paisaje mencionadas anteriormente como ejemplo de tres pinceles personalizados realizados a partir de un único conjunto de azulejos:

A la derecha puedes ver tres características que hemos realizado (resaltadas con un recuadro naranja en la imagen). Observe que hemos dejado un espacio de una baldosa entre cada característica - esto es porque cualquier grupo de baldosas que se toquen serán tratadas como un solo pincel en el editor de habitaciones, por lo que dejamos un espacio de una baldosa para mostrar que cada conjunto es un pincel distinto pincel que queremos crear. Mientras creas tus pinceles, pintas con el botón izquierdo del ratón y borras con el botón derecho . También puede ampliar la hoja de azulejos o el lienzo de pinceles utilizando / y la rueda del ratón cuando el ratón cuando el ratón está sobre una de las ventanas, o puede desplazarse utilizando + o el botón central del ratón y arrastrando. Tenga en cuenta que también puede utilizar las herramientas de zoom en la caja de herramientas dentro de las ventanas del editor para cambiar los niveles de zoom de cualquiera de las ventanas, así como para cambiar la visibilidad y el color de la cuadrícula.

En la parte superior derecha puedes ver la herramienta actualmente seleccionada, y también puedes establecer el tamaño del pincel con el que quieres pintar. El tamaño por defecto es 1, que es un solo azulejo, pero si lo ajustas a valores más altos entonces puedes pintar (y borrar) con un pincel más grande compuesto por el azulejo seleccionado repetido, como se muestra en la imagen siguiente:

La caja de herramientas es donde puede seleccionar la herramienta a utilizar para muchas tareas diferentes en el editor de conjuntos de azulejos, algunas de las cuales dependerán de si de si tiene algo definido en su biblioteca de azulejos automática. A continuación se ofrece un breve resumen de cada herramienta (tenga en cuenta que cuando ha seleccionado una capa de baldosas en el Editor de habitaciones, esta caja de herramientas se muestra en la parte superior del espacio de trabajo de la habitación junto con otras herramientas de dibujo):

Lápiz Esta es la herramienta lápiz. Utiliza el azulejo seleccionado para pintar en la ventana de Pinceles con el botón izquierdo del ratón y puede borrar con el botón derecho del ratón .
Borrador Con la herramienta de borrado puedes utilizar el botón izquierdo del ratón para borrar un azulejo determinado en la ventana de Pinceles. Esencialmente, todo lo que esto hace es poner el índice del azulejo a 0, que es el azulejo reservado "vacío".
Selección Esta es la herramienta de selección, que se puede utilizar para definir un área de la ventana de Pinceles para trabajar en ella. Puede pulsar el botón izquierdo del ratón y luego arrastrar el ratón para crear un área rectangular, o puede pulsar / + para añadir selecciones múltiples y + para borrar una parte de la selección. Para borrar toda la selección puede pulsar . Cuando tienes un área de la ventana de Pinceles seleccionada, el resto de las herramientas (Lápiz, Voltear, Rotar, etc...) sólo funcionarán dentro del área seleccionada. Tenga en cuenta que también puede seleccionar mosaicos en la ventana y luego copiar ( / + "C"), cortar ( / + "X") y pegar ( / + " V"), lo que cambiará la herramienta a Lápiz y le permitirá "pintar" con ellos.
AutoTile Al hacer clic en esta herramienta se activa el estilo de pintura Auto tiling. Cuando esto está activo, puede seleccionar cualquier azulejo de la ficha de Bibliotecas Autotile, y luego pintarlo en la ventana de Pinceles y GameMaker Studio 2 lo cambiará automáticamente para que coincida con los azulejos circundantes, siempre y cuando haya configurado correctamente la ficha Auto Tile. Tenga en cuenta que la selección de un azulejo del conjunto de azulejos que no es parte de las bibliotecas autotile restablecerá la herramienta de dibujo a la estándar Lápiz.
Flip Haciendo clic en la herramienta Voltear con el botón izquierdo del ratón (o utilizando el atajo de teclado "X" ) se volteará el azulejo o azulejos actualmente seleccionados para dibujar a lo largo del eje horizontal (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel se volteará). seleccionado para dibujar a lo largo del eje horizontal , sin cambiar la herramienta de dibujo (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel se volteará). Si no tiene ningún azulejo seleccionado para dibujar, pero tiene un grupo de azulejos seleccionados en la ventana del pincel, entonces la herramienta de volteo volteará los azulejos seleccionados.
Espejo Haciendo clic en la herramienta Espejo con el botón izquierdo del ratón (o utilizando el atajo de teclado " Y" ) se reflejará la(s) baldosa(s) actualmente seleccionada(s) para dibujar a lo largo del eje vertical. seleccionado para dibujar a lo largo del eje vertical , sin cambiar la herramienta de dibujo (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel se reflejará). Si no tiene ningún azulejo seleccionado para dibujar, pero tiene un grupo de azulejos seleccionados en la ventana del pincel, la herramienta Espejo reflejará los azulejos seleccionados.
Girar Haciendo clic en la herramienta Girar con el botón izquierdo del ratón (o utilizando el atajo de teclado " Z" ) girará la(s) baldosa(s) actualmente seleccionada(s) para dibujar seleccionado para dibujar 90° en el sentido de las agujas del reloj, sin cambiar de herramienta de dibujo (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel girará). Si no tiene ningún azulejo seleccionado para dibujar, pero tiene un grupo de azulejos seleccionados en la ventana de ventana del pincel, entonces la herramienta Rotar rotará los azulejos seleccionados.

Debajo de las herramientas, puede encontrar dos secciones diferentes para seleccionar cualquier auto azulejo o azulejosanimados que se han creado utilizando la imagen del conjunto de azulejos actual. Un solo sprite que se utiliza para un conjunto de baldosas puede tener muchas, muchas, imágenes de celdas individuales en él, y estas pueden ser combinadas en el editor de Animación o de Autobaldosas para crear pinceles personalizados que se mostrarán en estas secciones y pueden ser utilizados en conjunto con azulejos estáticos regulares para crear pinceles (tenga en cuenta que un azulejo animado se animará independientemente de si lo ha seleccionado de la biblioteca o del conjunto de azulejos base).

Una vez que hayas configurado todos los pinceles que necesites, podrás utilizarlos para pintar azulejos en cualquier capa del mapa de azulejos dentro del Editor de habitaciones.
