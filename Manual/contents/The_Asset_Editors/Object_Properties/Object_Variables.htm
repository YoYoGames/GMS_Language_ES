Variables del objeto

Al hacer clic en el botón de Variables se abrirá la ventana de Variables del Objeto , que puede utilizar para generar cualquier número de variables antes del Evento Crear del objeto:

En esta ventana puede hacer clic en el botón Add para añadir una nueva variable a la lista. A continuación, puede nombrar la variable, así como establecer su tipo y ajustar sus propiedades:

El nombre de la variable debe empezar por una letra y sólo puede contener letras, números y el símbolo de subrayado '_' con con una longitud máxima de 64 símbolos. Por lo tanto, las variables válidas son cosas como fish, foo_bar, num1, y las no válidas serían 6fish, foo bar , o *num. Una vez que hayas dado el nombre de la variable, tienes que establecer su tipo, que puede ser cualquiera de los siguientes: Vale la pena señalar que esta característica es particularmente útil cuando se trabaja con instancias padre/hijo - ya que puede tener un objeto padre con un conjunto de variables definidas y luego simplemente modificarlas en el objeto hijo (ver la sección Variables de Objeto y Objetos Hijos más adelante para más detalles) - y para cuando se agregan instancias en el Editor de Salas (ver la sección Capas > Capa de Instancia), ya que también puede modificar estas variables para instancias individuales que se han colocado en la sala.

A continuación se enumeran los diferentes tipos de variables que se pueden crear:

  • Real: Un número real es cualquier número que no es un entero y puede ser positivo o negativo. Así, 124.5, 45639.566546456, 0.9, -45.5 etc... son todos ejemplos de números reales. Todos los números reales se almacenan como valores de coma flotante de 64 bits (dobles). Los tipos de datos reales (y enteros) tienen una opción para establecer un rango de valores a la que se puede acceder haciendo clic en el botón Opciones :Cuando marque esta opción puede introducir un valor inicial y un valor final y en lugar de tener un valor fijo mostrado para la variable en la ventana de Variables de Objeto, tendrá un deslizador que se ajusta a estos valores:
  • Número entero: Un entero es un número entero y puede ser positivo o negativo, por ejemplo 30004, 19, 0, -300. Tenga en cuenta que puede establecer un rango de valores para el entero de salida, como se explicó anteriormente para los números reales.
  • Cadena: Una cadena es cualquier cosa que se haya colocado entre comillas dobles, por ejemplo "pez", "Hola Mundo" o "12345".
  • Booleano: Un booleano es un valor que es verdadero o falso. En la ventana de Variables del Editor de Objetos, esto se muestra simplemente como una casilla que se marca para verdadero y se desmarca para falso.
  • Expresión: Una expresión es una frase matemática que puede contener números ordinarios, variables, cadenas o funciones GML, así como uno o más operadores. Por ejemplo, sqrt(85 * 6) + 5.5 es una expresión.
  • Recurso: Un recurso es simplemente cualquiera de los recursos que ha definido en el Navegador de Activos. Cuando selecciona este tipo de recurso, puede hacer clic en el botón Abrir el Explorador de Activos para abrir el Explorador de Activos y seleccionar el recurso que necesita.
  • Lista: Seleccionar un tipo de entrada de lista significa que puede crear una selección de valores (pueden ser cadenas, reales, expresiones, etc.) de los cuales puede elegir uno o más para que la variable los devuelva.     Tenga en cuenta que si tiene marcada la opción Multiselección en las Opciones de la lista, la variable se convertirá en una matriz de todas las opciones seleccionadas:
  • Color: El tipo de color sirve para definir un valor de color que se almacenará en la variable. Puede introducir un número real (de 0 a 16777216, que tendrá un valor alfa de 255), o un valor hexadecimal (en el formato de $RRGGBBAA), o puede puede hacer doble clic en en la muestra de color para abrir el selector de color y definir el color allí.

Variables de objetos y objetos hijos

Una característica importante de las Variables de Objeto es que son heredadas por cualquier objeto hijo que tenga en el Navegador de Activos (vea la sección sobre Objetos Padre para más información), lo que significa que usted lo que significa que usted puede elegir anular o cambiar alguna o todas ellas si lo desea.

Cuando se crea un objeto hijo de otro recurso objeto que tiene Variables definidas para él, éstas se mostrarán en la ventana de Variables de Objeto de esta manera:

Puede ver en la imagen superior que el objeto Padre tiene cuatro Variables de Objeto y en la imagen inferior estas también se muestran, sólo que "atenuadas" y con el icono Heredado del Padre para indicar que han sido heredadas. Estas variables heredadas pueden ser editadas si hace clic en el botón Anular Variable , de modo que puede ajustar el deslizador de rango o los valores, o seleccionar diferentes elementos de las listas, etc. Tenga en cuenta que cuando edita una variable padre sólo puede cambiar el valor definido pero no el nombre, ni puede cambiar las opciones de la variable. Tenga en cuenta también que puede añadir nuevas variables de objeto a los objetos hijos también (y en la imagen de ejemplo de arriba puede ver que "Tipo de Carácter" es una nueva Variable de Objeto sólo para el objeto hijo).

Si ha editado una Variable de Objeto heredada, puede hacer clic en el botón Borrar para eliminar la edición, pero no la variable. La variable La variable seguirá siendo heredada del padre, pero utilizará de nuevo el valor por defecto del padre. Si necesita eliminar completamente la variable, entonces debe hacerlo desde el objeto padre.
