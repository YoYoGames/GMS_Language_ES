Nueve rodajas

El Nine Slicing (también conocido como 9-slicing) es una técnica utilizada para escalar imágenes rectangulares con el fin de preservar los detalles, haciendo que conserven su forma original tras el escalado.

Esto se consigue dividiendo la imagen en nueve partes separadas o "rodajas". Las rodajas de las esquinas se desplazan durante el escalado y se dibujan a la misma escala, luego las rodajas de los bordes y del centro se estiran o se repiten (según la configuración) para llenar el espacio.

Se colocan cuatro guías en un sprite para dividir la imagen en nueve cortes. La siguiente imagen muestra cómo las guías permanecen en su lugar después de que el después de que el sprite sea escalado usando Nine Slice:

El modo de mosaico de una rebanada determina cómo se muestra esa rebanada cuando se escala el sprite. Sólo puede establecerse para las rebanadas de los bordes y del centro (ya que las rebanadas de las esquinas no se escalan).

El siguiente visual demuestra cada modo de mosaico cuando se aplica a la rebanada central:Si se habilita Nine Slicing para un sprite, se utilizará para redimensionar el sprite dondequiera que aparezca, y funcionará en cualquier lugar existente en el que se haya utilizado el sprite sin que se requieran otros cambios.

NOTA: Nine Slicing no funcionará con funciones que dibujen sólo una parte del sprite o lo distorsionen, como draw_sprite_part() y draw_sprite_pos().

Junto con las secuencias, esta función puede utilizarse para construir interfaces de usuario, y puede utilizarse, por ejemplo, para crear paredes rectangulares en un nivel de juego. La memoria de las texturas también puede guardarse utilizando Nueve rebanadas, ya que se puede utilizar una imagen más pequeña para cubrir un área mayor en la pantalla.

Nine Slicing puede ser habilitado en Sprites en el IDE. Para más información, véase El editor de sprites.
