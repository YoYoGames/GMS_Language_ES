Notas de publicación

Cada versión de GameMaker Studio 2 viene con su propio conjunto de notas de la versión. Éstas contienen información detallada sobre los cambios que se han realizado, así como una lista completa de todos los errores que se han corregido. Se trata de una herramienta inestimable para decidir si se debe actualizar la versión actual, o cuando se encuentra algo que puede ser un error después de un lanzamiento.

Las notas de la versión están divididas en dos conjuntos diferentes debido a que tanto el IDE como los tiempos de ejecución reales pueden actualizarse de forma independiente. Puede encontrar las notas de la versión actual del IDE en el siguiente enlace:
