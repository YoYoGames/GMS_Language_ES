Preferencias de alimentaciónen tiempo real

Las preferencias de alimentación detiempo de ejecución se utilizan para mostrar y cambiar el tiempo de ejecución actualmente instalado Tiempo de ejecución así como el tiempo de ejecución en uso actualmente. GameMaker Studio 2 puede tener varias herramientas de tiempo de ejecución diferentes instaladas en cualquier momento y puede establecer cuál utilizar desde esta preferencia. Lo ideal es utilizar utilizar el tiempo de ejecución más actualizado, pero a veces puede tener un problema, o puede ser necesario utilizar uno más antiguo para la comparabilidad, etc ... y por lo que puede cambiar las herramientas de tiempo de ejecución que GameMaker Studio 2 utiliza para construir su juego aquí.

La sección inicial de estas preferencias está relacionada con la procedencia de las herramientas de ejecución y requiere que se introduzca una URL de un canal RSS. Esta fuente se escaneará en busca de actualizaciones de las herramientas de tiempo de ejecución, que pueden descargarse e instalarse. Esta sección mostrará la URL de la fuente de tiempo de ejecución principal por defecto, pero puede hacer clic en el botón Añadir para crear una nueva fuente de tiempo de ejecución en la que donde se da el nombre y la URL para el feed RSS, y luego haga clic en Aplicar para decirle a GameMaker Studio 2 que acceda a él. Tendrá que cerrar y volver a abrir la ventana de Preferencias para que se muestre el nuevo feed.

Una vez que se ha creado un feed de tiempo de ejecución, aparecerá en esta sección como una subcategoría, y siempre debería tener al menos un feed en la lista (por defecto sería el feed principal ). Las preferencias de alimentación de tiempo de ejecución se utilizan para seleccionar, instalar y desinstalar las diferentes herramientas de tiempo de ejecución. En la parte superior tienes las herramientas de tiempo de ejecución que se están utilizando, la fecha en que se crearon y la ubicación en la que están almacenadas en el disco duro. Debajo de estos detalles puedes encontrar una lista de todos los tiempos de ejecución disponibles, y puedes elegir cualquiera de esta para que se convierta en el tiempo de ejecución "en ejecución" utilizado para la compilación de tus proyectos.

Para seleccionar un tiempo de ejecución diferente, simplemente búsquelo en la lista y haga doble clic en . Si el tiempo de ejecución no ha sido descargado e instalado, se le pedirá que lo haga. y, una vez instalado, se le pedirá que cierre y reinicie el IDE para que el nuevo tiempo de ejecución pueda ser inicializado.

Esto marca el tiempo de ejecución actual que se ha seleccionado para su uso al construir sus proyectos.
Al hacer clic se iniciará la descarga e instalación del runtime seleccionado (se mostrará una barra de estado en la parte superior derecha del IDE).
Al hacer clic en esta opción, se eliminará el tiempo de ejecución seleccionado del disco duro.

Puedes actualizar la lista de tiempos de ejecución en cualquier momento desde la fuente RSS dada haciendo clic en el botón de actualización , y puedes limpiar tu disco duro de tiempos de ejecución antiguos haciendo clic en el botón "Limpiar tiempos de ejecución ahora".
