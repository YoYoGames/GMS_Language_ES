Preferencias lingüísticas

Las preferencias de lenguaje se utilizan para definir cómo se mostrarán algunas de las características del lenguaje para la programación en el IDE de GameMaker Studio 2. Las opciones para GML son las siguientes:

  • Mostrar los tipos de argumentos en la barra de estado: Esta opción se refiere a cuando usted está usando sus propios scripts personalizados y ha utilizado la notación JSDoc para identificar los diferentes argumentos, etc... (ver aquí para más información). Cuando esta opción está marcada, y ha especificado los tipos de argumentos en el script, entonces se mostrarán en la barra de estado en la parte inferior del editor, encerrados en {}. Esto está desactivado por defecto.
  • Mostrar descripciones de argumentos en la barra de estado: Si esta opción está marcada, cualquier descripción que haya dado a sus scripts personalizados utilizando la notación JSDoc se mostrará en la barra de estado en la parte inferior de la ventana del editor. Esta opción está desactivada por defecto.
  • Mostrar la ortografía de EE.UU. para las funciones del sistema: Por defecto, GameMaker Studio 2 utiliza la ortografía inglesa del Reino Unido para las funciones, por ejemplo draw_text_colour(), sin embargo, puede que no sea exactamente a lo que está acostumbrado, por lo que puede cambiar a la ortografía de EE.UU. utilizando esta opción, de modo que las funciones se convierten en draw_text_color(), por ejemplo. Esta opción está desactivada por defecto.
  • Número máximo de errores de sintaxis a mostrar: Este valor establece el número máximo de errores de sintaxis que se mostrarán en un momento dado. Si encuentra que el IDE se retrasa, puede intentar reducir este valor. Este valor está establecido en 50 por defecto.
  • Habilitar las advertencias de referencias a variables: Cuando se activa, esto marcará cualquier problema potencial con las variables, como cuando una ha sido definida en algún lugar pero no se utiliza realmente en el código. Si se desactiva, estas advertencias no se mostrarán. se mostrarán. Por defecto, esta opción está activada.
