Preferencias generales

Las preferencias generales se utilizan para definir el comportamiento general del IDE de GameMaker Studio 2. Las más generales están listadas en la página principal mientras que muchas otras están divididas en múltiples subcategorías, explicadas más abajo en esta página.

  • Idioma del IDE: Esta opción permite seleccionar el idioma del IDE de la lista de opciones suministrada. El idioma por defecto es el inglés.
  • Piel del IDE: Aquí puede seleccionar entre las diferentes pieles que el IDE puede utilizar. Por defecto GameMaker Studio 2 viene con dos skins, uno claro y otro oscuro, y la configuración inicial utiliza el skin oscuro.
  • Velocidad máxima del editor de GameMaker Studio 2: Esta opción le permite establecer la velocidad de fotogramas máxima a la que se actualizará el IDE de GameMaker Studio 2, permitiéndole establecer la velocidad de fotogramas para que coincida con la velocidad de actualización de su monitor. El valor predeterminado de es de 60.
  • Dispositivo de audiopor defecto: Aquí puede elegir el dispositivo de audio predeterminado que utilizará el IDE de GameMaker Studio 2. Puede actualizar la lista haciendo clic en el botón Actualizar .
  • Respuesta automática a la confirmación de salida: Esta opción permite establecer si se debe mostrar un mensaje de confirmación al salir de GameMaker Studio 2. Por defecto, está configurado como "mostrar mensaje", pero puede configurarse para omitir el mensaje como si hubiera hecho clic en Aceptar.
  • Respuesta automática al diálogo de cambio de fuente del sistema: Esta opción permite establecer si se debe mostrar un mensaje de confirmación cuando se ha añadido, eliminado o editado una fuente del sistema. Por defecto está configurado como "mostrar mensaje", pero puede configurarse para que se omita el mensaje como si se hubiera hecho clic en Aceptar.
  • Habilitar notificaciones: Al marcar esta opción, el IDE mostrará notificaciones sobre nuevos tiempos de ejecución, etc. Esta opción está activada por defecto.
  • Tiempo de espera de la notificación (ms): Establece el tiempo (en milisegundos) que se muestra una notificación. El valor predeterminado es 6000.
  • Cierre de sesión al salir: Si esta opción está marcada, el IDE cerrará automáticamente la sesión del usuario cuando se cierre GameMaker Studio 2. Esto está dirigido principalmente a los usuarios de la Educación, como - cuando se activa - esto significa que cada vez que el programa el programa se cierra, el usuario se desconecta y libera la sesión para que otros usuarios la utilicen. Esta opción está desactivada por defecto.
  • Activar la anulación de los PPP: Esta opción le permite anular la configuración de PPP por defecto con la suya propia. Primero debe habilitar la opción y luego establecer el valor de Anulación de DPi (el valor por defecto es 96) o establecer el Porcentaje del DPi nativo (valor por defecto 100%), y una vez cambiado se le pedirá que reinicie el IDE, lo que puede requerir que guarde cualquier trabajo que se esté realizando en el proyecto actualmente abierto. Tenga en cuenta que la DPi tiene un porcentaje mínimo del 50% y un máximo del 500%, y si pasa el ratón por encima del icono de información puede ver el ajuste máximo recomendado para la pantalla que se está utilizando:

    Tenga en cuenta que si supera el ajuste de DPi recomendado para el monitor, se le mostrará una notificación que le permite restablecer el DPi.
  • Mostrar pantalla de bienvenida al inicio: Esta opción está activada por defecto y simplemente indica a GameMaker Studio 2 que muestre una breve pantalla de bienvenida al iniciar el programa. Si se desmarca esta opción, no se mostrará la pantalla de bienvenida.
  • Permitir que las pestañas vacías se cierren mediante un atajo de teclado: Puedes cerrar cualquier pestaña abierta utilizando el atajo de teclado / + W, pero si desactivas esta opción ese atajo ya no funcionará. Esta opción está activada por defecto.
  • Abrir nuevos recursos para su edición al arrastrar los archivos al IDE: Ciertos recursos (como sprites o sonidos) pueden ser arrastrados al espacio de trabajo principal en el IDE de GameMaker Studio 2 y serán detectados y añadidos como un nuevo recurso en el Navegador de Activos. de activos. Por defecto, el editor de recursos correspondiente también se abrirá para su edición cuando esto ocurra, pero desmarcando esta opción se evitará que se abran los editores y se requerirá que los abras manualmente.
  • Abrir nuevos recursos para su edición en el momento de la creación: Si está marcada, cada vez que se cree un nuevo recurso en el Navegador de Activos, se abrirá automáticamente el editor de recursos correspondiente. Esta opción está marcada por defecto y si se desmarca significa que tendrá que abrir manualmente todos los recursos recién creados.
  • Recargar automáticamente los archivos modificados: Si esta opción está marcada, el IDE escaneará y comprobará automáticamente los archivos para ver si alguno de ellos ha sido modificado fuera de GameMaker Studio 2 y los recargará si es así.
  • Desactivar las animaciones de transición del IDE: El IDE de GameMaker Studio 2 tiene una serie de transiciones animadas para cosas como la apertura de ventanas, etc ... sin embargo, no son del gusto de todos y puede experimentar un aumento de rendimiento si si se desactivan en las máquinas más antiguas o de baja potencia si se desactivan. Puedes desactivarlas aquí, y están activadas por defecto.
  • Optar por las encuestas de opinión: Cuando esta opción está marcada, le está diciendo a GameMaker Studio 2 que desea participar en las encuestas de retroalimentación, lo que significa que ocasionalmente, mientras trabaja, se mostrará un mensaje sobre el área en la que está trabajando en la que está trabajando y le pedirá una valoración y un comentario. Esta opción está activada por defecto.
