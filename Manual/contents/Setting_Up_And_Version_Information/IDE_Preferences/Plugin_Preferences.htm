Preferencias de los plugins

Las Preferencias de los Plugins se utilizan para controlar cualquier plugin extra que se haya añadido al IDE de GameMaker Studio 2. Por defecto, tiene una sección dedicada al plugin de Control de Fuentes para integrar los proyectos de GameMaker Studio 2 con Git.

Control de fuentes(Git)

Esta sección trata de las preferencias que se deben establecer para obtener el Git SCM funcione con GameMaker Studio 2. En primer lugar tiene el conjunto de G eneral :

  • Abrir automáticamente la ventana de salida del control de fuentes: Cuando se marca esta opción, la ventana de salida del control de fuentes se mostrará automáticamente cuando algo cambie. Esta opción está activada por defecto.
  • Registro detallado: Activa el registro detallado para el registro de salida de Git. Está desactivado por defecto.

A continuación está la sección Identidad del usuario, con los siguientes campos que se deben rellenar obligatoriamente:

  • Nombre de usuario para confirmar: Aquí debes introducir el nombre de usuario con el que harás los commits en Git (debe ser el mismo que el especificado al configurar tu repositorio Git ).
  • Correo electrónico para confirmar como: El correo electrónico que está asociado a esta identidad Git.

Después de configurar la identidad del usuario, puede añadir una nueva combinación de nombre de usuario y contraseña para la autenticación haciendo clic en el botón denominado Añadir nueva autenticación de usuario/contraseña, que abrirá la siguiente ventana con diferentes campos para rellenar:

También puede Añadir un nuevo par de claves de autenticación SSH haciendo clic en el botón con esa etiqueta. Esto abrirá la siguiente ventana para que la rellenes:

Debajo de estos botones puede configurar la Herramienta de fusión y la Diff Tool para SCM con estas opciones:

  • Ruta de acceso a la herramienta de fusión: Aquí se especifica la ruta de acceso completa a la herramienta de fusión de Git.
  • Opciones de la herramienta de fusión: En este campo puede añadir cualquier comando extra que se ejecute cada vez que utilice la Herramienta de Fusión.
  • Ruta de acceso a la herramienta de diferencias: Aquí se especifica la ruta completa del archivo a la herramienta de diferencias de Git.
  • Opciones para la Herramienta Diff: En este campo puede añadir cualquier comando extra que se ejecute cada vez que utilice la Herramienta de Diferenciación.

Por último, después de configurarlas, tienes una última opción:

  • Habilitar automáticamente el control de fuentes: Si se marca esta opción, el control de fuentes se habilitará automáticamente para todos los nuevos proyectos. Esta opción está desactivada por defecto.
