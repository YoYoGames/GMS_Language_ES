Preferencias del editor de conjuntos de azulejos

Las Preferencias del Editor de Conjuntos de Azulejos tiene la siguiente opción para el Auto Alicatado:

Color de fondo de la baldosa automática: Esta opción establece el color de fondo dentro del editor de conjuntos de baldosas en la sección de creación de baldosas automáticas. El color por defecto es (rgba) #808080FF.

Las preferencias del editor de conjuntos de azulejos también tienen una subsección de diálogos de confirmación :

La opción disponible aquí controla el mensaje que aparece cuando se cambia el tamaño de la celda del conjunto de azulejos y esto afectará a la forma en que se renderizan las capas del mapa de azulejos de la habitación (dando lugar a capas recortadas). Por defecto se le pedirá que continúe o cancele la operación, y esta opción le permite suprimir este mensaje seleccionando "Okay" que seguirá adelante con el cambio de celdas del conjunto de azulejos sin tener en cuenta el impacto en las capas del mapa de azulejos de la habitación. El valor por defecto es "Mostrar mensaje".
