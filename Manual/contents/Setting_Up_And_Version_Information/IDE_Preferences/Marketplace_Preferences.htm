Preferenciasdel mercado

Las Preferencias del Mercado se utilizan para configurar el Certificado del Mercado para subir o comprar activos. El formulario del certificado consta de las siguientes secciones para cuando ya haya creado un certificado (véase más abajo en esta página para ver los detalles sobre cómo crear un certificado):

  • Archivo de certificado: En esta sección se especifica el archivo *.pem que el Marketplace utiliza para la certificación de los activos. Si tienes un certificado previamente creado, puedes hacer clic en el botón del explorador de archivos y añadirlo al foro, pero si no tienes un certificado y deseas crear uno, entonces debes omitir cualquier información aquí y en su lugar hacer clic en el botón Crear nuevo certificado (explicado más abajo).
  • Contraseña del certificado: Aquí debe introducir la contraseña del certificado para el archivo seleccionado anteriormente. La contraseña se crea cuando se crea por primera vez el archivo *.pem (véase Crear un nuevo certificado, más adelante) y se vinculará de forma irreversible al archivo de certificado creado.
  • Comprobar: Una vez que haya rellenado los campos anteriores, debe hacer clic en este botón para asegurarse de que la contraseña y la selección de archivos son correctas y funcionan juntas.

Confirmar diálogos

Aquí puede seleccionar el comportamiento de los diálogos de confirmación cuando se trabaja con un recurso desde el Navegador de Activos. Las opciones disponibles son:

  • Respuesta automática a la eliminación de contenidos sin conexión: Cuando eliminas un activo del mercado de tu biblioteca sin conexión a la web, normalmente recibes una advertencia preguntando si estás seguro de que quieres proceder. Al establecer esta opción en "Aceptar" el mensaje se suprimirá como si hubieras pulsado el botón "Aceptar" para proceder. El valor por defecto es "Mostrar mensaje".

Crear un nuevo certificado

Si no ha creado previamente un archivo de certificado para utilizar el Mercado, deberá hacer clic en el botón denominado Crear nuevo certificado, situado en la parte inferior de este formulario. Al hacerlo, se abrirá una nueva ventana en la que se le pedirá que que introduzca la siguiente información:

  • Nombre del editor: Este es el nombre que se utilizará como nombre del editor para el activo. En el panel de control de Marketplace puede configurar diferentes editores si lo desea y cada uno debe tener su propio archivo de certificado *.pem correspondiente.
  • Contraseña del certificado: Aquí debe dar una contraseña única para el archivo del certificado que se va a crear. Tenga cuidado con lo que elija ya que no podrá cambiarlo más tarde y si no tiene la contraseña correcta para el archivo de certificado en las secciones anteriores no podrá actualizar ningún activo del Mercado existente (ni subir nuevos).

Con estos datos, puede hacer clic en el botón Crear certificado para generar el archivo *.pem. Se le pedirá que indique una ubicación de almacenamiento y un nombre para el archivo, y deberá tomar nota de ello para futuras referencias. Una vez hecho esto, el nuevo archivo y la contraseña se añadirán automáticamente en los campos del formulario que aparecen más arriba y podrá hacer clic en el botón Comprobar para comprobar que el certificado se ha generado correctamente.

NOTA: Le recomendamos que haga una captura de pantalla de la configuración actual, así como una copia del archivo *.pem . Ambos deben guardarse juntos en una ubicación segura para futuras referencias, o en caso de que tenga que volver a instalar GameMaker Studio 2 en algún momento. No puede actualizar los activos que ya se han cargado a menos que el paquete actualizado esté firmado con el mismo certificado con el que se creó el original.
