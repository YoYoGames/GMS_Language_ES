Preferencias de la línea de tiempo

Las Preferencias de las Líneas de Tiempo se utilizan para definir ciertas propiedades para cuando se trabaja con el Editor de Líneas de Tiempo. Las opciones disponibles son:

  • Respuesta automática a la conversión de DnD™a lenguaje GameMaker: Cuando se utiliza el botón derecho del ratón opción de menú en un momento de la línea de tiempo para convertir las acciones de Drag and Drop™en script (código) obtendrá un aviso preguntándole si desea continuar, ya que se trata de una conversión de un solo sentido. Cambiando esta opción a "Okay" se suprimirá el mensaje y se seguirá adelante con la conversión como si hubieras hecho clic en "Okay" en el diálogo. La configuración por defecto es "Mostrar mensaje".
  • Una respuesta automática a la advertencia de acción permanente: Ciertas acciones dentro de una línea de tiempo no se pueden deshacer, y cuando eso ocurre recibirás un mensaje de advertencia preguntándote si quieres continuar. Si cambias esta opción a "Aceptar" suprimirá el mensaje y seguirá adelante con la acción como si hubiera hecho clic en "Aceptar" en el diálogo. La configuración por defecto es "Mostrar mensaje".
  • Abrirnuevos momentos para editar en la creación: Normalmente, cuando añades un nuevo momento a una línea de tiempo, GameMaker Studio 2 abrirá automáticamente el editor para que añadas tu código o arrastres y sueltes en él. Sin embargo, si desmarca esta opción, el momento se añadirá sin abrir el editor (puede abrirlo haciendo doble clic en en el momento). Por defecto esta opción está activada.
