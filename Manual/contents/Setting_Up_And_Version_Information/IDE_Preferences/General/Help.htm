Preferencias de ayuda

El archivo de ayuda está diseñado para funcionar como una página web y, aunque los archivos se almacenan localmente, requiere el uso de un navegador externo para verlos. Aquí se pueden cambiar los siguientes ajustes relacionados con el archivo de ayuda:

  • Idioma del manual: Establece el idioma que utilizará el IDE. El valor por defecto es "Inglés".
  • Puerto manual de ayuda: Establece el número de puerto que utilizará el microservidor web manual. El valor por defecto es 51291.
