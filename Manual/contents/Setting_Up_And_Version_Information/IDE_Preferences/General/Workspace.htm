Preferencias del espacio de trabajo

Las preferencias aquí afectarán a la forma de moverse dentro de los diferentes espacios de trabajo de GameMaker Studio 2 espacios de trabajo. Las opciones son:

  • Rango de grados de navegación del teclado del espacio de trabajo: Dentro de un espacio de trabajo puedes tener varias ventanas abiertas para diferentes objetos y recursos, por lo que para moverte rápidamente entre ellas puedes utilizar el atajo de teclado / + + <Arrow Keys> para moverte entre ellas. Dado que las ventanas dentro de un espacio de trabajo no están alineadas en cuadrícula, el IDE tiene que elegir a qué ventana se refiere cuando pulsa cualquiera de las 8 direcciones posibles, y esta opción permite establecer el "ángulo de visión" que se utiliza. El valor por defecto es de 40°.
  • Relleno de la columna del espacio de trabajo (px): Los elementos del espacio de trabajo suelen estar "encadenados" entre sí (por ejemplo, el Editor de código está encadenado al Editor de eventos está encadenado al Editor de objetos), y esta opción le permite establecer la distancia horizontal entre dos ventanas encadenadas. El valor por defecto es de 30px.
  • Relleno de la fila del espacio de trabajo (px): Como se ha mencionado anteriormente, los elementos del espacio de trabajo suelen estar "encadenados" entre sí y esta opción permite establecer la distancia vertical entre dos ventanas encadenadas cualesquiera. El valor por defecto es de 20px.
  • Recuento de segmentos de renderización de la cadena del espacio de trabajo: Aquí puedes establecer el número máximo de segmentos para las cadenas de renderizado. Bajar este valor puede ayudar al rendimiento en máquinas de gama baja. El valor por defecto es 20.
  • Color de la cadena del espacio de trabajo: Este es el color que se utilizará para mostrar las cadenas entre los elementos del espacio de trabajo. El color por defecto es ARGB $99DD00.
  • Lanavegación por el teclado del espacio de trabajo restablece el zoom: Como se ha mencionado anteriormente, puedes saltar de una ventana a otra dentro del espacio de trabajo utilizando los atajos de teclado. También puede acercar y alejar el espacio de trabajo utilizando / + la rueda del ratón . Si se configura esta opción, si tienes el espacio de trabajo ampliado y utilizas los atajos para saltar a otra ventana, el espacio de trabajo volverá a tener una vista 1:1 y estará centrado en la ventana seleccionada. Esta opción está desactivada por defecto.
  • Las cadenas de espacios de trabajopueden superponerse: Cuando está activada esta opción permitirá que las cadenas de espacios de trabajo se crucen y superpongan. Esta opción está desactivada por defecto.
