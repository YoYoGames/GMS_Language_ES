Preferencias del editor de objetos

Las Preferencias del Editor de Objetos se utilizan para definir ciertas propiedades de la ventana del Editor de Objetos. La sección principal contiene la siguiente opción:

  • Método de clic para abrir el evento: Esta opción controla si es necesario realizar un doble clic o un solo clic en un evento de objeto para abrirlo. Por defecto se establece en doble clic.
  • Alfa de los eventos heredados: Esta opción permite cambiar el alfa con el que se dibujarán los eventos ¡heredados en la lista de eventos del objeto. Por defecto está configurado a 0,5 (los valores deben estar entre 0 y 1).
  • Formato de la etiqueta de eventos: Aquí puede habilitar o deshabilitar los diferentes textos posibles que se pueden aplicar a la etiqueta del evento en la lista de eventos del editor de objetos. Por defecto sólo está habilitada la Descripción del Evento.
  • Añadir contenido de eventos por defecto: Por defecto esta opción está activada y lo único que hace es añadir un simple comentario en el editor de código de todos los nuevos eventos para recordarle que puede añadir su código allí.
  • Contenido del evento por defecto: Aquí puedes editar los comentarios que se añaden por defecto en la ventana de código para los nuevos eventos.

La siguiente subcategoría también existe para el editor de objetos:

Confirmar diálogos

Esta sección trata de los diálogos que se pueden mostrar mientras se utiliza el Editor de Objetos, con la siguiente opción disponible:

  • Respuesta automática a la conversión de DnD™ a lenguaje Gamemaker: Puede utilizar el botón derecho del ratón dentro del espacio de trabajo de Drag and Drop™para seleccionar la opción de convertir todos los Drag and Drop™de ese evento en código GML. La acción por defecto aquí es mostrar un mensaje de advertencia preguntando si quieres continuar, pero puedes establecerlo en "Okay" y el diálogo no se mostrará antes de la conversión.
  • Respuesta automática a la superación del límite de vértices de la física: Esta opción puede utilizarse para controlar el mensaje emergente que se muestra cuando se supera el número de vértices permitido en el editor de formas de la física. La acción por defecto aquí es mostrar un mensaje de advertencia, pero se puede establecer en "Okay" y el diálogo no se mostrará.
  • Elección automática al borrar un evento: Esta opción se puede utilizar para controlar el mensaje emergente que se muestra cuando se elimina un evento en la lista de eventos del Editor de objetos. La acción por defecto aquí es mostrar un mensaje de advertencia, pero puede pero puede establecerlo en "Aceptar" y el diálogo no se mostrará.
