Preferencias del editor de sprites

Las preferencias del editor de sprites se utilizan para definir el aspecto y el funcionamiento del editor de sprites. Existen las siguientes opciones:

  • Origen del spritepor defecto: Esta opción permite establecer la posición de origen por defecto para cualquier nuevo recurso de sprite que se cree. El valor por defecto es el superior izquierdo.
  • Tipo de máscara de colisión desprites por defecto: Esta opción permite establecer el tipo de máscara de colisión por defecto que se utilizará para los nuevos sprites. Las opciones son rectángulo, rectángulo girado, elipse, diamante, o precisa / precisa por cuadro. El valor por defecto es rectángulo.
  • Modo de animación desprites por defecto: Esta opción permite establecer el modo de animación por defecto para los sprites en el editor de sprites. Puedes elegir entre "Fotogramas por segundo" o "Fotogramas por cuadro de juego ", La opción por defecto es "Fotogramas por segundo ".
  • Velocidad de animación del sprite por defecto: Esta opción permite establecer el valor de la velocidad de animación por defecto para cuando se crea un nuevo activo de sprite. Tenga en cuenta que este valor está vinculado al modo de animación y, por ejemplo, 3 fotogramas por segundo no es lo mismo que 30 fotogramas por cuadro de juego, así que asegúrese de establecer este valor correctamente en referencia al modo preferido. El valor por defecto es 30.
  • Anchura por defecto del sprite: Esta opción permite establecer la anchura por defecto de todos los nuevos sprites creados. El valor por defecto es 64.
  • Altura delsprite por defecto: Esta opción permite establecer la altura por defecto para todos los nuevos sprites. El valor por defecto es 64.
  • Aplicar cambio de tamaño y escala al origen del sprite: Cuando está marcada esta opción cambiará el origen del sprite cuando se redimensione o escale el sprite, con el nuevo origen siendo movido/escalado para que coincida con los nuevos valores del sprite. Tenga en cuenta que esto sólo es aplicable a los orígenes de los sprites personalizados que no están bloqueados en el editor de sprites. Si el origen está bloqueado o el sprite utiliza una de las posiciones por defecto - por ejemplo: "Centro" o "Superior Izquierda", etc... - entonces esta opción estará estará en gris y no estará disponible para ser cambiada. Por defecto está activada.

La siguiente subcategoría también existe para el Editor de Sprite:

Confirmar diálogos

Las opciones disponibles aquí controlan los mensajes que aparecen cuando se requiere confirmar una acción. Puede cambiar estos ajustes para mostrar el mensaje siempre o descartarlo con una acción específica (ya sea Sí o No) :

  • Respuesta automática a la importación de archivos: Cuando importas una nueva imagen y ya existe una, por defecto se te pedirá que continúes la importación y sobrescribas la imagen existente o canceles la operación. Esta opción le permite suprimir este mensaje y cancelar siempre la importación seleccionando "No", o sobrescribir el sprite actual seleccionando "Sí". El valor por defecto es "Mostrar mensaje".
  • Respuesta automática a los fallos en la carga de imágenes: Cuando importas una nueva imagen y la importación falla, por defecto se te mostrará un mensaje de advertencia informándote de que ha habido un fallo. Esta opción le permite suprimir este mensaje seleccionando "Aceptar". El valor por defecto es "Mostrar mensaje".
