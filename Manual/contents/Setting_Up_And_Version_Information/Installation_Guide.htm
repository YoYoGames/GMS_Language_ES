Guía de instalación

GameMaker Studio 2 está diseñado para su uso en macOS y Windows ordenadores de sobremesa con las especificaciones mínimas mencionadas aquí.

Puedes descargar el instalador de GameMaker Studio 2 desde la página de tu cuenta de YoYo en el sitio webde YoYo Games. Una vez instalado, la primera vez que ejecutes GameMaker Studio 2 se te pedirá que actualices antes de poder utilizar el producto. Una vez instaladas las actualizaciones, se te pedirá que proporciones los datos de la licencia de tu cuenta de YoYo Games (ver aquí para más información) antes de poder utilizar el producto, lo que significa que la primera vez que utilices GameMaker Studio 2 deberás tener una conexión a Internet. Tras el inicio de sesión inicial, deberías poder utilizar el producto sin conexión, aunque periódicamente te pedirá que te conectes para verificar la cuenta de acceso.

Hay que tener en cuenta que, dependiendo del tipo de licencia que tengas, es posible que tengas que instalar componentes adicionales o seguir pasos adicionales para configurar y utilizar GameMaker Studio 2 para plataformas específicas. Para obtener información completa sobre lo que requiere cada plataforma requiere, consulta este artículo en el Helpdesk de YoYo Games: GMS2 Required SDKs.

Versiones estables y beta

La versión que descargues de la página web de YoYo Games debe ser la versión estable actual de GameMaker Studio 2, pero también puede haber otra versión "Beta" disponible que se utiliza para mostrar o probar nuevas funcionalidad antes de pasar a la versión estable final. Los detalles de la versión completa son los siguientes:

  • Estable: Esta versión es la que se usaría para el desarrollo general y se recomienda su uso con cualquier producto comercial. Puede carecer de ciertas características disponibles en las versiones Beta, pero esto se debe a que no han sido completamente probadas o requieren trabajo adicional.
  • Beta: La versión Beta de GameMaker Studio 2 es para aquellos que desean probar nuevas características antes de que se añadan a la versión estable, o para aquellos que necesitan una solución para un problema. No se recomienda utilizar la versión Beta para la producción final del juego, a menos que contenga una corrección importante de un error que esté retrasando su proyecto. Puedes obtener la versión Beta de GameMaker Studio 2 en la página de "Descargas" de tu cuenta de YoYo, y puede ser instalada y ejecutada junto con la versión estable. con la versión estable. Ten en cuenta que si ejecutas GameMaker Studio 2 a través de Vapor no tendrás acceso a la versión beta del producto a través de Steam, aunque podrás descargar y utilizar la versión de escritorio.
