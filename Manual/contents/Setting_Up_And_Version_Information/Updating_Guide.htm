Guía de actualización

El IDE de GameMaker Studio 2 buscará periódicamente actualizaciones y las descargará automáticamente. Algunas de estas actualizaciones serán opcionales y podrá optar por no instalarlas si no lo desea o cree que algunos de los cambios pueden romper un proyecto que está a punto de publicar. Sin embargo, hay algunas actualizaciones que incluyen correcciones o mejoras importantes que no se pueden omitir y GameMaker Studio 2 forzará una actualización en estos casos.

GameMaker Studio 2 es más que un simple IDE y tiene una serie de herramientas de ejecución diferentes que utiliza entre bastidores para crear tus juegos para las diferentes plataformas de destino disponibles. Estas herramientas son principalmente los diferentes ejecutores y compiladores que cada y compiladores que requiere cada plataforma, y en realidad puedes tener varios de cada uno en cualquier momento. Esto significa que puedes "elegir" qué versión del compilador y qué versión del ejecutor utilizar si has encontrado algún problema con la actual o si está apuntando a una versión o plataforma muy específica. Las herramientas se actualizarán automáticamente cuando haya una nueva versión disponible, pero puede utilizar las Preferencias de las Fuentes deTiempo de Ejecución para seleccionar las que realmente desea utilizar. Para más información sobre las actualizaciones de tiempo de ejecución, consulte Preferencias - Fuentes de tiempo de ejecución.
