Preferencias de HTML5

Las preferencias de HTML5 tienen las siguientes opciones:

  • Tipo de empaquetado por defecto: Cuando construyas un paquete ejecutable HTML5 final, puedes elegir entre Paquete como archivos sueltos - que simplemente generará todos los archivos necesarios en una carpeta - o Paquete como Zip - que creará un archivo comprimido ZIP que tendrá el juego y cualquier archivo adicional dentro. Por defecto se le dará a elegir cuál de estas dos opciones desea elegir en la compilación, pero puede establecer esta preferencia para usar siempre una u otra. Tenga en cuenta que cuando esta opción se establece en Mostrar diálogo, en la ventana de diálogo real que aparece en la compilación, la elección de "Recordar opción de embalaje" establecerá esta preferencia a la opción que usted seleccione.
  • Puerto del servidor web por defecto: Permite cambiar el puerto por defecto para todos los dispositivos creados a través del Administrador de dispositivos. El valor por defecto es 51264.
  • Dirección del servidor web por defecto: Aquí puede cambiar la dirección del servidor web por defecto para los dispositivos que añada al Administrador de dispositivos. El valor por defecto es 127.0.0.1.
  • IPs Permitidas por el Servidor Web por Defecto: Aquí puede añadir cualquier IP permitida para el Servidor Web y se establecerá por defecto para todos los dispositivos añadidos a través del Gestor de Destino.
