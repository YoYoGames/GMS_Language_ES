Preferencias deSteam

El SDK de Steam se utiliza para las construcciones de Windows, macOS y Ubuntu(Linux) que se van a distribuir a través de la plataforma Steam. Esta sección contiene las opciones de configuración necesarias para que funcione en tus proyectos:

  • Ruta de acceso alSDK de Steam: La ruta del sistema donde has instalado el SDK de Steam. Una vez que hayas establecido esta ruta al SDK, tendrás que activar Steam en las Opciones de Juego y luego para cada plataforma añadir el ID de la aplicación.
