Preferencias de la plataforma

Esta sección del manual cubre las diferentes preferencias específicas de la plataforma que necesitan ser configuradas antes de que puedas compilar tus juegos a ciertos objetivos. Por lo general, sólo es necesario configurarlas una vez para empezar (aunque las futuras actualizaciones a la tiempos de ejecución puede requerir que usted cambie ciertas cosas, pero estos serán mencionados en las notas de la versión) y se utilizan para apuntar a cualquier SDKque se requiere y establecer las opciones básicas de compilación para la forma en que desea GameMaker Studio 2 para compilar sus proyectos de juego para cualquier plataforma determinada. Tenga en cuenta que las plataformas disponibles dependerán de la Licencia que tengas y puede que no estén todas disponibles.

A continuación se muestra una lista de todas las plataformas de destino disponibles para las licencias estándar:

  • Android
  • HTML5
  • iOS
  • tvOS
  • macOS
  • Vapor
  • Windows
  • Windows UWP

Tenga en cuenta que, además de configurar estas preferencias, también tendrá que configurar los dispositivos a los que apuntan algunas de estas plataformas, de lo contrario no podrá compilar ni probar sus proyectos. Consulte la sección sobre compilación y la sección sobre el administrador de dispositivos para obtener más información.

El manual no muestra las distintas preferencias de las consolas (Xbox One, PS4 y Nintendo) debido a las diferentes restricciones legales que mantienen las distintas compañías. Si tienes una licencia de consola, puedes obtener esta información de el servicio de asistencia deYoYo Games, pero primero debes ponerte en contacto con YoYo Games con el correo electrónico del ID de tu consola y solicitar permiso para ver las diferentes secciones.
