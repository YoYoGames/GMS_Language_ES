La ventana del inspector

La ventana del Inspector es una ventana que muestra todas las propiedades asociadas a un elemento seleccionado en el IDE. La ventana mostrará información sobre el elemento que se está inspeccionando, así como todas las propiedades editables para ese elemento y le permitirá cambiarlas según sea necesario, actualizando el activo que se está inspeccionando para que coincida.

IMPORTANTE! Actualmente la Ventana del Inspector sólo está disponible para inspeccionar las propiedades de la Secuencia, pero en futuras actualizaciones se ampliará para incluir otros activos.

Puedes abrir varias ventanas de Inspector en cualquier momento, y cada una puede estar vinculada a un elemento específico en el IDE. Para asegurarse de que cada vez que haga clic en un elemento todos los inspectores abiertos no cambien para mostrar la nueva selección, primero debe bloquear la ventana del inspector. Para ello, debes abrir un inspector, seleccionar un elemento del IDE para inspeccionar, y luego hacer clic en el botón de bloqueo del inspector para bloquearlo en el elemento (se muestra en la imagen de abajo). Una vez que haya bloqueado el inspector, puede abrir otro y repetir el proceso.

Tenga en cuenta que una ventana de inspección bloqueada no bloquea la edición de ningún valor del elemento que se está inspeccionando. Simplemente bloquea el inspector a un elemento y puedes continuar editándolo en cualquier momento, incluso si no es el foco del IDE principal.
