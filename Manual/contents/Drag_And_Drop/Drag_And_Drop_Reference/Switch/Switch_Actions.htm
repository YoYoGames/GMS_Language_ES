Biblioteca de acciones deconmutación

La biblioteca de acciones de conmutación contiene las acciones necesarias para crear sentencias switch en el código de acción de su juego. En algunas situaciones querrás que tus instancias completen una acción dependiendo de un valor en particular, y aunque puedes hacer esto usando un número de sentencias "if" consecutivas, esto puede ser complicado cuando las opciones posibles son más de dos o tres. En estos casos es mejor utilizar la acción "switch". Una acción switch funciona como sigue:

  • Se suministra el valor a (normalmente de algún tipo de expresión o variable)
  • El valor se compara con el valor asignado a las distintas declaraciones "case"
  • Si el valor es el mismo que el del caso, se realiza el contenido del caso, de lo contrario se evalúa el siguiente caso, etc.
  • Una vez que se ha realizado un caso o se han evaluado y fallado todos los casos, el código continuará desde el final del interruptor
  • Si todos los casos fallan y tiene un caso por defecto, entonces se realizará antes de que el código continúe desde el final del interruptor
