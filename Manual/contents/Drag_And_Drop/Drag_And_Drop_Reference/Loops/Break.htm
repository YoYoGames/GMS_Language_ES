Pausa

Esta acción sólo puede utilizarse dentro de una acción de bucle (por ejemplo, Repetir o Mientras), y al llamarla se saldrá del bucle en ese momento. Normalmente un bucle se ejecuta un número fijo de veces o hasta que una condición específica, pero puedes usar la acción Break para terminar prematuramente un bucle en cualquier momento (las acciones dentro del bucle después del Break no se ejecutarán).
