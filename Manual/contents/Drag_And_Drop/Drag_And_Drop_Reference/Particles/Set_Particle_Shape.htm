Establecer la forma de las partículas

Esta acción puede utilizarse para establecer la forma de sprite incorporada que una partícula debe utilizar al dibujarse. Se suministra el valor de ID único para el tipo a establecer la forma (como se devuelve cuando se crea el tipo con la acción Crear Tipo de Partícula) y la forma a utilizar.
