Set Particle Sprite

Esta acción puede utilizarse para establecer el sprite que una partícula puede utilizar al dibujar. Se suministra el valor de ID único para el tipo que se va a establecer el sprite (como se devuelve cuando se crea el tipo con la acción Crear tipo de partícula) y el recurso de sprite a utilizar. Si el sprite tiene sub-imágenes, entonces la partícula se animará a través de los cuadros a la velocidad establecida en el Editor de Imágenes para el sprite. También hay que tener en cuenta que si se establece un valor de color para la partícula (utilizando la acción Establecer color de la partícula ) entonces este color se mezclará con el sprite utilizado, al igual que si se establece la mezcla de imágenes en una instancia se mezclará un color con el sprite de la instancia (utilice el blanco para no mezclar).

NOTA: Si usas esta acción no debes usar la acción Establecer Forma de Partícula ya que sólo puedes tener un sprite o una forma, pero no ambos.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Tipo El valor de la ID del tipo para establecer el sprite de
Sprite El sprite a asignar al tipo de partícula
