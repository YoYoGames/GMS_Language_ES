Si la instancia existe

Esta acción puede ser utilizada para comprobar si una instancia de un objeto específico existe en la sala de juego, o si una instancia específica en sí misma existe. Usted da el objeto desde el explorador de activos, y si cualquier instancia existe en la sala entonces la acción devolverá true de lo contrario devolverá false. Tenga en cuenta que puede utilizar una variable aquí con el ID único para la instancia específica (por ejemplo, como devuelto por la acción Crear Instancia de Objeto) o incluso el ID de la instancia devuelto por el editor de la sala al añadir la instancia.

Si se marca el argumento "No", la acción comprobará si no existe ninguna instancia y si no se encuentra ninguna devolverá true y si existe alguna devolverá false.
