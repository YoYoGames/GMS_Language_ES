Obtenga la alarma

Este bloque de acción recupera el valor de la cuenta atrás de una alarma y lo almacena en la variable de destino dada. Si marca la variable de destino como "Temp", se creará una nueva variable temporal (local) para mantener el valor de retorno, como se muestra en el ejemplo siguiente en el ejemplo siguiente.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Alarma La alarma que hay que conseguir.
Objetivo La variable de destino que se utilizará para almacenar el valor de la alarma.

Ejemplo:

El código del bloque de acción anterior crea una variable local (temporal) y almacena el valor actual de la alarma[0]. Luego se comprueba y si es menor que 0, la alarma se pone a 60.
