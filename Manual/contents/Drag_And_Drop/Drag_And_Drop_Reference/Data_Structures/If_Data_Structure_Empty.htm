Si la estructura de datos está vacía

Con esta acción puedes comprobar si la estructura de datos dada está vacía o no. Primero hay que elegir el tipo de estructura de datos que se va a comprobar (ya sea una Lista, un Mapa o una Pila - no hay cuadrículas porque ni siquiera pueden estar vacías) y luego se proporciona la variable que contiene el índice de la estructura de datos (tal y como lo devuelven las acciones Crear lista, Crear mapa y Crear pila). La acción devolverá true si la estructura de datos está vacía, y false si no lo está. Ten en cuenta que puedes marcar "no" en la acción para comprobar si una estructura de datos contiene datos, comprobando si no está vacía.
