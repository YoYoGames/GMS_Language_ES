Obtener el elemento de la lista en

Con esta acción puede recuperar la posición del índice de un elemento dentro de la lista dada. Se suministra la variable que almacena el índice de la lista (devuelto por la acción Crear lista) y la posición del índice dentro de la lista para obtener el valor del elemento, así como una variable de destino para almacenar el valor del elemento devuelto (que puede marcarse como una variable local temporal que se utilizará hasta el final del script o del evento).

NOTA: Si da una posición que está fuera del tamaño de la lista dada (por ejemplo, la posición 11 en una lista de 10 valores) entonces la acción devolverá undefined. Puede comprobarlo utilizando la acción If Undefined.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Lista El índice (almacenado en una variable) de la lista a eliminar de
Índice La posición del índice dentro de la lista para obtener el elemento
Objetivo La variable de destino para almacenar el valor devuelto

Ejemplo:

El código del bloque de acción anterior obtiene el tamaño de la lista referenciada en la variable global y luego utiliza un bucle for para recorrerla y destruir las instancias con los IDs almacenados en la lista. Una vez finalizado el bucle, la lista se libera de la memoria.
