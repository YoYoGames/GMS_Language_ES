Ajustar el volumen de audio

Con esta acción se puede ajustar el volumen de un sonido determinado. Se selecciona el sonido en el explorador de activos y luego se establece el valor del volumen. El volumen puede estar entre 0 (silencio) y 1 (volumen completo) y la escala es lineal, de manera que un valor de 0,5 sería la mitad del volumen. Tenga en cuenta que esta acción afectará a todas las instancias del sonido que se estén reproduciendo actualmente en la sala, y afectará a todas las instancias futuras del sonido reproducido.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sonido El recurso de sonido para ajustar el volumen de
Volumen El nuevo volumen del sonido (de 0 a 1, por defecto 1)

Ejemplo:

El código del bloque de acción anterior comprueba una variable global para ver si es verdadera o falsa. Si es verdadero, se recupera el volumen del sonido dado y se almacena en una variable local temporal. El valor se comprueba para ver si no es igual a 0,5, y si no lo es, entonces el volumen del sonido se establece en 0,5. Si la variable global se evalúa como falsa, entonces el volumen del sonido se fija en 1.
