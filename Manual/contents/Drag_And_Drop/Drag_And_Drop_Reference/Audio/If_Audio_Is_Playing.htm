Si se reproduce el audio

Esta acción se puede utilizar para comprobar si una instancia de cualquier recurso de sonido se está reproduciendo actualmente en su juego. Se da el recurso de sonido a comprobar desde el explorador de activos y la función devolverá true si se está reproduciendo y false si no lo está, aunque si marcas la bandera "no", esto se invierte y la función devolverá true si el sonido no se está reproduciendo y false en caso contrario.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el "if" se evalúa como verdadero, mientras que cualquier acciones lanzadas en otra parte se ejecutarán después del bloque "if".
