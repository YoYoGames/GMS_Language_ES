Stop Audio

Esta acción se puede utilizar para detener la reproducción de un sonido. Se selecciona el sonido en el explorador de activos y todas las instancias de ese sonido dejarán de reproducirse (por lo que si se ha configurado el mismo sonido para que se reproduzca en tres eventos diferentes, todos ellos se detendrán).

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sonido El recurso del sonido para parar

Ejemplo:

El código del bloque de acción anterior comprueba si se pulsa el botón del ratón y, si se detecta, comprueba una variable global variable global para ver si no es verdadera. Si no es verdadera, entonces se reproduce un sonido (y se hace un bucle) y la variable global se establece como verdadera, de lo contrario el sonido se detiene y la variable global se establece como falsa.
