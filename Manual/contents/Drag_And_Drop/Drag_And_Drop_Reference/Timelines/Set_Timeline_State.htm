Establecer el estado de la línea de tiempo

Esta acción puede utilizarse para iniciar, detener, pausar o reanudar una línea de tiempo. La línea de tiempo debe haberse establecido primero para la instancia utilizando la acción Establecer línea de tiempo de la instancia, y luego puede utilizar esta acción en cualquier momento para cambiar su estado de ejecución entre las siguientes opciones

Opción Descripción
Inicie Inicie la línea de tiempo asignada a la instancia (si la línea de tiempo estaba en pausa, esto establecerá la posición de inicio en el momento 0 de nuevo).
Currículum Vitae Reanudar una línea de tiempo en pausa (si la línea de tiempo estaba detenida y no en pausa, esto no hará nada).
En pausa Poner en pausa la línea de tiempo asignada a la instancia (mientras esté en pausa los momentos de una línea de tiempo no avanzarán con los fotogramas del juego).
Detener Detener la línea de tiempo asignada a la instancia.

Sintaxis:

Argumentos:

Argumento Descripción
Estado El estado en el que se encuentra la línea de tiempo (véase la tabla anterior)

El código del bloque de acción anterior comprobará el valor mantenido en una variable global y si es verdadero, la línea de tiempo se pausa, y si es falso, la línea de tiempo se reanuda.
