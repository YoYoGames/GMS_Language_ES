Obtener el tamaño del buffer

Esta acción puede utilizarse para recuperar el tamaño de un búfer, en bytes. Se suministra el valor único del ID del buffer devuelto por la acción Crear Buffer y luego se da una variable de destino para devolver el valor del tamaño. La variable de destino puede ser marcada como una variable local temporal, en cuyo caso se creará para usted y se eliminará de la memoria al final del script de la acción actual o del evento.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Tampón El índice del buffer (almacenado en una variable)
Objetivo La variable de destino para mantener el valor devuelto

Ejemplo:

El código del bloque de acción anterior obtiene el tamaño del búfer y luego utiliza ese valor para establecer la posición de lectura/escritura al final de los datos almacenados actualmente.
