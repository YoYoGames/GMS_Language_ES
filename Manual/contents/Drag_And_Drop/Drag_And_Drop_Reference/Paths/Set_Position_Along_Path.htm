Fijar la posición a lo largo de la ruta

Esta acción establece la posición de una instancia a lo largo de una ruta. El valor está normalizado de 0 a 1, por lo que si se establece, por ejemplo, en 0,5, la instancia se moverá exactamente a la mitad de la ruta. Tenga en cuenta que, para que esta acción tenga efecto, debe haber llamado a la acción Empezar a seguir la ruta para asignar una ruta a la instancia.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Posición La posición (de 0 a 1) a lo largo de la trayectoria para colocar la instancia

Ejemplo:

El código del bloque de acción anterior inicia la instancia a lo largo de un camino y luego genera un valor decimal aleatorio entre 0 y 1. Este valor se almacena en una variable local temporal y se utiliza para establecer la posición de la instancia a lo largo del camino.
