Si se pulsa la tecla

Con esta acción puedes sondear el teclado para ver si alguna tecla está siendo presionada. La acción devolverá true a cada paso que la tecla del teclado esté siendo presionada, o false en caso contrario, aunque si se pulsa el modificador not se puede comprobar si la tecla no está siendo ayudada, es decir: la acción devolverá true mientras no haya ninguna tecla presionada y false si la hay. Si sólo necesita comprobar la pulsación de una sola tecla, utilice la acción Si la tecla está pulsada.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Cualquier acción lanzada en otro lugar se realizará después del bloque "if" del bloque "if".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Clave El nombre de la tecla del teclado a comprobar.
No Negar la comprobación (verdadero se convierte en falso y viceversa)

Ejemplo:

El código del bloque de acción anterior sondea el estado del teclado a cada paso y si una tecla está siendo se comprueba la pulsación inicial de la tecla. Si la comprobación es verdadera en la pulsación inicial de la tecla, se reproduce un sonido, y luego, mientras la tecla se mantiene pulsada, la mezcla del sprite cambia a rojo. Si la tecla no se mantiene pulsada, se realiza una comprobación Si la tecla no se mantiene pulsada, se realiza una comprobación al soltar la tecla para restablecer el color de la mezcla a blanco.
