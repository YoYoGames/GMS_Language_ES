Si el ratón está abajo

Con esta acción puedes sondear el ratón para ver si el botón seleccionado se mantiene pulsado en ese momento. La acción devolverá true a cada paso que el botón se mantenga pulsado, o false en caso contrario, aunque si se pulsa el modificador not se puede comprobar si el botón no se mantiene pulsado, es decir: la acción devolverá true mientras no haya ningún botón pulsado y false si lo hay. Si sólo necesita comprobar si se ha pulsado un solo botón, utilice la acción Si el ratón está presionado.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el "if" se evalúa como verdadero, mientras que cualquier acción lanzada en otra parte se ejecutará después del bloque "if".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Botón del ratón El nombre del botón del ratón a comprobar (izquierdo, medio, derecho).
No Negar la comprobación (verdadero se convierte en falso y viceversa)

Ejemplo:

El código del bloque de acción anterior sondea el estado del botón del ratón a cada paso y si está siendo se comprueba la pulsación inicial del ratón. Si la comprobación es verdadera en la pulsación inicial, la velocidad de la instancia se establece, entonces, mientras el botón está abajo, la mezcla del sprite se cambia a rojo. Si el ratón no se mantiene pulsado, se realiza una comprobación Si el ratón no se mantiene pulsado, se realiza una comprobación al soltar el ratón para restablecer el color de la mezcla a blanco y establecer la velocidad a 0.
