Declarar una nueva función

Esta acción se puede utilizar para crear una nueva función definida por el usuario, que puede ser llamada en cualquier momento posterior en su código. Una función definida por el usuario es aquella que se define para realizar una tarea que no está cubierta por las acciones disponibles de DnD™ o por las funciones del corredor GML, y puede tener un alcance global, de instancia o local (ver aquí para más información sobre el alcance) y también pueden ser utilizadas como constructores para crear structs. Para obtener más información sobre el uso de esta acción para crear funciones globales, consulte la sección sobre la creación de funciones de acción. También puede marcar la función como estática, lo que significa que sólo se definirá una vez la primera vez que se utilice.

Para utilizar esta acción, puedes añadirla a un evento de instancia y luego dar un nombre a la función que quieres declarar. A continuación, proporciona los nombres de los argumentos necesarios para cada uno de los parámetros que tomará la función. Si la función se va a utilizar como constructor (es decir, va a crear una estructura), entonces debe marcar la bandera Constructor y luego para llamar a la función debe utilizar la acción Nuevo. Si la función sólo se va a utilizar localmente en el mismo evento y en ningún otro lugar, entonces también puede seleccionar la bandera Temp . Una vez que hayas definido el nombre y los parámetros puedes añadir las acciones que quieres que la función ejecute cuando sea llamada en tu código soltándolas al lado de la acción, como se muestra en la imagen siguiente:
