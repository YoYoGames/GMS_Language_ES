Asignar variable

Esta acción se utiliza para asignar un valor a una variable (creándola en el proceso) o para cambiar el valor de una variable existente. Se da el nombre de la variable y luego el valor, que puede añadirse en relación con el valor original si Si es necesario, se puede añadir un valor relativo al valor original. Si das un nombre de variable nuevo (es decir, una variable que nunca ha sido asignada previamente en la instancia) entonces se creará la nueva variable y se le asignará el valor que des, mientras que si das el nombre de una variable existente, se simplemente cambiará el valor por el nuevo. Tenga en cuenta que al marcar Es Relativo se sumará el valor al valor de la variable existente (utilice un número negativo para restar) y también puede utilizar expresiones s y variables creadas previamente como valor.

Cabe destacar que cuando hablamos de "valores", no sólo nos referimos a valores numéricos, ya que una variable puede ser una cadena, un puntero, un ID de recurso o cualquier otra cosa que una función pueda devolver o utilizar. En el caso de una cadena, al marcar Relative se concatenará el valor con lo que la variable contenía previamente, así que si su variable era "Hola" y le asigna " Mundo" utilizando la comprobación relativa, el valor final de la variable será "Hola Mundo". Hay que tener cuidado sin embargo, hay que asegurarse de que los valores de las variables son del mismo tipo de datos, ya que intentar sumar, por ejemplo, un entero y una cadena, dará lugar a un error del compilador.

Tenga en cuenta que puede añadir variables adicionales en la misma acción haciendo clic en el icono más al lado de la acción, y dando otro nombre de variable y valor. Para obtener información más avanzada sobre las variables, consulte la sección Variables y ámbito de las variables.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Nombre El nombre de la nueva variable a asignar, o el nombre de una variable existente a modificar.
Valor El valor que debe tener la variable.
Relativo Comprueba que el valor es relativo al anterior.
