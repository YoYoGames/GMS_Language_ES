Número a cadena

Esta acción convertirá un número real en una cadena. Si el número no es un valor entero y tiene decimales, éstos se redondearán a los dos lugares más cercanos, así que, por ejemplo, el número 123,45678 devolvería la cadena "123,46".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Valor El número real a convertir
Objetivo La variable de destino para mantener el valor devuelto

Ejemplo:

El código del bloque de acción anterior toma un valor decimal y lo convierte en un entero, almacenando el valor devuelto en una variable local temporal. Este valor se convierte entonces en una cadena y se dibuja en la pantalla.
