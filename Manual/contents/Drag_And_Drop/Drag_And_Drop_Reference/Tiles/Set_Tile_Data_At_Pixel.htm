Establecer los datos del azulejo en el píxel

Con esta acción puede establecer los datos de baldosa para una celda de baldosa que se encuentra en una posición específica en una capa de mapa de baldosas. Los datos del mosaico son simplemente un valor que refleja el índice del mosaico junto con los valores de valores de volteo, reflejo y rotación, así como cualquier máscara de mosaico personalizada que haya utilizado. Con esta acción, primero se suministra la capa a la que se apunta utilizando el nombre de la capa (una cadena) tal y como se define en el editor de la sala, y luego los ejes x e y en la sala para establecer los datos del mosaico. Tenga en cuenta que los datos de la baldosa se establecerán para la "celda" del mapa de baldosas en la que se encuentra la posición dada. Por ejemplo, si su conjunto de azulejos es de 16x16, entonces la capa del mapa de azulejos tendrá azulejos en celdas de 16x16 píxeles, así que si usas esta acción para apuntar a la posición de la habitación (8, 8), estarás cambiando la baldosa en la celda (0, 0) del mapa de baldosas.

NOTA: La capa seleccionada debe haber sido definida como una capa de mapa de baldosas en el editor de habitaciones, de lo contrario puede obtener errores.

El argumento final de la acción son los datos del mosaico en sí. Normalmente se recuperan los datos del mosaico para la celda del mapa del mosaico utilizando la acción Obtener datos del mosaico en el píxel y luego se manipulan utilizando la acción Establecer transformación de datos del m osaico antes de establecerlos de nuevo con esta acción. Para obtener más información sobre los datos de mosaico, consulte la sección GML sobre las funciones del mapa de mosaico.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Capa La capa con el mapa de azulejos a la que apuntar
X La posición a lo largo del eje horizontal (X) para establecer los datos del mosaico para
Y La posición a lo largo del eje vertical (Y) para establecer los datos del mosaico
Datos Los datos de la baldosa que se utilizarán para fijar la celda en la posición

Ejemplo:

El código del bloque de acción anterior comprueba si hay un clic del ratón y cuando se recibe uno recupera los datos de la celda que coincide con la posición del ratón. Estos datos se modifican y se devuelven a la capa del mapa de azulejos, girando el azulejo en esa celda.
