Obtener el índice de azulejos en el píxel

Esta acción se utiliza para obtener el índice de una baldosa en una capa determinada. La capa se especifica a partir del nombre de la capa dado (una cadena, como la que se utiliza para nombrar la capa en el Editor de Salas), y luego se da una posición x e y para encontrar la baldosa de la que obtener el índice. La baldosa será el que la posición dada cae dentro de la habitación, y el índice será devuelto a la variable de destino, donde 0 es ningún azulejo, y un número entero positivo es el índice del azulejo. Las posiciones x e y pueden ser marcadas como relativas a la posición de la instancia que llama, y la variable de destino puede ser marcada como una variable temporal (local) que puede ser utilizada hasta el final del evento.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Capa La capa en la que se va a colocar el azulejo
x La posición x a comprobar
y La posición y a comprobar
Objetivo La variable de destino para almacenar el índice de la baldosa

Ejemplo:

El código del bloque de acción anterior comprueba si el ratón pulsa el botón izquierdo y si se detecta uno, obtiene el índice de la celda de la baldosa en la que se encuentra la posición del ratón. Este índice de baldosa se comprueba entonces y si no es 0, se pone a 0.
