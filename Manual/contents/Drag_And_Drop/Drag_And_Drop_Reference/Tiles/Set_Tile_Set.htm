Juego de baldosas

Con esta acción puede configurar una capa para que utilice conjuntos de baldosas diferentes a los creados en el Navegador de Activos. Se da el nombre de la capa (una cadena, como se define en el Editor de Salas), y luego el recurso de conjunto de azulejos a utilizar, y todos los azulejos de la sala en esa capa se dibujarán con el nuevo conjunto de azulejos.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Capa La capa en la que se va a colocar el nuevo conjunto de baldosas.
Tileset El conjunto de azulejos a utilizar para la capa.

Ejemplo:

El código del bloque de acción anterior comprueba si los azulejos de la capa "Floor_Tiles" están usando el conjunto de baldosas "tl_PalaceRuins", y si no lo hacen, entonces se establecen para usarlo.
