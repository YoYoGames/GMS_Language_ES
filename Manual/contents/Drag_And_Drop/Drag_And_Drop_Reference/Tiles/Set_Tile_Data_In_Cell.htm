Establecer los datos de la baldosa en la celda

Con esta acción se pueden establecer los datos del mosaico para una celda de mosaico en una capa de mapa de mosaico. Los datos del mosaico son simplemente un valor que refleja el índice del mosaico junto con los valores de voltear, reflejar y rotar, así como las máscaras de mosaico personalizadas que haya utilizado. Con esta acción, primero se suministra la capa a la que se apunta utilizando el nombre de la capa (una cadena) tal y como se define en el editor de habitaciones, y luego la columna y la fila de la capa del mapa de azulejos para establecer los datos de azulejos para. La "celda" es el área de la rejilla del mapa de azulejos que contiene el azulejo para el que quieres establecer los datos, así que si tus azulejos son de 16x16, por ejemplo, y la habitación es de 1024x768 el mapa de azulejos tendrá 64 columnas y 48 filas.

NOTA: La capa seleccionada debe haber sido definida como una capa de mapa de baldosas en el editor de habitaciones, de lo contrario puede obtener errores.

El argumento final de la acción es el propio dato de la baldosa. Normalmente se recuperan los datos de mosaico para la celda del mapa de mosaico utilizando la acción Obtener datos de mosaico en la celda y luego se manipulan utilizando la acción Establecer transformación de datos de mosaico antes de establecerlos de nuevo con esta acción. Para más información sobre los datos de mosaico, consulte la sección GML sobre las funciones del mapa de mosaico.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Capa La capa con el mapa de azulejos a la que apuntar
Columnas La posición de la columna de la celda a lo largo del eje horizontal para establecer los datos del mosaico para
Fila La posición de la fila de celdas a lo largo del eje vertical para establecer los datos del mosaico
Datos Los datos del azulejo que se utilizarán para establecer la celda

Ejemplo:

El código del bloque de acción anterior recorre cada celda de la habitación, recupera los datos de las baldosas de la celda, los refleja, y luego establece la celda de nuevo.
