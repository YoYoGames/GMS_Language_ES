Obtener la longitud de la secuencia

Con esta acción puede recuperar la longitud (en fotogramas) del elemento de secuencia dado en la sala. Se suministra el ID del elemento de la secuencia (devuelto por la acción Crear secuencia), y esta acción devolverá su longitud en la variable de destino. Este será el número de fotogramas que dura la secuencia.

Tenga en cuenta que la variable de destino se puede marcar como temporal (local), que se creará para la acción y se puede utilizar en todas las acciones posteriores para el evento en el que se encuentra la acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento de la secuencia para obtener la longitud.
Objetivo La variable de destino para mantener la longitud.

Ejemplo:

El bloque de acción anterior recupera la longitud del ID del elemento de secuencia almacenado en la variable "seq_id". Ese valor se utiliza entonces para fijar la posición de la cabeza del mismo elemento de la secuencia a 10 fotogramas antes de su final.
