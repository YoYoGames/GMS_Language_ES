Fijar la posición de la secuencia

Con esta acción puede establecer la posición del elemento de secuencia dado en la sala. Se suministra el ID del elemento de la secuencia (tal y como lo devuelve la acción Crear Secuencia), y luego la nueva posición X e Y. Si usted Si marca la casilla "relativa", los valores dados para X e Y se añadirán a la posición actual de la secuencia, de lo contrario serán una posición absoluta.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento de la secuencia para establecer la posición.
X La nueva posición x en la sala para la secuencia.
Y La nueva posición y en la sala para la secuencia.
