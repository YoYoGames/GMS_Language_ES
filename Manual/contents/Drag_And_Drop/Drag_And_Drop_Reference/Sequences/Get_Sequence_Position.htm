Obtener la posición de la secuencia

Con esta acción puede recuperar la posición actual del elemento de secuencia dado en la sala. Se suministra el ID del elemento de la secuencia (devuelto por la acción Crear secuencia), y esta acción devolverá su posición X e Y en variables de destino. Tenga en cuenta que las variables de destino pueden ser marcadas como variables temporales (locales), que serán creadas para la acción y pueden ser utilizadas en todas las acciones posteriores para el evento en el que se encuentra la acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento de la secuencia para obtener la posición.
Objetivo X La variable de destino para mantener la posición x.
Objetivo Y La variable de destino para mantener la posición y.

El código del bloque de acción anterior recupera la posición actual del elemento de la secuencia ID almacenada en la variable "my_seq". Estos valores se utilizan entonces para fijar la posición de la secuencia, moviéndola 10 píxeles a la derecha.
