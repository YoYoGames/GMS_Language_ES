Si la secuencia existe

Esta acción se puede utilizar para comprobar si el elemento de secuencia dado existe en la sala de juego. Se da el ID del elemento, tal y como lo devuelve la acción Crear Secuencia, y si el elemento de la secuencia existe en la sala entonces la acción devolverá true, de lo contrario devolverá false.

Si se marca el argumento "No", la acción comprobará si no existe tal elemento de la secuencia y si no se encuentra ninguno devolverá true.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el bloque "if" se evalúa como verdadero, mientras que cualquier acción lanzada en otro lugar se ejecutará después del bloque "if".
