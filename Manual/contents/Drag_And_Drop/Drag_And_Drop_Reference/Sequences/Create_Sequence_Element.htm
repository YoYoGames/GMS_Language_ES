Crear elemento de secuencia

Con esta acción puede crear un elemento de cualquier secuencia desde el Navegador de Activos. Se da el nombre de la secuencia y una posición x/y dentro de la sala donde se va a crear, así como la capa en la que se va a crear (esta capa debe ser una capa de activos). Puedes marcar las banderas "relativas" para posicionar la secuencia en relación con la instancia que ejecuta los bloques de código de acción. Cada vez que se crea una secuencia en el juego de esta manera, se le da un valor de identificación de elemento único con el que podrá identificar la secuencia en acciones posteriores, por lo que esta acción le permite definir una variable "objetivo" para mantener este valor de ID (como se muestra en el ejemplo siguiente). Tenga en cuenta que el objetivo puede ser marcado como una variable temporal (local) que se creará para la acción y podrá utilizarse en todas las acciones posteriores para el evento en el que se encuentra la acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Secuencia El nombre de la secuencia para crear un elemento.
x La posición x en la sala para crear la secuencia (puede ser relativa a la instancia que llama).
y La posición y en la sala para crear la secuencia (puede ser relativa a la instancia que llama).
capa El nombre (una cadena) de la capa de activos para crear la secuencia.
Objetivo La variable a la que se dirige el valor de retorno.

Ejemplo:

El código del bloque de acción anterior crea una nueva secuencia y almacena su valor único de en una variable. Esta variable se utiliza entonces para pausar la secuencia utilizando la acción Pausar Secuencia.
