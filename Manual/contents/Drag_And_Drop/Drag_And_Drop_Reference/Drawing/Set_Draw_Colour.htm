Set Draw Colour

Con esta acción puedes establecer el color a utilizar para cualquier forma (rectángulo/elipse/línea) o para cualquier texto que se dibuje posteriormente. Tenga en cuenta que esto no tiene que ser llamado antes de cada acción de dibujo, y se puede establecer una sola vez al comienzo del juego. Por ejemplo, si todo su texto va a ser amarillo, puede llamar a esta acción en el Evento Crear de la primera sala del juego, y todo el texto (y las formas) se dibujarán en amarillo a partir de entonces. Sin embargo, si necesita utilizar varios colores diferentes, entonces tendrá que llamar a esta acción antes de cada texto o forma que requiera un color diferente.

La acción tiene una casilla de verificación en la parte inferior para Usar alfa. Esto permite que el componente alfa del color sea utilizado para establecer el alfa de dibujo también (muy parecido a lo que hace Set Draw Alpha. Si no quiere incluir el componente alfa y sólo tener un color sólido, esto puede ser desactivado (está activado por defecto).
