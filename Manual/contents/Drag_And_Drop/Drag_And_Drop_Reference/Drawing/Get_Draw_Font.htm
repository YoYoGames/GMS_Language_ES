Get Draw Font

Con esta acción puede obtener el valor del ID de la fuente actualmente asignada para el dibujo. El valor devuelto será o bien -1 para ninguna fuente asignada, o bien un valor >= 0 que representa una constante de fuente del Navegador de Activos. El valor devuelto se almacenará en la variable de destino que especifique, que puede haber sido creada previamente o puede ser una nueva temporal (si marca la casilla "Temp").
