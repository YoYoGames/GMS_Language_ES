Obtener la habitación actual

Esta acción devolverá el valor del índice único de la sala actual. Es necesario suministrar una variable de destino para mantener el valor devuelto, y esta variable puede ser marcada como una variable local temporal.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Objetivo La variable de destino para almacenar el índice de habitación devuelto

Ejemplo:

El código del bloque de acción anterior obtiene el valor del índice de la habitación actual y luego utiliza una Cambiar para comprobar la habitación y establecer una variable global en función del resultado.
