Sala de reinicio

Esta acción reiniciará la sala actual. Ten en cuenta que la sala no se reiniciará hasta el final del evento o script donde se llamó a la acción, por lo que cualquier código posterior a esta acción seguirá ejecutándose.

Ejemplo:

El código del bloque de acción anterior comprueba si se ha pulsado una tecla y, si se detecta una, se reinicia la sala.
