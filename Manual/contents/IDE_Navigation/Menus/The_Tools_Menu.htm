El menú de herramientas

Este menú te permite abrir las distintas herramientas independientes que están disponibles en GameMaker Studio 2, con las siguientes opciones:

  • Grupos de texturas - Ver la ventana de grupos de texturas.
  • Grupos de audio - Ver la ventana de grupos de audio.
  • Mezclador de sonido - Ver la ventana del mezclador de sonido.
  • Generar imágenes del proyecto - Abre la ventana Generar imágenes del proyecto.
  • Crear paquete local: abre la ventana del empaquetador de activos, para que pueda crear y guardar paquetes de activos locales para su uso futuro.
  • Importar paquete local - Abre una ventana del explorador de archivos donde puede seleccionar un paquete de activos locales e importarlo, añadiéndolo al proyecto actual o creando un nuevo proyecto para él.
