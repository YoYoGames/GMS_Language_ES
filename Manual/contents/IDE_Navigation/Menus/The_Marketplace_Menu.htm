El menú del mercado

Además de crear tus propios recursos y código para tus proyectos, también tienes acceso al Mercado de GameMaker Studio 2, donde puedes descargar recursos gratuitos y de pago para añadirlos a tus proyectos. Puedes encontrar más información sobre el Mercado en la sección GameMaker Marketplacey las opciones de este menú se indican a continuación:

  • Ir al Mercado - Abre la ventana del Mercado donde puedes buscar activos.
  • Mi biblioteca - Vea su biblioteca actual de contenidos de Marketplace.
  • Mi historial de compras - Esta opción abrirá un navegador y le llevará a la página de la cuenta del usuario conectado para que pueda ver el historial de compras de esa cuenta.
  • Actualizar Paquete Existente - Esta opción abrirá el Asistente de Actualización de Paquete para que pueda actualizar un paquete existente de Marketplace que haya creado previamente.
  • Crear Paquete - Con esta opción puede abrir el Asistente para Crear Paquete para crear un nuevo paquete de activos o tutorial para el Mercado.
