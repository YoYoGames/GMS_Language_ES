El menú de construcción

El menú de construcción es donde puede elegir construir su proyecto, ya sea para pruebas, depuración o como un ejecutable final. Las opciones disponibles son:

  • Ejecutar - Ejecute su proyecto actual en la plataforma de destino elegida para la prueba. Para conocer las diferentes opciones de compilación disponibles, consulte la sección sobre compilación. Tenga en cuenta que este comando es el mismo que pulsar el botón Ejecutar en el IDE.
  • Depuración - Ejecute su proyecto actual para probarlo con el depurador para que pueda ver información detallada sobre el rendimiento y depurar cualquier error o problema que pueda tener. Para más información sobre la depuración, consulte la sección Depuración. Tenga en cuenta que este comando es el mismo que pulsar el botón de depuración en el IDE.
  • Limpiar - GameMaker Studio 2 mantiene una caché de archivos para ayudar a acelerar el tiempo de compilación al hacer pruebas. Sin embargo, a veces un caché "rancio" puede conducir a errores extraños e inesperados en su juego, en cuyo caso lo primero que debe hacer es limpiar la caché desde aquí y volver a probar. También es una buena idea limpiar la caché antes de crear un ejecutable final para la plataforma de destino. Tenga en cuenta que este comando es lo mismo que pulsar el botón Limpiar caché en el IDE.
  • Crear ejecutable - Esto compilará el proyecto actual y creará un paquete ejecutable (o paquetes) para la plataforma de destino. Primero abrirá el explorador de archivos para que puedas seleccionar dónde quieres guardar el proyecto y luego compilar el juego. Consulta la sección sobre compilación para obtener más información sobre la configuración de los objetivos de compilación. Tenga en cuenta que este comando es el mismo que pulsar el botón Crear ejecutable en el IDE.
  • Mostrar instalador de Remote Worker - Esto abrirá una ventana del explorador de archivos en la ubicación del instalador de la herramienta Remote Worker (ver aquí para más información sobre Remote Workers).
