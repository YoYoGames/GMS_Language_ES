El menú Archivo

El menú de archivo es el lugar donde puede elegir cambiar, guardar o importar proyectos, así como establecer las preferencias del IDE de GameMaker Studio 2. Las opciones disponibles son:

  • Nuevo proyecto : sirve para crear un nuevo proyecto desde cero en GameMaker Studio 2. Se abrirá el diálogo "Guardar" y te pedirá que elijas una ubicación para guardar el nuevo proyecto. Una vez que haya dado un nombre al proyecto y una ubicación para guardarlo, se le llevará a un nuevo espacio de trabajo en blanco para empezar a trabajar en él. Los archivos del proyecto se guardan con la extensión *.yyp.
  • Abrir Proyecto - Esto abrirá un explorador de archivos para que usted seleccione un archivo de proyecto creado previamente en *.yyz para cargarlo. Tenga en cuenta que en la importación, si el proyecto fue creado con una versión inferior a la 2.3.0, recibirá el siguiente mensaje de advertencia: . Si decide continuar, se abrirá un explorador de archivos y podrá dar al proyecto un nombre actualizado y una ubicación para guardarlo, de modo que el proyecto original no se sobreescriba. Una vez que el proyecto se haya convertido al nuevo formato 2.3.0+, se abrirá en el IDE y recibirá un mensaje diciendo que la conversión se ha realizado con éxito. Deberá proceder a comprobar el informe de compatibilidad que ha generado el proceso de conversión y ver si hay algo que deba corregir antes de intentar ejecutar o compilar el proyecto. Tenga en cuenta que si obtiene errores repetidos al intentar abrir un proyecto realizado con una versión anterior a la 2.3.0, debes ponerte en contacto con YoYo Games (utilizando la opción "Informar de un error" del Menú de Ayuda) , proporcionando el archivo original *.yyp que estás intentando convertir.
  • Importar Proyecto - Aquí puede importar proyectos que han sido previamente exportados como un archivo *.yyz (ver "Exportar Proyecto", abajo), o - si tiene un proyecto hecho con el GameMaker: Studio 1.4 de YoYo Games, puede seleccionar esta opción para importar el archivo *.gmx o *.gmz de esa línea de productos. Al igual que al abrir un proyecto, si el *.yyz fue creado con una versión de GameMaker Studio 2 anterior a la actualización 2.3.0, será necesario convertirlo, ya que el formato de los archivos del proyecto cambió con dicha actualización.

NOTA: Para importar un proyecto anterior de GameMaker a GameMaker Studio 2 debe haber sido creado con la versión 1.4, e incluso entonces puede que no funcione directamente al importarlo. Hay muchas cosas diferentes entre GameMaker Studio 2 y GameMaker: Studio 1.4, por lo que sus proyectos más antiguos pueden necesitar algunos ajustes para que funcionen. Por favor, consulta el artículo Porting A GM:S 1.4 Game To GameMaker Studio 2 en la Base de Conocimiento de YoYo Games Knowledge Base y la sección del manual que cubre las funciones obsoletas y los scripts de compatibilidad.

IMPORTANTE! La importación de un proyecto de *.yyz que contenga recursos de fuentes puede requerir que "regenere" las fuentes eliminándolas/añadiéndolas, o cambiando la fuente en el editor de fuentes y volviéndola a establecer de nuevo. Esto se debe a que la información de las fuentes existentes es retenida por *.yyz cuando se exporta, y, al importarlo posteriormente o en otra máquina, el proyecto puede no tener ya instaladas las fuentes necesarias (o puede tener una fuente diferente con el mismo nombre). Esto sólo debe hacerse si se observan problemas con la representación de la fuente después de la importación.

  • Proyectos recientes - Puede utilizar esta opción para ver los proyectos utilizados recientemente. Al seleccionarla, se reabrirá la página de inicio (en caso de que la haya cerrado) y podrá seleccionar un proyecto utilizado recientemente de la lista que se encuentra allí. proyecto utilizado recientemente de la lista que se encuentra allí.
  • Guardar Proyecto - Esto guardará su juego en la misma ubicación que definió cuando creó el proyecto (o desde donde lo abrió).
  • Guardar proyecto como - Igual que el anterior, sólo que se especifica la ubicación y el nuevo nombre del proyecto.
  • Exportar proyecto - Esto exportará el proyecto como un único archivo *.yyz. Muy útil para enviar copias a la gente o para guardar copias de seguridad.
  • Preferencias - Al seleccionar esta opción, accederás a la ventana de preferencias, donde podrás configurar el aspecto y la sensación de GameMaker Studio 2.
  • Nuevo IDE - Al seleccionar esta opción, se abrirá una nueva instancia del IDE para su uso (se abrirá en la página de inicio).
  • Mi cuenta - Esta opción abrirá tu navegador web predeterminado y te llevará a la página de tu cuenta de YoYo.
  • Cierre de sesión: al seleccionar esta opción, se cerrará la sesión de GameMaker Studio 2. Tendrás que volver a iniciar sesión con la misma licencia (o cualquier otra) para poder utilizar el producto más adelante.

IMPORTANTE! Si ha estado utilizando una licencia de prueba para GameMaker Studio 2 y luego compra una de las otras licencias (escritorio, móvil, UWP, web o consola), es posible que tenga que cerrar la sesión del IDE y luego reiniciar GameMaker Studio 2 y volver a iniciar sesión para asegurarse de que la nueva licencia se aplica correctamente (aunque esto debería ocurrir automáticamente).

  • Salir - Salir de GameMaker Studio 2 al escritorio.
