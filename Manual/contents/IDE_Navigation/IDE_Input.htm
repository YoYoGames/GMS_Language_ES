Entrada y navegación del IDE

El IDE de GameMaker Studio 2 acepta la entrada del ratón y del teclado, y muchas operaciones pueden llevarse a cabo utilizando uno u otro o ambos. En general, puede hacer clic en el botón izquierdo del ratón para seleccionar cualquier cosa, utilizar / + para seleccionar varios elementos, mantener para arrastrar elementos a diferentes muelles o a los espacios de trabajo, y también el para abrir menús específicos del contexto. Tenga en cuenta que si está ejecutando GameMaker Studio 2 en un sistema macOS y está utilizando un ratón de un solo botón, utilice + para hacer clic con el botón derecho del ratón .

También hay un gran número de atajos de teclado que se pueden utilizar para navegar por los diferentes elementos del espacio de trabajo y los editores de recursos, y vamos a repasar rápidamente algunos de los más importantes (puedes encontrar una lista completa aquí ):

  • / + "Z" : Esto deshará la acción anterior y funciona en la mayoría de los editores. También tiene múltiples niveles de deshacer, por lo que puede pulsar esto varias veces para "retroceder" los cambios.
  • / + "Y" : Esto rehará una acción anterior acción que fue deshecha y funciona en la mayoría de los editores. Al igual que deshacer, también tiene múltiples niveles de rehacer.
  • / + "S" : Esto forzará a GameMaker Studio 2 a guardar el proyecto.
  • / + "T" : Esto abrirá la ventana Goto que le permite buscar rápidamente e ir a cualquier activo en su juego, así como buscar en las Preferencias y Opciones del Juego .   • / + : Esto abrirá la ventana de visión general del espacio de trabajo que se puede utilizar para navegar rápidamente entre los elementos que están abiertos en los diferentes espacios de trabajo.

  • : Se puede utilizar para contraer o expandir todas las diferentes ventanas acopladas en el IDE.

Aparte de estos métodos de entrada, también hay una compatibilidad limitada con dispositivos de lápiz, y también un modo especial para aquellas personas que trabajan en proyectos utilizando un ordenador portátil. Ambos se explican a continuación:

Modo portátil

Si utilizas GameMaker Studio 2 en un ordenador portátil, tendrás una opción adicional en la parte superior del IDE para el modo portátil:

Estará activado por defecto, pero puede desactivarse si lo prefiere pulsando este botón botón. El modo portátil se combina con algunas herramientas en el IDE para hacer una experiencia mucho mejor cuando se utiliza un touchpad, simplificando las 3 interacciones principales del ratón: Pan, zoom y desplazamiento. Para ello utiliza dos teclas modificadoras: Izquierda / e Izquierda . Cuando el modo portátil está activado, Izquierda / y Izquierda se "reservan" para las interacciones del ratón, como el desplazamiento del editor de salas, mientras que Izquierda / es el zoom, Izquierda es desplazamiento y paneo. El desplazamiento en el modo portátil se comporta igual que la panorámica, por lo que debería ser intuitivo y lo suficientemente preciso como para no tener que usar la barra de desplazamiento. Tenga en cuenta que al mantener estos botones, simplemente tiene que mover el ratón -no hacer clic ni arrastrar, sólo moverse- y que si hay un atajo general que normalmente requiere el uso de los botones / o , se utilizarán las teclas teclas de la derecha del teclado en su lugar.



Soporte de pantalla táctil

Una última cosa que mencionaremos aquí es que el IDE para GameMaker Studio 2 también tiene un soporte mínimo para pantallas táctiles. En todos los sistemas operativos se puede utilizar la pantalla táctil para hacer clic y arrastrar elementos en el espacio de trabajo principal, y se admiten 2 punteros simultáneos simultáneos, donde un segundo toque realizará un clic con el botón derecho. Ten en cuenta que en Windows 8 y superior, el IDE de GameMaker Studio 2 también admite dispositivos de lápiz.
