Opciones de juegode tvOS Esta sección describe las diferentes opciones disponibles que controlan cómo se compilarán tus proyectos de juegos para tvOS. Las diferentes secciones son: General Aquí puede rellenar la información del producto sobre su juego, incluyendo su nombre de visualización, su ID de paquete y la información de la versión . Tenga en cuenta que el ID del paquete debe estar en el formato correcto de URL inversa para que la compilación final del juego funcione correctamente, por ejemplo: com.[Company].[GameName]. También puede establecer la ubicación de salida de la aplicación en el Mac de construcción, y puede dar el identificador de equipo que desea utilizar para la firma de la aplicación final que GameMaker Studio 2 crea para usted. Si lo estableces aquí, anularás el identificador de equipo que hayas proporcionado en las Preferenciasde tvOS. Aquí también tienes la opción de generar un Podfile y un Podfile.lock. Estos archivos son necesarios si vas a añadir CocoaPods a una extensión de tvOS utilizando el Editor de extensiones. Los archivos contienen macros específicas requeridas por GameMaker Studio 2 para configurar cualquier CocaoPodsde la lista al construir tu juego. Al hacer clic en los botones se crearán los archivos con las macros pregeneradas, y también abrirá un explorador de archivos en la ubicación de los archivos, si desea editarlos manualmente. También hay un botón que te llevará a la siguiente guía: iOS y tvOS: Uso de Cocoa Pods Gráficos Aquí puede cambiar los siguientes detalles relacionados con la forma en que se mostrará su juego, con las siguientes opciones disponibles: Interpolar los colores entre píxeles: Activa la interpolación lineal, que básicamente "suaviza" los píxeles. Para gráficos de píxeles nítidos, debería estar desactivado, pero si tiene bonitas mezclas alfa y gráficos con bordes suavizados es mejor dejarlo activado. Está desactivado por defecto. Escala: Aquí puede seleccionar la escala del lienzo de dibujo manteniendo la relación de aspecto dentro de la pantalla del dispositivo - añadiendo "relleno" alrededor de los bordes para ajustarse a la pantalla - o puede seleccionar que el lienzo de dibujo se estire para ajustarse a la pantalla del dispositivo. A continuación está la opción de establecer el tamaño de la página de la textura. El tamaño por defecto (y más compatible) es 2048x2048, pero puedes elegir entre 256x256 y 8192x8192. En también hay un botón marcado como Vista Previa que generará las páginas de texturas para esta plataforma y luego abrirá una ventana para que puedas ver cómo se ven. Esto puede ser muy útil si desea ver cómo se estructuran las páginas de textura y para evitar tener páginas de textura más grandes (o más pequeñas) de lo necesario. Para obtener más información sobre las páginas de textura, consulte aquí. NOTA: Ten en cuenta que cuanto mayor sea el tamaño de la página de texturas, menos compatible será tu juego con los distintos navegadores y dispositivos. Por último, puede elegir si desea mostrar un cursor o no. Si está marcada, el sistema operativo mostrará un cursor siempre que se mueva el ratón (en la posición del ratón), y el cursor se desvanecerá y desaparecerá cuando no se mueva el ratón durante un periodo de tiempo. Si se desmarca, no se mostrará ningún cursor. Iconos Esta sección le permite añadir los diferentes iconos requeridos por los diferentes dispositivos y las diferentes partes de la App Store. Estos iconos deben ser creados como imágenes de 24bit *.png en el tamaño especificado para cada uno. Vale la pena señalar que GameMaker Studio 2 tiene una herramienta generadora de imágenes del proyecto que se puede utilizar para crear automáticamente todas las imágenes necesarias para todas las diferentes plataformas de destino de su juego para las que se compila el juego. Si utilizas esta herramienta, debes revisar las imágenes creadas para asegurarte de que son las que necesitas, especialmente para esta plataforma ya que los iconos requeridos son rectangulares y no cuadrados. Esta herramienta también generará Imágenes (explicadas más abajo) así como iconos. Imágenes Esta sección te permite añadir gráficos independientes como imágenes de "Top Shelf" y pantallas de bienvenida para cada uno de los diferentes tamaños de destino de tvOS. Estas imágenes deben ser creadas como imágenes individuales de 24bit *.png sin transparencias y en el tamaño correcto. También puedes establecer un tiempo mínimo para mostrar las pantallas de bienvenida desde el menú desplegable en en la parte inferior (de 0 a 10 segundos). Como se mencionó en la sección de Iconos, puede utilizar el Generador de Imágenes del Proyecto para crear estas imágenes. Social Esta sección es para habilitar diferentes características sociales en tu juego de tvOS. Se pueden elegir las siguientes opciones: Habilitar las notificaciones push: Para que su juego utilice las notificaciones push, primero deben estar habilitadas aquí, de lo contrario no funcionarán. Activar el inicio de sesión con Apple: Esta opción es para usar la extensión de inicio de sesión de Apple en iOS. Puedes obtener esta extensión desde la página Juegos YoYo Mercado, y la propia extensión contiene instrucciones completas de uso. Si no utiliza esta extensión, no debe marcar esta opción.
