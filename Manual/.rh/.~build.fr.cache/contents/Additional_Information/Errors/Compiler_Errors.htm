Errores de compilación El segundo tipo de error que puede ser reportado para su juego es el error del compilador. Un error del compilador ocurre cuando su juego encuentra algún tipo de error que el verificador de sintaxis no ha podido detectar - es muy fácil olvidar declarar una variable o escribir por error una cadena como un real, por ejemplo - o cuando un error está relacionado con la forma en que ha configurado las herramientas de compilación de las Preferencias de Plataforma para la plataforma de destino. Estos errores serán detectados por GameMaker Studio 2 mientras se compila el juego y esta información también se mostrará en la salida de errores del compilador, como se muestra a continuación: NOTA: Estos errores no suelen mostrar una ventana y simplemente mostrarán "BuildFailed" en la salida del compilador. Si aparece una ventana, normalmente se trata de un error de ejecución, que se explica aquí. Los mensajes de error del compilador seguirán todos el mismo formato: [objeto] - [evento] - [Número de línea]: [cadena de error] Si el error se encuentra en una secuencia de comandos, será simplemente: [script] - [Número de línea]: [cadena de error] A continuación, puede hacer doble clic en en cualquiera de las entradas de error del compilador para abrir el activo en cuestión en la posición marcada como error, y luego puede utilizar la información contenida en el mensaje para señalar exactamente en qué parte del objeto o del script se ha producido el error.
