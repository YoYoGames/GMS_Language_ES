Opciones del menú del botón derecho del ratón Cuando trabajes en tu código de DnD™, puedes hacer clic con el botón derecho del ratón en la acción, en el espacio de trabajo o en el texto de la lista reducida de la izquierda para abrir el menú del botón derecho del ratón: Este menú tiene las siguientes opciones: Cortar / Copiar / Pegar: Esto le permitirá cortar, copiar o pegar cualquier acción que esté seleccionada en el área de trabajo. Tenga en cuenta que puede utilizar + fuera de las acciones y arrastrar, o utilizar / + en acciones individuales, para seleccionar más de una y aplicar el comando a todas ellas. Comentar / Eliminar comentario: Esta opción le permite añadir y eliminar comentarios de sus bloques de código DnD™. Al seleccionar Comentario, se abrirá un editor de texto a la derecha del bloque DnD™seleccionado, y allí podrá añadir cualquier texto que necesites, ya sea un recordatorio para cambiar algo, o un comentario útil para un tutorial, o incluso instrucciones para tus compañeros de equipo: Una vez que hayas añadido el comentario, puedes hacer clic en él y arrastrarlo para colocarlo vertical y horizontalmente - aunque sólo a la derecha del nodo padre - y al colapsar el nodo DnD™también se colapsará el comentario y se mostrará un resumen de su contenido (en cuyo caso no se podrá reposicionar). Si el comentario ya no es necesario, puede eliminar todo el texto el texto del comentario, o utilizar la opción Eliminar comentario del menú. Tenga en cuenta que cuando copie un nodo, el comentario se copiará junto con él, y si convierte el nodo de DnD™a GML, entonces el comentario se conservará y se añadirá al GML que se cree, como se ilustra en la imagen siguiente: Como se ha mencionado anteriormente, un comentario sólo puede aplicarse a un único nodo, pero puede ampliarse para cubrir múltiples nodos simplemente añadiendo saltos de línea para extenderlo, aunque esto puede requerir que se reposicione el comentario para evitar que se superponga con los nodos siguientes: Una vez que se ha añadido un comentario, puede seleccionar la opción "Comentario" de nuevo en el menú RMB o puede hacer doble clic en para activar la edición. Desactivar / Activar: Estos comandos se pueden utilizar para deshabilitar o habilitar la acción (o acciones) seleccionada. Cuando una acción está deshabilitada, sigue siendo visible en el área de trabajo, pero no se ejecutará al compilar el juego. De esta manera puedes añadir/quitar acciones para probar y depurar. La imagen de abajo muestra algo de código de DnD™con las acciones deshabilitadas: Puedes ver que las acciones "If Score" y "Go To Next Room" han sido deshabilitadas - se han oscurecido, visualmente - y así se saltarán cuando se pruebe o compile el juego. Alternar punto de interrupción: Este comando activará o desactivará un punto de interrupción para la acción seleccionada. Un punto de interrupción es simplemente un punto en el código de DnD™donde el depurador debe detenerse cuando el juego cuando el juego se ejecuta en modo de depuración. Una acción con un punto de interrupción asignado a ella será resaltada, como se muestra en la imagen de abajo (la acción "Si la salud" se ha establecido como un punto de interrupción): Eliminar todos los puntos de interrupción: Con este comando puedes eliminar todos los puntos de interrupción del código DnD™. Colapsar todo / Expandir todo: Las acciones de DnD™pueden ser colapsadas o expandidas, donde una acción colapsada simplemente mostrará el nombre de la acción pero no ninguno de los valores internos maximizando así el espacio disponible para el espacio de trabajo. Por ejemplo: Como puedes ver, las acciones ocupan mucho menos espacio, y todavía puedes ver lo que hacen desde la breve descripción que aparece a la derecha. Puede seleccionar la opción "Expandir todo" para deshacer esto para cada nodo del espacio de trabajo. Restablecer diseño: Esto restablecerá el espacio de trabajo de DnD™, estableciendo el nivel de zoom al 100%, centrándose en la acción actualmente seleccionada, y expandiendo cualquier (y todas) las acciones colapsadas también. Vista previa en vivo: Al seleccionar esta opción, habilitarás o deshabilitarás la ventana de previsualización de código de DnD™a GML(GameMaker Language). Esto se explica con más detalle en la página Cambiar DnD™ a código. Convertir a GML: Esto convertirá todas sus acciones en el evento actual a código GML(Lenguaje GameMaker). Tenga mucho cuidado con esto ya que no puede convertir GML de nuevo en las acciones de DnD™ de las que fue creado. Puedes convertir el evento de vuelta a un evento DnD™desde el código, pero cualquier código será simplemente envuelto en una acción de Ejecutar Código. Esto se explica con más detalle en la página Cambiar DnD™ a código. Ventana: Esta sección abre el siguiente submenú: Los elementos de menú disponibles aquí son: Ir al objeto: Esto desplazará el espacio de trabajo al objeto que está llamando al evento Abrir Evento Heredado: Esta opción sólo está disponible si el objeto que se está editando es hijo de un objeto padre, y abrirá el evento que se está editando del objeto padre y desplazará el espacio de trabajo hacia él. Un Panel / 2 Columnas / 3 Columnas: Estas opciones se pueden utilizar para dividir el espacio de trabajo actual en 1, 2 o 3 "paneles" diferentes. Cada panel es un espacio de trabajo separado que puede tener scripts o eventos arrastrados a ellos para que puedas editar y comparar 2 o más bloques de código DnD™a la vez. La imagen de abajo muestra dos scripts en una ventana mostrados en su propio panel de espacio de trabajo: Las dos opciones siguientes sólo están disponibles cuando el menú RMB se abre en un nodo DnD™: Desplazamiento: Al seleccionar esta opción en cualquier nodo, el espacio de trabajo se desplazará para rodear ese nodo hacia el centro. Mostrar ayuda... : Al seleccionar esta opción se abrirá el Manual en un nuevo espacio de trabajo en la página correspondiente al nodo que se ha seleccionado.
