Cambiar DnD™a código Otra característica de DnD™es que puedes convertirlo en código GML(Lenguaje GameMaker), y de esta manera ver qué es exactamente lo que ocurre entre bastidores cuando utilizas acciones. Sin embargo, antes de cambiar las acciones a GML, puedes primero previsualizarlas utilizando el botón derecho del ratón opción de menú Vista previa en vivo: Que abrirá una nueva ventana: A medida que añada, cambie o elimine acciones de DnD™, la vista previa en vivo de GML se actualizará para mostrarle el código real que se está se está creando "entre bastidores". El código en la vista previa en vivo no puede editarse directamente, pero puede seleccionar secciones y copiarlas para pegarlas en scripts GML o Acciones de Código (por ejemplo). Si decides que quieres convertir el DnD™en código después de ver la vista previa, entonces esto se puede hacer de nuevo haciendo clic en el botón derecho del ratón en cualquier espacio de trabajo de eventos con acciones y seleccionando Convertir a GML. La primera vez que lo haga recibirá un mensaje de advertencia diciendo que se trata de una conversión de un solo sentido, ya que puede convertir acciones en código, pero no puede volver a convertirlas en acciones individuales más adelante. Al hacer clic en "Aceptar" se realizará la conversión por usted. El código resultante utilizará {} para delimitar las acciones individuales, y podrá ver claramente qué acciones se relacionan con qué funciones o declaraciones de variables dentro del código. Si el DnD™es más complejo, entonces el código también lo será, pero se aplican las mismas reglas generales y el código se estructurará secuencialmente exactamente igual a como has escrito el DnD™.
