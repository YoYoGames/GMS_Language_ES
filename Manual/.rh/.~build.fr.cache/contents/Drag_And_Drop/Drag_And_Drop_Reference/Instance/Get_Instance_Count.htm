Obtener el recuento de instancias Con esta acción puede encontrar el número de instancias de cualquier objeto determinado que esté actualmente activo dentro de la sala. Puede seleccionar cualquier objeto de la lista de recursos o puede utilizar la palabra clave especial todo para para obtener el número de todas las instancias activas. También es necesario suministrar una variable de destino para devolver el valor y poder utilizarlo en otras acciones. Si marca la variable de destino como "Temp", se creará una nueva variable temporal (local) para mantener el valor de retorno, como se muestra en el siguiente ejemplo. Sintaxis de la acción: Argumentos: Argumento Descripción Objeto El nombre del objeto a comprobar. Objetivo La variable a la que se dirige el valor de retorno. Ejemplo: El código del bloque de acción anterior crea una variable local (temporal) y luego le asigna el número de instancias del objeto "obj_Ardilla". A continuación se comprueba este valor y se destruye la instancia si el número de instancias es superior a 100.
