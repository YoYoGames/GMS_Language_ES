Rotación de la Instancia del Conjunto Este bloque de acción establece el ángulo_de_imagen (rotación) de la instancia. El ángulo se mide en grados, siendo el derecho 0°, el superior 90°, el izquierdo 180° y el inferior 270°. Establece esta variable a 0 para restablecer la instancia, lo que significa que dibujar cualquier sprite asignado como se definió en el editor de sprites. Tenga en cuenta que para que los cambios en esta acción sean visibles, la instancia no debe tener ningún evento de dibujo (y así GameMaker Studio 2 dibujará por defecto el sprite) o se dibujará utilizando la acción Draw Self. Sintaxis de la acción: Argumentos: Argumento Descripción Ángulo El nuevo ángulo a utilizar (de 0 a 360). Ejemplo: El código del bloque de acción anterior establece un nuevo sprite, así como una serie de otras propiedades para la visualización de ese sprite, incluyendo la rotación de 180°.
