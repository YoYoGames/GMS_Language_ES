Conjunto de instancias alfa Este bloque de acción establece el valor de image_alpha para la instancia. El alfa de la imagen es el valor de "transparencia" para dibujar cualquier sprite asignado a la instancia y tiene un valor entre 0 y 1, donde 0 sería totalmente transparente y 1 sería totalmente opaco. Tenga en cuenta que si el sprite tiene píxeles transparentes o semitransparentes, entonces su transparencia será escalada por la cantidad, por lo que si tiene algunos píxeles 50% transparentes en el sprite y establece el alfa de la instancia a 0,5, el alfa final para los píxeles transparentes será del 25%. Puedes establecer este valor para que sea una cantidad relativa Tenga en cuenta que para que los cambios en esta variable sean visibles, la instancia no debe tener ningún evento de dibujo (y por lo tanto GameMaker Studio 2 dibujará por defecto el sprite) o se dibujará utilizando la acción Draw Self . Sintaxis de la acción: Argumentos: Argumento Descripción alfa El valor alfa a establecer (de 0 a 1, por defecto es 1). Ejemplo: El código del bloque de acción anterior establece un nuevo sprite, así como una serie de otras propiedades para la visualización de ese sprite, incluyendo la configuración del alfa de la imagen a 0,5.
