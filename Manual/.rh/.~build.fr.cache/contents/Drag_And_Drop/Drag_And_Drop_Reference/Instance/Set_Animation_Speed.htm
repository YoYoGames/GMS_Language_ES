Establecer la velocidad de la animación Este bloque de acción establece la image_speed (velocidad de animación) para el sprite asignado a la instancia. Se establecen los fotogramas por segundo ideales en el Editor de Sprite y luego se puede utilizar este valor para modificarlo. La velocidad de animación por defecto se establece en 1, lo que significa que el juego tratará de mantener el número de fotogramas por segundo que hayas definido para el sprite, pero puedes utilizar valores superiores a 1 para animar más rápido (pueden saltarse fotogramas), o valores decimales para animar más lento (los fotogramas se mostrarán en varios pasos) o incluso valores negativos para invertir la velocidad de la animación. Sintaxis de la acción: Argumentos: Argumento Descripción Velocidad El nuevo valor de la velocidad del modificador. Ejemplo: El código del bloque de acción anterior establece un nuevo sprite, así como una serie de otras propiedades para la visualización de ese sprite, incluyendo su animación a la mitad de la velocidad definida.
