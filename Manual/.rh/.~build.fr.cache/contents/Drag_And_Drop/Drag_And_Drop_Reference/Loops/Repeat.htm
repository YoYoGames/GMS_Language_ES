Repetir Esta acción repetirá cualquier acción añadida a ella el número de veces especificado. Se da el número de veces que se deben repetir las acciones y luego se añade la acción al bucle de repetición, de la misma manera que se añadiría una acción a un "if", es decir, soltando es decir, colocándola al lado de la acción en lugar de debajo de ella: Todas las acciones añadidas al lado se incluirán en la repetición y se realizarán antes de que el evento o el script continúe con las acciones añadidas debajo (como se muestra en el ejemplo siguiente). Tenga en cuenta que puede terminar cualquier bucle prematuramente si llama a la acción Break. Sintaxis de la acción: Argumentos: Argumento Descripción Times El número de veces a repetir (un valor entero) Ejemplo: El código del bloque de acción anterior obtiene el número de elementos de una lista y almacena el valor en una variable local temporal. Esta variable se utiliza entonces para establecer el número de veces que se ejecuta un bucle de repetición, y en el bucle de repetición las entradas de la lista se leen en otra variable temporal que luego se utiliza para destruir las instancias a las que hacen referencia.
