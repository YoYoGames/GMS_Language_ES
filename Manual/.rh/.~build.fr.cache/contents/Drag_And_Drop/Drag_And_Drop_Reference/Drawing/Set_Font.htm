Set Font Con esta acción puede establecer el tipo de letra que se utilizará para todas las acciones posteriores de dibujar texto. Al hacer clic en el cuadro de diálogo se abrirá la siguiente ventana en la que podrá seleccionar la fuente que desea utilizar: Tenga en cuenta que esto establecerá el tipo de letra a utilizar para todas las llamadas posteriores a dibujar texto, y por lo que se puede llamar una vez al principio del juego - en cualquier caso, no sólo en el Evento de Dibujo - y todas las acciones de dibujar texto utilizarán la fuente establecida. Sin embargo, si usted requiere múltiples fuentes en su juego, entonces usted tendrá que llamar a esta acción antes de dibujar cualquier texto con la fuente requerida. Sintaxis de la acción: Ejemplo: El código del bloque de acción anterior establece la fuente, el color de dibujo y la alineación para un texto que se dibujado.
