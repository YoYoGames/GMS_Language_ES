Draw Healthbar Con esta acción puede dibujar una barra de color para mostrar una representación gráfica de un valor. La cantidad a mostrar es un valor porcentual y debe estar dentro del rango de 0 - 100, sin embargo puedes utilizar valores mayores con la fórmula (Valor_CURRENTE / Valor_MÁXIMO) * 100, así que, por ejemplo, puedes usar esto para mostrar el poder, la salud, el maná, el tiempo o cualquier otra cosa que se beneficie de la visualización de una barra. Aparte del valor a dibujar puedes establecer el ancla de dirección para dibujar la barra de salud también (de izquierda a derecha o de derecha a izquierda y de arriba a abajo o de abajo a arriba), así como la posición, ya sea dentro de la sala o en relación con la instancia que llama a la acción. Además de la posición, también se puede establecer el color que se mezclará desde cada una de las cuatro esquinas de la barra de salud, creando gradientes de rojo a verde (por ejemplo, pero puede utilizar cualquier color).
