Get Draw Alpha Con esta acción puedes obtener el alfa general que se está utilizando en el dibujo. La acción devolverá el alfa actual a la variable de destino que especifiques, que puede haber sido creada previamente o puede ser una nueva temporal (si marcas la casilla "Temp" ). Sintaxis de la acción: Ejemplo: El bloque de acción anterior obtiene el alfa de dibujo actual y si no está establecido en 1 (totalmente opaco) se ajusta a ese valor.
