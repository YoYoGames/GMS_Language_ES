Puntuación de la instancia de sorteo Con esta acción puedes dibujar el valor de la puntuación de la instancia. Cuando se utiliza cualquier acción de puntuación en GameMaker Studio 2 la instancia que la llama tendrá una nueva variable de ámbito de instancia llamada puntuación añadida. Esta variable se puede comprobar y establecer como cualquier otra variable. En el caso de esta acción, la variable se dibujará en la pantalla en la posición dada, junto con cualquier texto que desee proporcionar. El texto del título se dibujará antes del valor de la puntuación y si Si se trata de un texto, debe ir acompañado de "", y ambos serán formateados utilizando la fuente, el color y la alineación actuales. NOTA: Esta acción es sólo para su uso en los diversos eventos de sorteo, y no dibujará nada si se utiliza en otro lugar. Sintaxis de la acción: Argumentos: Argumento Descripción Título La cadena a dibujar junto con el valor de la puntuación (precede y puede dejarse en blanco) x La posición x para dibujar el marcador en y La posición y para dibujar el marcador en Ejemplo: El código del bloque de acción anterior dibuja la puntuación utilizando el formato de dibujo dado.
