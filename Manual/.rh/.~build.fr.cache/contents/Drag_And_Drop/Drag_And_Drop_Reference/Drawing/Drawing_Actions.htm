Biblioteca de acciones de dibujo La biblioteca de acciones de dibujo es donde se encuentran las acciones necesarias para dibujar sprites, texto o formas, así como para establecer ciertas propiedades de dibujo. La mayoría de estas acciones son sólo para su uso en los diferentes Eventos de Dibujo de un objeto, y pueden no funcionar si se utilizan fuera de del evento de dibujo. Las excepciones a esto son las acciones Set, que pueden ser añadidas a cualquier evento y afectarán a todo el dibujo de todas las instancias después. Es importante tener en cuenta que si usted agrega cualquier acción en el evento principal de dibujo de un objeto, entonces no dibujará el sprite que ha sido asignado a la instancia a menos que explícitamente le diga a GameMaker Studio 2 que lo dibuje, utilizando una acción como Draw Self. Básicamente, GameMaker Studio 2 dibujará por defecto cualquier sprite asignado a una instancia, sólo si no hay nada más en el Evento Draw.
