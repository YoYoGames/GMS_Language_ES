Insertar en la lista Con esta acción puede recuperar la posición del índice de un elemento dentro de la lista dada. Se suministra la variable que almacena el índice de la lista (tal y como la devuelve la acción Crear lista) y el valor del que se quiere encontrar la posición del índice dentro de la lista, así como una variable de destino para almacenar la posición de índice devuelta para el elemento (que puede marcarse como una variable local temporal que se utilizará hasta el final del script o evento). Cuando se inserta un valor en una lista, todos los índices por encima del índice dado para la inserción se moverán uno hacia arriba, así que si tienes una lista con 4 elementos en ella (posición de índice 0 a 3) y colocas un nuevo elemento en la posición de índice 2, el elemento anterior 2 se convertirá en el elemento 3 y el elemento anterior 3 se convertirá en el elemento 4. Tenga en cuenta que si el valor que está buscando no existe dentro de la lista, entonces no habrá ninguna posición de índice de la lista para devolver, y por lo tanto el valor de retorno será -1. Sintaxis de la acción: Argumentos: Argumento Descripción Lista El índice (almacenado en una variable) de la lista a insertar en Índice La posición del índice dentro de la lista para insertar un valor Valor El valor a almacenar en la posición Ejemplo: El código del bloque de acción anterior utiliza un bucle de repetición para obtener el tamaño de la lista y luego generar una posición aleatoria dentro de la lista. A continuación, genera un valor aleatorio y lo inserta en la lista en la posición aleatoria definida anteriormente.
