Obtener el recuento de elementos de la lista Puede utilizar esta acción para saber cuántas entradas hay en una lista. Se suministra la variable que contiene el índice de la lista (devuelto por la acción Crear Lista) y la variable de destino para almacenar el número devuelto de elementos de la lista (que puede marcarse como una variable local temporal que se utilizará hasta el final del script o del evento). Sintaxis de la acción: Argumentos: Argumento Descripción Lista El índice (almacenado en una variable) para obtener el recuento de elementos Objetivo La variable de destino para almacenar el recuento de artículos devueltos Ejemplo: El código del bloque de acción anterior obtiene el tamaño de la lista referenciada en la variable global y luego utiliza un bucle for para recorrerla y destruir las instancias con los IDs almacenados en la lista. Una vez finalizado el bucle, la lista se libera de la memoria.
