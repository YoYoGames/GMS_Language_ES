Añadir a la lista Esta acción puede utilizarse para añadir un nuevo valor (de cualquier tipo de datos) a la lista, y este valor se añadirá al final. Se suministra la variable que almacena el índice de la lista (devuelta por la acción Crear lista) y el valor a almacenar. Sintaxis de la acción: Argumentos: Argumento Descripción Lista El índice (almacenado en una variable) de la lista a añadir Valor El valor a añadir en la lista Ejemplo: El código del bloque de acción anterior crea una variable de ámbito global y luego crea una nueva estructura de datos de la lista, asignando su valor de índice a la variable global. El ámbito se cambia entonces para que todas las instancias del objeto "obj_Enemy_Parent" añadan su valor de ID de instancia único en la lista.
