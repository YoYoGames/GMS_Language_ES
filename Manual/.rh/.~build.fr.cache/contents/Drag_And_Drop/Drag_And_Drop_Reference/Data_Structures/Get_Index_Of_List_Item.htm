Obtener el índice del elemento de la lista Con esta acción puede recuperar la posición del índice de un elemento dentro de la lista dada. Se suministra la variable que almacena el índice de la lista (tal y como la devuelve la acción Crear lista) y el valor del que se quiere encontrar la posición del índice dentro de la lista, así como una variable de destino para almacenar la posición de índice devuelta para el elemento (que puede marcarse como una variable local temporal que se utilizará hasta el final del script o evento). Tenga en cuenta que si el valor que busca no existe no existe dentro de la lista, entonces no habrá una posición de índice de la lista para devolver, y por lo tanto el valor de retorno será -1. Sintaxis de la acción: Argumentos: Argumento Descripción Lista El índice (almacenado en una variable) de la lista a eliminar de Valor El valor para comprobar la lista Objetivo La variable de destino para almacenar el índice devuelto Ejemplo: El código del bloque de acción anterior comprueba si hay una colisión en la posición de la instancia y si se encuentra una, el valor de ID único de la instancia se almacena en una variable temporal y luego se comprueba si existe dentro de la estructura de datos de la lista. Si existe, el elemento se elimina de la lista y la instancia se destruye.
