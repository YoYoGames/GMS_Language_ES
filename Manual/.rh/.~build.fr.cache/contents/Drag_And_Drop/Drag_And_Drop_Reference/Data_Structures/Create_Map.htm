Crear mapa Esta acción creará una nueva estructura de datos de mapa y devolverá el valor del índice para que pueda acceder posteriormente al mapa a través de las otras acciones de estructura de datos. El índice del mapa será devuelto a la Variable de Destino que usted suministre, la cual puede haber sido creada anteriormente utilizando Assign Variable o Declare Temp, o puede marcar la casilla "Temp" para nombrar y crear una variable local temporal para almacenar el valor hasta el final del script o evento. Una estructura de datos de mapa recién creada se considera "vacía", es decir, no contiene entradas de clave/valor de mapa. Tenga en cuenta que puede crear mapas DS adicionales haciendo clic en el icono más junto a la acción, y seleccionando otra variable para mantener el ID del mapa. IMPORTANTE! La creación de cualquier estructura de datos consume memoria en la plataforma de destino, y como tal todas las estructuras de datos deben ser liberadas cuando ya no se necesitan utilizando la acción Liberar Estructura de Datos, de lo contrario se obtiene una de lo contrario se produce una fuga de memoria que puede perjudicar el rendimiento del juego o incluso hacer que se bloquee. Esto es particularmente relevante cuando se utilizan variables locales temporales para almacenar índices de estructuras de datos, ya que estas variables se eliminan al final del código o del evento, pero pero eso no significa que la estructura de datos se elimine también. La estructura de datos seguirá existiendo, sólo que no tendrás forma de referenciarla, así que o bien utilizas una variable de instancia y liberas la estructura más adelante, o liberas la estructura antes de que finalice el evento o la función si su índice está almacenado en una variable temporal.
