Establecer el valor del mapa Con esta acción puedes añadir un par clave/valor en la estructura de datos del mapa. Se suministra la variable que contiene el índice del mapa (devuelta por la acción Crear mapa) y luego se da una "clave" (que es el identificador dentro del mapa para un valor), y luego el "valor" que se asociará a esa clave. Ambas claves pueden ser valores enteros (como 0, 200, 36712, etc...) o cadenas (como "Nombre", "Escudo", etc...), pero el valor puede ser cualquier tipo de datos válido. Sintaxis de la acción: Argumentos: Argumento Descripción Mapa El índice (almacenado en una variable) del mapa a establecer Clave La clave a establecer (entero o cadena) Valor El valor a almacenar con la clave dada Ejemplo: El código del bloque de acción anterior crea una nueva estructura de datos de mapa y luego establece tres entradas de clave/valor en ella. en ella.
