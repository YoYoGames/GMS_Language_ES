Si existe una estructura de datos Con esta acción puedes comprobar una variable para ver si indexa una estructura de datos o no. Primero tienes que elegir el tipo de estructura de datos que quieres comprobar (ya sea una Lista, un Mapa, una Cuadrícula o una Pila) y luego suministrar la variable que contiene la estructura de datos (como la devuelta por las acciones Crear Lista, Crear Mapa, Crear Cuadrícula y Crear Pila). La acción devolverá true si la estructura de datos existe, y false si no existe. Ten en cuenta que puedes marcar "not" en la acción para comprobar si una variable no contiene un índice de estructura de datos. Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente: Estas acciones se ejecutarán ahora si el "if" se evalúa como verdadero, mientras que cualquier acciones lanzadas en otra parte se ejecutarán después del bloque "if".
