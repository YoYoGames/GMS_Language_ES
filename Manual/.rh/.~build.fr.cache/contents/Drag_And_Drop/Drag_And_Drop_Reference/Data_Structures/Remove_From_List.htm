Eliminar de la lista Esta acción puede utilizarse para eliminar un elemento del índice de la lista dada. Se suministra la variable que almacena el índice de la lista (devuelto por la acción Crear lista) y la posición del índice dentro de la lista a eliminar, donde la posición del índice está entre 1 y (longitud de la lista -1). Esta acción no devuelve nada, por lo que si necesita el valor en la posición del índice, debe utilizar Obtener el elemento de la lista antes de eliminar el índice. Sintaxis de la acción: Argumentos: Argumento Descripción Lista El índice (almacenado en una variable) de la lista a eliminar de Índice El índice dentro de la lista a eliminar Ejemplo: El código del bloque de acción anterior comprueba si hay una colisión en la posición de la instancia y si se encuentra una, el valor de ID único para la instancia se almacena en una variable temporal y luego se comprueba si existe dentro de la estructura de datos de la lista. Si existe, el elemento se elimina de la lista y la instancia se destruye.
