Conecta el Gamepad Puedes utilizar esta acción para comprobar si un "slot" de gamepad dado tiene un gamepad conectado. Suministra el índice del gamepad (este es el número de "ranura" que ocupa un gamepad) y el valor devuelto se almacenará en la variable de destino que suministres para su uso futuro (esto puede ser marcado como una variable local temporal que creas para este evento o script). Sintaxis de la acción: Argumentos: Argumento Descripción Gamepad El índice del gamepad para comprobar. Objetivo La variable de destino para almacenar el valor devuelto. Ejemplo: El código del bloque de acción anterior ejecuta un bucle for para comprobar si un gamepad está conectado a alguna de las 12 "ranuras" disponibles. Si se descubre un gamepad, se establece una variable global con el número de ranura actual y se rompe el bucle.
