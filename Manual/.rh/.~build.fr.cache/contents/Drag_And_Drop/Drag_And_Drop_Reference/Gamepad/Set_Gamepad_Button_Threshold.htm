Establecer el umbral de los botones del Gamepad Esta acción se puede utilizar para establecer el "umbral" de los botones analógicos del gamepad. Usted especifica el índice del gamepad para establecer, y luego establecer un valor de 0 a 1 y si la cantidad de entrada del botón analógico es menor que el valor dado, el botón gamepad se considera que está en 0. Tenga en cuenta que este es un ajuste global que afectará a todos los botones analógicos conectados gamepad. Este valor se utilizará en todas las comprobaciones de bajada, pulsación y liberación del gamepad dado, pero será ignorado por la acción Get Gamepad Trigger. Sintaxis de la acción: Argumentos: Argumento Descripción Gamepad El índice del gamepad. Zona muerta El valor del umbral (0 - 1) Ejemplo: El código del bloque de acción anterior establece el umbral del botón para todos los los índices del gamepad a 0,2.
