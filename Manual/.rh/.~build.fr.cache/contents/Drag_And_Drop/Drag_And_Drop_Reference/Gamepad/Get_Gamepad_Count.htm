Obtener el recuento del Gamepad Puedes utilizar esta acción para obtener el número de gamepads conectados al dispositivo que ejecuta tu juego. El número devuelto estará entre 0 y 12. Sintaxis de la acción: Argumentos: Argumento Descripción Objetivo La variable de destino para almacenar el valor devuelto. Ejemplo: El código del bloque de acción anterior obtiene el número de gamepads conectados y asigna el valor a una variable local temporal. A continuación, se comprueba si es más de 0 y se establece una variable global en consecuencia.
