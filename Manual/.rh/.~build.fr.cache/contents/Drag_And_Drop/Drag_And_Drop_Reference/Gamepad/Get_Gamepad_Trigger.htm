Obtenga el gatillo del Gamepad Puedes utilizar esta acción para obtener el valor de los diferentes disparos de un gamepad determinado. Se suministra el índice del gamepad (es el número de "ranura" que ocupa un gamepad) y se selecciona el disparador a comprobar (izquierdo o derecho). El valor devuelto estará entre 0 y 1, donde 0 significa que no hay presión en el gatillo y 1 que hay presión total. Este valor devuelto se almacenará en la variable de destino que proporcione para su uso futuro. Tenga en cuenta que esta función no tiene en cuenta el ajuste del umbral. Sintaxis de la acción: Argumentos: Argumento Descripción Gamepad El índice del gamepad. Disparador El gatillo a comprobar (izquierda/derecha) Objetivo La variable de destino para almacenar el valor devuelto. Ejemplo: El código del bloque de acción anterior crea una variable local (temporal) y la utiliza para almacenar el valor actual del disparador izquierdo. Si el valor del disparador es superior a 0,5, se crea una instancia en la posición de la instancia de llamada.
