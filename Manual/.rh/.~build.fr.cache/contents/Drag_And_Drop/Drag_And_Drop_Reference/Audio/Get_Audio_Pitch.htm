Obtener tono de audio Esta acción recuperará el tono del recurso sonoro que usted proporcione. El valor devuelto será un multiplicador para el tono real grabado de la muestra, donde un valor de 1 es ningún cambio. Para más información sobre el tono, consulte la acción Establecer tono de audio. Sintaxis de la acción: Argumentos: Argumento Descripción Sonido El recurso sonoro para obtener el tono de Objetivo La variable de destino para almacenar el tono devuelto Ejemplo: El código del bloque de acción anterior obtiene primero el tono del sonido dado y lo almacena en una variable local temporal temporal. A continuación, comprueba si se ha pulsado una tecla, y si se detecta una, se comprueba el valor del tono para ver si es menor que 2. Si lo es, el sonido se detiene, se añade 0,1 a la variable de tono, y luego se utiliza esta variable para establecer el tono del sonido antes de que se reproduzca.
