Si el audio está en pausa Esta acción se puede utilizar para comprobar si una instancia de cualquier recurso de sonido está actualmente en pausa en su juego. Se da el recurso de sonido a comprobar desde el explorador de activos y la función devolverá true si está en pausa y false si no lo está, aunque si marcas la bandera "no", esto se invierte y la función devolverá true si el sonido no está pausado y false en caso contrario. Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente: Estas acciones se ejecutarán ahora si el "if" se evalúa como verdadero, mientras que cualquier acción lanzada en otra parte se ejecutará después del bloque "if".
