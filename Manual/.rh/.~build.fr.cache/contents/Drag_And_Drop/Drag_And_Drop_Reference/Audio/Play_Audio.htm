Reproducir audio Esta acción hará que se reproduzca el sonido dado. Se selecciona el sonido en el Explorador de Activos y luego se establece si se desea que el sonido se reproduzca en bucle o no (un sonido en bucle se reproducirá una y otra vez hasta que se detenga) estableciendo la bandera "Bucle". Sintaxis de la acción: Argumentos: Argumento Descripción Sonido El recurso sonoro para jugar Bucle Si el sonido se reproduce en bucle o no (desactivado por defecto) Ejemplo: El código del bloque de acción anterior comprueba si se pulsa el botón del ratón y, si se detecta, comprueba una variable global variable global para ver si no es verdadera. Si no es verdadera, entonces se reproduce un sonido (y se hace un bucle) y la variable global se establece como verdadera, de lo contrario el sonido se detiene y la variable global se establece como falsa.
