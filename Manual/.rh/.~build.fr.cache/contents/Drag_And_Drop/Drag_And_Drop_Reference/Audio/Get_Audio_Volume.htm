Obtener volumen de audio Esta acción devuelve el volumen del sonido dado. Eliges el sonido desde el explorador de activos y le das una variable de destino para devolver el valor del volumen (puedes marcarla como una variable local temporal). El valor devuelto estará entre 0 y 1, donde 0 es silencio y 1 es volumen completo. Sintaxis de la acción: Argumentos: Argumento Descripción Sonido El recurso de sonido para obtener el volumen de Objetivo La variable de destino para almacenar el volumen devuelto Ejemplo: El código del bloque de acción anterior comprueba una variable global para ver si es verdadera o falsa. Si es verdadero, se recupera el volumen del sonido dado y se almacena en una variable local temporal. El valor se comprueba para ver si no es igual a 0,5, y si no lo es, entonces el volumen del sonido se establece en 0,5. Si la variable global se evalúa como falsa, entonces el volumen del sonido se fija en 1.
