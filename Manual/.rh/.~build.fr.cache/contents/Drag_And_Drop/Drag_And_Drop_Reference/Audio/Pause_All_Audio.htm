Pausar todo el audio Esta acción se puede utilizar para pausar todos los sonidos que se están reproduciendo, pero no se detendrá ningún sonido que se haya reproducido después de llamar a la acción. Puede comenzar a reproducirlos de nuevo utilizando la acción Reanudar todo el audio. Sintaxis de la acción: Ejemplo: El código del bloque de acción anterior comprobará una variable global y si se evalúa como verdadero entonces todo el sonido se pausa, de lo contrario todo se reanuda.
