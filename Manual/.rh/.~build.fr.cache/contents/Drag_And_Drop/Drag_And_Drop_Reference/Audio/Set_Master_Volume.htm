Establecer el volumen maestro Con esta acción puedes ajustar el volumen de todo el sistema de sonido. El valor del volumen que des puede estar entre 0 (silencio) y 1 (volumen completo) y la escala es lineal, de tal manera que un valor de 0,5 sería la mitad del volumen. Tenga en cuenta que esta acción afectará a todas las instancias de todos los sonidos que se están reproduciendo actualmente en la sala, y afectará a todas las instancias futuras de todos los sonidos reproducidos. Esta acción también es independiente del volumen establecido para cualquier sonido individual, por lo que si tiene un sonido configurado para reproducirse a un volumen de 0,5 y luego establece el volumen maestro a 0,5, el sonido se reproducirá a un volumen de 0,25. Sintaxis de la acción: Argumentos: Argumento Descripción Volumen El nuevo volumen maestro para todo el sonido (de 0 a 1, por defecto 1) Ejemplo: El código del bloque de acción anterior comprueba una variable global para ver si es verdadera o falso. Si es verdadero, se recupera el volumen maestro del sistema de sonido y se almacena en una variable local temporal. El valor se comprueba para ver si no es igual a 0,5, y si no lo es, entonces el volumen maestro del sistema de sonido se ajusta a 0,5. Si la variable global se evalúa como falsa, entonces el volumen maestro se establece en 1.
