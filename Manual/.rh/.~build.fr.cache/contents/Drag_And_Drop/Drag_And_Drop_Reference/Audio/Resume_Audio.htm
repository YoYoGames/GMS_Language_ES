Reanudar audio Esta acción se puede utilizar para reanudar un sonido previamente pausado. Esto puede ser usado en cualquier sonido que haya sido pausado usando Pausa de Audio o Pausa de Todo el Audio, y no tendrá efecto en aquellos sonidos que no han sido pausados previamente. Sintaxis de la acción: Argumentos: Argumento Descripción Sonido El recurso de sonido para reanudar la reproducción Ejemplo: El código del bloque de acción anterior comprobará una variable global y si se evalúa como verdadero entonces el sonido dado se pausa, de lo contrario se reanuda.
