Aplicar a... Esta acción se puede utilizar para cambiar el Ámbito de Acción para todas las acciones posteriores que se encadenen a ésta. Al igual que con una acción if, si se colocan más acciones a la derecha de la acción Aplicar a... a la derecha de la acción Aplicar a... entonces se ejecutarán desde el objeto o instancia con alcance. El ámbito se elige en el menú emergente que se muestra al hacer clic en la pequeña flecha situada en la parte superior derecha de la acción, y de esta forma se puede indicar un conjunto diferente de objetos - o incluso una sola instancia si tiene el ID de la instancia - para ejecutar múltiples acciones antes de continuar con el código del objeto actual. Tenga en cuenta que para añadir acciones en el bloque "Se aplica a", se deben soltar al lado de la acción, como se muestra en la imagen siguiente: Estas acciones se ejecutarán ahora desde el objeto o la instancia a la que se dirige el bloque "Se aplica a", mientras que cualquier acción lanzada en otro lugar se ejecutará después del bloque "Se aplica a". Sintaxis de la acción: Ejemplo: El código del bloque de acción anterior comprueba una variable para ver si es igual a 100 y si lo hace cambia el alcance a las instancias de " obj_Player" y les dice que ejecuten la acción Set Sprite ( todas las instancias de este objeto en la sala cambiarán de sprite en este punto), antes de continuar para añadir 1 a la variable dentro de la instancia que ejecuta el bloque de código completo.
