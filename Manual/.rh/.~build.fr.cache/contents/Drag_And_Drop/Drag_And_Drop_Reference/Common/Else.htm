Otros La acción "else" sólo se utiliza junto con la variable If (o cualquier otra acción "if"), y es una forma de definir un conjunto de acciones que deben realizarse si la evaluación "if" es falsa. Por ejemplo, puedes tener una acción "if" que compruebe si la variable de salud del jugador es menor o igual a 0, y si eso se evalúa como verdadero, destruyes la instancia. Sin embargo, si quieres realizar un conjunto diferente de Si quieres realizar un conjunto diferente de acciones de código si la salud del jugador no es menor que 0, entonces puedes añadir un "else" y luego especificar las acciones a realizar mientras el "if" se evalúa como falso.
