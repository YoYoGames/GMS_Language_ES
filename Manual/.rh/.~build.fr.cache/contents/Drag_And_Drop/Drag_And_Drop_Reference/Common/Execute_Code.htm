Ejecutar código Esta acción simplemente añade un bloque de editor de código en la lista de DnD™donde puede añadir cualquier código GML válido que necesite para el objeto. Tenga en cuenta que todo el código de la acción se ejecutará antes de que se ejecute la siguiente acción de la lista se ejecute. Para facilitar el seguimiento de lo que hacen estas acciones puede nombrarlas utilizando el identificador JSDoc @description, como se muestra en la imagen siguiente: Sintaxis de la acción: Ejemplo: El código del bloque de acción anterior crea un par de variables de ámbito global para crear una cuadrícula MP y una ruta.
