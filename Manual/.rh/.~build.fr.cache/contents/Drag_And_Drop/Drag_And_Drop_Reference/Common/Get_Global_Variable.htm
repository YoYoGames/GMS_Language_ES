Obtenga una variable global Esta acción se utiliza para recuperar el valor de una variable global y asignarlo a otra instancia o variable temporal (local). La variable a la que se asigna el valor global puede ser una creada previamente, o puede marcar la casilla "Temp" y dar un nuevo nombre de variable para que la acción cree una nueva variable temporal (local) específicamente para contener el valor devuelto. Tenga en cuenta que puede añadir variables adicionales en la misma acción haciendo clic en el icono más junto a la acción, y dando otro nombre de variable global y un nombre de variable local temporal. Para obtener información más avanzada sobre las variables, consulte la sección Variables y ámbito de las variables. Sintaxis de la acción: Argumentos: Argumento Descripción Nombre El nombre de la variable global para obtener el valor. Objetivo La variable de destino para mantener el valor devuelto - debería haber sido creada previamente si "Temp" no está marcada (ver más abajo). Temp Crear una variable temporal (local) para mantener el valor devuelto (desactivado por defecto) Ejemplo: El código del bloque de acción anterior recupera dos valores de variables globales y los asigna a variables temporales (locales) que luego se utilizan en una comparación "if".
