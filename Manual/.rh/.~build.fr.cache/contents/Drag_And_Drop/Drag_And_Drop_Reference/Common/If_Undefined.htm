Si no está definido Esta acción se utiliza para comprobar si una variable o expresión devuelve un valor "indefinido", y si es así esta acción devolverá true y si no lo es devolverá false . un valor indefinido puede causar un fallo o un error grave en tu proyecto, así que puedes usar esto para comprobarlo y detectarlo antes de que ocurra. Si marca el argumento "Not", entonces está comprobando si un valor es no es "indefinido". Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente: Estas acciones se ejecutarán ahora si el "if" se evalúa como verdadero, mientras que cualquier acción lanzada en otra parte se ejecutará después del bloque "if". Tenga en cuenta que puede "apilar" comprobaciones no definidas en la misma acción haciendo clic en el icono del signo más junto a la acción, y dando otra variable para comprobar. Esto comprobará si todas las variables dadas son indefinidas en una sola comprobación y sólo realizará el código de la acción añadida si lo son. Sintaxis de la acción: Argumentos: Argumento Descripción Variable El nombre de la variable a comprobar. No Se establece para comprobar si undefined no se evalúa como verdadero.
