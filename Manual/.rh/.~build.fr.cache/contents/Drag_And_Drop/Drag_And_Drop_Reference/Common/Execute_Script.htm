Ejecutar Script Esta acción puede utilizarse para llamar a una función de script o a una función definida por el usuario. En general, esta acción sólo se utilizaría para el soporte de legado y no debería usarse realmente en proyectos actuales o nuevos. Para utilizar la acción se le da al activo de la secuencia de comandos para llamar - o el nombre de la variable de la función definida por el usuario para llamar - y luego se suministran los argumentos que se requieren, es decir, cualquier valor (cadenas, enteros, índices de recursos, etc ...) que desea introducir en la secuencia de comandos o función para que pueda utilizarlos. Puede añadir los argumentos que necesite haciendo clic en el icono del signo más junto a la acción. Algunos scripts y funciones definidas por el usuario también pueden devolver un valor para su uso posterior en la lista de bloques de acción, por lo que se puede especificar la variable de destino que debe almacenar este valor de retorno. Esta puede haber sido creada previamente o puede ser una nueva variable temporal local (si marca la casilla "Temp"). Tanto el campo de destino como el de argumento pueden dejarse en blanco si no son necesarios. Tenga en cuenta que, por lo general, es mejor utilizar la acción Llamada a la Función para llamar a funciones definidas por el usuario, y que, por lo general, no debería necesitar utilizar esta acción, ni llamar a un activo de script directamente en el código. Sintaxis de la acción: Argumentos: Argumento Descripción Guión El nombre del script o de la función definida por el usuario a la que se va a llamar. Argumento0 ... Argumento3 Los diferentes argumentos (valores) que se van a pasar al script o a la función (los argumentos no utilizados pueden dejarse en blanco) Objetivo El nombre de la variable a la que se dirigen los valores devueltos (puede dejarse en blanco) Ejemplo: El código del bloque de acción anterior llama a una función definida por el usuario, pasando dos argumentos para una posición aleatoria dentro de la habitación.
