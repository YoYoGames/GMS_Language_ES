Retorno Esta acción es para usarla cuando se crean Funciones de Bloque de Acción y se usa para devolver un valor a la instancia que llama al script. El valor de retorno puede ser cualquier cosa que una variable pueda como una cadena, un valor numérico, un puntero o un ID de recurso - o puede ser una variable en sí misma. Tenga en cuenta que cuando llamamos a la acción de retorno en una función, la función saldrá - devolviendo el valor dado - y no se llamarán más acciones después del retorno, aunque si tienes acciones en el evento después de llamar a la función, éstas se seguirán ejecutando. Sintaxis de la acción: Argumentos: Argumento Descripción Volver El valor de retorno. Ejemplo: La función del bloque de acción anterior establece una variable temporal con la palabra clave nadie, y luego comprueba una variable de entrada para ver qué valor tiene actualmente. Si la variable es mayor que 100, entonces se crea una instancia y su ID de instancia único se almacena en la variable temporal, que luego se devuelve desde la función.
