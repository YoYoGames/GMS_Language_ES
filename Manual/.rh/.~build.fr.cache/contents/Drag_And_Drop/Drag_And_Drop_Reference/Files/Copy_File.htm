Copiar archivo Con esta acción puedes copiar cualquier archivo que haya sido guardado por tu juego o que haya sido incluido como parte de los Archivos Incluidos. Usted da el nombre del archivo a copiar (como una cadena) y luego el nombre del archivo a crear como una copia del archivo original (también como una cadena e incluyendo la extensión del archivo). El archivo dado se duplicará y se le dará el nuevo nombre. Sintaxis de la acción: Argumentos: Argumento Descripción Archivo El nombre (en forma de cadena) del archivo a copiar Copiar a El nombre (en forma de cadena) del archivo que se va a crear como copia Ejemplo: El código del bloque de acción anterior comprobará si un archivo ini no existe y si no existe entonces se crea copiando otro archivo.
