Guardar Buffer Con esta acción puede guardar un archivo que contenga los datos de un buffer. Se suministra el identificador único del buffer (tal y como se devolvió cuando se creó el buffer con Crear Buffer) y se da un nombre al archivo (como una cadena, con la extensión que elijas) y el buffer se escribirá en este archivo, listo para ser cargado de nuevo usando Cargar Buffer. Sintaxis de la acción: Argumentos: Argumento Descripción Tampón El valor único del ID del buffer a guardar Archivo El nombre (como una cadena) del archivo a guardar Ejemplo: El código del bloque de acción anterior creará una variable para mantener los datos del buffer y luego comprobará si se ha hecho un archivo de guardado del búfer se ha hecho previamente. Si el archivo existe, entonces se carga en la variable del búfer que creamos, pero si no existe, se crea, se escribe en él y luego se guarda listo para cargarlo la próxima vez.
