Si el archivo existe Con esta acción puede comprobar si existe un archivo con el nombre dado. Se selecciona el archivo a comprobar (como un sting e incluyendo la extensión) y la función devolverá true si existe, y false si no existe. Tenga en cuenta que esto devolverá true si el archivo existe en el paquete del juego como un Archivo Incluido o si el archivo ha sido guardado fuera mientras el juego se estaba ejecutando (ver la sección sobre El Sistema de Archivos para más información). Si el archivo existe, entonces se carga en la variable del búfer que creamos, pero si no existe, se crea, se escribe en él y luego se guarda listo para cargarlo la próxima vez.
