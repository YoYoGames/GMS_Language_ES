Si se muestra el teclado virtual Esta acción se puede utilizar para comprobar si el teclado virtual del sistema operativo se está mostrando o no. La acción evaluará a true cuando el teclado esté abierto o abriéndose, o false si el teclado virtual está cerrado o cerrándose. Si se habilita el modificador not, esta acción se invertirá y se puede almacenar el valor de retorno true / false en una variable de destino, que puede ser marcada como temp (local). Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente: Estas acciones se ejecutarán ahora si el bloque "if" se evalúa como verdadero, mientras que cualquier acción lanzada en otra parte se ejecutará después del bloque "if". Sintaxis de la acción: Ejemplo: El código del bloque de acción anterior comprueba la existencia de una acción de bajar el ratón, y si se detecta una, entonces se comprueba si se muestra el teclado virtual del sistema operativo. Si no lo está, entonces se ajusta para mostrarlo - usando un tipo de teclado numérico - y si ya se está mostrando, entonces se oculta.
