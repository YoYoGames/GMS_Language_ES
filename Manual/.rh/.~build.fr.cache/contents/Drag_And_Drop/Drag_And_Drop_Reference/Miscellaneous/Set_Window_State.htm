Establecer el estado de la ventana Con esta acción puedes establecer el estado de la ventana del juego a pantalla completa o a ventana. Aquí hay tres opciones: Alternar - Cuando se selecciona esta opción, cada vez que se llama a la acción se alternará la ventana entre los estados de pantalla completa y ventana Pantalla completa - Esto hará que la ventana pase a pantalla completa desde un estado de ventana Ventana - Esto hará que la ventana vuelva a un estado de ventana si está en pantalla completa Tenga en cuenta que el aspecto del juego en pantalla completa dependerá de la configuración de Opciones de Juego - Gráficos para las plataformas de destino, ya que si el juego no tiene algún mecanismo para ajustar a las diferentes relaciones de aspecto y resoluciones, puedes obtener un estiramiento (si la opción del juego está configurada como Escala completa), o puedes obtener barras negras añadidas (si la opción del juego está configurada como Mantener relación de aspecto). En general, si no tiene nada en su lugar para para tratar con diferentes resoluciones, deberías usar Keep Aspect Ratio para las plataformas de destino a las que quieras compilar. Sintaxis de la acción: Argumentos: Argumento Descripción Estado El estado de la ventana que se va a establecer (ver arriba las opciones) Ejemplo: El código del bloque de acción anterior cambiará el estado de la ventana entre pantalla completa y ventana cada vez que se pulse la tecla F4.
