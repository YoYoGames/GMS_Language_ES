Fijar la dirección de la variable Esta acción se utiliza para establecer una dirección para la instancia que no está fijada en incrementos de 45°. Se proporciona el valor de la dirección de 0° a 360° - donde derecha es 0°, arriba es 90°, izquierda es 180° y abajo es 270° - y también se puede establecer la bandera relativa para sumar/restar el valor que se da a la dirección actual. Sintaxis de la acción: Argumentos: Argumento Descripción Dirección Dirección de 0° a 360° (los valores fuera del rango se envolverán). Ejemplo: El código del bloque de acción anterior comprueba si el botón izquierdo del ratón está se mantiene presionado y si lo está, establece la dirección a un valor aleatorio de 0 a 360 y la velocidad a 5.
