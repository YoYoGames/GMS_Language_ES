Saltar al punto Esta acción se utiliza para mover una instancia de un lugar a otro. La instancia no se moverá suavemente a la nueva posición, sino que "saltará" para colocarse allí, por lo que si los valores son lo suficientemente grandes entonces "saltará" cualquier otra instancias en el camino. Puede establecer valores absolutos para mover la instancia, o puede marcar la bandera "relativa" para que la instancia se mueva en relación con la posición actual. Por ejemplo, si quiere que la instancia salte 10 píxeles a la izquierda solamente, debe establecer el valor x en -10 y el valor y en 0 y marcar "relativo" para ambos. Puedes utilizar valores negativos para las coordenadas absolutas y la instancia se moverá fuera del área de la sala base. Sintaxis de la acción: Argumentos: Argumento Descripción x La posición x a la que saltar y La posición y a la que saltar Ejemplo: El código del bloque de acción anterior comprueba si el botón izquierdo del ratón está pulsado, y si lo está, mueve la instancia a la posición del ratón.
