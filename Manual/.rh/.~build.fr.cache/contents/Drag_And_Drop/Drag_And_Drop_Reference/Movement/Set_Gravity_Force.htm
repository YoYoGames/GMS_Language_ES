Set Gravity Force Esta acción se utiliza para establecer la fuerza de gravedad que actúa sobre una instancia. Cada instancia puede tener una fuerza de gravedad diferente (y dirección) independientemente de todas las demás. La fuerza aplicada es un valor en píxeles por fotograma del juego, y es acumulativa, es decir: se es acumulativa, es decir, se añade en cada paso, por lo que si usted tiene una velocidad de 2 y una gravedad de 1, la velocidad se incrementará en 1 cada paso, lo que significa que normalmente querrá valores muy bajos aquí. Tenga en cuenta que puede introducir valores negativos para la gravedad y la instancia será "tirada" en la dirección opuesta a la dirección de la gravedad. Sintaxis de la acción: Argumentos: Argumento Descripción fuerza La fuerza de gravedad a aplicar (puede ser negativa) Ejemplo: El código del bloque de acción anterior establece varias propiedades de instancia relacionadas con el movimiento: la dirección de la gravedad se establece como abajo (270°), la fuerza de gravedad se establece como 0,05 y la fricción se establece como 0,2.
