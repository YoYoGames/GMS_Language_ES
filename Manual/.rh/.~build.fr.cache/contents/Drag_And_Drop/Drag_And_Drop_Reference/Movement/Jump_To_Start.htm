Jump To Start Esta acción se utiliza para mover una instancia desde su posición actual a la posición en la que fue creada originalmente. La posición original en la que se crea una instancia se almacena en dos variables de instancia llamadas xstart e ystart. Estas variables pueden utilizarse en cualquier otra acción y también pueden establecerse para cambiar la posición inicial registrada. Sintaxis de la acción: Ejemplo: El código del bloque de acción anterior comprueba si la parte superior del cuadro delimitador de la instancia está por debajo la altura de la sala y, si lo está, mueve la instancia de vuelta a la posición en la que fue creada.
