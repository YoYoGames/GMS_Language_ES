Biblioteca de Acción del Movimiento En cualquier juego, el movimiento y la posición son de suma importancia, por lo que GameMaker Studio 2 dispone de una completa selección de acciones para hacer frente a cualquier situación. Las dos formas principales de mover una instancia es establecer la posición real (x/y) dentro de la o establecer un vector de velocidad/dirección. Las acciones que se muestran aquí cubren ambos métodos y puedes utilizar la Variable de Asignación de Acción Común para establecer directamente la posición x e y por ti mismo.
