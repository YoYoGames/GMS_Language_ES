Establecer dirección fija Esta acción se utiliza para establecer una dirección de movimiento fija. Las direcciones disponibles están indicadas por los diferentes botones del bloque de acción, y al hacer clic en cualquiera de ellos se establecerá esa dirección. Tenga en cuenta que las direcciones en GameMaker Studio 2 se miden en grados, siendo la derecha 0°, arriba 90°, izquierda 180° y abajo 270°. Sintaxis de la acción: Ejemplo: El código del bloque de acción anterior comprueba si una pulsación del botón izquierdo del ratón en la instancia y si se detecta una, establece la dirección hacia abajo (270°) y la velocidad a 5.
