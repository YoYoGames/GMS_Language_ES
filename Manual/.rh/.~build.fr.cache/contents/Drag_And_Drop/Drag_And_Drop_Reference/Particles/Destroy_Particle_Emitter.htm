Destruye el emisor de partículas Esta acción "destruirá" el emisor de partículas dado, es decir, liberará la memoria utilizada por el emisor de partículas. Se suministra el ID del sistema de partículas al que pertenece el emisor (devuelto por la acción Crear Sistema de Partículas) y luego el ID del emisor real (como lo devuelve la acción Crear Emisor de Partículas). Esta acción debe ser llamada siempre que no necesites más un emisor de partículas en tu juego, o siempre que desees recrear el emisor de partículas (por ejemplo, justo antes de llamar a un reinicio del juego). Tenga en cuenta que si el sistema de partículas al que se ha asignado el emisor no ha sido marcado como Persistente entonces no necesita llamar a esta acción, ya que el emisor se limpiará automáticamente en la sala junto con el sistema de partículas. Sintaxis de la acción: Argumentos: Argumento Descripción Sistema El valor de identificación del sistema que contiene el emisor Emisor El valor del ID del emisor a destruir Ejemplo: El código del bloque de acción anterior comprobará una variable de instancia y si está por encima de 100 destruirá el emisor asignado a la instancia y luego pasará a la siguiente sala.
