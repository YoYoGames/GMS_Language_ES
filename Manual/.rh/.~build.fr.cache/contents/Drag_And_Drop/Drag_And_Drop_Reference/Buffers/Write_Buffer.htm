Buffer de escritura Esta acción puede utilizarse para escribir datos en un búfer. Se suministra el ID único del búfer en el que se va a escribir (devuelto por la acción Crear Búfer) y luego el valor a escribir, que puede ser un real o una cadena, pero tenga en cuenta que pero tenga en cuenta que todos los valores se almacenan como cadenas, así que cuando lea del búfer (usando Leer Búfer) puede que tenga que usar una de las Acciones de Tipo de Datos para convertirlo de una cadena al valor original. Ten en cuenta también que los búferes se leen y escriben secuencialmente, por lo que escribir en un búfer moverá la posición de "búsqueda" a lo largo de uno, lo que significa que para escribir varias piezas de datos sólo tienes que llamar a esta acción el número de veces necesario.
