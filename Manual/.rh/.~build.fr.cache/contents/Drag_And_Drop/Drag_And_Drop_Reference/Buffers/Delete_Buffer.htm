Borrar Buffer Esta acción se puede utilizar para borrar un buffer de la memoria, liberando los recursos utilizados para crearlo y eliminando cualquier dato que pueda contener actualmente. Se suministra el valor único del ID del búfer devuelto por la acción Crear búfer. Tenga en cuenta que puede seleccionar varios búferes para eliminarlos haciendo clic en el icono más junto a la acción, y añadiendo otro ID de búfer. NOTA: Es importante eliminar siempre los recursos creados dinámicamente de la memoria cuando ya no los necesite para evitar fugas de memoria. Sintaxis de la acción: Argumentos: Argumento Descripción Tampón El índice del buffer (almacenado en una variable) Ejemplo: El código del bloque de acción anterior comprueba si existe un búfer y, si es así, lo elimina.
