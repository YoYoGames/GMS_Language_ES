Si existe el búfer Esta acción puede utilizarse para comprobar si un ID de búfer dado es válido y el búfer existe realmente en la memoria. Si el ID del búfer es válido y existe, la acción devolverá true, de lo contrario devolverá false. Se puede marcar "no" para que la acción compruebe si un búfer no existe.
