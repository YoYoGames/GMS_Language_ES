Buffer de copia Esta acción puede utilizarse para copiar todos los datos almacenados en un búfer a otro. Cuando se utilizan dos búferes y se copia de uno a otro, ambos búferes deben haber sido creados previamente utilizando la acción Crear Búfer, y suministrar el ID de búfer único devuelto como argumentos de origen y destino. NOTA: No se puede copiar en el mismo búfer.
