Set Vidas Con esta acción puedes crear o establecer una variable de instancia para vidas. Se suministra el valor para establecer la variable de vidas o para añadirlo a la variable de vidas (si se marca la casilla de verificación relativa, el valor se añadirá al valor actual de "vidas actual). Tenga en cuenta que, a diferencia de las variables de instancia normales, el nombre de la variable "vidas" se almacena internamente, por lo que para recuperar el valor de la puntuación debe utilizar la acción Obtener vidas. Sintaxis de la acción: Argumentos: Argumento Descripción Vidas La cantidad a la que se fijan las vidas (o a la que se añaden las vidas si está marcada la opción relativa ) Ejemplo: El código del bloque de acción anterior comprobará si la sala es la primera del juego y si lo es establece las variables "score", "lives" y "health" para la instancia.
