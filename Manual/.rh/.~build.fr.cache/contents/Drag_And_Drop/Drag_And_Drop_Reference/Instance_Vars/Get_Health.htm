Get Health Puede utilizar esta acción para recuperar el valor de la variable de instancia creada para la "salud" del objeto. Usted proporciona la variable de destino para almacenar el valor de "salud" devuelto (que puede ser marcado como una variable local temporal), pero tenga en cuenta que si la variable no ha sido establecida previamente (usando Set Health) entonces la variable será creada por usted y la acción devolverá 0 (a diferencia de las variables de instancia regulares que darán un error si intenta acceder a ellas sin haberlas establecido primero). Sintaxis de la acción: Argumentos: Argumento Descripción Objetivo La variable de destino para almacenar el valor de "salud" devuelto (puede marcarse como una variable local temporal) Ejemplo: El código del bloque de acción anterior recuperará el valor de "salud" y lo almacenará en una variable local temporal temporal. Esta variable se utiliza entonces para dibujar una barra de salud para la instancia.
