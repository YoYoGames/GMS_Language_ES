Obtener puntuación Puede utilizar esta acción para recuperar el valor de la variable de instancia creada para la "puntuación". Usted proporciona la variable de destino para almacenar el valor de la "puntuación" devuelta (que puede ser marcada como una variable local temporal), pero tenga en cuenta que si la variable no ha sido configurada previamente (usando Set Score) entonces la variable será creada por usted y la acción devolverá 0 (a diferencia de las variables de instancia regulares que darán un error si intenta acceder a ellas sin haberlas configurado antes). Sintaxis de la acción: Argumentos: Argumento Descripción Objetivo La variable de destino para almacenar el valor de "puntuación" devuelto (puede marcarse como una variable local temporal) Ejemplo: El código del bloque de acción anterior recuperará el valor actual de la puntuación y lo almacenará en una variable local temporal temporal. Esta variable se convierte en una cadena y se almacena en otra variable local temporal antes de ser dibujada en la pantalla.
