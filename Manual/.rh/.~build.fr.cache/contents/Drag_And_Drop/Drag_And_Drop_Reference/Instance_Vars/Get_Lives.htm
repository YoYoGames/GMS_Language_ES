Consigue vidas Puede utilizar esta acción para recuperar el valor de la variable de instancia creada para las "vidas". Usted suministra la variable de destino para almacenar el valor de las "vidas" devuelto (que puede ser marcado como una variable local temporal), pero tenga en cuenta que si la variable no ha sido configurada previamente (usando Set Lives) entonces la variable será creada por usted y la acción devolverá 0 (a diferencia de las variables de instancia regulares que darán un error si intenta acceder a ellas sin haberlas configurado antes). Sintaxis de la acción: Argumentos: Argumento Descripción Objetivo La variable de destino para almacenar el valor de "vidas" devuelto (puede marcarse como una variable local temporal) Ejemplo: El código del bloque de acción anterior recuperará el valor de "vidas" y lo almacenará en una variable local temporal temporal. Esta variable se utiliza entonces para decidir cuántos sprites apilados deben dibujarse.
