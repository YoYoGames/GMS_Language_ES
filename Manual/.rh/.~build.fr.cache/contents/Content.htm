Bienvenido al manual de usuario de GameMaker Studio 2. Este documento está dividido en tres partes con el objetivo de que conozcas la interfaz y el funcionamiento básico de GameMaker Studio 2 y la programación general antes de pasar a un uso más avanzado y las funciones disponibles a través de nuestro lenguaje de scripting propietario GameMaker GML o nuestra herramienta de scripting visual Drag and Drop™. Para facilitarte las cosas, hemos creado una práctica guía de inicio rápido, e incluso si estás familiarizado con otras herramientas de creación de juegos, te recomendamos que empieces por ahí para tener una visión general del funcionamiento de GameMaker Studio 2. Verás que GameMaker Studio 2 es fácil e intuitivo de usar y, sea cual sea tu nivel de conocimientos, pronto podrás crear los juegos que siempre has soñado. NOTA: Debido a la naturaleza modular de GameMaker Studio 2 hay ciertas cosas que no se incluyen en este manual de usuario, en particular los aspectos de configuración y conexión de los diferentes dispositivos que pueden ser soportados. Para obtener información actualizada y consejos para la resolución de problemas relacionados con las diferentes plataformas de destino, consulta el Centro de Ayuda de YoYo Games. Cómo empezar Esta sección es la primera parada obligatoria para utilizar GameMaker Studio 2 y familiarizarse con el contenido del IDE y su funcionamiento. Verás que GameMaker Studio 2 es intuitivo y sencillo de usar, e incluso si nunca has usado ninguna incluso si nunca has utilizado ninguna herramienta de este tipo, no deberías tener ningún problema para empezar a crear juegos después de leer la Introducción y completar la Guía de Inicio Rápido ( más abajo). Una vez que te hayas familiarizado con los básicos puedes pasar a la mirada más profunda de los componentes de GameMaker Studio 2 en la sección de Referencia IDE más abajo.
